简介

欢迎使用 Logto 快速入门。本文档包含将 Logto 集成到你的应用程序中的实用指南。

大多数指南使用 Logto 官方 SDK。如果你有任何问题，请随时联系我们。

语言框架

Android (Kotlin / Java)Android 集成指南。AngularAngular 是一个用于构建用户界面的 JavaScript 库。Capacitor JSCapacitor JS 是一个用于构建混合移动应用的框架。Chrome extensionChrome 扩展是运行在 Google Chrome 中的小型 Web 应用程序。.NET Core (Blazor Server)将 Logto 集成到你的 .NET Core Blazor Server 应用中。.NET Core (Blazor WASM)将 Logto 集成到你的 .NET Core Blazor WebAssembly 应用中。.NET Core (MVC)将 Logto 集成到你的 .NET Core Web 应用中，使用模型-视图-控制器 (MVC)。.NET Core (Razor Pages)将 Logto 集成到你的 .NET Core Web 应用中，使用 Razor Pages 和视图模型。Expo (React Native)使用 React 创建通用原生应用程序，可在 Android、iOS 和 Web 上运行。ExpressExpress 是一个极简且灵活的 Node.js Web 应用框架，为 Web 和移动应用提供了一套强大的功能。FlutterFlutter 是一个用于构建混合移动应用的框架。FlutterFlowFlutterFlow 是一个用于构建混合 Flutter 应用的低代码框架。GoGo 是一种高效、并发、静态类型的编程语言，适用于 web 应用程序。Next.js (Pages Router)Next.js 是一个用于生产环境的 React 框架 - 它使构建全栈 React 应用变得轻而易举，并且内置了 SSR。Next.js (App Router)Next.js App Router 是一种使用 React 最新特性构建应用的新范式。Next AuthNext.js 的认证 (Authentication)。NuxtNuxt 是一个开源框架，使得 Web 开发直观且强大。Passport.jsPassport 是 Node.js 的认证 (Authentication) 中间件。PHP将 Logto 集成到你的 PHP Web 应用中，例如 Laravel。Python将 Logto 集成到你的 Python Web 应用中，例如 Django 和 Flask。ReactReact 是一个用于构建用户界面的 JavaScript 库。RubyRuby 是一种动态的开源编程语言，注重简单性和生产力。SvelteKitSvelteKit 是一个使用 Svelte 快速开发健壮、高性能 Web 应用的框架。iOS (Swift)iOS (Swift) 应用集成指南。Vanilla JS与框架无关的 JavaScript 集成。VueVue.js 是一个渐进式、可逐步采用的 JavaScript 框架，用于在 Web 上构建 UI。WebflowWebflow 是一个用于网站构建和托管的 SaaS 平台。WordPressWordPress 是一个免费且开源的内容管理系统。

无框架与通用

机器对机器启用机器之间的直接通信。传统 Web将 Logto 集成到在服务器上运行的传统 Web 应用中。

数据库

Hasura将 Logto 集成到 Hasura 中。Supabase将 Logto 集成到 Supabase 中。

第三方应用 (Logto as Idp)

OIDC (Third-party app)使用 Logto 作为你的应用程序的第三方 OIDC 身份提供商 (IdP)。
