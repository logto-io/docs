# Fluxos do usuário final

Os fluxos do usuário final cobrem todos os processos de verificação para interações do usuário, categorizados da seguinte forma:

- **Fluxos de autenticação (Authentication)**: Envolve login do usuário, registro e redefinição de senha. Logto oferece experiências prontas para uso configuradas em <CloudLink to="/sign-in-experience">Console > Experiência de login</CloudLink>, e você pode [trazer sua UI](/customization/bring-your-ui) via [Experience APIs](https://openapi.logto.io/group/endpoint-experience).
- **Fluxos de conta**: Envolve configurações de conta, perfil do usuário e verificação de segurança. Use [Account APIs](https://openapi.logto.io/group/endpoint-account-center) ou [Management APIs](https://openapi.logto.io/operation/operation-getuser) para implementar.
- **Fluxos de organização (Organization)**: Para produtos de multi-tenancy, como serviços B2B ou SaaS. Envolve criação de organização, convite de membros e gerenciamento. Use [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) para integrar isso em seus aplicativos.
