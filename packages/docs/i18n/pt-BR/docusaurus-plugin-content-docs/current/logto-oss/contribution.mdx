# Contribuição

Obrigado pelo seu interesse em contribuir para o Logto! Agradecemos as contribuições da comunidade para tornar nosso projeto mais robusto, amigável e versátil. Seja corrigindo bugs, melhorando a documentação, criando seu conector, adicionando novos recursos ou oferecendo feedback, seus esforços são inestimáveis.

## Como contribuir? \{#how-to-contribute}

Nos esforçamos para tornar a contribuição o mais simples possível. Antes de começar, por favor, revise nosso [Guia de Contribuição no GitHub](https://github.com/logto-io/logto/blob/master/.github/CONTRIBUTING.md). Este guia descreve nossos processos, padrões de codificação e os passos para fazer contribuições bem-sucedidas para o Logto.

## Precisa de ajuda? \{#need-help}

Se você tiver dúvidas ou precisar de orientação, sinta-se à vontade para entrar em contato através de nossas [discussões no GitHub](https://github.com/logto-io/logto/discussions), [abrir uma issue](https://github.com/logto-io/logto/issues) ou [nos contatar](https://logto.io/contact), e garantiremos que você tenha tudo o que precisa para ter sucesso.
