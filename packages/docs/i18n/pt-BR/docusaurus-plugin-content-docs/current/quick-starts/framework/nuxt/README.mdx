## Uso em middleware ou rotas de API \{#use-in-middleware-or-api-routes}

Os composables `useLogtoClient()` e `useLogtoUser()` não estão disponíveis em middleware ou rotas de API. Você pode usar a função `logtoEventHandler()` para obter o cliente Logto e outro contexto:

```ts title="api/auth/get-access-token.ts"
import { logtoEventHandler } from '@logto/nuxt';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
