Inicie o cliente criando uma instância de `LogtoClient` com um objeto `LogtoConfig`.

```swift title="ContentView.swift"
import Logto
import LogtoClient

let config = try? LogtoConfig(
  endpoint: "<your-logto-endpoint>", // Ex.: http://localhost:3001
  appId: "<your-app-id>"
)
let client = LogtoClient(useConfig: config)
```

:::info

Por padrão, armazenamos credenciais como Token de ID (ID Token) e Token de atualização (Refresh Token) no Keychain. Assim, o usuário não precisa fazer login novamente quando retornar.

Para desativar esse comportamento, defina `usingPersistStorage` como `false`:

```swift
let config = try? LogtoConfig(
  // ...
  usingPersistStorage: false
)
```

:::
