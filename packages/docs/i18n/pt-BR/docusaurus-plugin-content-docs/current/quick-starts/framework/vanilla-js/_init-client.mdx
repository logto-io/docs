Importe e inicie uma instância de `LogtoClient` passando a configuração:

```ts title="index.js"
import LogtoClient from '@logto/browser';

const logtoClient = new LogtoClient({
  endpoint: '<your-logto-endpoint>',
  appId: '<your-application-id>',
});
```

O `endpoint` e o `appId` podem ser encontrados na página de detalhes do aplicativo no Logto Console.
