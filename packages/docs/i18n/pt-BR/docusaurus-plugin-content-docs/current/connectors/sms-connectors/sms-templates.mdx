Logto fornece quatro modelos diferentes para personalizar o conteúdo de SMS, que são categorizados com base no tipo de uso: Register, SignIn, ForgotPassword e Generic. É altamente recomendável que você use modelos diferentes para vários casos de uso, caso contrário, pode atingir o limite de taxa, levando a uma interrupção temporária do seu serviço.

## Tipos de modelos de SMS e exemplos \{#sms-template-types-and-examples}

Aqui estão alguns exemplos apenas para referência:

| usageType                | Cenário                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | Exemplos de modelos                                                                         |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| SignIn                   | Usuários [fazem login usando seu número de telefone](/end-user-flows/sign-up-and-sign-in/sign-in) e verificam inserindo o código de verificação SMS em vez de inserir uma senha.                                                                                                                                                                                                                                                                                                                                                              | Código de verificação de login do Logto: `{{code}}`. Expira em 10 minutos.                  |
| Register                 | Usuários [criam uma conta usando seu número de telefone](/end-user-flows/sign-up-and-sign-in/sign-up) e a verificam inserindo um código de verificação enviado pelo Logto para seu número de telefone.                                                                                                                                                                                                                                                                                                                                        | Código de verificação de inscrição do Logto: `{{code}}`. Expira em 10 minutos.              |
| ForgotPassword           | Se os usuários esquecerem sua senha durante o login, eles podem optar por verificar sua identidade usando o número de telefone primeiro para [redefinir a senha](/end-user-flows/sign-up-and-sign-in/reset-password).                                                                                                                                                                                                                                                                                                                         | Código de verificação de redefinição de senha do Logto: `{{code}}`. Expira em 10 minutos.   |
| Generic                  | Este modelo pode ser usado como uma opção de backup geral para vários cenários, incluindo testar configurações de conector, [verificar ou vincular número de telefone após o login](/end-user-flows/account-settings/by-management-api#email-and-phone-number-verification), e assim por diante.                                                                                                                                                                                                                                              | Código de verificação do Logto: `{{code}}`. Expira em 10 minutos.                           |
| OrganizationInvitation   | Use este modelo para [enviar aos usuários um link de convite](/end-user-flows/organization-experience/invite-organization-members#configure-your-email-connector) para ingressar na organização.                                                                                                                                                                                                                                                                                                                                              | Código de verificação de convite da organização do Logto: `{{code}}`. Expira em 10 minutos. |
| UserPermissionValidation | Durante o uso do aplicativo, pode haver algumas operações de alto risco ou operações com um nível de risco relativamente alto que [requerem verificação adicional do usuário](/end-user-flows/account-settings/by-account-api#verify-by-sending-a-verification-code-to-the-users-email-or-phone), como transferências bancárias, exclusão de recursos em uso e cancelamento de assinaturas. O modelo `UserPermissionValidation` pode ser usado para definir o conteúdo do código de verificação SMS que os usuários recebem nessas situações. | Código de verificação do Logto: `{{code}}`. Expira em 10 minutos.                           |
| BindNewIdentifier        | Quando um usuário modifica seu perfil, ele pode [vincular um número de telefone à sua conta atual](/end-user-flows/account-settings/by-account-api#manage-phone). Nesse caso, o modelo `BindNewIdentifier` pode ser usado para personalizar o conteúdo do SMS de verificação.                                                                                                                                                                                                                                                                 | Código de verificação de vinculação de conta do Logto: `{{code}}`. Expira em 10 minutos.    |

É importante entender esses parâmetros:

- O código de verificação é válido por 10 minutos. Atualmente, não suportamos personalização no tempo de expiração.
- Logto substituirá o espaço reservado `{{code}}` no modelo de SMS por um código de verificação. Portanto, certifique-se de que o modelo tenha um espaço reservado reservado.

:::note
Alguns países e regiões podem não permitir o envio de conteúdo não aprovado via SMS devido a requisitos de conformidade. Os modelos de SMS precisam ser registrados e aprovados pelo provedor de SMS antes de serem usados. Nesses casos, o conteúdo pode ser indexado pelo ID do modelo para o modelo correspondente.
:::
