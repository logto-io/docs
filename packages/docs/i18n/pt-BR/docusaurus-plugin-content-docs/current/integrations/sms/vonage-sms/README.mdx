# Configurar verificação SMS com Vonage SMS

O conector oficial do Logto para Vonage SMS.

<GuideTip />

## Introdução \{#get-started}

Vonage é um provedor global de comunicações, oferecendo serviços de comunicação robustos baseados em nuvem, incluindo SMS (serviço de mensagens curtas). O Conector Vonage SMS é um plugin fornecido pela equipe Logto para permitir que os usuários finais do Logto registrem e façam login em sua conta Logto via códigos de verificação SMS.

## Configurar no Vonage \{#set-up-in-vonage}

:::tip
Você pode pular esta etapa se já a tiver concluído.
:::

Para trabalhar com este conector, você precisará [criar uma conta](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) no Vonage. Isso lhe dará uma chave de API e um segredo que você pode usar para acessar as APIs através deste conector.

Depois de ter uma conta, você pode encontrar sua chave de API e segredo de API no topo do Painel de API do Vonage.

E você pode precisar [alugar um número virtual](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) para enviar mensagens SMS.

Consulte a [API Vonage SMS](https://developer.vonage.com/en/messaging/sms/overview) para mais informações.

## Configurar no Logto \{#set-up-in-logto}

1. **Chave de API**: Sua chave de API do Vonage.
2. **Segredo de API**: Seu segredo de API do Vonage.
3. **Nome da Marca**: O nome da marca que você deseja usar para enviar o SMS, isso também é chamado de campo `from`, veja a [Identidade do Remetente](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) para mais informações.
4. **Modelos**: Os modelos que você deseja usar para enviar o SMS, você pode usar os modelos padrão ou modificá-los conforme necessário.
