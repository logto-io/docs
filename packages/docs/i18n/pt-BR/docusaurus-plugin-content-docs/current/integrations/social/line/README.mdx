# Configurar login social com Line

O conector oficial do Logto para login social com Line.

<GuideTip />

## Introdução \{#get-started}

O conector Line permite que os usuários finais façam login no seu aplicativo usando suas próprias contas Line através do protocolo de autenticação Line OAuth 2.0.

<Integration />

## Testar conector Line \{#test-line-connector}

É isso. O conector Line deve estar disponível agora. Não se esqueça de [Habilitar conector social na experiência de login](/connectors/social-connectors/#enable-social-sign-in).
