# Configurar login social com Feishu

Feishu Web: Uma plataforma avançada de colaboração e gestão empresarial, oferecendo colaboração de escritório perfeita e alinhando metas de equipe para ativar totalmente organizações e indivíduos.

<GuideTip />

## Introdução ao login social com Feishu \{#getting-started-with-feishu-social-sign-in}

O conector web do Feishu é projetado para aplicativos web de desktop e utiliza o processo de autenticação OAuth 2.0.

## Registrar uma conta de desenvolvedor Feishu \{#register-a-feishu-developer-account}

Se você não possui uma conta de desenvolvedor Feishu, registre-se na [Plataforma Aberta Feishu](https://open.feishu.cn/app).

## Criar um aplicativo \{#create-an-application}

1. No [Console do Desenvolvedor](https://open.feishu.cn/app), clique em "Criar App Personalizado".
2. Preencha o nome do aplicativo, descrição, selecione um ícone e clique no botão "Criar".
3. Na barra lateral esquerda, clique em "Configurações de Segurança", preencha o "URL de Redirecionamento" como `${logto_endpoint}/callback/${connector_id}`. O valor correspondente pode ser encontrado no campo `Callback URI` na página de detalhes do conector Feishu no Console Logto.
4. Em "Credenciais e Informações Básicas", você pode obter o "App ID" e o "App Secret".

:::note
Para uso interno não empresarial, você também precisa clicar no botão "Criar uma versão" na página "Gerenciamento de Versão e Lançamento".
O "App ID" e o "App Secret" só entrarão em vigor após o status do aplicativo mudar para "Habilitado."
:::
