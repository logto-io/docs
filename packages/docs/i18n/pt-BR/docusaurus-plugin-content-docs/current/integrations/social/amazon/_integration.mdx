### Configurar um aplicativo Amazon \{#setup-a-amazon-app}

Vá para o [Amazon Developer Portal](https://developer.amazon.com) e faça login com sua conta. Se você não tiver uma conta, pode se registrar para obter uma.

Em seguida, crie um aplicativo.

**Passo 1:** Vá para o bloco "Login with Amazon" na página inicial.

Ou você pode visitar [esta página](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html).

Siga o guia para criar um novo perfil de segurança.

Após criar o perfil de segurança, você obterá um Client ID e Client Secret da tabela.

**Passo 2:** Configurar URLs de callback.

Clique no botão "Manage" no lado direito da tabela e vá para a aba "Web Settings".

No campo "Allowed redirect URLs", adicione a seguinte URL:

- `${your_logto_origin}/callback/${connector_id}`.

O `connector_id` pode ser encontrado na barra superior da página de detalhes do conector no Logto Admin Console.

### Configurar seu conector \{#configure-your-connector}

Na configuração do seu conector Logto, preencha os seguintes campos com os valores obtidos na última etapa:

- **clientId:** O Client ID do seu aplicativo.
- **clientSecret:** O Client Secret do seu aplicativo.

`scope` é uma lista de escopos OIDC delimitada por espaços. Se não for fornecido, o escopo padrão é `openid profile`.

### Tipos de configuração \{#config-types}

| Nome         | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
