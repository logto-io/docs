### Configurar um canal Line \{#setup-a-line-channel}

Vá para o [Line Developers](https://developers.line.biz/console/) e faça login com sua conta comercial do Line. Se você não tiver uma conta, pode se registrar para obter uma.

Em seguida, acesse [este link](https://developers.line.biz/console/register/line-login/provider/) para criar um canal.

**Passo 1:** Preencha os Detalhes do Canal.

Complete o formulário e crie o canal.

**Passo 2:** Configure as URLs de callback.

Vá para a página de detalhes do canal, encontre a aba "LINE login" e edite o campo "Callback URL".

No nosso caso, será `${your_logto_endpoint}/callback/${connector_id}`. por exemplo, `https://foo.logto.app/callback/${connector_id}`. O `connector_id` pode ser encontrado na barra superior da página de detalhes do conector no Logto Admin Console.

### Configure seu conector \{#configure-your-connector}

Na configuração do seu conector Logto, preencha os seguintes campos com os valores obtidos na seção "Credenciais do aplicativo" da aba "Auth" do seu App:

- **clientId:** O ID do Canal do seu App.
- **clientSecret:** O Segredo do Canal do seu App.

`scope` é uma lista de escopos OIDC delimitada por espaços. Se não for fornecido, o escopo padrão é `openid profile`.

### Tipos de configuração \{#config-types}

| Nome         | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
