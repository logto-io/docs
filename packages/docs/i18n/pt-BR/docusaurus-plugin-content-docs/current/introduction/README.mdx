# Introdução ao Logto

Bem-vindo à documentação do Logto! Logto é uma solução de gerenciamento de identidade e acesso (IAM) projetada para aplicativos modernos e produtos SaaS. Ele fornece um sistema de autenticação e autorização seguro, escalável e personalizável para suas aplicações.
