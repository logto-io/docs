# 设计策略

- 每种编程语言都应该有一个名为 `${language}` 的独立 git 仓库。
- 每个编程语言仓库应该是一个 mono repo。
- SDK 及其相关示例项目都应放置在此仓库下。

示例：

<details>
  <summary>

### JS \{#js}

</summary>

- js (core)
- react
- react-sample

</details>

<details>
  <summary>

### Kotlin \{#kotlin}

</summary>

- kotlin (core)
- android
- android-sample

</details>
