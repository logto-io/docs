# 用户管理

用户管理是 Logto 的一个重要关注领域。你可以使用 Logto 控制台或 Management API 来访问和管理你的用户数据，这两者都非常有效。使用这些工具，你可以执行以下任务：

- 按用户名、ID、全名、电话号码或电子邮件搜索用户。
- 对用户执行多个操作，例如解除社交账户绑定、更新个人资料信息、删除用户或重置密码。
- 检查用户日志。
- 分配角色 (Role) 以进行访问控制。
- 创建和管理个人访问令牌 (Access token)。
