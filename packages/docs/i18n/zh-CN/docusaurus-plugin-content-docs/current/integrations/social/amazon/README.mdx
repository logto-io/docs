# 设置 Amazon 社交登录

Logto 官方的 Amazon 社交登录连接器。

<GuideTip />

## 开始使用 \{#get-started}

Amazon 连接器使终端用户能够通过 Amazon OAuth 2.0 认证 (Authentication) 协议使用他们自己的 Amazon 账户登录到你的应用程序。
