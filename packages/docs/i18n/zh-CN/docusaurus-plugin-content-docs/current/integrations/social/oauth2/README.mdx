# 设置 OAuth 2.0 协议的社交登录

Logto 的官方 OAuth 2.0 协议连接器。

<GuideTip />

## 开始使用 \{#get-started}

OAuth 连接器使 Logto 能够连接到支持 OAuth 2.0 协议的任意社交身份提供商。

:::note

OAuth 连接器是 Logto 中一种特殊的连接器，你可以添加一些基于 OAuth 协议的连接器。

:::
