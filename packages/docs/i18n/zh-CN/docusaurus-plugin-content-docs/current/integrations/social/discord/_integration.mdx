### 注册开发者应用程序 \{#register-a-developer-application}

- 访问 [Discord Developer Portal](https://discord.com/developers/applications) 并使用你的 Discord 账户登录。
- 点击 **New Application** 按钮创建一个应用程序，为其选择一个名称（例如：LogtoAuth），勾选框并点击 **Create**。
- 转到 **OAuth2** 页面并点击 **Reset Secret**
- 记录下 **CLIENT ID** 和 **CLIENT SECRET** 字段
- 添加有效的重定向（例如：**`http://auth.mycompany.io/callback/${connector_id}`**）。`connector_id` 可以在 Logto 管理控制台连接器详情页面的顶部栏找到。

## 配置 Logto \{#configure-logto}

### 配置类型 \{#config-types}

| 名称         | 类型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |

#### clientId \{#clientid}

`clientId` 是我们之前保存的 `CLIENT ID` 字段。
（你可以在 Discord Developer Portal 的 OAuth2 页面找到它。）

#### clientSecret \{#clientsecret}

`clientSecret` 是我们之前保存的 `CLIENT SECRET`。
（如果你丢失了它，你需要点击 **Reset Secret**。）

#### scope \{#scope}

`scope` 是用户授权授予的权限。默认值是 `identify email`。
你可以在 [这里](https://discord.com/developers/docs/topics/oauth2#shared-resources-oauth2-scopes) 查看完整的权限列表。
