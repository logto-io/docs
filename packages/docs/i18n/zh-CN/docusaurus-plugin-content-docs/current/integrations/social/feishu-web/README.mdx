# 设置 Feishu 社交登录

Feishu Web：一个先进的企业协作和管理平台，提供无缝的办公协作和团队目标对齐，以充分激活组织和个人。

<GuideTip />

## 开始使用 Feishu 社交登录 \{#getting-started-with-feishu-social-sign-in}

Feishu web 连接器专为桌面 Web 应用程序设计，并使用 OAuth 2.0 认证 (Authentication) 过程。

## 注册 Feishu 开发者账号 \{#register-a-feishu-developer-account}

如果你没有 Feishu 开发者账号，请在 [Feishu 开放平台](https://open.feishu.cn/app)上注册。

## 创建应用程序 \{#create-an-application}

1. 在 [开发者控制台](https://open.feishu.cn/app)上，点击“创建自定义应用”。
2. 填写应用名称、描述，选择一个图标，然后点击“创建”按钮。
3. 在左侧边栏中，点击“安全设置”，填写“重定向 URL”为 `${logto_endpoint}/callback/${connector_id}`。相应的值可以在 Logto 控制台的 Feishu 连接器详情页面的 `Callback URI` 字段中找到。
4. 在“凭证和基本信息”中，你可以获取“App ID”和“App Secret”。

:::note
对于非企业内部使用，你还需要在“版本管理和发布”页面中点击“创建版本”按钮。
“App ID”和“App Secret”只有在应用状态变为“已启用”后才会生效。
:::
