# 设置 X 社交登录

Logto 官方连接器用于 X（前称 Twitter）社交登录。

<GuideTip />

## 开始使用 \{#get-started}

X (Twitter) 连接器使终端用户能够通过 X OAuth 2.0 认证 (Authentication) 协议使用他们自己的 X (Twitter) 账户登录你的应用程序。
