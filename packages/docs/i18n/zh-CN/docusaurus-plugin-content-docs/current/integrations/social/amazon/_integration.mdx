### 设置 Amazon 应用 \{#setup-a-amazon-app}

前往 [Amazon Developer Portal](https://developer.amazon.com) 并使用你的账户登录。如果你没有账户，可以注册一个。

然后，创建一个应用。

**步骤 1：** 在主页中进入“Login with Amazon”模块。

或者你可以访问[此页面](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html)。

按照指南创建一个新的安全配置文件。

创建安全配置文件后，你将从表中获得 Client ID 和 Client Secret。

**步骤 2：** 设置回调 URL。

点击表格右侧的“Manage”按钮，进入“Web Settings”选项卡。

在“Allowed redirect URLs”字段中，添加以下 URL：

- `${your_logto_origin}/callback/${connector_id}`。

`connector_id` 可以在 Logto 管理控制台连接器详情页面的顶部栏找到。

### 配置你的连接器 \{#configure-your-connector}

在你的 Logto 连接器配置中，用上一步获得的值填写以下字段：

- **clientId：** 你的应用的 Client ID。
- **clientSecret：** 你的应用的 Client Secret。

`scope` 是一个以空格分隔的 OIDC 权限 (Scopes) 列表。如果未提供，默认权限 (Scope) 是 `openid profile`。

### 配置类型 \{#config-types}

| 名称         | 类型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
