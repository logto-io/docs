# 设置 OpenID Connect (OIDC) 社交登录

Logto 的 OpenID Connect (OIDC) 协议官方连接器。

<GuideTip />

## 开始使用 \{#get-started}

OIDC 连接器使 Logto 能够连接到支持 OIDC 协议的任意社交身份提供商 (IdP)。

:::note

OIDC 连接器是 Logto 中一种特殊的连接器，你可以添加一些基于 OIDC 的连接器。

:::
