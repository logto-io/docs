# 使用 AWS Direct Mail 设置电子邮件验证

Logto 的官方连接器用于 AWS 的直接邮件服务。

<GuideTip />

## 开始使用 \{#get-started}

Amazon SES 是一个云电子邮件服务提供商，可以集成到任何应用程序中进行批量电子邮件发送。

Logto 团队调用 Amazon Simple Email Service API，借助这些 API，Logto 终端用户可以通过邮件验证码注册并登录他们的 Logto 账户。
