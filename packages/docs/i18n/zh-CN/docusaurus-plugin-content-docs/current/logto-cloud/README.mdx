# Logto 云服务

[Logto Cloud](https://cloud.logto.io) 提供了一系列管理和运营服务，帮助你顺利且轻松地管理身份和资源。由 Logto 托管，它包括 [多区域支持](/logto-cloud/tenant-settings#tenant-region)、租户管理、[计费和定价](/logto-cloud/billing-and-pricing)、[成员管理](/logto-cloud/tenant-member-management) 和控制台基于角色的访问控制等功能。

如果你对云服务有任何疑问并需要额外支持，请联系我们。
