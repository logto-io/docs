# Logto in deine Anwendung integrieren

Folge diesen Schritten, um Authentifizierung zu deinen Anwendungen mit Logto hinzuzufügen, egal ob es sich um eine benutzerorientierte App oder einen Maschine-zu-Maschine-Dienst handelt:

1. Navigiere zu <CloudLink to="/applications">Konsole > Anwendungen</CloudLink>

2. Klicke auf "Anwendung erstellen", um eine neue Anwendung hinzuzufügen

3. Wähle dein Anwendungs-Framework, um zu beginnen. Wenn du dein Framework nicht finden kannst, klicke auf die Schaltfläche "App ohne Framework erstellen" unten rechts auf der Anwendungs-Erstellungsseite, um eine App durch Auswahl eines [Anwendungstyps](/integrate-logto/application-data-structure/#application-types) zu erstellen oder reiche eine Funktionsanfrage ein oder trage ein SDK bei, indem du unseren [SDK-Konventionen](/developers/sdk-conventions) folgst.

4. Nach der Auswahl deines Frameworks siehst du eine Schnellstartanleitung für das SDK des Frameworks. Folge den Schritten, um deine Anwendung zu konfigurieren und zu integrieren. Wenn du Hilfe beim Verständnis der im Integrationsprozess involvierten Konzepte benötigst, kannst du auf [Verständnis des Logto-Authentifizierungsflusses](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) verweisen, um ein tieferes Verständnis der Integration zu erlangen.

:::note
Die Anleitung in der Konsole ist nur für den Schnellstart mit Logto unter Verwendung unseres SDKs gedacht. Für vollständige Integrationsanleitungen, einschließlich fortgeschrittener SDK-Nutzung, schaue dir den Abschnitt [Schnellstarts](/quick-starts) an.
:::
