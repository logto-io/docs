<TabItem value="pub" label="pub.dev">

Du kannst das `logto_dart_sdk package` direkt mit dem Pub-Paketmanager installieren.
Führe den folgenden Befehl im Stammverzeichnis deines Projekts aus:

```bash
flutter pub add logto_dart_sdk
```

Oder füge das Folgende zu deiner `pubspec.yaml` Datei hinzu:

```yaml
dependencies:
  logto_dart_sdk: ^3.0.0
```

Dann führe aus:

```bash
flutter pub get
```

</TabItem>

<TabItem value="github" label="GitHub">

Wenn du es vorziehst, deine eigene Version des SDK zu forken, kannst du das Repository direkt von GitHub klonen.

```bash
git clone https://github.com/logto-io/dart
```
