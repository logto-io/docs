# Soziale Anmeldung mit Feishu einrichten

Feishu Web: Eine fortschrittliche Plattform für Unternehmenszusammenarbeit und -management, die nahtlose Bürokooperation bietet und Teamziele ausrichtet, um Organisationen und Einzelpersonen vollständig zu aktivieren.

<GuideTip />

## Erste Schritte mit der sozialen Anmeldung bei Feishu \{#getting-started-with-feishu-social-sign-in}

Der Feishu-Web-Connector ist für Desktop-Webanwendungen konzipiert und verwendet den OAuth 2.0-Authentifizierungsprozess.

## Ein Feishu-Entwicklerkonto registrieren \{#register-a-feishu-developer-account}

Wenn du kein Feishu-Entwicklerkonto hast, registriere dich bitte auf der [Feishu Open Platform](https://open.feishu.cn/app).

## Eine Anwendung erstellen \{#create-an-application}

1. Klicke in der [Developer Console](https://open.feishu.cn/app) auf "Create Custom App".
2. Fülle den Anwendungsnamen, die Beschreibung aus, wähle ein Symbol und klicke auf die Schaltfläche "Create".
3. Klicke in der linken Seitenleiste auf "Security Settings" und fülle die "Redirect URL" als `${logto_endpoint}/callback/${connector_id}` aus. Der entsprechende Wert kann im Feld `Callback URI` auf der Feishu-Connector-Detailseite in der Logto-Konsole gefunden werden.
4. In "Credentials & Basic Info" kannst du die "App ID" und das "App Secret" erhalten.

:::note
Für die interne Nutzung ohne Unternehmenszweck musst du auch auf der Seite "Version Management and Release" auf die Schaltfläche "Create a version" klicken.
Die "App ID" und das "App Secret" werden erst wirksam, nachdem sich der Anwendungsstatus auf "Enabled" geändert hat.
:::
