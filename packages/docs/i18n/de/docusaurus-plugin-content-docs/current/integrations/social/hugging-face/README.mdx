# Soziale Anmeldung mit Hugging Face einrichten

Der offizielle Logto-Connector für die soziale Anmeldung mit Hugging Face.

<GuideTip />

## Erste Schritte \{#get-started}

Der Hugging Face-Connector ermöglicht es Endbenutzern, sich mit ihren eigenen Hugging Face-Konten über den Hugging Face OAuth / OpenID Connect-Flow bei deiner Anwendung anzumelden.

<Integration />

## Hugging Face-Connector testen \{#test-hugging-face-connector}

Das war's. Der Hugging Face-Connector sollte jetzt verfügbar sein. Vergiss nicht, [Sozialen Connector in der Anmeldeerfahrung aktivieren](/connectors/social-connectors/#enable-social-sign-in).
