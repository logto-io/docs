# Soziale Anmeldung mit SAML einrichten

Der offizielle Logto-Connector für das SAML-Protokoll.

<GuideTip />

## Erste Schritte \{#get-started}

Der SAML-Connector ermöglicht Logto die Verbindung zu einem beliebigen sozialen Identitätsanbieter, der das SAML-Protokoll unterstützt.

:::note

Der SAML-Connector ist eine besondere Art von Connector in Logto, du kannst einige SAML-basierte Connectoren hinzufügen.

:::
