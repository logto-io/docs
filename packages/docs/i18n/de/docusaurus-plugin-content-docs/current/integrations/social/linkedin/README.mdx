# Soziale Anmeldung mit LinkedIn einrichten

Der offizielle Logto-Connector für die LinkedIn-Sozialanmeldung.

<GuideTip />

## Erste Schritte \{#get-started}

Der LinkedIn-Connector ermöglicht es Endbenutzern, sich mit ihren eigenen LinkedIn-Konten über das LinkedIn OAuth 2.0 Authentifizierungsprotokoll bei deiner Anwendung anzumelden.

<Integration />

## LinkedIn-Connector testen \{#test-linkedin-connector}

Das war's. Der LinkedIn-Connector sollte jetzt verfügbar sein. Vergiss nicht, [Sozialen Connector in der Anmeldeerfahrung aktivieren](/connectors/social-connectors/#enable-social-sign-in).
