### LinkedIn-App einrichten \{#setup-a-linkedin-app}

Gehe zu den [LinkedIn Developers](https://www.linkedin.com/developers) und melde dich mit deinem LinkedIn-Konto an. Wenn du noch kein Konto hast, kannst du dich registrieren.

Erstelle dann eine App.

**Schritt 1:** Fülle die App-Details aus.

Vervollständige das Formular und erstelle die App.

**Schritt 2:** Richte die Callback-URLs ein.

Gehe zur App-Detailseite und finde den "Auth"-Tab, den Abschnitt "OAuth 2.0" und das Feld "Authorized redirect URLs for your app".

In unserem Fall wird dies `${your_logto_endpoint}/callback/${connector_id}` sein, z. B. `https://foo.logto.app/callback/${connector_id}`. Die `connector_id` kann in der oberen Leiste der Logto Admin Console auf der Connector-Detailseite gefunden werden.

**Schritt 3:** Füge das Produkt hinzu.

Gehe zum Tab "Products" und füge das Produkt "Sign In with LinkedIn using OpenID Connect" hinzu.

### Konfiguriere deinen Connector \{#configure-your-connector}

In deiner Logto-Connector-Konfiguration fülle die folgenden Felder mit den Werten aus, die du im Abschnitt "Auth" der App unter "Application credentials" erhalten hast:

- **clientId:** Die Client-ID deiner App.
- **clientSecret:** Das primäre Client-Secret deiner App.

`scope` ist eine durch Leerzeichen getrennte Liste von OIDC-Berechtigungen. Wenn nicht angegeben, ist die Standardberechtigung `openid profile email`.

### Konfigurationstypen \{#config-types}

| Name         | Typ    |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
