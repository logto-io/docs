# Logto installieren

## Interaktiv \{#interactive}

Führe den folgenden Befehl aus, um eine interaktive Installation in deinem Terminal durchzuführen:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## Optionen und stille Installation \{#options-and-silent-installation}

Alternativ kannst du die folgenden Optionen hinzufügen, um einige Fragen zu überspringen:

```bash
# Alle diese Optionen sind optional
--db-url      Die Postgres-URL zur Logto-Datenbank
-p, --path    Pfad zu deiner Logto-Instanz, muss ein nicht vorhandener Pfad sein
--ss          Überspringe das Seeding der Logto-Datenbank
--oc          Füge nach der Installation offizielle Connectors hinzu
```

Führe `logto init --help` für vollständige Hilfeinformationen aus.

Hier ist ein Beispiel für die Durchführung einer stillen Installation (z. B. in CI):

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

Es wird die folgenden Schritte in der Reihenfolge ausführen:

1. Herunterladen und Extrahieren von Logto in `./logto`
2. Versuch, die Datenbank mit der URL `postgresql://your-postgres-dsn:5432/logto` zu initialisieren und zu seedieren
3. Hinzufügen der offiziellen Logto-Connectors zu `./logto/packages/core/connectors`
