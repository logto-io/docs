Logto proporciona servicios de correo electrónico integrados para tu conveniencia en los siguientes escenarios:

1. Explorar o probar rápidamente la experiencia de inicio de sesión por correo electrónico de Logto.
2. Usarlo directamente para tus productos en línea. Está destinado principalmente a nuevas startups que se sientan cómodas usando `logto.email` como su dominio de correo electrónico del remitente.

Las características del servicio de correo electrónico de Logto:

- **Gratis para usar:** Es completamente gratuito sin límites diarios de uso de correo electrónico, ahorrando tus costos.
- **Sin esfuerzo:** No se requiere configuración con ningún proveedor de servicios de correo electrónico de terceros. Simplemente personaliza la información básica de marca para tu plantilla de correo electrónico. Si aún no tienes tu propia información de marca, puedes optar por comenzar a usarlo con unos pocos clics.
- **Entrega asegurada:** Basado en el servicio de correo electrónico de Logto, puedes obtener un servicio estable y una entrega de correo electrónico confiable, asegurando que los usuarios puedan acceder a tu producto.

Sin embargo, aunque es conveniente, hay algunas limitaciones a tener en cuenta:

1. Los correos electrónicos se enviarán desde la dirección fija `no-reply@logto.email`.
2. No puedes agregar enlaces ni ningún otro contenido personalizado a los correos electrónicos.

Dependiendo de tus necesidades comerciales en evolución, puedes optar por usar otros proveedores de servicios de correo electrónico más adelante. Ofrecemos una gama de [conectores de servicios de correo electrónico listos para usar](/connectors/email-connectors#popular-email-providers), y también soportamos disparadores [SMTP](/integrations/smtp), [HTTP](/integrations/http-email) y [WebHook](/developers/webhooks) para enviar correos electrónicos, por lo que siempre encontrarás una manera que se adapte a ti.

:::note
El servicio de correo electrónico gratuito integrado de Logto está actualmente disponible solo para usuarios de [Cloud](https://cloud.logto.io/). Para los usuarios del servicio de código abierto, tienes la flexibilidad de configurar tu proveedor de servicios de correo electrónico para el inicio de sesión por correo electrónico.
:::

## Pasos de configuración \{#configuration-steps}

Sigue estos pasos para configurar el servicio de correo electrónico de Logto:

1. Ve a <CloudLink to="/connectors/passwordless">Conector > Conectores de correo electrónico y SMS</CloudLink>.
2. Para agregar un nuevo conector de correo electrónico, haz clic en el botón "**Configurar**" y selecciona el conector "**Servicio de correo electrónico de Logto**".
3. Una vez que el conector "Servicio de correo electrónico de Logto" se haya creado con éxito, puedes personalizar la información básica de marca que se muestra en las plantillas de correo electrónico.
4. Después de realizar estos cambios, recuerda enviar una plantilla de correo electrónico de prueba a tu dirección de correo electrónico antes de guardar los cambios.

Opciones de personalización:

- **Correo electrónico del remitente:** El correo electrónico del remitente está configurado como `no-reply@logto.email` y no se puede modificar.
- **Nombre del remitente:** Establece el nombre de tu marca como el nombre del remitente para asegurar el reconocimiento del usuario.
- **Información de la empresa:** Muestra el nombre de tu empresa, dirección o código postal para mejorar la confianza del usuario y cumplir con los requisitos de cumplimiento. _Ten en cuenta que no se permiten URLs._
- **Logo de la aplicación:** Sube el logo de la marca de tu aplicación para que el valor de la marca de la aplicación pueda mostrarse en los correos electrónicos recibidos por los usuarios.

## Plantillas de correo electrónico unificadas \{#unified-email-templates}

El servicio de correo electrónico de Logto utiliza plantillas de correo electrónico unificadas adaptadas para escenarios de autenticación específicos:

| Uso                                | Escenario                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Registro                           | Los usuarios crean una cuenta usando su correo electrónico y la verifican ingresando un código de verificación enviado por Logto a su correo electrónico.                                                                                                                                                                                                                                                                                                             |
| Iniciar sesión                     | Los usuarios inician sesión usando su correo electrónico y verifican ingresando un código de verificación en lugar de ingresar una contraseña.                                                                                                                                                                                                                                                                                                                        |
| Olvidó la contraseña               | Si los usuarios olvidan su contraseña durante el inicio de sesión, pueden optar por verificar su identidad usando el correo electrónico que ya han verificado con Logto.                                                                                                                                                                                                                                                                                              |
| Genérico                           | Esta plantilla se puede usar como una opción de respaldo general para varios escenarios, incluidas las pruebas de configuraciones de conectores, etc.                                                                                                                                                                                                                                                                                                                 |
| Invitación a la organización       | Usa esta plantilla para enviar a los usuarios un enlace de invitación para unirse a la organización.                                                                                                                                                                                                                                                                                                                                                                  |
| Validación de permisos del usuario | Durante el uso de la aplicación, puede haber algunas operaciones de alto riesgo o operaciones con un nivel de riesgo relativamente alto que requieran verificación adicional del usuario, como transferencias bancarias, eliminación de recursos en uso y cancelación de membresías. La plantilla `UserPermissionValidation` se puede usar para definir el contenido del código de verificación por correo electrónico que los usuarios reciben en estas situaciones. |
| Vincular nuevo identificador       | Cuando un usuario modifica su perfil, puede vincular una dirección de correo electrónico a su cuenta actual. En este caso, la plantilla `BindNewIdentifier` se puede usar para personalizar el contenido del correo electrónico de verificación.                                                                                                                                                                                                                      |

## Preguntas frecuentes \{#faqs}

<details>
<summary>

### ¿Es posible agregar CSS o HTML personalizados al servicio de correo electrónico de Logto? \{#is-it-possible-to-add-custom-css-or-html-to-logto-email-service}

</summary>

El servicio de correo electrónico integrado de Logto no admite CSS o HTML personalizados. Solo puedes modificar elementos genéricos de marca. Esta restricción está en su lugar para mantener la estabilidad del servicio de correo electrónico integrado, ya que todos los inquilinos comparten la misma dirección IP y dirección del remitente. Para más detalles, consulta "[Factores para mejorar la entrega de correos electrónicos](https://blog.logto.io/verification-email-delivery#factors-to-improve-email-delivery)".

Para personalizar las plantillas de correo electrónico, recomendamos usar otro conector de correo electrónico, como AWS Direct Mail, SendGrid, Mailgun, Postmark o SMTP.

</details>
