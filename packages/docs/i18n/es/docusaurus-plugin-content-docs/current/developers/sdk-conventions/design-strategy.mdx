# Estrategia de diseño

- Cada lenguaje de programación debe tener un repositorio git aislado llamado `${language}`.
- Cada repositorio de lenguaje de programación debe ser un mono repo.
- Tanto los SDKs como sus proyectos de ejemplo asociados deben colocarse bajo este repositorio.

Ejemplos:

<details>
  <summary>

### JS \{#js}

</summary>

- js (core)
- react
- react-sample

</details>

<details>
  <summary>

### Kotlin \{#kotlin}

</summary>

- kotlin (core)
- android
- android-sample

</details>
