# Integra la autenticación de Logto

Logto proporciona soluciones de autenticación completas para aplicaciones web, móviles y de escritorio, admite la autenticación [máquina a máquina (M2M)](/quick-starts/m2m) entre servicios, y puede servir como un Proveedor de Identidad (IdP) para [aplicaciones de terceros](/integrate-logto/third-party-applications) a través de protocolos estándar como [OpenID Connect(OIDC)](https://auth.wiki/openid-connect) y [OAuth 2.0](https://auth.wiki/oauth-2.0).

Comienza tu integración seleccionando la solución que mejor se adapte a tus necesidades:

## Añade autenticación a tu aplicación \{#add-authentication-for-your-application}

Ya sea que estés construyendo aplicaciones orientadas al usuario (como aplicaciones web, móviles o de escritorio) o aplicaciones máquina a máquina (M2M) para la comunicación de servicio a servicio, puedes implementar rápidamente características completas de [autenticación](/end-user-flows) y [gestión de usuarios](/user-management) integrando Logto.

## Usa Logto como un Proveedor de Identidad \{#use-logto-as-an-identity-provider}

Cuando deseas habilitar que aplicaciones de terceros se autentiquen con tu sistema de identidad Logto, puedes configurar Logto como un [Proveedor de Identidad (IdP)](https://auth.wiki/identity-provider).

Logto puede ser utilizado como un IdP a través de protocolos estándar como OAuth, OIDC o SAML (próximamente), y los servicios externos pueden implementar "Iniciar sesión con Logto" al igual que "Iniciar sesión con Google", permitiendo a sus usuarios autenticarse utilizando tu sistema de usuarios Logto.

Al crear aplicaciones de terceros en Logto, estableces conexiones seguras entre tu sistema de identidad y servicios externos.

## Extiende las características de Logto con Management API \{#extend-logto-features-with-management-api}

Logto [Management API](https://openapi.logto.io/) te permite controlar y automatizar programáticamente la funcionalidad de Logto en tus aplicaciones.

Cuando necesites extender las capacidades de Logto para mejorar tu producto (como [gestionar organizaciones multi-tenant](/organizations), o automatizar flujos de trabajo con Logto), puedes aprovechar Logto Management API para construir características avanzadas que se ajusten a las necesidades de tu producto.
