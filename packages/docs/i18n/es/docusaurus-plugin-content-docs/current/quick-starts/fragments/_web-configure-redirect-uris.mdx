#### Configurar URIs de redirección \{#configure-redirect-uris}

Cambia a la página de detalles de la aplicación en Logto Console. Añade una URI de redirección <code>{props.redirectUri ?? 'http://localhost:3000/callback'}</code>.

<img
  alt="URI de redirección en Logto Console"
  src="/img/assets/web-redirect-uri.png"
  width="600px"
/>

Al igual que al iniciar sesión, los usuarios deben ser redirigidos a Logto para cerrar la sesión de la sesión compartida. Una vez terminado, sería ideal redirigir al usuario de vuelta a tu sitio web. Por ejemplo, añade <code>{defaultBaseUrl}</code> como la sección de URI de redirección posterior al cierre de sesión.

Luego haz clic en "Guardar" para guardar los cambios.
