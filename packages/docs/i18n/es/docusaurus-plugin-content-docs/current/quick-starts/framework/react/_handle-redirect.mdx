Dado que usamos `http://localhost:3000/callback` como el URI de redirección, ahora necesitamos manejarlo adecuadamente.

Primero, vamos a crear una página de callback:

```tsx title="pages/Callback/index.tsx"
import { useHandleSignInCallback } from '@logto/react';

const Callback = () => {
  const { isLoading } = useHandleSignInCallback(() => {
    // Haz algo cuando termine, por ejemplo, redirigir a la página de inicio
  });

  // Cuando está en progreso
  if (isLoading) {
    return <div>Redirigiendo...</div>;
  }

  return null;
};
```

Finalmente, inserta el siguiente código para crear una ruta `/callback` que NO requiere autenticación:

```tsx title="App.tsx"
// Asumiendo react-router
<Route path="/callback" element={<Callback />} />
```
