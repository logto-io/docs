### Mostrar información del usuario \{#display-user-information}

Para mostrar la información del usuario, puedes usar el método `getIdTokenClaims()`. Por ejemplo, en una aplicación Capacitor:

```tsx
const userClaims = await logtoClient.getIdTokenClaims();
console.log(userClaims);
```

Para solicitar alcances adicionales, puedes pasar los alcances al objeto `LogtoConfig` al inicializar el cliente. Por ejemplo:

```tsx
const logtoConfig = {
  scopes: ['email', 'phone', 'custom_data', 'organizations'],
};
```
