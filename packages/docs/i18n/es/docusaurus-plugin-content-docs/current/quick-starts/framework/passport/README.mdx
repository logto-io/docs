# Añade autenticación a tu aplicación Passport.js

Esta guía te mostrará cómo integrar Logto en tu aplicación con [Passport.js](https://www.passportjs.org/) y la estrategia OIDC.

<GuideTip />

## Requisitos previos \{#prerequisites}

- Una cuenta de [Logto Cloud](https://cloud.logto.io) o un [Logto autoalojado](/introduction/set-up-logto-oss).
- Una aplicación tradicional de Logto creada.
- Un proyecto de express con sesión configurada. Consulta el [sitio web de Express.js](https://expressjs.com/).
