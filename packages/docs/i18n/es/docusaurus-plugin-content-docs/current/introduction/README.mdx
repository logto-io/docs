# Introducción a Logto

¡Bienvenido a la documentación de Logto! Logto es una solución de gestión de identidad y acceso (IAM) diseñada para aplicaciones modernas y productos SaaS. Proporciona un sistema de autenticación y autorización seguro, escalable y personalizable para tus aplicaciones.
