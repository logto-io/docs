# Gestión de miembros del inquilino

## Invitar colaboradores \{#invite-collaborators}

Ve a <CloudLink to="/tenant-settings/members">Consola > Configuración del inquilino > Miembros</CloudLink>. Verás a tus miembros actuales y podrás invitar a más. Ingresa direcciones de correo electrónico y asigna roles (puedes hacerlo en bloque) y luego envía.

Logto te permite invitar a miembros del equipo a tu inquilino con dos roles diferentes en Logto Cloud:

- **Admin:** Acceso completo y gestión de todos los recursos del inquilino.
- **Colaborador:** Puede gestionar recursos relacionados con la identidad, pero con limitaciones:
  - Sin acceso a la información de facturación
  - No puede invitar ni gestionar a otros miembros
  - No puede cambiar el nombre del inquilino
  - No puede eliminar el inquilino

### Límite de miembros \{#members-limit}

| Tipo de inquilino                | Límite de miembros                                      |
| -------------------------------- | ------------------------------------------------------- |
| Inquilino de desarrollo          | Hasta 3 miembros (contáctanos para más)                 |
| Inquilino de producción gratuito | 1 miembro (solo admin)                                  |
| Inquilino de producción Pro      | Comienza con 3 miembros, luego $8 por miembro adicional |

### Gestionar invitación \{#manage-invitation}

La pestaña de invitación muestra invitaciones pasadas con su estado

| Estado    | Explicación                                   | Acción disponible |
| --------- | --------------------------------------------- | ----------------- |
| Pendiente | Invitación enviada, pero aún no aceptada.     | Reenviar          |
| Aceptada  | El usuario es ahora un miembro del inquilino. | Eliminar          |
| Expirada  | La invitación superó el límite de 7 días.     | Eliminar          |
| Revocada  | El admin canceló la invitación.               | Eliminar          |

No puedes invitar a personas que ya tienen invitaciones pendientes o que son miembros.

### Gestionar miembros del inquilino (solo Admin) \{#manage-tenant-members-admin-only}

Como admin, tienes varias formas de gestionar tu equipo:

- **Invitar o eliminar miembros:** Agrega nuevos miembros del equipo o elimina el acceso según sea necesario.
- **Cambiar roles de miembros:** Asigna roles de 'admin' o 'colaborador' dependiendo del nivel de acceso requerido.
- **Transferir la propiedad del inquilino:** Si es necesario, puedes transferir la propiedad del inquilino a otro miembro. Puedes primero dar el rol de admin a otro miembro y luego elegir quedarte como miembro regular o dejar el inquilino por completo.
- **Agregar múltiples admins:** Comparte responsabilidades administrativas con otros miembros del equipo para una gestión colaborativa.
