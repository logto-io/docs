Usa el campo `Scope` para añadir alcances adicionales a tu solicitud de OAuth. Esto te permitirá solicitar más información del servidor OAuth de Okta. Por favor, consulta la [documentación de Okta](https://developer.okta.com/docs/reference/api/oidc/#scopes) para más detalles sobre los alcances disponibles.

Independientemente de la configuración de alcance personalizada, Logto siempre enviará los alcances `openid`, `profile` y `email` al IdP. Esto es para asegurar que Logto pueda recuperar correctamente la información de identidad y la dirección de correo electrónico del usuario.
