# Configura el inicio de sesión social con Amazon

El conector oficial de Logto para el inicio de sesión social de Amazon.

<GuideTip />

## Comenzar \{#get-started}

El conector de Amazon permite a los usuarios finales iniciar sesión en tu aplicación utilizando sus propias cuentas de Amazon a través del protocolo de autenticación Amazon OAuth 2.0.

<Integration />

## Probar el conector de Amazon \{#test-amazon-connector}

Eso es todo. El conector de Amazon debería estar disponible ahora. No olvides [habilitar el conector social en la experiencia de inicio de sesión](/connectors/social-connectors/#enable-social-sign-in).
