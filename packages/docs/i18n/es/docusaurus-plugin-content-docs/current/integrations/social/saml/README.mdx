# Configura el inicio de sesión social con SAML

El conector oficial de Logto para el protocolo SAML.

<GuideTip />

## Comenzar \{#get-started}

El conector SAML permite la conexión de Logto a un proveedor de identidad social arbitrario que admite el protocolo SAML.

:::note

El conector SAML es un tipo especial de conector en Logto, puedes agregar algunos conectores basados en SAML.

:::
