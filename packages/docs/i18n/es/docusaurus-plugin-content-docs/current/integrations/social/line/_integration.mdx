### Configura un canal de Line \{#setup-a-line-channel}

Ve a [Line Developers](https://developers.line.biz/console/) e inicia sesión con tu cuenta comercial de Line. Si no tienes una cuenta, puedes registrarte para obtener una.

Luego, ve a [este enlace](https://developers.line.biz/console/register/line-login/provider/) para crear un canal.

**Paso 1:** Completa los Detalles del Canal.

Completa el formulario y crea el canal.

**Paso 2:** Configura las URLs de callback.

Ve a la página de detalles del canal, encuentra la pestaña "LINE login" y edita el campo "Callback URL".

En nuestro caso, será `${your_logto_endpoint}/callback/${connector_id}`. por ejemplo, `https://foo.logto.app/callback/${connector_id}`. El `connector_id` se puede encontrar en la barra superior de la página de detalles del conector en Logto Admin Console.

### Configura tu conector \{#configure-your-connector}

En la configuración de tu conector de Logto, completa los siguientes campos con los valores obtenidos de la sección "Credenciales de la aplicación" en la pestaña "Auth" de tu App:

- **clientId:** El ID de Canal de tu App.
- **clientSecret:** El Secreto de Canal de tu App.

`scope` es una lista de alcances OIDC delimitada por espacios. Si no se proporciona, el alcance predeterminado es `openid profile`.

### Tipos de configuración \{#config-types}

| Nombre       | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
