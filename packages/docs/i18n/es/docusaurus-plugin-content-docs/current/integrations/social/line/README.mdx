# Configura el inicio de sesión social con Line

El conector oficial de Logto para el inicio de sesión social de Line.

<GuideTip />

## Comenzar \{#get-started}

El conector de Line permite a los usuarios finales iniciar sesión en tu aplicación utilizando sus propias cuentas de Line a través del protocolo de autenticación Line OAuth 2.0.

<Integration />

## Probar el conector de Line \{#test-line-connector}

Eso es todo. El conector de Line debería estar disponible ahora. No olvides [habilitar el conector social en la experiencia de inicio de sesión](/connectors/social-connectors/#enable-social-sign-in).
