# Configura el inicio de sesión social con Feishu

Feishu Web: Una plataforma avanzada de colaboración y gestión empresarial, que ofrece colaboración de oficina sin problemas y alinea los objetivos del equipo para activar completamente a las organizaciones e individuos.

<GuideTip />

## Comenzando con el inicio de sesión social de Feishu \{#getting-started-with-feishu-social-sign-in}

El conector web de Feishu está diseñado para aplicaciones web de escritorio y utiliza el proceso de autenticación OAuth 2.0.

## Registra una cuenta de desarrollador de Feishu \{#register-a-feishu-developer-account}

Si no tienes una cuenta de desarrollador de Feishu, por favor regístrate en la [Plataforma Abierta de Feishu](https://open.feishu.cn/app).

## Crea una aplicación \{#create-an-application}

1. En la [Consola de Desarrollador](https://open.feishu.cn/app), haz clic en "Crear aplicación personalizada".
2. Rellena el nombre de la aplicación, la descripción, selecciona un ícono y haz clic en el botón "Crear".
3. En la barra lateral izquierda, haz clic en "Configuración de seguridad", completa el "Redirect URL" como `${logto_endpoint}/callback/${connector_id}`. El valor correspondiente se puede encontrar en el campo `Callback URI` en la página de detalles del conector Feishu en la Consola de Logto.
4. En "Credenciales e Información Básica", puedes obtener el "App ID" y el "App Secret".

:::note
Para uso interno no empresarial, también necesitas hacer clic en el botón "Crear una versión" en la página "Gestión de Versiones y Lanzamiento".
El "App ID" y el "App Secret" solo tendrán efecto después de que el estado de la aplicación cambie a "Habilitado".
:::
