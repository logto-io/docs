# Gestion des utilisateurs

La gestion des utilisateurs est un domaine d'intérêt majeur pour Logto. Vous pouvez accéder à vos données utilisateur et les gérer en utilisant soit la Console Logto, soit l'API de gestion, qui sont toutes deux efficaces. Avec ces outils, vous pouvez effectuer des tâches telles que

- Rechercher des utilisateurs par nom d'utilisateur, ID, nom complet, numéro de téléphone ou email.
- Effectuer plusieurs actions sur l'utilisateur, telles que dissocier des comptes sociaux, mettre à jour les informations de profil, supprimer des utilisateurs ou réinitialiser des mots de passe.
- Vérifier les journaux utilisateur.
- Attribuer des rôles pour le contrôle d'accès.
- Créer et gérer des jetons d’accès personnels.
