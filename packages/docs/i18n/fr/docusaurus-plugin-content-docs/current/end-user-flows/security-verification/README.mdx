# Vérification de sécurité

Lorsque des utilisateurs authentifiés tentent d'effectuer des actions sensibles ou à haut risque, telles que changer leur mot de passe, effectuer un paiement ou accéder à des informations sensibles comme des fiches de paie ou des détails de compte bancaire, des mesures de sécurité supplémentaires sont essentielles. Ce flux est particulièrement critique dans les applications sensibles telles que les services bancaires, de santé et gouvernementaux.

Ce processus, connu sous le nom de _vérification de sécurité_, exige que les utilisateurs revérifient leur identité pour confirmer qu'ils sont le titulaire de compte autorisé. En mettant en œuvre la vérification de sécurité, le système renforce la protection contre l'accès non autorisé, ajoutant une couche de sécurité cruciale pour les actions à haut risque et aidant à protéger les données sensibles.
