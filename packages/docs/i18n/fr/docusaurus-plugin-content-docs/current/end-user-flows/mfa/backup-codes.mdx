# Codes de secours

## Concepts \{#concepts}

Les codes de secours, également appelés code de récupération, sont des codes à usage unique pour l'authentification multi-facteurs (MFA), agissant comme une sauvegarde au cas où les facteurs d'authentification principaux de l'utilisateur (par exemple, application d'authentification ou jeton matériel) ne seraient pas disponibles.

Les perdre peut entraîner des défis de récupération de compte. Par conséquent, il est recommandé de configurer un facteur principal supplémentaire avant d'activer les codes de secours, en leur donnant la priorité.

Logto génère automatiquement 10 codes de secours pour les utilisateurs une fois qu'ils configurent un facteur supplémentaire. Chaque code est à usage unique. Il est conseillé aux utilisateurs de régénérer un nouvel ensemble de codes dans les paramètres de compte utilisateur (accessible via le [Management API](/integrate-logto/interact-with-management-api/)) avant d'utiliser tous les codes existants.
