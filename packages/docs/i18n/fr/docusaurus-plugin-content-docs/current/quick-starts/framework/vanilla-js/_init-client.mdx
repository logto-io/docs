Importez et initialisez une instance `LogtoClient` en passant la configuration :

```ts title="index.js"
import LogtoClient from '@logto/browser';

const logtoClient = new LogtoClient({
  endpoint: '<your-logto-endpoint>',
  appId: '<your-application-id>',
});
```

Les `endpoint` et `appId` peuvent être trouvés dans la page des détails de l'application dans la Logto Console.
