Ce guide vous montrera comment intégrer Logto dans votre application Vue 3.

- Un compte [Logto Cloud](https://cloud.logto.io) ou un [Logto auto-hébergé](/introduction/set-up-logto-oss).
- Une application monopage (SPA) créée dans Logto Console.
- Un projet Vue 3.

### Attacher le jeton d’accès aux en-têtes de requête \{#attach-access-token-to-request-headers}

Placez le jeton dans le champ `Authorization` des en-têtes HTTP avec le format Bearer (`Bearer YOUR_TOKEN`), et vous êtes prêt à partir.

:::note
Le flux d'intégration du jeton Bearer peut varier en fonction du framework ou du demandeur que vous utilisez. Choisissez votre propre méthode pour appliquer l'en-tête de requête `Authorization`.
:::
