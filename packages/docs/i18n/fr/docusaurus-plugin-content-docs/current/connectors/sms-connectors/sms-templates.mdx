Logto propose quatre modèles différents pour personnaliser le contenu des SMS, qui sont classés en fonction de leur type d'utilisation : Register, SignIn, ForgotPassword et Generic. Il est fortement recommandé d'utiliser différents modèles pour divers cas d'utilisation, sinon cela pourrait atteindre la limite de taux, entraînant une interruption temporaire de votre service.

## Types de modèles SMS et exemples \{#sms-template-types-and-examples}

Voici quelques exemples à titre de référence :

| usageType                | Scénario                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | Exemples de modèles                                                                                  |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| SignIn                   | Les utilisateurs [se connectent en utilisant leur numéro de téléphone](/end-user-flows/sign-up-and-sign-in/sign-in) et vérifient en entrant le code de vérification SMS au lieu de saisir un mot de passe.                                                                                                                                                                                                                                                                                                                                                                                                                     | Code de vérification de connexion Logto : `{{code}}`. Expire dans 10 minutes.                        |
| Register                 | Les utilisateurs [créent un compte en utilisant leur numéro de téléphone](/end-user-flows/sign-up-and-sign-in/sign-up) et le vérifient en entrant un code de vérification envoyé par Logto à leur numéro de téléphone.                                                                                                                                                                                                                                                                                                                                                                                                         | Code de vérification d'inscription Logto : `{{code}}`. Expire dans 10 minutes.                       |
| ForgotPassword           | Si les utilisateurs oublient leur mot de passe lors de la connexion, ils peuvent choisir de vérifier leur identité en utilisant d'abord le numéro de téléphone pour [réinitialiser le mot de passe](/end-user-flows/sign-up-and-sign-in/reset-password).                                                                                                                                                                                                                                                                                                                                                                       | Code de vérification de réinitialisation du mot de passe Logto : `{{code}}`. Expire dans 10 minutes. |
| Generic                  | Ce modèle peut être utilisé comme option de secours générale pour divers scénarios, y compris la configuration de connecteurs de test, [la vérification ou la liaison du numéro de téléphone après la connexion](/end-user-flows/account-settings/by-management-api#email-and-phone-number-verification), etc.                                                                                                                                                                                                                                                                                                                 | Code de vérification Logto : `{{code}}`. Expire dans 10 minutes.                                     |
| OrganizationInvitation   | Utilisez ce modèle pour [envoyer aux utilisateurs un lien d'invitation](/end-user-flows/organization-experience/invite-organization-members#configure-your-email-connector) pour rejoindre l'organisation.                                                                                                                                                                                                                                                                                                                                                                                                                     | Code de vérification d'invitation à l'organisation Logto : `{{code}}`. Expire dans 10 minutes.       |
| UserPermissionValidation | Lors de l'utilisation de l'application, il peut y avoir certaines opérations à haut risque ou des opérations avec un niveau de risque relativement élevé qui [nécessitent une vérification utilisateur supplémentaire](/end-user-flows/account-settings/by-account-api#verify-by-sending-a-verification-code-to-the-users-email-or-phone), telles que les virements bancaires, la suppression de ressources en cours d'utilisation et l'annulation d'adhésions. Le modèle `UserPermissionValidation` peut être utilisé pour définir le contenu du code de vérification SMS que les utilisateurs reçoivent dans ces situations. | Code de vérification Logto : `{{code}}`. Expire dans 10 minutes.                                     |
| BindNewIdentifier        | Lorsqu'un utilisateur modifie son profil, il peut [lier un numéro de téléphone à son compte actuel](/end-user-flows/account-settings/by-account-api#manage-phone). Dans ce cas, le modèle `BindNewIdentifier` peut être utilisé pour personnaliser le contenu du SMS de vérification.                                                                                                                                                                                                                                                                                                                                          | Code de vérification de liaison de compte Logto : `{{code}}`. Expire dans 10 minutes.                |

Il est important de comprendre ces paramètres :

- Le code de vérification est valable pendant 10 minutes. Nous ne prenons actuellement pas en charge la personnalisation du temps d'expiration.
- Logto remplacera l'espace réservé `{{code}}` dans le modèle SMS par un code de vérification. Par conséquent, veuillez vous assurer que le modèle a un espace réservé réservé.

:::note
Certains pays et régions peuvent ne pas autoriser l'envoi de contenu non approuvé par SMS en raison des exigences de conformité. Les modèles SMS doivent être enregistrés et approuvés par le fournisseur de SMS avant de pouvoir être utilisés. Dans de tels cas, le contenu peut être indexé par ID de modèle vers le modèle correspondant.
:::
