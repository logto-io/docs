# Hugging Face 소셜 로그인을 설정하세요

Hugging Face 소셜 로그인에 대한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Hugging Face 커넥터를 통해 최종 사용자는 Hugging Face OAuth / OpenID Connect 흐름을 통해 자신의 Hugging Face 계정을 사용하여 애플리케이션에 로그인할 수 있습니다.

<Integration />

## Hugging Face 커넥터 테스트하기 \{#test-hugging-face-connector}

이제 Hugging Face 커넥터가 사용 가능해야 합니다. [로그인 경험에서 소셜 커넥터 활성화](/connectors/social-connectors/#enable-social-sign-in)를 잊지 마세요.
