# Line 소셜 로그인을 설정하세요

Line 소셜 로그인에 대한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Line 커넥터를 사용하면 최종 사용자가 Line OAuth 2.0 인증 프로토콜을 통해 자신의 Line 계정을 사용하여 애플리케이션에 로그인할 수 있습니다.

<Integration />

## Line 커넥터 테스트 \{#test-line-connector}

이제 완료되었습니다. Line 커넥터가 이제 사용 가능해야 합니다. [로그인 경험에서 소셜 커넥터 활성화](/connectors/social-connectors/#enable-social-sign-in)를 잊지 마세요.
