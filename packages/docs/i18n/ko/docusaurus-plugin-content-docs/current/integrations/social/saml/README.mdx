# SAML로 소셜 로그인을 설정하세요

SAML 프로토콜을 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

SAML 커넥터는 SAML 프로토콜을 지원하는 임의의 소셜 아이덴티티 제공자 (IdP)와 Logto의 연결을 가능하게 합니다.

:::note

SAML 커넥터는 Logto에서 특별한 종류의 커넥터로, 몇 가지 SAML 기반 커넥터를 추가할 수 있습니다.

:::
