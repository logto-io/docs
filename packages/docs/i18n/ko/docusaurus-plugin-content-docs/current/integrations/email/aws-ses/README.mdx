# AWS Direct Mail로 이메일 인증을 설정하세요

AWS의 직접 메일 서비스를 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Amazon SES는 대량 이메일 발송을 위해 어떤 애플리케이션에도 통합할 수 있는 클라우드 이메일 서비스 제공업체입니다.

Logto 팀은 Amazon Simple Email Service API를 호출하여 Logto 최종 사용자가 메일 인증 코드를 통해 Logto 계정에 등록하고 로그인할 수 있도록 지원합니다.
