## AWS 서비스 콘솔에서 메일 서비스 구성하기 \{#configure-a-mail-service-in-the-aws-service-console}

### AWS 계정 등록하기 \{#register-aws-account}

[AWS](https://aws.amazon.com/)로 이동하여 계정을 등록하세요.

### 아이덴티티 생성하기 \{#create-a-identity}

- `Amazon Simple Email Service` 콘솔로 이동합니다.
- 아이덴티티를 생성하고 다음 옵션 중 하나를 선택합니다.
  - 도메인 생성
  - 이메일 주소 생성

### 커넥터 구성하기 \{#configuration-of-the-connector}

1. Amazon 콘솔의 오른쪽 상단에서 사용자 이름을 클릭하여 `Security Credentials`로 들어갑니다. 만약 없다면, `AccessKey`를 생성하고 주의 깊게 저장하세요.
2. `Amazon Simple Email Service` 커넥터의 설정을 완료합니다:
   - 1단계에서 얻은 `AccessKey ID`와 `AccessKey Secret`을 사용하여 각각 `accessKeyId`와 `accessKeySecret`을 입력합니다.
   - `region`: 메일을 보내는 데 사용하는 아이덴티티의 지역을 `region` 필드에 입력합니다.
   - `emailAddress`: 메일을 보내는 데 사용하는 이메일 주소로, `Logto\<noreply@logto.io>` 또는 `\<noreply@logto.io>` 형식으로 입력합니다.

다음 매개변수는 선택 사항입니다. 매개변수 설명은 [AWS SES API 문서](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html)에서 찾을 수 있습니다.

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### Amazon SES 커넥터 테스트하기 \{#test-the-amazon-ses-connector}

이메일 주소를 입력하고 "Send"를 클릭하여 설정이 작동하는지 "Save and Done" 전에 확인할 수 있습니다.

이제 완료되었습니다. [로그인 경험에서 커넥터 활성화하기](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)를 잊지 마세요.

### 유형 구성하기 \{#configure-types}

| 이름                                      | 유형              |
| ----------------------------------------- | ----------------- |
| accessKeyId                               | string            |
| accessKeySecret                           | string            |
| region                                    | string            |
| emailAddress                              | string (OPTIONAL) |
| emailAddressIdentityArn                   | string (OPTIONAL) |
| feedbackForwardingEmailAddress            | string (OPTIONAL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPTIONAL) |
| configurationSetName                      | string (OPTIONAL) |
| templates                                 | Template[]        |

| 템플릿 속성 | 유형        | 열거형 값                                               |
| ----------- | ----------- | ------------------------------------------------------- |
| subject     | string      | N/A                                                     |
| content     | string      | N/A                                                     |
| usageType   | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
