1. [Microsoft Entra 관리자 센터](https://entra.microsoft.com/)로 이동하여 관리자 계정으로 로그인하세요.

2. Identity > Applications > App registrations로 이동합니다.

<center>
  <img src="/img/assets/entra_id_oidc_create_application.webp" alt="애플리케이션 생성" />
</center>

3. `New registration`을 선택합니다.

4. 애플리케이션 이름을 입력하고 애플리케이션에 적합한 계정 유형을 선택합니다.

5. 애플리케이션 플랫폼으로 `Web`을 선택합니다.

6. Logto의 SSO 설정 페이지에서 `redirect URI`를 복사하여 붙여넣습니다. `redirect URI`는 사용자가 Microsoft Entra ID로 인증한 후 리디렉션되는 URL입니다.

<center>
  <img src="/img/assets/entra_id_oidc_config_application.webp" alt="애플리케이션 구성" />
</center>

6. 애플리케이션을 생성하려면 `Register`를 클릭하세요.
