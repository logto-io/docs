# 기여

Logto에 기여해 주셔서 감사합니다! 우리의 프로젝트를 더 견고하고, 사용자 친화적이며, 다재다능하게 만드는 데 커뮤니티의 기여를 환영합니다. 버그 수정, 문서 개선, 커넥터 생성, 새로운 기능 추가, 피드백 제공 등 어떤 형태로든 여러분의 노력이 매우 소중합니다.

## 어떻게 기여할 수 있나요? \{#how-to-contribute}

우리는 기여를 가능한 한 원활하게 만들기 위해 노력하고 있습니다. 시작하기 전에, [GitHub의 기여 가이드](https://github.com/logto-io/logto/blob/master/.github/CONTRIBUTING.md)를 검토해 주세요. 이 가이드는 우리의 프로세스, 코딩 표준, Logto에 성공적으로 기여하는 방법을 설명합니다.

## 도움이 필요하신가요? \{#need-help}

질문이 있거나 안내가 필요하시면, [GitHub 토론](https://github.com/logto-io/logto/discussions), [이슈 열기](https://github.com/logto-io/logto/issues), 또는 [문의하기](https://logto.io/contact)를 통해 언제든지 연락해 주세요. 성공에 필요한 모든 것을 제공해 드리겠습니다.
