# 데이터베이스 설정 관리

Logto는 OIDC 및 쿠키 키와 같은 일부 기술 설정을 유지 관리하는 데 도움을 줍니다. Admin Console 또는 Management API를 통해 이를 관리하기 전에 CLI를 사용하여 목표를 달성할 수 있습니다.

## 키로 설정 가져오기 \{#get-config-by-key}

사용법:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get <key> [keys...]
```

  </TabItem>

</Tabs>

OIDC 개인 키를 가져오고 싶다면:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get oidc.privateKeys
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get oidc.privateKeys
```

  </TabItem>

</Tabs>

출력은 다음과 같습니다:

```bash
oidc.privateKeys=["\n-----BEGIN PRIVATE KEY-----\nMIIJRAIBA..."]
```

사용 가능한 모든 키에 대해서는 `logto db config get --help`를 실행하거나, 자세한 설명은 [Configuration](/concepts/core-service/configuration/#database-configs)을 참조하세요.

## 키로 설정하기 \{#set-config-by-key}

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set <key> [keys...]
```

  </TabItem>

</Tabs>

OIDC 쿠키 키를 설정하고 싶다면:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>

</Tabs>

일반적으로 설정할 값은 유효한 JSON 문자열이어야 하며, 사전 정의된 타입 정의를 따라야 합니다. 사용 가능한 모든 키에 대해서는 `logto db config set --help`를 실행하거나, 자세한 설명은 [Configuration](/concepts/core-service/configuration/)을 참조하세요.
