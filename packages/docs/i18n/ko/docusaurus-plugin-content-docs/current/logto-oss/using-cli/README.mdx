# Logto CLI

Logto CLI는 지루한 기술 작업을 수동으로 처리하지 않고 Logto 인스턴스를 설치, 유지 관리 및 업데이트할 수 있는 편리한 방법을 제공합니다.

:::note
현재 바이너리를 제공하지 않으므로, 환경에 Node.js `^18.12.0`이 필요합니다.
:::

### CLI 전역 설치 \{#install-cli-globally}

선호하는 패키지 관리자를 통해 `logto` 명령을 전역으로 사용하세요:

<Tabs>

  <TabItem value="npm" label="npm">

```bash
npm i @logto/cli -g
```

  </TabItem>
  <TabItem value="yarn" label="Yarn">

```bash
yarn global add @logto/cli
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm add -g @logto/cli
```

  </TabItem>

</Tabs>

각 명령의 도움말 정보를 표시하려면 항상 `--help`를 추가할 수 있습니다:

```bash
logto --help
logto db seed --help
```

### 로컬 CLI \{#local-cli}

CLI의 내장 버전은 v1.0.0-beta.11부터 모든 Logto 릴리스에 포함되어 있습니다. 이미 `~/logto`에 Logto 인스턴스가 있다고 가정하면, 아래 명령을 실행할 수 있습니다:

```bash
cd ~/logto
npm run cli # 로컬 CLI 호출
```

옵션과 함께 실행하려면 Logto CLI 옵션 앞에 `--`를 추가해야 합니다. 예를 들어:

```bash
npm run cli db seed -- --db-url postgresql://your-database-url
```

### `npx`로 CLI 사용 \{#use-cli-by-npx}

Logto CLI를 사용하는 또 다른 방법은 설치 없이 패키지를 실행할 수 있는 `npx`입니다.

```bash
npx @logto/cli
```

이는 일회성 호출에 유용합니다. 예를 들어:

```bash
npx @logto/cli db seed --db-url postgresql://your-database-url
```
