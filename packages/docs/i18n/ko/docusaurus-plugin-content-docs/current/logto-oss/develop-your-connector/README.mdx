# 커넥터 개발하기

현재 Logto는 WeChat, Alipay, Google, Facebook 등과 같은 널리 사용되는 소셜 로그인 커넥터와 AliCloud, SendGrid, Twilio와 같은 SMS / 이메일 커넥터를 제공하고 있습니다.

다음은 커넥터가 Logto 핵심 서비스와 어떻게 작동하는지를 보여주는 아키텍처 다이어그램입니다.

<img src="/img/assets/logto-connector-framework.png" alt="Connector framework" />

이들은 현재 대부분의 사용자의 요구를 충족시킬 수 있지만, 우리는 계속해서 발전해 나갈 것입니다.

더 많은 커넥터를 개발하는 동안, 여러분도 커넥터를 개발할 수 있습니다. 여러분의 커넥터를 전체 커뮤니티와 공유하는 것을 권장합니다.

커넥터를 구축하는 것부터 시작하여 Logto 프로젝트에 기여하는 방법을 알아봅시다.
