# 커스터마이제이션

브랜딩과 사용자 인터페이스부터 언어 및 도메인 설정까지 모든 측면을 커스터마이즈하여 Logto를 여러분만의 것으로 만드세요.

## 사전 구축된 UI \{#prebuilt-ui}

사전 구축된 UI 컴포넌트로 시작하여 필수적인 브랜드 요소를 커스터마이즈하세요. 다음 중 하나를 만드세요:

- 플랫폼 전반에 걸친 통합된 인증 (Authentication) 경험.
- 특정 애플리케이션 또는 조직을 위한 전용 로그인 페이지.

<DocCardList
  items={[
    {
      type: 'link',
      label: '브랜드에 맞추기',
      href: '/customization/match-your-brand',
      description:
        '사용할 준비가 된 로그인 경험을 위해 기본 브랜드 요소를 커스터마이즈하는 방법을 알아보세요.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## 콘텐츠 및 언어 \{#content--languages}

Logto는 기본적으로 15개 언어를 지원합니다. 언어 지원을 확장하거나 기존 번역을 수정하여 필요에 맞추세요.

## 사용자 정의 도메인 \{#custom-domain}

인증 (Authentication) URL에 자신의 도메인을 사용하여 브랜드 존재감을 강화하세요. 기본 Logto 도메인을 브랜드 도메인으로 교체하세요.

  items={[

## 실시간 미리보기 \{#live-preview}
