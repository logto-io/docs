# 계정 설정

Logto는 사용자가 Logto에 저장된 계정과 프로필을 관리할 수 있도록 두 가지 계정 설정 API 컬렉션을 제공합니다.

## 사용자 관리 API \{#user-management-apis}

Logto Management API는 사용자의 프로필 데이터를 관리하기 위한 가장 포괄적인 기능을 제공합니다. 이 API를 사용하여 자체 호스팅 사용자 관리 시스템이나 최종 사용자 계정 설정 페이지를 구현할 수 있습니다.

## 계정 API \{#account-apis}

Management API 외에도, Logto는 최종 사용자가 직접 계정 설정과 프로필 데이터를 관리할 수 있도록 하는 최종 사용자 계정 API 세트를 제공합니다. Management API와 달리, 계정 API는 세부적인 사용자 권한 제어와 함께 최종 사용자 컨텍스트에서 사용하도록 설계되었습니다. 클라이언트 측 애플리케이션에서 계정 API를 직접 사용하여 사용자 친화적인 계정 설정 페이지를 구축할 수 있습니다.
