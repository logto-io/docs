# 최종 사용자 흐름

최종 사용자 흐름은 사용자 상호작용에 대한 모든 검증 과정을 다루며, 다음과 같이 분류됩니다:

- **인증 (Authentication) 흐름**: 사용자 로그인, 등록, 비밀번호 재설정을 포함합니다. Logto는 <CloudLink to="/sign-in-experience">콘솔 > 로그인 경험</CloudLink>에서 구성된 기본 제공 경험을 제공하며, [경험 API](https://openapi.logto.io/group/endpoint-experience)를 통해 [UI를 가져올 수 있습니다](/customization/bring-your-ui).
- **계정 흐름**: 계정 설정, 사용자 프로필, 보안 검증을 포함합니다. [계정 API](https://openapi.logto.io/group/endpoint-account-center) 또는 [관리 API](https://openapi.logto.io/operation/operation-getuser)를 사용하여 구현할 수 있습니다.
- **조직 흐름**: B2B 또는 SaaS 서비스와 같은 다중 테넌시 제품을 위한 것입니다. 조직 생성, 멤버 초대 및 관리를 포함합니다. [관리 API](https://openapi.logto.io/operation/operation-listorganizations)를 사용하여 앱에 통합할 수 있습니다.
