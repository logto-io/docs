# 보안 검증

인증된 사용자가 비밀번호 변경, 결제, 급여 명세서나 은행 계좌 정보와 같은 민감한 정보에 접근하는 등의 민감하거나 고위험 행동을 시도할 때 추가적인 보안 조치가 필수적입니다. 이 흐름은 특히 은행, 의료, 정부 서비스와 같은 민감한 애플리케이션에서 매우 중요합니다.

*보안 검증*으로 알려진 이 과정은 사용자가 권한 있는 계정 소유자인지 확인하기 위해 아이덴티티를 다시 검증하도록 요구합니다. 보안 검증을 구현함으로써 시스템은 무단 접근에 대한 보호를 강화하고, 고위험 행동에 대한 중요한 보안 계층을 추가하며, 민감한 데이터를 보호하는 데 도움을 줍니다.
