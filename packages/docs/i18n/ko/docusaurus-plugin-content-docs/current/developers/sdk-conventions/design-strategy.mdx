# 디자인 전략

- 모든 프로그래밍 언어는 `${language}`라는 이름의 독립된 git 저장소를 가져야 합니다.
- 각 프로그래밍 언어 저장소는 모노 레포여야 합니다.
- SDK와 관련 샘플 프로젝트는 이 저장소 아래에 배치되어야 합니다.

예시:

<details>
  <summary>

### JS \{#js}

</summary>

- js (core)
- react
- react-sample

</details>

<details>
  <summary>

### Kotlin \{#kotlin}

</summary>

- kotlin (core)
- android
- android-sample

</details>
