# Logto를 애플리케이션에 통합하세요

Logto를 사용하여 사용자 대상 앱이든 기계 간 서비스든 애플리케이션에 인증을 추가하려면 다음 단계를 따르세요:

1. <CloudLink to="/applications">콘솔 > 애플리케이션</CloudLink>으로 이동하세요.

2. "애플리케이션 생성"을 클릭하여 새 애플리케이션을 추가하세요.

3. 애플리케이션 프레임워크를 선택하여 시작하세요. 프레임워크를 찾을 수 없는 경우, 애플리케이션 생성 페이지의 오른쪽 하단에 있는 "프레임워크 없이 앱 생성" 버튼을 클릭하여 [애플리케이션 유형](/integrate-logto/application-data-structure/#application-types)을 선택하여 앱을 생성하거나 기능 요청을 제출하거나 [SDK 규칙](/developers/sdk-conventions)을 따라 SDK를 기여하세요.

4. 프레임워크를 선택한 후, 프레임워크의 SDK에 대한 빠른 시작 가이드를 보게 됩니다. 애플리케이션을 구성하고 통합하는 단계를 따르세요. 통합 과정에 관련된 개념을 이해하는 데 도움이 필요하다면, [Logto 인증 흐름 이해하기](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/)를 참조하여 통합에 대한 깊은 이해를 얻을 수 있습니다.

:::note
콘솔의 가이드는 SDK를 사용하여 Logto를 빠르게 시작하기 위한 것입니다. 고급 SDK 사용을 포함한 완전한 통합 가이드는 [빠른 시작](/quick-starts) 섹션을 확인하세요.
:::
