Logto는 SMS 콘텐츠를 사용자 정의할 수 있는 네 가지 다른 템플릿을 제공합니다. 이 템플릿은 사용 유형에 따라 Register, SignIn, ForgotPassword, Generic으로 분류됩니다. 다양한 사용 사례에 대해 다른 템플릿을 사용하는 것이 좋습니다. 그렇지 않으면 속도 제한에 도달하여 서비스가 일시적으로 중단될 수 있습니다.

## SMS 템플릿 유형 및 예시 \{#sms-template-types-and-examples}

참고용 예시가 있습니다:

| usageType                | 시나리오                                                                                                                                                                                                                                                                                                                                                                                                      | 템플릿 예시                                                      |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| SignIn                   | 사용자가 [전화번호로 로그인](/end-user-flows/sign-up-and-sign-in/sign-in)하고 비밀번호 대신 SMS 인증 코드를 입력하여 인증합니다.                                                                                                                                                                                                                                                                              | Logto 로그인 인증 코드: `{{code}}`. 10분 후 만료됩니다.          |
| Register                 | 사용자가 [전화번호로 계정을 생성](/end-user-flows/sign-up-and-sign-in/sign-up)하고 Logto가 전화번호로 보낸 인증 코드를 입력하여 인증합니다.                                                                                                                                                                                                                                                                   | Logto 가입 인증 코드: `{{code}}`. 10분 후 만료됩니다.            |
| ForgotPassword           | 사용자가 로그인 중 비밀번호를 잊어버린 경우, 전화번호를 사용하여 신원을 확인한 후 [비밀번호를 재설정](/end-user-flows/sign-up-and-sign-in/reset-password)할 수 있습니다.                                                                                                                                                                                                                                      | Logto 비밀번호 재설정 인증 코드: `{{code}}`. 10분 후 만료됩니다. |
| Generic                  | 이 템플릿은 커넥터 구성 테스트, [로그인 후 전화번호 인증 또는 연결](/end-user-flows/account-settings/by-management-api#email-and-phone-number-verification) 등 다양한 시나리오에 대한 일반적인 백업 옵션으로 사용할 수 있습니다.                                                                                                                                                                              | Logto 인증 코드: `{{code}}`. 10분 후 만료됩니다.                 |
| OrganizationInvitation   | 이 템플릿을 사용하여 사용자가 조직에 가입하도록 초대 링크를 [보냅니다](/end-user-flows/organization-experience/invite-organization-members#configure-your-email-connector).                                                                                                                                                                                                                                   | Logto 조직 초대 인증 코드: `{{code}}`. 10분 후 만료됩니다.       |
| UserPermissionValidation | 앱 사용 중, 은행 송금, 사용 중인 리소스 삭제, 멤버십 취소와 같은 [추가 사용자 인증이 필요한](/end-user-flows/account-settings/by-account-api#verify-by-sending-a-verification-code-to-the-users-email-or-phone) 고위험 작업이나 상대적으로 높은 위험 수준의 작업이 있을 수 있습니다. `UserPermissionValidation` 템플릿은 이러한 상황에서 사용자가 받는 SMS 인증 코드의 내용을 정의하는 데 사용할 수 있습니다. | Logto 인증 코드: `{{code}}`. 10분 후 만료됩니다.                 |
| BindNewIdentifier        | 사용자가 프로필을 수정할 때, 현재 계정에 [전화번호를 연결](/end-user-flows/account-settings/by-account-api#manage-phone)할 수 있습니다. 이 경우, `BindNewIdentifier` 템플릿을 사용하여 인증 SMS의 내용을 사용자 정의할 수 있습니다.                                                                                                                                                                           | Logto 계정 연결 인증 코드: `{{code}}`. 10분 후 만료됩니다.       |

이러한 매개변수를 이해하는 것이 중요합니다:

- 인증 코드는 10분 동안 유효합니다. 현재 만료 시간을 사용자 정의하는 것은 지원하지 않습니다.
- Logto는 SMS 템플릿의 `{{code}}` 자리 표시자를 인증 코드로 대체합니다. 따라서 템플릿에 자리 표시자가 예약되어 있는지 확인하세요.

:::note
일부 국가 및 지역에서는 규정 준수 요구 사항으로 인해 승인되지 않은 콘텐츠를 SMS로 전송하는 것이 허용되지 않을 수 있습니다. SMS 템플릿은 사용 전에 SMS 제공업체에 등록 및 승인을 받아야 합니다. 이러한 경우, 콘텐츠는 템플릿 ID로 인덱싱되어 해당 템플릿에 매핑될 수 있습니다.
:::
