# 커넥터

커넥터는 Logto와 외부 서비스 간의 다리 역할을 합니다. 이를 통해 [비밀번호 없는 인증](https://auth.wiki/passwordless) 방법인 [이메일 인증](/end-user-flows/sign-up-and-sign-in/sign-in), [SMS 인증](/end-user-flows/sign-up-and-sign-in/sign-in), [소셜 로그인](/end-user-flows/sign-up-and-sign-in/social-sign-in), 및 [엔터프라이즈 SSO](/end-user-flows/enterprise-sso)를 사용할 수 있습니다.

사용자 이름과 비밀번호를 사용한 전통적인 계정 등록은 번거롭고 좌절감을 줄 수 있는 과정입니다. 사용자는 복잡한 비밀번호를 기억하는 데 어려움을 겪고, 불편함 때문에 새로운 계정을 만드는 것을 주저할 수 있습니다. 사용자가 이메일이나 전화번호를 제공하고 인증 코드를 통해 이를 인증하거나, Google/Facebook과 같은 계정이나 Microsoft와 같은 기업 계정을 사용하여 로그인할 수 있다면, 로그인 과정을 크게 단순화하고 사용자 이탈을 줄일 수 있습니다. 커넥터는 이러한 문제를 해결하기 위해 만들어졌습니다.
