Logto 메서드를 사용하여 프로그래밍 방식으로 사용자 정보를 검색할 수 있습니다:

- <code>{props.getIdTokenClaimsApi}</code>: 로컬 ID 토큰을 디코딩하여 사용자 정보를 가져옵니다. 일부
  클레임은 사용할 수 없을 수 있습니다.
- <code>{props.fetchUserInfoApi}</code>: [userinfo
  엔드포인트](https://openid.net/specs/openid-connect-core-1_0.html#UserInfo)에 요청을 보내 사용자
  정보를 가져옵니다.

사용자 정보 클레임은 사용자가 로그인할 때 사용한 스코프에 따라 검색할 수 있으며, 성능과 데이터 크기를 고려할 때 ID 토큰에 모든 사용자 클레임이 포함되지 않을 수 있습니다. 일부 사용자 클레임은 userinfo 엔드포인트에서만 사용할 수 있습니다 (아래 관련 목록 참조).
