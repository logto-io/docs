거의 다 왔습니다! 마지막 단계에서는 `https://your-awesome-site.webflow.io/callback`을 Redirect URI로 사용했으며, 이제 이를 적절히 처리해야 합니다.

먼저 Webflow에서 "Callback" 페이지를 만들고, 단순히 "Redirecting..."이라는 정적 텍스트를 넣습니다. 그런 다음 "Callback" 페이지에 다음과 같은 페이지 수준의 사용자 정의 코드를 추가하세요.

```html
<script type="module">
  (async () => {
    // SDK 메서드를 호출하여 로그인 콜백 로직을 처리합니다
    await logtoClient.handleSignInCallback(window.location.href);

    // 처리가 완료되면 홈 페이지로 리디렉션합니다
    window.location.assign('https://your-awesome-site.webflow.io');
  })();
</script>
```
