:::

이 단계에서는 Webflow 사이트에 전역 수준의 사용자 정의 코드를 추가하겠습니다. Webflow에서는 NPM이 지원되지 않으므로, Logto SDK를 가져오기 위해 [jsdelivr.com](https://www.jsdelivr.com/) CDN 서비스를 사용할 것입니다.

"Site settings" 페이지를 열고 "Custom code" 섹션으로 이동합니다. 다음 코드를 "Head code" 섹션에 추가하세요.

```html
<script type="module">
  // jsdelivr CDN에서 \`@logto/browser\` SDK를 가져옵니다
  import LogtoClient from 'https://esm.run/@logto/browser';

  // \`logtoClient\` 인스턴스를 window 객체에 할당하여
  // 다른 페이지에서 전역적으로 사용할 수 있도록 합니다
  window.logtoClient = new LogtoClient({
    endpoint: '<your-logto-endpoint>', // 예: http://localhost:3001
    appId: '<your-application-id>',
  });
</script>
```
