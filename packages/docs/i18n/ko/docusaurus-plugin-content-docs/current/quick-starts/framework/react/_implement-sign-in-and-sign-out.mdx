### 로그인 및 로그아웃 구현하기 \{#implement-sign-in-and-sign-out}

우리는 인증 (Authentication) 흐름을 쉽게 관리할 수 있도록 두 가지 훅 `useHandleSignInCallback()`과 `useLogto()`를 제공합니다.

<SignInNote calling=".signIn()" />

```tsx title="pages/Home/index.tsx"
import { useLogto } from '@logto/react';

const Home = () => {
  const { signIn, signOut, isAuthenticated } = useLogto();

  return isAuthenticated ? (
    <button onClick={signOut}>Sign Out</button>
  ) : (
    <button onClick={() => signIn('http://localhost:3000/callback')}>Sign In</button>
  );
};
```
