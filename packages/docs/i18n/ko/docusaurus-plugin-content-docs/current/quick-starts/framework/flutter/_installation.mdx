<TabItem value="pub" label="pub.dev">

`logto_dart_sdk 패키지`를 pub 패키지 관리자를 사용하여 직접 설치할 수 있습니다. 프로젝트 루트에서 다음 명령을 실행하세요:

```bash
flutter pub add logto_dart_sdk
```

또는 `pubspec.yaml` 파일에 다음을 추가하세요:

```yaml
dependencies:
  logto_dart_sdk: ^3.0.0
```

그런 다음 다음을 실행하세요:

```bash
flutter pub get
```
