`.signOut()`을 호출하면 메모리와 localStorage에 존재하는 모든 Logto 데이터를 지웁니다.

로그아웃 후, 사용자를 웹사이트로 리디렉션하는 것이 좋습니다. Admin Console에서 Post Sign-out URI 중 하나로 `https://your-awesome-site.webflow.io`를 추가하고, `.signOut()`을 호출할 때 이 URL을 매개변수로 사용하세요.

#### 로그아웃 버튼 구현하기 \{#implement-a-sign-out-button}

Webflow 디자이너로 돌아가서 홈 페이지에 "로그아웃" 버튼을 추가하세요. 마찬가지로 버튼에 ID "sign-out"을 할당하고, 페이지 수준의 사용자 정의 코드에 다음 코드를 추가하세요.

```js
const signOutButton = document.getElementById('sign-out');
const onClickSignOut = () => logtoClient.signOut('https://your-awesome-site.webflow.io');
signOutButton.addEventListener('click', onClickSignOut);
```
