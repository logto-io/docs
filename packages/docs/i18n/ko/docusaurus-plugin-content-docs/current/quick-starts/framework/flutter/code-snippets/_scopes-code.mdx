SDK 패키지에는 미리 정의된 스코프를 사용할 수 있도록 도와주는 내장 `LogtoUserScope` 열거형도 제공합니다.

```dart
// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",
  appId: "<your-app-id>",
  // highlight-start
  scopes: [LogtoUserScope.email.value, LogtoUserScope.phone.value],
  // highlight-end
);
```
