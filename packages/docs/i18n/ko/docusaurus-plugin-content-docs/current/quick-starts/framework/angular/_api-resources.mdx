이제 액세스 토큰은 불투명 토큰 대신 JSON Web Token (JWT) 형식으로 제공됩니다.

:::warning
`resource`가 설정되면 `autoUserInfo`와 `renewUserInfoAfterTokenRenew`는 비활성화됩니다. 이는 액세스 토큰이 사용자 정보 엔드포인트가 아닌 특정 API 리소스를 위해 요청되기 때문입니다.
:::

현재, Logto 공식 SDK만이 사용자 정보와 API 리소스 액세스 토큰을 모두 요청할 수 있는 기능을 지원합니다. 둘 다 요청해야 하는 경우, 주저하지 말고 저희에게 연락해 주세요.
