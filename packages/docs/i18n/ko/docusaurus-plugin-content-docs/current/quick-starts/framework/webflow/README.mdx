# Webflow 애플리케이션에 인증 (Authentication)을 추가하세요

<GuideTip />

## 사전 준비 사항 \{#prerequisits}

1. Webflow와 Logto를 통합하려면 Webflow의 "Custom code" 기능이 필요하며, 최소한 "Basic" 플랜이 필요합니다.
2. Webflow 사이트, 기존 사이트를 사용하거나 새로 만드세요.
