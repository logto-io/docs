`LogtoConfig` 객체로 `LogtoClient` 인스턴스를 생성하여 클라이언트를 초기화하세요.

```swift title="ContentView.swift"
import Logto
import LogtoClient

let config = try? LogtoConfig(
  endpoint: "<your-logto-endpoint>", // 예: http://localhost:3001
  appId: "<your-app-id>"
)
let client = LogtoClient(useConfig: config)
```

:::info

기본적으로, 우리는 ID 토큰 및 리프레시 토큰과 같은 자격 증명을 Keychain에 저장합니다. 따라서 사용자가 다시 돌아왔을 때 다시 로그인할 필요가 없습니다.

이 동작을 끄려면, `usingPersistStorage`를 `false`로 설정하세요:

```swift
let config = try? LogtoConfig(
  // ...
  usingPersistStorage: false
)
```

:::
