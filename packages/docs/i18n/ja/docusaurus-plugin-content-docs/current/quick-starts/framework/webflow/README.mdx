## 前提条件 \{#prerequisits}

1. Webflow と Logto を統合するには、Webflow の「カスタムコード」機能が必要で、少なくとも「Basic」プランが必要です。
2. Webflow サイト。既存のサイトを使用するか、新しいサイトを作成します。
