### ユーザー情報を表示する \{#display-user-information}

ユーザーの情報を表示するには、`logtoClient.idTokenClaims` ゲッターを使用できます。例えば、Flutter アプリで：

<DisplayUserInfo />

追加のスコープ (Scopes) をリクエストするには、スコープを `LogtoConfig` オブジェクトに渡すことができます。例えば：

<ClaimsNeedNetworkRequest
  type="method"
  method="logtoClient.getUserInfo()"
