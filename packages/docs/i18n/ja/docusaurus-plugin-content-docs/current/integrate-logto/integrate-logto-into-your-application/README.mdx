# Logto をアプリケーションに統合する

Logto を使用してアプリケーションに認証 (Authentication) を追加するには、次の手順に従ってください。これは、ユーザー向けアプリでもマシン間通信サービスでも同様です：

1. <CloudLink to="/applications">Console > Applications</CloudLink> に移動します

2. 「Create application」をクリックして新しいアプリケーションを追加します

3. アプリケーションフレームワークを選択して開始します。フレームワークが見つからない場合は、アプリケーション作成ページの右下にある「Create app without framework」ボタンをクリックして、[アプリケーションタイプ](/integrate-logto/application-data-structure/#application-types) を選択してアプリを作成するか、[SDK 規約](/developers/sdk-conventions) に従って機能リクエストを提出するか、SDK を提供してください。

4. フレームワークを選択すると、そのフレームワークの SDK のクイックスタートガイドが表示されます。手順に従ってアプリケーションを設定し、統合します。統合プロセスに関与する概念を理解するのに役立つ場合は、[Logto 認証 (Authentication) フローの理解](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) を参照して、統合の深い理解を得ることができます。

:::note
コンソールのガイドは、Logto を使用して SDK を使用したクイックスタートのためのものです。高度な SDK の使用を含む完全な統合ガイドについては、[クイックスタート](/quick-starts) セクションを確認してください。
:::

5. 完了したら、Logto についてさらに探索する準備が整いました：
