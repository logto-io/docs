# 組織体験

[組織 (Organization)](/organizations) 体験とは、特に [マルチテナントアプリケーション](https://auth.wiki/multi-tenancy) において、ビジネスクライアントとその従業員が使用するユーザーインターフェースとフローを指します。Logto は、この体験を Logto Management API を使用してアプリに迅速に統合するためのいくつかのガイドを提供しています。
