# Amazon を使用してソーシャルログインを設定する

Amazon ソーシャルサインインのための公式 Logto コネクター。

<GuideTip />

## 始めましょう \{#get-started}

Amazon コネクターを使用すると、エンドユーザーは Amazon OAuth 2.0 認証プロトコルを介して自分の Amazon アカウントを使用してアプリケーションにサインインできます。

<Integration />

## Amazon コネクターをテストする \{#test-amazon-connector}

これで完了です。Amazon コネクターが利用可能になりました。[サインイン体験でソーシャルコネクターを有効にする](/connectors/social-connectors/#enable-social-sign-in) ことを忘れないでください。
