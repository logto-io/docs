### Line チャンネルの設定 \{#setup-a-line-channel}

[Line Developers](https://developers.line.biz/console/) にアクセスし、Line ビジネスアカウントでサインインします。アカウントをお持ちでない場合は、登録できます。

次に、[このリンク](https://developers.line.biz/console/register/line-login/provider/) にアクセスしてチャンネルを作成します。

**ステップ 1:** チャンネルの詳細を入力します。

フォームに記入し、チャンネルを作成します。

**ステップ 2:** コールバック URL を設定します。

チャンネルの詳細ページに移動し、「LINE ログイン」タブを見つけ、「Callback URL」フィールドを編集します。

この場合、`${your_logto_endpoint}/callback/${connector_id}` になります。例：`https://foo.logto.app/callback/${connector_id}`。`connector_id` は Logto 管理コンソールのコネクター詳細ページの上部バーに表示されます。

### コネクターの設定 \{#configure-your-connector}

Logto コネクター設定で、アプリの「Auth」タブ、「アプリケーションクレデンシャル」セクションから取得した値で次のフィールドを入力します：

- **clientId:** アプリのチャンネル ID。
- **clientSecret:** アプリのチャンネルシークレット。

`scope` はスペースで区切られた OIDC スコープのリストです。指定しない場合、デフォルトのスコープは `openid profile` です。

### 設定タイプ \{#config-types}

| 名前         | タイプ |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
