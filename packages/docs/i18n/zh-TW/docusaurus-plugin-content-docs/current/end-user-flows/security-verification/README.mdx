# 安全驗證

當已驗證的使用者嘗試執行敏感或高風險操作時，例如更改密碼、進行付款或存取敏感資訊（如薪資或銀行帳戶詳細資料），需要額外的安全措施。此流程在銀行、醫療保健和政府服務等敏感應用程式中特別重要。

這個過程稱為 _安全驗證_，要求使用者重新驗證其身分，以確認他們是授權的帳戶持有人。透過實施安全驗證，系統加強了對未經授權存取的保護，為高風險操作增加了一個關鍵的安全層，並有助於保護敏感資料。
