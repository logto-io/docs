# 終端使用者流程

終端使用者流程涵蓋所有使用者互動的驗證過程，分類如下：

- **驗證流程**：涉及使用者登入、註冊和重設密碼。Logto 提供現成的體驗，可在 <CloudLink to="/sign-in-experience">Console > 登入體驗</CloudLink> 中配置，你也可以透過 [Experience APIs](https://openapi.logto.io/group/endpoint-experience) [自定義你的 UI](/customization/bring-your-ui)。
- **帳戶流程**：涉及帳戶設置、使用者資料和安全驗證。使用 [Account APIs](https://openapi.logto.io/group/endpoint-account-center) 或 [Management APIs](https://openapi.logto.io/operation/operation-getuser) 來實現。
- **組織流程**：適用於多租戶產品，如 B2B 或 SaaS 服務。涉及組織創建、成員邀請和管理。使用 [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) 將其整合到你的應用程式中。
