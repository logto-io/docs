# 組織使用體驗 (Experience)

[組織 (Organization)](/organizations) 使用體驗指的是你的商業客戶及其員工所使用的使用者介面和流程，特別是在[多租戶應用程式](https://auth.wiki/multi-tenancy)中。Logto 提供多個指南，幫助你快速將此體驗整合到你的應用程式中，使用 Logto Management API。
