# 帳戶設定

Logto 提供兩組帳戶設定 API，讓使用者管理儲存在 Logto 的帳戶和個人資料。

## 使用者管理 API \{#user-management-apis}

Logto Management API 提供最全面的功能來管理使用者及其個人資料數據。你可以使用這些 API 實現自託管的使用者管理系統或終端使用者帳戶設定頁面。

## 帳戶 API \{#account-apis}

除了 Management API，Logto 還提供一組終端使用者帳戶 API，讓終端使用者直接管理其帳戶設定和個人資料數據。與 Management API 不同，帳戶 API 設計用於終端使用者環境，具有詳細的使用者權限控制。你可以直接在客戶端應用程式中使用帳戶 API，建立使用者友好的帳戶設定頁面。
