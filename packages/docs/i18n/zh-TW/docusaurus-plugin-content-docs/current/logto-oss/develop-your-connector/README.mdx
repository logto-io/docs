# 開發你的連接器

目前，Logto 已提供許多廣泛使用的社交登入連接器，如微信、支付寶、Google、Facebook 等，以及簡訊 / 電子郵件連接器，如阿里雲、SendGrid 和 Twilio。

以下是一個架構圖，說明連接器如何與 Logto 核心服務協作。

這些應該能滿足大多數使用者的需求，但我們將持續進步。

在我們開發更多連接器的同時，你也可以開發自己的連接器。我們鼓勵你與整個社群分享你的連接器。
