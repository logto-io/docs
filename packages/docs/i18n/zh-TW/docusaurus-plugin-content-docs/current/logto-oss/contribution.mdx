# 貢獻

感謝你對貢獻 Logto 的興趣！我們非常感謝社群的貢獻，讓我們的專案更加健全、使用者友好且多功能。無論是修復錯誤、改善文件、創建你的連接器、添加新功能，或提供反饋，你的努力都非常寶貴。

## 如何貢獻？\{#how-to-contribute}

我們努力讓貢獻過程盡可能順暢。在開始之前，請查看我們的 [GitHub 貢獻指南](https://github.com/logto-io/logto/blob/master/.github/CONTRIBUTING.md)。此指南概述了我們的流程、編碼標準，以及成功貢獻 Logto 的步驟。

## 需要幫助？\{#need-help}

如果你有任何問題或需要指導，隨時透過我們的 [GitHub 討論](https://github.com/logto-io/logto/discussions)、[提交問題](https://github.com/logto-io/logto/issues)，或 [聯繫我們](https://logto.io/contact)，我們將確保你擁有成功所需的一切。
