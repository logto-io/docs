# 安裝 Logto

## 互動式 \{#interactive}

在終端中運行以下命令以進行互動式安裝：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## 選項與靜默安裝 \{#options-and-silent-installation}

或者，你可以添加以下選項以跳過某些問題：

```bash
# 所有這些選項都是可選的
--db-url      Logto 資料庫的 Postgres URL
-p, --path    Logto 實例的路徑，必須是不存在的路徑
--ss          跳過 Logto 資料庫的預設資料
--oc          安裝後添加官方連接器
```

運行 `logto init --help` 以獲取完整的幫助信息。

以下是一個執行靜默安裝的範例（例如，在 CI 中）：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

它將按以下順序執行：

1. 下載並解壓 Logto 到 `./logto`
2. 嘗試使用 URL `postgresql://your-postgres-dsn:5432/logto` 初始化並填充資料庫
3. 將 Logto 官方連接器添加到 `./logto/packages/core/connectors`
