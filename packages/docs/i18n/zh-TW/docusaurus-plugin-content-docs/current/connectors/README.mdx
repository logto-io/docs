# 連接器 (Connectors)

連接器是 Logto 與外部服務之間的橋樑。它們啟用 [無密碼驗證](https://auth.wiki/passwordless) 方法，如 [電子郵件驗證](/end-user-flows/sign-up-and-sign-in/sign-in)、[簡訊驗證](/end-user-flows/sign-up-and-sign-in/sign-in)、[社交登入](/end-user-flows/sign-up-and-sign-in/social-sign-in) 和 [企業級單一登入 (SSO)](/end-user-flows/enterprise-sso)。

傳統的使用者名稱和密碼註冊帳戶過程可能繁瑣且令人沮喪。使用者經常難以記住複雜的密碼，並可能因不便而不願意創建新帳戶。如果使用者可以提供電子郵件或電話號碼並通過驗證碼進行驗證；或使用 Google/Facebook 等帳戶或 Microsoft 等企業帳戶登入，這可以大大簡化登入過程並減少使用者流失。連接器就是為了解決這個問題而創建的。
