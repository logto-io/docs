# 設定 OpenID Connect (OIDC) 社交登入

Logto 的 OpenID Connect (OIDC) 協議官方連接器。

<GuideTip />

## 開始使用 \{#get-started}

OIDC 連接器使 Logto 能夠連接到支持 OIDC 協議的任意社交身分提供者。

:::note

OIDC 連接器是 Logto 中一種特殊的連接器，你可以新增一些基於 OIDC 的連接器。

:::
