# 設定 Apple 社交登入

Logto 的 Apple 社交登入官方連接器。

<GuideTip />

## 開始使用 \{#get-started}

如果你不瞭解連接器的概念或不知道如何將此連接器新增到你的登入體驗中，請參閱 [Logto 教程](/connectors/social-connectors)。

<Integration />

### 配置權限範圍的陷阱 \{#pitfalls-of-configuring-scope}

如果你已配置應用程式在使用者已使用 Apple 登入後請求其電子郵件地址，則無法檢索這些現有使用者的電子郵件地址，即使他們再次使用 Apple ID 登入。為了解決此問題，你需要指導使用者訪問 [Apple ID 帳戶管理頁面](https://appleid.apple.com/account/manage) 並從「使用 Apple 登入」部分中移除你的應用程式。這可以通過在應用程式的詳細資訊頁面上選擇「停止使用 Apple 登入」來完成。

例如，如果你的應用程式請求使用者的電子郵件和姓名（`email name` 權限範圍），新使用者在首次登入時看到的使用者授權頁面應類似如下：
