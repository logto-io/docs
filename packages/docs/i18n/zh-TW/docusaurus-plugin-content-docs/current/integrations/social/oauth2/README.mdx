# 設定 OAuth 2.0 協議的社交登入

Logto 的官方 OAuth 2.0 協議連接器。

<GuideTip />

## 開始使用 \{#get-started}

OAuth 連接器使 Logto 能夠連接支援 OAuth 2.0 協議的任意社交身分提供者。

:::note

OAuth 連接器是 Logto 中一種特殊的連接器，你可以新增一些基於 OAuth 協議的連接器。

:::
