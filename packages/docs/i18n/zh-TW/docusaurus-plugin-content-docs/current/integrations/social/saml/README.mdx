# 設定 SAML 社交登入

Logto 的 SAML 協議官方連接器。

<GuideTip />

## 開始使用 \{#get-started}

SAML 連接器使 Logto 能夠連接到支援 SAML 協議的任意社交身分提供者 (IdP)。

:::note

SAML 連接器是 Logto 中一種特殊的連接器，你可以新增幾個基於 SAML 的連接器。

:::
