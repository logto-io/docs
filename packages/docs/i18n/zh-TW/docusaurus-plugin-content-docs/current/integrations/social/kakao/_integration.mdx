## 在 Kakao 開發者控制台中設定專案 \{#set-up-a-project-in-the-kakao-devlopers-console}

- 訪問 [Kakao 開發者控制台](https://developers.kakao.com/console/app) 並使用你的 Kakao 帳戶登入。
- 點擊 **新增應用程式** 以創建新專案或選擇現有專案。

## 配置 Kakao 登入 \{#configure-kakao-login}

### 啟用 Kakao 登入 \{#activate-kakao-login}

- 從選單中點擊 **產品設定 -> Kakao 登入**。
- 開啟 `Kakao 登入啟用`
- 在 `Redirect URI` 中新增以下 URL
  - `http(s)://YOUR_URL/callback/${connector_id}`（`connector_id` 可以在 Logto 管理控制台連接器詳細資訊頁面的頂部欄位找到。）
  - （請將 `YOUR_URL` 替換為你的 `Logto` URL，並根據情況選擇 `http` 或 `https`。）

### 隱私設定 \{#privacy-setting}

- 從選單中點擊 **產品設定 -> Kakao 登入 -> 同意項目**。
- 將 `暱稱`、`個人資料圖片` 和 `電子郵件` 的狀態更改為 **需要同意**（由於專案設定，你可能無法將 `電子郵件` 更改為 **需要同意**。）

### 安全設定（可選）\{#security-setting-optional}

- 從選單中點擊 **產品設定 -> Kakao 登入 -> 安全**。
- 點擊 `客戶端密碼碼` 以生成密碼碼。
- 將 `啟用狀態` 更改為啟用。（如果啟用，則需要 `密碼碼`。）

## 配置 Logto \{#configure-logto}

### 配置類型 \{#config-types}

| 名稱         | 類型    |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId \{#clientid}

`clientId` 是專案的 `REST API key`。
（你可以在 Kakao 開發者控制台的專案 `摘要` 中找到。）

#### clientSecret \{#clientseceret}

`clientSecret` 是專案的 `密碼碼`。
（請查看 [安全設定（可選）](#security-setting-optional)）
