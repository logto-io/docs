# 設定 GitHub 社交登入

Logto 的 GitHub 社交登入官方連接器。

<GuideTip />

## 開始使用 \{#get-started}

GitHub 連接器使終端使用者能夠透過 GitHub OAuth 2.0 驗證協議，使用他們自己的 GitHub 帳戶登入你的應用程式。
