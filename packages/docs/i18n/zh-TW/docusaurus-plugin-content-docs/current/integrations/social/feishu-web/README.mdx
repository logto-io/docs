# 設定 Feishu 社交登入

Feishu Web：一個先進的企業協作與管理平台，提供無縫的辦公協作，協調團隊目標，充分激活組織和個人。

<GuideTip />

## 開始使用 Feishu 社交登入 \{#getting-started-with-feishu-social-sign-in}

Feishu web 連接器專為桌面網頁應用程式設計，使用 OAuth 2.0 驗證流程。

## 註冊 Feishu 開發者帳戶 \{#register-a-feishu-developer-account}

如果你還沒有 Feishu 開發者帳戶，請在 [Feishu 開放平台](https://open.feishu.cn/app)上註冊。

## 建立應用程式 \{#create-an-application}

1. 在 [開發者控制台](https://open.feishu.cn/app)上，點擊「建立自定義應用程式」。
2. 填寫應用程式名稱、描述，選擇圖示，然後點擊「建立」按鈕。
3. 在左側邊欄中，點擊「安全設置」，將「重定向 URL」填寫為 `${logto_endpoint}/callback/${connector_id}`。對應的值可以在 Logto Console 的 Feishu 連接器詳細資訊頁面的 `Callback URI` 欄位中找到。
4. 在「憑證與基本資訊」中，你可以獲取「App ID」和「App Secret」。

:::note
對於非企業內部使用，你還需要在「版本管理與發佈」頁面中點擊「建立版本」按鈕。
「App ID」和「App Secret」只有在應用程式狀態變更為「啟用」後才會生效。
:::
