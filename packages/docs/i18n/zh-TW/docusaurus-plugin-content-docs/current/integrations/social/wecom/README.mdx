# 設定 WeCom 社交登入 (OAuth 2.0)

WeCom（可能稱為 WXwork）社交登入的自訂連接器。

<GuideTip />

## 開始使用 \{#get-started}

使用管理員帳戶登入 [WeCom WebUI](https://work.weixin.qq.com/) 或從 WeCom 應用程式中點擊 **管理企業**。

在「應用管理」標籤中，向下滾動頁面並點擊「創建應用」。

根據你的應用程式填寫適當的信息，然後創建。

現在我們有了 Agent ID（不是 APPID）和 Secret。

### 網站信息 \{#website-info}

在此頁面上設置你需要的內容。可能會是：

- 允許的使用者：_誰可以看到此應用_
- 應用首頁：_你的應用首頁。例如，`logto.io/demo-app`_

**重要**
此頁面上有三個關於「開發者接口」的項目。

1. 網頁授權與 JS-SDK；
2. 通過 WeCom 授權登入；
3. 企業可信 IP；

根據 WeCom 的指南填寫它們。

### Corp ID \{#corp-id}

如果你熟悉微信開發，你可能會注意到 Corp ID 的使用與 APP ID 相同。

你可以在「我的企業」標籤頁底部找到 Corp ID，看起來像 **ww\*\*\*\*** 。

## 配置連接器 \{#configure-the-connector}

所以我們有了 Agent ID、Secret 和 Corp ID。

讓我們完成連接器的表單。

你可以將 `Scope` 欄位留空，因為它是可選的。或者，你可以填寫 `snsapi_base` 或 `snsapi_privateinfo`。

**保存並完成**
