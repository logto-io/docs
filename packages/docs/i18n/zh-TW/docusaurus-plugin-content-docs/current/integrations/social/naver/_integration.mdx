## 開發者網站目前僅支援韓文 \{#developer-site-only-korean-support-now}

目前 `Naver Developers` 網站僅支援韓文。請考慮使用翻譯工具。

## 用於生產環境 \{#for-the-production}

- 在生產環境中，你必須獲得 Naver 團隊的審核。否則，只有註冊的使用者可以登入。
  - 你可以從 `맴버관리(Member Manage)` 菜單中新增測試者。
- 要獲得審核，請檢查應用程式專案設定中的 `API 설정(API Setting)` 下的 `애플리케이션 개발 상태(Application Development Status)`。

## 在 Naver Developers 設定專案 \{#set-up-a-project-in-the-naver-developers}

- 訪問 [Naver Developers](https://developers.naver.com/main/) 並使用你的 Naver 帳戶登入。
- 從菜單中點擊 **Application -> 어플리케이션 등록** 來創建新專案。
- 按照以下指示創建應用程式。

### 應用程式名稱 (어플리케이션 이름) \{#application-name-어플리케이션-이름}

- 在 `어플리케이션 이름` 中輸入你的應用程式名稱（此名稱會在使用者登入時顯示）。

### API 使用 (사용 API) \{#api-usage-사용-api}

- 在 `사용 API(API Usage)` 中選擇 `네이버 로그인(Naver Login)`
- 在 `권한(Role)` 中將 `이메일 주소(Email Address), 별명(Nickname), 프로필 사진(Profile Image)` 勾選為 `필수(Necessary)`（你可以將這些選項勾選為 `추가(Add)` 作為可選項，但無法從使用者獲取資訊）。

### 登入開放 API 服務環境 (로그인 오픈 API 서비스 환경) \{#sign-in-open-api-service-environment-로그인-오픈-api-서비스-환경}

- 對於 `로그인 오픈 API 서비스 환경(Sign in Open API Service Environment)`，新增兩個環境 `PC웹(PC Web)` 和 `모바일웹(Mobile Web)`。

#### PC Web (PC 웹) \{#pc-web-pc-웹}

- 對於 `서비스 URL(Service URL)`，輸入 `http(s)://YOUR_URL`（例如：https://logto.io）
- 對於 `네이버 로그인(Naver Login) Callback URL`，輸入 `http(s)://YOUR_URL/callback/${connector_id}`（例如：https://logto.io/callback/${connector_id}）

#### Mobile Web (Mobile 웹) \{#mobile-web-mobile-웹}

- 對於 `서비스 URL(Service URL)`，輸入 `http(s)://YOUR_URL`（例如：https://logto.io）
- 對於 `네이버 로그인(Naver Login) Callback URL`，輸入 `http(s)://YOUR_URL/callback/${connector_id}`（例如：https://logto.io/callback/${connector_id}）

:::caution

`connector_id` 可以在 Logto 管理控制台連接器詳細頁面的頂部欄位找到。

:::

## 配置 Logto \{#configure-logto}

### 配置類型 \{#config-types}

| 名稱         | 類型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |

#### clientId \{#clientid}

`clientId` 是你專案的 `Client ID`。
（你可以在 Naver developers 的專案 `애플리케이션 정보(Application Info)` 中找到它。）

#### clientSecret \{#clientseceret}

`clientSecret` 是你專案的 `Client Secret`。
（你可以在 Naver developers 的專案 `애플리케이션 정보(Application Info)` 中找到它。）
