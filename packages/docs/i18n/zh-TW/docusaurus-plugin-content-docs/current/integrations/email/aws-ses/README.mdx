# 使用 AWS Direct Mail 設定電子郵件驗證

Logto 的官方 AWS 連接器，用於直接郵件服務。

<GuideTip />

## 開始使用 \{#get-started}

Amazon SES 是一個雲端電子郵件服務提供者，可以整合到任何應用程式中進行大量電子郵件發送。

Logto 團隊調用 Amazon Simple Email Service API，藉此 Logto 終端使用者可以透過郵件驗證碼註冊並登入他們的 Logto 帳戶。
