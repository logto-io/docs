# 將 Logto 整合到你的應用程式中

按照以下步驟，使用 Logto 為你的應用程式新增驗證 (Authentication)，無論是面向使用者的應用程式還是機器對機器服務：

1. 前往 <CloudLink to="/applications">控制台 > 應用程式</CloudLink>

2. 點擊「建立應用程式」以新增應用程式

3. 選擇你的應用程式框架開始。如果找不到你的框架，請點擊應用程式建立頁面右下角的「建立無框架應用程式」按鈕，透過選擇 [應用程式類型](/integrate-logto/application-data-structure/#application-types) 來建立應用程式，或根據我們的 [SDK 規範](/developers/sdk-conventions) 提交功能請求或貢獻 SDK。

4. 選擇框架後，你將看到該框架 SDK 的快速入門指南。按照步驟配置並整合你的應用程式。如果需要幫助理解整合過程中的概念，可以參考 [理解 Logto 驗證流程](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) 以深入了解整合。

:::note
控制台中的指南僅適用於使用我們的 SDK 快速開始 Logto。完整的整合指南，包括進階 SDK 使用，請查看 [快速入門](/quick-starts) 部分。
:::
