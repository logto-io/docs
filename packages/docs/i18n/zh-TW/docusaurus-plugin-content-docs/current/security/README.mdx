# 安全性

現代驗證安全性面臨的威脅包括網路釣魚、憑證填充、暴力破解攻擊、勒索軟體、DDoS 到 AI 驅動的攻擊。保護使用者身分對於維護品牌信任和合規性至關重要。

Logto 提供強大的安全存取管理，旨在直接應對這些風險。我們優先考慮主動威脅預防和彈性，確保你的系統在不影響可用性的情況下保持防護。對於 Logto 來說，安全性不是事後考慮，而是基礎，讓企業在威脅不斷演變的時代中蓬勃發展，而防禦措施發展得更快。
