# 設計策略

- 每個程式語言應有一個獨立的 git 儲存庫，命名為 `${language}`。
- 每個程式語言的儲存庫應為單一儲存庫（mono repo）。
- SDK 及其相關範例專案應放置於此儲存庫下。

範例：

<details>
  <summary>

### JS \{#js}

</summary>

- js (核心)
- react
- react-sample

</details>

<details>
  <summary>

### Kotlin \{#kotlin}

</summary>

- kotlin (核心)
- android
- android-sample

</details>
