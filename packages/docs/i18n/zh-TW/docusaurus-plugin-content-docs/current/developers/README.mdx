# 開發者

Logto 是一個基於 [OAuth 2](https://auth.wiki/oauth-2.0) 和 [OIDC](https://auth.wiki/openid-connect) 協議的 [身分與存取管理 (IAM, Identity and Access Management)](https://auth.wiki/iam) 服務。像 Logto 這樣的 IAM 服務通常作為其他網路服務的基礎；這些網路服務中的各種授權狀態直接受到 Logto 的影響。

為了方便使用者，Logto 提供了一系列常用的開發者功能。
