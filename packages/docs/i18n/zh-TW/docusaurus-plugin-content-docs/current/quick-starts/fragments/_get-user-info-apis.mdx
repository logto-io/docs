你可以使用這些 Logto 方法以程式化方式檢索使用者資訊：

- <code>{props.getIdTokenClaimsApi}</code>：透過解碼本地 ID 權杖 (ID token) 獲取使用者資訊。某些宣告
  (Claims) 可能無法取得。
- <code>{props.fetchUserInfoApi}</code>：透過向 [userinfo
  endpoint](https://openid.net/specs/openid-connect-core-1_0.html#UserInfo)
  發送請求來獲取使用者資訊。

需要注意的是，可以檢索的使用者資訊宣告 (Claims) 取決於使用者登入時使用的權限範圍 (Scopes)。考慮到效能和資料大小，ID 權杖 (ID token) 可能不包含所有使用者宣告 (Claims)，某些使用者宣告 (Claims) 僅在 userinfo endpoint 中可用（請參閱下方相關列表）。
