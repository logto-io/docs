#### 組織 API 資源 (Organization API resources) \{#organization-api-resources}

若要為組織中的 API 資源取得存取權杖 (Access token)，可以使用 `getAccessToken` 方法，並將 API 資源和組織 ID 作為參數：

```go
accessToken, error := client.GetAccessToken(
  'https://shopping.your-app.com/api',
  organizationId
);
```
