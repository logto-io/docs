// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",

  // highlight-start
  scopes: ["email", "phone"],
  // highlight-end

我們也在 SDK 套件中提供了一個內建的 `LogtoUserScope` 列舉，幫助你使用預定義的權限範圍 (Scopes)。

```dart
// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",
  appId: "<your-app-id>",
  // highlight-start
  scopes: [LogtoUserScope.email.value, LogtoUserScope.phone.value],
  // highlight-end
);
```
