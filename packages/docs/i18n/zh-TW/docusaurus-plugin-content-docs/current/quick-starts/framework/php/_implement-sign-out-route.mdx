當使用者發出登出請求後，Logto 將會清除會話中的所有使用者驗證資訊。

要清理 PHP 會話和 Logto 會話，可以實作一個登出路由，如下所示：

<CodeBlock language="php">
  {`Route::get('/sign-out', function () {
  return redirect(
    // 成功登出後將使用者重定向到首頁
    $client->signOut('${defaultBaseUrl}')
  );
});`}
</CodeBlock>
