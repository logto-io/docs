### 實作登入與登出 \{#implement-sign-in-and-sign-out}

我們提供兩個可組合的函式 `useHandleSignInCallback()` 和 `useLogto()`，可以幫助你輕鬆管理驗證 (Authentication) 流程。

<SignInNote calling=".signIn()" />

```tsx title="views/HomeView.vue"
import { useLogto } from '@logto/vue';

const { signIn, signOut, isAuthenticated } = useLogto();

const onClickSignIn = () => signIn('http://localhost:3000/callback');
const onClickSignOut = () => signOut('http://localhost:3000');
```

```html
<template>
  <button v-if="!isAuthenticated" @click="onClickSignIn">登入</button>
  <button v-else @click="onClickSignOut">登出</button>
</template>
```
