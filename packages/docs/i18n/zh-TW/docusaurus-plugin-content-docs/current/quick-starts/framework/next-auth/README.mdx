# 為你的 Auth.js (Next Auth) 應用程式新增驗證 (Authentication)

本指南將向你展示如何將 Logto 整合到你的 Next.js 應用程式中，使用 [Auth.js](https://authjs.dev/)，之前稱為 Next Auth。

<GuideTip />

## 先決條件 \{#prerequisites}

- 一個 [Logto Cloud](https://cloud.logto.io) 帳戶或 [自行架設的 Logto](/introduction/set-up-logto-oss)。
- 已建立的 Logto 傳統應用程式。
- 一個包含 Auth.js 的 Next.js 專案，請參閱 [Auth.js 文件](https://authjs.dev/getting-started/installation)。
