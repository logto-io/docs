### 將存取權杖 (Access token) 附加到請求標頭 \{#attach-access-token-to-request-headers}

將權杖放入 HTTP 標頭的 `Authorization` 欄位，使用 Bearer 格式（`Bearer YOUR_TOKEN`），即可完成。

:::note
Bearer 權杖的整合流程可能會根據你使用的框架或請求者而有所不同。選擇適合你的方式來應用請求的 `Authorization` 標頭。
:::
