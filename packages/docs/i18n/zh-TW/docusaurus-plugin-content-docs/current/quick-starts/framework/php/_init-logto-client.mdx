首先，建立一個 Logto 配置：

```php title="index.php"
use Logto\Sdk\LogtoClient;
use Logto\Sdk\LogtoConfig;

$client = new LogtoClient(
  new LogtoConfig(
    endpoint: "https://you-logto-endpoint.app",
    appId: "replace-with-your-app-id",
    appSecret: "replace-with-your-app-secret",
  ),
);
```

預設情況下，SDK 使用內建的 PHP session 來儲存 Logto 資料。如果你想使用其他儲存方式，可以將自訂的儲存物件作為第二個參數傳入：

```php title="index.php"
$client = new LogtoClient(
  new LogtoConfig(
    // ...
  ),
  new YourCustomStorage(),
);
```

詳情請參閱 [Storage](https://github.com/logto-io/php/blob/master/docs/api/classes/Logto/Sdk/Storage/Storage.md)。
