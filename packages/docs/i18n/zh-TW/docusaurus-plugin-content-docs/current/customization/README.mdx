# 自訂化

透過自訂品牌、使用者介面、語言和網域設定，讓 Logto 成為你的專屬平台。

## 預建 UI \{#prebuilt-ui}

從我們的預建 UI 元件開始，自訂基本品牌元素。你可以創建：

- 整個平台統一的驗證 (Authentication) 體驗。
- 特定應用程式或組織專屬的登入頁面。

<DocCardList
  items={[
    {
      type: 'link',
      label: '符合你的品牌',
      href: '/customization/match-your-brand',
      description: '了解如何自訂基本品牌元素，以提供即用型的登入體驗。',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## 內容與語言 \{#content--languages}

Logto 預設支援 15 種語言。擴展語言支援或修改現有翻譯以滿足你的需求。

## 自訂網域 \{#custom-domain}

透過使用你的網域作為驗證 (Authentication) URL，強化品牌形象。將預設的 Logto 網域替換為你的品牌網域。

<DocCardList
  items={[
    {
      type: 'link',
      label: '自訂網域',
      href: '/logto-cloud/custom-domain',
      description: '使用你自己的網域作為登入或註冊 URL，以增強品牌和使用者信任。',
      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>

## 即時預覽 \{#live-preview}

使用我們的即時預覽功能，試驗不同的驗證 (Authentication) 流程（[註冊](/end-user-flows/sign-up-and-sign-in/sign-up)、[登入](/end-user-flows/sign-up-and-sign-in/sign-in)、[社交登入](/end-user-flows/sign-up-and-sign-in/social-sign-in)、[忘記密碼](/end-user-flows/sign-up-and-sign-in/reset-password)、[MFA](/end-user-flows/mfa)、[SSO](/end-user-flows/enterprise-sso) 等）。透過我們內建的示範應用程式，即時測試你的登入體驗。
