# 使用者管理

使用者管理是 Logto 的一個重要關注領域。你可以使用 Logto Console 或 Management API 存取並管理你的使用者資料，這兩者都非常有效。透過這些工具，你可以執行以下任務：

- 根據使用者名稱、ID、全名、電話號碼或電子郵件搜尋使用者。
- 對使用者執行多項操作，例如解除連結社交帳戶、更新個人資料資訊、刪除使用者或重設密碼。
- 檢查使用者日誌。
- 指派角色 (Roles) 以進行存取控制。
- 建立和管理個人存取權杖。
