# 租戶設定

## 租戶區域 \{#tenant-region}

當你創建租戶時，可以選擇租戶資料儲存的區域。租戶創建後，區域將無法更改。以下是可用的區域：

- 歐盟（荷蘭）
- 美國（美國西部）
- 澳洲（澳大利亞）

通常，你應選擇最接近客戶的區域，以最大限度地減少延遲並提高性能。

Logto 利用全球邊緣網路為你的應用程式提供最佳性能和可用性。請求路由經過優化，以確保你的使用者始終連接到性能最佳的選項。

:::note
尋找其他區域？請[聯繫我們](https://logto.io/contact)。
:::

## 租戶類型：開發 vs. 生產 \{#tenant-types-dev-vs-prod}

在 Logto Cloud 中有兩種類型的租戶：開發和生產。通過這種租戶區分，你可以更好地管理不同環境中的專案，提高效率，同時享受 Logto 的全部價值。

請注意，你只能在創建時指定租戶類型；租戶類型一旦創建後便無法編輯。

## 開發 \{#development}

開發租戶（dev 租戶）主要用於測試目的，不應在生產環境中使用。這些租戶允許訪問付費計劃中的高級和付費功能，免費且無需訂閱。

然而，開發租戶有一些限制：

- 開發租戶將自動刪除超過 90 天的使用者和組織。
- 在登入體驗中會出現一個橫幅，指示租戶處於開發模式。
- 開發租戶可能對特定功能有配額限制。這些限制在功能詳細頁面中說明（如適用）。
- Logto 可能會更新開發租戶的配額限制，我們將盡力提前通知你。

## 生產 \{#production}

生產租戶是終端使用者訪問實時應用程式的地方，你可能需要[付費訂閱](https://logto.io/pricing)。你可以訂閱免費計劃或專業計劃來創建生產租戶。如果你訂閱免費計劃，則最多只能創建 10 個租戶。
