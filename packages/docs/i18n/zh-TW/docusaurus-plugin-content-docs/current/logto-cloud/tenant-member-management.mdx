# 租戶成員管理

## 邀請協作者 \{#invite-collaborators}

前往 <CloudLink to="/tenant-settings/members">控制台 > 租戶設定 > 成員</CloudLink>。你將看到當前的成員，並可以邀請更多成員。輸入電子郵件地址並分配角色（可以批量操作），然後發送邀請。

Logto 允許你在 Logto Cloud 中以兩種不同的角色邀請團隊成員加入你的租戶：

- **管理員 (Admin)：** 完全存取和管理所有租戶資源。
- **協作者 (Collaborator)：** 可以管理與身分相關的資源，但有以下限制：
  - 無法存取帳單資訊
  - 無法邀請或管理其他成員
  - 無法更改租戶名稱
  - 無法刪除租戶

### 成員限制 \{#members-limit}

| 租戶類型     | 成員限制                                 |
| ------------ | ---------------------------------------- |
| 開發租戶     | 最多 3 名成員（如需更多，請聯繫我們）    |
| 免費生產租戶 | 1 名成員（僅限管理員）                   |
| 專業生產租戶 | 起始 3 名成員，之後每增加一名成員收費 $8 |

### 管理邀請 \{#manage-invitation}

邀請標籤顯示過去的邀請及其狀態

| 狀態   | 說明                     | 可用操作 |
| ------ | ------------------------ | -------- |
| 待處理 | 邀請已發送，但尚未接受。 | 重新發送 |
| 已接受 | 使用者現在是租戶成員。   | 刪除     |
| 已過期 | 邀請超過 7 天限制。      | 刪除     |
| 已撤銷 | 管理員取消了邀請。       | 刪除     |

你不能邀請已經有待處理邀請或已是成員的人。

### 管理租戶成員（僅限管理員） \{#manage-tenant-members-admin-only}

作為管理員，你有多種方式管理你的團隊：

- **邀請或移除成員：** 根據需要新增團隊成員或移除存取權。
- **更改成員角色：** 根據所需的存取級別分配「管理員」或「協作者」角色。
- **轉移租戶所有權：** 如有需要，你可以將租戶的所有權轉移給其他成員。你可以先將管理員角色給予其他成員，然後選擇繼續作為普通成員或完全離開租戶。
- **新增多位管理員：** 與其他團隊成員共享管理責任，以便協作管理。
