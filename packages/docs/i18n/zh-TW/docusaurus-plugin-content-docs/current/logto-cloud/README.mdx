# Logto 雲端服務

[Logto Cloud](https://cloud.logto.io) 提供一系列管理和運營服務，幫助你順利且輕鬆地管理身分和資源。由 Logto 託管，包含 [多區域支援](/logto-cloud/tenant-settings#tenant-region)、租戶管理、[計費與定價](/logto-cloud/billing-and-pricing)、[成員管理](/logto-cloud/tenant-member-management) 和控制台角色型存取控制等功能。

如果你對雲端服務有任何疑問並需要額外支援，請聯繫我們。
