## 使用条款

如果启用，用户在登录或注册前将必须接受使用条款。

输入使用条款的 URL，在需要时 Logto 将会通过 `<iframe>` 加载。

## 语言

### 自动模式

在默认状态下，Logto 使用 _自动_ 模式来动态检测环境并变更语言。如果 Logto 没有找到合适的语言包，将回退至备选语言：英文。

你可以通过点按下拉框来变更备选语言。

### 固定模式

如果你希望固定显示一种语言，则需使用 _固定_ 模式。在该模式下，不管环境设置成何种语言，Logto 都会显示你指定的语言。
