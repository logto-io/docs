## 深色模式

_启用深色模式_ 决定了当 Logto 检测到相关系统偏好时，是否让 UI 有一种「深色的感觉」。

## 品牌色

_品牌色_ 将影响组件的主要颜色，包括但不限于按钮、链接等。试试将 Logto 蓝紫色替换成你的品牌色。

如果启用了深色模式，你将看到一个名为 _品牌色（深色）_ 的额外输入框。
