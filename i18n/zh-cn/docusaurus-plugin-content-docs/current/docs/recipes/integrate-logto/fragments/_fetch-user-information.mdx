Logto SDK 可助你从 OIDC 的 [UserInfo Endpoint](https://openid.net/specs/openid-connect-core-1_0.html#UserInfo) 获取用户信息。

<p>
  你可以在登录后调用 <code>{props.apiCodeFragment}</code> 获得用户的详细信息。
</p>

Logto 返回的用户信息内容将因你在初始化 `LogtoClient` 时 `LogtoConfig` 配置里面的 scope 组合差异而有所不同，下表列出了用户信息内容与 scope 之间的关系：

| 字段名                | 类型       | 所需 Scope    | 备注                                |
| --------------------- | ---------- | ------------- | ----------------------------------- |
| sub                   | `string`   | `openid`      | `openid` 已由 Logto SDK 默认添加。  |
| name                  | `string`   | `profile`     | `profile` 已由 Logto SDK 默认添加。 |
| username              | `string`   | `profile`     | `profile` 已由 Logto SDK 默认添加。 |
| picture               | `string`   | `profile`     | `profile` 已由 Logto SDK 默认添加。 |
| role_names            | `string[]` | `profile`     | `profile` 已由 Logto SDK 默认添加。 |
| email                 | `string`   | `email`       |                                     |
| email_verified        | `boolean`  | `email`       |                                     |
| phone_number          | `string`   | `phone`       |                                     |
| phone_number_verified | `boolean`  | `phone`       |                                     |
| custom_data           | `object`   | `custom_data` |                                     |
| identities            | `object`   | `identities`  |                                     |
