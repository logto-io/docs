# 搭建连接器指南

对连接器的文件结构有了一个初步概念之后，让我们讨论一下开发、实现一个连接器的思路。

## 在连接器对接的服务商网站注册一个账号

仔细浏览服务商的网站和相应的管理页面，总是我们开始实现一个连接器的第一步。在这里你可以找到关于连接器服务提供商的一些基本信息。

## 查找讲解连接器要对接的服务的功能和工作流程的文档

由于不同类型的连接器对应着不同的使用场景，我们在搜索相关的文档的时候也需要注意用不同的关键词。

_社交连接器_ 是用于引导终端用户授权登录的；而 _短信连接器_ 和 _邮件连接器_ 适用于调用发送短信或邮件服务 API 的。

:::tip
_社交连接器_ 文档关键词：授权流程，获取用户档案以及 OAuth 2.0 协议（大部分 _社交连接器_ 对接的服务商都遵从此协议）。<br/>
_短信连接器_ 和 _邮件连接器_ 文档关键词：集成发送短信（或邮件）API。
:::
