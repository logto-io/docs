# 設定 LinkedIn 社交登入

Logto 官方的 LinkedIn 社交登入連接器。

## 開始使用 \{#get-started}

LinkedIn 連接器讓終端使用者可以透過 LinkedIn OAuth 2.0 驗證協議，使用他們自己的 LinkedIn 帳號登入你的應用程式。
