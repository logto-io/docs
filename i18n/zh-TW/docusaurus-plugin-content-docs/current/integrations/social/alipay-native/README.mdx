# 設定 Alipay (原生) 社交登入

官方 Logto 連接器，適用於行動裝置原生應用的 Alipay 社交登入。

<GuideTip />

## 開始使用 \{#get-started}

Alipay 原生連接器與 Logto SDK 在行動平台上緊密配合。它利用 Alipay 的 OAuth 2.0 驗證 (Authentication) 流程，讓 Alipay 使用者能以公開的 Alipay 使用者資料登入其他 App，無需繁瑣的註冊流程。

<Integration />

### 測試 Alipay 原生連接器 \{#test-alipay-native-connector}

就是這麼簡單。別忘了<MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">在登入體驗中啟用社交連接器</MainSiteUrl>。

啟用 Alipay 原生連接器後，你可以建置並執行你的應用程式來測試是否運作正常。
