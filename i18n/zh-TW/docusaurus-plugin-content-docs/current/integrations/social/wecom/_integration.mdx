## 開始使用 \{#get-started}

使用管理員帳號登入 [WeCom WebUI](https://work.weixin.qq.com/)，或從 WeCom 應用程式點擊 **管理企業 (Manage the enterprise/管理企业)**。

在「應用管理 (App Management)」分頁中，向下捲動頁面並點擊「建立應用 (Create an app)」。

根據你的應用程式填寫相關資訊，然後建立。

現在你已取得 Agent ID（**不是** APPID）以及 Secret。

### 網站資訊 \{#website-info}

在此頁面設定你需要的項目，例如：

- 允許使用者：_誰可以看到這個應用程式_
- 應用首頁：_你的應用程式首頁。例如：`logto.io/demo-app`_

**重要說明**  
此頁面有三個關於「開發者 API (Developer API/开发者接口)」的項目：

1. 網頁授權與 JS-SDK (Web Authorization and JS-SDK)；
2. 透過 WeCom 授權登入 (Log in to via authorization by WeCom)；
3. 企業可信 IP (Enterprise Trusted IP)；

請依照 WeCom 指引填寫。

### Corp ID \{#corp-id}

如果你熟悉 WeChat 開發，會發現 Corp ID 的用途與 APP ID 相同。

你可以在「我的企業 (My Enterprise/我的企业)」分頁頁面底部找到 Corp ID，格式類似 **ww\*\*\*\*** 。

## 設定連接器 \{#configure-the-connector}

現在我們已經有 Agent ID、Secret 和 Corp ID。

讓我們完成連接器的表單。

`Scope` 欄位可以留空（非必填），也可以填入 `snsapi_base` 或 `snsapi_privateinfo`。

**儲存並完成**
