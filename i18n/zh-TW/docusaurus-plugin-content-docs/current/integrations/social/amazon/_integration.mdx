### 設定 Amazon 應用程式 \{#setup-a-amazon-app}

前往 [Amazon Developer Portal](https://developer.amazon.com) 並使用你的帳戶登入。如果你還沒有帳戶，可以註冊一個。

然後，創建一個應用程式。

**步驟 1：** 在首頁找到 "Login with Amazon" 區塊。

或者你可以訪問[此頁面](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html)。

按照指南創建一個新的安全設定檔。

創建安全設定檔後，你將從表格中獲得 Client ID 和 Client Secret。

**步驟 2：** 設定回呼 URL。

點擊表格右側的 "Manage" 按鈕，然後進入 "Web Settings" 標籤。

在 "Allowed redirect URLs" 欄位中，新增以下 URL：

- `${your_logto_origin}/callback/${connector_id}`。

`connector_id` 可以在 Logto Admin Console 連接器詳細資訊頁面的頂部欄位找到。

### 配置你的連接器 \{#configure-your-connector}

在你的 Logto 連接器配置中，使用上一步獲得的值填寫以下欄位：

- **clientId：** 你的應用程式的 Client ID。
- **clientSecret：** 你的應用程式的 Client Secret。

`scope` 是一個以空格分隔的 OIDC 權限範圍列表。如果未提供，預設權限範圍為 `openid profile`。

### 配置類型 \{#config-types}

| 名稱         | 類型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
