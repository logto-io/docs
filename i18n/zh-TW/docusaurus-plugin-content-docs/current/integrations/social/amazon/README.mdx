# 設定 Amazon 社交登入

Logto 的官方 Amazon 社交登入連接器。

<GuideTip />

## 開始使用 \{#get-started}

Amazon 連接器使終端使用者能夠透過 Amazon OAuth 2.0 驗證協議，使用其 Amazon 帳戶登入你的應用程式。
