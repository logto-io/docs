# 設定支付寶 (Web) 社交登入

Logto 官方的支付寶社交登入連接器，適用於網頁應用程式。

<GuideTip />

## 開始使用 \{#get-started}

支付寶 Web 連接器專為桌面網頁應用程式設計。它利用支付寶的 OAuth 2.0 驗證 (Authentication) 流程，讓支付寶用戶可以直接使用公開支付寶用戶資料登入其他應用程式，無需繁瑣的註冊流程。

<Integration />

## 測試支付寶 Web 連接器 \{#test-alipay-web-connector}

就是這麼簡單。別忘了<MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">在登入體驗中啟用社交連接器</MainSiteUrl>。

啟用支付寶 Web 連接器後，你可以建置並執行你的網頁應用程式來測試是否運作正常。
