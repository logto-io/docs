# 設定 Apple 社交登入

Apple 社交登入的官方 Logto 連接器 (Connector)。

<GuideTip />

## 開始使用 \{#get-started}

如果你還不瞭解連接器 (Connector) 的概念，或不知道如何將此連接器新增到「登入與帳號 > 註冊與登入」中，請參閱 [Logto 教學](/connectors/social-connectors)。

<Integration />

### 設定權限範圍 (Scope) 的注意事項 \{#pitfalls-of-configuring-scope}

如果你在應用程式已經讓使用者用 Apple 登入後，才設定要求使用者的電子郵件地址，則即使這些現有使用者再次使用 Apple ID 登入，你也無法取得他們的電子郵件地址。為解決此問題，你需要指示使用者前往 [Apple ID 帳號管理頁面](https://appleid.apple.com/account/manage)，並在「使用 Apple 登入」區塊移除你的應用程式。這可以在你的應用程式詳細頁面選擇「停止使用 Apple 登入」來完成。

舉例來說，如果你的應用程式同時要求使用者的電子郵件與姓名（`email name` 權限範圍），新使用者首次登入時看到的使用者授權頁面 (Consent screen) 應如下圖所示：
