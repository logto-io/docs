## 在微信開放平台創建網頁應用 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

如果你已經完成某些部分，可以跳過這些部分。

:::

### 創建帳戶 \{#create-an-account}

打開 https://open.weixin.qq.com/，點擊右上角的「註冊」按鈕，然後完成註冊流程。

### 創建網頁應用 \{#create-a-web-app}

使用你剛創建的帳戶登入。在「網頁應用」(网页应用) 標籤中，點擊大綠色按鈕「創建網頁應用」(创建网页应用)。

<img src="/img/assets/app-tabs.png" alt="應用標籤" />

讓我們在申請表中填寫所需的資訊。

<img src="/img/assets/create-web-app.png" alt="創建網頁應用" />

#### 基本資訊 \{#basic-info}

大多數都很簡單明瞭。完成表單後，點擊「下一步」繼續。

#### 網站資訊 \{#website-info}

在「授權回調域」(授权回调域) 中填寫你的 Logto 網域。例如，`logto.io`。

#### 等待審核結果 \{#waiting-for-the-review-result}

完成網站資訊後，點擊「提交審核」繼續。通常審核速度很快，會在 1-2 天內完成。

我們懷疑每次提交的審核員是隨機分配的，因為標準不固定。你可能第一次會被拒絕，但不要放棄！說明你的現狀並詢問審核員如何修改。

### 配置你的微信連接器 \{#configure-your-wechat-connector}

在 `clientId` 和 `clientSecret` 欄位中填入你從 OAuth 應用詳細頁面獲得的 _Client ID_ 和 _Client Secret_。

在 `scope` 欄位中填入 'snsapi_userinfo' 或 'snsapi_base'。你可以將此欄位留空，因為它是可選的。預設值為 'snsapi_userinfo'。你可以查看不同值之間的[差異](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)。
