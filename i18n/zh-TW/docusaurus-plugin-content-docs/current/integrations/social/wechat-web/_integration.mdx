## 在微信開放平台建立網頁應用程式 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

如果你已經完成部分步驟，可以略過相關章節。

:::

### 建立帳號 \{#create-an-account}

打開 https://open.weixin.qq.com/，點擊右上角的「註冊」按鈕，然後完成註冊流程。

### 建立網頁應用程式 \{#create-a-web-app}

使用剛剛建立的帳號登入。在「網頁應用」(网页应用) 分頁中，點擊綠色大按鈕「建立網頁應用」(创建网页应用)。

<img src="/img/assets/app-tabs.png" alt="應用分頁 (App tabs)" />

接下來填寫應用程式表單中的必要資訊。

<img src="/img/assets/create-web-app.png" alt="建立網頁應用 (Create web app)" />

#### 基本資訊 \{#basic-info}

大多數欄位都很直觀。填寫完畢後，點擊「下一步」繼續。

#### 網站資訊 \{#website-info}

在「授權回調域」(授权回调域) 欄位填入你的 Logto 網域。例如：`logto.io`。

#### 等待審核結果 \{#waiting-for-the-review-result}

完成網站資訊後，點擊「提交審核」繼續。通常審核速度很快，1-2 天內會有結果。

我們推測每次提交時審核人員是隨機分配的，因此標準會有所浮動。你可能第一次會被拒絕，但不要放棄！說明你的現況並詢問審核人員如何修改。

### 設定你的 WeChat 連接器 (Connector) \{#configure-your-wechat-connector}

將 _Client ID_ 和 _Client Secret_ 分別填入 `clientId` 和 `clientSecret` 欄位，這些資訊可在 OAuth 應用程式詳情頁面取得。

`scope` 欄位可填入 'snsapi_userinfo' 或 'snsapi_base'。這個欄位是「選填」的，可以留空。預設值為 'snsapi_userinfo'。你可以參考[不同值的差異](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)。
