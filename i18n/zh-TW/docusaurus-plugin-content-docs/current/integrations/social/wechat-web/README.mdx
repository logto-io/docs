# 設定 WeChat（Web）社交登入

Logto 官方提供的 WeChat 社交登入 Web 應用程式連接器。

<GuideTip />

## 開始使用 \{#get-started}

如果你還不了解連接器（Connector）的概念，或不知道如何將此連接器新增至你的登入體驗（Sign-in experience），請參閱 [Logto 教學](/connectors/social-connectors)。

:::caution

此連接器僅適用於 Web 應用程式。如果你想在原生應用程式中實現 WeChat 登入，請參閱 [WeChat 原生連接器](/integrations/wechat-native)。

:::

<Integration />

### 測試 WeChat Web 連接器 \{#test-wechat-web-connector}

就是這麼簡單。別忘了 [在登入體驗中啟用社交連接器](/connectors/social-connectors/#enable-social-sign-in)。

啟用 WeChat Web 連接器後，你可以再次登入你的應用程式來確認是否運作正常。
