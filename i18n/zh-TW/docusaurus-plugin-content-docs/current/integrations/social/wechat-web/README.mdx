# 設定 WeChat 社交登入 (Web)

Logto 官方的 WeChat 社交登入連接器，適用於網頁應用程式。

<GuideTip />

## 開始使用 \{#get-started}

如果你不瞭解連接器的概念或不知道如何將此連接器新增到你的登入體驗中，請參閱 [Logto 教學](/connectors/social-connectors)。

:::caution

此連接器僅適用於網頁應用程式。如果你正在尋找在原生應用程式中使用 WeChat 登入的方法，請參閱 [WeChat 原生連接器](/integrations/wechat-native)。

:::

## 在 WeChat 開放平台創建網頁應用程式 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

如果你已經完成某些部分，可以跳過。

:::

### 創建帳戶 \{#create-an-account}

打開 https://open.weixin.qq.com/，點擊右上角的「註冊」按鈕，然後完成註冊流程。

### 創建網頁應用程式 \{#create-a-web-app}

使用你剛創建的帳戶登入。在「網頁應用」(网页应用) 標籤中，點擊大綠色按鈕「創建網頁應用」(创建网页应用)。

<img src="/img/assets/app-tabs.png" alt="App tabs" />

在應用程式表單中填寫所需資訊。

<img src="/img/assets/create-web-app.png" alt="Create web app" />

#### 基本資訊 \{#basic-info}

大多數都很簡單明瞭。完成表單後，點擊「下一步」繼續。

#### 網站資訊 \{#website-info}

在「授權回調域」(授权回调域) 中填寫你的 Logto 網域。例如，`logto.io`。

#### 等待審核結果 \{#waiting-for-the-review-result}

完成網站資訊後，點擊「提交審核」繼續。通常審核速度很快，會在 1-2 天內完成。

我們懷疑每次提交的審核員是隨機分配的，因為標準浮動不定。你可能第一次被拒絕，但不要放棄！說明你的現狀並詢問審核員如何修改。

### 配置你的 WeChat 連接器 \{#configure-your-wechat-connector}

在 `clientId` 和 `clientSecret` 欄位中填入你從 OAuth 應用程式詳細頁面獲得的 _Client ID_ 和 _Client Secret_。

在 `scope` 欄位中填入 'snsapi_userinfo' 或 'snsapi_base'。此欄位為選填，可以留空。預設值為 'snsapi_userinfo'。你可以查看不同值之間的[差異](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)。

### 測試 WeChat 網頁連接器 \{#test-wechat-web-connector}

就是這樣。別忘了在 [登入體驗中啟用社交連接器](/connectors/social-connectors/#enable-social-sign-in)。

一旦啟用了 WeChat 網頁連接器，你可以再次登入你的應用程式以查看是否正常運作。
