# 設定 WeChat 社交登入（Web）

Logto 官方提供的 WeChat 社交登入連接器，適用於網頁應用程式。

<GuideTip />

## 開始使用 \{#get-started}

如果你還不清楚連接器（Connector）的概念，或不知道如何將此連接器新增到你的登入體驗（Sign-in experience），請參閱 [Logto 教學](/connectors/social-connectors)。

:::caution

此連接器僅適用於網頁應用程式。如果你想在原生應用程式中實現 WeChat 登入，請參閱 [WeChat 原生連接器](/integrations/wechat-native)。

:::

## 在 WeChat 開放平台建立網頁應用程式 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

如果你已經完成部分步驟，可以略過相關章節。

:::

### 建立帳號 \{#create-an-account}

前往 https://open.weixin.qq.com/，點擊右上角的「註冊」按鈕，並完成註冊流程。

### 建立網頁應用程式 \{#create-a-web-app}

使用剛剛建立的帳號登入。在「網頁應用」分頁（网页应用）中，點擊綠色大按鈕「建立網頁應用」（创建网页应用）。

<img src="/img/assets/app-tabs.png" alt="App tabs" />

接下來填寫應用程式申請表中的必要資訊。

<img src="/img/assets/create-web-app.png" alt="Create web app" />

#### 基本資訊 \{#basic-info}

大多數欄位都很直觀。填寫完畢後，點擊「下一步」繼續。

#### 網站資訊 \{#website-info}

在「授權回調域」（授权回调域）欄位填入你的 Logto 網域。例如：`logto.io`。

#### 等待審核結果 \{#waiting-for-the-review-result}

完成網站資訊後，點擊「提交審核」繼續。通常審核速度很快，1-2 天內會有結果。

我們推測每次送審時審核人員是隨機分配的，因此標準會有所浮動。你可能第一次會被拒絕，但別灰心！如實說明你的現況，並詢問審核人員如何修改。

### 設定你的 WeChat 連接器 \{#configure-your-wechat-connector}

將 _Client ID_ 與 _Client Secret_ 分別填入 `clientId` 與 `clientSecret` 欄位，這些資訊可在 OAuth 應用程式詳情頁取得。

`scope` 欄位可填入 'snsapi_userinfo' 或 'snsapi_base'。此欄位為「選填」，可留空，預設值為 'snsapi_userinfo'。你可以參考[官方文件](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)了解不同值的差異。

### 測試 WeChat Web 連接器 \{#test-wechat-web-connector}

就這樣！別忘了 [在註冊與登入流程中啟用社交連接器](/connectors/social-connectors/#enable-social-sign-in)。

啟用 WeChat Web 連接器後，你可以再次登入你的應用程式，確認是否運作正常。
