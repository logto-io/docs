# 設定 WeChat（原生）社交登入

Logto 官方提供的 WeChat 社交登入連接器，適用於原生應用程式（iOS 與 Android）。

<GuideTip />

## 開始使用 \{#get-started}

如果你還不了解連接器的概念，或是不知道如何將此連接器新增至你的登入體驗 (Sign-in experience)，請參閱 [Logto 教學](/connectors/social-connectors)。

在原生應用程式中，無法使用網頁作為 WeChat 的登入方式：必須導向至 WeChat 應用程式，並且需要使用他們的 SDK。

我們知道這聽起來有點複雜，但別擔心，本教學將帶你輕鬆完成。

<Integration />

## 測試 WeChat 原生連接器 \{#test-wechat-native-connector}

就是這麼簡單。別忘了<MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">在登入體驗中啟用社交連接器</MainSiteUrl>。

啟用 WeChat 原生連接器後，你可以建置並執行你的應用程式，檢查是否運作正常。

:::caution

WeChat 並未針對未安裝 WeChat 應用程式的裝置提供方案。若遇此情況，Logto 會在登入時隱藏此連接器（這也是 [官方開發指南](https://developers.weixin.qq.com/doc/oplatform/en/Mobile_App/WeChat_Login/Development_Guide.html) 建議的做法）。

:::
