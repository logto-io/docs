## 測試 Mailgun 電子郵件連接器 \{#test-mailgun-email-connector}

你可以輸入一個電子郵件地址並點擊「發送」，在「儲存並完成」之前測試設定是否可用。

就是這麼簡單。別忘了 <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">在登入體驗中啟用連接器</MainSiteUrl>。
