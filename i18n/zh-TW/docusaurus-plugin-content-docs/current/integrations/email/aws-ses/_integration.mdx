## 在 AWS 服務主控台中配置郵件服務 \{#configure-a-mail-service-in-the-aws-service-console}

### 註冊 AWS 帳戶 \{#register-aws-account}

前往 [AWS](https://aws.amazon.com/) 並註冊一個帳戶。

### 建立身分 (Identity) \{#create-a-identity}

- 前往 `Amazon Simple Email Service` 主控台
- 建立一個身分，選擇以下選項之一
  - 建立一個網域
  - 建立一個電子郵件地址

### 連接器的配置 \{#configuration-of-the-connector}

1. 點擊 Amazon 主控台右上角的用戶名進入 `Security Credentials`。如果沒有，請建立一個 `AccessKey` 並妥善保存。
2. 完成 `Amazon Simple Email Service` 連接器的設置：
   - 使用在步驟 1 中獲得的 `AccessKey ID` 和 `AccessKey Secret` 分別填寫 `accessKeyId` 和 `accessKeySecret`。
   - `region`：填寫你用來發送郵件的身分所在的區域。
   - `emailAddress`：你用來發送郵件的電子郵件地址，格式為 `Logto\<noreply@logto.io>` 或 `\<noreply@logto.io>`

以下參數為可選；參數說明可參考 [AWS SES API 文件](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html)。

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### 測試 Amazon SES 連接器 \{#test-the-amazon-ses-connector}

你可以輸入一個電子郵件地址並點擊「Send」來檢查設置是否有效，然後再「Save and Done」。

就是這樣。別忘了 <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">在登入體驗中啟用連接器</MainSiteUrl>。

### 配置類型 \{#configure-types}

| 名稱                                      | 類型              |
| ----------------------------------------- | ----------------- |
| accessKeyId                               | string            |
| accessKeySecret                           | string            |
| region                                    | string            |
| emailAddress                              | string (OPTIONAL) |
| emailAddressIdentityArn                   | string (OPTIONAL) |
| feedbackForwardingEmailAddress            | string (OPTIONAL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPTIONAL) |
| configurationSetName                      | string (OPTIONAL) |
| templates                                 | Template[]        |

| 模板屬性  | 類型        | 枚舉值                                                  |
| --------- | ----------- | ------------------------------------------------------- |
| subject   | string      | N/A                                                     |
| content   | string      | N/A                                                     |
| usageType | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
