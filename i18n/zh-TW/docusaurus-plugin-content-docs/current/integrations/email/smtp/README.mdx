## 開始使用 \{#get-started}

SMTP（簡單郵件傳輸協定，Simple Mail Transfer Protocol）是一種用於電子郵件傳輸的網際網路標準通訊協定。郵件伺服器與其他訊息傳遞代理會使用 SMTP 來傳送與接收訊息。
