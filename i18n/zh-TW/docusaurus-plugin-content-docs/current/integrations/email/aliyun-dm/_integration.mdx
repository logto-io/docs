## 在阿里雲 DirectMail 控制台中設定電子郵件服務 \{#set-up-an-email-service-in-aliyun-directmail-console}

### 建立阿里雲帳戶 \{#create-an-aliyun-account}

前往 [阿里雲](https://aliyun.com/) 並建立你的阿里雲帳戶，如果你還沒有的話。

### 啟用並配置阿里雲 Direct Mail \{#enable-and-configure-aliyun-direct-mail}

進入 [DM 服務控制台頁面](https://www.aliyun.com/product/directmail) 並登入。點擊頁面左上角的「申請開通」按鈕啟用 Direct Mail 服務，然後開始配置流程。

從 [DM 管理控制台頁面](https://dm.console.aliyun.com/) 開始，你應該：

1. 從側邊欄進入「發信域名」，按照指示新增「新建域名」。
2. 分別自訂「發信地址」和「郵件標籤」。

完成設定後，有兩種不同的測試方式：

- 前往 [DirectMail 概覽頁面](https://dm.console.aliyun.com/)，在頁面底部找到「操作引導」，然後點擊「發送郵件」。你會找到所有不同類型的測試方法。
- 按照側邊欄的路徑「發送郵件」->「發送郵件」來建立測試任務。

## 編寫連接器 JSON \{#compose-the-connector-json}

1. 從 [DM 管理控制台頁面](https://dm.console.aliyun.com/)，將滑鼠懸停在右上角的頭像上，進入「AccessKey 管理」，然後點擊「創建 AccessKey」。完成安全驗證後，你將獲得一對「AccessKey ID」和「AccessKey Secret」。請妥善保管。
2. 前往你剛才訪問的 [DM 管理控制台頁面](https://dm.console.aliyun.com/)中的「發信地址」或「郵件標籤」標籤，你可以輕鬆找到 _發信地址_ 或 _郵件標籤_。
3. 填寫阿里雲 DM 連接器設置：
   - 使用你在步驟 1 中獲得的 access key 對填寫 `accessKeyId` 和 `accessKeySecret` 欄位。
   - 使用在步驟 2 中找到的「發信地址」和「郵件標籤」填寫 `accountName` 和 `fromAlias` 欄位。所有模板將共享此簽名名稱。（`fromAlias` 是可選的，可以留空。）
   - 你可以為不同情況添加多個 DM 連接器模板。以下是一個添加單一模板的示例：
     - 填寫 `subject` 欄位，這將作為發送電子郵件的標題。
     - 使用任意字串類型內容填寫 `content` 欄位。不要忘記留 `{{code}}` 佔位符作為隨機驗證碼。
     - 使用 `Register`、`SignIn`、`ForgotPassword` 或 `Generic` 填寫 `usageType` 欄位以適應不同的使用情境。（`usageType` 是 Logto 的屬性，用於識別適當的使用情境。）為了啟用完整的使用者流程，需要具有 `Register`、`SignIn`、`ForgotPassword` 和 `Generic` 的 usageType 的模板。

### 測試阿里雲 DM 連接器 \{#test-aliyun-dm-connector}

你可以輸入一個電子郵件地址並點擊「發送」來查看設置是否能在「保存並完成」之前正常運作。

就是這樣。別忘了 [在登入體驗中啟用連接器](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)。

### 配置類型 \{#config-types}

| 名稱            | 類型          |
| --------------- | ------------- |
| accessKeyId     | string        |
| accessKeySecret | string        |
| accountName     | string        |
| fromAlias       | string (可選) |
| templates       | Template[]    |

| 模板屬性  | 類型        | 枚舉值                                                  |
| --------- | ----------- | ------------------------------------------------------- |
| subject   | string      | N/A                                                     |
| content   | string      | N/A                                                     |
| usageType | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
