# 使用 Aliyun Direct Mail 設定電子郵件驗證 (Set up email verification with Aliyun Direct Mail)

官方 Logto 連接器 (Connector) 支援 Aliyun Direct Mail 服務。

<GuideTip />

## 開始使用 \{#get-started}

阿里雲 (Aliyun) 是亞洲主要的雲端服務供應商，提供多種雲端服務，包括 DM（Direct Mail）。Aliyun DM Connector 是由 Logto 團隊提供的外掛，可用於呼叫阿里雲 DM 服務 API，協助 Logto 終端使用者透過郵件驗證碼（或稱驗證碼）註冊並登入他們的 Logto 帳號。
