# 使用 Postmark 電子郵件服務設定電子郵件驗證

Logto 的 Postmark 電子郵件服務連接器。

<GuideTip />

## 開始使用 \{#get-started}

Postmark 是一個用於交易和行銷電子郵件的郵件平台。我們可以使用其電子郵件發送功能來發送 _驗證碼_。

## 註冊 Postmark 帳戶 \{#register-postmark-account}

在 [Postmark 網站](https://postmark.com/)創建一個新帳戶。如果你已經有帳戶，可以跳過此步驟。

## 配置你的連接器 \{#configure-your-connector}

在 Postmark 中，填寫 `serverToken` 欄位，使用你在伺服器設定中找到的伺服器權杖。

填寫 `fromEmail` 欄位，使用發件人的 _From Address_。

為了啟用完整的使用者流程，需要具有 `Register`、`SignIn`、`ForgotPassword` 和 `Generic` 使用類型的模板。

以下是 Postmark 連接器模板 JSON 的範例。

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## 測試 Postmark 電子郵件連接器 \{#test-postmark-email-connector}

你可以輸入一個電子郵件地址並點擊「發送」來檢查設定是否有效，然後再點擊「儲存並完成」。

就是這樣。別忘了 <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">在登入體驗中啟用連接器</MainSiteUrl>。

## 配置類型 \{#config-types}

| 名稱        | 類型   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| 模板屬性      | 類型        | 枚舉值                                                  |
| ------------- | ----------- | ------------------------------------------------------- |
| usageType     | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias | string      | N/A                                                     |
