## 註冊 Postmark 帳號 \{#register-postmark-account}

請前往 [Postmark 官方網站](https://postmark.com/) 建立新帳號。如果你已經有帳號，可以跳過此步驟。

## 設定你的連接器 (Connector) \{#configure-your-connector}

將你在 Postmark 伺服器設定中找到的 Server Token 填入 `serverToken` 欄位。

將寄件人的 _From Address_ 填入 `fromEmail` 欄位。

為了啟用完整的使用者流程，必須建立 `Register`、`SignIn`、`ForgotPassword` 和 `Generic` 這四種 usageType 的範本 (template)。

以下是 Postmark 連接器範本的 JSON 範例。

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## 測試 Postmark 電子郵件連接器 (Connector) \{#test-postmark-email-connector}

你可以輸入一個電子郵件地址並點擊「Send」來測試設定是否可用，再點選「Save and Done」完成設定。

就是這麼簡單。別忘了 <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">在登入體驗 (Sign-in experience) 中啟用連接器 (Connector)</MainSiteUrl>。

## 設定類型 \{#config-types}

| 名稱        | 類型   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| 範本屬性 (Template Properties) | 類型        | 列舉值 (Enum values)                                 |
| ------------------------------ | ----------- | ---------------------------------------------------- |
| usageType                      | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| templateAlias                  | string      | N/A                                                  |
