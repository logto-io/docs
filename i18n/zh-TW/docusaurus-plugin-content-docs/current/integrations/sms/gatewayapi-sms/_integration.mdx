### 建立 GatewayAPI 帳號 \{#create-an-gatewayapi-account}

前往 [GatewayAPI 官方網站](https://www.gatewayapi.com/)，如果你還沒有 GatewayAPI 帳號，請註冊一個。

### 啟用帳號 \{#enable-account}

在使用簡訊服務前，你可能需要先啟用帳號。你可以聯繫 GatewayAPI 客服以啟用你的帳號。

### 取得 API 權杖 \{#get-api-token}

從 GatewayAPI 控制台進入 API Keys 頁面，找到現有的 API 權杖或建立新的 API 權杖。

## 在 Logto 中設定 \{#set-up-in-logto}

1. **Endpoint**：如果你的 GatewayAPI 帳號屬於歐盟區域，請使用 endpoint https://gatewayapi.com/rest/mtsms。如果你的 GatewayAPI 帳號屬於美國區域，也請使用 endpoint https://gatewayapi.com/rest/mtsms。
2. **API 權杖 (API Token)**：你在前一步建立的 API 權杖。
3. **Sender**：你想用來發送簡訊的發送者名稱。
4. **範本 (Templates)**：你想用來發送簡訊的範本，可以使用預設範本或根據需求進行修改。
