# 使用 GatewayAPI SMS 設定簡訊驗證 (Set up SMS verification with GatewayAPI SMS)

官方 Logto GatewayAPI SMS 連接器 (connector)。

<GuideTip />

## 開始使用 \{#get-started}

GatewayAPI 是歐洲的雲端服務供應商，提供多種雲端服務，包括簡訊（SMS, short message service）。GatewayAPI SMS 連接器 (Connector) 是 Logto 團隊提供的外掛，可用於呼叫 GatewayAPI SMS 服務，協助 Logto 終端使用者透過簡訊驗證碼註冊與登入他們的 Logto 帳號。
