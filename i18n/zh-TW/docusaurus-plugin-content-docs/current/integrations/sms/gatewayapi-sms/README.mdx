# 使用 GatewayAPI SMS 設定簡訊驗證

Logto 的 GatewayAPI SMS 官方連接器。

<GuideTip />

## 開始使用 \{#get-started}

GatewayAPI 是歐洲的一家雲服務提供商，提供多種雲服務，包括簡訊（SMS, Short Message Service）。GatewayAPI SMS 連接器是由 Logto 團隊提供的插件，用於調用 GatewayAPI SMS 服務，藉此 Logto 終端使用者可以通過簡訊驗證碼註冊並登入他們的 Logto 帳戶。

## 在 GatewayAPI 中設定 \{#set-up-in-gatewayapi}

### 建立 GatewayAPI 帳戶 \{#create-an-gatewayapi-account}

前往 [GatewayAPI 網站](https://www.gatewayapi.com/) 並註冊你的 GatewayAPI 帳戶，如果你還沒有的話。

### 啟用帳戶 \{#enable-account}

在使用簡訊服務之前，你可能需要啟用你的帳戶。你可以聯繫 GatewayAPI 客戶服務來啟用你的帳戶。

### 獲取 API 權杖 \{#get-api-token}

從 GatewayAPI 控制台進入 API Keys 頁面，找到 API 權杖或創建一個新的 API 權杖。

## 在 Logto 中設定 \{#set-up-in-logto}

1. **Endpoint**：如果你的 GatewayAPI 帳戶在歐盟地區，你應該使用端點 https://gatewayapi.com/rest/mtsms。如果你的 GatewayAPI 帳戶在美國地區，你也應該使用端點 https://gatewayapi.com/rest/mtsms。
2. **API 權杖**：你在前一步創建的 API 權杖。
3. **Sender**：你想用來發送簡訊的發送者。
4. **Templates**：你想用來發送簡訊的模板，你可以使用預設模板或根據需要進行修改。
