# 使用阿里雲簡訊服務 (Aliyun Short Message Service) 設定簡訊驗證

Logto 官方提供的阿里雲簡訊服務連接器。

<GuideTip />

## 開始使用 \{#get-started}

阿里雲是亞洲主要的雲端服務供應商，提供多種雲端服務，包括簡訊服務（SMS, Short Message Service）。Aliyun SMS Connector 是 Logto 團隊提供的外掛，可用於呼叫阿里雲簡訊服務，讓 Logto 終端使用者能透過簡訊驗證碼註冊並登入他們的 Logto 帳號。
