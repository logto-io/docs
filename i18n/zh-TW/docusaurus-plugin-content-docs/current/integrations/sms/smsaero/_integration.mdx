## 註冊帳號 \{#register-account}

在 [SMSAero](https://smsaero.ru/) 建立新帳號。（如果你已經有帳號，請直接跳到下一步。）

## 取得帳號憑證 \{#get-account-credentials}

我們需要 API 憑證來讓連接器運作。請從 [API 和 SMPP](https://smsaero.ru/cabinet/settings/apikey/) 開始。

複製「API-key」或產生新的。

## 編寫連接器 JSON \{#compose-the-connector-json}

請將 _email_、_apiKey_ 與 _senderName_ 欄位分別填入你的電子郵件、API 金鑰與發送者名稱。

你可以將發送者名稱填為「SMSAero」，以使用 SMSAero 預設的發送者名稱。

你可以針對不同情境新增多個 SMS 連接器範本。以下為新增單一範本的範例：

- 將 `content` 欄位填入任意字串內容。別忘了保留 `{{code}}` 佔位符以插入隨機驗證碼。
- 將 `usageType` 欄位填為 `Register`、`SignIn`、`ForgotPassword`、`Generic` 之一，以對應不同用途。
  若要啟用完整的使用者流程，需同時提供 `Register`、`SignIn`、`ForgotPassword` 與 `Generic` 這四種 usageType 的範本。

### 測試 SMSAero 連接器 \{#test-smsaero-connector}

你可以輸入手機號碼並點擊「發送 (Send)」，在「儲存並完成 (Save and Done)」前測試設定是否可用。

就是這麼簡單。別忘了 <MainSiteUrl href="/connectors/sms-connectors/#enable-phone-number-sign-up-or-sign-in">在登入體驗中啟用連接器</MainSiteUrl>。

### 設定類型 \{#config-types}

| 名稱       | 類型        |
| ---------- | ----------- |
| email      | string      |
| apiKey     | string      |
| senderName | string      |
| templates  | Templates[] |

| 範本屬性  | 類型        | 列舉值                                               |
| --------- | ----------- | ---------------------------------------------------- |
| content   | string      | N/A                                                  |
| usageType | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
