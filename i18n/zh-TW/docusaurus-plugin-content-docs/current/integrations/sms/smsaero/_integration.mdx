## 註冊帳戶 \{#register-account}

在 [SMSAero](https://smsaero.ru/) 上創建一個新帳戶。（如果你已經有帳戶，請跳到下一步。）

## 獲取帳戶憑證 \{#get-account-credentials}

我們需要 API 憑證來使連接器運作。從 [API 和 SMPP](https://smsaero.ru/cabinet/settings/apikey/) 開始。

複製 "API-key" 或生成新的。

## 編寫連接器 JSON \{#compose-the-connector-json}

用你的電子郵件、API key 和發送者名稱填寫 _email_、_apiKey_ 和 _senderName_ 欄位。

你可以用 "SMSAero" 填寫發送者名稱，以使用 SMSAero 提供的預設發送者名稱。

你可以為不同情況添加多個 SMS 連接器模板。以下是添加單個模板的範例：

- 用任意字串類型的內容填寫 `content` 欄位。不要忘記保留 `{{code}}` 佔位符以放置隨機驗證碼。
- 用 `Register`、`SignIn`、`ForgotPassword`、`Generic` 填寫 `usageType` 欄位以適應不同的使用情境。為了啟用完整的使用者流程，`Register`、`SignIn`、`ForgotPassword` 和 `Generic` 的模板是必需的。

### 測試 SMSAero 連接器 \{#test-smsaero-connector}

你可以輸入電話號碼並點擊 "Send" 來查看設置是否能在 "Save and Done" 之前運作。

就是這樣。別忘了 [在登入體驗中啟用連接器](/connectors/sms-connectors/#enable-phone-number-sign-up-or-sign-in)。

### 配置類型 \{#config-types}

| 名稱       | 類型        |
| ---------- | ----------- |
| email      | string      |
| apiKey     | string      |
| senderName | string      |
| templates  | Templates[] |

| 模板屬性  | 類型        | 枚舉值                                                  |
| --------- | ----------- | ------------------------------------------------------- |
| content   | string      | N/A                                                     |
| usageType | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
