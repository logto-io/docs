# YunPian SMS 連接器 (Connector)

官方 Logto YunPian SMS 服務連接器 (Connector)。

<GuideTip />

## 開始使用 \{#get-started}

雲片（YunPian）是一家提供多種通訊服務（包含簡訊）的服務供應商。YunPian SMS 連接器 (Connector) 是由 Logto 團隊提供的插件，用於整合 YunPian 的簡訊服務，讓 Logto 終端使用者可透過簡訊驗證碼註冊與登入。

## 在 YunPian 控制台設置簡訊服務 \{#set-up-sms-service-in-yunpian-console}

### 建立 YunPian 帳號 \{#create-a-yunpian-account}

前往 [雲片官網](https://www.yunpian.com/) 註冊帳號並完成實名認證。

### 取得 API KEY \{#get-api-key}

1. 登入 YunPian 控制台
2. 前往「帳戶設置」->「子帳戶管理」
3. 找到並複製 API KEY

### 配置簡訊模板 \{#configure-sms-templates}

1. 在 YunPian 控制台，前往「國內短信」->「簽名報備」
2. 建立並提交簽名，等待運營商審核
3. 前往「國內短信」->「模板報備」，選擇「驗證碼」
4. 建立驗證碼模板，確保包含 `#code#` 變數（也可使用「通用模板」加速審核流程）
5. 等待模板審核通過
6. 若需發送國際簡訊，請重複上述步驟，並選擇「國際短信」->「模板報備」

## 在 Logto 中配置 \{#configure-in-logto}

1. 進入 Logto 控制台，前往「連接器 (Connectors)」
2. 找到並點擊「YunPian SMS 服務」
3. 填寫配置表單：
   - API KEY：從 YunPian 取得的 API KEY
   - 簡訊模板：根據用途配置模板，並確保與 YunPian 審核通過的模板內容完全一致

## 注意事項 \{#notes}

1. 簡訊模板內容必須與 YunPian 審核通過的模板完全一致
2. YunPian 模板中的驗證碼變數佔位符為 `#code#`，而連接器 (Connector) 配置中為 `{{code}}`
3. YunPian 會根據 API KEY 自動添加預設簽名，模板內容無需包含簽名
4. 建議正式使用前先進行測試
