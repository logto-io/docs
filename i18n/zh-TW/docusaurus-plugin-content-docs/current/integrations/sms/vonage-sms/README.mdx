# 使用 Vonage SMS 設定簡訊驗證 (Set up SMS verification with Vonage SMS)

Vonage SMS 的官方 Logto 連接器。

<GuideTip />

## 開始使用 \{#get-started}

Vonage 是全球性的通訊服務提供商，提供強大的雲端通訊服務，包括簡訊（SMS, short message service）。Vonage SMS 連接器 (Vonage SMS Connector) 是由 Logto 團隊提供的外掛，讓 Logto 終端使用者能透過簡訊驗證碼註冊並登入他們的 Logto 帳號。
