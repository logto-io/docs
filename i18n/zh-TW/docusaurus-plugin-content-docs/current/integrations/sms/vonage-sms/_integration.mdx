## 在 Vonage 中設定 \{#set-up-in-vonage}

:::tip
如果你已經完成這些步驟，可以跳過此步。
:::

要使用此連接器，你需要在 Vonage [註冊一個帳戶](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account)。這將為你提供一個 API key 和 secret，讓你可以透過此連接器存取 API。

擁有帳戶後，你可以在 Vonage API Dashboard 的頂部找到你的 API key 和 API secret。

你可能需要[租用一個虛擬號碼](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number)來發送簡訊。

更多資訊請參閱 [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview)。

## 在 Logto 中設定 \{#set-up-in-logto}

1. **API Key**：你的 Vonage API key。
2. **API Secret**：你的 Vonage API secret。
3. **品牌名稱 (Brand Name)**：你想用來發送簡訊的品牌名稱，也稱為 `from` 欄位，詳情請參閱 [發送者身分 (Sender Identity)](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id)。
4. **模板 (Templates)**：你想用來發送簡訊的模板，你可以使用預設模板或根據需要進行修改。
