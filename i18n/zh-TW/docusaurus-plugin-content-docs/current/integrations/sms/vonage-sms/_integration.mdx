## 在 Vonage 中設定 \{#set-up-in-vonage}

:::tip
如果你已經完成這些步驟，可以跳過本節。
:::

要使用此連接器 (Connector)，你需要先在 Vonage [註冊帳號](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account)。這樣你就能取得 API 金鑰 (API key) 和密鑰 (API secret)，以便透過此連接器存取 API。

註冊帳號後，你可以在 Vonage API 控制台頂部找到你的 API key 和 API secret。

你可能還需要[租用虛擬號碼](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number)來發送簡訊 (SMS)。

更多資訊請參閱 [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview)。

## 在 Logto 中設定 \{#set-up-in-logto}

1. **API Key**：你的 Vonage API key。
2. **API Secret**：你的 Vonage API secret。
3. **品牌名稱 (Brand Name)**：你希望用來發送簡訊的品牌名稱，也稱為 `from` 欄位，詳情請參閱 [發送者身分 (Sender Identity)](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id)。
4. **範本 (Templates)**：你想用來發送簡訊的範本，可以使用預設範本或根據需求進行修改。
