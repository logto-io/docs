# 使用騰訊雲簡訊服務設定簡訊驗證

Logto 官方提供的騰訊雲簡訊服務連接器。

<GuideTip />

## 開始使用 \{#get-started}

騰訊雲是亞洲主要的雲端服務供應商，提供多種服務，包括簡訊（SMS, Short Message Service）。

騰訊雲簡訊服務連接器是 Logto 的官方整合，協助終端使用者透過簡訊驗證碼註冊或登入。
