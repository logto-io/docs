# 自訂使用者授權頁面品牌設定

當使用者被重定向到第三方應用程式的使用者授權頁面時，確保正確顯示第三方的品牌資訊和條款連結是很重要的。

Logto 允許你自訂第三方應用程式的品牌資訊，包括應用程式名稱、標誌和條款連結。

## 自訂品牌資訊 \{#customize-the-branding-information}

確保妥善配置第三方應用程式的品牌資訊，以確保使用者獲得一致且安全的驗證 (Authentication) 體驗。

1. 前往 <CloudLink to="/applications/third-party-applications">控制台 > 應用程式 > 第三方應用程式</CloudLink>，打開特定 OIDC 第三方應用程式的詳細資訊頁面。

2. 導航到 **品牌** 標籤。

3. 配置使用者授權頁面的顯示資訊：

- **顯示名稱**：將在使用者授權頁面上顯示的第三方應用程式名稱。它將代表請求存取你使用者資訊的第三方應用程式名稱。如果此欄位留空，將使用 **應用程式名稱**。
- **應用程式標誌（淺色）**：將在使用者授權頁面上顯示的第三方應用程式標誌。它將代表請求存取你使用者資訊的第三方應用程式品牌。如果提供了第三方應用程式的標誌和你的通用登入體驗標誌，兩者都將顯示在使用者授權頁面上。
- **應用程式標誌（深色）**：僅在啟用深色模式登入體驗時可用。在 <CloudLink to="/sign-in-experience/branding">控制台 > 登入體驗 > 品牌</CloudLink> 頁面管理深色模式設定。
- **使用條款 URL**：將在使用者授權頁面上顯示的第三方應用程式條款連結。
- **隱私權政策 URL**：將在使用者授權頁面上顯示的第三方應用程式隱私連結。
