# 使用者管理 (User management)

使用者管理是 Logto 的重要核心之一。你可以透過 Logto Console 或 Management API 存取並管理你的使用者資料，這兩種方式都非常有效。利用這些工具，你可以執行以下操作：

- 依使用者名稱、ID、全名、電話號碼或電子郵件搜尋使用者。
- 對使用者執行多種操作，例如解除社交帳號綁定、更新個人資料、刪除使用者或重設密碼。
- 檢查使用者日誌。
- 指派角色 (Roles) 以進行存取控制。
- 建立與管理個人存取權杖 (Personal access tokens)。
- 管理使用者工作階段 (sessions)。
