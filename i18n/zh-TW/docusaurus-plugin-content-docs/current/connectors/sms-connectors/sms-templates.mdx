Logto 提供四種不同的簡訊內容自訂範本，依用途分為：Register、SignIn、ForgotPassword 及 Generic。強烈建議你針對不同情境使用不同範本，否則可能觸發速率限制，導致服務暫時中斷。

## 簡訊範本類型與範例 \{#sms-template-types-and-examples}

以下僅為部分範例，供參考：

| usageType                | 情境說明                                                                                                                                                                                                                                                                                                            | 範本範例                                            |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| SignIn                   | 使用者[以手機號碼登入](/end-user-flows/sign-up-and-sign-in/sign-in)，並以輸入簡訊驗證碼取代密碼驗證。                                                                                                                                                                                                               | Logto 登入驗證碼：`{{code}}`。10 分鐘內有效。       |
| Register                 | 使用者[以手機號碼註冊帳號](/end-user-flows/sign-up-and-sign-in/sign-up)，並輸入 Logto 發送至手機的驗證碼完成驗證。                                                                                                                                                                                                  | Logto 註冊驗證碼：`{{code}}`。10 分鐘內有效。       |
| ForgotPassword           | 若使用者登入時忘記密碼，可選擇先以手機號碼驗證身分以[重設密碼](/end-user-flows/sign-up-and-sign-in/reset-password)。                                                                                                                                                                                                | Logto 密碼重設驗證碼：`{{code}}`。10 分鐘內有效。   |
| Generic                  | 此範本可作為多種情境的備用選項，包括測試連接器設定、[登入後驗證或綁定手機號碼](/end-user-flows/account-settings/by-management-api#email-and-phone-number-verification)等。                                                                                                                                          | Logto 驗證碼：`{{code}}`。10 分鐘內有效。           |
| OrganizationInvitation   | 使用此範本[發送邀請連結給使用者](/end-user-flows/organization-experience/invite-organization-members#configure-your-email-connector)加入組織。                                                                                                                                                                      | Logto 組織邀請驗證碼：`{{code}}`。10 分鐘內有效。   |
| UserPermissionValidation | 應用程式使用過程中，部分高風險或較高風險等級的操作（如銀行轉帳、刪除使用中資源、取消會員資格等）[需額外驗證使用者身分](/end-user-flows/account-settings/by-account-api#verify-by-sending-a-verification-code-to-the-users-email-or-phone)。可用 `UserPermissionValidation` 範本自訂這些情境下發送的簡訊驗證碼內容。 | Logto 驗證碼：`{{code}}`。10 分鐘內有效。           |
| BindNewIdentifier        | 使用者修改個人資料時，可能[將手機號碼綁定至現有帳號](/end-user-flows/account-settings/by-account-api#manage-phone)。此時可用 `BindNewIdentifier` 範本自訂驗證簡訊內容。                                                                                                                                             | Logto 帳號綁定驗證碼：`{{code}}`。10 分鐘內有效。   |
| MfaVerification          | 啟用 [簡訊 MFA](/end-user-flows/mfa/sms-mfa) 時，此範本用於多重要素驗證 (MFA) 流程中發送驗證碼給使用者。                                                                                                                                                                                                            | Logto 兩步驟驗證碼：`{{code}}`。10 分鐘內有效。     |
| BindMfa                  | 啟用 [簡訊 MFA](/end-user-flows/mfa/sms-mfa) 時，此範本用於設定 MFA 簡訊驗證碼。使用者將手機號碼綁定或設定為 MFA 因子時會收到此驗證碼。                                                                                                                                                                             | Logto 新增兩步驟驗證碼：`{{code}}`。10 分鐘內有效。 |

請注意以下參數：

- 驗證碼有效時間為 10 分鐘，目前尚不支援自訂有效期限。
- Logto 會將簡訊範本中的 `{{code}}` 佔位符替換為實際驗證碼，請務必保留此佔位符於範本中。

:::note
部分國家／地區因合規要求，可能禁止發送未經審核的簡訊內容。簡訊範本需先向簡訊供應商註冊並審核通過後方可使用。在這些情況下，內容可能會以範本 ID 對應至實際範本。
:::
