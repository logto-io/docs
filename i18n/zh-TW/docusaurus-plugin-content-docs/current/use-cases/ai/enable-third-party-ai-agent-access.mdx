# 啟用第三方 AI agent 存取你的應用程式

:::caution
👷 本教學仍在建置中。請稍後回來查看最新進度。🚧
:::

本指南將帶你整合 Logto 與你的服務，並啟用 AI agent 存取該服務。

你將學會：

- 將 Logto 設定為你的服務的授權伺服器 (Authorization server)。
- 取得 AI agent 存取你服務所需的存取權杖 (Access token)。
- 使用 AI agent 測試整個流程。

## 先決條件 \{#prerequisites}

- 一個 [Logto Cloud](https://cloud.logto.io)（或自架）租戶
- 一個可供 AI agent 存取 API 端點的服務

### 瞭解流程 \{#understanding-the-flow}

- **服務 (Service)**：你希望開放給 AI agent 存取的服務。
- **AI agent**：將會存取你服務的 AI agent。
- **Logto**：作為 OpenID Connect 提供者（授權伺服器），負責管理使用者身分。
