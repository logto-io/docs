# 終端使用者流程 (End-user flows)

終端使用者流程涵蓋所有與使用者互動的驗證 (Authentication) 流程，主要分為以下幾類：

- **驗證 (Authentication) 流程**：包含使用者登入、註冊與重設密碼。Logto 提供開箱即用的體驗，可在 <CloudLink to="/sign-in-experience">Console > 登入體驗 (Sign-in experience)</CloudLink> 設定，亦可透過 [Experience API](https://openapi.logto.io/group/endpoint-experience) [自訂 UI](/customization/bring-your-ui)。
- **帳號流程 (Account flows)**：包含帳號設定、使用者個人資料與安全驗證。可透過 [Account API](https://openapi.logto.io/group/endpoint-account-center) 或 [Management API](https://openapi.logto.io/operation/operation-getuser) 實作。
- **組織流程 (Organization flows)**：適用於多租戶產品，如 B2B 或 SaaS 服務。包含組織建立、成員邀請與管理。可透過 [Management API](https://openapi.logto.io/operation/operation-listorganizations) 整合至你的應用程式。
