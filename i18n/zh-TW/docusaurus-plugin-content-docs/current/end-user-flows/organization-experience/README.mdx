# 組織使用體驗 (Organization experience)

[組織 (Organization)](/organizations) 使用體驗指的是你的企業客戶及其員工在 [多租戶應用程式 (multi-tenant applications)](https://auth.wiki/multi-tenancy) 中所使用的介面與流程。Logto 提供多份指南，協助你透過 Logto Management API 快速將這些體驗整合進你的應用程式。
