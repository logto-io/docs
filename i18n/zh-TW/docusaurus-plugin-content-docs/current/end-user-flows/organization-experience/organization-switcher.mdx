# 組織切換器 (Organization switcher)

在組織使用體驗 (Organization experience) 中，使用者可以在不同組織間切換。每次切換時，應用程式會重新取得新的組織權杖 (Organization token) 以載入相關資訊。

<img src="/img/assets/organization-switcher.png" alt="組織切換器 (Organization switcher)" />

## 如何實作 \{#how-to-implement-it}

1. 建立下拉選單或類似的 UI 元件，列出使用者可用的所有組織 (Organizations)。
2. 當使用者選擇不同組織時，取得其 ID，並呼叫 SDK 方法 `getOrganizationToken(organizationId)` 以取得對應的組織權杖 (Organization token)。（SDK 會自動為你處理權杖快取。）
3. 儲存選取的組織 ID（可放在 URL 或用戶端儲存空間），並始終用它來取得對應的組織權杖 (Organization token) 以存取組織專屬資源。
