要初始化一個新的 Node.js 專案並使用 Koa，你可以按照以下步驟操作：

```bash
npm init -y
npm install koa @koa/router
```

如果你使用 TypeScript，請記得相應地設定你的 TypeScript 環境。

接著，建立一個基本的 Koa 伺服器設定：

```ts title="app.ts"
import Koa from 'koa';

const app = new Koa();

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

:::note
如需更多關於如何設定路由、中介軟體及其他功能的詳細資訊，請參閱 Koa 官方文件。
:::
