要初始化一個新的 Node.js 專案並使用 Fastify，可以按照以下步驟進行：

```bash
npm init -y
npm install fastify
```

如果你使用 TypeScript，請記得相應地設定你的 TypeScript 環境。

接著，建立一個基本的 Fastify 伺服器設定：

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Server running on http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
請參閱 Fastify 官方文件，以獲取更多關於如何設定路由、插件及其他功能的詳細資訊。
:::
