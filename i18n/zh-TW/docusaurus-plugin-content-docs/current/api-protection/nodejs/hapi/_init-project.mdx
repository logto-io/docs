要用 Hapi 初始化一個新的 Node.js 專案，你可以按照以下步驟操作：

```bash
npm init -y
npm install @hapi/hapi
```

如果你使用 TypeScript，請記得相應地設定你的 TypeScript 環境。

接著，建立一個基本的 Hapi 伺服器設定：

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Server running on http://localhost:3000');
```

:::note
更多有關如何設定路由、插件及其他功能，請參考 Hapi 官方文件。
:::
