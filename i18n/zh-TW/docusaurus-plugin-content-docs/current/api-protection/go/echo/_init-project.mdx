要用 Echo 初始化一個新的 Go 專案，你可以按照以下步驟操作：

```bash
go mod init your-api-name
go get github.com/labstack/echo/v4
```

接著，建立一個基本的 Echo 伺服器設定：

```go title="main.go"
package main

import (
    "github.com/labstack/echo/v4"
)

func main() {
    e := echo.New()

    e.Logger.Fatal(e.Start(":3000"))
}
```

:::note
更多關於如何設定路由、中介軟體及其他功能的細節，請參考 Echo 官方文件。
:::
