要用 Chi 初始化一個新的 Go 專案，你可以按照以下步驟操作：

```bash
go mod init your-api-name
go get github.com/go-chi/chi/v5
```

接著，建立一個基本的 Chi 伺服器設定：

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
如需更多關於路由、middleware 及其他功能的設定細節，請參考 Chi 官方文件。
:::
