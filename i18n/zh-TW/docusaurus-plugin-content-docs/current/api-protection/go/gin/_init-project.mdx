要使用 Gin 初始化新的 Go 專案，你可以依照以下步驟操作：

```bash
go mod init your-api-name
go get github.com/gin-gonic/gin
```

接著，建立一個基本的 Gin 伺服器設定：

```go title="main.go"
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()

    r.Run(":3000") // 監聽並服務於 0.0.0.0:3000
}
```

:::note
更多關於如何設定路由、中介軟體及其他功能，請參考 Gin 官方文件。
:::
