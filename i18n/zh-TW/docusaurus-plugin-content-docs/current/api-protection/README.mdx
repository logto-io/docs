# API 保護

學習如何實作 [角色型存取控制 (RBAC, Role-based Access Control)](/authorization/role-based-access-control) 與 [JSON Web Token (JWT)](https://auth.wiki/jwt) 驗證，保護你的 API 端點免於未經授權的存取。

我們的指南涵蓋中介軟體（middleware）設置、權限模型，以及多種程式語言與框架的實用範例。

## 開始使用 \{#get-started}

### API 安全新手？ \{#new-to-api-security}

如果你是 API 驗證 (Authentication) 與授權 (Authorization) 的新手，請從以下基礎開始：

- **[角色型存取控制 (RBAC, Role-based Access Control)](/authorization/role-based-access-control)** — 學習權限與角色的運作方式
- **[JSON Web Token (JWT)](https://auth.wiki/jwt)** — 探索 JSON Web Token 如何保護你的 API
