要初始化一個新的 Grape API 專案，請建立一個目錄並設置基本結構：

```bash
mkdir your-api-name
cd your-api-name
```

建立 Gemfile：

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

安裝相依套件：

```bash
bundle install
```

建立一個基本的 Grape API：

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

建立 config.ru 檔案：

```ruby title="config.ru"
require_relative 'api'

run API
```

啟動開發伺服器：

```bash
bundle exec rackup
```

:::note
如需更多關於如何設置資源、參數驗證及其他功能的細節，請參考 Grape 官方文件。
:::
