要初始化一個新的 Sinatra 專案，請建立一個目錄並設置基本結構：

```bash
mkdir your-api-name
cd your-api-name
```

建立一個 Gemfile：

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'sinatra'
```

安裝相依套件：

```bash
bundle install
```

建立一個基本的 Sinatra 應用程式：

```ruby title="app.rb"
require 'sinatra'
require 'json'

get '/' do
  content_type :json
  { message: 'Hello from Sinatra API' }.to_json
end
```

啟動開發伺服器：

```bash
ruby app.rb
```

:::note
如需更多關於如何設置路由、中介軟體及其他功能的詳細資訊，請參閱 Sinatra 官方文件。
:::
