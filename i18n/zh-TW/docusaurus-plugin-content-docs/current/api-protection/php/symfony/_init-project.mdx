要初始化一個新的 Symfony 專案以進行 API 開發，可以使用 Symfony CLI 或 Composer：

推薦使用 Symfony CLI：

```bash
symfony new your-api-name --webapp
cd your-api-name
```

或使用 Composer：

```bash
composer create-project symfony/skeleton your-api-name
cd your-api-name
composer require webapp
```

安裝額外的 API 開發套件：

```bash
composer require symfony/security-bundle
composer require symfony/serializer
composer require doctrine/annotations
```

啟動開發伺服器：

```bash
symfony serve
```

或使用 PHP 內建伺服器：

```bash
php -S localhost:8000 -t public/
```

這樣會建立一個基本的 Symfony 專案。接著設定框架以支援 API 開發：

```yaml title="config/packages/framework.yaml"
framework:
  secret: '%env(APP_SECRET)%'
  serializer:
    enabled: true
  property_access:
    enabled: true
```

:::note
更多關於如何設定控制器、服務與其他功能，請參考 Symfony 官方文件。
:::
