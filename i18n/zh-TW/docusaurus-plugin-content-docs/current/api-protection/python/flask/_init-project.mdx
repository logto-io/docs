要初始化一個新的 Flask 專案，請建立一個目錄並設置基本結構：

```bash
mkdir your-api-name
cd your-api-name
```

建立需求檔案：

```txt title="requirements.txt"
Flask
```

安裝相依套件：

```bash
pip install -r requirements.txt
```

建立一個基本的 Flask 應用程式：

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

啟動開發伺服器：

```bash
python app.py
```

:::note
如需更多關於如何設置路由、藍圖及其他功能的詳細資訊，請參閱 Flask 官方文件。
:::
