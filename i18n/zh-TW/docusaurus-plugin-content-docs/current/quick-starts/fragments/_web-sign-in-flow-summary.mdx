在進入細節之前，這裡先快速說明一下終端使用者的體驗。登入流程可簡化如下：

```mermaid
graph LR
    A(你的應用程式 (Your app)) -->|1\. 呼叫登入 (Invoke sign-in)| B(Logto)
    B -->|2\. 完成登入 (Finish sign-in)| A
```

1. 你的應用程式呼叫登入方法。
2. 使用者被重新導向至 Logto 登入頁面。對於原生應用程式，會開啟系統瀏覽器。
3. 使用者登入後，會被重新導向回你的應用程式（設定為 redirect URI）。
