{/* 此片段結合了重定向 URI 和登出後重定向 URI 的配置，可能對某些框架有幫助。 */}

import redirectUriFigure from '@site/docs/quick-starts/assets/web-redirect-uri.png';

export const defaultBaseUrl = 'http://localhost:3000/';
export const defaultRedirectUri = `${defaultBaseUrl}callback`;

#### 配置重定向 URI \{#configure-redirect-uris}

切換到 Logto Console 的應用程式詳細資訊頁面。新增一個重定向 URI <code>{props.redirectUri ?? 'http://localhost:3000/callback'}</code>。

<img alt="Logto Console 中的重定向 URI" src={redirectUriFigure} width="600px" />

就像登入一樣，使用者應被重定向到 Logto 以登出共享會話。完成後，將使用者重定向回你的網站會很不錯。例如，將 <code>{defaultBaseUrl}</code> 新增為登出後重定向 URI 區段。

然後點擊「儲存」以保存更改。
