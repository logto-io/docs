# 為你的 Java Spring Boot 應用程式新增驗證 (Authentication)

本指南將帶你將 Logto 整合進你的 Java Spring Boot 應用程式。

<GuideTip />

## 先決條件 \{#prerequisites}

- 一個 [Logto Cloud](https://cloud.logto.io) 帳號或 [自行架設 Logto](/introduction/set-up-logto-oss)。
- 我們的範例程式碼是以 Spring Boot 的 [securing web starter](https://spring.io/guides/gs/securing-web) 建立。如果你還沒有應用程式，請依照官方指引建立新的網頁應用程式。
- 本指南將使用 [Spring Security](https://spring.io/projects/spring-security) 與 [Spring Security OAuth2](https://spring.io/guides/tutorials/spring-boot-oauth2) 套件來處理與 Logto 的 OIDC 驗證流程。請務必參閱官方文件以瞭解相關概念。
