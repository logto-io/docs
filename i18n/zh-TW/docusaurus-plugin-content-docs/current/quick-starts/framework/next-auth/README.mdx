# 為你的 Auth.js (Next Auth) 應用程式新增驗證 (Authentication)

本指南將說明如何將 Logto 整合至你的 Next.js 應用程式，並搭配 [Auth.js](https://authjs.dev/)（前稱 Next Auth）使用。

<GuideTip />

## 先決條件 \{#prerequisites}

- 一個 [Logto Cloud](https://cloud.logto.io) 帳號或 [自行架設 Logto](/introduction/set-up-logto-oss)。
- 已建立的 Logto 傳統應用程式。
- 已安裝 Auth.js 的 Next.js 專案，請參考 [Auth.js 文件](https://authjs.dev/getting-started/installation)。
