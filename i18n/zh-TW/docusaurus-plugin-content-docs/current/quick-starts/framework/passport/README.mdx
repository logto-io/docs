# 為你的 Passport.js 應用程式新增驗證 (Authentication)

本指南將向你展示如何使用 [Passport.js](https://www.passportjs.org/) 和 OIDC 策略將 Logto 整合到你的應用程式中。

<GuideTip />

## 先決條件 \{#prerequisites}

- 一個 [Logto Cloud](https://cloud.logto.io) 帳戶或 [自託管 Logto](/introduction/set-up-logto-oss)。
- 一個已建立的 Logto 傳統應用程式。
- 一個已配置 session 的 express 專案。請參閱 [Express.js 網站](https://expressjs.com/)。
