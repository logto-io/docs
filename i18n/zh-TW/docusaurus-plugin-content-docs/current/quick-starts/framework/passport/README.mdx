- 一個 [Logto Cloud](https://cloud.logto.io) 帳號或 [自行架設 Logto](/introduction/set-up-logto-oss)。
- 已建立的 Logto 傳統應用程式。
- 已設定 session 的 express 專案。請參考 [Express.js 官方網站](https://expressjs.com/)。
