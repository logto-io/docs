# 快速入門

歡迎來到 Logto 快速入門。本文件提供將 Logto 整合進你的應用程式的實用指南。

大多數指南都使用 Logto 官方 SDK。如果你正在使用 AI 驅動的開發工具，歡迎嘗試我們的 [Logto MCP Server](/logto-cloud/logto-mcp-server) 以獲得 AI 協助整合。

如有任何疑問，請隨時聯絡我們。

:::
