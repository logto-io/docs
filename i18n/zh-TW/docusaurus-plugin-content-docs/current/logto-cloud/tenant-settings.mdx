# 租戶設定 (Tenant settings)

## 租戶區域 (Tenant region) \{#tenant-region}

當你建立租戶時，可以選擇租戶資料儲存的區域。租戶建立後無法更改區域。目前可用區域如下：

- 歐洲（荷蘭）
- 美國西部（亞利桑那）
- 澳洲（澳洲東部）
- 日本（日本東部）

通常建議選擇最接近你客戶的區域，以降低延遲並提升效能。

Logto 利用全球邊緣網路，為你的應用程式提供最佳效能與可用性。請求路由經過最佳化，確保你的使用者始終連接到效能最佳的節點。

:::note
需要其他區域嗎？[聯絡我們](https://logto.io/contact)：

- 申請新的公有雲區域
- 詢問在你偏好的地點部署 Logto 私有雲 (Logto Private Cloud)
  :::

## 租戶類型：開發 (Dev) vs. 生產 (Prod) \{#tenant-types-dev-vs-prod}

Logto Cloud 有兩種租戶類型：開發（development）與生產（production）。透過區分租戶類型，你可以更有效率地管理不同環境下的專案，同時充分享受 Logto 的完整價值。

你可以在建立租戶時選擇租戶類型。當你準備好正式上線時，有兩種選擇：

- **建立新的生產租戶 (Production tenant)**  
  從零開始設定全新的生產租戶。若你希望將開發與生產環境分開，這是理想選擇。
- **將現有開發租戶 (Dev tenant) 轉換為生產租戶 (Production)**  
  若你不想重新設定或遷移使用者，可以透過訂閱 Pro 方案（每月 $16 起）將現有開發租戶升級為付費生產租戶。
  - 你在開發租戶中使用的所有付費功能都會帶入 Stripe 結帳流程。
  - **轉換後，租戶無法再回復為開發環境，請確認準備好再進行操作。**

### 開發 (Development) \{#development}

開發租戶（dev tenant）主要用於測試，不應用於生產環境。這些租戶可免費存取付費方案中的高級功能，且無需訂閱。

但開發租戶有以下限制：

- 開發租戶會自動刪除 90 天以上的使用者與組織。
- 登入體驗期間會顯示橫幅，提示租戶處於開發模式。
- 開發租戶部分功能可能有配額限制，詳情請參閱功能說明頁。
- Logto 可能會調整開發租戶的配額限制，並會盡力提前通知你。

### 生產 (Production) \{#production}

生產租戶是終端使用者存取正式應用程式的環境，你可能需要[付費訂閱](https://logto.io/pricing)。你可以訂閱 Free 方案或 Pro 方案來建立生產租戶。若訂閱 Free 方案，最多只能建立 10 個租戶。

## 啟用多重要素驗證 (MFA) \{#enable-mfa}

透過要求 Logto Pro／Enterprise 租戶所有成員啟用多重要素驗證 (MFA, Multi-Factor Authentication)，提升工作區安全性。

目前尚未開放自助設定，請[聯絡我們](https://logto.io/contact)以啟用此功能。

## 啟用企業級單一登入 (Enterprise SSO) \{#enable-enterprise-sso}

Logto Cloud 支援付費租戶整合企業級單一登入 (Enterprise SSO)，包含 Google Workspace、Okta、Azure AD 等提供者。

如需開始設定，請[聯絡我們](https://logto.io/contact)。我們將協助你快速完成設定。
