# Logto 私有雲 (Private Cloud)

在專屬的 Azure 環境中部署 Logto，以獲得更強的隔離性、合規性與基礎架構控制。

## 概述 \{#overview}

Logto 私有雲 (Private Cloud) 是為需要比共享公有雲環境更高基礎架構隔離與控制的客戶所設計的專屬部署選項。

它專為對安全性、合規性、採購或部署有更嚴格要求的組織設計，同時保有與 Logto Cloud 相同的核心產品體驗。

以下是雲端選項的差異：

- **Logto Cloud**：受管公有雲，最快進入生產環境的途徑
- **Logto 私有雲 (Private Cloud)**：專為對資料存放地或合規性有更嚴格要求的客戶提供的專屬部署

## 何時選擇私有雲 (Private Cloud) \{#when-to-choose-private-cloud}

當你的組織有以下一項或多項需求時，Logto 私有雲 (Private Cloud) 通常是合適的選擇：

- 更強的環境隔離
- 具 SLA 保證的正常運作承諾，並有 Logto 團隊的支援與協作
- 共享公有雲無法滿足的特定合規性要求
- 需部署於 Logto Cloud 不支援的特定區域或基礎架構
- 需連接無法從公有雲環境存取的內部資源

如果你沒有上述需求，[Logto Cloud](https://cloud.logto.io/?sign_up=1) 通常是更簡單採用且更易於操作的選擇。

### 部署範圍 \{#deployment-scope}

Logto 私有雲 (Private Cloud) 目前僅支援 **Azure**。

支援的區域取決於該 Azure 區域是否提供 Logto 所需的服務。如果你有目標區域，請聯絡我們，我們會根據你的需求確認可行性。

### 安全性與合規性 \{#security-and-compliance}

私有雲 (Private Cloud) 提供比共享公有雲部署更明確隔離界線的專屬環境。

我們以實際需求為導向討論安全性與合規性。如果你的團隊需要安全審查、合規討論或部署細節釐清，歡迎聯絡我們，我們會直接與你討論細節。
