# 系統限制

在 Logto，我們為所有方案設定了寬鬆的限制，並提供彈性的按量付費選項，讓使用者只需為實際使用的部分付費。

你可能會注意到，部分定價頁上的項目標註為 _無限制_ 或 _持續按量付費無上限_。這表示通常可無限制使用，但 Logto 保留隨時間調整實際限制的權利，以維護所有使用者的公平使用。換句話說，實體限制是為了保護平台整體健康而設立的嚴格上限。這些限制並非定價的一部分，但可能會因不同方案組而有所差異。

如果你的使用情境合理但達到系統限制，歡迎[聯絡我們](https://logto.io/contact)並分享你的回饋。這有助於我們更好地理解實際使用情境，並調整系統限制以更好地支持忠實客戶。
