# 介紹

歡迎來到 Logto 文件！Logto 是專為現代應用程式與 SaaS 產品設計的身分與存取管理（IAM, Identity and Access Management）解決方案。它為你的應用程式提供安全、可擴展且可自訂的驗證 (Authentication) 與授權 (Authorization) 系統。
