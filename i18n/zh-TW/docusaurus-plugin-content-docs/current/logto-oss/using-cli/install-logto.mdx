# 安裝 Logto

## 互動式安裝 \{#interactive}

在終端機執行下方指令，進行互動式安裝：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## 參數與靜默安裝 \{#options-and-silent-installation}

你也可以加上下方任一選項（皆為可選），以略過部分提問：

| Option                                         | Description（說明）                                                                                                                                                                             |
| ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--db-url`                                     | Logto 資料庫的 Postgres URL。                                                                                                                                                                   |
| `-p, --path`                                   | Logto 實例的路徑，必須為不存在的路徑。                                                                                                                                                          |
| `--ss`                                         | 略過 Logto 資料庫初始化資料。                                                                                                                                                                   |
| `--disable-admin-pwned-password-check, --dapc` | 為管理租戶（admin tenant）停用 Have I Been Pwned（HIBP）密碼洩漏檢查，適用於隔離或離線部署。詳見 [離線 / 隔離部署初始化資料](/logto-oss/using-cli#seed-for-air-gapped-or-offline-deployments)。 |

執行 `logto init --help` 以獲得完整說明。

以下為靜默安裝（例如在 CI 中）的範例：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

它將依序執行下列操作：

1. 下載並解壓 Logto 至 `./logto`
2. 嘗試以 URL `postgresql://your-postgres-dsn:5432/logto` 初始化並填充資料庫
3. 將 Logto 官方連接器新增至 `./logto/packages/core/connectors`
