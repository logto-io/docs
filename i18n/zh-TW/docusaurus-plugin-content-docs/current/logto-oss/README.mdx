# Logto OSS

Logto 提供開源（OSS）與 [雲端](/logto-cloud)版本。本節聚焦於 OSS 方案：<Url href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS 是免費使用的身分解決方案，讓開發者能完全自訂並完全掌控自己的資料。Logto OSS 提供靈活的程式碼基礎，可依需求調整，非常適合重視資料主權並希望擁有可隨應用成長的專屬身分提供者解決方案的你。

## Logto OSS 支援的功能 \{#feature-supported-by-logto-oss}

Logto OSS 支援 Logto 服務的大多數核心功能，並持續獲得更新與維護。

部分進階功能目前僅限於 Logto Cloud 版本，包括：

| OSS 功能限制                                                                  | 說明                                                                                  |
| ----------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| [Logto 控制台：多租戶](/logto-cloud/tenant-settings)                          | 在控制台中建立並管理多個 Logto 租戶。                                                 |
| [Logto 控制台：協作者邀請](/logto-cloud/tenant-member-management)             | 邀請並管理單一 Logto 租戶的多位成員。                                                 |
| Logto 控制台：啟用 MFA                                                        | 登入 Logto 控制台時強制多重要素驗證 (MFA, Multi-factor authentication) 以提升安全性。 |
| [Logto Protected App](/integrate-logto/protected-app)                         | 由 Cloudflare 提供的快速、無需 SDK 的驗證 (Authentication) 整合。                     |
| [Logto 內建電子郵件服務](/connectors/email-connectors/built-in-email-service) | 使用內建且免費的電子郵件服務進行郵件發送。                                            |
| [自帶 UI](/customization/bring-your-ui)                                       | 使用此功能自訂登入體驗的 UI 或流程（OSS 可直接 fork GitHub 程式碼自訂登入體驗）。     |
| [IdP 發起 SSO](/end-user-flows/enterprise-sso/idp-initiated-sso)              | 啟用身分提供者 (IdP, Identity provider) 發起的企業級單一登入 (SSO, Single Sign-On)。  |
| [SAML 應用程式](/integrate-logto/saml-app)                                    | 透過 SAML 協定整合應用程式。OSS 版本僅限 3 個 SAML 應用程式。                         |
| [隱藏 Logto 品牌](/customization/match-your-brand#hide-logto-branding)        | 從登入體驗中移除「Powered by Logto」標記。                                            |

提示：多租戶、成員邀請與 MFA 無法用於你的團隊登入開源 Logto 控制台。但你可以用 Logto OSS 在自己的產品中實作這些功能，讓終端使用者使用。
