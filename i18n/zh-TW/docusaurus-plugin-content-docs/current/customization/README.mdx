# 自訂化 (Customization)

透過自訂品牌、使用者介面、語言與網域設定，讓 Logto 完全符合你的需求。

## 預設 UI \{#prebuilt-ui}

從我們預設的 UI 元件開始，自訂關鍵品牌元素。你可以：

- 打造平台一致的驗證 (Authentication) 體驗。
- 為特定應用程式或組織設計專屬登入頁面。

<DocCardList
  items={[
    {
      type: 'link',
      label: '品牌一致化 (Match your brand)',
      href: '/customization/match-your-brand',
      description: '瞭解如何自訂基本品牌元素，打造即用型登入體驗與帳號中心。',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## 自訂 UI \{#custom-ui}

你可以透過以下兩種方式，完全掌控驗證 (Authentication) 體驗：

## 內容與語言 \{#content--languages}

Logto 預設支援 15 種語言。你可以擴充語言支援，或調整現有翻譯內容以符合需求。

<DocCardList
  items={[
    {
      type: 'link',
      label: '語言與內容 (Languages & Content)',
      href: '/customization/localized-languages',
      description: '瞭解如何新增語言並自訂使用者內容。',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## 自訂網域 \{#custom-domain}

透過自有網域作為驗證 (Authentication) URL，強化品牌形象。將預設 Logto 網域替換為你的品牌網域。

<DocCardList
  items={[
    {
      type: 'link',
      label: '自訂網域 (Custom domain)',
      href: '/logto-cloud/custom-domain',
      description: '使用自有網域作為登入或註冊 URL，提升品牌感與使用者信任。',
      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>

## 即時預覽 \{#live-preview}

利用即時預覽功能，體驗不同驗證 (Authentication) 流程（[註冊](/end-user-flows/sign-up-and-sign-in/sign-up)、[登入](/end-user-flows/sign-up-and-sign-in/sign-in)、[社交登入](/end-user-flows/sign-up-and-sign-in/social-sign-in)、[忘記密碼](/end-user-flows/sign-up-and-sign-in/reset-password)、[MFA](/end-user-flows/mfa)、[SSO](/end-user-flows/enterprise-sso) 等）。可透過內建的 Demo App 立即測試你的登入體驗。
