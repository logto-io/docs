# Logto 介绍

欢迎阅读 Logto 文档！Logto 是一个为现代应用和 SaaS 产品设计的身份和访问管理 (IAM) 解决方案。它为你的应用程序提供了一个安全、可扩展且可定制的认证和授权系统。

## 按功能探索

Logto 提供了一套强大的功能，包括认证 (Authentication)、授权 (Authorization) 以及用户和组织 (Organization) 管理，专为现代身份需求量身定制。我们强大的 CIAM（客户身份和访问管理）支持使其在该领域具有明显优势。
