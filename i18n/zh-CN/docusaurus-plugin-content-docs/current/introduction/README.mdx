# Logto 简介

欢迎来到 Logto 文档！Logto 是专为现代应用和 SaaS 产品设计的身份与访问管理 (IAM) 解决方案。它为你的应用提供安全、可扩展且可定制的认证 (Authentication) 和授权 (Authorization) 系统。
