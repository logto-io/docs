# B2B 架构

## 架构

B2B 应用通常使用[多租户](https://auth.wiki/multi-tenancy)架构。在这些应用中，用户拥有他们的账户并管理他们的身份和认证 (Authentication)，同时涉及其他方，如企业或组织。终端用户的身份通常不是个人消费者，而是企业组织内的员工或合作者。
