# B2C 架构

## 架构

在消费者应用中，用户完全拥有他们的账户，并控制他们的身份和认证，不涉及其他“中间层”方，如企业或组织。这是 B2C 和 B2B 身份架构之间的关键区别。

## 将你的需求连接到 Logto 的支持工具包

此架构包括管理场景中涉及的两个主要方。根据你的具体需求和目标，所有或仅部分这些方可能会参与其中。

我们总结了常见的用例，突出每个用户身份管理任务的关键目标以及我们提供的相关产品和 API。你可以将你的需求映射到我们的服务，以快速入门。
