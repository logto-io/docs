# 组织 (Organization) 数据结构

在组织 (Organization) 层面，有几个属性，其值可以被创建、更新、读取和删除 (CRUD)。

## 组织 (Organization) ID \{#organization-id}

_组织 (Organization) id_ 是每个组织 (Organization) 的唯一标识符。它对于实现组织 (Organization) 级别的登录体验和检索组织令牌 (Organization tokens) 很有用。

## 名称 \{#name}

_名称_ 支持组织 (Organization) 级别的登录，并可以根据需要集成到组织 (Organization) 级别的产品界面中。

## 描述 \{#description}

_描述_ 字段允许你添加文本以帮助识别和标记组织 (Organization)。

## 组织 (Organization) 标志 \{#organization-logos}

为了在登录体验中动态显示客户的组织 (Organization) 标志，你可以将组织 (Organization) 标志上传到组织 (Organization) 设置页面。

有关更多详细信息，请参见 [组织 (Organization) 特定标志](/customization/match-your-brand/#organization-specific-logos)。

## 自定义数据 \{#custom-data}

_自定义数据_ 是一个 JSON 对象，用于存储有关组织 (Organization) 的额外信息。它为开发人员提供了灵活性，可以存储额外的组织 (Organization) 详细信息以用于开发目的。
