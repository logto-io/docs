# 组织数据结构

在组织层面，有几个属性的值可以被创建、更新、读取和删除（CRUD）。

## 组织 ID

组织 ID 是每个组织的唯一标识符。它对于实现组织层面的登录体验和检索组织令牌 (Organization tokens) 很有用。

## 名称

名称支持组织层面的登录，并可以根据需要集成到组织层面的产品界面中。

## 描述

描述字段允许你添加文本以帮助识别和标记组织。

## 组织标志

为了在登录体验中动态显示客户的组织标志，你可以将组织标志上传到组织设置页面。

有关更多详细信息，请参见 [特定于组织的标志](/customization/match-your-brand/#organization-specific-logos)。

## 自定义数据

自定义数据是一个 JSON 对象，用于存储有关组织的额外信息。它为开发人员提供了灵活性，可以存储额外的组织详细信息以用于开发目的。
