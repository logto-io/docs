### 设置 LinkedIn 应用 \{#setup-a-linkedin-app}

访问 [LinkedIn Developers](https://www.linkedin.com/developers) 并使用你的 LinkedIn 账户登录。如果你没有账户，可以注册一个。

然后，创建一个应用。

**步骤 1：** 填写应用详情。

完成表单并创建应用。

**步骤 2：** 设置回调 URL。

进入应用详情页面，找到“Auth”选项卡，“OAuth 2.0”部分，并找到“Authorized redirect URLs for your app”字段。

在我们的例子中，这将是 `${your_logto_endpoint}/callback/${connector_id}`。例如 `https://foo.logto.app/callback/${connector_id}`。`connector_id` 可以在 Logto 管理控制台连接器详情页面的顶部栏找到。

**步骤 3：** 添加产品。

进入“Products”选项卡并添加产品“使用 OpenID Connect 登录 LinkedIn”。

### 配置你的连接器 \{#configure-your-connector}

在你的 Logto 连接器配置中，使用从应用的“Auth”选项卡，“Application credentials”部分获得的值填写以下字段：

- **clientId：** 你的应用的 Client ID。
- **clientSecret：** 你的应用的 Primary Client Secret。

`scope` 是一个以空格分隔的 OIDC 权限 (Scopes) 列表。如果未提供，默认权限 (Scope) 是 `openid profile email`。

### 配置类型 \{#config-types}

| 名称         | 类型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
