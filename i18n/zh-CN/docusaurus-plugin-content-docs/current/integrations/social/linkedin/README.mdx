# 设置 LinkedIn 社交登录

Logto 官方的 LinkedIn 社交登录连接器。

<GuideTip />

## 开始 \{#get-started}

LinkedIn 连接器使终端用户能够通过 LinkedIn OAuth 2.0 认证 (Authentication) 协议使用他们自己的 LinkedIn 账户登录到你的应用程序。
