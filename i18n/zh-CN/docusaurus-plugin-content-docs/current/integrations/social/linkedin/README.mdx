# 设置 LinkedIn 社交登录

Logto 官方的 LinkedIn 社交登录连接器。

<GuideTip />

## 入门指南 \{#get-started}

LinkedIn 连接器允许终端用户通过 LinkedIn OAuth 2.0 认证 (Authentication) 协议，使用他们自己的 LinkedIn 账户登录你的应用程序。
