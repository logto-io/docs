## 在 KOOK 开发者中心创建应用程序 \{#create-an-application-in-the-kook-developer-center}

提示：如果你已经完成了部分步骤，可以跳过相关部分。

:::note
你需要在创建 OAuth2 应用程序之前申请 KOOK OAuth2 权限。请查看 OAuth2 页面上的信息，或在 KOOK 官方开发者服务器向 KOOK 工作人员咨询更多信息。
:::

1. 前往 [KOOK 开发者中心](https://developer.kookapp.cn/app/index) 并使用你的 KOOK 账号登录。如果你还没有账号，可以注册一个新账号。
2. 点击 **创建应用** 按钮来创建一个应用程序，为其选择一个名称（例如 LogtoAuth），然后点击 **确认**。
3. 点击你创建的应用程序图标，进入 **OAuth2** 页面。
4. 在 **回调 URI** 字段中添加有效的重定向地址（例如 `${your_logto_origin}/callback/${connector_id}`）。你可以在 logto 管理控制台连接器详情页找到重定向 uri。
5. 别忘了在 **OAuth2 链接生成器** 中勾选 **get_user_info** 权限 (Scope)。

## 配置你的 KOOK 连接器 \{#configure-your-kook-connector}

将你在上一节 OAuth2 页面获得的 Client ID 和 Client Secret 分别填写到 clientId 和 clientSecret 字段中。

### 配置类型 \{#config-types}

| 名称         | 类型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
