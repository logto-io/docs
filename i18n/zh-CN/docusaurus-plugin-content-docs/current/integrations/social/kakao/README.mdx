# 设置 Kakao 社交登录

Kakao 连接器为你的应用程序提供了一种简洁的方式来使用 Kakao 的 OAuth 2.0 认证系统。

<GuideTip />

## 在 Kakao 开发者控制台中设置项目 \{#set-up-a-project-in-the-kakao-devlopers-console}

- 访问 [Kakao 开发者控制台](https://developers.kakao.com/console/app) 并使用你的 Kakao 账户登录。
- 点击 **添加应用程序** 来创建新项目或选择现有项目。

## 配置 Kakao 登录 \{#configure-kakao-login}

### 激活 Kakao 登录 \{#activate-kakao-login}

- 从菜单中点击 **产品设置 -> Kakao 登录**。
- 打开 `Kakao 登录激活`
- 在 `Redirect URI` 中添加以下 URL
  - `http(s)://YOUR_URL/callback/${connector_id}` (`connector_id` 可以在 Logto 管理控制台连接器详情页面的顶部栏找到。)
  - （请将 `YOUR_URL` 替换为你的 `Logto` URL，并根据你的情况选择 `http` 或 `https`。）

### 隐私设置 \{#privacy-setting}

- 从菜单中点击 **产品设置 -> Kakao 登录 -> 同意项**。
- 将 `昵称`、`头像` 和 `邮箱` 的状态更改为 **必需同意**（由于项目设置的原因，你可能无法将 `邮箱` 更改为 **必需同意**。）

### 安全设置（可选） \{#security-setting-optional}

- 从菜单中点击 **产品设置 -> Kakao 登录 -> 安全**。
- 点击 `客户端密钥` 以生成密钥。
- 将 `激活状态` 更改为启用。（如果你启用了它，则需要 `密钥`。）

## 配置 Logto \{#configure-logto}

### 配置类型 \{#config-types}

| 名称         | 类型    |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId \{#clientid}

`clientId` 是你项目的 `REST API key`。
（你可以在 Kakao 开发者控制台的项目 `概览` 中找到它。）

#### clientSecret \{#clientseceret}

`clientSecret` 是你项目的 `Secret Code`。
（请查看 [安全设置（可选）](#security-setting-optional)）
