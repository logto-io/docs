### 设置 Line 渠道

访问 [Line Developers](https://developers.line.biz/console/) 并使用你的 Line 商业账户登录。如果你没有账户，可以注册一个。

然后，访问[此链接](https://developers.line.biz/console/register/line-login/provider/)创建一个渠道。

**步骤 1：** 填写渠道详情。

完成表单并创建渠道。

**步骤 2：** 设置回调 URL。

进入渠道详情页面，找到“LINE 登录”选项卡并编辑“回调 URL”字段。

在我们的例子中，这将是 `${your_logto_endpoint}/callback/${connector_id}`。例如 `https://foo.logto.app/callback/${connector_id}`。`connector_id` 可以在 Logto 管理控制台连接器详情页面的顶部栏找到。

### 配置你的连接器

在你的 Logto 连接器配置中，使用从应用程序的“Auth”选项卡、“应用程序凭证”部分获得的值填写以下字段：

- **clientId：** 你的应用程序的渠道 ID。
- **clientSecret：** 你的应用程序的渠道密钥。

`scope` 是一个以空格分隔的 OIDC 权限 (Scopes) 列表。如果未提供，默认权限 (Scope) 是 `openid profile`。

### 配置类型

| 名称         | 类型   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
