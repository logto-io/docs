# 设置 WeChat (Web) 社交登录

Logto 官方为 Web 应用提供的 WeChat 社交登录连接器。

<GuideTip />

## 入门指南 \{#get-started}

如果你不了解连接器的概念，或者不知道如何将此连接器添加到你的登录体验中，请参阅 [Logto 教程](/connectors/social-connectors)。

:::caution

此连接器仅适用于 Web 应用。如果你需要在原生应用中使用 WeChat 登录，请参阅 [WeChat 原生连接器](/integrations/wechat-native)。

:::

## 在微信开放平台创建 Web 应用 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

如果你已经完成了部分步骤，可以跳过相关章节。

:::

### 创建账号 \{#create-an-account}

打开 https://open.weixin.qq.com/，点击右上角的“注册”按钮，然后完成注册流程。

### 创建 Web 应用 \{#create-a-web-app}

使用刚刚创建的账号登录。在“网页应用”标签页，点击绿色大按钮“创建网页应用”。

<img src="/img/assets/app-tabs.png" alt="应用标签" />

接下来填写应用表单中的必填信息。

<img src="/img/assets/create-web-app.png" alt="创建网页应用" />

#### 基本信息 \{#basic-info}

大部分内容都很直观。填写完表单后，点击“下一步”继续。

#### 网站信息 \{#website-info}

在“授权回调域”中填写你的 Logto 域名。例如：`logto.io`。

#### 等待审核结果 \{#waiting-for-the-review-result}

完成网站信息后，点击“提交审核”继续。通常审核速度较快，一般 1-2 天内完成。

我们推测每次提交的审核员是随机分配的，因为审核标准会有浮动。你可能第一次会被拒绝，但不要灰心！如实说明你的情况，并向审核员询问如何修改。

### 配置你的 WeChat 连接器 \{#configure-your-wechat-connector}

将 _Client ID_ 和 _Client Secret_ 填写到 `clientId` 和 `clientSecret` 字段中，这些信息可以在 OAuth 应用详情页获取。

`scope` 字段可以填写 'snsapi_userinfo' 或 'snsapi_base'。该字段为可选项，可以留空，默认值为 'snsapi_userinfo'。你可以查看[不同取值的区别](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)。

### 测试 WeChat Web 连接器 \{#test-wechat-web-connector}

就是这样。别忘了在 [注册和登录中启用社交连接器](/connectors/social-connectors/#enable-social-sign-in)。

启用 WeChat Web 连接器后，你可以再次登录你的应用，检查是否生效。
