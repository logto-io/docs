# 设置 GitLab 社交登录

官方 Logto GitLab 社交登录连接器。使用 GitLab 作为 OAuth 2.0 认证 (Authentication) 身份提供商 (IdP)。

<GuideTip />

## 入门指南 \{#get-started}

GitLab 连接器允许终端用户通过 GitLab OAuth 2.0 认证 (Authentication) 协议，使用他们自己的 GitLab 账户登录你的应用程序。
