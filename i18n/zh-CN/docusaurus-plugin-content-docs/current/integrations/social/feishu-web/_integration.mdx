## 开始使用飞书社交登录 \{#getting-started-with-feishu-social-sign-in}

飞书 Web 连接器专为桌面 Web 应用程序设计，并使用 OAuth 2.0 认证 (Authentication) 流程。

## 注册飞书开发者账号 \{#register-a-feishu-developer-account}

如果你还没有飞书开发者账号，请在 [飞书开放平台](https://open.feishu.cn/app) 注册。

## 创建应用程序 \{#create-an-application}

1. 在 [开发者后台](https://open.feishu.cn/app) 点击“创建自建应用”。
2. 填写应用名称、描述，选择图标，然后点击“创建”按钮。
3. 在左侧边栏点击“安全设置”，将“重定向 URL”填写为 `${logto_endpoint}/callback/${connector_id}`。对应的值可以在 Logto 控制台的飞书连接器详情页的 `Callback URI` 字段中找到。
4. 在“凭证与基础信息”中，你可以获取“App ID”和“App Secret”。

:::note
如为非企业内部自用，还需在“版本管理与发布”页面点击“创建版本”按钮。
“App ID”和“App Secret”仅在应用状态变为“已启用”后才会生效。
:::
