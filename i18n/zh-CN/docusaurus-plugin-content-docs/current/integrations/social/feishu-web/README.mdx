# 配置飞书社交登录

飞书：先进企业协作与管理平台，一站式无缝办公协作，团队上下对齐目标，全面激活组织和个人。

<GuideTip />

## 开始上手配置飞书登录 \{#getting-started-with-feishu-social-login}

飞书连接器是为桌面网页应用设计的。它采用了 OAuth 2.0 认证流程。

## 注册飞书开发者账号 \{#register-a-feishu-developer-account}

如果你还没有飞书开发者账号，请在 [飞书开放平台](https://open.feishu.cn/?lang=zh-CN) 注册。

## 创建应用 \{#create-an-application}

1. 在 [开发者后台](https://open.feishu.cn/app?lang=zh-CN) 页面中点击「创建企业自建应用」按钮。
2. 填写名称、应用描述，选择应用图标，最后点击「创建」按钮。
3. 左侧菜单「安全设置」，填写「重定向 URL」为 `${logto_endpoint}/callback/${connector_id}`。其对应值可参见 Logto Console 飞书连接器详情页中的 `Callback URI`。
4. 左侧「凭证与基础信息」中可以获取「App ID」和「App Secret」。

:::note
如非企业内部使用，还需在左侧「应用发布」中点击「创建版本」，待应用发布状态变更为「已启用」之后，「App ID」、「App Secret」才会生效。
:::
