## 入门指南 \{#get-started}

使用管理员账号登录 [企业微信 WebUI](https://work.weixin.qq.com/)，或在企业微信应用中点击 **管理企业**。

在“应用管理”标签页中，向下滚动页面并点击“创建应用”。

根据你的应用填写相应信息，然后创建。

现在我们得到了 Agent ID（不是 APPID）和 Secret。

### 网站信息 \{#website-info}

在此页面设置你需要的内容。通常包括：

- 允许的用户：_谁可以看到此应用_
- 应用主页：_你的应用主页。例如，`logto.io/demo-app`_

**重要提示**
此页面有三项与“开发者接口”相关的内容：

1. 网页授权及 JS-SDK；
2. 通过企业微信授权登录；
3. 企业可信 IP；

请根据企业微信的指引填写。

### 企业 ID（Corp ID）\{#corp-id}

如果你熟悉微信开发，你可能会注意到企业 ID（Corp ID）的用法与 APP ID 相同。

你可以在“我的企业”标签页底部找到企业 ID，看起来像 **ww\*\*\*\*** 。

## 配置连接器 \{#configure-the-connector}

现在我们已经有了 Agent ID、Secret 和企业 ID（Corp ID）。

让我们来填写连接器的表单。

你可以将 `Scope` 字段留空（可选），也可以填写 `snsapi_base` 或 `snsapi_privateinfo`。

**保存并完成**
