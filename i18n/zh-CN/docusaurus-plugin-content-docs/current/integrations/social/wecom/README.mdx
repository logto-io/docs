# 设置 WeCom (OAuth 2.0) 社交登录

WeCom（可能称为 WXwork）社交登录的自定义连接器。

<GuideTip />

## 开始

使用管理员账户登录 [WeCom WebUI](https://work.weixin.qq.com/) 或从 WeCom 应用中点击 **管理企业**。

在“应用管理”标签中，向下滚动页面并点击“创建应用”。

根据你的应用填写适当的信息。然后创建。

现在我们有了 Agent ID（不是 APPID）和 Secret。

### 网站信息

在此页面上设置你需要的内容。可能是这样的：

- 允许的用户：_谁可以看到此应用_
- 应用主页：_你的应用主页。例如，`logto.io/demo-app`_

**重要**
此页面上有三个关于“开发者接口”的项目。

1. 网页授权和 JS-SDK；
2. 通过 WeCom 授权登录；
3. 企业可信 IP；

根据 WeCom 的指南填写它们。

### Corp ID

如果你熟悉微信开发，你可能会注意到 Corp ID 的使用与 APP ID 相同。

你可以在“我的企业”标签页的底部找到 Corp ID。看起来像 **ww\*\*\*\*** 。

## 配置连接器

所以我们有了 Agent ID、Secret 和 Corp ID。

让我们完成连接器的表单。

你可以将 `Scope` 字段留空，因为它是可选的。或者，你可以填写 `snsapi_base` 或 `snsapi_privateinfo`。

**保存并完成**
