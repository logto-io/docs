# 设置 Hugging Face 社交登录

## 开始使用 \{#get-started}

Hugging Face 连接器允许终端用户通过 Hugging Face OAuth / OpenID Connect 流程，使用他们自己的 Hugging Face 账户登录你的应用程序。
