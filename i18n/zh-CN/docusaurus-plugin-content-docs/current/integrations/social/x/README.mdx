# 设置 X 社交登录

Logto 官方的 X（原 Twitter）社交登录连接器。

<GuideTip />

## 开始使用 \{#get-started}

X (Twitter) 连接器允许终端用户通过 X OAuth 2.0 认证 (Authentication) 协议，使用他们自己的 X (Twitter) 账户登录你的应用程序。
