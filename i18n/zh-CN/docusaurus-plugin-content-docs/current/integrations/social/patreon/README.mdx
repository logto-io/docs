# 设置 Patreon 社交登录

官方 Logto 连接器，用于 Patreon 社交登录。

<GuideTip />

## 入门指南 \{#get-started}

Patreon 连接器允许终端用户通过 Patreon OAuth 2.0 认证 (Authentication) 协议，使用他们自己的 Patreon 账户登录你的应用程序。
