## 注册支付宝开发者账号 \{#register-alipay-developer-account}

[注册一个支付宝开发者账号](https://certifyweb.alipay.com/certify/reg/guide#/) 如果你还没有的话。

## 创建和配置支付宝应用 \{#create-and-configure-alipay-app}

1. 使用你刚注册的账号登录 [支付宝控制台](https://open.alipay.com/)。
2. 在“我的应用” (我的应用) 面板中，进入“网页&移动应用” (网页&移动应用) 标签。
3. 点击“立即创建” (立即创建) 按钮开始配置你的应用。
4. 在“应用名称” (应用名称) 中按照命名规范命名你的应用，并上传你的“应用图标” (应用图标)，确保选择“网页应用” (网页应用) 作为“应用类型” (应用类型)。
5. 完成应用创建后，我们来到概览页面，在这里我们应该点击“添加能力” (添加能力) 来添加“第三方应用授权” (第三方应用授权)、“获取会员信息” (获取会员信息) 和 “App 支付宝登录” (App 支付宝登录)，然后才能启用支付宝登录。
6. 前往 [支付宝客户中心](https://b.alipay.com/index2.htm)，并使用支付宝开发者账号登录。点击顶部栏的“账号中心” (账号中心)，然后进入“APPID 绑定” (APPID 绑定)，入口可以在侧边栏底部找到。通过输入你在步骤 4 中创建的网页应用的 APPID 来“添加绑定” (添加绑定)。
7. 点击“App 支付宝登录”的“签约”按钮，并按照指南完成签约过程。完成此步骤后，你应该会发现你在步骤 5 中刚刚添加的能力生效。
8. 返回支付宝开放平台控制台页面，你可以在“开发信息” (开发信息) 部分找到“接口加签方式（密钥/证书）” (接口加签方式（密钥/证书）)。点击“设置” (设置) 按钮，你会进入一个设置签名方式的页面。“公钥” (公钥) 是首选的签名模式，并在文本输入框中填写你生成的公钥文件的内容。
9. 通过点击支付宝控制台页面底部的“设置” (设置) 按钮来设置“授权回调地址” (授权回调地址)。`${your_logto_origin}/callback/${connector_id}` 是 Logto 核心中使用的默认回调 URI。`connector_id` 可以在 Logto 管理控制台连接器详情页面的顶部栏找到。
10. 完成所有这些步骤后，返回支付宝控制台页面的右上角，点击“提交审核” (提交审核)。一旦审核通过，你就可以顺利进行支付宝登录流程。

:::note
你可以在本地机器上使用 _openssl_ 通过在终端中执行以下代码片段来生成密钥对。

```bash
openssl genrsa -out private.pem 2048
openssl rsa -in private.pem -outform PEM -pubout -out public.pem
```

在支付宝应用设置网站上填写公钥时，你需要移除 `public.pem` 的头部和尾部，删除所有换行符，并将其余内容粘贴到“公钥”文本输入框中。

:::

## 设置 Logto Alipay Web 连接器设置 \{#set-up-the-logto-alipay-web-connector-settings}

1. 在 [支付宝控制台工作区](https://open.alipay.com/dev/workspace) 中，进入“我的应用” (我的应用) 面板并点击“网页&移动应用” (网页&移动应用) 标签，你可以找到所有应用的 APPID。
2. 在前一部分的步骤 7 中，你已经生成了一个包含私钥和公钥的密钥对。
3. 填写 Logto 连接器设置：
   - 使用步骤 1 中获得的 APPID 填写 `appId` 字段。
   - 使用步骤 2 中提到的私钥文件的内容填写 `privateKey` 字段。请确保使用 '\n' 替换所有换行符，并且不要移除私钥文件中的头部和尾部。
   - 由于我们在“创建和配置支付宝应用”步骤 7 中选择了 `Public key` 签名模式，填写 `signType` 字段为 'RSA2'。
   - 使用 'gbk' 或 'utf8' 填写 `charset` 字段。你可以将此字段留空，因为它是可选的。默认值设置为 'utf8'。
   - 使用 'auth_base' 或 'auth_user' 填写 `scope` 字段。你可以将此字段留空，因为它是可选的。默认值设置为 'auth_user'。你可以查看不同值之间的[区别](https://opendocs.alipay.com/fw/api/105942)。

### 配置类型 \{#config-types}

| 名称       | 类型               | 枚举值                       |
| ---------- | ------------------ | ---------------------------- |
| appId      | string             | N/A                          |
| privateKey | string             | N/A                          |
| signType   | enum string        | 'RSA' \| 'RSA2'              |
| charset    | enum string (可选) | 'gbk' \| 'utf8' \| undefined |
| scope      | enum string (可选) | 'auth_user' \| 'auth_base'   |
