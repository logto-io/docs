# 设置支付宝（原生）社交登录

Logto 官方为移动端原生应用提供的支付宝社交登录连接器。

<GuideTip />

## 入门指南 \{#get-started}

支付宝原生连接器与 Logto SDK 在移动平台上紧密配合。它利用支付宝的 OAuth 2.0 认证 (Authentication) 流程，让支付宝用户无需繁琐注册流程即可使用其公开支付宝用户资料登录其他应用。

<Integration />

### 测试支付宝原生连接器 \{#test-alipay-native-connector}

就是这样。别忘了<MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">在登录体验中启用社交连接器</MainSiteUrl>。

启用支付宝原生连接器后，你可以构建并运行你的应用，查看其是否正常工作。
