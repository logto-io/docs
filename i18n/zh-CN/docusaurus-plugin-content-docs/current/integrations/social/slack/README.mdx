# 设置 Slack 社交登录

Logto 官方的 Slack 社交登录连接器。

## 开始使用 \{#get-started}

Slack 连接器允许终端用户通过 Slack OAuth 2.0 认证 (Authentication) 协议，使用他们自己的 Slack 账户登录你的应用程序。
