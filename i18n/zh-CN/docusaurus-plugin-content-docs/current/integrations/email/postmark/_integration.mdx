## 注册 Postmark 账户 \{#register-postmark-account}

在 [Postmark 网站](https://postmark.com/)创建一个新账户。如果你已经有账户，可以跳过此步骤。

## 配置你的连接器 \{#configure-your-connector}

在 Postmark 中找到你的服务器设置下的 Server Token，并填写到 `serverToken` 字段中。

在 `fromEmail` 字段中填写发件人的 _From Address_。

为了启用完整的用户流程，需要使用 `Register`、`SignIn`、`ForgotPassword` 和 `Generic` 的模板。

以下是 Postmark 连接器模板 JSON 的示例。

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## 测试 Postmark 邮件连接器 \{#test-postmark-email-connector}

你可以输入一个电子邮件地址并点击“发送”以查看设置是否可以在“保存并完成”之前正常工作。

就是这样。别忘了[在登录体验中启用连接器](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)。

## 配置类型 \{#config-types}

| 名称        | 类型   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| 模板属性      | 类型        | 枚举值                                                  |
| ------------- | ----------- | ------------------------------------------------------- |
| usageType     | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias | string      | N/A                                                     |
