# 使用 Postmark 邮件服务设置电子邮件验证

Logto 用于 Postmark 邮件服务的连接器 (Connector)。

<GuideTip />

## 入门 \{#get-started}

Postmark 是一个用于事务性和营销邮件的邮件平台。我们可以利用它的邮件发送功能来发送 _验证码_。

## 注册 Postmark 账号 \{#register-postmark-account}

在 [Postmark 官网](https://postmark.com/) 创建一个新账号。如果你已经有账号，可以跳过此步骤。

## 配置你的连接器 (Connector) \{#configure-your-connector}

在 `serverToken` 字段中填写你在 Postmark 服务器设置下找到的 Server Token。

在 `fromEmail` 字段中填写发件人的 _From Address_。

为了启用完整的用户流程，需要包含 usageType 为 `Register`、`SignIn`、`ForgotPassword` 和 `Generic` 的模板。

以下是 Postmark 连接器 (Connector) 模板 JSON 的示例。

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## 测试 Postmark 邮件连接器 (Connector) \{#test-postmark-email-connector}

你可以输入一个邮箱地址并点击“发送”，在“保存并完成”之前测试设置是否可用。

就是这样。别忘了 <MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">在注册和登录中启用连接器 (Connector)</MainSiteUrl>。

## 配置类型 \{#config-types}

| 名称        | 类型   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| 模板属性      | 类型        | 枚举值                                                  |
| ------------- | ----------- | ------------------------------------------------------- |
| usageType     | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias | string      | N/A                                                     |
