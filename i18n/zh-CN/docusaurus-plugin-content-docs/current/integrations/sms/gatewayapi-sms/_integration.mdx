### 创建 GatewayAPI 账户 \{#create-an-gatewayapi-account}

前往 [GatewayAPI 官网](https://www.gatewayapi.com/)，如果你还没有 GatewayAPI 账户，请注册一个。

### 启用账户 \{#enable-account}

在使用短信服务之前，你可能需要先启用你的账户。你可以联系 GatewayAPI 客服来启用你的账户。

### 获取 API token \{#get-api-token}

在 GatewayAPI 控制台进入 API Keys 页面，找到 API token 或创建一个新的 API token。

## 在 Logto 中设置 \{#set-up-in-logto}

1. **Endpoint**：如果你的 GatewayAPI 账户在欧盟地区，应使用 https://gatewayapi.com/rest/mtsms 作为 endpoint。如果你的 GatewayAPI 账户在美国地区，也应使用 https://gatewayapi.com/rest/mtsms 作为 endpoint。
2. **API Token**：你在上一步创建的 API token。
3. **Sender**：你希望用于发送短信的发送方。
4. **Templates**：你希望用于发送短信的模板，可以使用默认模板，也可以根据需要进行修改。
