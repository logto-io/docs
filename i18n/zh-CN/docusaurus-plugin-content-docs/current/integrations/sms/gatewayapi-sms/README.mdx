# 使用 GatewayAPI SMS 设置短信验证

Logto 的 GatewayAPI SMS 官方连接器。

<GuideTip />

## 开始使用 \{#get-started}

GatewayAPI 是一家欧洲的云服务提供商，提供包括短信（短消息服务）在内的多种云服务。GatewayAPI SMS 连接器是由 Logto 团队提供的一个插件，借助该插件，Logto 终端用户可以通过短信验证码注册和登录他们的 Logto 账户。

## 在 GatewayAPI 中设置 \{#set-up-in-gatewayapi}

### 创建 GatewayAPI 账户 \{#create-an-gatewayapi-account}

如果你还没有 GatewayAPI 账户，请访问 [GatewayAPI 网站](https://www.gatewayapi.com/) 并注册你的 GatewayAPI 账户。

### 启用账户 \{#enable-account}

在使用短信服务之前，你可能需要启用你的账户。你可以联系 GatewayAPI 客户服务来启用你的账户。

### 获取 API 令牌 \{#get-api-token}

从 GatewayAPI 控制台的 API Keys 页面获取 API 令牌或创建一个新的 API 令牌。

## 在 Logto 中设置 \{#set-up-in-logto}

1. **Endpoint**：如果你的 GatewayAPI 账户在欧盟地区，你应该使用端点 https://gatewayapi.com/rest/mtsms。如果你的 GatewayAPI 账户在美国地区，你应该使用端点 https://gatewayapi.com/rest/mtsms。
2. **API Token**：你在上一步创建的 API 令牌。
3. **Sender**：你想要用来发送短信的发送者。
4. **Templates**：你想要用来发送短信的模板，你可以使用默认模板或根据需要进行修改。
