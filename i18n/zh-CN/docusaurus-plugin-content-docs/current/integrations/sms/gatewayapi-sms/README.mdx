# 使用 GatewayAPI SMS 设置短信验证

官方 Logto GatewayAPI SMS 连接器。

<GuideTip />

## 入门指南 \{#get-started}

GatewayAPI 是欧洲的一家云服务提供商，提供多种云服务，包括短信（SMS，短消息服务）。GatewayAPI SMS 连接器是 Logto 团队提供的插件，用于调用 GatewayAPI SMS 服务，借助该插件，Logto 终端用户可以通过短信验证码注册和登录他们的 Logto 账户。
