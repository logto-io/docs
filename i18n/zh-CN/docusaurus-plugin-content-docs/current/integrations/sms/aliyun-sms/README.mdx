# 使用阿里云短信服务设置短信验证

Logto 官方的阿里云短信服务连接器。

<GuideTip />

## 入门指南 \{#get-started}

阿里云是亚洲主要的云服务提供商，提供多种云服务，包括短信服务（SMS）。阿里云短信连接器是 Logto 团队提供的插件，用于调用阿里云短信服务，借助该插件，Logto 终端用户可以通过短信验证码注册和登录他们的 Logto 账户。
