# 使用 Vonage SMS 设置短信验证

Logto 的官方 Vonage SMS 连接器。

<GuideTip />

## 开始使用 \{#get-started}

Vonage 是一家全球通信提供商，提供强大的基于云的通信服务，包括短信 (SMS)。Vonage SMS 连接器是由 Logto 团队提供的一个插件，使 Logto 终端用户可以通过短信验证码注册和登录他们的 Logto 账户。

## 在 Vonage 中设置 \{#set-up-in-vonage}

:::tip
如果你已经完成了这些步骤，可以跳过此步骤。
:::

要使用此连接器，你需要在 Vonage [注册一个账户](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account)。这将为你提供一个 API key 和 secret，你可以通过此连接器访问 API。

一旦你有了账户，你可以在 Vonage API Dashboard 的顶部找到你的 API key 和 API secret。

你可能还需要 [租用一个虚拟号码](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) 来发送短信。

有关更多信息，请参阅 [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview)。

## 在 Logto 中设置 \{#set-up-in-logto}

1. **API Key**：你的 Vonage API key。
2. **API Secret**：你的 Vonage API secret。
3. **品牌名称**：你想用来发送短信的品牌名称，这也被称为 `from` 字段，更多信息请参阅 [发送者身份](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id)。
4. **模板**：你想用来发送短信的模板，你可以使用默认模板或根据需要进行修改。
