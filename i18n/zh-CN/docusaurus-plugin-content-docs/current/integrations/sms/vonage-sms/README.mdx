# 使用 Vonage SMS 设置短信验证 (Set up SMS verification with Vonage SMS)

Vonage SMS 的官方 Logto 连接器。

<GuideTip />

## 入门指南 (Get started) \{#get-started}

Vonage 是一家全球通信服务提供商，提供强大的基于云的通信服务，包括短信（SMS，短消息服务）。Vonage SMS 连接器是由 Logto 团队提供的插件，使 Logto 终端用户能够通过短信验证码注册和登录他们的 Logto 账户。
