# 云片短信连接器

Logto 官方提供的云片短信服务连接器。

<GuideTip />

## 入门指南 \{#get-started}

云片是一家通信服务提供商，提供包括短信在内的多种服务。云片短信连接器是 Logto 团队提供的插件，用于集成云片的短信服务，使 Logto 终端用户能够通过短信验证码注册和登录。

## 在云片控制台设置短信服务 \{#set-up-sms-service-in-yunpian-console}

### 创建云片账号 \{#create-a-yunpian-account}

访问[云片官网](https://www.yunpian.com/)注册账号并完成实名认证。

### 获取 API KEY \{#get-api-key}

1. 登录云片控制台
2. 进入“账户设置” -> “子账户管理”
3. 找到并复制 API KEY

### 配置短信模板 \{#configure-sms-templates}

1. 在云片控制台，进入“国内短信” -> “签名报备”
2. 创建并提交签名，等待运营商审核
3. 进入“国内短信” -> “模板报备”，选择“验证码”
4. 创建验证码模板，确保包含 `#code#` 变量（也可以使用“通用模板”加快审核）
5. 等待模板审核通过
6. 如需发送国际短信，重复上述步骤，但选择“国际短信” -> “模板报备”

## 在 Logto 中配置 \{#configure-in-logto}

1. 在 Logto 控制台，进入“连接器”
2. 找到并点击“云片短信服务”
3. 填写配置表单：
   - API KEY：从云片获取的 API KEY
   - 短信模板：根据使用场景配置模板，确保与云片审核通过的模板内容完全一致

## 注意事项 \{#notes}

1. 短信模板内容必须与云片审核通过的模板完全一致
2. 云片模板中的验证码变量占位符为 `#code#`，而连接器配置中为 `{{code}}`
3. 云片会根据 API KEY 自动添加默认签名，无需在模板内容中包含签名
4. 建议在正式使用前先进行配置测试
