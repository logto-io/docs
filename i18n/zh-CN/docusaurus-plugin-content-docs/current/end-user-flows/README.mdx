# 终端用户流程

终端用户流程涵盖了用户交互的所有验证过程，分类如下：

- **登录体验**：涉及用户登录、注册和密码重置。 Logto 提供了在 Console 中配置的开箱即用体验，你可以通过 Experience API 提供自己的用户界面。
- **账户体验**：涉及账户设置、用户资料和安全验证。使用 Account API 或 Management API 实现。
- **组织体验**：适用于多租户产品，如 B2B 或 SaaS 服务。涉及组织创建、成员邀请和管理。使用 Management API 将其集成到你的应用中。
