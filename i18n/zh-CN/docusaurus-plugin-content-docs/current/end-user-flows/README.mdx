# 终端用户流程

终端用户流程涵盖了所有用户交互的验证过程，分为以下几类：

- **认证 (Authentication) 流程**：涉及用户登录、注册和重置密码。Logto 在 <CloudLink to="/sign-in-experience">控制台 > 登录体验</CloudLink> 中提供了开箱即用的体验，你也可以通过 [Experience API](https://openapi.logto.io/group/endpoint-experience) [自定义你的 UI](/customization/bring-your-ui)。
- **账户流程**：涉及账户设置、用户资料和安全验证。可通过 [Account API](https://openapi.logto.io/group/endpoint-account-center) 或 [Management API](https://openapi.logto.io/operation/operation-getuser) 实现。
- **组织 (Organization) 流程**：适用于多租户产品，如 B2B 或 SaaS 服务。涉及组织创建、成员邀请和管理。可通过 [Management API](https://openapi.logto.io/operation/operation-listorganizations) 集成到你的应用中。
