# 组织 (Organization) 体验

[组织 (Organization)](/organizations) 体验指的是你的企业客户及其员工所使用的用户界面和流程，尤其是在[多租户应用](https://auth.wiki/multi-tenancy)中。Logto 提供了多种指南，帮助你通过 Logto Management API 快速将这种体验集成到你的应用中。
