# 组织 (Organization) 体验

[组织 (Organization)](/organizations) 体验指的是你的商业客户及其员工使用的用户界面和流程，特别是在[多租户应用程序](https://auth.wiki/multi-tenancy)中。Logto 提供了多个指南，帮助你快速使用 Logto Management API 将这种体验集成到你的应用中。
