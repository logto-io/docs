# 组织切换器

在组织体验中，用户可以在不同组织之间切换。每次切换时，应用会获取新的组织令牌 (Organization token) 以加载相关信息。

## 如何实现 \{#how-to-implement-it}

1. 创建一个下拉菜单或类似的 UI 组件，列出用户可用的所有组织 (Organizations)。
2. 当用户选择不同的组织 (Organization) 时，获取其 ID，并调用 SDK 方法 `getOrganizationToken(organizationId)` 来获取对应的组织令牌 (Organization token)。（SDK 会为你处理令牌缓存。）
3. 存储所选组织 (Organization) 的 ID（在 URL 或客户端存储中），并始终使用它来获取与组织相关资源配对的组织令牌 (Organization token)。
