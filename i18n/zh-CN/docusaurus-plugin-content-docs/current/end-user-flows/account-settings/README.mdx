# 账户设置

日志 (Logto) 提供了两组账户设置 API，允许用户管理存储在日志 (Logto) 中的账户和个人资料。

## 用户管理 API

日志 (Logto) 管理 API 提供了最全面的功能来管理用户及其个人资料数据。你可以使用这些 API 实现一个自托管的用户管理系统或终端用户账户设置页面。

了解更多关于如何使用 [用户管理 API](/end-user-flows/account-settings/by-management-api) 构建你自己的账户设置页面。

## 账户 API

除了管理 API，日志 (Logto) 还提供了一组终端用户账户 API，允许终端用户直接管理他们的账户设置和个人资料数据。与管理 API 不同，账户 API 设计用于终端用户上下文中，具有详细的用户权限控制。你可以在客户端应用程序中直接使用账户 API 来构建一个用户友好的账户设置页面。

了解更多关于如何使用 [账户 API](/end-user-flows/account-settings/by-account-api) 构建你自己的账户设置页面。
