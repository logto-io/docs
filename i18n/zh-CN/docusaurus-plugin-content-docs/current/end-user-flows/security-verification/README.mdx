# 安全验证

当经过认证的用户尝试执行敏感或高风险操作时，例如更改密码、进行支付或访问敏感信息（如工资单或银行账户详细信息），额外的安全措施是必不可少的。在银行、医疗和政府服务等敏感应用中，这一流程尤为关键。

这个过程被称为 _安全验证_，要求用户重新验证其身份，以确认他们是授权的账户持有人。通过实施安全验证，系统加强了对未授权访问的保护，为高风险操作增加了关键的安全层，有助于保护敏感数据。

了解如何使用 Logto [Management APIs](/end-user-flows/security-verification/by-management-api) 将安全验证过程添加到你的应用程序中。
