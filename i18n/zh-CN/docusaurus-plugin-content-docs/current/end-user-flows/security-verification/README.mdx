# 安全验证

当已认证 (Authentication) 用户尝试执行敏感或高风险操作时——例如更改密码、进行支付，或访问诸如工资单、银行账户等敏感信息——就需要额外的安全措施。此流程在银行、医疗和政府服务等敏感应用中尤为关键。

这个过程被称为 _安全验证_，要求用户重新验证其身份，以确认他们是被授权的账户持有者。通过实施安全验证，系统能够加强对未授权访问的防护，为高风险操作增加关键的安全层，有助于保护敏感数据。
