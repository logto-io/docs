# 个性化定制

通过自定义每一个细节，让 Logto 成为你的专属——从品牌和用户界面到语言和域名设置。

## 预设 UI \{#prebuilt-ui}

从我们预设的 UI 组件开始，自定义关键品牌元素。你可以创建：

- 整个平台统一的认证 (Authentication) 体验。
- 针对特定应用或组织 (Organizations) 的专属登录页面。

<DocCardList
  items={[
    {
      type: 'link',
      label: '匹配你的品牌',
      href: '/customization/match-your-brand',
      description: '了解如何自定义基础品牌元素，打造即用型登录体验和账户中心。',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## 内容与语言 \{#content--languages}

Logto 默认支持 15 种语言。你可以扩展语言支持，或修改现有翻译以满足你的需求。

<DocCardList
  items={[
    {
      type: 'link',
      label: '语言与内容',
      href: '/customization/localized-languages',
      description: '了解如何为你的用户添加新语言并自定义内容。',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## 自定义域名 \{#custom-domain}

通过为认证 (Authentication) URL 使用你的专属域名，强化品牌形象。用你的品牌域名替换默认的 Logto 域名。

<DocCardList
  items={[
    {
      type: 'link',
      label: '自定义域名',
      href: '/logto-cloud/custom-domain',

      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>
