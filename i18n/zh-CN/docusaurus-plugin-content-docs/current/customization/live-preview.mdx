# 实时预览

你可以使用 **实时预览** 功能方便地测试你的登录体验配置。

导航到 <CloudLink to="/sign-in-experience">控制台 > 登录体验</CloudLink>，点击右上角的“实时预览”按钮，将会在新的浏览器标签页中打开一个 Logto 功能的完整演示。此演示展示了我们在登录和注册流程中的体验设计方法。它具有高度的可定制性，可以无缝集成到你的应用程序中。

:::note

**注意**：在此预览中，你应首先创建一个账户以测试体验。你可以稍后在 <CloudLink to="/user-management">控制台 > 用户管理</CloudLink> 中管理此账户。

:::

如果你尝试使用你创建的用户账户登录，并且过程顺利无误，你将被重定向到一个成功页面，该页面显示你的标识符和 ID。
