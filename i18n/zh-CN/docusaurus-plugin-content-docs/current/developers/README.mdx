# 开发者

Logto 是一个基于 [OAuth 2](https://auth.wiki/oauth-2.0) 和 [OIDC](https://auth.wiki/openid-connect) 协议的[身份和访问管理 (IAM)](https://auth.wiki/iam) 服务。像 Logto 这样的 IAM 服务通常作为其他网络服务的基础；这些网络服务中的各种授权 (Authorization) 状态直接受到 Logto 的影响。

为了给我们的用户提供便利，Logto 提供了一系列常用的开发者功能。
