# 审计日志的事件类型

你可以在 <CloudLink to="/audit-logs">Logto 控制台 > 审计日志</CloudLink> 中筛选事件类型。

:::note

Logto 现已支持通过 [体验 (Experience) API](https://openapi.logto.io/group/endpoint-experience) 获取与终端用户交互相关的日志。

[管理 (Management) API](/integrate-logto/interact-with-management-api) 和 [账户 (Account) API](https://openapi.logto.io/group/endpoint-account-center) 的审计日志即将上线。

如果你有相关需求，欢迎 [联系我们](https://logto.io/contact)。

:::
