要使用 Gin 初始化一个新的 Go 项目，你可以按照以下步骤操作：

```bash
go mod init your-api-name
go get github.com/gin-gonic/gin
```

然后，创建一个基础的 Gin 服务器设置：

```go title="main.go"
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()

    r.Run(":3000") // 监听并服务于 0.0.0.0:3000
}
```

:::note
有关如何设置路由、中间件及其他功能的更多信息，请参考 Gin 官方文档。
:::
