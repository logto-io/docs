要使用 Fiber 初始化一个新的 Go 项目，你可以按照以下步骤操作：

```bash
go mod init your-api-name
go get github.com/gofiber/fiber/v2
```

然后，创建一个基础的 Fiber 服务器设置：

```go title="main.go"
package main

import (
    "log"

    "github.com/gofiber/fiber/v2"
)

func main() {
    app := fiber.New()

    log.Fatal(app.Listen(":3000"))
}
```

:::note
更多关于如何设置路由、中间件及其他功能的信息，请参考 Fiber 文档。
:::
