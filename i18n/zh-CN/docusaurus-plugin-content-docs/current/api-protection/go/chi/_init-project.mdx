要使用 Chi 初始化一个新的 Go 项目，你可以按照以下步骤操作：

```bash
go mod init your-api-name
go get github.com/go-chi/chi/v5
```

然后，创建一个基础的 Chi 服务器设置：

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
更多关于如何设置路由、中间件及其他功能的信息，请参考 Chi 官方文档。
:::
