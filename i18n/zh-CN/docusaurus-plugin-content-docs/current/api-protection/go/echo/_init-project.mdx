要使用 Echo 初始化一个新的 Go 项目，你可以按照以下步骤操作：

```bash
go mod init your-api-name
go get github.com/labstack/echo/v4
```

然后，创建一个基础的 Echo 服务器设置：

```go title="main.go"
package main

import (
    "github.com/labstack/echo/v4"
)

func main() {
    e := echo.New()

    e.Logger.Fatal(e.Start(":3000"))
}
```

:::note
有关如何设置路由、中间件及其他功能的更多细节，请参考 Echo 官方文档。
:::
