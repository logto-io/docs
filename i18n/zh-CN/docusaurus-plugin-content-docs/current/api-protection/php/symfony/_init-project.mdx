要初始化一个用于 API 开发的新 Symfony 项目，可以使用 Symfony CLI 或 Composer：

使用 Symfony CLI（推荐）：

```bash
symfony new your-api-name --webapp
cd your-api-name
```

或者使用 Composer：

```bash
composer create-project symfony/skeleton your-api-name
cd your-api-name
composer require webapp
```

安装用于 API 开发的其他依赖包：

```bash
composer require symfony/security-bundle
composer require symfony/serializer
composer require doctrine/annotations
```

启动开发服务器：

```bash
symfony serve
```

或者使用 PHP 内置服务器：

```bash
php -S localhost:8000 -t public/
```

这样就创建了一个基础的 Symfony 项目。接下来配置框架以支持 API 开发：

```yaml title="config/packages/framework.yaml"
framework:
  secret: '%env(APP_SECRET)%'
  serializer:
    enabled: true
  property_access:
    enabled: true
```

:::note
更多关于如何设置控制器、服务和其他功能的信息，请参考 Symfony 官方文档。
:::
