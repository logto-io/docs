要初始化一个新的 Sinatra 项目，创建一个目录并设置基本结构：

```bash
mkdir your-api-name
cd your-api-name
```

创建一个 Gemfile：

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'sinatra'
```

安装依赖：

```bash
bundle install
```

创建一个基础的 Sinatra 应用：

```ruby title="app.rb"
require 'sinatra'
require 'json'

get '/' do
  content_type :json
  { message: 'Hello from Sinatra API' }.to_json
end
```

启动开发服务器：

```bash
ruby app.rb
```

:::note
有关如何设置路由、中间件和其他功能的更多信息，请参考 Sinatra 文档。
:::
