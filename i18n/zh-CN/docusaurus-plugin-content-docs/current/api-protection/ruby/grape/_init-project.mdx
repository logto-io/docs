要初始化一个新的 Grape API 项目，创建一个目录并设置基本结构：

```bash
mkdir your-api-name
cd your-api-name
```

创建一个 Gemfile：

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

安装依赖：

```bash
bundle install
```

创建一个基础的 Grape API：

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

创建一个 config.ru 文件：

```ruby title="config.ru"
require_relative 'api'

run API
```

启动开发服务器：

```bash
bundle exec rackup
```

:::note
更多关于如何设置资源、参数校验以及其他功能的详细信息，请参考 Grape 文档。
:::
