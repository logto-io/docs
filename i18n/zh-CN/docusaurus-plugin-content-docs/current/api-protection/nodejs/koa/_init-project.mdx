要初始化一个新的 Node.js 项目并使用 Koa，你可以按照以下步骤操作：

```bash
npm init -y
npm install koa @koa/router
```

如果你使用 TypeScript，请记得相应地配置你的 TypeScript 环境。

然后，创建一个基础的 Koa 服务器设置：

```ts title="app.ts"
import Koa from 'koa';

const app = new Koa();

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

:::note
更多关于如何设置路由、中间件及其他功能的信息，请参考 Koa 官方文档。
:::
