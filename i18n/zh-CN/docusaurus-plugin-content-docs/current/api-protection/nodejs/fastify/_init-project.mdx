要使用 Fastify 初始化一个新的 Node.js 项目，你可以按照以下步骤操作：

```bash
npm init -y
npm install fastify
```

如果你使用 TypeScript，请记得相应地配置你的 TypeScript 环境。

然后，创建一个基础的 Fastify 服务器设置：

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Server running on http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
更多关于如何设置路由、插件和其他功能的详细信息，请参考 Fastify 官方文档。
:::
