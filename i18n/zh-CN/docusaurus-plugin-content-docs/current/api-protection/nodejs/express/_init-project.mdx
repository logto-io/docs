要初始化一个带有 Express 的全新 Node.js 项目，你可以按照以下步骤操作：

```bash
npm init -y
npm install express
```

如果你使用 TypeScript，请记得相应地配置你的 TypeScript 环境。

然后，创建一个基础的 Express 服务器设置：

```ts title="app.ts"
import express from 'express';

const app = express();

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

:::note
有关如何设置路由、中间件及其他功能的更多信息，请参考 Express 官方文档。
:::
