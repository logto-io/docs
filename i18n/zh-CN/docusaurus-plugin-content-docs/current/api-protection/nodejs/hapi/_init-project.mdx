要使用 Hapi 初始化一个新的 Node.js 项目，你可以按照以下步骤操作：

```bash
npm init -y
npm install @hapi/hapi
```

如果你使用 TypeScript，请记得相应地配置你的 TypeScript 环境。

然后，创建一个基础的 Hapi 服务器设置：

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Server running on http://localhost:3000');
```

:::note
更多关于如何设置路由、插件和其他功能的信息，请参考 Hapi 官方文档。
:::
