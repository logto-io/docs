要初始化一个新的 Flask 项目，创建一个目录并设置基本结构：

```bash
mkdir your-api-name
cd your-api-name
```

创建一个依赖文件：

```txt title="requirements.txt"
Flask
```

安装依赖：

```bash
pip install -r requirements.txt
```

创建一个基础的 Flask 应用：

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

启动开发服务器：

```bash
python app.py
```

:::note
更多关于如何设置路由、蓝图以及其他功能的信息，请参考 Flask 官方文档。
:::
