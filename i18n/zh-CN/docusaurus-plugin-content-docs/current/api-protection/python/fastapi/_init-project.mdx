要初始化一个新的 FastAPI 项目，创建一个目录并设置基本结构：

```bash
mkdir your-api-name
cd your-api-name
```

创建一个依赖文件：

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

安装依赖：

```bash
pip install -r requirements.txt
```

创建一个基础的 FastAPI 应用：

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

启动开发服务器：

```bash
uvicorn main:app --reload
```

:::note
更多关于如何设置路径操作、依赖注入以及其他功能的详细信息，请参考 FastAPI 文档。
:::
