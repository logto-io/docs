# API 保护

了解如何实现[基于角色的访问控制 (RBAC)](/authorization/role-based-access-control) 和 [JSON Web Token (JWT)](https://auth.wiki/jwt) 校验，以保护你的 API 端点免受未授权访问。

我们的指南涵盖中间件设置、权限模型，以及多种编程语言和框架下的实用示例。
