M2M 应用通过向令牌端点发送 `POST` 请求，并在 HTTP 请求实体主体中以 `application/x-www-form-urlencoded` 格式添加以下参数来获取访问令牌 (Access token)：

- **grant_type**：必须设置为 `client_credentials`
- **resource**：你想要访问的资源
- **scope**：访问请求的权限 (Scope)

你还需要在请求头中包含 M2M 应用的凭据，以便令牌端点对你的 M2M 应用进行认证 (Authentication)。

这可以通过在请求的 `Authorization` 头中以 [Basic 认证 (Authentication)](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Authorization#basic_authentication) 形式包含应用凭据来实现：将 `{App ID}:{App Secret}` 进行 base64 编码，并将其作为 `Basic` 前缀后的值。
