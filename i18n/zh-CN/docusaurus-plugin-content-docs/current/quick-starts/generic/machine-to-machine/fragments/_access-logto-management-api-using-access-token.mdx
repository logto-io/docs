<TabItem value="with-@logto-api-sdk" label="Node.js 使用 `@logto/api` SDK">

使用 `@logto/api` SDK，你可以直接与任何 Management API 交互，参考下面的代码片段。访问令牌 (Access token) 会在内部缓存，并在需要时自动刷新。

```js
import { createManagementApi } from '@logto/api/management';

const { apiClient } = createManagementApi('your-tenant-id', {
  clientId: 'your-client-id',
  clientSecret: 'your-client-secret',
});

const response = await apiClient.GET('/api/applications');
console.log(response.data);
```

</TabItem>

<TabItem value="nodejs" label="Node.js">

<AccessTokenUsage />

```js
const logtoEndpoint = 'https://your.logto.endpoint'; // 替换为你的 Logto endpoint
const accessToken = 'eyJhb...2g'; // 访问令牌 (Access token)

const fetchLogtoApplications = async () => {
  return await fetch(`${logtoEndpoint}/api/applications`, {
    method: 'GET',
    headers: {
      Authorization: `Bearer ${accessToken}`,
    },
  });
};
```

</TabItem>

<TabItem value="curl" label="cURL">

<AccessTokenUsage />

```bash
curl --location \
  --request GET 'https://your.logto.endpoint/api/applications' \
  --header 'Authorization: Bearer eyJhbG...2g'
```

请记得将实际值替换为你自己的。`Bearer` 后面的值应为你收到的访问令牌 (Access token)（JWT）。
