# 为你的 Passport.js 应用添加认证 (Authentication)

本指南将向你展示如何将 Logto 集成到你的应用中，使用 [Passport.js](https://www.passportjs.org/) 和 OIDC 策略。

<GuideTip />

## 先决条件

- 一个 [Logto Cloud](https://cloud.logto.io) 账户或一个 [自托管 Logto](/introduction/set-up-logto-oss)。
- 一个 Logto 传统应用已创建。
- 一个配置了会话的 express 项目。查看 [Express.js 网站](https://expressjs.com/)。
