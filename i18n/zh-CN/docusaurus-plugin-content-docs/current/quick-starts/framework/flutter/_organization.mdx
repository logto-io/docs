### 为组织获取访问令牌

就像 API 资源一样，你也可以请求组织的访问令牌。这在你需要访问使用组织权限而不是 API 资源权限定义的资源时非常有用。

  organizationScope="LogtoUserScope.Organizations"
  configOrganizationCode={<ConfigOrganizationCode />}
