### 模块 \{#modules}

`logto_dart_sdk` 包含两个主要模块：

- **logto_core.dart** 这个核心模块提供了 Logto SDK 的基本功能和接口。

- **logto_client.dart** 这个客户端模块提供了一个高级的 Logto 客户端类，用于与 Logto 服务器交互。
