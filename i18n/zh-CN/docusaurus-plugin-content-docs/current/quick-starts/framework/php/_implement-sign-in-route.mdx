import CodeBlock from '@theme/CodeBlock';

import {
  defaultRedirectUri,
  defaultBaseUrl,
} from '../../fragments/_web-configure-redirect-uris.mdx';

在你的 Web 应用中，添加一个路由以正确处理用户的登录请求。例如：

<CodeBlock language="php">
  {`Route::get('/sign-in', function () {
  return redirect($client->signIn('${defaultRedirectUri}'));
});`}
</CodeBlock>

将 <code>{defaultRedirectUri}</code> 替换为你在 Logto Console 中为此应用设置的回调 URL。

如果你想在第一个屏幕显示注册页面，可以将 `interactionMode` 设置为 `signUp`：

<CodeBlock language="php">
  {`Route::get('/sign-in', function () {
  return redirect($client->signIn('${defaultRedirectUri}', InteractionMode::signUp));
});`}
</CodeBlock>

现在，每当你的用户访问 <code>{defaultBaseUrl}sign-in</code> 时，它将启动一个新的登录尝试并将用户重定向到 Logto 登录页面。

> **注意**
> 创建一个登录路由并不是启动登录尝试的唯一方法。你始终可以使用 `signIn` 方法获取登录 URL 并将用户重定向到该 URL。
