### 将访问令牌 (Access token) 附加到请求头

将令牌放入 HTTP 头的 `Authorization` 字段中，使用 Bearer 格式（`Bearer YOUR_TOKEN`），然后你就可以开始了。

:::note
Bearer 令牌的集成流程可能会根据你使用的框架或请求者而有所不同。选择你自己的方式来应用请求 `Authorization` 头。
:::
