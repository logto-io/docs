首先，让我们配置 **Logto 重定向 URI**。将以下 URI 添加到 Logto 应用详情页面的“重定向 URI”列表中：

<CodeBlock>{`http://${defaultBaseUrl}/Callback`}</CodeBlock>

要配置 **Logto 签出后重定向 URI**，请将以下 URI 添加到 Logto 应用详情页面的“签出后重定向 URI”列表中：

<CodeBlock>{`http://${defaultBaseUrl}/SignedOutCallback`}</CodeBlock>

#### 更改默认路径

**Logto 重定向 URI** 的默认路径是 `/Callback`，而 **Logto 签出后重定向 URI** 的默认路径是 `/SignedOutCallback`。

如果没有特殊要求，你可以保持原样。如果你想更改它，可以为 `LogtoOptions` 设置 `CallbackPath` 和 `SignedOutCallbackPath` 属性：

```csharp title="Program.cs"
builder.Services.AddLogtoAuthentication(options =>
{
  // 其他配置...
  // highlight-start
  options.CallbackPath = "/Foo";
  options.SignedOutCallbackPath = "/Bar";
  // highlight-end
});
```

记得在 Logto 应用详情页面中相应地更新值。
