# 为你的 Next Auth 应用添加认证 (Authentication)

本指南将向你展示如何将日志 (Logto) 集成到你的 Next.js 应用中，使用 [Next Auth](https://next-auth.js.org/)。

<GuideTip />

## 前提条件

- 一个 [Logto Cloud](https://cloud.logto.io) 账户或一个 [自托管 Logto](/introduction/set-up-logto-oss)。
- 一个已创建的 Logto 传统应用。
- 一个带有 Next Auth 的 Next.js 项目，查看 [Next Auth 文档](https://next-auth.js.org/getting-started/introduction)。
