# 为你的 Auth.js (Next Auth) 应用添加认证 (Authentication)

本指南将向你展示如何将 Logto 集成到你的使用 [Auth.js](https://authjs.dev/)（以前称为 Next Auth）的 Next.js 应用中。

<GuideTip />

## 前提条件 \{#prerequisites}

- 一个 [Logto Cloud](https://cloud.logto.io) 账户或一个 [自托管 Logto](/introduction/set-up-logto-oss)。
- 一个已创建的 Logto 传统应用。
- 一个带有 Auth.js 的 Next.js 项目，查看 [Auth.js 文档](https://authjs.dev/getting-started/installation)。
