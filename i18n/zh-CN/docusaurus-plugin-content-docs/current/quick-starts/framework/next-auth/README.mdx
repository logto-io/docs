# 为你的 Auth.js (Next Auth) 应用添加认证 (Authentication)

本指南将向你展示如何将 Logto 集成到你的 Next.js 应用中，使用 [Auth.js](https://authjs.dev/)，前身为 Next Auth。

<GuideTip />

## 前置条件 \{#prerequisites}

- 一个 [Logto Cloud](https://cloud.logto.io) 账户或一个 [自托管 Logto](/introduction/set-up-logto-oss)。
- 已创建的 Logto 传统应用。
- 一个集成了 Auth.js 的 Next.js 项目，参考 [Auth.js 文档](https://authjs.dev/getting-started/installation)。
