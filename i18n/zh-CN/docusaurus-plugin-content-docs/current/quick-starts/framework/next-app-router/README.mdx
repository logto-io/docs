### 获取 API 资源的访问令牌 (Access token) \{#fetch-access-token-for-the-api-resource}

<FetchAccessTokenForApiResources />

如果你需要在服务端组件中获取访问令牌 (Access token)，可以使用 `getAccessTokenRSC` 函数：

```tsx title="app/page.tsx"
// 引入所需的函数和配置
import { getAccessTokenRSC } from '@logto/next/server-actions';
import { logtoConfig } from './logto';

export default async function Home() {
  const accessToken = await getAccessTokenRSC(logtoConfig, 'https://shopping.your-app.com/api');

  return (
    <main>
      <p>访问令牌 (Access token): {accessToken}</p>
    </main>
  );
}
```

:::tip

HTTP 不允许在流式传输开始后设置 cookie，`getAccessTokenRSC` 无法更新 cookie 值，因此如果访问令牌 (Access token) 被刷新，它不会持久化到会话中。推荐在客户端或路由处理器中使用 `getAccessToken` 函数。

:::

### 获取组织令牌 (Organization tokens) \{#fetch-organization-tokens}

<FetchOrganizationTokenForUser />

如果你需要在服务端组件中获取组织令牌 (Organization token)，可以使用 `getOrganizationTokenRSC` 函数：

```tsx title="app/page.tsx"
// 引入所需的函数和配置
import { getOrganizationTokenRSC } from '@logto/next/server-actions';
import { logtoConfig } from './logto';

export default async function Home() {
  const token = await getOrganizationTokenRSC(logtoConfig, 'organization-id');

  return (
    <main>
      <p>组织令牌 (Organization token): {token}</p>
    </main>
  );
}
```

:::tip

HTTP 不允许在流式传输开始后设置 cookie，`getOrganizationTokenRSC` 无法更新 cookie 值，因此如果访问令牌 (Access token) 被刷新，它不会持久化到会话中。推荐在客户端或路由处理器中使用 `getOrganizationToken` 函数。

:::

## 未授权时自动重定向到登录 \{#automatic-sign-in-redirect\}

:::tip
`signIn` 辅助函数会修改 cookie 以建立登录会话，因此不能直接在 React Server Component (RSC) 中调用。若要在 RSC 检测到未授权用户时自动触发登录，请在专用路由处理器中调用 `signIn` 并重定向到该路由。
:::

```ts title="app/sign-in/route.ts"
import { signIn } from '@logto/next/server-actions';
import { logtoConfig } from '../../logto';

export async function GET() {
  await signIn(logtoConfig);
}
```

```tsx title="app/protected/page.tsx"
import { getLogtoContext } from '@logto/next/server-actions';
import { redirect } from 'next/navigation';
import { logtoConfig } from '../logto';

export default async function ProtectedPage() {
  const { isAuthenticated } = await getLogtoContext(logtoConfig);

  if (!isAuthenticated) {
    redirect('/sign-in');
  }

  return <div>受保护内容</div>;
}
```
