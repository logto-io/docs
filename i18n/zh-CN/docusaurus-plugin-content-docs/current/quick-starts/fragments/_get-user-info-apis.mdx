你可以使用这些日志 (Logto) 方法以编程方式检索用户信息：

- <code>{props.getIdTokenClaimsApi}</code>：通过解码本地 ID 令牌获取用户信息。某些声明可能不可用。
- <code>{props.fetchUserInfoApi}</code>：通过向 [userinfo
  endpoint](https://openid.net/specs/openid-connect-core-1_0.html#UserInfo) 发送请求获取用户信息。

需要注意的是，可以检索的用户信息声明取决于用户在登录时使用的权限，并且考虑到性能和数据大小，ID 令牌可能不包含所有用户声明，某些用户声明仅在 userinfo endpoint 中可用（请参阅下面的相关列表）。
