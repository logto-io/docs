# 用户管理

用户管理是 Logto 的一个重要关注领域。你可以通过 Logto 控制台或 Management API 访问和管理你的用户数据，这两种方式都非常高效。借助这些工具，你可以执行如下任务：

- 通过用户名、ID、全名、手机号或邮箱搜索用户。
- 对用户执行多种操作，如解绑社交账号、更新个人信息、删除用户或重置密码。
- 查看用户日志。
- 分配角色 (Role) 以进行访问控制。
- 创建和管理个人访问令牌 (Personal access token)。
- 管理用户会话。
