# 将 Logto 集成到你的应用程序中

按照以下步骤，将认证 (Authentication) 添加到你的应用程序中，无论是面向用户的应用还是机器对机器服务：

1. 导航到 <CloudLink to="/applications">控制台 > 应用程序</CloudLink>

2. 点击“创建应用程序”以添加新应用程序

3. 选择你的应用程序框架开始。如果找不到你的框架，请点击应用程序创建页面右下角的“创建无框架应用”按钮，通过选择一个[应用程序类型](/integrate-logto/application-data-structure/#application-types)来创建应用，或者按照我们的 [SDK 约定](/developers/sdk-conventions)提交功能请求或贡献 SDK。

4. 选择框架后，你将看到该框架 SDK 的快速入门指南。按照步骤配置和集成你的应用程序。如果你需要帮助理解集成过程中涉及的概念，可以参考[理解 Logto 认证 (Authentication) 流程](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/)以更深入地了解集成。

:::note
控制台中的指南仅用于使用我们的 SDK 快速开始 Logto。有关完整的集成指南，包括高级 SDK 使用，请查看[快速入门](/quick-starts)部分。
:::
