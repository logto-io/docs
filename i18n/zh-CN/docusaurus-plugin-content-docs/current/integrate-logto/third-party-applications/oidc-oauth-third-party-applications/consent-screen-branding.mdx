# 用户授权页面品牌

确保第三方的品牌信息和条款链接在用户被重定向到第三方应用程序的用户授权页面时正确显示是很重要的。

Logto 允许你自定义第三方应用程序的品牌信息，包括应用程序名称、标志和条款链接。

## 自定义品牌信息

确保正确配置第三方应用程序的品牌信息，以确保为用户提供一致且安全的认证 (Authentication) 体验。

1. 前往 <CloudLink to="/applications/third-party-applications">控制台 > 应用程序 > 第三方应用</CloudLink> 并打开特定 OIDC 第三方应用程序的详细信息页面。

2. 导航到 **品牌** 标签。

3. 配置用户授权页面的显示信息：

- **显示名称**：将在用户授权页面上显示的第三方应用程序名称。它将代表请求访问用户信息的第三方应用程序的名称。如果此字段为空，将使用 **应用程序名称**。
- **应用标志（浅色）**：将在用户授权页面上显示的第三方应用程序标志。它将代表请求访问用户信息的第三方应用程序的品牌。如果同时提供了第三方应用程序的标志和你的通用登录体验标志，则两者都将在用户授权页面上显示。
- **应用标志（深色）**：仅在启用深色模式登录体验时可用。在 <CloudLink to="/sign-in-experience/branding">控制台 > 登录体验 > 品牌</CloudLink> 页面管理深色模式设置。
- **使用条款 URL**：将在用户授权页面上显示的第三方应用程序的条款链接。
- **隐私政策 URL**：将在用户授权页面上显示的第三方应用程序的隐私链接。
