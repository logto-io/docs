# 自定义用户授权页面 (Consent screen) 品牌

当用户被重定向到第三方应用的用户授权页面 (Consent screen) 时，确保第三方的品牌信息和条款链接正确展示给用户非常重要。

Logto 允许你自定义第三方应用的品牌信息，包括应用名称、Logo 和条款链接。

## 自定义品牌信息 \{#customize-the-branding-information}

请确保妥善配置第三方应用的品牌信息，为你的用户提供一致且安全的认证 (Authentication) 体验。

1. 前往 <CloudLink to="/applications/third-party-applications">控制台 > 应用 > 第三方应用</CloudLink>，打开某个 OIDC 第三方应用的详情页。

2. 切换到 **品牌** 标签页。

3. 配置用户授权页面 (Consent screen) 的展示信息：

- **显示名称**：将在用户授权页面 (Consent screen) 上展示的第三方应用名称。它代表请求访问你用户信息的第三方应用名称。如果此字段为空，将使用 **应用名称**。
- **应用 Logo（浅色）**：将在用户授权页面 (Consent screen) 上展示的第三方应用 Logo。它代表请求访问你用户信息的第三方应用品牌。如果同时提供了第三方应用 Logo 和你的通用登录体验 Logo，二者都会在用户授权页面 (Consent screen) 上展示。
- **应用 Logo（深色）**：仅在启用深色模式登录体验时可用。可在 <CloudLink to="/sign-in-experience/branding">控制台 > 登录体验 > 品牌</CloudLink> 页面管理深色模式设置。
- **使用条款 URL**：将在用户授权页面 (Consent screen) 上展示的第三方应用条款链接。
- **隐私政策 URL**：将在用户授权页面 (Consent screen) 上展示的第三方应用隐私链接。
