# 租户成员管理

## 邀请协作者

前往 <CloudLink to="/tenant-settings/members">控制台 > 租户设置 > 成员</CloudLink>。你将看到当前的成员，并可以邀请更多成员。输入电子邮件地址并分配角色（可以批量操作），然后发送。

Logto 允许你在 Logto Cloud 中以两种不同的角色邀请团队成员到你的租户：

- **管理员：** 完全访问和管理所有租户资源。
- **协作者：** 可以管理与身份相关的资源，但有以下限制：
  - 无法访问账单信息
  - 无法邀请或管理其他成员
  - 无法更改租户名称
  - 无法删除租户

### 成员限制

| 租户类型     | 成员限制                                   |
| ------------ | ------------------------------------------ |
| 开发租户     | 最多 3 名成员（联系我们以获取更多信息）    |
| 免费生产租户 | 1 名成员（仅限管理员）                     |
| 专业生产租户 | 从 3 名成员开始，然后每增加一名成员收费 $8 |

### 管理邀请

邀请选项卡显示过去的邀请及其状态

| 状态   | 说明                     | 可用操作 |
| ------ | ------------------------ | -------- |
| 待处理 | 邀请已发送，但尚未接受。 | 重新发送 |
| 已接受 | 用户现在是租户成员。     | 删除     |
| 已过期 | 邀请超过了 7 天的限制。  | 删除     |
| 已撤销 | 管理员取消了邀请。       | 删除     |

你不能邀请已经有待处理邀请或已经是成员的人。

### 管理租户成员（仅限管理员）

作为管理员，你有多种方式来管理你的团队：

- **邀请或移除成员：** 根据需要添加新团队成员或移除访问权限。
- **更改成员角色：** 根据所需的访问级别分配“管理员”或“协作者”角色。
- **转移租户所有权：** 如果需要，你可以将租户的所有权转移给其他成员。你可以先将管理员角色授予其他成员，然后选择继续作为普通成员或完全离开租户。
- **添加多个管理员：** 与其他团队成员共享管理职责，以便协作管理。
