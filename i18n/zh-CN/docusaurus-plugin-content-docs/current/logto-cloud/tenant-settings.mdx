# 租户设置

## 租户区域 \{#tenant-region}

当你创建租户时，可以选择租户数据存储的区域。租户创建后无法更改区域。可用的区域如下：

- 欧洲（荷兰）
- 美国西部（亚利桑那）
- 澳大利亚（澳大利亚东部）
- 日本（日本东部）

通常，你应该选择离你的客户最近的区域，以最小化延迟并提升性能。

Logto 利用全球边缘网络，为你的应用提供最佳性能和可用性。请求路由经过优化，确保你的用户始终连接到表现最优的选项。

:::note
需要其他区域？[联系我们](https://logto.io/contact)以：

- 申请新的公有云区域
- 咨询在你首选位置部署 Logto 私有云
  :::

## 租户类型：开发与生产 \{#tenant-types-dev-vs-prod}

在 Logto Cloud 中有两种租户类型：开发（development）和生产（production）。通过这种租户区分，你可以更好地在不同环境下管理项目，提高效率，同时充分享受 Logto 的全部价值。

请注意，租户类型只能在创建时指定；一旦创建后无法编辑。

## 开发环境 \{#development}

开发租户（dev tenant）主要用于测试目的，不应在生产环境中使用。这些租户可以免费访问付费计划中的高级和付费功能，无需订阅。

但开发租户有以下限制：

- 开发租户会自动删除 90 天以上的用户和组织。
- 登录体验过程中会显示横幅，提示租户处于开发模式。
- 开发租户的某些功能可能有配额限制。如有适用，这些限制会在功能详情页说明。
- Logto 可能会更新开发租户的配额限制，并会尽力提前通知你。

## 生产环境 \{#production}

生产租户是终端用户访问线上应用的地方，你可能需要[付费订阅](https://logto.io/pricing)。你可以订阅免费计划或专业计划来创建生产租户。如果你订阅免费计划，最多只能创建 10 个租户。
