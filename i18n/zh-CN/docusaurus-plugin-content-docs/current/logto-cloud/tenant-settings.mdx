# 租户设置

## 租户区域

当你创建一个租户时，你可以选择存储租户数据的区域。租户创建后无法更改。以下是可用的区域：

- 欧盟（荷兰）
- 美国（美国西部）
- 澳大利亚（澳大利亚）

通常，你应该选择离你的客户最近的区域，以最小化延迟并提高性能。

Logto 利用全球边缘网络为你的应用程序提供最佳性能和可用性。请求路由经过优化，以确保你的用户始终连接到性能最佳的选项。

:::note
寻找其他区域？请[联系我们](https://logto.io/contact)。
:::

## 租户类型：开发 vs. 生产

在 Logto Cloud 中有两种类型的租户：开发和生产。通过这种租户区分，你可以更好地管理不同环境中的项目，提高效率，同时享受 Logto 的全部价值。

请注意，你只能在创建时指定租户类型；租户类型一旦创建后无法编辑。

## 开发

开发租户（dev 租户）主要用于测试目的，不应在生产环境中使用。这些租户允许访问付费计划中的高级和付费功能，免费且无需订阅。

然而，开发租户有一些限制：

- 开发租户将在 90 天后自动删除用户和组织。
- 在登录体验期间会出现一个横幅，指示租户处于开发模式。
- 开发租户可能对特定功能有配额限制。这些限制在功能详情页面上有说明（如果适用）。
- Logto 可能会更新开发租户的配额限制，我们将尽力提前通知你。

## 生产

生产租户是终端用户访问实时应用程序的地方，你可能需要一个[付费订阅](https://logto.io/pricing)。你可以订阅免费计划或专业计划来创建生产租户。如果你订阅免费计划，你最多只能创建 10 个租户。
