# Logto OSS

Logto 提供开源 (OSS) 和 [云](/logto-cloud) 版本。本节将重点介绍 OSS 解决方案：<Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS 是一个免费使用的身份解决方案，为开发者提供完全的定制能力和对其数据的完全控制。Logto OSS 提供了灵活性，可以根据特定需求调整代码库，非常适合那些重视数据主权并希望拥有一个可以随着应用程序成长的定制身份提供商解决方案的人。

## Logto OSS 支持的功能 \{#feature-supported-by-logto-oss}

Logto OSS 支持 Logto 服务的大多数核心功能，并定期更新和维护。

一些高级功能目前仅限于 Logto Cloud 版本，包括：

| OSS 中的功能限制                                                          | 描述                                                                                                                                        |
| ----------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Logto 控制台：多个租户](/logto-cloud/tenant-settings)                     | 在控制台中创建和管理多个 Logto 租户。                                                                                             |
| [Logto 控制台：协作者邀请](/logto-cloud/tenant-member-management)     | 邀请和管理一个 Logto 租户的多个成员。                                                                                            |
| Logto 控制台：启用 MFA                                                           | 通过要求在登录 Logto 控制台时进行多因素认证 (MFA) 来增强安全性。                                                    |
| [Logto 受保护应用](/integrate-logto/protected-app)                               | 由 Cloudflare 提供支持的快速和非 SDK 认证 (Authentication) 集成。                                                                                |
| [Logto 内置电子邮件服务](/connectors/email-connectors/built-in-email-service) | 利用内置和免费的电子邮件服务进行电子邮件发送。                                                                                    |
| [自定义你的 UI](/customization/bring-your-ui)                                       | 使用此功能自定义登录体验的 UI 或流程（OSS 可以在 GitHub 上 fork 代码以直接自定义登录体验）。 |
| [IdP 发起的 SSO](/end-user-flows/enterprise-sso/idp-initiated-sso)               | 启用身份提供商 (IdP) 发起的企业单点登录 (SSO)。                                                                                      |
| [SAML 应用](/integrate-logto/saml-app)                                              | 通过 SAML 协议集成应用程序。OSS 版本限制为 3 个 SAML 应用。                                                               |

提示：多租户、成员邀请和 MFA 不适用于你的团队登录开源 Logto 控制台。然而，你可以在自己的产品中使用 Logto OSS 实现这些功能，使它们对你的终端用户可用。
