# Logto OSS

Logto 提供开源 (OSS) 和 [云](/logto-cloud) 版本。本节将重点介绍 OSS 解决方案：<Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS 是一个免费使用的身份解决方案，为开发者提供完全的定制能力和对其数据的完全控制。Logto OSS 提供了灵活性，可以根据特定需求调整代码库，非常适合那些重视数据主权并希望拥有一个可以随着应用程序成长的定制身份提供商解决方案的人。

## Logto OSS 支持的功能

Logto OSS 支持 Logto 服务的大多数核心功能，并定期更新和维护。

一些高级功能目前仅限于 Logto Cloud 版本，包括 [Logto 受保护应用](/integrate-logto/protected-app/)、[免费电子邮件服务](/connectors/email-connectors/built-in-email-service)、[自定义 UI](/customization/bring-your-ui)、[IdP 发起的 SSO](/end-user-flows/enterprise-sso/idp-initiated-sso) 和 SAML IdP。
