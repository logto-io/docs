# 贡献

感谢你对 Logto 的贡献兴趣！我们感谢社区的贡献，以使我们的项目更加健壮、用户友好和多功能。无论你是在修复错误、改进文档、创建你的连接器、添加新功能，还是提供反馈，你的努力都是无价的。

## 如何贡献？

我们努力使贡献过程尽可能顺利。在开始之前，请查看我们在 [GitHub 上的贡献指南](https://github.com/logto-io/logto/blob/master/.github/CONTRIBUTING.md)。该指南概述了我们的流程、编码标准以及成功贡献给 Logto 的步骤。

## 需要帮助？

如果你有问题或需要指导，请随时通过我们的 [GitHub 讨论](https://github.com/logto-io/logto/discussions)、[打开一个 issue](https://github.com/logto-io/logto/issues) 或 [联系我们](https://logto.io/contact)，我们将确保你拥有成功所需的一切。
