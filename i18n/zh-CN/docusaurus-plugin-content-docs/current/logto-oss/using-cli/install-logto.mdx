# 安装 Logto

## 交互式

在终端中运行以下命令以进行交互式安装：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## 选项和静默安装

或者，你可以添加以下选项以跳过某些问题：

```bash
# 所有这些选项都是可选的
--db-url      Logto 数据库的 Postgres URL
-p, --path    Logto 实例的路径，必须是一个不存在的路径
--ss          跳过 Logto 数据库播种
--oc          安装后添加官方连接器
```

运行 `logto init --help` 获取完整的帮助信息。

以下是执行静默安装的示例（例如，在 CI 中）：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

它将按顺序执行以下操作：

1. 下载并解压 Logto 到 `./logto`
2. 尝试使用 URL `postgresql://your-postgres-dsn:5432/logto` 初始化和播种数据库
3. 将 Logto 官方连接器添加到 `./logto/packages/core/connectors`
