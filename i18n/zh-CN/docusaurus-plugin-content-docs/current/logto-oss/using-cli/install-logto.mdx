# 安装 Logto

## 交互式安装 \{#interactive}

在终端中运行以下命令以进行交互式安装：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## 选项与静默安装 \{#options-and-silent-installation}

你也可以添加以下任意选项（均为可选），以跳过部分问题：

| Option                                         | Description（描述）                                                                                                                                                                           |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--db-url`                                     | Logto 数据库的 Postgres URL。                                                                                                                                                                 |
| `-p, --path`                                   | Logto 实例的路径，必须是不存在的路径。                                                                                                                                                        |
| `--ss`                                         | 跳过 Logto 数据库初始化数据。                                                                                                                                                                 |
| `--disable-admin-pwned-password-check, --dapc` | 为隔离或离线部署，使用禁用 Have I Been Pwned (HIBP) 密码泄露检查的方式初始化 admin 租户。详见 [为隔离或离线部署初始化数据](/logto-oss/using-cli#seed-for-air-gapped-or-offline-deployments)。 |

运行 `logto init --help` 获取完整帮助信息。

以下是进行静默安装（例如在 CI 中）的示例：

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

它将按以下顺序执行：

1. 下载并解压 Logto 到 `./logto`
2. 尝试使用 URL `postgresql://your-postgres-dsn:5432/logto` 初始化并填充数据库
3. 将 Logto 官方连接器添加到 `./logto/packages/core/connectors`
