# 开发你的连接器

目前，Logto 已经提供了许多广泛使用的社交登录连接器，如微信、支付宝、Google、Facebook 等，以及短信 / 电子邮件连接器，如阿里云、SendGrid 和 Twilio。

下面是一个架构图，展示了连接器如何与 Logto 核心服务协作。

<img src="/img/assets/logto-connector-framework.png" alt="Connector framework" />

这些连接器应该能够满足大多数用户在此阶段的需求，但我们将继续取得进展。

在我们开发更多连接器的同时，你也可以开发自己的连接器。我们鼓励你与整个社区分享你的连接器。

让我们从构建一个连接器开始，看看如何为 Logto 项目做出贡献。
