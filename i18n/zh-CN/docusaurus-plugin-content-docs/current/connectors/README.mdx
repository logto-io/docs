# 连接器

连接器是 Logto 和外部服务之间的桥梁。它们支持[无密码验证](https://auth.wiki/passwordless)方法，如[电子邮件验证](/end-user-flows/sign-up-and-sign-in/sign-in)、[短信验证](/end-user-flows/sign-up-and-sign-in/sign-in)、[社交登录](/end-user-flows/sign-up-and-sign-in/social-sign-in)和[企业单点登录 (SSO)](/end-user-flows/enterprise-sso)。

使用用户名和密码进行传统账户注册可能是一个繁琐且令人沮丧的过程。用户经常难以记住复杂的密码，并可能由于不便而不愿创建新账户。如果用户可以提供他们的电子邮件或电话号码并通过验证码进行验证；或者使用 Google/Facebook 等账户或 Microsoft 等企业账户登录，这可以极大地简化登录过程并减少用户流失。连接器就是为了解决这个问题而创建的。
