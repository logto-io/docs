# 启用第三方 AI 代理访问你的应用

:::caution
👷 本教程仍在编写中。请稍后回来查看更新。🚧
:::

本指南将带你完成将 Logto 集成到你的服务中，并启用 AI 代理访问该服务的流程。

你将学会如何：

- 配置 Logto 作为你的服务的授权 (Authorization) 服务器。
- 获取 AI 代理访问你的服务所需的访问令牌 (Access token)。
- 使用 AI 代理测试整个流程。

## 前置条件 \{#prerequisites}

- 一个 [Logto Cloud](https://cloud.logto.io)（或自托管）租户
- 一个向 AI 代理开放 API 端点的服务

### 理解流程 \{#understanding-the-flow}

- **服务**：你希望向 AI 代理开放的服务。
- **AI 代理**：将访问你服务的 AI 代理。
- **Logto**：作为 OpenID Connect 提供方（授权 (Authorization) 服务器），管理用户身份。
