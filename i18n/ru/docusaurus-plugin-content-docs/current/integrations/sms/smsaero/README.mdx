# Настройка проверки SMS через SMS Aero

Официальный коннектор Logto для службы коротких сообщений SMS Aero.

<GuideTip />

## Регистрация аккаунта \{#register-account}

Создайте новый аккаунт на [SMSAero](https://smsaero.ru/). (Перейдите к следующему шагу, если у вас уже есть аккаунт.)

## Получение учетных данных аккаунта \{#get-account-credentials}

Нам понадобятся учетные данные API, чтобы коннектор работал. Начнем с
[API и SMPP](https://smsaero.ru/cabinet/settings/apikey/).

Скопируйте "API-key" или создайте новый.

## Составление JSON коннектора \{#compose-the-connector-json}

Заполните поля _email_, _apiKey_ и _senderName_ вашим email, API ключом и именем отправителя.

Вы можете заполнить имя отправителя "SMSAero", чтобы использовать имя отправителя по умолчанию, предоставленное SMSAero.

Вы можете добавить несколько шаблонов SMS коннекторов для разных случаев. Вот пример добавления одного шаблона:

- Заполните поле `content` произвольным строковым содержимым. Не забудьте оставить `{{code}}` как место для случайного кода подтверждения.
- Заполните поле `usageType` одним из `Register`, `SignIn`, `ForgotPassword`, `Generic` для разных случаев использования.
  Чтобы включить полные пользовательские потоки, требуются шаблоны с usageType `Register`, `SignIn`, `ForgotPassword` и `Generic`.

### Тестирование коннектора SMSAero \{#test-smsaero-connector}

Вы можете ввести номер телефона и нажать "Отправить", чтобы проверить, работают ли настройки, перед "Сохранить и завершить".

Вот и все. Не забудьте
[включить коннектор в опыте входа](/connectors/sms-connectors/#enable-phone-number-sign-up-or-sign-in).

### Типы конфигурации \{#config-types}

| Имя        | Тип         |
| ---------- | ----------- |
| email      | string      |
| apiKey     | string      |
| senderName | string      |
| templates  | Templates[] |

| Свойства шаблона | Тип         | Значения перечисления                                   |
| ---------------- | ----------- | ------------------------------------------------------- |
| content          | string      | N/A                                                     |
| usageType        | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
