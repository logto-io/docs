# Настройка проверки SMS через Twilio SMS

Официальный коннектор Logto для службы коротких сообщений Twilio.

<GuideTip />

## Начало работы \{#get-started}

Twilio предоставляет программируемые инструменты связи для совершения и приема телефонных звонков, отправки и получения текстовых сообщений и других функций связи. Мы можем отправлять текстовые сообщения с "кодом подтверждения" через его веб-сервис API.
