# Настройка проверки SMS через Vonage SMS

Официальный коннектор Logto для Vonage SMS.

<GuideTip />

## Начало работы \{#get-started}

Vonage — это глобальный поставщик коммуникационных услуг, предлагающий надежные облачные коммуникационные сервисы, включая SMS (службу коротких сообщений). Коннектор Vonage SMS — это плагин, предоставляемый командой Logto, который позволяет конечным пользователям Logto регистрироваться и входить в свою учетную запись Logto через коды проверки SMS.

## Настройка в Vonage \{#set-up-in-vonage}

:::tip
Вы можете пропустить этот шаг, если уже выполнили его.
:::

Чтобы работать с этим коннектором, вам нужно [зарегистрировать учетную запись](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) в Vonage. Это даст вам API ключ и секрет, которые вы можете использовать для доступа к API через этот коннектор.

После того как у вас будет учетная запись, вы можете найти ваш API ключ и API секрет в верхней части панели управления Vonage API.

Вам также может понадобиться [арендовать виртуальный номер](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) для отправки SMS-сообщений.

Смотрите [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview) для получения дополнительной информации.

## Настройка в Logto \{#set-up-in-logto}

1. **API Key**: Ваш API ключ Vonage.
2. **API Secret**: Ваш API секрет Vonage.
3. **Brand Name**: Название бренда, которое вы хотите использовать для отправки SMS, это также называется полем `from`, смотрите [Идентификация отправителя](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) для получения дополнительной информации.
4. **Templates**: Шаблоны, которые вы хотите использовать для отправки SMS, вы можете использовать шаблоны по умолчанию или изменить их по мере необходимости.
