# Настройка проверки SMS через GatewayAPI SMS

Официальный коннектор Logto для GatewayAPI SMS.

<GuideTip />

## Начало работы \{#get-started}

GatewayAPI — это облачный провайдер в Европе, предлагающий множество облачных сервисов, включая SMS (службу коротких сообщений). Коннектор GatewayAPI SMS — это плагин, предоставляемый командой Logto для вызова сервиса GatewayAPI SMS, с помощью которого конечные пользователи Logto могут регистрироваться и входить в свою учетную запись Logto через код проверки SMS.

## Настройка в GatewayAPI \{#set-up-in-gatewayapi}

### Создание учетной записи GatewayAPI \{#create-an-gatewayapi-account}

Перейдите на [веб-сайт GatewayAPI](https://www.gatewayapi.com/) и зарегистрируйте свою учетную запись GatewayAPI, если у вас ее нет.

### Активация учетной записи \{#enable-account}

Вам может понадобиться активировать свою учетную запись перед использованием SMS-сервиса. Вы можете связаться с службой поддержки GatewayAPI для активации вашей учетной записи.

### Получение API токена \{#get-api-token}

Перейдите на страницу API Keys в консоли GatewayAPI и найдите API токен или создайте новый API токен.

## Настройка в Logto \{#set-up-in-logto}

1. **Endpoint**: Если ваша учетная запись GatewayAPI находится в регионе ЕС, вы должны использовать endpoint https://gatewayapi.com/rest/mtsms. Если ваша учетная запись GatewayAPI находится в регионе США, вы должны использовать endpoint https://gatewayapi.com/rest/mtsms.
2. **API Token**: API токен, который вы создали на предыдущем шаге.
3. **Sender**: Отправитель, которого вы хотите использовать для отправки SMS.
4. **Templates**: Шаблоны, которые вы хотите использовать для отправки SMS, вы можете использовать стандартные шаблоны или изменить их по мере необходимости.
