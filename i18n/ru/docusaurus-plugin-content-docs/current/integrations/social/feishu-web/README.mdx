# Настройка социального входа через Feishu

Feishu Web: Продвинутая платформа для корпоративного сотрудничества и управления, предлагающая бесшовное офисное взаимодействие и согласование целей команды для полной активации организаций и индивидуумов.

<GuideTip />

## Начало работы с социальным входом через Feishu \{#getting-started-with-feishu-social-sign-in}

Коннектор Feishu web предназначен для веб-приложений на настольных компьютерах и использует процесс аутентификации OAuth 2.0.

## Регистрация учетной записи разработчика Feishu \{#register-a-feishu-developer-account}

Если у вас нет учетной записи разработчика Feishu, пожалуйста, зарегистрируйтесь на [Feishu Open Platform](https://open.feishu.cn/app).

## Создание приложения \{#create-an-application}

1. На [Консоли разработчика](https://open.feishu.cn/app) нажмите "Создать пользовательское приложение".
2. Заполните имя приложения, описание, выберите значок и нажмите кнопку "Создать".
3. В левой боковой панели нажмите "Настройки безопасности", заполните "Redirect URL" как `${logto_endpoint}/callback/${connector_id}`. Соответствующее значение можно найти в поле `Callback URI` на странице деталей коннектора Feishu в Logto Console.
4. В разделе "Учетные данные и основная информация" вы можете получить "App ID" и "App Secret".

:::note
Для внутреннего использования не в корпоративных целях вам также нужно нажать кнопку "Создать версию" на странице "Управление версиями и выпуск". "App ID" и "App Secret" вступят в силу только после изменения статуса приложения на "Включено".
:::
