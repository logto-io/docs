# Настройка социального входа через OpenID Connect (OIDC)

Официальный коннектор Logto для протокола OpenID Connect (OIDC).

<GuideTip />

## Начало работы \{#get-started}

Коннектор OIDC позволяет Logto подключаться к произвольному социальному провайдеру идентификации, который поддерживает протокол OIDC.

:::note

Коннектор OIDC — это особый вид коннектора в Logto, вы можете добавить несколько коннекторов на основе OIDC.

:::
