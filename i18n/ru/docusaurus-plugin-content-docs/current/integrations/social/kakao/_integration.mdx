## Настройка проекта в консоли Kakao Developers \{#set-up-a-project-in-the-kakao-devlopers-console}

- Посетите [Kakao Developers Console](https://developers.kakao.com/console/app) и войдите в систему с вашим аккаунтом Kakao.
- Нажмите **Добавить приложение**, чтобы создать новый проект, или выберите существующий проект.

## Настройка входа через Kakao \{#configure-kakao-login}

### Активация входа через Kakao \{#activate-kakao-login}

- Нажмите **Настройки продукта -> Вход через Kakao** в меню.
- Включите `Активация входа через Kakao`
- Добавьте нижеуказанный URL в `Redirect URI`
  - `http(s)://YOUR_URL/callback/${connector_id}` (`connector_id` можно найти на верхней панели страницы деталей коннектора в Logto Admin Console.)
  - (Пожалуйста, замените `YOUR_URL` на ваш URL `Logto` и выберите `http` или `https` в зависимости от вашей ситуации.)

### Настройки конфиденциальности \{#privacy-setting}

- Нажмите **Настройки продукта -> Вход через Kakao -> Элемент согласия** в меню.
- Измените состояние `Никнейм`, `Изображение профиля` и `Электронная почта` на **Требуется согласие** (Вы можете не иметь возможности изменить `Электронная почта` на **Требуется согласие** из-за настроек вашего проекта.)

### Настройки безопасности (необязательно) \{#security-setting-optional}

- Нажмите **Настройки продукта -> Вход через Kakao -> Безопасность** в меню.
- Нажмите `Код клиентского секрета`, чтобы сгенерировать секретный код.
- Измените `Состояние активации` на Включено. (Если вы включите это, `секретный код` будет необходим.)

## Настройка Logto \{#configure-logto}

### Типы конфигурации \{#config-types}

| Имя          | Тип     |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId \{#clientid}

`clientId` — это `REST API ключ` вашего проекта.
(Вы можете найти его в `резюме` вашего проекта в консоли разработчиков Kakao.)

#### clientSecret \{#clientseceret}

`clientSecret` — это `Секретный код` вашего проекта.
(Пожалуйста, проверьте [Настройки безопасности (необязательно)](#security-setting-optional))
