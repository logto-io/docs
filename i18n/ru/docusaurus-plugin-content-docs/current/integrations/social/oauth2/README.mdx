# Настройка социального входа через протокол OAuth 2.0

Официальный коннектор Logto для протокола OAuth 2.0.

<GuideTip />

## Начало работы \{#get-started}

Коннектор OAuth позволяет Logto подключаться к произвольному провайдеру социальной идентификации, который поддерживает протокол OAuth 2.0.

:::note

Коннектор OAuth — это особый вид коннектора в Logto, вы можете добавить несколько коннекторов на основе протокола OAuth.

:::
