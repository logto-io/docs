# Настройка социального входа через WeChat (Web)

Официальный коннектор Logto для социального входа через WeChat в веб-приложениях.

<GuideTip />

## Начало работы \{#get-started}

Если вы не знаете концепцию коннектора или не знаете, как добавить этот коннектор в ваш опыт входа, пожалуйста, ознакомьтесь с [руководством Logto](/connectors/social-connectors).

:::caution

Этот коннектор предназначен только для веб-приложений. Если вы ищете метод для входа через WeChat в нативных приложениях, пожалуйста, ознакомьтесь с [нативным коннектором WeChat](/integrations/wechat-native).

:::

## Создание веб-приложения на платформе WeChat Open Platform \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

Вы можете пропустить некоторые разделы, если уже завершили их.

:::

### Создание аккаунта \{#create-an-account}

Откройте https://open.weixin.qq.com/, нажмите кнопку "Sign Up" в правом верхнем углу, затем завершите процесс регистрации.

### Создание веб-приложения \{#create-a-web-app}

Войдите в систему с только что созданным аккаунтом. На вкладке "Web Application" (网页应用) нажмите большую зеленую кнопку "Create a web app" (创建网页应用).

![App tabs](./assets/app-tabs.png)

Давайте заполним необходимую информацию в форме заявки.

![Create a web app](./assets/create-web-app.png)

#### Основная информация \{#basic-info}

Большинство из них довольно просты. После заполнения формы нажмите "Next step", чтобы продолжить.

#### Информация о сайте \{#website-info}

Заполните "Authorization callback domain" (授权回调域) вашим доменом Logto. Например, `logto.io`.

#### Ожидание результата проверки \{#waiting-for-the-review-result}

После завершения информации о сайте нажмите "Submit Review", чтобы продолжить. Обычно проверка проходит быстро и завершается в течение 1-2 дней.

Мы подозреваем, что проверяющий назначается случайным образом при каждом отправлении, так как стандарт плавающий. Вас могут отклонить в первый раз, но не сдавайтесь! Укажите ваше текущее состояние и спросите у проверяющего, как его изменить.

### Настройка вашего коннектора WeChat \{#configure-your-wechat-connector}

Заполните поля `clientId` и `clientSecret` с _Client ID_ и _Client Secret_, которые вы получили на страницах деталей OAuth приложения.

Заполните поле `scope` значением 'snsapi_userinfo' или 'snsapi_base'. Вы можете оставить это поле пустым, так как оно НЕОБЯЗАТЕЛЬНО. Значение по умолчанию установлено как 'snsapi_userinfo'. Вы можете ознакомиться с [различиями](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) между различными значениями.

### Тестирование веб-коннектора WeChat \{#test-wechat-web-connector}

Вот и все. Не забудьте [включить социальный коннектор в опыте входа](/connectors/social-connectors/#enable-social-sign-in).

Как только веб-коннектор WeChat будет включен, вы можете снова войти в ваше приложение, чтобы проверить, работает ли он.
