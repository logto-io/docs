# Настройка социального входа через SAML

Официальный коннектор Logto для протокола SAML.

<GuideTip />

## Начало работы \{#get-started}

Коннектор SAML позволяет Logto подключаться к произвольному провайдеру социальной идентификации, поддерживающему протокол SAML.

:::note

Коннектор SAML — это особый вид коннектора в Logto, вы можете добавить несколько коннекторов на основе SAML.

:::
