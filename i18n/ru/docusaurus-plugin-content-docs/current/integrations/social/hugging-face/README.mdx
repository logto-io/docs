# Настройка социального входа через Hugging Face

Официальный коннектор Logto для социального входа через Hugging Face.

<GuideTip />

## Начало работы \{#get-started}

Коннектор Hugging Face позволяет конечным пользователям входить в ваше приложение, используя свои собственные аккаунты Hugging Face через поток Hugging Face OAuth / OpenID Connect.

<Integration />

## Тестирование коннектора Hugging Face \{#test-hugging-face-connector}

Вот и все. Коннектор Hugging Face теперь должен быть доступен. Не забудьте [включить социальный коннектор в опыте входа](/connectors/social-connectors/#enable-social-sign-in).
