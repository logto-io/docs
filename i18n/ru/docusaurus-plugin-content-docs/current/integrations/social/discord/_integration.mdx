### Регистрация приложения для разработчиков \{#register-a-developer-application}

- Посетите [Discord Developer Portal](https://discord.com/developers/applications) и войдите в систему с вашей учетной записью Discord.
- Нажмите кнопку **New Application**, чтобы создать приложение, выберите для него имя (например, LogtoAuth), отметьте галочку и нажмите **Create**.
- Перейдите на страницу **OAuth2** и нажмите **Reset Secret**.
- Обратите внимание на поля **CLIENT ID** и **CLIENT SECRET**.
- Добавьте допустимые перенаправления (например, **`http://auth.mycompany.io/callback/${connector_id}`**). `connector_id` можно найти на верхней панели страницы деталей коннектора в Logto Admin Console.

## Настройка Logto \{#configure-logto}

### Типы конфигурации \{#config-types}

| Имя          | Тип    |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |

#### clientId \{#clientid}

`clientId` — это поле `CLIENT ID`, которое мы сохранили ранее.
(Вы можете найти его на странице Oauth2 в Discord Developer Portal.)

#### clientSecret \{#clientsecret}

`clientSecret` — это `CLIENT SECRET`, который мы сохранили ранее.
(Если вы его потеряли, вам нужно нажать **Reset Secret**.)

#### scope \{#scope}

`scope` — это разрешения, предоставленные авторизацией пользователя. Значение по умолчанию — `identify email`.
Вы можете увидеть полный список областей [здесь](https://discord.com/developers/docs/topics/oauth2#shared-resources-oauth2-scopes).
