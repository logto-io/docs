# Настройка социального входа через GitHub

Официальный коннектор Logto для социального входа через GitHub.

<GuideTip />

## Начало работы \{#get-started}

Коннектор GitHub позволяет конечным пользователям входить в ваше приложение, используя свои собственные учетные записи GitHub через протокол аутентификации GitHub OAuth 2.0.

<Integration />

## Тестирование коннектора GitHub \{#test-github-connector}

Вот и все. Коннектор GitHub должен быть доступен сейчас. Не забудьте [включить социальный коннектор в опыте входа](/connectors/social-connectors/#enable-social-sign-in).
