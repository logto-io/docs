Чтобы интеграция SAML SSO работала, вам нужно предоставить метаданные IdP в Logto. Метаданные IdP — это [XML](https://blog.logto.io/what-is-xml) документ, содержащий всю информацию, необходимую для установления доверия между Logto и IdP.

Перейдите на вкладку `Connection`. Logto предоставляет три различных способа настройки метаданных IdP:

1. **URL метаданных**: Укажите URL XML-документа метаданных IdP. Logto получит метаданные по URL и автоматически настроит интеграцию SAML SSO.
2. **Загрузка метаданных**: Загрузите XML-документ метаданных IdP. Logto проанализирует XML-документ и автоматически настроит интеграцию SAML SSO.
3. **Ручная настройка**: Настройте метаданные IdP вручную.

- IdP entity ID: Идентификатор сущности IdP.
- URL единого входа: URL сервиса единого входа IdP.
- Сертификат подписи: Сертификат x509, используемый для проверки подписи ответа SAML от IdP.

При любой из вышеперечисленных конфигураций Logto проанализирует метаданные IdP и настроит интеграцию SAML SSO соответствующим образом.
