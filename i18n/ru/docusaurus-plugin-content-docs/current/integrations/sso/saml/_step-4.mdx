Укажите `домен электронной почты` вашей организации на вкладке `SSO experience` коннектора Logto. Это позволит использовать SSO коннектор в качестве метода аутентификации для этих пользователей.

Пользователи с адресами электронной почты в указанных доменах будут перенаправлены на использование SAML SSO коннектора в качестве единственного метода аутентификации.
