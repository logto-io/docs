Чтобы интеграция SAML SSO работала, вам нужно предоставить метаданные провайдера идентификации (IdP) обратно в Logto. Давайте вернемся на сторону Logto и перейдем на вкладку `Connection` вашего коннектора Azure AD SSO.

Logto предоставляет три разных способа настройки метаданных IdP. Самый простой способ — предоставить `metadata URL` приложения Azure AD SSO.

Скопируйте `App Federation Metadata Url` из раздела `SAML Certificates section` вашего приложения Azure AD SSO и вставьте его в поле `Metadata URL` в Logto.

![Azure AD Metadata URL.webp](./assets/azure_ad_metadata_url.webp)

Logto получит метаданные с URL и автоматически настроит интеграцию SAML SSO.
