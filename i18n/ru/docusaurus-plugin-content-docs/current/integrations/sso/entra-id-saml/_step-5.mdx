Укажите `домен электронной почты` вашей организации на вкладке `SSO experience` коннектора Logto. Это позволит использовать коннектор SSO в качестве метода аутентификации для этих пользователей.

Пользователи с адресами электронной почты в указанных доменах будут перенаправлены на использование коннектора SAML SSO в качестве единственного метода аутентификации.

Пожалуйста, ознакомьтесь с официальной [документацией](https://learn.microsoft.com/en-us/entra/identity/enterprise-apps/add-application-portal-setup-sso) Azure AD для получения более подробной информации об интеграции Azure AD SSO.
