Перейдите в раздел `Users and groups` вашего приложения Azure AD SSO. Нажмите на `Add user/group`, чтобы назначить пользователей для приложения Azure AD SSO. Только пользователи, назначенные вашему приложению Azure AD SSO, смогут аутентифицироваться через коннектор Azure AD SSO.
