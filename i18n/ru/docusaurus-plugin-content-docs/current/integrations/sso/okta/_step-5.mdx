Укажите `домен электронной почты` вашей организации на вкладке `SSO experience` коннектора Logto. Это позволит использовать коннектор SSO в качестве метода аутентификации для этих пользователей.

Пользователи с адресами электронной почты в указанных доменах будут перенаправлены на использование вашего коннектора SSO в качестве единственного метода аутентификации.

Для получения более подробной информации о создании интеграции OIDC с Okta, пожалуйста, ознакомьтесь с [Создание интеграций OIDC приложений](https://help.okta.com/oie/en-us/content/topics/apps/apps_app_integration_wizard_oidc.htm).
