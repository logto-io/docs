После успешного создания OIDC приложения вы будете перенаправлены на страницу с деталями приложения.

![Okta client credentials.webp](./assets/okta_client_credentials.webp)

Скопируйте `client ID` и `client secret` и заполните соответствующие поля на вкладке `Connection` коннектора Logto SSO.

Используйте ваш домен Okta в качестве `эмитента`. Пример: `https://dev-12345678.okta.com`. После заполнения всех полей нажмите кнопку `Save`, чтобы сохранить настройки коннектора.

Если предоставленная вами ссылка `эмитента` действительна, вы увидите разобранный полный список конфигураций Okta IdP, показанный ниже поля `эмитента`.
