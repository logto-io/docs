- Посетите портал администратора Okta и войдите как администратор.
- Перейдите на страницу `Applications` / `Applications` с помощью бокового меню.
- Нажмите кнопку `Create App Integration`, чтобы создать новое приложение OIDC.
- Выберите опцию `OIDC - OpenID Connect` в качестве `Sign-in method`.
- Выберите опцию `Web Application` в качестве `Application type`.

![Okta create application.webp](./assets/okta_create_application.webp)

Нажмите кнопку `Next`, чтобы продолжить.
