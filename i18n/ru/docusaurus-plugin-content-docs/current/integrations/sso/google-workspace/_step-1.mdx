Прежде чем вы сможете использовать Google Workspace в качестве провайдера аутентификации, вы должны настроить проект в [Google API Console](https://console.developers.google.com/), чтобы получить учетные данные OAuth 2.0. Если у вас уже есть проект, вы можете пропустить этот шаг. В противном случае создайте новый проект в вашей организации Google.
