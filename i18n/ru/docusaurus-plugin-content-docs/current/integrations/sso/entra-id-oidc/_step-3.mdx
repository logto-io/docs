Укажите `domains` электронной почты вашей организации на вкладке `experience` коннектора. Это позволит использовать коннектор SSO в качестве метода аутентификации для этих пользователей.

Пользователи с адресами электронной почты в указанных доменах будут исключительно ограничены использованием вашего коннектора SSO в качестве единственного метода аутентификации.
