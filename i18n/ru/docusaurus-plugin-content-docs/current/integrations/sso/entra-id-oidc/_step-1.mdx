1. Перейдите в [Microsoft Entra admin center](https://entra.microsoft.com/) и войдите как администратор.

2. Перейдите в Identity > Applications > App registrations.

<center>
  <img src={createApplication} alt="Create Application" />
</center>

3. Выберите `New registration`.

4. Введите имя приложения и выберите соответствующий тип учетной записи для вашего приложения.

5. Выберите `Web` в качестве платформы приложения.

6. Скопируйте и вставьте `redirect URI` со страницы настроек SSO Logto. `Redirect URI` — это URL, на который пользователь перенаправляется после аутентификации с Microsoft Entra ID.

<center>
  <img src={configApplication} alt="Configure Application" />
</center>

6. Нажмите `Register`, чтобы создать приложение.
