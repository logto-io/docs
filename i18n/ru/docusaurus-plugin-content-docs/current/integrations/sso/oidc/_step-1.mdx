Инициируйте интеграцию OIDC SSO, создав приложение на стороне провайдера идентификации (IdP). Вам потребуется предоставить следующие настройки с сервера Logto.

- **Callback URI**: Callback URI Logto, также известный как Redirect URI или Reply URL, это конкретная конечная точка или URL, который IdP использует для перенаправления браузера пользователя после успешной аутентификации. После успешной аутентификации пользователя с IdP, IdP перенаправляет браузер пользователя обратно на этот указанный URI вместе с кодом авторизации. Logto завершит процесс аутентификации на основе кода авторизации, полученного с этого URI.

Заполните Callback URI Logto в форме настроек OIDC приложения вашего IdP и продолжите создание приложения. (Большинство OIDC IdP предоставляют широкий выбор типов приложений. Чтобы создать веб-основанный SSO коннектор в Logto, пожалуйста, выберите тип `Web Application`.)
