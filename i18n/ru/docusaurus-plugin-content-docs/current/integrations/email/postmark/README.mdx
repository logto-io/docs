# Настройка проверки электронной почты через Postmark email service

Коннектор Logto для Postmark email service.

<GuideTip />

## Начало работы \{#get-started}

Postmark — это почтовая платформа для транзакционной и маркетинговой электронной почты. Мы можем использовать ее функцию отправки электронной почты для отправки _кода подтверждения_.

## Регистрация аккаунта Postmark \{#register-postmark-account}

Создайте новый аккаунт на [веб-сайте Postmark](https://postmark.com/). Вы можете пропустить этот шаг, если у вас уже есть аккаунт.

## Настройка вашего коннектора \{#configure-your-connector}

Заполните поле `serverToken` токеном сервера, который вы найдете в настройках вашего сервера в Postmark.

Заполните поле `fromEmail` адресом отправителя _From Address_.

Для включения полных пользовательских потоков требуются шаблоны с usageType `Register`, `SignIn`, `ForgotPassword` и `Generic`.

Вот пример JSON шаблона коннектора Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Тестирование коннектора электронной почты Postmark \{#test-postmark-email-connector}

Вы можете ввести адрес электронной почты и нажать "Отправить", чтобы проверить, работают ли настройки, перед тем как нажать "Сохранить и завершить".

Вот и все. Не забудьте [включить коннектор в опыте входа](/connectors/email-connectors/#enable-email-sign-up-or-sign-in).

## Типы конфигурации \{#config-types}

| Имя         | Тип    |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Свойства шаблона | Тип         | Значения перечисления                                   |
| ---------------- | ----------- | ------------------------------------------------------- |
| usageType        | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias    | string      | N/A                                                     |
