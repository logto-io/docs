## Настройка почтового сервиса в консоли AWS \{#configure-a-mail-service-in-the-aws-service-console}

### Регистрация аккаунта AWS \{#register-aws-account}

Перейдите на [AWS](https://aws.amazon.com/) и зарегистрируйте аккаунт.

### Создание идентичности \{#create-a-identity}

- Перейдите в консоль `Amazon Simple Email Service`
- Создайте идентичность, выбрав один из следующих вариантов:
  - Создать домен
  - Создать адрес электронной почты

### Настройка коннектора \{#configuration-of-the-connector}

1. Нажмите на ваше имя пользователя в правом верхнем углу консоли Amazon, чтобы войти в `Security Credentials`. Если у вас его нет, создайте `AccessKey` и сохраните его в надежном месте.
2. Завершите настройку коннектора `Amazon Simple Email Service`:
   - Используйте `AccessKey ID` и `AccessKey Secret`, полученные на шаге 1, чтобы заполнить `accessKeyId` и `accessKeySecret` соответственно.
   - `region`: Заполните поле `region` регионом идентичности, которую вы используете для отправки почты.
   - `emailAddress`: Адрес электронной почты, который вы используете для отправки почты, в формате `Logto\<noreply@logto.io>` или `\<noreply@logto.io>`

следующие параметры являются необязательными; описание параметров можно найти в [документации AWS SES API](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html).

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### Тестирование коннектора Amazon SES \{#test-the-amazon-ses-connector}

Вы можете ввести адрес электронной почты и нажать "Отправить", чтобы проверить, работают ли настройки, перед тем как нажать "Сохранить и завершить".

Вот и все. Не забудьте [включить коннектор в процессе входа](/connectors/email-connectors/#enable-email-sign-up-or-sign-in).

### Настройка типов \{#configure-types}

| Имя                                       | Тип                    |
| ----------------------------------------- | ---------------------- |
| accessKeyId                               | string                 |
| accessKeySecret                           | string                 |
| region                                    | string                 |
| emailAddress                              | string (НЕОБЯЗАТЕЛЬНО) |
| emailAddressIdentityArn                   | string (НЕОБЯЗАТЕЛЬНО) |
| feedbackForwardingEmailAddress            | string (НЕОБЯЗАТЕЛЬНО) |
| feedbackForwardingEmailAddressIdentityArn | string (НЕОБЯЗАТЕЛЬНО) |
| configurationSetName                      | string (НЕОБЯЗАТЕЛЬНО) |
| templates                                 | Template[]             |

| Свойства шаблона | Тип         | Значения перечисления                                   |
| ---------------- | ----------- | ------------------------------------------------------- |
| subject          | string      | N/A                                                     |
| content          | string      | N/A                                                     |
| usageType        | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
