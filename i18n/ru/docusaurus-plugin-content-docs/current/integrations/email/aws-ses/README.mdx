# Настройка проверки электронной почты через AWS Direct Mail

Официальный коннектор Logto для AWS коннектора для сервиса прямой почты.

<GuideTip />

## Начало работы \{#get-started}

Amazon SES — это облачный провайдер электронной почты, который может интегрироваться в любое приложение для массовой отправки электронной почты.

Команда Logto вызывает API Amazon Simple Email Service, с помощью которых конечные пользователи Logto могут зарегистрироваться и войти в свою учетную запись Logto через код проверки по электронной почте.
