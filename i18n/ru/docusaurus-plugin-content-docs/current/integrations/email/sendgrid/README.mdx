## Начало работы \{#get-started}

SendGrid (т.е. Twilio SendGrid) — это платформа для коммуникации с клиентами через транзакционные и маркетинговые электронные письма. Мы можем использовать функцию отправки электронной почты для отправки _кода подтверждения_.
