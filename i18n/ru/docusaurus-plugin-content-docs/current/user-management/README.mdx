# Управление пользователями

Управление пользователями является значительной областью внимания для Logto. Вы можете получить доступ к данным своих пользователей и управлять ими, используя либо Logto Console, либо Management API, оба из которых эффективны. С помощью этих инструментов вы можете выполнять такие задачи, как

- Поиск пользователей по имени пользователя, ID, полному имени, номеру телефона или электронной почте.
- Выполнение нескольких действий с пользователем, таких как отвязка социальных аккаунтов, обновление информации профиля, удаление пользователей или сброс паролей.
- Проверка журналов пользователей.
- Назначение ролей для управления доступом.
- Создание и управление личными токенами доступа.
