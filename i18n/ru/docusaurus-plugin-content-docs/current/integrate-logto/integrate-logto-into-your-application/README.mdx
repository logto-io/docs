# Интеграция Logto в ваше приложение

Следуйте этим шагам, чтобы добавить аутентификацию в ваши приложения с помощью Logto, будь то приложение для пользователей или сервис машина-машина:

1. Перейдите в <CloudLink to="/applications">Консоль > Приложения</CloudLink>

2. Нажмите "Создать приложение", чтобы добавить новое приложение

3. Выберите ваш фреймворк приложения для начала. Если вы не можете найти ваш фреймворк, нажмите кнопку "Создать приложение без фреймворка" в нижнем правом углу страницы создания приложения, чтобы создать приложение, выбрав [Тип приложения](/integrate-logto/application-data-structure/#application-types) или подайте запрос на добавление функции или внесите вклад в SDK, следуя нашим [Конвенциям SDK](/developers/sdk-conventions).

4. После выбора вашего фреймворка вы увидите руководство по быстрому старту для SDK фреймворка. Следуйте шагам для настройки и интеграции вашего приложения. Если вам нужна помощь в понимании концепций, связанных с процессом интеграции, вы можете обратиться к [Понимание потока аутентификации Logto](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) для более глубокого понимания интеграции.

:::note
Руководство в консоли предназначено только для быстрого старта с Logto, используя наш SDK. Для полных руководств по интеграции, включая расширенное использование SDK, ознакомьтесь с разделом [Быстрые старты](/quick-starts).
:::
