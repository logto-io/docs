# Интеграция аутентификации Logto

Logto предоставляет комплексные решения для аутентификации веб-, мобильных и настольных приложений, поддерживает [аутентификацию между машинами (M2M)](/quick-starts/m2m) между сервисами и может служить провайдером идентификации (IdP) для [сторонних приложений](/integrate-logto/third-party-applications) через стандартные протоколы, такие как [OpenID Connect(OIDC)](https://auth.wiki/openid-connect) и [OAuth 2.0](https://auth.wiki/oauth-2.0).

Начните интеграцию, выбрав решение, которое наилучшим образом соответствует вашим потребностям:

## Добавление аутентификации в ваше приложение \{#add-authentication-for-your-application}

Независимо от того, создаете ли вы приложения для пользователей (такие как веб-, мобильные или настольные приложения) или приложения машина-машина (M2M) для связи между сервисами, вы можете быстро реализовать комплексные функции [аутентификации](/end-user-flows) и [управления пользователями](/user-management), интегрировав Logto.

## Использование Logto как провайдера идентификации \{#use-logto-as-an-identity-provider}

Когда вы хотите, чтобы сторонние приложения могли аутентифицироваться с вашей системой идентификации Logto, вы можете настроить Logto как [провайдера идентификации (IdP)](https://auth.wiki/identity-provider).

Logto может использоваться как IdP через стандартные протоколы, такие как OAuth, OIDC, и внешние сервисы могут реализовать "Вход через Logto" так же, как "Вход через Google", позволяя их пользователям аутентифицироваться с использованием вашей системы пользователей Logto.

Создавая сторонние приложения в Logto, вы устанавливаете безопасные соединения между вашей системой идентификации и внешними сервисами.

## Расширение возможностей Logto с помощью Management API \{#extend-logto-features-with-management-api}

Logto [Management API](https://openapi.logto.io/) позволяет программно контролировать и автоматизировать функциональность Logto в ваших приложениях.

Когда вам нужно расширить возможности Logto для улучшения вашего продукта (например, [управление мультитенантными организациями](/organizations) или автоматизация рабочих процессов с Logto), вы можете использовать Logto Management API для создания расширенных функций, соответствующих потребностям вашего продукта.
