# Настройка брендинга экрана согласия

Важно убедиться, что информация о бренде и ссылка на условия третьей стороны правильно отображаются пользователям, когда они перенаправляются на экран согласия приложения третьей стороны.

Logto позволяет настраивать информацию о бренде ваших приложений третьей стороны, включая название приложения, логотип и ссылку на условия.

## Настройка информации о бренде \{#customize-the-branding-information}

Убедитесь, что информация о бренде ваших приложений третьей стороны правильно настроена, чтобы обеспечить последовательный и безопасный опыт аутентификации для ваших пользователей.

1. Перейдите в <CloudLink to="/applications/third-party-applications">Консоль > Приложение > Приложения третьих сторон</CloudLink> и откройте страницу деталей для конкретного OIDC приложения третьей стороны.

2. Перейдите на вкладку **Брендинг**.

3. Настройте информацию для отображения на экране согласия:

- **Отображаемое имя**: Имя приложения третьей стороны, которое будет отображаться на экране согласия. Оно будет представлять имя приложения третьей стороны, запрашивающего доступ к информации ваших пользователей. **Имя приложения** будет использоваться, если это поле оставлено пустым.
- **Логотип приложения (Светлый)**: Логотип приложения третьей стороны, который будет отображаться на экране согласия. Он будет представлять бренд приложения третьей стороны, запрашивающего доступ к информации ваших пользователей. Логотип приложения третьей стороны и ваш универсальный логотип опыта входа будут отображаться на экране согласия, если оба предоставлены.
- **Логотип приложения (Темный)**: Доступно только при включенном режиме входа в темной теме. Управляйте настройками темной темы на странице <CloudLink to="/sign-in-experience/branding">Консоль > Опыт входа > Брендинг</CloudLink>.
- **URL условий использования**: Ссылка на условия приложения третьей стороны, которая будет отображаться на экране согласия.
- **URL политики конфиденциальности**: Ссылка на политику конфиденциальности приложения третьей стороны, которая будет отображаться на экране согласия.
