# Кастомизация

Сделайте Logto своим, настроив каждый аспект - от брендинга и пользовательского интерфейса до языков и настроек домена.

## Готовый UI \{#prebuilt-ui}

Начните с наших готовых компонентов пользовательского интерфейса и настройте основные элементы бренда. Создайте либо:

- Единый опыт аутентификации на всей вашей платформе.
- Специальные страницы входа для конкретных приложений или организаций.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Соответствие вашему бренду',
      href: '/customization/match-your-brand',
      description:
        'Узнайте, как настроить основные элементы бренда для готового к использованию опыта входа.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## Кастомный UI \{#custom-ui}

Получите полный контроль над опытом аутентификации с двумя вариантами:

## Контент и языки \{#content--languages}

Logto по умолчанию поддерживает 15 языков. Расширьте поддержку языков или измените существующие переводы в соответствии с вашими потребностями.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Языки и контент',
      href: '/customization/localized-languages',
      description: 'Узнайте, как добавить новые языки и настроить контент для ваших пользователей.',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## Кастомный домен \{#custom-domain}

Укрепите присутствие вашего бренда, используя ваш домен для URL аутентификации. Замените домен Logto по умолчанию на ваш брендированный домен.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Кастомный домен',
      href: '/logto-cloud/custom-domain',
      description:
        'Улучшите брендинг и доверие пользователей, используя ваш собственный домен в качестве URL для входа или регистрации.',
      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>

## Живой предпросмотр \{#live-preview}

Экспериментируйте с различными потоками аутентификации ([регистрация](/end-user-flows/sign-up-and-sign-in/sign-up), [вход](/end-user-flows/sign-up-and-sign-in/sign-in), [социальный вход](/end-user-flows/sign-up-and-sign-in/social-sign-in), [забыли пароль](/end-user-flows/sign-up-and-sign-in/reset-password), [MFA](/end-user-flows/mfa), [SSO](/end-user-flows/enterprise-sso) и т.д.) с помощью нашей функции Живого Предпросмотра. Тестируйте ваш опыт входа мгновенно через наше встроенное демо-приложение.
