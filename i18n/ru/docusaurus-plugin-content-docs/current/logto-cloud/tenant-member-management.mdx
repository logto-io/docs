# Управление участниками арендатора

## Приглашение сотрудников \{#invite-collaborators}

Перейдите в <CloudLink to="/tenant-settings/members">Консоль > Настройки арендатора > Участники</CloudLink>. Вы увидите текущих участников и сможете пригласить новых. Введите адреса электронной почты и назначьте роли (вы можете сделать это массово), а затем отправьте приглашение.

Logto позволяет приглашать членов команды в ваш арендатор с двумя различными ролями в Logto Cloud:

- **Администратор:** Полный доступ и управление всеми ресурсами арендатора.
- **Сотрудник:** Может управлять ресурсами, связанными с идентификацией, но с ограничениями:
  - Нет доступа к информации о выставлении счетов
  - Не может приглашать или управлять другими участниками
  - Не может изменять имя арендатора
  - Не может удалять арендатора

### Ограничение по количеству участников \{#members-limit}

| Тип арендатора      | Ограничение по количеству участников                          |
| ------------------- | ------------------------------------------------------------- |
| Dev арендатор       | До 3 участников (свяжитесь с нами для увеличения)             |
| Бесплатный продакшн | 1 участник (только администратор)                             |
| Pro продакшн        | Начинается с 3 участников, затем $8 за каждого доп. участника |

### Управление приглашениями \{#manage-invitation}

Вкладка приглашений показывает прошлые приглашения с их статусом

| Статус     | Объяснение                                          | Доступное действие |
| ---------- | --------------------------------------------------- | ------------------ |
| В ожидании | Приглашение отправлено, но еще не принято.          | Повторно отправить |
| Принято    | Пользователь теперь является участником арендатора. | Удалить            |
| Истекло    | Приглашение превысило 7-дневный лимит.              | Удалить            |
| Отозвано   | Администратор отменил приглашение.                  | Удалить            |

Вы не можете приглашать людей, у которых уже есть ожидающие приглашения или которые уже являются участниками.

### Управление участниками арендатора (только администратор) \{#manage-tenant-members-admin-only}

Как администратор, у вас есть несколько способов управления вашей командой:

- **Приглашение или удаление участников:** Добавляйте новых членов команды или удаляйте доступ по мере необходимости.
- **Изменение ролей участников:** Назначайте роли 'администратор' или 'сотрудник' в зависимости от необходимого уровня доступа.
- **Передача владения арендатором:** При необходимости вы можете передать владение арендатором другому участнику. Вы можете сначала дать роль администратора другому участнику, а затем выбрать, оставаться обычным участником или полностью покинуть арендатора.
- **Добавление нескольких администраторов:** Разделите административные обязанности с другими членами команды для совместного управления.
