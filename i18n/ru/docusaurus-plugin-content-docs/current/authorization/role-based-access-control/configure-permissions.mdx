# Настройка разрешений

## Настройка разрешений API через Logto Console \{#configure-api-permissions-via-logto-console}

Разрешения устанавливаются на уровне **ресурса API**. Перейдите в <CloudLink to="/api-resources">Console > API resources</CloudLink>, нажмите на ресурс API, который вы хотите управлять, а затем перейдите на вкладку "Permissions".

Важно указать имя и детали для разрешения, так как это упростит идентификацию и управление разрешениями.

## Настройка разрешений API через Logto Management API \{#configure-api-permissions-via-logto-management-api}

Управляйте с помощью Logto Management API. Сделайте вызов к соответствующей конечной точке. Ознакомьтесь с этой [справкой](https://openapi.logto.io/operation/operation-listresourcescopes).

| метод  | путь                                                                                                                 | описание                                                                                              |
| ------ | -------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| GET    | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-listresourcescopes)              | Получить области действия, определенные для ресурса API.                                              |
| POST   | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-createresourcescope)             | Создать новую область действия для ресурса API.                                                       |
| DELETE | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-deleteresourcescope) | Удалить область действия ресурса API из данного ресурса.                                              |
| PATCH  | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-updateresourcescope) | Обновить область действия ресурса API для данного ресурса. Этот метод выполняет частичное обновление. |
