# Коннекторы

Коннекторы являются мостом между Logto и внешними сервисами. Они позволяют использовать методы [аутентификации без пароля](https://auth.wiki/passwordless), такие как [проверка электронной почты](/end-user-flows/sign-up-and-sign-in/sign-in), [проверка SMS](/end-user-flows/sign-up-and-sign-in/sign-in), [социальный вход](/end-user-flows/sign-up-and-sign-in/social-sign-in) и [корпоративный единый вход (SSO)](/end-user-flows/enterprise-sso).

Традиционная регистрация аккаунта с использованием имен пользователей и паролей может быть утомительным и раздражающим процессом. Пользователи часто испытывают трудности с запоминанием сложных паролей и могут неохотно создавать новые аккаунты из-за неудобства. Если пользователи могут предоставить свою электронную почту или номер телефона и подтвердить его с помощью кодов подтверждения; или войти, используя аккаунты, такие как Google / Facebook, или корпоративные аккаунты, такие как Microsoft, это может значительно упростить процесс входа и уменьшить отток пользователей. Коннектор был создан для решения этой проблемы.
