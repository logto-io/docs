# Конвенции Platform SDK

Logto предоставляет очень мощный и гибкий веб-сервис аутентификации.

На практике использования сервисов Logto, для удобства, часто необходимо, чтобы разработчики интегрировали Logto SDK в свои клиентские приложения для управления статусом входа пользователей, разрешениями и другими функциями.

Вы можете найти SDK для всех языков программирования / фреймворков, поддерживаемых Logto, [здесь](/quick-starts).

Если вам не повезло и вы не нашли нужный SDK, вот конвенция, которую вы можете использовать для реализации SDK для вашего желаемого языка программирования, что облегчит использование сервисов Logto.

Эта конвенция содержит три основные части:
