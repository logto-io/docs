# Стратегия проектирования

- Каждый язык программирования должен иметь изолированный git-репозиторий с именем `${language}`.
- Каждый репозиторий языка программирования должен быть моно-репозиторием.
- Оба SDK и их связанные примерные проекты должны быть размещены в этом репозитории.

Примеры:

<details>
  <summary>

### JS \{#js}

</summary>

- js (core)
- react
- react-sample

</details>

<details>
  <summary>

### Kotlin \{#kotlin}

</summary>

- kotlin (core)
- android
- android-sample

</details>
