# Разработчик

Logto — это сервис [управления идентификацией и доступом (IAM)](https://auth.wiki/iam), основанный на протоколах [OAuth 2](https://auth.wiki/oauth-2.0) и [OIDC](https://auth.wiki/openid-connect). Сервисы IAM, такие как Logto, часто служат основой для других веб-сервисов; различные состояния авторизации в этих веб-сервисах напрямую зависят от Logto.

Для удобства наших пользователей Logto предлагает серию часто используемых функций для разработчиков.
