# Введение в Logto

Добро пожаловать в документацию Logto! Logto — это решение для управления идентификацией и доступом (IAM), разработанное для современных приложений и SaaS-продуктов. Оно предоставляет безопасную, масштабируемую и настраиваемую систему аутентификации и авторизации для ваших приложений.
