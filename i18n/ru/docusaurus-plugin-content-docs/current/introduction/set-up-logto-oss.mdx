# Настройка Logto OSS

Этот раздел охватывает основные шаги настройки и ключевые действия для эффективной доставки вашего продукта и управления вашим рабочим процессом разработки для [Logto с открытым исходным кодом (OSS)](https://github.com/logto-io/logto).

## Начало работы с Logto OSS \{#get-started-with-logto-oss}

Следуйте [руководству по началу работы](/logto-oss/get-started-with-oss/), чтобы запустить Logto сегодня, а позже вы можете обратиться к полному [руководству по развертыванию](/logto-oss/deployment-and-configuration) для использования в производстве.

## Функции, поддерживаемые Logto OSS \{#feature-supported-by-logto-oss}

Logto OSS поддерживает большинство основных возможностей сервиса Logto и регулярно обновляется и поддерживается.

Некоторые расширенные функции в настоящее время эксклюзивны для версии Logto Cloud, включая [Logto Protected App](/integrate-logto/protected-app), [бесплатный сервис электронной почты](/connectors/email-connectors/built-in-email-service), [IdP-инициированный единый вход (SSO)](/end-user-flows/enterprise-sso/idp-initiated-sso) и [SAML IdP](/integrate-logto/third-party-applications).

## Оставайтесь в курсе обновлений Logto \{#stay-updated-with-logto-releases}

Чтобы ваша инстанция Logto была обновлена до последних функций, обязательно следите за страницей [Logto GitHub Releases](https://github.com/logto-io/logto/releases), где вы можете найти все журналы выпусков.

Прочтите [руководство по обновлению](/logto-oss/upgrading-oss-version), чтобы узнать, как обновить Logto без изменения вашего кода или схемы базы данных.

## Вклад в Logto OSS \{#contributing-to-logto-oss}

Спасибо за ваш интерес к вкладу в Logto! Вот [руководство по вкладу](/logto-oss/contribution).
