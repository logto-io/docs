После того как пользователь отправляет запрос на выход, Logto очистит всю информацию аутентификации пользователя в сессии.

Чтобы очистить PHP-сессию и сессию Logto, маршрут выхода можно реализовать следующим образом:

<CodeBlock language="php">
  {`Route::get('/sign-out', function () {
  return redirect(
    // Перенаправить пользователя на главную страницу после успешного выхода
    $client->signOut('${defaultBaseUrl}')
  );
});`}
</CodeBlock>
