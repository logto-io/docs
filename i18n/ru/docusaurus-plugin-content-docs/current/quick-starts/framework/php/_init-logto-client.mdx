Сначала создайте конфигурацию Logto:

```php title="index.php"
use Logto\Sdk\LogtoClient;
use Logto\Sdk\LogtoConfig;

$client = new LogtoClient(
  new LogtoConfig(
    endpoint: "https://you-logto-endpoint.app",
    appId: "replace-with-your-app-id",
    appSecret: "replace-with-your-app-secret",
  ),
);
```

По умолчанию SDK использует встроенную PHP сессию для хранения данных Logto. Если вы хотите использовать другое хранилище, вы можете передать объект пользовательского хранилища в качестве второго параметра:

```php title="index.php"
$client = new LogtoClient(
  new LogtoConfig(
    // ...
  ),
  new YourCustomStorage(),
);
```

Смотрите [Storage](https://github.com/logto-io/php/blob/master/docs/api/classes/Logto/Sdk/Storage/Storage.md) для получения более подробной информации.
