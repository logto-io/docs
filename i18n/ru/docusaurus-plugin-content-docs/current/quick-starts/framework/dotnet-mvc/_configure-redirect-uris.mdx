Сначала давайте настроим **Logto redirect URI**. Добавьте следующий URI в список "Redirect URIs" на странице сведений о приложении Logto:

<CodeBlock>{`${defaultBaseUrl}Callback`}</CodeBlock>

Чтобы настроить **Logto post sign-out redirect URI**, добавьте следующий URI в список "Post sign-out redirect URIs" на странице сведений о приложении Logto:

<CodeBlock>{`${defaultBaseUrl}SignedOutCallback`}</CodeBlock>

#### Изменение путей по умолчанию \{#change-the-default-paths}

**Logto redirect URI** имеет путь по умолчанию `/Callback`, а **Logto post sign-out redirect URI** имеет путь по умолчанию `/SignedOutCallback`.

Вы можете оставить их как есть, если нет особых требований. Если вы хотите изменить их, вы можете установить свойства `CallbackPath` и `SignedOutCallbackPath` для `LogtoOptions`:

```csharp title="Program.cs"
builder.Services.AddLogtoAuthentication(options =>
{
  // Другие настройки...
  // highlight-start
  options.CallbackPath = "/Foo";
  options.SignedOutCallbackPath = "/Bar";
  // highlight-end
});
```

Не забудьте обновить значение на странице сведений о приложении Logto соответствующим образом.
