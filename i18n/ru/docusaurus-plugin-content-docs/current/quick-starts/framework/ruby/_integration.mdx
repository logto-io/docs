### Инициализация Logto клиента \{#initialize-logto-client}

В файле, где вы хотите инициализировать Logto клиент (например, в базовом контроллере или middleware), добавьте следующий код:

```ruby
require "logto/client"

@client = LogtoClient.new(
  config: LogtoClient::Config.new(
    endpoint: "https://your-logto-endpoint.com",
    app_id: "your-logto-app-id",
    app_secret: "your-logto-app-secret"
  ),
  navigate: ->(uri) { a_redirect_method(uri) },
  storage: LogtoClient::SessionStorage.new(the_session_object)
)
end
```

Например, в контроллере Rails код может выглядеть так:

```ruby title="app/controllers/sample_controller.rb"
require "logto/client"

class SampleController < ApplicationController
  before_action :initialize_logto_client

  private

  def initialize_logto_client
    @client = LogtoClient.new(
      config: LogtoClient::Config.new(
        # ...ваша конфигурация
      ),
      # Позволяет клиенту перенаправлять на другие хосты (т.е. ваш Logto tenant)
      navigate: ->(uri) { redirect_to(uri, allow_other_host: true) },
      # Контроллер имеет доступ к объекту сессии
      storage: LogtoClient::SessionStorage.new(session)
    )
  end
end
```

### Обработка callback \{#handle-the-callback}

Поскольку redirect URI был установлен на `http://localhost:3000/callback`, его необходимо обработать в нашем приложении. В контроллере Rails вы можете добавить следующий код:

```ruby title="app/controllers/sample_controller.rb"
class SampleController < ApplicationController
  # highlight-start
  def callback
    @client.handle_sign_in_callback(url: request.original_url)
  end
  # highlight-end
end
```

И настройте маршрут в `config/routes.rb`:

```ruby title="config/routes.rb"
Rails.application.routes.draw do
  # highlight-next-line
  get "/callback", to: "sample#callback"
end
```

### Вызов входа и выхода \{#invoke-sign-in-and-sign-out}

Существует несколько способов вызова входа и выхода в вашем приложении. Например, вы можете реализовать два маршрута в вашем приложении Rails:

```ruby title="app/controllers/sample_controller.rb"
class SampleController < ApplicationController
  # highlight-start
  def sign_in
    @client.sign_in(redirect_uri: request.base_url + "/callback")
  end

  def sign_out
    @client.sign_out(post_logout_redirect_uri: request.base_url)
  end
  # highlight-end

  # ...
end
```

```ruby title="config/routes.rb"
Rails.application.routes.draw do
  # highlight-start
  get "/sign_in", to: "sample#sign_in"
  get "/sign_out", to: "sample#sign_out"
  # highlight-end

  # ...
end
```

Затем вы можете создать кнопки или ссылки в ваших представлениях для запуска этих действий. Например:

```erb title="app/views/sample/index.html.erb"
<% if @client.is_authenticated? %>
  <a href="<%= sign_out_path %>">Выйти</a>
<% else %>
  <a href="<%= sign_in_path %>">Войти</a>
<% end %>
```
