### Отображение информации о пользователе \{#display-user-information}

Чтобы отобразить информацию о пользователе, вы можете использовать метод `@client.id_token_claims`. Например, в представлении:

```erb title="app/views/sample/index.html.erb"
<% if @client.is_authenticated? %>
  <p>Добро пожаловать, <%= @client.id_token_claims["name"] %></p>
<% else %>
  <p>Пожалуйста, войдите в систему</p>
<% end %>
```

Пожалуйста, обратитесь к методу `#id_token_claims` в [gemdocs](https://gemdocs.org/gems/logto/latest) для получения дополнительной информации.

Чтобы запросить дополнительные области действия, вы можете настроить опцию `scopes` в объекте `LogtoClient::Config`:

```ruby
require "logto/client"

@client = LogtoClient.new(
  config: LogtoClient::Config.new(
    # ...другие настройки
    # highlight-next-line
    scopes: ["email", "phone"] # Добавьте больше областей действия по мере необходимости
  ),
  # ...другие настройки
)
```

Затем вы можете получить доступ к дополнительным утверждениям через `id_token_claims`:

```erb title="app/views/sample/index.html.erb"
<% if @client.is_authenticated? %>
  <p>Имя: <%= @client.id_token_claims["name"] %></p>
  <p>Email: <%= @client.id_token_claims["email"] %></p>
  <p>Телефон: <%= @client.id_token_claims["phone"] %></p>
<% else %>
  <p>Пожалуйста, войдите в систему</p>
<% end %>
```
