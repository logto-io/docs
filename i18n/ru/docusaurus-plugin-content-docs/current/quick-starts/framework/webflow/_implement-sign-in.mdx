#### Реализация кнопки входа \{#implement-a-sign-in-button}

Вернитесь к вашему дизайнеру Webflow, перетащите кнопку "Sign in" на главную страницу и назначьте ей ID “sign-in” для последующего использования с помощью `getElementById()`.

```html
<script type="module">
  const signInButton = document.getElementById('sign-in');
  const onClickSignIn = () => logtoClient.signIn('https://your-awesome-site.webflow.io/callback');
  signInButton.addEventListener('click', onClickSignIn);
</script>
```
