В Logto SDK, обычно мы можем использовать метод `logtoClient.isAuthenticated()` для проверки статуса аутентификации. Если пользователь вошел в систему, значение будет `true`; в противном случае оно будет `false`.

На вашем сайте Webflow вы также можете использовать его для программного отображения и скрытия кнопок входа и выхода. Примените следующий пользовательский код, чтобы соответствующим образом настроить CSS кнопок.

```js
const isAuthenticated = await logtoClient.isAuthenticated();

signInButton.style.display = isAuthenticated ? 'none' : 'block';
signOutButton.style.display = isAuthenticated ? 'block' : 'none';
```
