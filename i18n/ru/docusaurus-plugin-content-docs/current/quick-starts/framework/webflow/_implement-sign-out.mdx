Вызов `.signOut()` очистит все данные Logto в памяти и localStorage, если они существуют.

После выхода будет полезно перенаправить вашего пользователя обратно на ваш веб-сайт. Давайте добавим `https://your-awesome-site.webflow.io` в качестве одного из URI после выхода в Admin Console (отображается под Redirect URIs) и используем этот URL в качестве параметра при вызове `.signOut()`.

#### Реализация кнопки выхода \{#implement-a-sign-out-button}

Вернитесь в дизайнер Webflow и добавьте кнопку «Выйти» на вашу домашнюю страницу. Аналогично, присвойте кнопке ID «sign-out» и добавьте следующий код в пользовательский код на уровне страницы.

```js
const signOutButton = document.getElementById('sign-out');
const onClickSignOut = () => logtoClient.signOut('https://your-awesome-site.webflow.io');
signOutButton.addEventListener('click', onClickSignOut);
```
