:::note
В следующих шагах мы предполагаем, что ваш сайт на Webflow работает на `https://your-awesome-site.webflow.io`.
:::

На этом этапе мы добавим пользовательский код на глобальном уровне на ваш сайт Webflow. Поскольку NPM не поддерживается в Webflow, мы будем использовать сервис CDN [jsdelivr.com](https://www.jsdelivr.com/) для импорта Logto SDK.

Откройте страницу "Настройки сайта" и перейдите в раздел "Пользовательский код". Добавьте следующий код в раздел "Код в заголовке".

```html
<script type="module">
  // Импорт SDK \`@logto/browser\` из CDN jsdelivr
  import LogtoClient from 'https://esm.run/@logto/browser';

  // Присваиваем экземпляр \`logtoClient\` объекту window,
  // чтобы обеспечить глобальное использование на других страницах
  window.logtoClient = new LogtoClient({
    endpoint: '<your-logto-endpoint>', // Например, http://localhost:3001
    appId: '<your-application-id>',
  });
</script>
```
