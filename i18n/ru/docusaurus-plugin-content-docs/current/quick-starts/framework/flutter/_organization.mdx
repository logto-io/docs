### Получение токена доступа для организаций \{#fetch-access-token-for-organizations}

Так же, как и для ресурсов API, вы можете запросить токен доступа для организаций. Это полезно, когда вам нужно получить доступ к ресурсам, которые определены с использованием области действия организации вместо области действия ресурса API.

<FetchOrganizationTokenForUser
  organizationScope="LogtoUserScope.Organizations"
  configOrganizationCode={<ConfigOrganizationCode />}
  getOrganizationAccessTokenCode={<GetOrganizationAccessTokenCode />}
/>
