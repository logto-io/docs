Мы также предоставляем встроенный enum `LogtoUserScope` в пакете SDK, чтобы помочь вам использовать предопределенные области действия.

```dart
// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",
  appId: "<your-app-id>",
  // highlight-start
  scopes: [LogtoUserScope.email.value, LogtoUserScope.phone.value],
  // highlight-end
);
```
