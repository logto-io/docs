<TabItem value="pub" label="pub.dev">

Вы можете установить пакет `logto_dart_sdk` напрямую, используя менеджер пакетов pub. Выполните следующую команду в корне вашего проекта:

```bash
flutter pub add logto_dart_sdk
```

Или добавьте следующее в ваш файл `pubspec.yaml`:

```yaml
dependencies:
  logto_dart_sdk: ^3.0.0
```

Затем выполните:

```bash
flutter pub get
```

git clone https://github.com/logto-io/dart
