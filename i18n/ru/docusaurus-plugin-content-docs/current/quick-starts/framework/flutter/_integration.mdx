### Инициализация LogtoClient \{#init-logtoclient}

Импортируйте пакет `logto_dart_sdk` и инициализируйте экземпляр `LogtoClient` в корне вашего приложения.

```dart title="lib/main.dart"
import 'package:logto_dart_sdk/logto_dart_sdk.dart';
import 'package:http/http.dart' as http;

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      title: 'Flutter Demo',
      home: MyHomePage(title: 'Logto Demo Home Page'),
    );
  }
}

class MyHomePage extends StatefulWidget {
  const MyHomePage({Key? key, required this.title}) : super(key: key);
  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  late LogtoClient logtoClient;

  void render() {
    // изменение состояния
  }

  // highlight-start
  // LogtoConfig
  final logtoConfig = const LogtoConfig(
    endpoint: "<your-logto-endpoint>",
    appId: "<your-app-id>"
  );

  void _init() {
    logtoClient = LogtoClient(
      config: logtoConfig,
      httpClient: http.Client(), // Необязательный http клиент
    );
    render();
  }
  // highlight-end

  @override
  void initState() {
    super.initState();
    _init();
  }

  // ...
}
```

### Реализация входа \{#implement-sign-in}

<SignInFlowSummary />

Перед началом вам нужно добавить redirect URI в Admin Console для вашего приложения.

<ConfigureRedirectUri figureSrc={redirectUriFigure} redirectUri="io.logto://callback" />

- Для iOS схема redirect URI не имеет значения, так как класс `ASWebAuthenticationSession` будет слушать redirect URI независимо от того, зарегистрирован он или нет.
- Для Android схема redirect URI должна быть зарегистрирована в файле `AndroidManifest.xml`.

После настройки redirect URI мы добавляем кнопку входа на вашу страницу, которая будет вызывать `logtoClient.signIn` API для запуска процесса входа Logto:

```dart title="lib/main.dart"
class _MyHomePageState extends State<MyHomePage> {
  // ...
  final redirectUri = 'io.logto://callback';

  @override
  Widget build(BuildContext context) {
    // ...

    Widget signInButton = TextButton(
      onPressed: () async {
        // highlight-start
        await logtoClient.signIn(redirectUri);
        render();
        // highlight-end
      },
      child: const Text('Sign In'),
    );

    return Scaffold(
      appBar: AppBar(
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            SelectableText('My Demo App'),
            signInButton,
          ],
        ),
      ),
    );
  }
}
```

### Реализация выхода \{#implement-sign-out}

/>

Теперь давайте добавим кнопку выхода на главную страницу, чтобы пользователи могли выйти из вашего приложения.

```dart title="lib/main.dart"
class _MyHomePageState extends State<MyHomePage> {
  // ...

  final postSignOutRedirectUri = 'io.logto//home';

  @override
  Widget build(BuildContext context) {
    // ...

    Widget signOutButton = TextButton(
      onPressed: () async {
        // highlight-start
        await logtoClient.signOut(postSignOutRedirectUri);
        render();
        // highlight-end
      },
      child: const Text('Sign Out'),
    );

    return Scaffold(
      appBar: AppBar(
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            SelectableText('My Demo App'),
            signInButton,
            signOutButton,
          ],
        ),
      ),
    );
  }
}
```

### Обработка статуса аутентификации \{#handle-authentication-status}

Logto SDK предоставляет асинхронный метод для проверки статуса аутентификации. Метод называется `logtoClient.isAuthenticated`. Метод возвращает логическое значение, `true`, если пользователь аутентифицирован, иначе `false`.

В примере мы условно отображаем кнопки входа и выхода на основе статуса аутентификации. Теперь давайте обновим метод `render` в нашем виджете для обработки изменения состояния:

```dart title="lib/main.dart"
class _MyHomePageState extends State<MyHomePage> {
  // ...
  bool? isAuthenticated = false;

  void render() {
    setState(() async {
      // highlight-start
      isAuthenticated = await logtoClient.isAuthenticated;
      // highlight-end
    });
  }

  @override
  Widget build(BuildContext context) {
    // ...

    return Scaffold(
      appBar: AppBar(
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            SelectableText('My Demo App'),
            isAuthenticated == true ? signOutButton : signInButton,
          ],
        ),
      ),
    );
  }
}
```
