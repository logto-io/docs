Инициализируйте клиент, создав экземпляр `LogtoClient` с объектом `LogtoConfig`.

```swift title="ContentView.swift"
import Logto
import LogtoClient

let config = try? LogtoConfig(
  endpoint: "<your-logto-endpoint>", // Например, http://localhost:3001
  appId: "<your-app-id>"
)
let client = LogtoClient(useConfig: config)
```

:::info

По умолчанию мы храним учетные данные, такие как ID токен и токен обновления, в Keychain. Таким образом, пользователю не нужно снова входить в систему при возвращении.

Чтобы отключить это поведение, установите `usingPersistStorage` в `false`:

```swift
let config = try? LogtoConfig(
  // ...
  usingPersistStorage: false
)
```

:::
