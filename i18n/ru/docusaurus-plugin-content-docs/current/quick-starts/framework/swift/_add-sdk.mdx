Используйте следующий URL, чтобы добавить Logto SDK в качестве зависимости в Swift Package Manager.

```bash
https://github.com/logto-io/swift.git
```

Начиная с Xcode 11, вы можете [непосредственно импортировать Swift пакет](https://developer.apple.com/documentation/swift_packages/adding_package_dependencies_to_your_app) без использования дополнительных инструментов.

Мы не поддерживаем **Carthage** и **CocoaPods** в настоящее время из-за некоторых технических проблем.

<details>

<summary>

### Carthage \{#carthage}

</summary>

Carthage [требует файл `xcodeproj` для сборки](https://github.com/Carthage/Carthage/issues/1226#issuecomment-290931385), но `swift package generate-xcodeproj` выдаст ошибку, так как мы используем бинарные цели для нативных социальных плагинов. Мы постараемся найти обходной путь позже.

</details>

<details>

<summary>

### CocoaPods \{#cocoapods}

</summary>

CocoaPods [не поддерживает локальные зависимости](https://github.com/CocoaPods/CocoaPods/issues/3276) и монорепозитории, поэтому сложно создать `.podspec` для этого репозитория.

</details>
