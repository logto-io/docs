### Отображение информации о пользователе \{#display-user-information}

Когда пользователь вошел в систему, возвращаемое значение `useLogtoUser()` будет объектом, содержащим информацию о пользователе. Вы можете отобразить эту информацию в вашем приложении:

```html title="index.vue"
<script setup lang="ts">
  const user = useLogtoUser();
</script>
<template>
  <!-- highlight-start -->
  <!-- Отображение информации о пользователе при входе в систему -->
  <ul v-if="Boolean(user)">
    <li v-for="(value, key) in user"><b>{{ key }}:</b> {{ value }}</li>
  </ul>
  <!-- highlight-end -->
  <!-- Упрощенная кнопка для входа и выхода -->
  <nuxt-link :to="`/sign-${ user ? 'out' : 'in' }`"> Sign {{ user ? 'out' : 'in' }} </nuxt-link>
</template>
```

Чтобы запросить дополнительные области действия, вы можете настроить параметры модуля `logto`:

```ts title="nuxt.config.ts"
import { UserScope } from '@logto/nuxt';

export default defineNuxtConfig({
  logto: {
    // highlight-next-line
    scopes: [UserScope.Email, UserScope.Phone], // Добавьте больше областей, если необходимо
    // ...другие настройки
  },
});
```

Затем вы можете получить доступ к дополнительным утверждениям в объекте `user`:

```html title="index.vue"
<template>
  <div v-if="user">
    <p>Имя: {{ user.name }}</p>
    <p>Email: {{ user.email }}</p>
    <p>Телефон: {{ user.phone }}</p>
  </div>
</template>
```

### Получение информации о пользователе вручную \{#get-user-information-manually}

Чтобы получить доступ ко всем методам, предоставляемым клиентом Logto, вы можете использовать композицию `useLogtoClient()`:

```ts title="index.vue"
const client = useLogtoClient();
```

:::warning
Клиент Logto доступен только на стороне сервера. Композиция вернет `undefined` на стороне клиента.
:::

<GetUserInfoApis
  getIdTokenClaimsApi="client.getIdTokenClaims()"
  fetchUserInfoApi="client.fetchUserInfo()"
/>

Например, чтобы вручную получить информацию о пользователе:

```ts title="index.vue"
import { useLogtoClient, useState, callOnce } from '#imports';

const client = useLogtoClient();
const userInfo = useState(null);

// Вызовите один раз, чтобы предотвратить выполнение на стороне клиента
await callOnce(async () => {
  if (!client) {
    throw new Error('Клиент Logto недоступен');
  }

  if (!(await client.isAuthenticated())) {
    return;
  }

  try {
    userInfo.value = await client.fetchUserInfo();
  } catch (error) {
    console.error('Не удалось получить информацию о пользователе:', error);
  }
});
```
