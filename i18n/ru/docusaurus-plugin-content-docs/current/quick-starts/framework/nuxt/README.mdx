## Использование в middleware или маршрутах API \{#use-in-middleware-or-api-routes}

Композиции `useLogtoClient()` и `useLogtoUser()` недоступны в middleware или маршрутах API. Вы можете использовать функцию `logtoEventHandler()` для получения клиента Logto и другого контекста:

```ts title="api/auth/get-access-token.ts"
import { logtoEventHandler } from '@logto/nuxt';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
