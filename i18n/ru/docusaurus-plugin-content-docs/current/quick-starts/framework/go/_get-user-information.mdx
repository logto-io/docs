## Получение информации о пользователе \{#get-user-information}

### Отображение информации о пользователе \{#display-user-information}

Чтобы отобразить информацию о пользователе, вы можете использовать метод `client.GetIdTokenClaims`. Например, добавьте маршрут:

```go title="main.go"
func main() {
    //...

    router.GET("/user-id-token-claims", func(ctx *gin.Context) {
      session := sessions.Default(ctx)
      logtoClient := client.NewLogtoClient(logtoConfig, &SessionStorage{session: session})

      // highlight-start
      idTokenClaims, err := logtoClient.GetIdTokenClaims()
      // highlight-end

      if err != nil {
        ctx.String(http.StatusOK, err.Error())
      }

      ctx.JSON(http.StatusOK, idTokenClaims)
    })
}
```

Чтобы запросить дополнительные области действия, вы можете передать их в объект `LogtoConfig`. Например:

```go title="main.go"
logtoConfig := &client.LogtoConfig{
  // ...другие настройки
  // highlight-next-line
  Scopes: []string{"email", "phone"},
}
```

Затем вы можете получить доступ к дополнительным утверждениям в возвращаемом значении `client.GetIdTokenClaims()`:

```go
idTokenClaims, error := client.GetIdTokenClaims()
// Теперь вы можете получить доступ к дополнительным утверждениям `claims.email`, `claims.phone` и т.д.
```
