Аналогично процессу входа, когда пользователь выходит, Logto перенаправит его на URI перенаправления после выхода.

Теперь давайте добавим маршрут `sign-out` для обработки запроса на выход и также добавим ссылку на выход на главной странице:

```go title="main.go"
func main() {
	// ...

	// Добавить ссылку для выполнения запроса на выход на главной странице
	router.GET("/", func(ctx *gin.Context) {
		// ...
		homePage := `<h1>Hello Logto</h1>` +
			"<div>" + authState + "</div>" +
			`<div><a href="/sign-in">Sign In</a></div>` +
			// Добавить ссылку
			`<div><a href="/sign-out">Sign Out</a></div>`

		ctx.Data(http.StatusOK, "text/html; charset=utf-8", []byte(homePage))
	})

	// Добавить маршрут для обработки запросов на выход
	router.GET("/sign-out", func(ctx *gin.Context) {
		session := sessions.Default(ctx)
		logtoClient := client.NewLogtoClient(
			logtoConfig,
			&SessionStorage{session: session},
		)

		// Запрос на выход обрабатывается Logto.
		// Пользователь будет перенаправлен на URI перенаправления после выхода.
		signOutUri, signOutErr := logtoClient.SignOut("http://localhost:3000")

		if signOutErr != nil {
			ctx.String(http.StatusOK, signOutErr.Error())
			return
		}

		ctx.Redirect(http.StatusTemporaryRedirect, signOutUri)
	})

	// ...
}
```

После того как пользователь делает запрос на выход, Logto очистит всю информацию аутентификации пользователя в сессии.
