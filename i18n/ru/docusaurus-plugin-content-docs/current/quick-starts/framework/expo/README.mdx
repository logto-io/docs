## Устранение неполадок \{#troubleshooting}

### Не удается разрешить `@logto/client/shim` \{#unable-to-resolve-logtoclientshim}

:::note

- Для пользователей, использующих версию SDK \<= v0.3.0, пожалуйста, следуйте инструкциям ниже для решения проблемы.
- Для пользователей, использующих версию SDK \>= v0.4.0, пожалуйста, проигнорируйте этот раздел. Эта проблема была исправлена в последней версии.

:::

Для проектов Expo, если вы столкнулись с ошибкой `Unable to resolve "@logto/client/shim" from "node_modules/@logto/rn/lib/index.js"`, вы можете решить ее, добавив следующее в ваш файл `metro.config.js`:

```js title="metro.config.js"
const config = {
  // ...
  resolver: {
    unstable_enablePackageExports: true,
  },
};

module.exports = config;
```

Эта ошибка указывает на то, что пакет `@logto/rn` не может разрешить модуль `@logto/client/shim`.

Поскольку в пакете `@logto/client` использовались экспорты узлов, а экспорты пакетов по умолчанию не включены в Metro bundler, вам нужно включить их вручную.

Смотрите [поддержку экспортов пакетов React Native](https://reactnative.dev/blog/2023/06/21/package-exports-support#enabling-package-exports-beta) для получения более подробной информации.
