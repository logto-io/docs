import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

Установите Logto SDK и зависимые пакеты через ваш любимый менеджер пакетов:

<Tabs>

  <TabItem value="npm" label="npm">

```bash
npm i @logto/rn
npm i expo-crypto expo-secure-store expo-web-browser @react-native-async-storage/async-storage
```

  </TabItem>
  <TabItem value="yarn" label="Yarn">

```bash
yarn add @logto/rn
yarn add expo-crypto expo-secure-store expo-web-browser @react-native-async-storage/async-storage
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm add @logto/rn
pnpm add expo-crypto expo-secure-store expo-web-browser @react-native-async-storage/async-storage
```

  </TabItem>

</Tabs>

Пакет `@logto/rn` является SDK для Logto. Остальные пакеты являются его зависимостями. Они не могут быть указаны как прямые зависимости, потому что Expo CLI требует, чтобы все зависимости для нативных модулей были установлены непосредственно в `package.json` корневого проекта.

:::note
Если вы устанавливаете это в [bare React Native приложении](https://docs.expo.dev/bare/overview), вам следует также следовать этим [дополнительным инструкциям по установке](https://docs.expo.dev/bare/installing-expo-modules/).
:::
