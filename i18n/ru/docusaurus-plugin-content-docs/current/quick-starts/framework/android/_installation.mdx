:::note
Минимально поддерживаемый уровень Android API для Logto Android SDK — уровень 24.
:::

Прежде чем установить Logto Android SDK, убедитесь, что `mavenCentral()` добавлен в конфигурацию вашего репозитория в файле сборки проекта Gradle:

```kotlin title="settings.gradle.kts"
dependencyResolutionManagement {
  repositories {
    mavenCentral()
  }
}
```

Добавьте Logto Android SDK в ваши зависимости:

<Tabs>

<TabItem value="kotlin" label="Kotlin">

```kotlin title="build.gradle.kts"
dependencies {
  implementation("io.logto.sdk:android:1.1.3")
}
```

</TabItem>

<TabItem value="groovy" label="Groovy">

```groovy title="build.gradle"
dependencies {
  implementation 'io.logto.sdk:android:1.1.3'
}
```

</TabItem>

</Tabs>

Поскольку SDK требует доступа в интернет, вам нужно добавить следующее разрешение в ваш файл `AndroidManifest.xml`:

```xml title="AndroidManifest.xml"
<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
  xmlns:tools="http://schemas.android.com/tools">

  <!-- добавьте разрешение на интернет -->
  <uses-permission android:name="android.permission.INTERNET" />

  <!-- другие конфигурации... -->
</manifest>
```
