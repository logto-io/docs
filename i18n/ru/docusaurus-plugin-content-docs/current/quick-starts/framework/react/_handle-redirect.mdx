Поскольку мы используем `http://localhost:3000/callback` в качестве redirect URI, теперь нам нужно правильно его обработать.

Сначала давайте создадим страницу обратного вызова:

```tsx title="pages/Callback/index.tsx"
import { useHandleSignInCallback } from '@logto/react';

const Callback = () => {
  const { isLoading } = useHandleSignInCallback(() => {
    // Выполните что-то, когда закончите, например, перенаправьте на домашнюю страницу
  });

  // Когда идет процесс
  if (isLoading) {
    return <div>Перенаправление...</div>;
  }

  return null;
};
```

Наконец, вставьте код ниже, чтобы создать маршрут `/callback`, который НЕ требует аутентификации:

```tsx title="App.tsx"
// Предполагается использование react-router
<Route path="/callback" element={<Callback />} />
```
