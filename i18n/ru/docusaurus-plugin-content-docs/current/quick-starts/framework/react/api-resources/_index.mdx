### Присоединение токена доступа к заголовкам запроса \{#attach-access-token-to-request-headers}

Поместите токен в поле `Authorization` HTTP-заголовков в формате Bearer (`Bearer YOUR_TOKEN`), и вы готовы к работе.

:::note
Процесс интеграции Bearer Token может варьироваться в зависимости от используемого вами фреймворка или отправителя. Выберите свой способ применения заголовка запроса `Authorization`.
:::
