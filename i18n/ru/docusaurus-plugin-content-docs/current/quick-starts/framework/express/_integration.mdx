### Подготовка конфигураций и необходимых промежуточных программ \{#prepare-configs-and-required-middlewares}

Подготовьте конфигурацию для клиента Logto:

```ts title="app.ts"
import { LogtoExpressConfig } from '@logto/express';

const config: LogtoExpressConfig = {
  appId: '<your-application-id>',
  appSecret: '<your-application-secret>',
  endpoint: '<your-logto-endpoint>', // Например, http://localhost:3001
  baseUrl: '<your-express-app-base-url>', // Например, http://localhost:3000
};
```

SDK требует предварительной настройки [express-session](https://www.npmjs.com/package/express-session).

```ts title="app.ts"
import cookieParser from 'cookie-parser';
import session from 'express-session';

app.use(cookieParser());
app.use(
  session({
    secret: 'random_session_key', // Замените на ваш собственный секрет
    cookie: { maxAge: 14 * 24 * 60 * 60 * 1000 }, // В миллисекундах
  })
);
```

### Регистрация маршрутов \{#register-routes}

SDK предоставляет вспомогательную функцию `handleAuthRoutes` для регистрации 3 маршрутов:

1. `/logto/sign-in`: Вход через Logto.
2. `/logto/sign-in-callback`: Обработка обратного вызова входа.
3. `/logto/sign-out`: Выход через Logto.

Добавьте следующий код в ваше приложение:

```ts title="app.ts"
import { handleAuthRoutes } from '@logto/express';

app.use(handleAuthRoutes(config));
```

### Реализация входа и выхода \{#implement-sign-in-and-sign-out}

С зарегистрированными маршрутами теперь давайте реализуем кнопки входа и выхода на главной странице. Нам нужно перенаправить пользователя на маршрут входа или выхода, когда это необходимо. Чтобы помочь с этим, используйте `withLogto` для внедрения статуса аутентификации в `req.user`.

```ts title="app.ts"
import { withLogto } from '@logto/express';

app.get('/', withLogto(config), (req, res) => {
  res.setHeader('content-type', 'text/html');

  if (req.user.isAuthenticated) {
    res.end(`<div>Привет ${req.user.claims?.sub}, <a href="/logto/sign-out">Выйти</a></div>`);
  } else {
    res.end('<div><a href="/logto/sign-in">Войти</a></div>');
  }
});
```
