### Отображение информации о пользователе \{#display-user-information}

Когда пользователь вошел в систему, значение `req.user` будет объектом, содержащим информацию о пользователе. Вы можете отобразить эту информацию в вашем приложении:

```ts title="app.ts"
import { withLogto } from '@logto/express';

app.get('/', withLogto(config), (req, res) => {
  response.json(request.user.claims);
});
```

Чтобы запросить дополнительные области действия, вы можете добавить их в конфигурацию:

```ts title="app.ts"
import { LogtoExpressConfig, UserScope } from '@logto/express';

const config: LogtoExpressConfig = {
  // highlight-next-line
  scopes: [UserScope.Email, UserScope.Phone], // Добавьте больше областей, если необходимо
  // ...другие настройки
};
```

Затем вы можете получить доступ к дополнительным утверждениям в объекте `req.user`:

```ts title="app.ts"
import { withLogto } from '@logto/express';

app.get('/', withLogto(config), (req, res) => {
  response.end(`User email: ${req.user.claims.email}`);
});
```
