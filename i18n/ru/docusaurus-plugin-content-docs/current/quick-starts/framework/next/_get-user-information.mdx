### Отображение информации о пользователе \{#display-user-information}

Когда пользователь вошел в систему, есть три способа получить информацию о пользователе.

#### Через API запрос на фронтенде \{#through-api-request-in-front-end}

```ts title="pages/index.tsx"
import { type LogtoContext } from '@logto/next';
import { useMemo } from 'react';
import useSWR from 'swr';

const Home = () => {
  const { data } = useSWR<LogtoContext>('/api/logto/user');

  const claims = useMemo(() => {
    if (!data?.isAuthenticated || !data.claims) {
      return null;
    }

    return (
      <div>
        <h2>Утверждения:</h2>
        <table>
          <thead>
            <tr>
              <th>Имя</th>
              <th>Значение</th>
            </tr>
          </thead>
          <tbody>
            {Object.entries(data.claims).map(([key, value]) => (
              <tr key={key}>
                <td>{key}</td>
                <td>{String(value)}</td>
              </tr>
            ))}
          </tbody>
        </table>
      </div>
    );
  }, [data]);

  return (
    <div>
      {claims}
    </div>
  );
};

export default Home;
```

#### Через `getServerSideProps` \{#through-getserversideprops}

```ts title="pages/index.tsx"
import { LogtoContext } from '@logto/next';
import { logtoClient } from '../../libraries/logto';

type Props = {
  user: LogtoContext;
};

const Home = ({ user }: Props) => {
  const claims = useMemo(() => {
    if (!user.isAuthenticated || !user.claims) {
      return null;
    }

    return (
      <div>
        <h2>Утверждения:</h2>
        <table>
          <thead>
            <tr>
              <th>Имя</th>
              <th>Значение</th>
            </tr>
          </thead>
          <tbody>
            {Object.entries(user.claims).map(([key, value]) => (
              <tr key={key}>
                <td>{key}</td>
                <td>{String(value)}</td>
              </tr>
            ))}
          </tbody>
        </table>
      </div>
    );
  }, [user]);

  return (
    <div>
      {claims}
    </div>
  );
};

export default Home;

export const getServerSideProps = logtoClient.withLogtoSsr(async function ({ request }) {
  const { user } = request;

  return {
    props: { user },
  };
});
```

#### В маршруте API \{#in-the-api-route}

```ts title="pages/api/get-user-info.ts"
import { logtoClient } from '../../libraries/logto';

export default logtoClient.withLogtoApiRoute((request, response) => {
  if (!request.user.isAuthenticated) {
    response.status(401).json({ message: 'Unauthorized' });

    return;
  }

  response.json({
    data: request.user.claims,
  });
});
```

Чтобы запросить дополнительные области действия, вы можете настроить параметры при инициализации клиента Logto:

```ts title="libraries/logto.ts"
import LogtoClient, { UserScope } from '@logto/next';

export const logtoClient = new LogtoClient({
  // highlight-next-line
  scopes: [UserScope.Email, UserScope.Phone], // Добавьте больше областей действия, если необходимо
  // ...другие настройки
});
```

Затем вы можете получить доступ к дополнительным утверждениям в ответе контекста:

```ts title="pages/index.tsx"
const Home = () => {
  const { data } = useSWR<LogtoContext>('/api/logto/user');

  const email = data?.claims?.email;

  return (
    <div>
      {email && <p>Email: {email}</p>}
    </div>
  );
};

export default Home;
```

### Получение информации о пользователе вручную \{#fetch-user-information-manually}

Вы можете вручную получить информацию о пользователе в маршруте API:

```ts title="pages/api/get-user-info.ts"
import { logtoClient } from '../../libraries/logto';

export default logtoClient.withLogtoApiRoute(
  (request, response) => {
    if (!request.user.isAuthenticated) {
      response.status(401).json({ message: 'Unauthorized' });

      return;
    }

    response.json({
      userInfo: request.user.userInfo,
    });
  },
  { fetchUserInfo: true }
);
```
