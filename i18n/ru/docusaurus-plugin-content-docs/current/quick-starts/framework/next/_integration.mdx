### Инициализация LogtoClient \{#init-logtoclient}

Импортируйте и инициализируйте LogtoClient:

```ts title="libraries/logto.ts"
import LogtoClient from '@logto/next';

export const logtoClient = new LogtoClient({
  appId: '<your-application-id>',
  appSecret: '<your-app-secret-copied-from-console>',
  endpoint: '<your-logto-endpoint>', // Например, http://localhost:3001
  baseUrl: 'http://localhost:3000',
  cookieSecret: 'complex_password_at_least_32_characters_long',
  cookieSecure: process.env.NODE_ENV === 'production',
});
```

### Подготовка маршрутов API \{#prepare-api-routes}

Подготовьте [маршруты API](https://nextjs.org/docs/pages/building-your-application/routing/api-routes) для подключения к Logto.

Вернитесь в вашу IDE / редактор и используйте следующий код для реализации маршрутов API:

```ts title="pages/api/logto/[action].ts"
import { logtoClient } from '../../../libraries/logto';

export default logtoClient.handleAuthRoutes();
```

Это автоматически создаст 4 маршрута:

1. `/api/logto/sign-in`: Вход с Logto.
2. `/api/logto/sign-in-callback`: Обработка обратного вызова входа.
3. `/api/logto/sign-out`: Выход с Logto.
4. `/api/logto/user`: Проверка, аутентифицирован ли пользователь с Logto, если да, возвращает информацию о пользователе.

### Реализация входа и выхода \{#implement-sign-in-and-sign-out}

Мы подготовили маршруты API, теперь давайте реализуем кнопки входа и выхода на вашей домашней странице. Нам нужно перенаправить пользователя на маршрут входа или выхода, когда это необходимо. Чтобы помочь с этим, используйте `useSWR` для получения статуса аутентификации с `/api/logto/user`.

Ознакомьтесь с [этим руководством](https://swr.vercel.app/docs/getting-started), чтобы узнать больше о `useSWR`.

```tsx title="/pages/index.tsx"
import { type LogtoContext } from '@logto/next';
import useSWR from 'swr';

const Home = () => {
  const { data } = useSWR<LogtoContext>('/api/logto/user');

  return (
    <nav>
      {data?.isAuthenticated ? (
        <p>
          Здравствуйте, {data.claims?.sub},
          <button
            onClick={() => {
              window.location.assign('/api/logto/sign-out');
            }}
          >
            Выйти
          </button>
        </p>
      ) : (
        <p>
          <button
            onClick={() => {
              window.location.assign('/api/logto/sign-in');
            }}
          >
            Войти
          </button>
        </p>
      )}
    </nav>
  );
};

export default Home;
```
