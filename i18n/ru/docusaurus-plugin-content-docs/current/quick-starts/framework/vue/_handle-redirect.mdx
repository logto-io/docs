После того как пользователь будет перенаправлен обратно в ваше приложение из Logto, необходимо выполнить несколько действий. Давайте правильно с этим справимся.

Сначала создадим страницу обратного вызова:

```ts title="views/CallbackView.vue"
import { useHandleSignInCallback } from '@logto/vue';
import router from '@/router';

const { isLoading } = useHandleSignInCallback(() => {
  // Выполните что-то по завершении, например, перенаправьте на главную страницу
});
```

```html
<template>
  <!-- Когда идет обработка -->
  <p v-if="isLoading">Перенаправление...</p>
</template>
```

Вставьте код ниже в ваш маршрут `/callback`, который НЕ требует аутентификации:

```ts title="router/index.ts"
// Предполагается использование vue-router
const router = createRouter({
  routes: [
    {
      path: '/callback',
      name: 'callback',
      component: CallbackView,
    },
  ],
});
```
