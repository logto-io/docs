### Реализация входа и выхода \{#implement-sign-in-and-sign-out}

Мы предоставляем два композита `useHandleSignInCallback()` и `useLogto()`, которые могут помочь вам легко управлять процессом аутентификации.

<SignInNote calling=".signIn()" />

```tsx title="views/HomeView.vue"
import { useLogto } from '@logto/vue';

const { signIn, signOut, isAuthenticated } = useLogto();

const onClickSignIn = () => signIn('http://localhost:3000/callback');
const onClickSignOut = () => signOut('http://localhost:3000');
```

```html
<template>
  <button v-if="!isAuthenticated" @click="onClickSignIn">Войти</button>
  <button v-else @click="onClickSignOut">Выйти</button>
</template>
```
