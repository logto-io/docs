### Отображение информации о пользователе \{#display-user-information}

Чтобы отобразить информацию о пользователе, вы можете использовать метод `getIdTokenClaims()`. Например, в приложении на Capacitor:

```tsx
const userClaims = await logtoClient.getIdTokenClaims();
console.log(userClaims);
```

Чтобы запросить дополнительные области действия, вы можете передать их в объект `LogtoConfig` при инициализации клиента. Например:

```tsx
const logtoConfig = {
  scopes: ['email', 'phone', 'custom_data', 'organizations'],
};
```

Затем вы можете получить доступ к дополнительным утверждениям в возвращаемом значении `client.getIdTokenClaims()`:
