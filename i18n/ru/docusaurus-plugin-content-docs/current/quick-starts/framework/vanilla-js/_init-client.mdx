Импортируйте и инициализируйте экземпляр `LogtoClient`, передав конфигурацию:

```ts title="index.js"
import LogtoClient from '@logto/browser';

const logtoClient = new LogtoClient({
  endpoint: '<your-logto-endpoint>',
  appId: '<your-application-id>',
});
```

`endpoint` и `appId` можно найти на странице деталей приложения в Logto Console.
