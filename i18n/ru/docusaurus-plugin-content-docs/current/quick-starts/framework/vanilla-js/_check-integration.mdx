Теперь вы можете протестировать ваше приложение:

1. Запустите ваше приложение, вы увидите кнопку входа.
2. Нажмите на кнопку входа, SDK инициализирует процесс входа и перенаправит вас на страницу входа Logto.
3. После входа вы будете перенаправлены обратно в ваше приложение и увидите ID пользователя и кнопку выхода.
4. Нажмите на кнопку выхода, чтобы выйти.
