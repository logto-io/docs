### Отображение информации о пользователе \{#display-user-information}

Чтобы отобразить информацию о пользователе, вы можете использовать метод `logtoClient.getIdTokenClaims()`. Например, на вашей домашней странице:

```js title="Home.js"
const userInfo = await logtoClient.getIdTokenClaims();

// Создание таблицы отображения для утверждений ID токена
const table = document.createElement('table');
const thead = document.createElement('thead');
const tr = document.createElement('tr');
const thName = document.createElement('th');
const thValue = document.createElement('th');
thName.innerHTML = 'Name';
thValue.innerHTML = 'Value';
tr.append(thName, thValue);
thead.append(tr);
table.append(thead);

const tbody = document.createElement('tbody');

for (const [key, value] of Object.entries(userInfo)) {
  const tr = document.createElement('tr');
  const tdName = document.createElement('td');
  const tdValue = document.createElement('td');
  tdName.innerHTML = key;
  tdValue.innerHTML = typeof value === 'string' ? value : JSON.stringify(value);
  tr.append(tdName, tdValue);
  tbody.append(tr);
}

table.append(tbody);
```

Чтобы запросить дополнительные области действия, вы можете настроить конфигурации Logto:

```js title="index.js"
// highlight-next-line
import LogtoClient, { UserScope } from '@logto/browser';

const logtoClient = new LogtoClient({
  appId: '<your-application-id>',
  endpoint: '<your-logto-endpoint>',
  // highlight-next-line
  scopes: [UserScope.Email, UserScope.Phone],
});
```

Затем вы можете получить доступ к дополнительным утверждениям в возвращаемом значении `logtoClient.getIdTokenClaims()`:

```ts
const claims = await getIdTokenClaims();
// Теперь вы можете получить доступ к дополнительным утверждениям `claims.email`, `claims.phone` и т. д.
```
