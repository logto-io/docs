После того как пользователь будет перенаправлен обратно в ваше приложение из Logto, необходимо выполнить еще несколько действий. Давайте правильно с этим справимся.

```js title="pages/Callback.js"
const callbackHandler = async (logtoClient) => {
  await logtoClient.handleSignInCallback(window.location.href);

  if (!logtoClient.isAuthenticated) {
    // Обработка неудачного входа
    alert('Не удалось войти');
    return;
  }

  // Обработка успешного входа
  window.location.assign('/');
};
```
