### Добавление Logto hook \{#add-logto-hook}

В вашем файле `hooks.server.ts` добавьте следующий код, чтобы внедрить Logto hook в ваш сервер:

```ts title="hooks.server.ts"
import { handleLogto } from '@logto/sveltekit';

export const handle = handleLogto(
  {
    endpoint: '<your-logto-endpoint>',
    appId: '<your-logto-app-id>',
    appSecret: '<your-logto-app-secret>',
  },
  {
    encryptionKey: '<a-random-string>',
  }
);
```

Поскольку эта информация является конфиденциальной, рекомендуется использовать переменные окружения:

```ts title="hooks.server.ts"
import { handleLogto } from '@logto/sveltekit';
import { env } from '$env/dynamic/private';

export const handle = handleLogto(
  {
    endpoint: env.LOGTO_ENDPOINT,
    appId: env.LOGTO_APP_ID,
    appSecret: env.LOGTO_APP_SECRET,
  },
  {
    encryptionKey: env.LOGTO_COOKIE_ENCRYPTION_KEY,
  }
);
```

Если у вас есть несколько hooks, вы можете использовать [функцию-помощник sequence()](https://kit.svelte.dev/docs/modules#sveltejs-kit-hooks) для их объединения:

```ts title="hooks.server.ts"
import { sequence } from '@sveltejs/kit/hooks';

export const handle = sequence(handleLogto, handleOtherHook);
```

Теперь вы можете получить доступ к клиенту Logto в объекте `locals`. Для TypeScript вы можете добавить тип в `app.d.ts`:

```ts
import type { LogtoClient, UserInfoResponse } from '@logto/sveltekit';

declare global {
  namespace App {
    interface Locals {
      logtoClient: LogtoClient;
      user?: UserInfoResponse;
    }
  }
}
```

Мы обсудим объект `user` позже.

### Реализация входа и выхода \{#implement-sign-in-and-sign-out}

На странице, где вы хотите реализовать вход и выход, определите следующие действия:

```ts title="+page.server.ts"
import type { Actions } from './$types';

export const actions: Actions = {
  signIn: async ({ locals }) => {
    await locals.logtoClient.signIn('http://localhost:3000/callback');
  },
  signOut: async ({ locals }) => {
    await locals.logtoClient.signOut('http://localhost:3000/');
  },
};
```

Затем используйте эти действия в вашем Svelte компоненте:

```html title="+page.svelte"
<form method="POST" action="?/{data.user ? 'signOut' : 'signIn'}">
  <button type="submit">Sign {data.user ? 'out' : 'in'}</button>
</form>
```
