### Утверждения, требующие сетевых запросов \{#claims-that-need-network-requests}

Чтобы предотвратить раздувание ID токена, некоторые утверждения требуют выполнения сетевых запросов для их получения. Например, утверждение `custom_data` не включено в объект пользователя, даже если оно запрошено в областях действия.
