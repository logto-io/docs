import redirectUriFigure from '../assets/web-redirect-uri.png';

export const defaultBaseUrl = 'http://localhost:3000/';
export const defaultRedirectUri = `${defaultBaseUrl}callback`;

#### Настройка URI перенаправления \{#configure-redirect-uris}

Перейдите на страницу сведений о приложении в Logto Console. Добавьте URI перенаправления <code>{props.redirectUri ?? 'http://localhost:3000/callback'}</code>.

<img alt="URI перенаправления в Logto Console" src={redirectUriFigure} width="600px" />

Как и при входе, пользователи должны быть перенаправлены в Logto для выхода из общей сессии. После завершения будет хорошо перенаправить пользователя обратно на ваш веб-сайт. Например, добавьте <code>{defaultBaseUrl}</code> в раздел URI перенаправления после выхода.

Затем нажмите "Сохранить", чтобы сохранить изменения.
