Чтобы получить токен доступа для конкретного ресурса API, вы можете использовать метод <code>{props.getAccessTokenApi}</code>:

{props.getAccessTokenCode}

Этот метод вернет JWT токен доступа, который можно использовать для доступа к ресурсу API, если у пользователя есть соответствующие разрешения. Если текущий кэшированный токен доступа истек, этот метод автоматически попытается использовать токен обновления, чтобы получить новый токен доступа.
