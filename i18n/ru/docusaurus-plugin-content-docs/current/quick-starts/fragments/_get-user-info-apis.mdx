Вы можете использовать эти методы Logto для программного получения информации о пользователе:

- <code>{props.getIdTokenClaimsApi}</code>: Получение информации о пользователе путем декодирования
  локального ID токена. Некоторые утверждения могут быть недоступны.
- <code>{props.fetchUserInfoApi}</code>: Получение информации о пользователе путем отправки запроса
  на [userinfo endpoint](https://openid.net/specs/openid-connect-core-1_0.html#UserInfo).

Важно отметить, что утверждения информации о пользователе, которые могут быть получены, зависят от областей действия, используемых пользователем во время входа, и учитывая производительность и размер данных, ID токен может не содержать всех утверждений пользователя, некоторые утверждения пользователя доступны только в userinfo endpoint (см. соответствующий список ниже).
