### Относительно входа на основе перенаправления \{#regarding-redirect-based-sign-in}

</summary>

1. Этот процесс аутентификации следует протоколу [OpenID Connect (OIDC)](https://openid.net/specs/openid-connect-core-1_0.html), и Logto применяет строгие меры безопасности для защиты входа пользователей.
2. Если у вас есть несколько приложений, вы можете использовать одного и того же провайдера идентификации (Logto). Как только пользователь войдет в одно приложение, Logto автоматически завершит процесс входа, когда пользователь получит доступ к другому приложению.

Чтобы узнать больше о причинах и преимуществах входа на основе перенаправления, см. [Объяснение опыта входа в Logto](/concepts/sign-in-experience).
