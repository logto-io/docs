Прежде чем мы углубимся в детали, вот краткий обзор опыта конечного пользователя. Процесс входа можно упростить следующим образом:

```mermaid
graph LR
    A(Ваше приложение) -->|1. Вызов входа| B(Logto)
    B -->|2. Завершение входа| A
```

1. Ваше приложение вызывает метод входа.
2. Пользователь перенаправляется на страницу входа Logto. Для нативных приложений открывается системный браузер.
3. Пользователь входит в систему и перенаправляется обратно в ваше приложение (настроенное как URI перенаправления).
