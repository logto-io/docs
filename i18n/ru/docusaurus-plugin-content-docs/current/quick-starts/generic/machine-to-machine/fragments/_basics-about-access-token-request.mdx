M2M приложение делает `POST` запрос к конечной точке токена, чтобы получить токен доступа, добавляя следующие параметры в формате `application/x-www-form-urlencoded` в тело HTTP запроса:

- **grant_type**: Должен быть установлен в `client_credentials`
- **resource**: Ресурс, к которому вы хотите получить доступ
- **scope**: Область действия запроса доступа

Также вам нужно включить учетные данные вашего M2M приложения в заголовок запроса к конечной точке токена, чтобы аутентифицировать ваше M2M приложение.

Это достигается путем включения учетных данных приложения в форме [Базовой аутентификации](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Authorization#basic_authentication) в заголовке запроса `Authorization`, где имя пользователя — это App ID, а пароль — это App Secret.
