Использование запрашиваемого токена доступа с встроенным ресурсом Logto Management API `https://[your-tenant-id].logto.app/api` для получения всех приложений в Logto:

<Tabs>
<TabItem value="Node.js" label="Node.js">

```js
const logtoEndpoint = 'https://your.logto.endpoint'; // Замените на ваш Logto endpoint
const accessToken = 'eyJhb...2g'; // Токен доступа

const fetchLogtoApplications = async () => {
  return await fetch(`${logtoEndpoint}/api/applications`, {
    method: 'GET',
    headers: {
      Authorization: `Bearer ${accessToken}`,
    },
  });
};
```

</TabItem>

<TabItem value="cURL" label="cURL">

```bash
curl --location \
  --request GET 'https://your.logto.endpoint/api/applications' \
  --header 'Authorization: Bearer eyJhbG...2g'
```

Не забудьте заменить фактические значения на ваши собственные. Значение после `Bearer` должно быть токеном доступа (JWT), который вы получили.

</TabItem>

</Tabs>
