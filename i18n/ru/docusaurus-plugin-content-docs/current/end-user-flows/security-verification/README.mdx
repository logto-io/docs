# Проверка безопасности

Когда аутентифицированные пользователи пытаются выполнить чувствительные или высокорисковые действия, такие как изменение пароля, совершение платежа или доступ к конфиденциальной информации, такой как зарплатные ведомости или данные банковского счета, необходимы дополнительные меры безопасности. Этот процесс особенно важен в чувствительных приложениях, таких как банковские, медицинские и государственные сервисы.

Этот процесс, известный как _проверка безопасности_, требует от пользователей повторной проверки их личности, чтобы подтвердить, что они являются авторизованными владельцами учетной записи. Реализуя проверку безопасности, система усиливает защиту от несанкционированного доступа, добавляя критический уровень безопасности для высокорисковых действий и помогая защитить конфиденциальные данные.
