# Реализация проверки безопасности с помощью Management API

## Проверка пароля \{#password-verification}

Когда пользователи пытаются изменить свой пароль, система должна предложить им повторно ввести текущий пароль для подтверждения их личности. Этот шаг гарантирует, что только авторизованный владелец аккаунта может изменить пароль, предотвращая несанкционированный доступ к аккаунту.

| метод | путь                                                                                                     | описание                                          |
| ----- | -------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| POST  | [/api/users/\{userId\}/password/verify](https://openapi.logto.io/operation/operation-verifyuserpassword) | Проверка текущего пароля пользователя по user ID. |

## Проверка одноразового кода по электронной почте / SMS \{#emailsms-one-time-code-verification}

Отправьте код подтверждения на электронную почту или номер телефона пользователя и предложите ему ввести код для подтверждения своей личности. Эти конечные точки могут использоваться для проверки личности пользователя или для подтверждения владения конкретным адресом электронной почты или номером телефона. Этот шаг проверки настоятельно рекомендуется, когда пользователь пытается связать новый адрес электронной почты или номер телефона с своим аккаунтом, обеспечивая подлинность предоставленной информации.

| метод | путь                                                                                                               | описание                                                       |
| ----- | ------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------- |
| POST  | [/api/verification/verification-codes](https://openapi.logto.io/operation/operation-createverificationcode)        | Отправка кода подтверждения по электронной почте или телефону. |
| POST  | [/api/verification/verification-codes/verify](https://openapi.logto.io/operation/operation-verifyverificationcode) | Проверка электронной почты или телефона по коду подтверждения. |
