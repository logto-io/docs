# Настройки аккаунта

Logto предоставляет две коллекции API настроек аккаунта, позволяющие пользователям управлять своими аккаунтами и профилями, хранящимися в Logto.

## API управления пользователями \{#user-management-apis}

Logto Management API предоставляет наиболее полную функциональность для управления пользователями и их данными профиля. Вы можете реализовать систему управления пользователями с собственным хостингом или страницу настроек аккаунта конечного пользователя, используя эти API.

## Account API \{#account-apis}

Помимо Management API, Logto также предоставляет набор API аккаунта для конечных пользователей, позволяя им напрямую управлять настройками аккаунта и данными профиля. В отличие от Management API, Account API предназначены для использования в контексте конечного пользователя с детальным контролем разрешений пользователя. Вы можете использовать Account API непосредственно в вашем клиентском приложении для создания удобной страницы настроек аккаунта.
