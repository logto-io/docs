# Опыт организации

Опыт [Организации](/organizations) относится к пользовательским интерфейсам и потокам, используемым вашими бизнес-клиентами и их сотрудниками, особенно в [мультитенантных приложениях](https://auth.wiki/multi-tenancy). Logto предоставляет несколько руководств, которые помогут вам быстро интегрировать этот опыт в ваше приложение с использованием Logto Management API.
