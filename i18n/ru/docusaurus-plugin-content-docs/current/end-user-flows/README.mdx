# Потоки конечных пользователей

Потоки конечных пользователей охватывают все процессы проверки для взаимодействий с пользователями, которые классифицируются следующим образом:

- **Потоки аутентификации**: Включает вход пользователя, регистрацию и сброс пароля. Logto предлагает готовые решения, настроенные в <CloudLink to="/sign-in-experience">Консоль > Опыт входа</CloudLink>, и вы можете [добавить свой интерфейс](/customization/bring-your-ui) через [Experience APIs](https://openapi.logto.io/group/endpoint-experience).
- **Потоки учетной записи**: Включает настройки учетной записи, профиль пользователя и проверку безопасности. Используйте [Account APIs](https://openapi.logto.io/group/endpoint-account-center) или [Management APIs](https://openapi.logto.io/operation/operation-getuser) для реализации.
- **Потоки организации**: Для продуктов с мультитенантностью, таких как B2B или SaaS сервисы. Включает создание организации, приглашение участников и управление. Используйте [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) для интеграции этого в ваши приложения.
