# Logto OSS

Logto предлагает как open-source (OSS), так и [cloud](/logto-cloud) версии. Этот раздел будет сосредоточен на OSS решении: <Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS — это бесплатное решение для идентификации, которое предоставляет разработчикам полные возможности для кастомизации и полный контроль над их данными. Logto OSS предлагает гибкость в адаптации кодовой базы для удовлетворения специфических нужд, это идеальный выбор для тех, кто придает приоритет суверенитету данных и хочет получить индивидуальное решение провайдера идентификации, которое может расти вместе с их приложением.

## Функции, поддерживаемые Logto OSS \{#feature-supported-by-logto-oss}

Logto OSS поддерживает большинство основных возможностей сервиса Logto и регулярно обновляется и поддерживается.

Некоторые расширенные функции в настоящее время эксклюзивны для версии Logto Cloud, включая [Logto Protected App](/integrate-logto/protected-app/), [бесплатный email сервис](/connectors/email-connectors/built-in-email-service), [bring your UI](/customization/bring-your-ui), [инициированный IdP единый вход (SSO)](/end-user-flows/enterprise-sso/idp-initiated-sso) и SAML IdP.
