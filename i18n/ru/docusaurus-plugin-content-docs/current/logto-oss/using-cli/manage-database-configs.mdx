# Управление конфигурациями базы данных

Logto помогает поддерживать некоторые технические конфигурации, такие как ключи OIDC и Cookie. Прежде чем управлять ими через Admin Console или Management API, вы можете использовать CLI для достижения цели.

## Получение конфигурации по ключу \{#get-config-by-key}

Использование:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get <key> [keys...]
```

  </TabItem>

</Tabs>

Допустим, вы хотите получить приватные ключи OIDC:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get oidc.privateKeys
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get oidc.privateKeys
```

  </TabItem>

</Tabs>

Вывод будет следующим:

```bash
oidc.privateKeys=["\n-----BEGIN PRIVATE KEY-----\nMIIJRAIBA..."]
```

Запустите `logto db config get --help` для получения всех доступных ключей или смотрите [Конфигурация](/concepts/core-service/configuration/#database-configs) для подробного объяснения.

## Установка конфигурации по ключу \{#set-config-by-key}

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set <key> [keys...]
```

  </TabItem>

</Tabs>

Допустим, вы хотите установить ключи Cookie OIDC:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>

</Tabs>

Обычно значение, которое нужно установить, должно быть допустимой строкой JSON и соответствовать предопределенному определению типа. Запустите `logto db config set --help` для получения всех доступных ключей или смотрите [Конфигурация](/concepts/core-service/configuration/) для подробного объяснения.
