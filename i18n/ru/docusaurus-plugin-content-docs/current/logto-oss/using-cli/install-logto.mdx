# Установка Logto

## Интерактивная установка \{#interactive}

Выполните команду ниже для интерактивной установки в вашем терминале:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## Опции и тихая установка \{#options-and-silent-installation}

В качестве альтернативы, вы можете добавить опции ниже, чтобы пропустить некоторые вопросы:

```bash
# Все эти опции являются необязательными
--db-url      URL Postgres для базы данных Logto
-p, --path    Путь к вашей инстанции Logto, должен быть несуществующим путем
--ss          Пропустить заполнение базы данных Logto
--oc          Добавить официальные коннекторы после установки
```

Выполните `logto init --help` для получения полной информации о помощи.

Вот пример выполнения тихой установки (например, в CI):

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

Это выполнит следующие действия по порядку:

1. Загрузит и извлечет Logto в `./logto`
2. Попробует инициализировать и заполнить базу данных с URL `postgresql://your-postgres-dsn:5432/logto`
3. Добавит официальные коннекторы Logto в `./logto/packages/core/connectors`
