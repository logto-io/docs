{props.connectorType} коннектор — это метод, используемый для отправки одноразовых паролей (OTP) для аутентификации. Он позволяет проверять {props.passwordlessSignUpIdentifier} для поддержки аутентификации без пароля, включая регистрацию на основе {props.connectorType}, вход, двухфакторную аутентификацию (2FA) и восстановление аккаунта. Вы можете легко подключить {props.connector} в качестве вашего провайдера {props.connectorType}. С коннектором Logto {props.connectorType} вы можете настроить это всего за несколько минут.

Чтобы добавить коннектор {props.connectorType}, просто выполните следующие шаги:

1. Перейдите в [Console > Connector > Email and SMS connectors](https://cloud.logto.io/to/connectors/passwordless).
2. Чтобы добавить новый коннектор {props.connectorType}, нажмите кнопку "Set up" и выберите "{props.connector}".
3. Ознакомьтесь с документацией README для выбранного вами провайдера.
4. Заполните поля конфигурации в разделе "Parameter Configuration".
5. Настройте шаблон {props.connectorType} с помощью JSON-редактора.
6. Проверьте вашу конфигурацию, отправив код подтверждения на ваш {props.passwordlessSignUpIdentifier}.
