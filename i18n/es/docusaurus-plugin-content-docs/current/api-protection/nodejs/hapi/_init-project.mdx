Para inicializar un nuevo proyecto Node.js con Hapi, puedes seguir estos pasos:

```bash
npm init -y
npm install @hapi/hapi
```

Si estás utilizando TypeScript, recuerda configurar tu entorno de TypeScript adecuadamente.

Luego, crea una configuración básica de servidor Hapi:

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Servidor ejecutándose en http://localhost:3000');
```

:::note
Consulta la documentación de Hapi para más detalles sobre cómo configurar rutas, plugins y otras características.
:::
