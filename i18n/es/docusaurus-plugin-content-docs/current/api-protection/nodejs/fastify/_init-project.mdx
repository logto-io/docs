Para inicializar un nuevo proyecto de Node.js con Fastify, puedes seguir estos pasos:

```bash
npm init -y
npm install fastify
```

Si estás utilizando TypeScript, recuerda configurar tu entorno de TypeScript adecuadamente.

Luego, crea una configuración básica de servidor Fastify:

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Servidor ejecutándose en http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
Consulta la documentación de Fastify para más detalles sobre cómo configurar rutas, plugins y otras características.
:::
