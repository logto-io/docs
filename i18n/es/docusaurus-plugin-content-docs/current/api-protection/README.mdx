# Protección de API

Aprende cómo implementar [control de acceso basado en roles (RBAC)](/authorization/role-based-access-control) y validación de [JSON Web Token (JWT)](https://auth.wiki/jwt) para proteger tus endpoints de API contra accesos no autorizados.

Nuestras guías cubren la configuración de middleware, modelos de permisos y ejemplos prácticos en varios lenguajes de programación y frameworks.

## Comienza \{#get-started}

### ¿Nuevo en la seguridad de API? \{#new-to-api-security}

Si eres nuevo en la autenticación y autorización de API, comienza con estos fundamentos:

- **[Autenticación vs. autorización](/concepts/authn-vs-authz)**: Comprende las diferencias clave
- **[Control de acceso basado en roles (RBAC)](/authorization/role-based-access-control)**: Aprende cómo funcionan los permisos y roles
- **[JSON Web Token (JWT)](https://auth.wiki/jwt)**: Descubre cómo los JSON Web Tokens aseguran tus APIs

### Elige tu lenguaje o framework \{#choose-your-language-or-framework}

¿Listo para implementar la protección de API? Selecciona tu stack tecnológico en la barra lateral para obtener guías de integración paso a paso con ejemplos de código y mejores prácticas.
