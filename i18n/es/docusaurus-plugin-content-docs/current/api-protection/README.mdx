# Protege tus API con Logto

Aprende cómo implementar [control de acceso basado en roles (RBAC)](/authorization/role-based-access-control) y validación de [JSON Web Token (JWT)](https://auth.wiki/jwt) para proteger los endpoints de tu API contra accesos no autorizados.

Nuestras guías cubren la configuración de middleware, modelos de permisos y ejemplos prácticos en múltiples lenguajes de programación y frameworks.

## Comienza \{#get-started}

### ¿Nuevo en la seguridad de API? \{#new-to-api-security}

Si eres nuevo en la autenticación y autorización de API, comienza con estos fundamentos:

- **[Autenticación vs. autorización](/concepts/authn-vs-authz)** - Comprende las diferencias clave
- **[Control de acceso basado en roles (RBAC)](/authorization/role-based-access-control)** - Aprende cómo funcionan los permisos y los roles
- **[JSON Web Token (JWT)](https://auth.wiki/jwt)** - Descubre cómo los JSON Web Tokens aseguran tus API

¿Listo para implementar la protección de API? Selecciona tu stack tecnológico en la barra lateral para obtener guías de integración paso a paso con ejemplos de código y mejores prácticas.
