Para inicializar un nuevo proyecto Flask, crea un directorio y configura la estructura básica:

```bash
mkdir your-api-name
cd your-api-name
```

Crea un archivo de requisitos:

```txt title="requirements.txt"
Flask
```

Instala las dependencias:

```bash
pip install -r requirements.txt
```

Crea una aplicación Flask básica:

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

Inicia el servidor de desarrollo:

```bash
python app.py
```

:::note
Consulta la documentación de Flask para más detalles sobre cómo configurar rutas, blueprints y otras características.
:::
