Para inicializar un nuevo proyecto de Sinatra, crea un directorio y configura la estructura básica:

```bash
mkdir your-api-name
cd your-api-name
```

Crea un Gemfile:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'sinatra'
```

Instala las dependencias:

```bash
bundle install
```

Crea una aplicación básica de Sinatra:

```ruby title="app.rb"
require 'sinatra'
require 'json'

get '/' do
  content_type :json
  { message: 'Hello from Sinatra API' }.to_json
end
```

Inicia el servidor de desarrollo:

```bash
ruby app.rb
```

:::note
Consulta la documentación de Sinatra para más detalles sobre cómo configurar rutas, middleware y otras características.
:::
