Para inicializar un nuevo proyecto de Grape API, crea un directorio y configura la estructura básica:

```bash
mkdir your-api-name
cd your-api-name
```

Crea un Gemfile:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

Instala las dependencias:

```bash
bundle install
```

Crea una Grape API básica:

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

Crea un archivo config.ru:

```ruby title="config.ru"
require_relative 'api'

run API
```

Inicia el servidor de desarrollo:

```bash
bundle exec rackup
```

:::note
Consulta la documentación de Grape para más detalles sobre cómo configurar recursos, validación de parámetros y otras funcionalidades.
:::
