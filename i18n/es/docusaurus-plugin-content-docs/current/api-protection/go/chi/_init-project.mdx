Para inicializar un nuevo proyecto Go con Chi, puedes seguir estos pasos:

```bash
go mod init your-api-name
go get github.com/go-chi/chi/v5
```

Luego, crea una configuración básica de servidor Chi:

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
Consulta la documentación de Chi para más detalles sobre cómo configurar rutas, middleware y otras funcionalidades.
:::
