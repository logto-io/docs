Para inicializar un nuevo proyecto Go con Gin, puedes seguir estos pasos:

```bash
go mod init your-api-name
go get github.com/gin-gonic/gin
```

Luego, crea una configuración básica de servidor Gin:

```go title="main.go"
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()

    r.Run(":3000") // escuchar y servir en 0.0.0.0:3000
}
```

:::note
Consulta la documentación de Gin para más detalles sobre cómo configurar rutas, middleware y otras funcionalidades.
:::
