# Estructura de datos de la organización

A nivel de organización, hay varios atributos cuyos valores pueden ser creados, actualizados, leídos y eliminados (CRUD).

## ID de la organización \{#organization-id}

El _ID de la organización_ es un identificador único para cada organización. Es útil para implementar experiencias de inicio de sesión a nivel de organización y recuperar tokens de organización.

## Nombre \{#name}

El _nombre_ admite el inicio de sesión a nivel de organización y puede integrarse en las interfaces de productos a nivel de organización según sea necesario.

## Descripción \{#description}

El campo _descripción_ te permite agregar texto para ayudar a identificar y etiquetar la organización.

## Logotipos de la organización \{#organization-logos}

Para mostrar dinámicamente el logotipo de la organización de tu cliente en la experiencia de inicio de sesión, puedes cargar los logotipos de la organización en la página de configuración de la organización.

Consulta [logotipos específicos de la organización](/customization/match-your-brand/#organization-specific-branding) para más detalles.

## Datos personalizados \{#custom-data}

Los _datos personalizados_ son un objeto JSON utilizado para almacenar información adicional sobre la organización. Ofrece a los desarrolladores flexibilidad para almacenar detalles adicionales de la organización con fines de desarrollo.

## ¿Es MFA requerido? \{#is-mfa-required}

_isMfaRequired_ indica si la Autenticación Multifactor (MFA) es obligatoria para la organización. Si se establece en `true`, todos los miembros deben pasar por MFA durante el inicio de sesión para acceder a la organización. Esta configuración de política de seguridad se configura a nivel de organización.

Consulta [Gestionar organización](/organizations/organization-management#require-mfa-for-organization-members) para más detalles.

## Creado en \{#created-at}

_createdAt_ es la marca de tiempo con la zona horaria cuando se creó la organización.

## ID del inquilino \{#tenant-id}

_tenantId_ identifica al inquilino al que pertenece la organización.
