# Gestión de usuarios

La gestión de usuarios es un área de enfoque importante para Logto. Puedes acceder a los datos de tus usuarios y gestionarlos utilizando tanto la Consola de Logto como la Management API, ambas opciones son efectivas. Con estas herramientas, puedes realizar tareas como

- Buscar usuarios por nombre de usuario, ID, nombre completo, número de teléfono o correo electrónico.
- Realizar varias acciones sobre el usuario, como desvincular cuentas sociales, actualizar información del perfil, eliminar usuarios o restablecer contraseñas.
- Consultar los registros de usuario.
- Asignar roles para el control de acceso.
- Crear y gestionar tokens de acceso personal.
- Gestionar sesiones de usuario.
