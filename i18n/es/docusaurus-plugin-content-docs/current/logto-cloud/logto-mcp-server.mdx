# Logto MCP Server

Logto MCP Server es un servidor remoto de [MCP (Model Context Protocol)](https://modelcontextprotocol.io/). Añade la URL a tu aplicación de IA, inicia sesión con tu cuenta de Logto Cloud y comienza a interactuar con Logto.

:::tip
Logto MCP Server está construido con Logto y [MCP Auth](https://mcp-auth.dev). Si quieres construir tu propio servidor remoto MCP, consulta [Construye tu propio servidor remoto MCP](#build-your-own-remote-mcp-server).
:::

## Qué puedes hacer \{#what-you-can-do}

- **Añade autenticación a tu proyecto**: La IA detecta tu framework, crea una aplicación Logto y genera el código de integración funcional. Soporta más de 38 frameworks incluyendo React, Next.js, Vue, Nuxt, SvelteKit, Express, Go, Python, iOS, Android, Flutter y más.
- **Haz preguntas sobre Logto**: La IA busca en la documentación de Logto y te da respuestas con fuentes.

Próximamente habrá más capacidades, incluyendo configuración de la experiencia de inicio de sesión, gestión de RBAC y permisos con guía de mejores prácticas, configuración de multi-tenancy para construir aplicaciones B2B con organizaciones, y más.

## Configuración en tu aplicación de IA \{#set-up-in-your-ai-application}

### Claude Code \{#claude-code}

<Tabs>
  <TabItem value="claude-mcp-add" label="claude mcp add">

    ```bash
    claude mcp add --transport http --client-id mcp-client --callback-port 8080 logto https://mcp.logto.io
    ```

  </TabItem>
  <TabItem value="claude-mcp-add-json" label="claude mcp add-json">

    ```bash
    claude mcp add-json logto '{"type":"http","url":"https://mcp.logto.io","oauth":{"clientId":"mcp-client","callbackPort":8080}}'
    ```

  </TabItem>
</Tabs>

Puedes cambiar `8080` por cualquier puerto disponible en tu máquina.

Luego, ejecuta `/mcp` en Claude Code y sigue el flujo de inicio de sesión en el navegador para autenticarte con tu cuenta de Logto Cloud.

### Claude Desktop \{#claude-desktop}

1. Abre **Settings** desde el menú principal, navega a **Connectors** y haz clic en **Add custom connector**.
2. Ingresa el nombre `Logto MCP Server` y la URL del servidor remoto MCP: `https://mcp.logto.io`.
3. Expande **Advanced settings** e ingresa el OAuth Client ID: `mcp-client`. Haz clic en **Add**.
4. En la lista **Connectors**, busca **Logto MCP Server** y haz clic en **Connect**.
5. Primero serás redirigido a la página web de Claude. Si no tienes una sesión activa, inicia sesión en tu cuenta de Claude primero.
6. Después de iniciar sesión en Claude, serás redirigido a la página de inicio de sesión de Logto Cloud. Inicia sesión con tu cuenta de Logto Cloud y autoriza la conexión.

### Cursor \{#cursor}

1. Abre la paleta de comandos (`Command + Shift + P`) y selecciona **View: Open MCP Settings**.
2. En la sección **Installed MCP Servers**, haz clic en el botón **Add Custom MCP**.
3. En el archivo `mcp.json` que se abre, añade la siguiente configuración:

   ```json
   {
     "mcpServers": {
       "Logto": {
         "command": "npx",
         "args": [
           "mcp-remote",
           "https://mcp.logto.io",
           "--allow-http",
           "--static-oauth-client-info",
           "{ \"client_id\": \"mcp-client\", \"scope\": \"mcp:all\" }",
           "--resource",
           "https://mcp.logto.io/"
         ]
       }
     }
   }
   ```

Aparecerá automáticamente una página de inicio de sesión de Logto Cloud. Inicia sesión con tu cuenta de Logto Cloud para conectar.

### OpenCode \{#opencode}

1. Añade la siguiente configuración a tu archivo `opencode.json`:

   ```json
   {
     "$schema": "https://opencode.ai/config.json",
     "mcp": {
       "Logto": {
         "enabled": true,
         "type": "remote",
         "url": "https://mcp.logto.io",
         "oauth": {
           "clientId": "mcp-client"
         }
       }
     }
   }
   ```

2. Ejecuta el siguiente comando para autenticarte:
   ```bash
   opencode mcp auth Logto
   ```
   Ten en cuenta que `Logto` corresponde al nombre del servidor MCP en tu archivo de configuración.

Se abrirá una ventana del navegador. Inicia sesión con tu cuenta de Logto Cloud para autorizar y conectar.

### VS Code \{#vs-code}

1. Abre la paleta de comandos (`Command + Shift + P`) y selecciona **MCP: Add Server...**.
2. Elige **HTTP** como tipo de servidor.
3. Ingresa la URL: `https://mcp.logto.io`. Puedes elegir instalarlo en tu espacio de trabajo para mejor aislamiento y gestión.
4. Cuando comience el flujo OAuth, ingresa el App ID: `mcp-client`. Deja el App Secret vacío y presiona Enter para omitir.
5. Serás redirigido a la página de inicio de sesión de Logto Cloud. Inicia sesión con tu cuenta de Logto Cloud para conectar.

### Windsurf \{#windsurf}

1. Abre la paleta de comandos (`Command + Shift + P`) y selecciona **Windsurf: MCP Marketplace**.
2. En la sección **Installed MCPs**, haz clic en el icono de configuración a la derecha para crear y abrir el archivo `~/.codeium/windsurf/mcp_config.json`.
3. Añade la siguiente [configuración](https://docs.windsurf.com/windsurf/cascade/mcp#mcp_config-json):

   ```json
   {
     "mcpServers": {
       "Logto": {
         "command": "npx",
         "args": [
           "mcp-remote",
           "https://mcp.logto.io",
           "--allow-http",
           "--static-oauth-client-info",
           "{ \"client_id\": \"mcp-client\", \"scope\": \"mcp:all\" }",
           "--resource",
           "https://mcp.logto.io/"
         ]
       }
     }
   }
   ```

Aparecerá automáticamente una página de inicio de sesión de Logto Cloud. Inicia sesión con tu cuenta de Logto Cloud para conectar.

### Otros clientes MCP \{#other-mcp-clients}

Logto MCP Server funciona con cualquier cliente MCP que soporte autorización OAuth. Utiliza la siguiente configuración:

- **URL del servidor MCP:** `https://mcp.logto.io`
- **OAuth client ID:** `mcp-client`
- **Resource:** `https://mcp.logto.io/` (si tu cliente lo requiere)
- **Scope:** `mcp:all` (si tu cliente lo requiere)

Si eres desarrollador de aplicaciones de IA y quieres soportar la conexión a Logto MCP Server, pero Logto MCP Server aún no soporta el redirect URI de tu aplicación, por favor [contáctanos](https://logto.io/contact) con los detalles de tu aplicación.
