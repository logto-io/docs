# Servicio en la nube de Logto

[Logto Cloud](https://cloud.logto.io) ofrece una gama de servicios de gestión y operación para ayudarte a gestionar identidades y recursos de manera fluida y sencilla. Alojado por Logto, incluye características como [soporte multirregional](/logto-cloud/tenant-settings#tenant-region), gestión de inquilinos, [facturación y precios](/logto-cloud/billing-and-pricing), [gestión de miembros](/logto-cloud/tenant-member-management) y control de acceso basado en roles en la consola.

Si tienes alguna pregunta sobre los servicios en la nube y necesitas soporte adicional, por favor contáctanos.
