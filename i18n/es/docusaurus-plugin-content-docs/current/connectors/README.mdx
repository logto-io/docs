# Conectores

Los conectores son el puente entre Logto y los servicios externos. Permiten métodos de [verificación sin contraseña](https://auth.wiki/passwordless) como [verificación por correo electrónico](/end-user-flows/sign-up-and-sign-in/sign-in), [verificación por SMS](/end-user-flows/sign-up-and-sign-in/sign-in), [inicio de sesión social](/end-user-flows/sign-up-and-sign-in/social-sign-in) y [SSO empresarial](/end-user-flows/enterprise-sso).

El registro tradicional de cuentas con nombres de usuario y contraseñas puede ser un proceso tedioso y frustrante. Los usuarios a menudo tienen dificultades para recordar contraseñas complejas y pueden ser reacios a crear nuevas cuentas debido a la inconveniencia. Si los usuarios pueden proporcionar su correo electrónico o número de teléfono y verificarlo a través de códigos de verificación; o iniciar sesión usando cuentas como Google/Facebook, o cuentas corporativas como Microsoft, esto puede simplificar enormemente el proceso de inicio de sesión y reducir la pérdida de usuarios. El conector fue creado para resolver este problema.
