Logto proporciona cuatro plantillas diferentes para personalizar el contenido de los SMS, que se categorizan según su tipo de uso: Registro, Inicio de sesión, Olvidó contraseña y Genérico. Se recomienda encarecidamente que utilices diferentes plantillas para varios casos de uso, o podrías alcanzar el límite de tasa, lo que llevaría a una interrupción temporal de tu servicio.

## Tipos de plantillas de SMS y ejemplos

Aquí hay algunos ejemplos solo para referencia:

| Uso               | Escenario                                                                                                                                                    | Ejemplo de plantilla                                                                                                         |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| Registro          | Los usuarios crean una cuenta usando su correo electrónico y lo verifican ingresando un código de verificación enviado por Logto a su correo.                | Tu código de verificación de registro de Logto es `{{code}}`. El código permanecerá activo durante 10 minutos.               |
| Inicio de sesión  | Los usuarios inician sesión usando su correo electrónico y verifican ingresando un código de verificación en lugar de una contraseña.                        | Tu código de verificación de inicio de sesión de Logto es `{{code}}`. El código permanecerá activo durante 10 minutos.       |
| Olvidó contraseña | Si los usuarios olvidan su contraseña durante el inicio de sesión, pueden optar por verificar su identidad usando el correo que ya han verificado con Logto. | Tu código de verificación para cambio de contraseña de Logto es `{{code}}`. El código permanecerá activo durante 10 minutos. |
| Genérico          | Esta plantilla se puede usar como una opción de respaldo general para varios escenarios, incluyendo la prueba de configuraciones de conectores, etc.         | Tu código de verificación de Logto es `{{code}}`. El código permanecerá activo durante 10 minutos.                           |

Es importante entender estos parámetros:

- El código de verificación es válido por 10 minutos. Actualmente no admitimos la personalización del tiempo de expiración.
- Logto reemplazará el marcador de posición `{{code}}` en la plantilla de SMS con un código de verificación. Por lo tanto, asegúrate de que la plantilla tenga un marcador de posición reservado.

:::note
Algunos países y regiones pueden no permitir el envío de contenido no aprobado a través de SMS debido a requisitos de cumplimiento. Las plantillas de SMS deben ser registradas y aprobadas por el proveedor de SMS antes de que puedan ser utilizadas. En tales casos, el contenido podría ser indexado por ID de plantilla a la plantilla correspondiente.
:::
