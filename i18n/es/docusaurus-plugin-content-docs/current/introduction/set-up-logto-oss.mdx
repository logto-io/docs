# Configurar Logto OSS

Esta sección cubre los pasos básicos de configuración y acciones clave para entregar efectivamente tu producto y gestionar tu flujo de trabajo de desarrollo para el servicio de código abierto de Logto (OSS).

## Comenzar con Logto OSS

Sigue la [guía de inicio](/logto-oss/get-started-with-oss/) para lanzar Logto hoy, y más tarde puedes consultar la [guía de despliegue](/logto-oss/deployment-and-configuration) completa para uso en producción.

## Funcionalidades soportadas por Logto OSS

Logto OSS soporta la mayoría de las capacidades centrales del servicio Logto y se actualiza y mantiene regularmente.

Algunas características avanzadas son actualmente exclusivas de la versión Logto Cloud, incluyendo Logto Protected App, servicio de correo electrónico gratuito, SSO iniciado por IdP y SAML IdP.

## Mantente actualizado con las versiones de Logto

Para mantener tu instancia de Logto actualizada con las últimas características, asegúrate de seguir la página de [Logto GitHub Releases](https://github.com/logto-io/logto/releases), donde puedes encontrar todos los registros de versiones.

Lee la guía sobre actualizaciones para aprender cómo actualizar Logto sin cambiar tu código o esquema de base de datos.
