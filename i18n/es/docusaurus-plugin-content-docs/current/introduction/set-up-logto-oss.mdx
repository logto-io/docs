# Configurar Logto OSS

Esta sección cubre los pasos básicos de configuración y acciones clave para entregar eficazmente tu producto y gestionar tu flujo de trabajo de desarrollo para [Logto servicio de código abierto (OSS)](https://github.com/logto-io/logto).

## Comenzar con Logto OSS \{#get-started-with-logto-oss}

Sigue la [guía de inicio](/logto-oss/get-started-with-oss/) para lanzar Logto hoy, y más adelante puedes consultar la [guía de implementación](/logto-oss/deployment-and-configuration) completa para uso en producción.

## Funcionalidades soportadas por Logto OSS \{#feature-supported-by-logto-oss}

Logto OSS admite la mayoría de las capacidades principales del servicio Logto y se actualiza y mantiene regularmente.

Algunas características avanzadas son actualmente exclusivas de la versión Logto Cloud, incluyendo:

| Limitaciones de características en OSS                                              | Descripción                                                                                                                                        |
| ----------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Consola Logto: múltiples inquilinos](/logto-cloud/tenant-settings)                 | Crear y gestionar múltiples inquilinos de Logto en una consola.                                                                                    |
| [Consola Logto: invitación de colaboradores](/logto-cloud/tenant-member-management) | Invitar y gestionar múltiples miembros de un inquilino de Logto.                                                                                   |
| Consola Logto: Habilitar MFA                                                        | Mejorar la seguridad al requerir autenticación multifactor al iniciar sesión en la Consola Logto.                                                  |
| [Aplicación protegida de Logto](/integrate-logto/protected-app)                     | Integración rápida de autenticación sin SDK impulsada por Cloudflare.                                                                              |
| [Servicio de correo electrónico integrado de Logto](/connectors/email-connectors/built-in-email-service) | Utilizar el servicio de correo electrónico integrado y gratuito para la entrega de correos electrónicos.                                           |
| [Trae tu UI](/customization/bring-your-ui)                                          | Personalizar la interfaz de usuario o los flujos de la experiencia de inicio de sesión usando esta característica (OSS puede bifurcar el código en GitHub para personalizar la experiencia de inicio de sesión directamente). |
| [SSO iniciado por IdP](/end-user-flows/enterprise-sso/idp-initiated-sso)            | Habilitar el inicio de sesión único empresarial iniciado por el proveedor de identidad.                                                            |
| [Aplicaciones SAML](/integrate-logto/saml-app)                                      | Integrar aplicaciones a través del protocolo SAML. La versión OSS está limitada a 3 aplicaciones SAML.                                             |

Consejos: La multi-tenencia, las invitaciones de miembros y MFA no están disponibles para que tu equipo inicie sesión en una consola Logto de código abierto. Sin embargo, puedes implementar estas características en tu propio producto usando Logto OSS, haciéndolas disponibles para tus usuarios finales.

## Mantente actualizado con los lanzamientos de Logto \{#stay-updated-with-logto-releases}

Para mantener tu instancia de Logto actualizada con las últimas características, asegúrate de seguir la página de [Lanzamientos de Logto en GitHub](https://github.com/logto-io/logto/releases), donde puedes encontrar todos los registros de lanzamientos.

Lee la [guía sobre cómo actualizar](/logto-oss/upgrading-oss-version) para aprender cómo actualizar Logto sin cambiar tu código o esquema de base de datos.
