# Introducción

¡Bienvenido a la documentación de Logto! Logto es una solución de gestión de identidad y acceso (IAM) diseñada para aplicaciones modernas y productos SaaS. Proporciona un sistema seguro, escalable y personalizable de autenticación (Authentication) y autorización (Authorization) para tus aplicaciones.
