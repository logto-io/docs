# Configura Logto Cloud

Esta sección cubre los pasos básicos de configuración y acciones clave para entregar efectivamente tu producto y gestionar tu flujo de trabajo de desarrollo.

## Crear un inquilino de Logto

Primero, regístrate para un negocio y crea un inquilino de Logto. Un inquilino de Logto Cloud es un entorno aislado donde puedes gestionar identidades de usuarios, aplicaciones y todos los demás recursos de Logto.

Tu primer inquilino se configurará automáticamente como un entorno de **Desarrollo**. Esto es ideal para propósitos de prueba y desarrollo y es gratuito.

Consulta la configuración del inquilino y [aprende más](/logto-cloud) sobre el servicio en la nube de Logto.

## Invitar colaboradores

Durante la incorporación, tienes la oportunidad de invitar a colaboradores para realizar el trabajo de desarrollo juntos. Si deseas hacer más o más tarde, ve a <CloudLink to="/tenant-settings/members">Configuración del inquilino > Miembros</CloudLink>. Verás a tus miembros actuales y podrás invitar a más. Ingresa direcciones de correo electrónico y asigna roles (puedes hacerlo en bloque) y luego envía.

## Moverse al inquilino de producción

Puedes pasar algún tiempo desarrollando y probando las capacidades y características de Logto para armar una prueba de concepto.

Después de que tu proyecto complete las pruebas y esté listo para la siguiente fase, recuerda crear un nuevo inquilino de tipo producción (requiriendo un inquilino Free o Pro). Porque el inquilino de desarrollo no está destinado para uso en producción debido a sus limitaciones y restricciones.

:::note

Asegúrate de ingresar el dominio personalizado tan pronto como crees un inquilino de producción, ya que impactará en la configuración de tu siguiente paso.

:::

Esta configuración permite que tu proyecto tenga tanto entornos de prueba de desarrollo como de producción, ayudándote a gestionar tu flujo de trabajo de desarrollo sin problemas.

## Migrar desde el sistema existente

Si estás trabajando en un proyecto existente y necesitas cambiar de proveedor de autenticación, Logto admite la migración de usuarios desde otras plataformas. Puedes migrar datos básicos, datos personalizados, identidades sociales y hashes de contraseñas. Para más detalles, consulta la guía [Migrar a Logto](../user-management/user-migration.mdx).
