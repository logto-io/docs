# Flujos de usuario final

Los flujos de usuario final cubren todos los procesos de verificación para las interacciones de los usuarios, categorizados de la siguiente manera:

- **Experiencia de inicio de sesión**: Involucra el inicio de sesión de usuario, registro y restablecimiento de contraseña. Logto ofrece experiencias listas para usar configuradas en Console, y puedes traer tu propia interfaz de usuario a través de las Experience APIs.
- **Experiencia de cuenta**: Involucra configuraciones de cuenta, perfil de usuario y verificación de seguridad. Usa Account APIs o Management APIs para implementar.
- **Experiencia de organización**: Para productos de multi-tenencia, como servicios B2B o SaaS. Involucra la creación de organizaciones, invitación de miembros y gestión. Usa Management APIs para integrar esto en tus aplicaciones.
