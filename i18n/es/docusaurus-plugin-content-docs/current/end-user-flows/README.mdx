# Flujos de usuario final

Los flujos de usuario final cubren todos los procesos de verificación para interacciones de usuario, categorizados de la siguiente manera:

- **Flujos de autenticación**: Involucra el inicio de sesión de usuario, registro y restablecimiento de contraseña. Logto ofrece experiencias listas para usar configuradas en <CloudLink to="/sign-in-experience">Consola > Experiencia de inicio de sesión</CloudLink>, y puedes [traer tu UI](/customization/bring-your-ui) a través de [Experience APIs](https://openapi.logto.io/group/endpoint-experience).
- **Flujos de cuenta**: Involucra configuraciones de cuenta, perfil de usuario y verificación de seguridad. Usa [Account APIs](https://openapi.logto.io/group/endpoint-account-center) o [Management APIs](https://openapi.logto.io/operation/operation-getuser) para implementar.
- **Flujos de organización**: Para productos de multi-tenencia, como servicios B2B o SaaS. Involucra la creación de organizaciones, invitación de miembros y gestión. Usa [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) para integrar esto en tus aplicaciones.
