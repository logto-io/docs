# Flujos de usuario final

Los flujos de usuario final cubren todos los procesos de verificación para las interacciones de los usuarios, categorizados de la siguiente manera:

- **Flujos de autenticación (Authentication flows)**: Incluye inicio de sesión, registro y restablecimiento de contraseña del usuario. Logto ofrece experiencias listas para usar configuradas en <CloudLink to="/sign-in-experience">Consola > Experiencia de inicio de sesión</CloudLink>, y puedes [traer tu propia interfaz](/customization/bring-your-ui) mediante las [Experience APIs](https://openapi.logto.io/group/endpoint-experience).
- **Flujos de cuenta (Account flows)**: Incluye configuración de cuenta, perfil de usuario y verificación de seguridad. Utiliza las [Account APIs](https://openapi.logto.io/group/endpoint-account-center) o las [Management APIs](https://openapi.logto.io/operation/operation-getuser) para implementarlo.
- **Flujos de organización (Organization flows)**: Para productos multi-inquilino, como servicios B2B o SaaS. Incluye creación de organización, invitación de miembros y gestión. Utiliza las [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) para integrarlo en tus aplicaciones.
