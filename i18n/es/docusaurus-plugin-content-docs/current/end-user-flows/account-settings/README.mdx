# Configuración de la cuenta

Logto proporciona dos colecciones de APIs de configuración de cuenta para permitir a los usuarios gestionar su cuenta y perfiles almacenados en Logto.

## APIs de gestión de usuarios \{#user-management-apis}

Las Management APIs de Logto proporcionan la funcionalidad más completa para gestionar usuarios y sus datos de perfil. Puedes implementar un sistema de gestión de usuarios autoalojado o una página de configuración de cuenta para el usuario final utilizando estas APIs.

## APIs de cuenta \{#account-apis}

Además de las Management APIs, Logto también proporciona un conjunto de APIs de cuenta para el usuario final que permiten a los usuarios finales gestionar directamente la configuración de su cuenta y los datos de su perfil. A diferencia de las Management APIs, las APIs de cuenta están diseñadas para ser utilizadas en el contexto del usuario final con un control detallado de permisos de usuario. Puedes usar las APIs de cuenta directamente en tu aplicación del lado del cliente para construir una página de configuración de cuenta amigable para el usuario.
