# Configuración de cuenta

Logto proporciona dos colecciones de APIs de configuración de cuenta para permitir a los usuarios gestionar su cuenta y perfiles almacenados en Logto.

## APIs de gestión de usuarios

Las Management APIs de Logto proporcionan la funcionalidad más completa para gestionar usuarios y sus datos de perfil. Puedes implementar un sistema de gestión de usuarios autoalojado o una página de configuración de cuenta para el usuario final utilizando estas APIs.

Aprende más sobre cómo usar las [Management APIs de usuario](/end-user-flows/account-settings/by-management-api) para construir tu propia página de configuración de cuenta.

## APIs de cuenta

Además de las Management APIs, Logto también proporciona un conjunto de APIs de cuenta para usuarios finales que permiten a los usuarios finales gestionar directamente la configuración de su cuenta y los datos de perfil. A diferencia de las Management APIs, las APIs de cuenta están diseñadas para ser utilizadas en el contexto del usuario final con un control detallado de permisos de usuario. Puedes usar las APIs de cuenta directamente en tu aplicación del lado del cliente para construir una página de configuración de cuenta amigable para el usuario.

Aprende más sobre cómo usar las [APIs de cuenta](/end-user-flows/account-settings/by-account-api) para construir tu propia página de configuración de cuenta.
