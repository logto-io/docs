# Experiencia de organización

La experiencia de [Organización (Organization)](/organizations) se refiere a las interfaces de usuario y flujos utilizados por tus clientes empresariales y sus empleados, particularmente en [aplicaciones multi-tenant](https://auth.wiki/multi-tenancy). Logto proporciona varias guías para ayudarte a integrar rápidamente esta experiencia en tu aplicación utilizando el Logto Management API.
