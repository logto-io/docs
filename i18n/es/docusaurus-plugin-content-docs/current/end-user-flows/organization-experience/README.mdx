# Experiencia de organización

La experiencia de [organización](/organizations) se refiere a las interfaces de usuario y flujos utilizados por tus clientes empresariales y sus empleados, especialmente en [aplicaciones multi-inquilino](https://auth.wiki/multi-tenancy). Logto proporciona varias guías para ayudarte a integrar rápidamente esta experiencia en tu aplicación utilizando la Management API de Logto.
