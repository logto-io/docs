# Selector de organización

En la experiencia de organización, los usuarios pueden cambiar entre organizaciones. Cada vez que cambian, la aplicación obtiene un nuevo token de organización (Organization token) para cargar la información relevante.

## Cómo implementarlo \{#how-to-implement-it}

1. Crea un menú desplegable u otro componente de interfaz similar para listar todas las organizaciones (Organizations) disponibles para el usuario.
2. Cuando el usuario seleccione una organización diferente, obtén su ID y llama al método del SDK `getOrganizationToken(organizationId)` para obtener el token de organización (Organization token) correspondiente. (El SDK gestiona el almacenamiento en caché de tokens por ti).
3. Almacena el ID de la organización seleccionada (en la URL o en el almacenamiento del cliente), y utilízalo siempre para obtener el token de organización (Organization token) emparejado para los recursos específicos de la organización.
