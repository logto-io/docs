# Configurar permisos

## Configurar permisos de API a través de Logto Console \{#configure-api-permissions-via-logto-console}

Los permisos se establecen a nivel de **recurso de API (API resource)**. Navega a <CloudLink to="/api-resources">Console > API resources</CloudLink>, haz clic en el recurso de API que deseas gestionar y luego ve a la pestaña "Permissions".

Es importante proporcionar un nombre y detalles para el permiso, ya que facilitará la identificación y gestión de los permisos.

## Configurar permisos de API a través de Logto Management API \{#configure-api-permissions-via-logto-management-api}

Gestiona usando Logto Management API. Realiza una llamada al punto final relativo. Consulta esta [referencia](https://openapi.logto.io/operation/operation-listresourcescopes).

| método | ruta                                                                                                                 | descripción                                                                                                  |
| ------ | -------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| GET    | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-listresourcescopes)              | Obtener los alcances definidos para un recurso de API.                                                       |
| POST   | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-createresourcescope)             | Crear un nuevo alcance para un recurso de API.                                                               |
| DELETE | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-deleteresourcescope) | Eliminar un alcance de recurso de API del recurso dado.                                                      |
| PATCH  | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-updateresourcescope) | Actualizar un alcance de recurso de API para el recurso dado. Este método realiza una actualización parcial. |
