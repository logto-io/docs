## Configura un proyecto en la Consola de Desarrolladores de Kakao \{#set-up-a-project-in-the-kakao-devlopers-console}

- Visita la [Consola de Desarrolladores de Kakao](https://developers.kakao.com/console/app) e inicia sesión con tu cuenta de Kakao.
- Haz clic en **Agregar una aplicación** para crear un nuevo proyecto o elige un proyecto existente.

## Configura el inicio de sesión de Kakao \{#configure-kakao-login}

### Activa el inicio de sesión de Kakao \{#activate-kakao-login}

- Haz clic en **Configuración del producto -> Inicio de sesión de Kakao** en el menú.
- Activa `Activación de inicio de sesión de Kakao`
- Añade la siguiente URL en `Redirect URI`
  - `http(s)://YOUR_URL/callback/${connector_id}` (El `connector_id` se puede encontrar en la barra superior de la página de detalles del conector en la Consola de Administración de Logto.)
  - (Por favor, reemplaza `YOUR_URL` con tu URL de `Logto`, y elige `http` o `https` según tu situación.)

### Configuración de privacidad \{#privacy-setting}

- Haz clic en **Configuración del producto -> Inicio de sesión de Kakao -> Elemento de consentimiento** en el menú.
- Cambia el estado de `Apodo`, `Imagen de perfil` y `Correo electrónico` a **Consentimiento requerido** (Es posible que no puedas cambiar `Correo electrónico` a **Consentimiento requerido** debido a la configuración de tu proyecto.)

### Configuración de seguridad (Opcional) \{#security-setting-optional}

- Haz clic en **Configuración del producto -> Inicio de sesión de Kakao -> Seguridad** en el menú.
- Haz clic en `Código secreto del cliente` para generar el código secreto.
- Cambia el `Estado de activación` a Habilitar. (Si lo habilitas, el `código secreto` es necesario.)

## Configura Logto \{#configure-logto}

### Tipos de configuración \{#config-types}

| Nombre       | Tipo    |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId \{#clientid}

`clientId` es la `clave de API REST` de tu proyecto.
(Puedes encontrarla en el `resumen` de tu proyecto en la consola de desarrolladores de Kakao.)

#### clientSecret \{#clientseceret}

`clientSecret` es el `Código Secreto` de tu proyecto.
(Por favor, consulta [Configuración de seguridad (Opcional)](#security-setting-optional))
