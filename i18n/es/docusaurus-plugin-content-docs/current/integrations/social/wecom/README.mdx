# Configura el inicio de sesión social con WeCom (OAuth 2.0)

El conector personalizado para el inicio de sesión social de WeCom (quizás llamado WXwork).

<GuideTip />

## Comenzar \{#get-started}

Inicia sesión en [WeCom WebUI](https://work.weixin.qq.com/) con una cuenta de administrador o haz clic en **Gestionar la empresa(管理企业)** desde la aplicación WeCom.

En la pestaña "Gestión de aplicaciones", desplaza la página hacia abajo y haz clic en "Crear una aplicación".

Rellena la información adecuada según tu aplicación. Luego crea.

Ahora tenemos el Agent ID (NO APPID) y el Secret.

### Información del sitio web \{#website-info}

Configura lo que necesitas en esta página. Sería algo como:

- Usuarios permitidos: _quién puede ver esta aplicación_
- Página de inicio de la aplicación: _La página de inicio de tu aplicación. Por ejemplo, `logto.io/demo-app`_

**Importante**
Hay tres elementos en esta página relacionados con el "Developer API(开发者接口)".

1. Autorización web y JS-SDK;
2. Iniciar sesión a través de la autorización por WeCom;
3. IP de confianza de la empresa;

Rellénalos según la guía de WeCom.

### Corp ID \{#corp-id}

Si estás familiarizado con el desarrollo de WeChat, puedes notar que el uso del Corp ID es el mismo que el APP ID.

Puedes encontrar el Corp ID en la parte inferior de la página de la pestaña "Mi Empresa(我的企业)". Parece algo como **ww\*\*\*\*** .

## Configurar el conector \{#configure-the-connector}

Así que tenemos el Agent ID, el Secret y el Corp ID.

Completemos el formulario para el conector.

Puedes dejar el campo `Scope` en blanco ya que es opcional. Alternativamente, puedes rellenarlo con `snsapi_base` o `snsapi_privateinfo`.

**Guardar y listo**
