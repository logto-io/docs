## Crea una aplicación en el Centro de Desarrolladores de KOOK \{#create-an-application-in-the-kook-developer-center}

Consejo: Puedes omitir algunas secciones si ya las has completado.

:::note
Debes solicitar el permiso OAuth2 de KOOK antes de crear una Aplicación OAuth2. Por favor, consulta la información en la página de OAuth2 o pregunta al personal de KOOK en el servidor oficial de desarrolladores de KOOK para obtener más información.
:::

1. Ve al [Centro de Desarrolladores de KOOK](https://developer.kookapp.cn/app/index) e inicia sesión con tu cuenta de KOOK. Puedes registrar una nueva cuenta si no tienes una.
2. Haz clic en el botón **Crear aplicación** para crear una aplicación, elige un nombre para ella (por ejemplo, LogtoAuth) y haz clic en **Confirmar**.
3. Haz clic en el icono de la aplicación que creaste y ve a la página **OAuth2**.
4. Añade las redirecciones válidas (por ejemplo, `${your_logto_origin}/callback/${connector_id}`) en los campos **Callback URIs**. Puedes encontrar el URI de redirección en la página de detalles del conector en la consola de administración de Logto.
5. No olvides marcar el alcance **get_user_info** en el **constructor de enlaces OAuth2**.

## Configura tu conector KOOK \{#configure-your-kook-connector}

Rellena los campos clientId y clientSecret con el Client ID y Client Secret que obtuviste en la página de OAuth2 mencionada en la sección anterior.

### Tipos de configuración \{#config-types}

| Nombre       | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
