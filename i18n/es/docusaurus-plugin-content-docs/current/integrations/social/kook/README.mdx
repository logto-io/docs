# Configura el inicio de sesión social con KOOK (Set up social login with KOOK)

El conector oficial de Logto para aplicaciones web con inicio de sesión social de KOOK.

<GuideTip />

## Comenzar \{#get-started}

El conector de KOOK permite a los usuarios finales iniciar sesión en tu aplicación usando su propia cuenta de KOOK a través del protocolo de autenticación OAuth 2.0 de KOOK.
