# Configura el inicio de sesión social con Alipay (Native) (Set up social login with Alipay (Native))

El conector oficial de Logto para el inicio de sesión social con Alipay en aplicaciones nativas de dispositivos móviles.

<GuideTip />

## Comenzar \{#get-started}

El conector Alipay Native funciona en estrecha colaboración con el SDK de Logto en plataformas móviles. Aprovecha el flujo de autenticación OAuth 2.0 de Alipay y permite que los usuarios de Alipay inicien sesión en otras aplicaciones utilizando perfiles públicos de usuario de Alipay sin pasar por un proceso de registro complicado.

<Integration />

### Probar el conector nativo de Alipay \{#test-alipay-native-connector}

Eso es todo. No olvides [habilitar el conector social en la experiencia de inicio de sesión](/connectors/social-connectors/#enable-social-sign-in).

Una vez que el conector nativo de Alipay esté habilitado, puedes compilar y ejecutar tu aplicación para ver si funciona.
