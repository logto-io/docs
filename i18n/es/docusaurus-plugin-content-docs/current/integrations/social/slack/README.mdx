# Configura el inicio de sesión social con Slack

El conector oficial de Logto para el inicio de sesión social con Slack.

<GuideTip />

## Comenzar \{#get-started}

El conector de Slack permite a los usuarios finales iniciar sesión en tu aplicación usando sus propias cuentas de Slack a través del protocolo de autenticación Slack OAuth 2.0.

<Integration />

## Probar el conector de Slack \{#test-slack-connector}

Eso es todo. El conector de Slack debería estar disponible ahora. No olvides [habilitar el conector social en el registro e inicio de sesión](/connectors/social-connectors/#enable-social-sign-in).
