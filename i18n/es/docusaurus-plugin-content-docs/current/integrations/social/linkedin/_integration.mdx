### Configura una aplicación de LinkedIn \{#setup-a-linkedin-app}

Ve a [LinkedIn Developers](https://www.linkedin.com/developers) e inicia sesión con tu cuenta de LinkedIn. Si no tienes una cuenta, puedes registrarte para obtener una.

Luego, crea una aplicación.

**Paso 1:** Completa los Detalles de la Aplicación.

Completa el formulario y crea la aplicación.

**Paso 2:** Configura las URLs de callback.

Ve a la página de detalles de la aplicación y encuentra la pestaña "Auth", sección "OAuth 2.0" y busca el campo "Authorized redirect URLs for your app".

En nuestro caso, será `${your_logto_endpoint}/callback/${connector_id}`. por ejemplo, `https://foo.logto.app/callback/${connector_id}`. El `connector_id` se puede encontrar en la barra superior de la página de detalles del conector en la Consola de Administración de Logto.

**Paso 3:** Añade el producto.

Ve a la pestaña "Products" y añade el producto "Sign In with LinkedIn using OpenID Connect".

### Configura tu conector \{#configure-your-connector}

En la configuración de tu conector de Logto, completa los siguientes campos con los valores obtenidos de la pestaña "Auth" de tu aplicación, sección "Application credentials":

- **clientId:** El Client ID de tu aplicación.
- **clientSecret:** El Primary Client Secret de tu aplicación.

`scope` es una lista de alcances OIDC delimitada por espacios. Si no se proporciona, el alcance predeterminado es `openid profile email`.

### Tipos de configuración \{#config-types}

| Nombre       | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
