### Configura una aplicación de Amazon \{#setup-a-amazon-app}

Ve al [Portal de Desarrolladores de Amazon](https://developer.amazon.com) e inicia sesión con tu cuenta. Si no tienes una cuenta, puedes registrarte para obtener una.

Luego, crea una aplicación.

**Paso 1:** Ve al bloque "Login with Amazon" en la página de inicio.

O puedes visitar [esta página](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html).

Sigue la guía para crear un nuevo perfil de seguridad.

Después de crear el perfil de seguridad, obtendrás un Client ID y un Client Secret de la tabla.

**Paso 2:** Configura las URLs de callback.

Haz clic en el botón "Manage" en el lado derecho de la tabla y ve a la pestaña "Web Settings".

En el campo "Allowed redirect URLs", añade la siguiente URL:

- `${your_logto_origin}/callback/${connector_id}`.

El `connector_id` se puede encontrar en la barra superior de la página de detalles del conector en la Logto Admin Console.

### Configura tu conector \{#configure-your-connector}

En la configuración de tu conector de Logto, completa los siguientes campos con los valores obtenidos en el último paso:

- **clientId:** El Client ID de tu aplicación.
- **clientSecret:** El Client Secret de tu aplicación.

`scope` es una lista de OIDC scopes delimitada por espacios. Si no se proporciona, el scope predeterminado es `openid profile`.

### Tipos de configuración \{#config-types}

| Nombre       | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
