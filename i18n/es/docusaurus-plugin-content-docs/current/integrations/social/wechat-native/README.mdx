# Configura el inicio de sesión social con WeChat (Native) (Set up social login with WeChat (Native))

El conector oficial de Logto para el inicio de sesión social con WeChat en aplicaciones nativas (iOS y Android).

<GuideTip />

## Comenzar \{#get-started}

Si no conoces el concepto de conector o no sabes cómo añadir este conector a tu experiencia de inicio de sesión, por favor consulta el [tutorial de Logto](/connectors/social-connectors).

En las aplicaciones nativas, no puedes usar la web como método de inicio de sesión de WeChat: se requiere la navegación a la aplicación de WeChat, y también es necesario utilizar su SDK.

Sabemos que suena complicado, pero no te preocupes. Lo resolveremos fácilmente en este tutorial.

<Integration />

## Probar el conector nativo de WeChat \{#test-wechat-native-connector}

Eso es todo. No olvides [habilitar el conector social en la experiencia de inicio de sesión](/connectors/social-connectors/#enable-social-sign-in).

Una vez habilitado el conector nativo de WeChat, puedes compilar y ejecutar tu aplicación para ver si funciona.

:::caution

WeChat no tiene un plan para aquellos dispositivos sin la aplicación de WeChat instalada. Logto ocultará este conector durante el inicio de sesión en ese caso (lo cual es la forma recomendada según la [guía oficial de desarrollo](https://developers.weixin.qq.com/doc/oplatform/en/Mobile_App/WeChat_Login/Development_Guide.html)).

:::
