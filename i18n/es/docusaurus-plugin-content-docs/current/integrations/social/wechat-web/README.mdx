# Configura el inicio de sesión social con WeChat (Web) (Set up social login with WeChat (Web))

El conector oficial de Logto para el inicio de sesión social con WeChat en aplicaciones web.

<GuideTip />

## Comenzar \{#get-started}

Si no conoces el concepto de conector o no sabes cómo añadir este conector a tu experiencia de inicio de sesión, consulta el [tutorial de Logto](/connectors/social-connectors).

:::caution

Este conector es solo para aplicaciones web. Si buscas el método para iniciar sesión con WeChat en aplicaciones nativas, consulta el [conector nativo de WeChat](/integrations/wechat-native).

:::

<Integration />

### Probar el conector web de WeChat \{#test-wechat-web-connector}

Eso es todo. No olvides [habilitar el conector social en la experiencia de inicio de sesión](/connectors/social-connectors/#enable-social-sign-in).

Una vez que el conector web de WeChat esté habilitado, puedes volver a iniciar sesión en tu aplicación para comprobar si funciona.
