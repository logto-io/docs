## Registrar cuenta de Postmark \{#register-postmark-account}

Crea una nueva cuenta en el [sitio web de Postmark](https://postmark.com/). Puedes omitir este paso si ya tienes una cuenta.

## Configura tu conector \{#configure-your-connector}

Rellena el campo `serverToken` con el Server Token que encuentras en la configuración de tu servidor en Postmark.

Rellena el campo `fromEmail` con la _dirección From_ del remitente.

Para habilitar todos los flujos de usuario, se requieren plantillas con usageType `Register`, `SignIn`, `ForgotPassword` y `Generic`.

Aquí tienes un ejemplo de JSON de plantilla de conector de Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Probar el conector de correo electrónico de Postmark \{#test-postmark-email-connector}

Puedes escribir una dirección de correo electrónico y hacer clic en "Enviar" para comprobar si la configuración funciona antes de "Guardar y finalizar".

Eso es todo. No olvides <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">habilitar el conector en la experiencia de inicio de sesión</MainSiteUrl>.

## Tipos de configuración \{#config-types}

| Nombre      | Tipo   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propiedades de la plantilla | Tipo        | Valores enum                                         |
| --------------------------- | ----------- | ---------------------------------------------------- |
| usageType                   | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| templateAlias               | string      | N/A                                                  |
