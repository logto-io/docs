# Configura la verificación por correo electrónico con el servicio de correo Postmark

Conector de Logto para el servicio de correo Postmark.

<GuideTip />

## Comenzar \{#get-started}

Postmark es una plataforma de correo para correos transaccionales y de marketing. Podemos usar su función de envío de correos para enviar un _código de verificación_.

## Registrar cuenta de Postmark \{#register-postmark-account}

Crea una nueva cuenta en el [sitio web de Postmark](https://postmark.com/). Puedes omitir este paso si ya tienes una cuenta.

## Configura tu conector \{#configure-your-connector}

Rellena el campo `serverToken` con el Token del Servidor que encuentres en la configuración de tu servidor en Postmark.

Rellena el campo `fromEmail` con la _Dirección de Remitente_.

Para habilitar completamente los flujos de usuario, se requieren plantillas con `usageType` `Register`, `SignIn`, `ForgotPassword` y `Generic`.

Aquí tienes un ejemplo de JSON de plantilla de conector de Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Probar el conector de correo de Postmark \{#test-postmark-email-connector}

Puedes escribir una dirección de correo electrónico y hacer clic en "Enviar" para ver si la configuración funciona antes de "Guardar y Listo".

Eso es todo. No olvides [Habilitar el conector en la experiencia de inicio de sesión](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)

## Tipos de configuración \{#config-types}

| Nombre      | Tipo   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propiedades de la plantilla | Tipo        | Valores del enum                                      |
| --------------------------- | ----------- | ----------------------------------------------------- |
| usageType                   | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias               | string      | N/A                                                   |
