# Conector YunPian SMS

El conector oficial de Logto para el servicio SMS de YunPian.

<GuideTip />

## Comenzar \{#get-started}

YunPian es un proveedor de servicios de comunicación que ofrece varios servicios, incluido el SMS. El Conector YunPian SMS es un complemento proporcionado por el equipo de Logto para integrarse con el servicio SMS de YunPian, permitiendo que los usuarios finales de Logto se registren e inicien sesión mediante códigos de verificación por SMS.

## Configurar el servicio SMS en la Consola de YunPian \{#set-up-sms-service-in-yunpian-console}

### Crear una cuenta de YunPian \{#create-a-yunpian-account}

Visita el [sitio web de YunPian](https://www.yunpian.com/) para registrar una cuenta y completar la verificación de identidad.

### Obtener API KEY \{#get-api-key}

1. Inicia sesión en la Consola de YunPian
2. Ve a "Configuración de cuenta" -> "Gestión de subcuentas"
3. Busca y copia la API KEY

### Configurar plantillas SMS \{#configure-sms-templates}

1. En la Consola de YunPian, ve a "SMS nacional" -> "Registro de firma"
2. Crea y envía una firma, espera la aprobación del operador
3. Ve a "SMS nacional" -> "Registro de plantilla" y selecciona "Código de verificación"
4. Crea una plantilla de código de verificación, asegúrate de que incluya la variable `#code#` (también puedes usar "Plantillas comunes" para acelerar el proceso de aprobación)
5. Espera la aprobación de la plantilla
6. Si necesitas enviar SMS internacionales, repite los pasos anteriores pero selecciona "SMS internacional" -> "Registro de plantilla"

## Configurar en Logto \{#configure-in-logto}

1. En la Consola de Logto, ve a "Connectors"
2. Busca y haz clic en "YunPian SMS Service"
3. Completa el formulario de configuración:
   - API KEY: La API KEY obtenida de YunPian
   - Plantillas SMS: Configura las plantillas según el uso, asegúrate de que coincidan exactamente con las plantillas aprobadas en YunPian

## Notas \{#notes}

1. El contenido de la plantilla SMS debe coincidir exactamente con la plantilla aprobada en YunPian
2. El marcador de posición de la variable del código de verificación en las plantillas de YunPian es `#code#`, mientras que en la configuración del conector es `{{code}}`
3. YunPian agrega automáticamente la firma predeterminada según la API KEY, no es necesario incluir la firma en el contenido de la plantilla
4. Se recomienda probar la configuración antes de su uso formal
