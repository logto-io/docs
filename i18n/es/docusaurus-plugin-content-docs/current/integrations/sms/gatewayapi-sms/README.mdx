# Configura la verificación por SMS con GatewayAPI SMS

El conector oficial de Logto para GatewayAPI SMS.

<GuideTip />

## Comenzar

GatewayAPI es un proveedor de servicios en la nube en Europa, que ofrece muchos servicios en la nube, incluido SMS (servicio de mensajes cortos). El Conector SMS de GatewayAPI es un complemento proporcionado por el equipo de Logto para llamar al servicio SMS de GatewayAPI, con la ayuda del cual los usuarios finales de Logto pueden registrarse e iniciar sesión en su cuenta de Logto a través del código de verificación por SMS.

## Configuración en GatewayAPI

:::tip
Puedes omitir algunas secciones si ya las has completado.
:::

### Crear una cuenta en GatewayAPI

Ve al [sitio web de GatewayAPI](https://www.gatewayapi.com/) y registra tu cuenta de GatewayAPI si no tienes una.

### Habilitar cuenta

Es posible que necesites habilitar tu cuenta antes de usar el servicio SMS. Puedes contactar al servicio al cliente de GatewayAPI para habilitar tu cuenta.

### Obtener token de API

Ve a la página de API Keys desde la consola de GatewayAPI y encuentra el token de API o crea uno nuevo.

## Configuración en Logto

1. **Endpoint**: Si tu cuenta de GatewayAPI está en la región de la UE, debes usar el endpoint https://gatewayapi.com/rest/mtsms. Si tu cuenta de GatewayAPI está en la región de EE. UU., debes usar el endpoint https://gatewayapi.com/rest/mtsms.
2. **API Token**: El token de API que creaste en el paso anterior.
3. **Sender**: El remitente que deseas usar para enviar el SMS.
4. **Templates**: Las plantillas que deseas usar para enviar el SMS, puedes usar las plantillas predeterminadas o modificarlas según sea necesario.
