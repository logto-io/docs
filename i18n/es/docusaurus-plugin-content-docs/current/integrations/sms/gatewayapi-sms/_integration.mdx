### Crea una cuenta en GatewayAPI \{#create-an-gatewayapi-account}

Ve al [sitio web de GatewayAPI](https://www.gatewayapi.com/) y registra tu cuenta de GatewayAPI si aún no tienes una.

### Habilita la cuenta \{#enable-account}

Es posible que necesites habilitar tu cuenta antes de usar el servicio de SMS. Puedes contactar al servicio de atención al cliente de GatewayAPI para habilitar tu cuenta.

### Obtén el token de API \{#get-api-token}

Ve a la página de API Keys desde la consola de GatewayAPI y encuentra el token de API o crea uno nuevo.

## Configura en Logto \{#set-up-in-logto}

1. **Endpoint**: Si tu cuenta de GatewayAPI está en la región de la UE, debes usar el endpoint https://gatewayapi.com/rest/mtsms. Si tu cuenta de GatewayAPI está en la región de EE. UU., debes usar el endpoint https://gatewayapi.com/rest/mtsms.
2. **API Token**: El token de API que creaste en el paso anterior.
3. **Remitente (Sender)**: El remitente que deseas usar para enviar el SMS.
4. **Plantillas (Templates)**: Las plantillas que deseas usar para enviar el SMS; puedes usar las plantillas predeterminadas o modificarlas según sea necesario.
