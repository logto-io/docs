### Crear una cuenta de GatewayAPI \{#create-an-gatewayapi-account}

Ve al [sitio web de GatewayAPI](https://www.gatewayapi.com/) y registra tu cuenta de GatewayAPI si no tienes una.

### Habilitar cuenta \{#enable-account}

Es posible que necesites habilitar tu cuenta antes de usar el servicio de SMS. Puedes contactar al servicio de atención al cliente de GatewayAPI para habilitar tu cuenta.

### Obtener token de API \{#get-api-token}

Ve a la página de API Keys desde la consola de GatewayAPI y encuentra el token de API o crea un nuevo token de API.

## Configurar en Logto \{#set-up-in-logto}

1. **Endpoint**: Si tu cuenta de GatewayAPI está en la región de la UE, debes usar el endpoint https://gatewayapi.com/rest/mtsms. Si tu cuenta de GatewayAPI está en la región de EE. UU., debes usar el endpoint https://gatewayapi.com/rest/mtsms.
2. **Token de API**: El token de API que creaste en el paso anterior.
3. **Remitente**: El remitente que deseas usar para enviar el SMS.
4. **Plantillas**: Las plantillas que deseas usar para enviar el SMS, puedes usar las plantillas predeterminadas o modificarlas según sea necesario.
