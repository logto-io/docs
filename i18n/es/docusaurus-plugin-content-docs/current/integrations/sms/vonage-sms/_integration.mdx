## Configura en Vonage \{#set-up-in-vonage}

:::tip
Puedes omitir este paso si ya los has completado.
:::

Para trabajar con este conector, necesitarás [registrarte para obtener una cuenta](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) en Vonage. Esto te proporcionará una API key y un secreto que puedes usar para acceder a las APIs a través de este conector.

Una vez que tengas una cuenta, puedes encontrar tu API key y API secret en la parte superior del Vonage API Dashboard.

Y es posible que necesites [alquilar un número virtual](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) para enviar mensajes SMS.

Consulta la [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview) para más información.

## Configura en Logto \{#set-up-in-logto}

1. **API Key**: Tu API key de Vonage.
2. **API Secret**: Tu API secret de Vonage.
3. **Nombre de Marca**: El nombre de la marca que deseas usar para enviar el SMS, esto también se llama el campo `from`, consulta la [Identidad del Remitente](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) para más información.
4. **Plantillas**: Las plantillas que deseas usar para enviar el SMS, puedes usar las plantillas predeterminadas o modificarlas según sea necesario.
