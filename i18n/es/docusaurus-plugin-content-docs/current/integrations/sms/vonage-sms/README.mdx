# Configura la verificación por SMS con Vonage SMS

El conector oficial de Logto para Vonage SMS.

<GuideTip />

## Comenzar \{#get-started}

Vonage es un proveedor global de comunicaciones, que ofrece servicios de comunicación en la nube robustos, incluido el SMS (servicio de mensajes cortos). El Conector Vonage SMS es un complemento proporcionado por el equipo de Logto para permitir que los usuarios finales de Logto se registren e inicien sesión en su cuenta Logto mediante códigos de verificación por SMS.
