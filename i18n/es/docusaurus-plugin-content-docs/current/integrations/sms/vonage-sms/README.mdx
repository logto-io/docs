# Configura la verificación por SMS con Vonage SMS

El conector oficial de Logto para Vonage SMS.

<GuideTip />

## Comenzar \{#get-started}

Vonage es un proveedor de comunicaciones global, que ofrece servicios de comunicación robustos basados en la nube, incluyendo SMS (servicio de mensajes cortos). El Conector SMS de Vonage es un complemento proporcionado por el equipo de Logto para permitir que los usuarios finales de Logto se registren e inicien sesión en su cuenta de Logto a través de códigos de verificación por SMS.

## Configuración en Vonage \{#set-up-in-vonage}

:::tip
Puedes omitir este paso si ya lo has completado.
:::

Para trabajar con este conector, necesitarás [registrarte para obtener una cuenta](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) en Vonage. Esto te proporcionará una clave de API y un secreto que puedes usar para acceder a las APIs a través de este conector.

Una vez que tengas una cuenta, puedes encontrar tu clave de API y secreto de API en la parte superior del Panel de API de Vonage.

Y es posible que necesites [alquilar un número virtual](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) para enviar mensajes SMS.

Consulta la [API de SMS de Vonage](https://developer.vonage.com/en/messaging/sms/overview) para más información.

## Configuración en Logto \{#set-up-in-logto}

1. **API Key**: Tu clave de API de Vonage.
2. **API Secret**: Tu secreto de API de Vonage.
3. **Brand Name**: El nombre de la marca que deseas usar para enviar el SMS, esto también se llama el campo `from`, consulta la [Identidad del Remitente](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) para más información.
4. **Templates**: Las plantillas que deseas usar para enviar el SMS, puedes usar las plantillas predeterminadas o modificarlas según sea necesario.
