# Integra la autenticación de Logto

Logto ofrece soluciones integrales de autenticación para aplicaciones web, móviles y de escritorio, admite la autenticación [máquina a máquina (M2M)](/quick-starts/m2m) entre servicios y puede funcionar como Proveedor de Identidad (IdP) para [aplicaciones de terceros](/integrate-logto/third-party-applications) a través de protocolos estándar como [OpenID Connect (OIDC)](https://auth.wiki/openid-connect) y [OAuth 2.0](https://auth.wiki/oauth-2.0).

Comienza tu integración seleccionando la solución que mejor se adapte a tus necesidades:

## Añade autenticación para tus aplicaciones \{#add-authentication-for-your-applications}

Ya sea que estés desarrollando aplicaciones orientadas al usuario (como aplicaciones web, móviles o de escritorio) o aplicaciones máquina a máquina (M2M) para la comunicación entre servicios, puedes implementar rápidamente funciones completas de [autenticación (Authentication)](/end-user-flows) y [gestión de usuarios](/user-management) integrando Logto.

## Añade aplicaciones SAML \{#add-saml-apps}

Logto (como Proveedor de Identidad) admite la integración con aplicaciones basadas en el protocolo SAML (como Proveedor de Servicios). A través de la integración de aplicaciones SAML, puedes ofrecer a los usuarios empresariales una experiencia segura y estandarizada de inicio de sesión único (SSO).

## Añade aplicaciones OIDC / OAuth \{#add-oidc-oauth-apps}

Cuando quieras permitir que aplicaciones de terceros se autentiquen con tu sistema de identidad Logto, puedes configurar Logto como un [Proveedor de Identidad (IdP)](https://auth.wiki/identity-provider).

Logto puede usarse como IdP a través de protocolos estándar como OAuth, OIDC, y los servicios externos pueden implementar "Iniciar sesión con Logto" igual que "Iniciar sesión con Google", permitiendo a sus usuarios autenticarse usando tu sistema de usuarios Logto.

Al crear aplicaciones de terceros en Logto, estableces conexiones seguras entre tu sistema de identidad y servicios externos.

## Amplía las funciones de Logto con Management API \{#extend-logto-features-with-management-api}

Logto [Management API](https://openapi.logto.io/) te permite controlar y automatizar programáticamente la funcionalidad de Logto en tus aplicaciones.

Cuando necesites ampliar las capacidades de Logto para mejorar tu producto (como [gestionar organizaciones multi-inquilino](/organizations), o automatizar flujos de trabajo con Logto), puedes aprovechar Logto Management API para construir funciones avanzadas que se adapten a las necesidades de tu producto.
