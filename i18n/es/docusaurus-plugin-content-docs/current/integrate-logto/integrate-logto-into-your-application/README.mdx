# Integra Logto en tu aplicación

Sigue estos pasos para añadir autenticación a tus aplicaciones con Logto, ya sea una aplicación orientada al usuario o un servicio máquina a máquina:

1. Navega a <CloudLink to="/applications">Consola > Aplicaciones</CloudLink>

2. Haz clic en "Crear aplicación" para añadir una nueva aplicación

3. Elige el marco de tu aplicación para comenzar. Si no encuentras tu marco, haz clic en el botón "Crear aplicación sin marco" en la parte inferior derecha de la página de creación de aplicaciones para crear una aplicación seleccionando un [tipo de aplicación](/integrate-logto/application-data-structure/#application-types) o presenta una solicitud de función o contribuye con un SDK siguiendo nuestras [convenciones de SDK](/developers/sdk-conventions).

4. Después de seleccionar tu marco, verás una guía de inicio rápido para el SDK del marco. Sigue los pasos para configurar e integrar tu aplicación. Si necesitas ayuda para entender los conceptos involucrados en el proceso de integración, puedes consultar [Entendiendo el flujo de autenticación de Logto](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) para una comprensión más profunda de la integración.

:::note
La guía en la consola es solo para un inicio rápido con Logto usando nuestro SDK. Para guías de integración completas, incluyendo el uso avanzado del SDK, consulta la sección de [Inicios rápidos](/quick-starts).
:::

5. Una vez completado, estás listo para explorar más sobre Logto:
