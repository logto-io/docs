# Tipos de eventos de los registros de auditoría

Puedes filtrar los tipos de eventos en <CloudLink to="/audit-logs">Logto Console > Audit Logs</CloudLink>.

:::note

Logto ahora admite la recuperación de registros relacionados con interacciones de usuarios finales a través de las [Experience APIs](https://openapi.logto.io/group/endpoint-experience).

Los registros de auditoría para las [Management APIs](/integrate-logto/interact-with-management-api) y las [Account APIs](https://openapi.logto.io/group/endpoint-account-center) estarán disponibles próximamente.

No dudes en [contactarnos](https://logto.io/contact) si deseas compartir tus requisitos.

:::
