# Registros de auditoría

El registro de auditoría de Logto te permite monitorear fácilmente la actividad de los usuarios y los eventos. Proporciona una base sólida para varios escenarios de gestión de usuarios y verificación de salud empresarial.

## Ver todos los registros \{#view-all-logs}

Navega a <CloudLink to="/audit-logs">Consola > Registros de auditoría</CloudLink>. Logto captura y organiza eventos de autenticación en una tabla. Realiza un seguimiento del nombre del evento, usuario, aplicación y marca de tiempo. Puedes reducir los resultados filtrando según el nombre del evento y el nombre de la aplicación. Al hacer clic en un evento específico, se proporcionarán detalles adicionales.

:::warning
Los registros de auditoría solo contienen registros que ocurren durante el proceso de autenticación de usuario, los registros de operaciones de Management API no se registran.
:::

## Capturar actividad de usuario a nivel de inquilino \{#capture-user-activity-at-the-tenant-level}

Los registros de Logto ofrecen detalles completos, asegurando facilidad de acción y seguridad del cliente. Capturan y registran la siguiente información:

- Tipo de evento (la lista completa de eventos de registro de auditoría se puede encontrar [aquí](https://github.com/logto-io/logto/blob/master/packages/console/src/consts/logs.ts))
- Aplicación involucrada
- Dirección IP
- Usuario involucrado
- ID del registro
- Marca de tiempo
- User-agent

Al mantener estos registros de eventos, las organizaciones pueden detectar efectivamente posibles riesgos de seguridad y abordarlos rápidamente para prevenir el acceso no autorizado al sistema.

## Realizar un análisis detallado a nivel de usuario \{#perform-a-detailed-analysis-at-the-user-level}

Los administradores pueden realizar un análisis detallado de los registros asociados con usuarios específicos, facilitando investigaciones exhaustivas sobre eventos específicos. El proceso de navegación es sencillo y fácil de usar.

Para acceder a los registros específicos de usuarios, sigue estos pasos:

1. Navega a <CloudLink to="/users">Consola > Gestión de usuarios</CloudLink>.
2. Selecciona el usuario deseado y ve a la página de detalles.
3. Haz clic en "Registros de usuario". La tabla resultante mostrará exclusivamente eventos de registro realizados y desencadenados por ese usuario en particular.
