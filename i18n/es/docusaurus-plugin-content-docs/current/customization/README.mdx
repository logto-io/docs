# Personalización

Haz que Logto sea tuyo personalizando cada aspecto: desde la marca y la interfaz de usuario hasta los idiomas y la configuración de dominio.

## Interfaz preconstruida \{#prebuilt-ui}

Comienza con nuestros componentes de interfaz preconstruidos y personaliza los elementos esenciales de tu marca. Crea ya sea:

- Una experiencia de autenticación unificada en toda tu plataforma.
- Páginas de inicio de sesión dedicadas para aplicaciones u organizaciones específicas.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Adapta a tu marca',
      href: '/customization/match-your-brand',
      description:
        'Aprende cómo personalizar los elementos básicos de la marca para una experiencia de inicio de sesión y centro de cuentas listos para usar.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## Contenido e idiomas \{#content--languages}

Logto ofrece soporte para 15 idiomas por defecto. Amplía el soporte de idiomas o modifica las traducciones existentes para adaptarlas a tus necesidades.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Idiomas y contenido',
      href: '/customization/localized-languages',
      description:
        'Aprende cómo añadir nuevos idiomas y personalizar el contenido para tus usuarios.',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## Dominio personalizado \{#custom-domain}

Refuerza la presencia de tu marca usando tu propio dominio para las URLs de autenticación. Sustituye el dominio predeterminado de Logto por tu dominio de marca.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Dominio personalizado',
      href: '/logto-cloud/custom-domain',
      description:
        'Mejora la marca y la confianza del usuario usando tu propio dominio como URL de inicio de sesión o registro.',
      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>

## Vista previa en vivo \{#live-preview}

Experimenta con diferentes flujos de autenticación ([registro](/end-user-flows/sign-up-and-sign-in/sign-up), [inicio de sesión](/end-user-flows/sign-up-and-sign-in/sign-in), [inicio de sesión social](/end-user-flows/sign-up-and-sign-in/social-sign-in), [olvidé mi contraseña](/end-user-flows/sign-up-and-sign-in/reset-password), [MFA](/end-user-flows/mfa), [SSO](/end-user-flows/enterprise-sso), etc.) usando nuestra función de Vista Previa en Vivo. Prueba tu experiencia de inicio de sesión al instante a través de nuestra aplicación de demostración integrada.
