# Personalización

Haz que Logto sea tuyo personalizando cada aspecto, desde la marca y la interfaz de usuario hasta los idiomas y la configuración del dominio.

## Interfaz de usuario preconstruida

Comienza con nuestros componentes de interfaz de usuario preconstruidos y personaliza los elementos esenciales de la marca. Crea ya sea:

- Una experiencia de autenticación unificada en toda tu plataforma.
- Páginas de inicio de sesión dedicadas para aplicaciones u organizaciones específicas.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Ajusta a tu marca',
      href: '/customization/match-your-brand',
      description:
        'Aprende cómo personalizar los elementos básicos de la marca para una experiencia de inicio de sesión lista para usar.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## Contenido e idiomas

Logto proporciona soporte para 15 idiomas por defecto. Amplía el soporte de idiomas o modifica las traducciones existentes para satisfacer tus necesidades.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Idiomas y contenido',
      href: '/customization/localized-languages',
      description:
        'Aprende cómo agregar nuevos idiomas y personalizar el contenido para tus usuarios.',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## Dominio personalizado

Fortalece la presencia de tu marca utilizando tu dominio para las URLs de autenticación. Reemplaza el dominio predeterminado de Logto con tu dominio de marca.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Dominio personalizado',
      href: '/logto-cloud/custom-domain',
      description:
        'Mejora la marca y la confianza del usuario utilizando tu propio dominio como la URL de inicio de sesión o registro.',
      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>

## Vista previa en vivo

Experimenta con diferentes flujos de autenticación ([registro](/end-user-flows/sign-up-and-sign-in/sign-up), [inicio de sesión](/end-user-flows/sign-up-and-sign-in/sign-in), [inicio de sesión social](/end-user-flows/sign-up-and-sign-in/social-sign-in), [olvidó su contraseña](/end-user-flows/sign-up-and-sign-in/reset-password), [MFA](/end-user-flows/mfa), [SSO](/end-user-flows/enterprise-sso), etc.) utilizando nuestra función de Vista Previa en Vivo. Prueba tu experiencia de inicio de sesión instantáneamente a través de nuestra aplicación de demostración integrada.
