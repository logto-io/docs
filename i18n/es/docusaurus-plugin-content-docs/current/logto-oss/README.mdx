# Logto OSS

Logto ofrece versiones tanto de código abierto (OSS) como en la nube; esta sección se centrará en la solución OSS.

Logto OSS es una solución de identidad de uso gratuito que proporciona a los desarrolladores capacidades de personalización completa y control total sobre sus datos. Logto OSS ofrece la flexibilidad de adaptar la base de código para satisfacer necesidades específicas, es una opción perfecta para aquellos que priorizan la soberanía de los datos y desean una solución de proveedor de identidad personalizada que pueda crecer con su aplicación.

## Funciones soportadas por Logto OSS

Logto OSS soporta la mayoría de las capacidades centrales del servicio Logto y se actualiza y mantiene regularmente.

Algunas funciones avanzadas son actualmente exclusivas de la versión Logto Cloud, incluyendo [Logto Protected App](/integrate-logto/protected-app/), [servicio de correo electrónico gratuito](/connectors/email-connectors/built-in-email-service), [SSO iniciado por IdP](/end-user-flows/enterprise-sso/idp-initiated-sso), y SAML IdP.
