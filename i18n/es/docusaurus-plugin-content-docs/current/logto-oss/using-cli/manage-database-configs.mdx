# Gestionar configuraciones de base de datos

Logto ayuda a mantener algunas configuraciones técnicas, como claves OIDC y de Cookie. Antes de poder gestionarlas a través de Admin Console o Management API, puedes usar CLI para lograr el objetivo.

## Obtener configuración por clave

Uso:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get <key> [keys...]
```

  </TabItem>

</Tabs>

Supongamos que quieres obtener las claves privadas de OIDC:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get oidc.privateKeys
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get oidc.privateKeys
```

  </TabItem>

</Tabs>

La salida será como:

```bash
oidc.privateKeys=["\n-----BEGIN PRIVATE KEY-----\nMIIJRAIBA..."]
```

Ejecuta `logto db config get --help` para ver todas las claves disponibles, o consulta [Configuración](/concepts/core-service/configuration/#database-configs) para una explicación detallada.

## Establecer configuración por clave

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set <key> [keys...]
```

  </TabItem>

</Tabs>

Supongamos que quieres establecer las claves de Cookie de OIDC:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>

</Tabs>

Normalmente, el valor a establecer debe ser una cadena JSON válida y seguir la definición de tipo predefinida. Ejecuta `logto db config set --help` para ver todas las claves disponibles, o consulta [Configuración](/concepts/core-service/configuration/) para una explicación detallada.
