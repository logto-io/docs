# Instalar Logto

## Interactivo \{#interactive}

Ejecuta el siguiente comando para proceder con una instalación interactiva en tu terminal:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## Opciones e instalación silenciosa \{#options-and-silent-installation}

Alternativamente, puedes agregar las siguientes opciones para omitir algunas preguntas:

```bash
# Todas estas opciones son opcionales
--db-url      La URL de Postgres para la base de datos de Logto
-p, --path    Ruta a tu instancia de Logto, debe ser una ruta no existente
--ss          Omitir la siembra de la base de datos de Logto
--oc          Agregar conectores oficiales después de la instalación
```

Ejecuta `logto init --help` para obtener información completa de ayuda.

Aquí tienes un ejemplo para realizar una instalación silenciosa (por ejemplo, en CI):

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

Hará lo siguiente en orden:

1. Descargar y extraer Logto en `./logto`
2. Intentar inicializar y sembrar la base de datos con la URL `postgresql://your-postgres-dsn:5432/logto`
3. Agregar los conectores oficiales de Logto a `./logto/packages/core/connectors`
