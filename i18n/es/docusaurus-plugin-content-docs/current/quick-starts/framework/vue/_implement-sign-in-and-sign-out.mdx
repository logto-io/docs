### Implementar inicio y cierre de sesión

Proporcionamos dos composables `useHandleSignInCallback()` y `useLogto()` que pueden ayudarte a gestionar fácilmente el flujo de autenticación.

<SignInNote calling=".signIn()" />

```tsx title="views/HomeView.vue"
import { useLogto } from '@logto/vue';

const { signIn, signOut, isAuthenticated } = useLogto();

const onClickSignIn = () => signIn('http://localhost:3000/callback');
const onClickSignOut = () => signOut('http://localhost:3000');
```

```html
<template>
  <button v-if="!isAuthenticated" @click="onClickSignIn">Iniciar sesión</button>
  <button v-else @click="onClickSignOut">Cerrar sesión</button>
</template>
```
