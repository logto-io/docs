# Añade autenticación a tu aplicación Auth.js (Next Auth)

Esta guía te mostrará cómo integrar Logto en tu aplicación Next.js con [Auth.js](https://authjs.dev/), anteriormente conocido como Next Auth.

<GuideTip />

## Prerrequisitos \{#prerequisites}

- Una cuenta de [Logto Cloud](https://cloud.logto.io) o un [Logto autoalojado](/introduction/set-up-logto-oss).
- Una aplicación tradicional de Logto creada.
- Un proyecto Next.js con Auth.js, consulta la [documentación de Auth.js](https://authjs.dev/getting-started/installation).
