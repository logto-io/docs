## Uso en middleware o rutas de API \{#use-in-middleware-or-api-routes}

Los composables `useLogtoClient()` y `useLogtoUser()` no están disponibles en middleware o rutas de API. Puedes usar la función `logtoEventHandler()` para obtener el cliente de Logto y otro contexto:

```ts title="api/auth/get-access-token.ts"
import { logtoEventHandler } from '@logto/nuxt';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
