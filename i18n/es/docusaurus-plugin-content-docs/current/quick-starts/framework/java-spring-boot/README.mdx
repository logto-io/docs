# Añade autenticación a tu aplicación Java Spring Boot (Add authentication to your Java Spring Boot application)

Esta guía te mostrará cómo integrar Logto en tu aplicación Java Spring Boot.

<GuideTip />

## Requisitos previos \{#prerequisites}

- Una cuenta de [Logto Cloud](https://cloud.logto.io) o un [Logto autoalojado](/introduction/set-up-logto-oss).
- Nuestro código de ejemplo fue creado usando el [securing web starter](https://spring.io/guides/gs/securing-web) de Spring Boot. Sigue las instrucciones para iniciar una nueva aplicación web si aún no tienes una.
- En esta guía, utilizaremos las librerías [Spring Security](https://spring.io/projects/spring-security) y [Spring Security OAuth2](https://spring.io/guides/tutorials/spring-boot-oauth2) para manejar el flujo de autenticación OIDC con Logto. Por favor, asegúrate de revisar la documentación oficial para comprender los conceptos.
