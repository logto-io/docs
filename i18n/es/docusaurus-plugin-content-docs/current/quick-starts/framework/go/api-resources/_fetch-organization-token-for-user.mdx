#### Recursos de API de la organización (Organization API resources) \{#organization-api-resources}

Para obtener un token de acceso para un recurso de API en una organización, puedes usar el método `GetAccessTokenWithOptions` con tanto el recurso de API como el ID de la organización como parámetros:

```go
accessToken, error := client.GetAccessTokenWithOptions(
  client.GetAccessTokenOptions{
    Resource: 'https://shopping.your-app.com/api',
    OrganizationId: organizationId,
  },
);
```
