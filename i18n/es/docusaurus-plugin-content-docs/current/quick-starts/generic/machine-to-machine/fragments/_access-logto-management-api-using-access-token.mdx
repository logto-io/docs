<TabItem value="with-@logto-api-sdk" label="Node.js con `@logto/api` SDK">

Con el SDK `@logto/api`, puedes interactuar directamente con cualquier Management API utilizando el siguiente fragmento de código. El token de acceso (Access token) se almacena en caché internamente y se actualiza automáticamente si es necesario.

```js
import { createManagementApi } from '@logto/api/management';

const { apiClient } = createManagementApi('your-tenant-id', {
  clientId: 'your-client-id',
  clientSecret: 'your-client-secret',
});

const response = await apiClient.GET('/api/applications');
console.log(response.data);
```

<TabItem value="nodejs" label="Node.js">

<AccessTokenUsage />

```js
const logtoEndpoint = 'https://your.logto.endpoint'; // Reemplaza con tu endpoint de Logto
const accessToken = 'eyJhb...2g'; // Token de acceso (Access Token)

const fetchLogtoApplications = async () => {
  return await fetch(`${logtoEndpoint}/api/applications`, {
    method: 'GET',
    headers: {
      Authorization: `Bearer ${accessToken}`,
    },
  });
};
```

</TabItem>

<TabItem value="curl" label="cURL">

<AccessTokenUsage />

```bash
curl --location \
  --request GET 'https://your.logto.endpoint/api/applications' \
  --header 'Authorization: Bearer eyJhbG...2g'
```

Recuerda reemplazar los valores reales por los tuyos propios. El valor después de `Bearer` debe ser el token de acceso (Access token, JWT) que recibiste.
