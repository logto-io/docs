La aplicación M2M realiza una solicitud `POST` al endpoint de token para obtener un token de acceso agregando los siguientes parámetros utilizando el formato `application/x-www-form-urlencoded` en el cuerpo de la entidad de la solicitud HTTP:

- **grant_type**: Debe establecerse en `client_credentials`
- **resource**: El recurso al que deseas acceder
- **scope**: El alcance de la solicitud de acceso

Además, también necesitas incluir las credenciales de tu aplicación M2M en el encabezado de la solicitud para que el endpoint de token pueda autenticar tu aplicación M2M.

Esto se logra incluyendo las credenciales de la aplicación en el formulario de [autenticación básica (Basic authentication)](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Authorization#basic_authentication) en el encabezado `Authorization` de la solicitud: codifica en base64 `{App ID}:{App Secret}` y colócalo como el valor después del prefijo `Basic`.
