# コネクター

コネクターは、Logto と外部サービスをつなぐ橋です。これにより、[メール認証](/end-user-flows/sign-up-and-sign-in/sign-in)、[SMS 認証](/end-user-flows/sign-up-and-sign-in/sign-in)、[ソーシャルログイン](/end-user-flows/sign-up-and-sign-in/social-sign-in)、および [エンタープライズシングルサインオン (SSO)](/end-user-flows/enterprise-sso) などの [パスワードレス認証](https://auth.wiki/passwordless) 方法が可能になります。

ユーザー名とパスワードを使用した従来のアカウント登録は、面倒でストレスの多いプロセスになることがあります。ユーザーは複雑なパスワードを覚えるのに苦労し、不便さから新しいアカウントを作成することをためらうことがあります。ユーザーがメールアドレスや電話番号を提供し、認証コードで確認するか、Google/Facebook などのアカウントや Microsoft などの企業アカウントを使用してログインできれば、ログインプロセスを大幅に簡素化し、ユーザーの離脱を減らすことができます。この問題を解決するためにコネクターが作られました。
