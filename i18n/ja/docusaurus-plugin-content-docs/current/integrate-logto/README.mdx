# Logto 認証 (Authentication) の統合

:::tip[AI で統合する (Logto Cloud)]
AI 搭載の開発ツールを利用している場合は、[Logto MCP Server](/logto-cloud/logto-mcp-server) を試してみてください。AI のサポートで Logto を統合できます。フレームワークを自動検出し、アプリケーションを作成し、動作するコードを生成します。
:::

Logto は、Web、モバイル、デスクトップアプリケーション向けの包括的な認証 (Authentication) ソリューションを提供し、サービス間の [マシン間通信 (M2M)](/quick-starts/m2m) 認証 (Authentication)、入力制限デバイス向けの [デバイスフロー](/quick-starts/device-flow)、[OpenID Connect(OIDC)](https://auth.wiki/openid-connect) や [OAuth 2.0](https://auth.wiki/oauth-2.0) などの標準プロトコルを通じて [サードパーティアプリケーション](/integrate-logto/third-party-applications) のためのアイデンティティプロバイダー (IdP) としても機能します。

ニーズに最適なソリューションを選択して統合を始めましょう：

## アプリケーションへの認証 (Authentication) 機能の追加 \{#add-authentication-for-your-applications}

ユーザー向けアプリケーション（Web、モバイル、デスクトップアプリなど）や、サービス間通信のためのマシン間通信 (M2M) アプリケーションを構築する場合でも、Logto を統合することで包括的な [認証 (Authentication)](/end-user-flows) および [ユーザー管理](/user-management) 機能をすばやく実装できます。

OIDC 標準に基づく Logto により、すべてのアプリケーションで **オムニサインイン** が可能です。複数のアプリケーションを Logto と統合すると、同じアイデンティティシステムと認証 (Authentication) 方法を共有できます。つまり、ユーザーは一度サインインするだけで、すべての接続済みアプリケーションにシームレスにアクセスでき、統一された認証 (Authentication) 体験を得られます。

## サードパーティ OIDC/OAuth アプリの追加 \{#add-third-party-oidc-oauth-apps}

サードパーティアプリケーションで Logto のアイデンティティシステムによる認証 (Authentication) を有効にしたい場合は、Logto を [アイデンティティプロバイダー (IdP)](https://auth.wiki/identity-provider) として設定できます。

Logto は OAuth や OIDC などの標準プロトコルを通じて IdP として利用でき、外部サービスは「Logto でサインイン」を実装できます（「Google でサインイン」と同様）。これにより、外部サービスのユーザーは Logto のユーザーシステムを使って認証 (Authentication) できます。

ファーストパーティアプリケーションとは異なり、サードパーティアプリは限定された権限のみリクエストできます。認証 (Authentication) フロー中、ユーザーは **同意画面** で、サードパーティアプリがリクエストするデータや権限を確認・承認してからアクセスを許可します。

Logto でサードパーティアプリケーションを作成することで、アイデンティティシステムと外部サービス間の安全な接続を確立し、ユーザーがデータ共有をコントロールできるようになります。

## Management API で Logto 機能を拡張 \{#extend-logto-features-with-management-api}

Logto [Management API](https://openapi.logto.io/) を利用すると、アプリケーション内で Logto の機能をプログラムで制御・自動化できます。

Logto の機能を拡張してプロダクトを強化したい場合（例：[マルチテナント組織の管理](/organizations) や Logto を使ったワークフローの自動化など）、Logto Management API を活用してプロダクトニーズに合わせた高度な機能を構築できます。
