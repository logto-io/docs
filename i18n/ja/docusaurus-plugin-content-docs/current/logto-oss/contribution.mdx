# 貢献

Logto への貢献に興味を持っていただき、ありがとうございます！私たちは、プロジェクトをより堅牢でユーザーフレンドリーかつ多用途にするために、コミュニティからの貢献を大いに歓迎しています。バグの修正、ドキュメントの改善、コネクターの作成、新機能の追加、フィードバックの提供など、あなたの努力は非常に貴重です。

## どのように貢献できますか？

貢献をできるだけスムーズにするよう努めています。始める前に、[GitHub 上の貢献ガイド](https://github.com/logto-io/logto/blob/master/.github/CONTRIBUTING.md) をご覧ください。このガイドには、私たちのプロセス、コーディング標準、および Logto への貢献を成功させるための手順が記載されています。

## 助けが必要ですか？

質問がある場合やガイダンスが必要な場合は、[GitHub discussions](https://github.com/logto-io/logto/discussions)、[issue を開く](https://github.com/logto-io/logto/issues)、または [お問い合わせ](https://logto.io/contact) を通じてお気軽にご連絡ください。成功するために必要なすべての情報を提供します。
