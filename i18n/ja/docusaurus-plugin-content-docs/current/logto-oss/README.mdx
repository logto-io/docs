# Logto OSS

Logto はオープンソース (OSS) バージョンと [クラウド](/logto-cloud) バージョンの両方を提供しています。このセクションでは、OSS ソリューションに焦点を当てます：<Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS は、開発者に完全なカスタマイズ機能とデータに対する完全なコントロールを提供する無料のアイデンティティソリューションです。Logto OSS は、特定のニーズに合わせてコードベースを適応させる柔軟性を提供し、データ主権を重視し、アプリケーションと共に成長できるカスタマイズされたアイデンティティプロバイダーソリューションを求める人々に最適です。

## Logto OSS がサポートする機能 \{#feature-supported-by-logto-oss}

Logto OSS は、Logto サービスのほとんどのコア機能をサポートしており、定期的に更新および保守されています。

一部の高度な機能は現在、Logto Cloud バージョンにのみ限定されています。これには、[Logto Protected App](/integrate-logto/protected-app/)、[無料のメールサービス](/connectors/email-connectors/built-in-email-service)、[UI の持ち込み](/customization/bring-your-ui)、[IdP 主導の SSO](/end-user-flows/enterprise-sso/idp-initiated-sso)、および SAML IdP が含まれます。
