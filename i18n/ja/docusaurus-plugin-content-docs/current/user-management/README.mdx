# ユーザー管理

ユーザー管理は Logto にとって重要な注力分野です。ユーザーデータへのアクセスや管理は、Logto Console または Management API のいずれかを使用して行うことができ、どちらも効果的です。これらのツールを使って、次のようなタスクを実行できます。

- ユーザー名、ID、氏名、電話番号、メールアドレスでユーザーを検索。
- ソーシャルアカウントの連携解除、プロフィール情報の更新、ユーザーの削除、パスワードのリセットなど、ユーザーに対してさまざまな操作を実行。
- ユーザーログの確認。
- アクセス制御のためのロール (Role) の割り当て。
- パーソナルアクセストークンの作成と管理。
- ユーザーセッションの管理。
