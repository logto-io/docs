# ユーザー管理

ユーザー管理は、Logto の重要な注力領域です。Logto Console または Management API を使用して、ユーザーデータにアクセスし管理することができます。これらのツールを使用して、次のようなタスクを実行できます：

- ユーザー名、ID、フルネーム、電話番号、またはメールアドレスでユーザーを検索する。
- ソーシャルアカウントのリンク解除、プロフィール情報の更新、ユーザーの削除、パスワードのリセットなど、ユーザーに対していくつかのアクションを実行する。
- ユーザーログを確認する。
- アクセス制御のためのロールを割り当てる。
- 個人用アクセス トークンを作成および管理する。
