# Passport.js アプリケーションへ認証機能の追加

このガイドでは、 [Passport.js](https://www.passportjs.org/) と OIDC ストラテジーを使って Logto をアプリケーションに統合する方法を紹介します。

<GuideTip />

## 前提条件 \{#prerequisites}

- [Logto Cloud](https://cloud.logto.io) アカウント、または [セルフホスト型 Logto](/introduction/set-up-logto-oss)。
- 作成済みの Logto 従来型アプリケーション。
- セッションが設定された express プロジェクト。 [Express.js 公式サイト](https://expressjs.com/) をご覧ください。
