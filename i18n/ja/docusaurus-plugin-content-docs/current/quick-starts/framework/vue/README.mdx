### アクセス トークンをリクエストヘッダーに添付

トークンを HTTP ヘッダーの `Authorization` フィールドに Bearer 形式 (`Bearer YOUR_TOKEN`) で配置すれば準備完了です。

:::note
Bearer トークンの統合フローは、使用しているフレームワークやリクエスターによって異なる場合があります。リクエスト `Authorization` ヘッダーを適用する方法を選択してください。
:::
