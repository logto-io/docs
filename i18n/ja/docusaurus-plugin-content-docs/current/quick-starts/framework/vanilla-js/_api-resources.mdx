### アクセス トークンをリクエストヘッダーに添付する

トークンを HTTP ヘッダーの `Authorization` フィールドに Bearer 形式 (`Bearer YOUR_TOKEN`) で入れれば準備完了です。

:::note
Bearer トークンの統合フローは、使用しているフレームワークやリクエスターによって異なる場合があります。リクエスト `Authorization` ヘッダーを適用する独自の方法を選択してください。
:::
