# Auth.js (Next Auth) アプリケーションへ認証機能の追加

このガイドでは、[Auth.js](https://authjs.dev/)（以前の Next Auth）を使って Logto を Next.js アプリケーションに統合する方法を紹介します。

<GuideTip />

## 前提条件 \{#prerequisites}

- [Logto Cloud](https://cloud.logto.io) アカウント、または [セルフホスト型 Logto](/introduction/set-up-logto-oss)。
- 作成済みの Logto 従来型アプリケーション。
- Auth.js を導入した Next.js プロジェクト。詳細は [Auth.js ドキュメント](https://authjs.dev/getting-started/installation) を参照してください。
