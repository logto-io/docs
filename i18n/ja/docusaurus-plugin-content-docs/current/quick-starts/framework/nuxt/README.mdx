## ミドルウェアや API ルートでの利用 \{#use-in-middleware-or-api-routes}

`useLogtoClient()` および `useLogtoUser()` のコンポーザブルは、ミドルウェアや API ルートでは利用できません。`logtoEventHandler()` 関数を使って Logto クライアントやその他のコンテキストを取得できます：

```ts title="/server/api/auth/get-access-token.ts"
import { logtoEventHandler } from '#logto';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
