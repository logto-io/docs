# アカウント設定

Logto は、ユーザーが Logto に保存されたアカウントとプロファイルを管理できるようにするための 2 つのアカウント設定 API コレクションを提供しています。

## ユーザー管理 API

Logto Management API は、ユーザーとそのプロファイルデータを管理するための最も包括的な機能を提供します。これらの API を使用して、セルフホスト型のユーザー管理システムやエンドユーザーのアカウント設定ページを実装することができます。

## アカウント API

Management API とは別に、Logto はエンドユーザーが直接アカウント設定とプロファイルデータを管理できるようにする一連のエンドユーザーアカウント API も提供しています。Management API とは異なり、アカウント API は詳細なユーザー権限制御を伴うエンドユーザーコンテキストで使用するように設計されています。クライアントサイドアプリケーションでアカウント API を直接使用して、ユーザーフレンドリーなアカウント設定ページを構築できます。
