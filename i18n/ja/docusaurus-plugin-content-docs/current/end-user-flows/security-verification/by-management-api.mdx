# Management API によるセキュリティ認証の実装

## パスワード認証 (Password verification) \{#password-verification}

ユーザーがパスワードを変更しようとする際、システムは本人確認のために現在のパスワードの再入力を求める必要があります。このステップにより、認可 (Authorization) されたアカウント所有者のみがパスワードを変更でき、不正なアカウントアクセスを防止できます。

| method | path                                                                                                     | description                                                            |
| ------ | -------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| POST   | [/api/users/\{userId\}/password/verify](https://openapi.logto.io/operation/operation-verifyuserpassword) | ユーザー ID で現在のユーザーパスワードを認証 (Authentication) します。 |
