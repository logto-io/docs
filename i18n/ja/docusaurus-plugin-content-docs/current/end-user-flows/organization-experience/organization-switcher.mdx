# 組織スイッチャー

組織体験 (Experience) では、ユーザーが複数の組織間を切り替えることができます。切り替えるたびに、アプリは新しい組織トークン (Organization token) を取得し、関連情報を読み込みます。

## 実装方法 \{#how-to-implement-it}

1. ドロップダウンなどの UI コンポーネントを作成し、ユーザーが利用可能なすべての組織を一覧表示します。
2. ユーザーが別の組織を選択した際、その ID を取得し、SDK の `getOrganizationToken(organizationId)` メソッドを呼び出して対応する組織トークン (Organization token) を取得します。（SDK がトークンのキャッシュを自動で処理します。）
3. 選択した組織 ID を（URL やクライアントストレージに）保存し、常にその ID を使って組織固有リソース用の組織トークン (Organization token) を取得します。
