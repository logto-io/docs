# 組織スイッチャー

組織体験 (Experience) では、ユーザーは複数の組織 (Organization) を切り替えることができます。切り替えるたびに、アプリは新しい組織トークン (Organization token) を取得して関連情報を読み込みます。

<img src="/img/assets/organization-switcher.png" alt="組織スイッチャー" />

## 実装方法 \{#how-to-implement-it}

1. ユーザーが利用可能なすべての組織 (Organization) を一覧表示するドロップダウンや類似の UI コンポーネントを作成します。
2. ユーザーが別の組織 (Organization) を選択した際、その ID を取得し、SDK の `getOrganizationToken(organizationId)` メソッドを呼び出して対応する組織トークン (Organization token) を取得します。（トークンのキャッシュは SDK が自動で処理します。）
3. 選択した組織 (Organization) の ID を（URL やクライアントストレージに）保存し、組織固有リソースのために常に対応する組織トークン (Organization token) を取得する際に使用します。
