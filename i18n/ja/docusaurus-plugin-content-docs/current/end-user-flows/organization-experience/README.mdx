# 組織体験 (Organization experience)

[組織 (Organization)](/organizations) 体験とは、ビジネスクライアントやその従業員が利用するユーザーインターフェースやフローのことを指します。特に [マルチテナントアプリケーション](https://auth.wiki/multi-tenancy) において重要です。Logto は、Logto Management API を利用してこの体験をアプリに素早く統合するためのガイドをいくつか提供しています。
