# エンドユーザーフロー

エンドユーザーフローは、ユーザーインタラクションのすべての検証プロセスをカバーし、次のように分類されます：

- **認証 (Authentication) フロー**：ユーザーのサインイン、登録、パスワードリセットを含みます。Logto は、<CloudLink to="/sign-in-experience">Console > サインイン体験</CloudLink> で設定された即時利用可能な体験を提供し、[Experience APIs](https://openapi.logto.io/group/endpoint-experience) を介して [独自の UI を導入する](/customization/bring-your-ui) ことができます。
- **アカウントフロー**：アカウント設定、ユーザープロファイル、およびセキュリティ検証を含みます。[Account APIs](https://openapi.logto.io/group/endpoint-account-center) または [Management APIs](https://openapi.logto.io/operation/operation-getuser) を使用して実装します。
- **組織フロー**：B2B や SaaS サービスのようなマルチテナンシー製品向けです。組織の作成、メンバーの招待、および管理を含みます。[Management APIs](https://openapi.logto.io/operation/operation-listorganizations) を使用してアプリに統合します。
