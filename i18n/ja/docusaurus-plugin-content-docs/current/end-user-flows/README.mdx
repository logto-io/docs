# エンドユーザーフロー

エンドユーザーフローは、ユーザーインタラクションに関するすべての認証 (Authentication) プロセスをカバーしており、次のように分類されます：

- **認証 (Authentication) フロー**：ユーザーのサインイン、登録、パスワードリセットを含みます。Logto では <CloudLink to="/sign-in-experience">コンソール > サインイン体験</CloudLink> で設定できる標準の体験を提供しており、[独自 UI の導入](/customization/bring-your-ui) や [Experience API](https://openapi.logto.io/group/endpoint-experience) を利用したカスタマイズも可能です。
- **アカウントフロー**：アカウント設定、ユーザープロファイル、セキュリティ認証 (Authentication) を含みます。[Account API](https://openapi.logto.io/group/endpoint-account-center) または [Management API](https://openapi.logto.io/operation/operation-getuser) を利用して実装します。
- **組織フロー**：B2B や SaaS サービスなどのマルチテナンシー製品向け。組織の作成、メンバー招待、管理などを含みます。[Management API](https://openapi.logto.io/operation/operation-listorganizations) を利用してアプリに統合できます。
