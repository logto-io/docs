# エンドユーザーフロー

エンドユーザーフローは、ユーザーインタラクションに関するすべての認証プロセスをカバーしており、次のように分類されます：

- **認証フロー (Authentication flows)**：ユーザーのサインイン、登録、パスワードリセットを含みます。Logto では <CloudLink to="/sign-in-experience">コンソール > サインイン体験</CloudLink> で設定できる標準の体験を提供しており、[Experience API](https://openapi.logto.io/group/endpoint-experience) を利用して [独自の UI を持ち込む](/customization/bring-your-ui) ことも可能です。
- **アカウントフロー (Account flows)**：アカウント設定、ユーザープロフィール、セキュリティ認証を含みます。[Account API](https://openapi.logto.io/group/endpoint-account-center) または [Management API](https://openapi.logto.io/operation/operation-getuser) を利用して実装します。
- **組織フロー (Organization flows)**：B2B や SaaS サービスなどのマルチテナンシー製品向けです。組織の作成、メンバー招待、管理を含みます。[Management API](https://openapi.logto.io/operation/operation-listorganizations) を利用してアプリに統合できます。
