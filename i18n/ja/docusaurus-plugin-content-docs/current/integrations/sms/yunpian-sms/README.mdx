# YunPian SMS コネクター

YunPian SMS サービス向けの公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

YunPian は、SMS を含むさまざまなサービスを提供する通信サービスプロバイダーです。YunPian SMS コネクターは、Logto チームが提供する YunPian の SMS サービスと連携するためのプラグインであり、Logto のエンドユーザーが SMS 認証コードで登録・サインインできるようにします。

## YunPian コンソールで SMS サービスを設定する \{#set-up-sms-service-in-yunpian-console}

### YunPian アカウントを作成する \{#create-a-yunpian-account}

[ YunPian のウェブサイト ](https://www.yunpian.com/) にアクセスし、アカウント登録と実名認証を完了してください。

### API KEY を取得する \{#get-api-key}

1. YunPian コンソールにログイン
2. 「アカウント設定」→「サブアカウント管理」へ進む
3. API KEY を探してコピーする

### SMS テンプレートを設定する \{#configure-sms-templates}

1. YunPian コンソールで「国内 SMS」→「署名申請」へ進む
2. 署名を作成・申請し、キャリアの承認を待つ
3. 「国内 SMS」→「テンプレート申請」に進み、「認証コード」を選択
4. 認証コード用テンプレートを作成し、`#code#` 変数を含める（「共通テンプレート」を利用すると承認が早くなります）
5. テンプレートの承認を待つ
6. 国際 SMS を送信する場合は、上記手順を「国際 SMS」→「テンプレート申請」で繰り返してください

## Logto での設定 \{#configure-in-logto}

1. Logto コンソールで「コネクター」へ進む
2. 「YunPian SMS サービス」を探してクリック
3. 設定フォームに入力：
   - API KEY：YunPian で取得した API KEY
   - SMS テンプレート：用途に応じてテンプレートを設定し、YunPian で承認されたテンプレートと完全に一致していることを確認してください

## 注意事項 \{#notes}

1. SMS テンプレートの内容は YunPian で承認されたテンプレートと完全に一致している必要があります
2. YunPian テンプレートでの認証コード変数プレースホルダーは `#code#` ですが、コネクター設定では `{{code}}` です
3. YunPian は API KEY に基づき自動的にデフォルト署名を追加するため、テンプレート内容に署名を含める必要はありません
4. 本番利用前に設定のテストを推奨します
