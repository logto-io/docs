# GatewayAPI SMS を使用して SMS 認証を設定する

GatewayAPI SMS の公式 Logto コネクター。

<GuideTip />

## はじめに \{#get-started}

GatewayAPI はヨーロッパのクラウドサービスプロバイダーで、SMS（ショートメッセージサービス）を含む多くのクラウドサービスを提供しています。GatewayAPI SMS コネクターは、Logto チームによって提供されるプラグインで、GatewayAPI SMS サービスを呼び出すために使用されます。これにより、Logto のエンドユーザーは SMS 認証コードを使用して Logto アカウントに登録およびサインインできます。

## GatewayAPI での設定 \{#set-up-in-gatewayapi}

### GatewayAPI アカウントを作成する \{#create-an-gatewayapi-account}

[GatewayAPI のウェブサイト](https://www.gatewayapi.com/)にアクセスし、GatewayAPI アカウントをお持ちでない場合は登録してください。

### アカウントを有効にする \{#enable-account}

SMS サービスを使用する前にアカウントを有効にする必要があるかもしれません。アカウントを有効にするには、GatewayAPI のカスタマーサービスに連絡してください。

### API トークンを取得する \{#get-api-token}

GatewayAPI コンソールの API キーページに移動し、API トークンを見つけるか、新しい API トークンを作成してください。

## Logto での設定 \{#set-up-in-logto}

1. **エンドポイント**: GatewayAPI アカウントが EU 地域にある場合は、エンドポイント https://gatewayapi.com/rest/mtsms を使用する必要があります。GatewayAPI アカウントが US 地域にある場合は、エンドポイント https://gatewayapi.com/rest/mtsms を使用する必要があります。
2. **API トークン**: 前のステップで作成した API トークン。
3. **送信者**: SMS を送信するために使用したい送信者。
4. **テンプレート**: SMS を送信するために使用したいテンプレート。デフォルトのテンプレートを使用するか、必要に応じて変更できます。
