# GatewayAPI SMS を使用してSMS認証を設定する

GatewayAPI SMS 用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

GatewayAPI はヨーロッパのクラウドサービスプロバイダーであり、SMS（ショートメッセージサービス）を含む多くのクラウドサービスを提供しています。GatewayAPI SMS コネクターは、Logto チームが提供するプラグインで、GatewayAPI SMS サービスを呼び出すことができます。これにより、Logto のエンドユーザーは SMS 認証コードを使って Logto アカウントの登録やサインインが可能になります。
