### GatewayAPI アカウントの作成 \{#create-an-gatewayapi-account}

[GatewayAPI のウェブサイト](https://www.gatewayapi.com/) にアクセスし、GatewayAPI アカウントをお持ちでない場合は登録してください。

### アカウントの有効化 \{#enable-account}

SMS サービスを使用する前に、アカウントを有効化する必要があるかもしれません。GatewayAPI のカスタマーサービスに連絡してアカウントを有効化してください。

### API トークンの取得 \{#get-api-token}

GatewayAPI コンソールの API Keys ページに移動し、API トークンを見つけるか、新しい API トークンを作成してください。

## Logto での設定 \{#set-up-in-logto}

1. **エンドポイント**: GatewayAPI アカウントが EU リージョンにある場合は、エンドポイント https://gatewayapi.com/rest/mtsms を使用してください。US リージョンにある場合も同様に、エンドポイント https://gatewayapi.com/rest/mtsms を使用してください。
2. **API トークン**: 前のステップで作成した API トークン。
3. **送信者**: SMS を送信するために使用したい送信者。
4. **テンプレート**: SMS を送信するために使用したいテンプレート。デフォルトのテンプレートを使用するか、必要に応じて変更できます。
