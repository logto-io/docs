### GatewayAPI アカウントの作成 \{#create-an-gatewayapi-account}

[ GatewayAPI のウェブサイト ](https://www.gatewayapi.com/) にアクセスし、まだアカウントをお持ちでない場合は GatewayAPI アカウントを登録してください。

### アカウントの有効化 \{#enable-account}

SMS サービスを利用する前にアカウントを有効化する必要がある場合があります。GatewayAPI のカスタマーサービスに連絡してアカウントを有効化してください。

### API トークンの取得 \{#get-api-token}

GatewayAPI コンソールの API Keys ページに移動し、API トークンを探すか、新しい API トークンを作成してください。

## Logto での設定 \{#set-up-in-logto}

1. **エンドポイント**：GatewayAPI アカウントが EU リージョンの場合は https://gatewayapi.com/rest/mtsms エンドポイントを使用してください。US リージョンの場合も https://gatewayapi.com/rest/mtsms エンドポイントを使用してください。
2. **API トークン**：前のステップで作成した API トークン。
3. **送信者**：SMS を送信する際に使用したい送信者。
4. **テンプレート**：SMS 送信に使用するテンプレート。デフォルトのテンプレートを使用するか、必要に応じて変更できます。
