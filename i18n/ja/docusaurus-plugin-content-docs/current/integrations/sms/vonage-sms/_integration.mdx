## Vonage での設定 \{#set-up-in-vonage}

:::tip
すでに完了している場合は、このステップをスキップできます。
:::

このコネクターを利用するには、Vonage で [アカウント登録](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) を行う必要があります。これにより、API キーとシークレットが発行され、このコネクターを通じて API にアクセスできるようになります。

アカウント作成後、Vonage API ダッシュボードの上部で API キーと API シークレットを確認できます。

また、SMS メッセージを送信するには [バーチャル番号をレンタル](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) する必要がある場合があります。

詳細は [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview) をご覧ください。

## Logto での設定 \{#set-up-in-logto}

1. **API Key**：Vonage の API キー。
2. **API Secret**：Vonage の API シークレット。
3. **Brand Name**：SMS 送信時に使用するブランド名（`from` フィールドとも呼ばれます）。詳細は [送信者 ID (Sender Identity)](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) を参照してください。
4. **Templates**：SMS 送信時に使用するテンプレート。デフォルトテンプレートを利用するか、必要に応じて編集できます。
