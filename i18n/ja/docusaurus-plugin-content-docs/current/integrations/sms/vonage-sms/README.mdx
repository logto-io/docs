# Vonage SMS を使用してSMS認証を設定する

Vonage SMS 用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Vonage はグローバルなコミュニケーションプロバイダーであり、SMS（ショートメッセージサービス）を含む堅牢なクラウドベースのコミュニケーションサービスを提供しています。Vonage SMS コネクターは、Logto チームが提供するプラグインで、Logto のエンドユーザーが SMS 認証コードを使って Logto アカウントに登録・サインインできるようにします。
