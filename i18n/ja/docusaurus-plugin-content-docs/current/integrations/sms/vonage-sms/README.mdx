# Vonage SMS を使用して SMS 認証を設定する

Vonage SMS の公式 Logto コネクター。

<GuideTip />

## はじめに

Vonage は、SMS（ショートメッセージサービス）を含む強力なクラウドベースの通信サービスを提供するグローバルな通信プロバイダーです。Vonage SMS コネクターは、Logto チームによって提供されるプラグインで、Logto のエンドユーザーが SMS 認証コードを使用して Logto アカウントに登録およびサインインできるようにします。

## Vonage での設定

:::tip
すでに完了している場合は、このステップをスキップできます。
:::

このコネクターを使用するには、Vonage で [アカウントを登録](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) する必要があります。これにより、このコネクターを通じて API にアクセスするための API キーとシークレットが取得できます。

アカウントを取得したら、Vonage API ダッシュボードの上部で API キーと API シークレットを見つけることができます。

SMS メッセージを送信するために [仮想番号を借りる](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) 必要があるかもしれません。

詳細については、[Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview) を参照してください。

## Logto での設定

1. **API キー**: あなたの Vonage API キー。
2. **API シークレット**: あなたの Vonage API シークレット。
3. **ブランド名**: SMS を送信するために使用したいブランド名。これは `from` フィールドとも呼ばれます。詳細は [送信者 ID](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) を参照してください。
4. **テンプレート**: SMS を送信するために使用したいテンプレート。デフォルトのテンプレートを使用するか、必要に応じて変更できます。
