## はじめに \{#get-started}

SMTP（Simple Mail Transfer Protocol）は、電子メール送信のためのインターネット標準通信プロトコルです。メールサーバーやその他のメッセージ転送エージェントは、SMTP を使用してメッセージを送受信します。
