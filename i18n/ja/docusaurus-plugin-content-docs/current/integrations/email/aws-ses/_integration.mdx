## AWS サービスコンソールでメールサービスを設定する \{#configure-a-mail-service-in-the-aws-service-console}

### AWS アカウントの登録 \{#register-aws-account}

[AWS](https://aws.amazon.com/) にアクセスし、アカウントを登録します。

### アイデンティティの作成 \{#create-a-identity}

- `Amazon Simple Email Service` コンソールにアクセスします。
- アイデンティティを作成し、以下のいずれかを選択します。
  - ドメインを作成
  - メールアドレスを作成

### コネクターの設定 \{#configuration-of-the-connector}

1. Amazon コンソールの右上にあるユーザー名をクリックして `Security Credentials` に入ります。まだ作成していない場合は `AccessKey` を作成し、大切に保管してください。
2. `Amazon Simple Email Service` コネクターの設定を完了します：
   - ステップ 1 で取得した `AccessKey ID` と `AccessKey Secret` をそれぞれ `accessKeyId` と `accessKeySecret` に入力します。
   - `region`：メール送信に使用するアイデンティティのリージョンを `region` フィールドに入力します。
   - `emailAddress`：メール送信に使用するメールアドレス。形式は `Logto\<noreply@logto.io>` または `\<noreply@logto.io>` です。

以下のパラメーターはオプションです。パラメーターの説明は [AWS SES API ドキュメント](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html) を参照してください。

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### Amazon SES コネクターのテスト \{#test-the-amazon-ses-connector}

メールアドレスを入力し、「送信」をクリックして設定が機能するか「保存して完了」前に確認できます。

以上です。<MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">サインアップ / サインインでコネクターを有効化</MainSiteUrl>するのをお忘れなく。

### 設定タイプ \{#configure-types}

| Name                                      | Type              |
| ----------------------------------------- | ----------------- |
| accessKeyId                               | string            |
| accessKeySecret                           | string            |
| region                                    | string            |
| emailAddress                              | string (OPTIONAL) |
| emailAddressIdentityArn                   | string (OPTIONAL) |
| feedbackForwardingEmailAddress            | string (OPTIONAL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPTIONAL) |
| configurationSetName                      | string (OPTIONAL) |
| templates                                 | Template[]        |

| Template Properties | Type        | Enum values                                          |
| ------------------- | ----------- | ---------------------------------------------------- |
| subject             | string      | N/A                                                  |
| content             | string      | N/A                                                  |
| usageType           | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
