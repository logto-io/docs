## Aliyun DirectMail コンソールでメールサービスを設定する \{#set-up-an-email-service-in-aliyun-directmail-console}

### Aliyun アカウントを作成する \{#create-an-aliyun-account}

[Aliyun](https://aliyun.com/) にアクセスし、アカウントを持っていない場合は Aliyun アカウントを作成してください。

### Aliyun Direct Mail を有効化し設定する \{#enable-and-configure-aliyun-direct-mail}

[DM サービスコンソールページ](https://www.aliyun.com/product/directmail) にアクセスしてサインインします。ページの左上にある「Apply to enable」(申请开通) ボタンをクリックして Direct Mail サービスを有効化し、設定プロセスを開始します。

[DM 管理コンソールページ](https://dm.console.aliyun.com/) から始めて、次のことを行います：

1. サイドバーから「Email Domains」(発信域名) に移動し、指示に従って「New Domain」(新建域名) を追加します。
2. 「Sender Addresses」(発信アドレス) と「Email Tags」(メールタグ) をそれぞれカスタマイズします。

設定が完了したら、テストする方法は 2 つあります：

- [DirectMail 概要ページ](https://dm.console.aliyun.com/) に移動し、ページの下部にある「Operation Guide」(操作引导) を見つけ、「Send Emails」(发送邮件) をクリックします。さまざまなテスト方法が見つかります。
- サイドバーの「Send Emails」(发送邮件) -> 「Email Tasks」(发送邮件) のパスに従ってテストタスクを作成します。

## コネクター JSON を作成する \{#compose-the-connector-json}

1. [DM 管理コンソールページ](https://dm.console.aliyun.com/) から、右上のアバターにカーソルを合わせて「AccessKey Management」(AccessKey 管理) に移動し、「Create AccessKey」(创建 AccessKey) をクリックします。セキュリティ検証を完了すると、「AccessKey ID」と「AccessKey Secret」のペアが取得できます。これらを適切に保管してください。
2. [DM 管理コンソールページ](https://dm.console.aliyun.com/) から訪れた「Sender Addresses」(発信アドレス) または「Email Tags」(メールタグ) タブに移動し、_Sender Address_ または _Email Tag_ を簡単に見つけることができます。
3. Aliyun DM コネクター設定を入力します：
   - `accessKeyId` と `accessKeySecret` フィールドに、ステップ 1 で取得したアクセスキーのペアを入力します。
   - `accountName` と `fromAlias` フィールドに、ステップ 2 で見つけた「Sender Address」と「Email Tag」を入力します。すべてのテンプレートはこの署名名を共有します。（`fromAlias` はオプションなので空白のままにしておくこともできます。）
   - 異なるケースに対して複数の DM コネクターテンプレートを追加できます。単一のテンプレートを追加する例を以下に示します：
     - 送信メールのタイトルとして機能する `subject` フィールドを入力します。
     - 任意の文字列型コンテンツで `content` フィールドを入力します。ランダムな認証コードのために `{{code}}` プレースホルダーを忘れずに残してください。
     - `usageType` フィールドに、異なるユースケースに応じて `Register`、`SignIn`、`ForgotPassword`、または `Generic` のいずれかを入力します。（`usageType` は適切なユースケースを識別するための Logto プロパティです。）完全なユーザーフローを有効にするためには、`Register`、`SignIn`、`ForgotPassword`、および `Generic` の usageType を持つテンプレートが必要です。

### Aliyun DM コネクターをテストする \{#test-aliyun-dm-connector}

メールアドレスを入力し、「Send」をクリックして、設定が機能するかどうかを「Save and Done」する前に確認できます。

これで完了です。[サインイン体験でコネクターを有効化する](/connectors/email-connectors/#enable-email-sign-up-or-sign-in) ことを忘れないでください。

### 設定タイプ \{#config-types}

| 名前            | タイプ            |
| --------------- | ----------------- |
| accessKeyId     | string            |
| accessKeySecret | string            |
| accountName     | string            |
| fromAlias       | string (OPTIONAL) |
| templates       | Template[]        |

| テンプレートプロパティ | タイプ      | 列挙値                                                  |
| ---------------------- | ----------- | ------------------------------------------------------- |
| subject                | string      | N/A                                                     |
| content                | string      | N/A                                                     |
| usageType              | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
