## Postmark アカウントの登録 \{#register-postmark-account}

[Postmark のウェブサイト](https://postmark.com/)で新しいアカウントを作成します。すでにアカウントをお持ちの場合は、このステップをスキップできます。

## コネクターの設定 \{#configure-your-connector}

Postmark のサーバー設定で見つけた Server Token を `serverToken` フィールドに入力します。

送信者の _From Address_ を `fromEmail` フィールドに入力します。

完全なユーザーフローを有効にするには、`Register`、`SignIn`、`ForgotPassword`、`Generic` の usageType を持つテンプレートが必要です。

以下は、Postmark コネクターテンプレートの JSON の例です。

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Postmark メールコネクターのテスト \{#test-postmark-email-connector}

メールアドレスを入力し、「送信」をクリックして、設定が「保存して完了」する前に機能するかどうかを確認できます。

以上です。[サインイン体験でコネクターを有効にする](/connectors/email-connectors/#enable-email-sign-up-or-sign-in) のを忘れないでください。

## 設定タイプ \{#config-types}

| 名前        | タイプ |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| テンプレートプロパティ | タイプ      | 列挙値                                                  |
| ---------------------- | ----------- | ------------------------------------------------------- |
| usageType              | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias          | string      | N/A                                                     |
