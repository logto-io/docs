# Facebook を使用してソーシャルログインを設定する

Facebook ソーシャルサインインのための公式 Logto コネクター。

## はじめに \{#get-started}

Facebook コネクターは、あなたのアプリケーションが Facebook の OAuth 2.0 認証 (Authentication) システムを使用するための簡潔な方法を提供します。
