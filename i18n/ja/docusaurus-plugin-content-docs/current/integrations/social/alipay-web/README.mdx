# Alipay を使用してソーシャルログインを設定する (Set up social login with Alipay (Web))

Web アプリでの Alipay ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Alipay Web コネクターは、デスクトップ Web アプリケーション向けに設計されています。Alipay の OAuth 2.0 認証 (Authentication) ワークフローを活用し、Alipay の公開ユーザープロフィールを利用して、面倒な登録プロセスを経ずに他のアプリへサインインできるようにします。

<Integration />

## Alipay Web コネクターのテスト \{#test-alipay-web-connector}

これで完了です。[サインイン体験でソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。

Alipay Web コネクターが有効化されたら、Web アプリをビルドして実行し、動作を確認できます。
