# Amazon を使用してソーシャルログインを設定する

Amazon ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Amazon コネクターを利用すると、エンドユーザーは Amazon OAuth 2.0 認証 (Authentication) プロトコルを通じて自身の Amazon アカウントでアプリケーションにサインインできます。

<Integration />

## Amazon コネクターのテスト \{#test-amazon-connector}

これで完了です。Amazon コネクターが利用可能になっているはずです。[サインアップ / サインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。
