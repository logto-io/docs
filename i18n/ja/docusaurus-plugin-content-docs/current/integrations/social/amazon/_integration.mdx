### Amazon アプリのセットアップ \{#setup-a-amazon-app}

[Amazon Developer Portal](https://developer.amazon.com) にアクセスし、アカウントでサインインします。アカウントをお持ちでない場合は、登録できます。

その後、アプリを作成します。

**ステップ 1:** ホームページの「Login with Amazon」ブロックに移動します。

または、[このページ](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html) を訪問することもできます。

ガイドに従って、新しいセキュリティプロファイルを作成します。

セキュリティプロファイルを作成した後、テーブルから Client ID と Client Secret を取得します。

**ステップ 2:** コールバック URL を設定します。

テーブルの右側にある「Manage」ボタンをクリックし、「Web Settings」タブに移動します。

「Allowed redirect URLs」フィールドに次の URL を追加します：

- `${your_logto_origin}/callback/${connector_id}`.

`connector_id` は Logto Admin Console のコネクター詳細ページの上部バーにあります。

### コネクターの設定 \{#configure-your-connector}

Logto コネクター設定で、前のステップで取得した値を次のフィールドに入力します：

- **clientId:** アプリの Client ID。
- **clientSecret:** アプリの Client Secret。

`scope` はスペースで区切られた OIDC スコープのリストです。指定しない場合、デフォルトのスコープは `openid profile` です。

### 設定タイプ \{#config-types}

| 名前         | タイプ |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
