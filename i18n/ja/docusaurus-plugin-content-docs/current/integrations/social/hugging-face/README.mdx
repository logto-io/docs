# Hugging Face を使用してソーシャルログインを設定する

Hugging Face ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Hugging Face コネクターを利用すると、エンドユーザーは Hugging Face OAuth / OpenID Connect フローを通じて自身の Hugging Face アカウントでアプリケーションにサインインできます。

<Integration />

## Hugging Face コネクターのテスト \{#test-hugging-face-connector}

これで完了です。Hugging Face コネクターが利用可能になっているはずです。[サインアップ / サインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。
