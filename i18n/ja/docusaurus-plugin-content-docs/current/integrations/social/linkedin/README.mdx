# LinkedIn を使用してソーシャルログインを設定する

LinkedIn ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

LinkedIn コネクターを利用すると、エンドユーザーは LinkedIn OAuth 2.0 認証プロトコルを通じて自身の LinkedIn アカウントでアプリケーションにサインインできます。

<Integration />

## LinkedIn コネクターのテスト \{#test-linkedin-connector}

これで完了です。LinkedIn コネクターが利用可能になっているはずです。[サインアップ / サインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。
