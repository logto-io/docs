# SAML を使用してソーシャルログインを設定する

SAML プロトコル用の公式 Logto コネクター。

<GuideTip />

## はじめに \{#get-started}

SAML コネクターは、SAML プロトコルをサポートする任意のソーシャルアイデンティティプロバイダー (IdP) への Logto の接続を可能にします。

:::note

SAML コネクターは Logto における特別な種類のコネクターであり、いくつかの SAML ベースのコネクターを追加できます。

:::
