# X を使用してソーシャルログインを設定する

X（旧 Twitter）ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

X (Twitter) コネクターを利用すると、エンドユーザーは X OAuth 2.0 認証プロトコルを通じて自身の X (Twitter) アカウントでアプリケーションにサインインできます。

<Integration />

## X (Twitter) コネクターのテスト \{#test-x-connector}

これで完了です。X (Twitter) コネクターが利用可能になっているはずです。[サインアップおよびサインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。
