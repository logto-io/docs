# Alipay を使用してソーシャルログインを設定する (Set up social login with Alipay (Native))

モバイルデバイスのネイティブアプリ向け Alipay ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Alipay ネイティブコネクターは、モバイルプラットフォーム上の Logto SDK と密接に連携します。Alipay の OAuth 2.0 認証 (Authentication) ワークフローを活用し、Alipay ユーザーが面倒な登録プロセスを経ることなく、公開 Alipay ユーザープロファイルを使って他のアプリにサインインできるようにします。

<Integration />

### Alipay ネイティブコネクターのテスト \{#test-alipay-native-connector}

以上です。[サインイン体験でソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。

Alipay ネイティブコネクターが有効化されたら、アプリをビルドして実行し、動作を確認できます。
