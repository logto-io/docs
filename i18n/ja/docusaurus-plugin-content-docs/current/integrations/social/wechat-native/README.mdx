# WeChat を使用してソーシャルログインを設定する (Set up social login with WeChat (Native))

ネイティブアプリ（iOS および Android）向けの WeChat ソーシャルサインイン用公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

コネクターの概念が分からない場合や、このコネクターをサインイン体験に追加する方法が分からない場合は、 [Logto チュートリアル](/connectors/social-connectors) を参照してください。

ネイティブアプリでは、WeChat のサインイン方法として Web を利用することはできません。WeChat アプリへの遷移が必要であり、また WeChat の SDK を利用する必要があります。

少し難しそうに聞こえるかもしれませんが、ご安心ください。このチュートリアルで簡単に対応できます。

<Integration />

## WeChat ネイティブコネクターのテスト \{#test-wechat-native-connector}

以上です。 [サインイン体験でソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のを忘れないでください。

WeChat ネイティブコネクターが有効化されると、アプリをビルドして実行し、動作を確認できます。

:::caution

WeChat アプリがインストールされていないデバイス向けの対応策は WeChat にはありません。Logto はその場合、サインイン時にこのコネクターを非表示にします（これは [公式開発ガイド](https://developers.weixin.qq.com/doc/oplatform/en/Mobile_App/WeChat_Login/Development_Guide.html) でも推奨されている方法です）。

:::
