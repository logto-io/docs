# Line を使用してソーシャルログインを設定する

Line ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Line コネクターを利用すると、エンドユーザーは Line OAuth 2.0 認証プロトコルを通じて自身の Line アカウントでアプリケーションにサインインできます。

<Integration />

## Line コネクターのテスト \{#test-line-connector}

これで完了です。Line コネクターが利用可能になっているはずです。[サインアップ / サインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。
