# WeChat を使用してソーシャルログインを設定する (Web)

Web アプリでの WeChat ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

コネクターの概念が分からない場合や、このコネクターをサインイン体験に追加する方法が分からない場合は、 [Logto チュートリアル](/connectors/social-connectors) を参照してください。

:::caution

このコネクターは Web アプリ専用です。ネイティブアプリで WeChat でサインインする方法を探している場合は、 [WeChat ネイティブコネクター](/integrations/wechat-native) を参照してください。

:::

## WeChat オープンプラットフォームで Web アプリを作成する \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

すでに完了しているセクションはスキップできます。

:::

### アカウントを作成する \{#create-an-account}

https://open.weixin.qq.com/ を開き、右上の「Sign Up」ボタンをクリックし、サインアッププロセスを完了してください。

### Web アプリを作成する \{#create-a-web-app}

作成したアカウントでサインインします。「Web Application」（网页应用）タブで、大きな緑色の「Create a web app」（创建网页应用）ボタンをクリックします。

<img src="/img/assets/app-tabs.png" alt="アプリタブ" />

申請フォームに必要な情報を入力しましょう。

<img src="/img/assets/create-web-app.png" alt="Web アプリ作成" />

#### 基本情報 \{#basic-info}

ほとんどは直感的に入力できます。フォーム入力後、「Next step」をクリックして進みます。

#### ウェブサイト情報 \{#website-info}

「Authorization callback domain」（授权回调域）には Logto ドメインを入力します。例： `logto.io` 。

#### 審査結果を待つ \{#waiting-for-the-review-result}

ウェブサイト情報の入力が完了したら、「Submit Review」をクリックして続行します。通常、審査は迅速で、1～2日以内に終了します。

審査基準は流動的で、審査担当者は毎回ランダムに割り当てられているようです。最初は却下される場合もありますが、諦めずに現状を説明し、どのように修正すればよいか審査担当者に尋ねてみてください。

### WeChat コネクターを設定する \{#configure-your-wechat-connector}

`clientId` と `clientSecret` フィールドには、OAuth アプリ詳細ページで取得した _Client ID_ と _Client Secret_ を入力します。

`scope` フィールドには 'snsapi_userinfo' または 'snsapi_base' のいずれかを入力します。このフィールドは省略可能なので空欄でも構いません。デフォルト値は 'snsapi_userinfo' です。異なる値の [違い](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) も確認できます。

### WeChat Web コネクターをテストする \{#test-wechat-web-connector}

これで完了です。 [サインアップ / サインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のを忘れずに。

WeChat Web コネクターが有効化されると、アプリで再度サインインして動作を確認できます。
