# WeChat を使用してソーシャルログインを設定する (Set up social login with WeChat (Web))

Web アプリでの WeChat ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

コネクターの概念が分からない場合や、このコネクターをサインイン体験に追加する方法が分からない場合は、 [Logto チュートリアル](/connectors/social-connectors) をご覧ください。

:::caution

このコネクターは Web アプリ専用です。ネイティブアプリで WeChat でサインインする方法を探している場合は、 [WeChat ネイティブコネクター](/integrations/wechat-native) をご覧ください。

:::

<Integration />

### WeChat Web コネクターのテスト \{#test-wechat-web-connector}

以上です。 <MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">サインイン体験でソーシャルコネクターを有効化する</MainSiteUrl> のを忘れないでください。

WeChat Web コネクターが有効になると、アプリに再度サインインして動作を確認できます。
