## WeChat オープンプラットフォームでウェブアプリを作成する \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

すでに完了しているセクションはスキップできます。

:::

### アカウントを作成する \{#create-an-account}

https://open.weixin.qq.com/ を開き、右上の「サインアップ」ボタンをクリックし、サインアッププロセスを完了します。

### ウェブアプリを作成する \{#create-a-web-app}

先ほど作成したアカウントでサインインします。「Web Application」(网页应用) タブで、大きな緑色のボタン「Create a web app」(创建网页应用) をクリックします。

<img src="/img/assets/app-tabs.png" alt="アプリタブ" />

アプリケーションフォームに必要な情報を入力しましょう。

<img src="/img/assets/create-web-app.png" alt="ウェブアプリを作成" />

#### 基本情報 \{#basic-info}

ほとんどは非常に簡単です。フォームを完成させたら、「次のステップ」をクリックして進みます。

#### ウェブサイト情報 \{#website-info}

「Authorization callback domain」(授权回调域) に Logto ドメインを入力します。例：`logto.io`。

#### レビュー結果を待つ \{#waiting-for-the-review-result}

ウェブサイト情報を入力した後、「Submit Review」をクリックして続行します。通常、レビューは迅速に行われ、1〜2日以内に終了します。

レビュー基準は変動するため、各提出時にレビュー担当者がランダムに割り当てられると考えられます。最初は拒否されるかもしれませんが、諦めないでください！現状を説明し、どのように修正すればよいかをレビュー担当者に尋ねてください。

### WeChat コネクターを設定する \{#configure-your-wechat-connector}

OAuth アプリの詳細ページから取得した _Client ID_ と _Client Secret_ を `clientId` と `clientSecret` フィールドに入力します。

`scope` フィールドには 'snsapi_userinfo' または 'snsapi_base' を入力します。このフィールドはオプションであり、空白のままにしておくこともできます。デフォルト値は 'snsapi_userinfo' に設定されています。異なる値の [違い](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) を確認できます。
