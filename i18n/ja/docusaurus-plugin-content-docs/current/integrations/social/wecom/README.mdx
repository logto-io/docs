# WeCom を使用してソーシャルログインを設定する (OAuth 2.0)

WeCom (WXwork と呼ばれることもあります) ソーシャルサインインのためのカスタムコネクター。

<GuideTip />

## はじめに \{#get-started}

管理者アカウントで [WeCom WebUI](https://work.weixin.qq.com/) にサインインするか、WeCom アプリから **Manage the enterprise(管理企業)** をクリックします。

「アプリ管理」タブでページを下にスクロールし、「アプリを作成」をクリックします。

アプリに応じて適切な情報を入力し、作成します。

これで、Agent ID (APPID ではありません) と Secret を取得しました。

### ウェブサイト情報 \{#website-info}

このページで必要な項目を設定します。以下のようになります：

- 許可されたユーザー：_このアプリを閲覧できる人_
- アプリホームページ：_あなたのアプリのホームページ。例：`logto.io/demo-app`_

**重要**
このページには「Developer API(开发者接口)」に関する 3 つの項目があります。

1. Web 認証と JS-SDK;
2. WeCom による認証でログイン;
3. 企業信頼 IP;

WeCom のガイドに従ってそれらを記入してください。

### Corp ID \{#corp-id}

WeChat 開発に精通している場合、Corp ID の使用が APP ID と同じであることに気付くかもしれません。

「My Enterprise(我的企业)」タブページの下部に Corp ID を見つけることができます。**ww\*\*\*\*** のように表示されます。

## コネクターを設定する \{#configure-the-connector}

これで、Agent ID、Secret、および Corp ID が揃いました。

コネクターのフォームを完成させましょう。

`Scope` フィールドはオプションなので空白のままにしておくことができます。あるいは、`snsapi_base` または `snsapi_privateinfo` を入力することもできます。

**保存して完了**
