# KOOK を使用してソーシャルログインを設定する

KOOK ソーシャルサインイン Web アプリ向けの公式 Logto コネクター (Connector) です。

<GuideTip />

## はじめに \{#get-started}

KOOK コネクター (Connector) を利用することで、エンドユーザーは KOOK OAuth 2.0 認証 (Authentication) プロトコルを通じて自身の KOOK アカウントでアプリケーションにサインインできます。
