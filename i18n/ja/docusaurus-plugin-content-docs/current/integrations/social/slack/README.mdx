# Slack を使用してソーシャルログインを設定する

Slack ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Slack コネクターを利用すると、エンドユーザーは Slack OAuth 2.0 認証プロトコルを通じて自身の Slack アカウントでアプリケーションにサインインできます。

<Integration />

## Slack コネクターのテスト \{#test-slack-connector}

これで完了です。Slack コネクターが利用可能になっているはずです。[サインアップ / サインインでソーシャルコネクターを有効化する](/connectors/social-connectors/#enable-social-sign-in) のをお忘れなく。
