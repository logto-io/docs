# Patreon を使用してソーシャルログインを設定する

Patreon ソーシャルサインイン用の公式 Logto コネクターです。

<GuideTip />

## はじめに \{#get-started}

Patreon コネクターを利用すると、エンドユーザーは Patreon OAuth 2.0 認証プロトコルを通じて自身の Patreon アカウントでアプリケーションにサインインできます。
