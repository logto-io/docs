# Logto の紹介

Logto ドキュメントへようこそ！Logto は、モダンなアプリや SaaS 製品向けに設計されたアイデンティティとアクセス管理 (IAM) ソリューションです。アプリケーションに対して、安全でスケーラブル、かつカスタマイズ可能な認証 (Authentication) と認可 (Authorization) システムを提供します。

## 機能別に探る \{#explore-by-features}

Logto は、モダンなアイデンティティニーズに合わせた認証 (Authentication)、認可 (Authorization)、ユーザーおよび組織管理を含む強力な機能群を提供します。強力な CIAM (顧客アイデンティティとアクセス管理) サポートにより、この分野での明確な優位性を持っています。
