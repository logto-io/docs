Chi を使って新しい Go プロジェクトを初期化するには、次の手順に従います：

```bash
go mod init your-api-name
go get github.com/go-chi/chi/v5
```

次に、基本的な Chi サーバーセットアップを作成します：

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
ルート、ミドルウェア、その他の機能のセットアップ方法については、Chi のドキュメントを参照してください。
:::
