Fiber を使用して新しい Go プロジェクトを初期化するには、次の手順を実行します：

```bash
go mod init your-api-name
go get github.com/gofiber/fiber/v2
```

次に、基本的な Fiber サーバーのセットアップを作成します：

```go title="main.go"
package main

import (
    "log"

    "github.com/gofiber/fiber/v2"
)

func main() {
    app := fiber.New()

    log.Fatal(app.Listen(":3000"))
}
```

:::note
ルートやミドルウェア、その他の機能の設定方法については、Fiber のドキュメントを参照してください。
:::
