新しい Rails API プロジェクトを初期化するには、Rails ジェネレーターを使用できます：

```bash
rails new your-api-name --api
cd your-api-name
```

開発サーバーを起動します：

```bash
rails server
```

基本的な API コントローラーを作成します：

```ruby title="app/controllers/api/base_controller.rb"
class Api::BaseController < ApplicationController
  def index
    render json: { message: 'Hello from Rails API' }
  end
end
```

ルートを追加します：

```ruby title="config/routes.rb"
Rails.application.routes.draw do
  namespace :api do
    root 'base#index'
  end
end
```

:::note
コントローラー、モデル、その他の機能のセットアップ方法については、Rails のドキュメントを参照してください。
:::
