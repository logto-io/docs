Hapi を使って新しい Node.js プロジェクトを初期化するには、次の手順を実行します：

```bash
npm init -y
npm install @hapi/hapi
```

TypeScript を使用する場合は、TypeScript の環境も適切にセットアップしてください。

次に、基本的な Hapi サーバーのセットアップを作成します：

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Server running on http://localhost:3000');
```

:::note
ルートやプラグイン、その他の機能のセットアップ方法については、 Hapi のドキュメント を参照してください。
:::
