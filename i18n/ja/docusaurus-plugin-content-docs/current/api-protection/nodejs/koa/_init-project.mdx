Koa を使って新しい Node.js プロジェクトを初期化するには、次の手順に従ってください：

```bash
npm init -y
npm install koa @koa/router
```

TypeScript を使用する場合は、TypeScript の環境も適切にセットアップしてください。

次に、基本的な Koa サーバーのセットアップを作成します：

```ts title="app.ts"
import Koa from 'koa';

const app = new Koa();

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

:::note
ルートやミドルウェア、その他の機能のセットアップ方法については、Koa のドキュメントを参照してください。
:::
