Fastify を使って新しい Node.js プロジェクトを初期化するには、次の手順に従います：

```bash
npm init -y
npm install fastify
```

TypeScript を使用する場合は、TypeScript の環境を適切にセットアップしてください。

次に、基本的な Fastify サーバーのセットアップを作成します：

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Server running on http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
ルートやプラグイン、その他の機能のセットアップ方法については、 Fastify のドキュメント を参照してください。
:::
