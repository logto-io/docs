新しい FastAPI プロジェクトを初期化するには、ディレクトリを作成し、基本的な構成をセットアップします：

```bash
mkdir your-api-name
cd your-api-name
```

requirements ファイルを作成します：

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

依存関係をインストールします：

```bash
pip install -r requirements.txt
```

基本的な FastAPI アプリケーションを作成します：

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

開発サーバーを起動します：

```bash
uvicorn main:app --reload
```

:::note
パス操作、依存性注入、その他の機能のセットアップ方法については、 FastAPI のドキュメント を参照してください。
:::
