新しい Flask プロジェクトを初期化するには、ディレクトリを作成し、基本的な構成をセットアップします：

```bash
mkdir your-api-name
cd your-api-name
```

requirements ファイルを作成します：

```txt title="requirements.txt"
Flask
```

依存関係をインストールします：

```bash
pip install -r requirements.txt
```

基本的な Flask アプリケーションを作成します：

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

開発サーバーを起動します：

```bash
python app.py
```

:::note
ルート、ブループリント、その他の機能のセットアップ方法については、 Flask のドキュメント を参照してください。
:::
