# 開発者

Logto は、[OAuth 2](https://auth.wiki/oauth-2.0) および [OIDC](https://auth.wiki/openid-connect) プロトコルに基づいた [アイデンティティとアクセス管理 (IAM)](https://auth.wiki/iam) サービスです。Logto のような IAM サービスは、他の Web サービスの基盤として機能することが多く、これらの Web サービス内のさまざまな認可 (Authorization) 状態は Logto によって直接影響を受けます。

ユーザーに利便性を提供するために、Logto は一般的に使用される開発者向け機能を一連の形で提供しています。
