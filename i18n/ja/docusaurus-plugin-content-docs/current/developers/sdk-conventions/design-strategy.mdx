# デザイン戦略

- すべてのプログラミング言語は、`${language}` という名前の独立した git リポジトリを持つべきです。
- 各プログラミング言語リポジトリはモノレポであるべきです。
- SDK とその関連するサンプルプロジェクトは、このリポジトリの下に配置されるべきです。

例：

<details>
  <summary>JS</summary>

- js (コア)
- react
- react-sample

</details>

<details>
  <summary>Kotlin</summary>

- kotlin (コア)
- android
- android-sample

</details>
