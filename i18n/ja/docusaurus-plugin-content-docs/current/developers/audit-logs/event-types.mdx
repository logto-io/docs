# 監査ログのイベントタイプ

<CloudLink to="/audit-logs">Logto Console > 監査ログ</CloudLink> でイベントタイプをフィルタリングできます。

:::note

Logto は現在、 [体験 (Experience) API](https://openapi.logto.io/group/endpoint-experience) を通じてエンドユーザーの操作に関連するログの取得をサポートしています。

[Management API](/integrate-logto/interact-with-management-api) および [Account API](https://openapi.logto.io/group/endpoint-account-center) の監査ログは近日公開予定です。

ご要望があれば、ぜひ [お問い合わせ](https://logto.io/contact) ください。

:::

## トークン交換

| Key                               | Name                                         |
| --------------------------------- | -------------------------------------------- |
| ExchangeTokenBy.AuthorizationCode | コードによるトークン交換                     |
| ExchangeTokenBy.ClientCredentials | クライアントクレデンシャルによるトークン交換 |
| ExchangeTokenBy.RefreshToken      | リフレッシュ トークンによるトークン交換      |
| ExchangeTokenBy.TokenExchange     | トークン交換                                 |

## カスタムトークンクレーム (Claims)

| Key                            | Name                                                       |
| ------------------------------ | ---------------------------------------------------------- |
| JwtCustomizer.AccessToken      | カスタムユーザー アクセス トークンのクレーム (Claims) 取得 |
| JwtCustomizer.ClientCredential | カスタム M2M アクセス トークンのクレーム (Claims) 取得     |
