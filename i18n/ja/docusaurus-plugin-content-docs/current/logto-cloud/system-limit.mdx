# システム制限

Logto では、すべてのプランで寛大な制限を設定し、柔軟な従量課金オプションも提供しています。そのため、実際に使用した分だけお支払いいただけます。

料金ページの一部の項目には _無制限_ または _上限なしの継続的な従量課金_ と記載されている場合があります。これは、基本的に制限なく利用できることを意味しますが、Logto はすべてのユーザーの公平な利用を維持するため、実際の制限を随時調整する権利を留保しています。つまり、エンティティ制限はプラットフォーム全体の健全性を守るための厳格な上限です。これらは料金体系の一部ではありませんが、プランごとに異なる場合があります。

利用ケースが合理的であってもシステム制限に達した場合は、遠慮なく [お問い合わせ](https://logto.io/contact) ください。フィードバックを共有いただくことで、実際の利用パターンをより深く理解し、システム制限を調整してロイヤルカスタマーをより良くサポートできるようになります。

## テナント単位のレート制限保護 \{#tenant-level-rate-limit-protection}

### 開発テナント \{#dev-tenants}

[開発テナント](/logto-cloud/tenant-settings#development) では、Logto の無料機能や提供内容を活用できます。悪用防止と明確な期待値設定のため、一定のシステム制限を設けています。これらの制限は、テストや開発目的での無料利用を維持しつつ、プラットフォームを持続的に運用するためのものです。

クォータの増加をご希望の場合は、お問い合わせいただければサポートいたします。また、[**Dev** から **Pro** へのアップグレード](/logto-cloud/billing-and-pricing#dev-tenant-upgrade-to-pro) もおすすめします。これにより上限が解除され、すぐにフルアクセスが可能となります。

### エンタープライズテナント \{#enterprise-tenants}

エンタープライズプランでは、制限や機能はすべてカスタマイズ可能で、契約を通じて管理されます。詳細については [お問い合わせ](https://logto.io/contact) ください。
