# Logto クラウドサービス

[Logto Cloud](https://cloud.logto.io) は、アイデンティティとリソースをスムーズかつ簡単に管理するためのさまざまな管理および運用サービスを提供します。Logto によってホストされ、[マルチリージョンサポート](/logto-cloud/tenant-settings#tenant-region)、テナント管理、[請求と価格設定](/logto-cloud/billing-and-pricing)、[メンバー管理](/logto-cloud/tenant-member-management)、およびコンソールロールベースのアクセス制御などの機能が含まれています。

クラウドサービスに関する質問や追加サポートが必要な場合は、お問い合わせください。
