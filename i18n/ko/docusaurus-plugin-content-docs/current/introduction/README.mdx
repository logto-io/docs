# Logto 소개

Logto 문서에 오신 것을 환영합니다! Logto는 현대적인 앱과 SaaS 제품을 위해 설계된 아이덴티티 및 접근 관리 (IAM) 솔루션입니다. 애플리케이션을 위한 안전하고 확장 가능하며 맞춤화 가능한 인증 (Authentication) 및 인가 (Authorization) 시스템을 제공합니다.
