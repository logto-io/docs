# 써드파티 AI 에이전트가 앱에 접근할 수 있도록 허용하세요

:::caution
👷 이 튜토리얼은 아직 진행 중입니다. 추후 업데이트를 위해 다시 확인해 주세요. 🚧
:::

이 가이드는 Logto를 서비스와 통합하고 AI 에이전트가 서비스에 접근할 수 있도록 설정하는 과정을 안내합니다.

다음 내용을 학습할 수 있습니다:

- 서비스의 인가 (Authorization) 서버로 Logto를 구성하는 방법
- AI 에이전트가 서비스에 접근할 수 있도록 액세스 토큰 (Access token)을 얻는 방법
- AI 에이전트로 플로우를 테스트하는 방법

## 사전 준비 사항 \{#prerequisites}

- [Logto Cloud](https://cloud.logto.io) (또는 셀프 호스팅) 테넌트
- AI 에이전트가 접근할 수 있도록 API 엔드포인트를 제공하는 서비스

### 플로우 이해하기 \{#understanding-the-flow}

- **서비스**: AI 에이전트에게 노출하려는 서비스입니다.
- **AI 에이전트**: 서비스에 접근할 AI 에이전트입니다.
- **Logto**: OpenID Connect 제공자 (인가 (Authorization) 서버) 역할을 하며, 사용자 아이덴티티를 관리합니다.
