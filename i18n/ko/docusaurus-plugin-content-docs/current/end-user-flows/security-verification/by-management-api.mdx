# Management API로 보안 검증 구현하기

## 비밀번호 검증 \{#password-verification}

사용자가 비밀번호를 변경하려고 할 때, 시스템은 현재 비밀번호를 다시 입력하도록 요청하여 본인임을 검증해야 합니다. 이 단계는 인가된 계정 소유자만 비밀번호를 변경할 수 있도록 하여, 계정에 대한 무단 접근을 방지합니다.

| method | path                                                                                                     | description                                                                               |
| ------ | -------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| POST   | [/api/users/\{userId\}/password/verify](https://openapi.logto.io/operation/operation-verifyuserpassword) | 사용자 ID로 현재 사용자 비밀번호를 검증합니다. (Verify current user password by user ID.) |
