# 조직 전환기

조직 경험(Experience)에서는 사용자가 여러 조직 간에 전환할 수 있습니다. 사용자가 전환할 때마다, 앱은 새로운 조직 토큰을 가져와 관련 정보를 로드합니다.

## 구현 방법 \{#how-to-implement-it}

1. 사용자가 접근할 수 있는 모든 조직을 나열하는 드롭다운 또는 유사한 UI 컴포넌트를 만드세요.
2. 사용자가 다른 조직을 선택하면, 해당 조직의 ID를 가져와 SDK의 `getOrganizationToken(organizationId)` 메서드를 호출하여 해당 조직 토큰을 가져옵니다. (SDK가 토큰 캐싱을 자동으로 처리합니다.)
3. 선택한 조직 ID를 (URL 또는 클라이언트 저장소에) 저장하고, 항상 이를 사용하여 조직별 리소스에 대한 조직 토큰을 가져오세요.
