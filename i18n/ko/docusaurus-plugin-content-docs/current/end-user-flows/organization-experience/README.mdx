# 조직 경험 (Organization experience)

[조직](/organizations) 경험은 비즈니스 클라이언트와 그들의 직원들이 사용하는 사용자 인터페이스 및 플로우를 의미하며, 특히 [다중 테넌트 애플리케이션](https://auth.wiki/multi-tenancy)에서 중요합니다. Logto는 Logto Management API를 사용하여 이 경험을 앱에 빠르게 통합할 수 있도록 여러 가이드를 제공합니다.
