# 최종 사용자 플로우

최종 사용자 플로우는 사용자 상호작용에 대한 모든 검증 과정을 다루며, 다음과 같이 분류됩니다:

- **인증 (Authentication) 플로우**: 사용자 로그인, 회원가입, 비밀번호 재설정이 포함됩니다. Logto는 <CloudLink to="/sign-in-experience">콘솔 > 로그인 경험</CloudLink>에서 바로 사용할 수 있는 경험을 제공하며, [Experience API](https://openapi.logto.io/group/endpoint-experience)를 통해 [UI를 직접 구현](/customization/bring-your-ui)할 수도 있습니다.
- **계정 플로우**: 계정 설정, 사용자 프로필, 보안 검증이 포함됩니다. [Account API](https://openapi.logto.io/group/endpoint-account-center) 또는 [Management API](https://openapi.logto.io/operation/operation-getuser)를 사용하여 구현할 수 있습니다.
- **조직 플로우**: B2B 또는 SaaS 서비스와 같은 다중 테넌시 제품을 위한 플로우입니다. 조직 생성, 멤버 초대, 관리가 포함됩니다. [Management API](https://openapi.logto.io/operation/operation-listorganizations)를 사용하여 앱에 통합할 수 있습니다.
