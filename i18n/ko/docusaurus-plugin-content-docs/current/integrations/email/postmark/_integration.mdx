## Postmark 계정 등록 \{#register-postmark-account}

[Postmark 웹사이트](https://postmark.com/)에서 새 계정을 만드세요. 이미 계정이 있는 경우 이 단계를 건너뛸 수 있습니다.

## 커넥터 구성 \{#configure-your-connector}

Postmark에서 서버 설정 아래에서 찾을 수 있는 서버 토큰으로 `serverToken` 필드를 채우세요.

발신자의 *From Address*로 `fromEmail` 필드를 채우세요.

전체 사용자 흐름을 활성화하려면 `Register`, `SignIn`, `ForgotPassword` 및 `Generic` 사용 유형의 템플릿이 필요합니다.

다음은 Postmark 커넥터 템플릿 JSON의 예입니다.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Postmark 이메일 커넥터 테스트 \{#test-postmark-email-connector}

이메일 주소를 입력하고 "Send"를 클릭하여 설정이 "Save and Done" 전에 작동하는지 확인할 수 있습니다.

이제 완료되었습니다. [로그인 경험에서 커넥터 활성화](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)를 잊지 마세요.

## 구성 유형 \{#config-types}

| 이름        | 유형   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| 템플릿 속성   | 유형        | 열거형 값                                               |
| ------------- | ----------- | ------------------------------------------------------- |
| usageType     | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias | string      | N/A                                                     |
