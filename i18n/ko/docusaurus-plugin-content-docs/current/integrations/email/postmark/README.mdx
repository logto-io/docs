# Postmark로 이메일 인증을 설정하세요 (Set up email verification with Postmark email service)

Postmark 이메일 서비스용 Logto 커넥터 (Connector)입니다.

<GuideTip />

## 시작하기 \{#get-started}

Postmark는 트랜잭션 및 마케팅 이메일을 위한 메일 플랫폼입니다. 우리는 이 플랫폼의 이메일 발송 기능을 사용하여 *인증 코드*를 보낼 수 있습니다.

## Postmark 계정 등록하기 \{#register-postmark-account}

[Postmark 웹사이트](https://postmark.com/)에서 새 계정을 만드세요. 이미 계정이 있다면 이 단계를 건너뛸 수 있습니다.

## 커넥터 (Connector) 구성하기 \{#configure-your-connector}

Postmark에서 서버의 설정에서 찾을 수 있는 Server Token을 `serverToken` 필드에 입력하세요.

발신자의 *From Address*를 `fromEmail` 필드에 입력하세요.

전체 사용자 플로우를 활성화하려면, `Register`, `SignIn`, `ForgotPassword`, `Generic`의 usageType을 가진 템플릿이 필요합니다.

다음은 Postmark 커넥터 (Connector) 템플릿 JSON 예시입니다.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Postmark 이메일 커넥터 (Connector) 테스트하기 \{#test-postmark-email-connector}

이메일 주소를 입력하고 "Send"를 클릭하여 설정이 정상적으로 동작하는지 "Save and Done" 전에 확인할 수 있습니다.

이제 완료되었습니다. <MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">회원가입 및 로그인에서 커넥터 (Connector) 활성화</MainSiteUrl>를 잊지 마세요.

## 구성 타입 \{#config-types}

| Name        | Type   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| 템플릿 속성 (Template Properties) | Type        | Enum 값 (Enum values)                                   |
| --------------------------------- | ----------- | ------------------------------------------------------- |
| usageType                         | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias                     | string      | N/A                                                     |
