# Postmark로 이메일 인증을 설정하세요

Postmark 이메일 서비스를 위한 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Postmark는 트랜잭션 및 마케팅 이메일을 위한 메일 플랫폼입니다. 우리는 이 플랫폼의 이메일 발송 기능을 사용하여 *인증 코드*를 보낼 수 있습니다.
