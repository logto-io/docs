## AWS 서비스 콘솔에서 메일 서비스 구성하기 \{#configure-a-mail-service-in-the-aws-service-console}

### AWS 계정 등록하기 \{#register-aws-account}

[AWS](https://aws.amazon.com/)에 접속하여 계정을 등록하세요.

### 아이덴티티 생성하기 \{#create-a-identity}

- `Amazon Simple Email Service` 콘솔로 이동하세요.
- 아이덴티티를 생성하고, 다음 옵션 중 하나를 선택하세요.
  - 도메인 생성
  - 이메일 주소 생성

### 커넥터 설정 \{#configuration-of-the-connector}

1. Amazon 콘솔 우측 상단에서 사용자 이름을 클릭하여 `Security Credentials`로 들어갑니다. 만약 없다면, `AccessKey`를 생성하고 안전하게 보관하세요.
2. `Amazon Simple Email Service` 커넥터의 설정을 완료하세요:
   - 1단계에서 얻은 `AccessKey ID`와 `AccessKey Secret`을 각각 `accessKeyId`와 `accessKeySecret`에 입력하세요.
   - `region`: 메일을 발송하는 데 사용하는 아이덴티티의 리전을 `region` 필드에 입력하세요.
   - `emailAddress`: 메일 발송에 사용하는 이메일 주소로, `Logto<noreply@logto.io>` 또는 `<noreply@logto.io>` 형식입니다.

다음 파라미터들은 선택 사항입니다. 파라미터 설명은 [AWS SES API 문서](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html)에서 확인할 수 있습니다.

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### Amazon SES 커넥터 테스트하기 \{#test-the-amazon-ses-connector}

이메일 주소를 입력하고 "Send"를 클릭하여 "Save and Done" 전에 설정이 정상적으로 동작하는지 확인할 수 있습니다.

이제 완료되었습니다. <MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">회원가입 및 로그인에서 커넥터 활성화</MainSiteUrl>를 잊지 마세요.

### 설정 타입 \{#configure-types}

| 이름                                      | 타입              |
| ----------------------------------------- | ----------------- |
| accessKeyId                               | string            |
| accessKeySecret                           | string            |
| region                                    | string            |
| emailAddress                              | string (OPTIONAL) |
| emailAddressIdentityArn                   | string (OPTIONAL) |
| feedbackForwardingEmailAddress            | string (OPTIONAL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPTIONAL) |
| configurationSetName                      | string (OPTIONAL) |
| templates                                 | Template[]        |

| 템플릿 속성 | 타입        | Enum 값                                              |
| ----------- | ----------- | ---------------------------------------------------- |
| subject     | string      | N/A                                                  |
| content     | string      | N/A                                                  |
| usageType   | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
