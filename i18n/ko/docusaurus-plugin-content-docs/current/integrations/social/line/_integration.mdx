### Line 채널 설정

[Line Developers](https://developers.line.biz/console/)로 이동하여 Line 비즈니스 계정으로 로그인하세요. 계정이 없다면, 등록할 수 있습니다.

그런 다음, [이 링크](https://developers.line.biz/console/register/line-login/provider/)로 이동하여 채널을 생성하세요.

**1단계:** 채널 세부 정보 입력.

양식을 작성하고 채널을 생성하세요.

**2단계:** 콜백 URL 설정.

채널 세부 정보 페이지로 이동하여 "LINE 로그인" 탭을 찾고 "Callback URL" 필드를 편집하세요.

우리의 경우, 이는 `${your_logto_endpoint}/callback/${connector_id}`가 됩니다. 예를 들어, `https://foo.logto.app/callback/${connector_id}`입니다. `connector_id`는 Logto Admin Console 커넥터 세부 정보 페이지의 상단 바에서 찾을 수 있습니다.

### 커넥터 구성

Logto 커넥터 구성에서, 앱의 "Auth" 탭, "Application credentials" 섹션에서 얻은 값으로 다음 필드를 채우세요:

- **clientId:** 앱의 채널 ID.
- **clientSecret:** 앱의 채널 비밀.

`scope`는 OIDC 스코프의 공백으로 구분된 목록입니다. 제공되지 않으면 기본 스코프는 `openid profile`입니다.

### 구성 유형

| 이름         | 유형   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
