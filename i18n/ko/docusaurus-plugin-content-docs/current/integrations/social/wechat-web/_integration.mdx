## WeChat 오픈 플랫폼에서 웹 앱 생성하기 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

이미 완료한 섹션은 건너뛸 수 있습니다.

:::

### 계정 생성하기 \{#create-an-account}

https://open.weixin.qq.com/ 에 접속하여 오른쪽 상단의 "Sign Up" 버튼을 클릭한 후, 회원가입 과정을 완료하세요.

### 웹 앱 생성하기 \{#create-a-web-app}

방금 생성한 계정으로 로그인하세요. "Web Application" (网页应用) 탭에서 큰 녹색 버튼 "Create a web app" (创建网页应用)을 클릭하세요.

<img src="/img/assets/app-tabs.png" alt="앱 탭" />

신청서에 필요한 정보를 입력해 봅시다.

<img src="/img/assets/create-web-app.png" alt="웹 앱 생성" />

#### 기본 정보 \{#basic-info}

대부분은 매우 간단합니다. 양식을 완료한 후, "Next step"을 클릭하여 계속 진행하세요.

#### 웹사이트 정보 \{#website-info}

"Authorization callback domain" (授权回调域)에 Logto 도메인을 입력하세요. 예: `logto.io`.

#### 검토 결과 대기 \{#waiting-for-the-review-result}

웹사이트 정보를 완료한 후, "Submit Review"를 클릭하여 계속 진행하세요. 일반적으로 검토는 빠르게 진행되며, 1-2일 내에 끝납니다.

검토 기준이 유동적이기 때문에 각 제출 시 검토자가 무작위로 배정되는 것으로 보입니다. 처음에는 거절될 수 있지만 포기하지 마세요! 현재 상태를 설명하고 검토자에게 어떻게 수정해야 하는지 물어보세요.

### WeChat 커넥터 구성하기 \{#configure-your-wechat-connector}

OAuth 앱 세부 정보 페이지에서 받은 *Client ID*와 *Client Secret*으로 `clientId`와 `clientSecret` 필드를 채우세요.

`scope` 필드는 'snsapi_userinfo' 또는 'snsapi_base'로 채울 수 있습니다. 이 필드는 선택 사항이므로 비워둘 수 있습니다. 기본값은 'snsapi_userinfo'로 설정되어 있습니다. 서로 다른 값의 [차이점](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)을 확인할 수 있습니다.
