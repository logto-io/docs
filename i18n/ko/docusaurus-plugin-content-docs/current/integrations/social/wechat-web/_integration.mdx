## WeChat 오픈 플랫폼에서 웹 앱 생성하기 \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

이미 완료한 부분이 있다면 일부 섹션은 건너뛸 수 있습니다.

:::

### 계정 생성하기 \{#create-an-account}

https://open.weixin.qq.com/ 에 접속하여 오른쪽 상단의 "Sign Up" 버튼을 클릭한 후, 회원가입 절차를 완료하세요.

### 웹 앱 생성하기 \{#create-a-web-app}

방금 생성한 계정으로 로그인하세요. "Web Application" (网页应用) 탭에서 큰 초록색 버튼인 "웹 앱 생성" (创建网页应用)을 클릭하세요.

<img src="/img/assets/app-tabs.png" alt="앱 탭" />

이제 신청서에 필요한 정보를 입력해봅시다.

<img src="/img/assets/create-web-app.png" alt="웹 앱 생성" />

#### 기본 정보 \{#basic-info}

대부분 매우 직관적입니다. 양식을 모두 작성한 후 "Next step"을 클릭하여 다음 단계로 이동하세요.

#### 웹사이트 정보 \{#website-info}

"Authorization callback domain" (授权回调域)에는 본인의 Logto 도메인을 입력하세요. 예: `logto.io`.

#### 검토 결과 대기 \{#waiting-for-the-review-result}

웹사이트 정보를 모두 입력한 후 "Submit Review"를 클릭하여 계속 진행하세요. 일반적으로 검토는 빠르게 진행되며 1~2일 내에 완료됩니다.

검토 기준이 유동적이기 때문에, 제출할 때마다 검토자가 무작위로 배정되는 것으로 추정됩니다. 처음에 거절될 수도 있지만 포기하지 마세요! 현재 상황을 설명하고, 어떻게 수정해야 하는지 검토자에게 문의하세요.

### WeChat 커넥터 구성하기 \{#configure-your-wechat-connector}

OAuth 앱 상세 페이지에서 받은 *Client ID*와 *Client Secret*을 각각 `clientId`와 `clientSecret` 필드에 입력하세요.

`scope` 필드에는 'snsapi_userinfo' 또는 'snsapi_base' 중 하나를 입력할 수 있습니다. 이 필드는 선택 사항이므로 비워둘 수 있습니다. 기본값은 'snsapi_userinfo'로 설정되어 있습니다. 각 값의 [차이점](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html)을 참고하세요.
