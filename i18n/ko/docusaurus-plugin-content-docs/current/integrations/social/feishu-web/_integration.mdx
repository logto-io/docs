## Feishu 소셜 로그인 시작하기 \{#getting-started-with-feishu-social-sign-in}

Feishu 웹 커넥터는 데스크톱 웹 애플리케이션을 위해 설계되었으며, OAuth 2.0 인증 (Authentication) 프로세스를 사용합니다.

## Feishu 개발자 계정 등록하기 \{#register-a-feishu-developer-account}

Feishu 개발자 계정이 없다면, [Feishu Open Platform](https://open.feishu.cn/app)에서 등록하세요.

## 애플리케이션 생성하기 \{#create-an-application}

1. [Developer Console](https://open.feishu.cn/app)에서 "Create Custom App"을 클릭하세요.
2. 애플리케이션 이름, 설명을 입력하고 아이콘을 선택한 후 "Create" 버튼을 클릭하세요.
3. 왼쪽 사이드바에서 "Security Settings"를 클릭하고, "Redirect URL"에 `${logto_endpoint}/callback/${connector_id}`를 입력하세요. 해당 값은 Logto 콘솔의 Feishu 커넥터 상세 페이지의 `Callback URI` 필드에서 확인할 수 있습니다.
4. "Credentials & Basic Info"에서 "App ID"와 "App Secret"을 얻을 수 있습니다.

:::note
엔터프라이즈 내부용이 아닌 경우, "Version Management and Release" 페이지에서 "Create a version" 버튼을 클릭해야 합니다.
"App ID"와 "App Secret"은 애플리케이션 상태가 "Enabled"로 변경된 후에만 유효합니다.
:::
