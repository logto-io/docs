# Feishu 소셜 로그인 설정하기

Feishu Web: 고급 엔터프라이즈 협업 및 관리 플랫폼으로, 원활한 사무실 협업을 제공하고 팀 목표를 조정하여 조직과 개인을 완전히 활성화합니다.

<GuideTip />

## Feishu 소셜 로그인 시작하기 \{#getting-started-with-feishu-social-sign-in}

Feishu 웹 커넥터는 데스크톱 웹 애플리케이션을 위해 설계되었으며 OAuth 2.0 인증 (Authentication) 프로세스를 사용합니다.

## Feishu 개발자 계정 등록하기 \{#register-a-feishu-developer-account}

Feishu 개발자 계정이 없는 경우, [Feishu Open Platform](https://open.feishu.cn/app)에서 등록하세요.

## 애플리케이션 생성하기 \{#create-an-application}

1. [개발자 콘솔](https://open.feishu.cn/app)에서 "사용자 정의 앱 생성"을 클릭하세요.
2. 애플리케이션 이름, 설명을 입력하고 아이콘을 선택한 후 "생성" 버튼을 클릭하세요.
3. 왼쪽 사이드바에서 "보안 설정"을 클릭하고 "리디렉션 URL"을 `${logto_endpoint}/callback/${connector_id}`로 입력하세요. 해당 값은 Logto 콘솔의 Feishu 커넥터 세부 정보 페이지의 `Callback URI` 필드에서 찾을 수 있습니다.
4. "자격 증명 및 기본 정보"에서 "앱 ID"와 "앱 비밀"을 얻을 수 있습니다.

:::note
비엔터프라이즈 내부 사용의 경우, "버전 관리 및 릴리스" 페이지에서 "버전 생성" 버튼을 클릭해야 합니다.
"앱 ID"와 "앱 비밀"은 애플리케이션 상태가 "활성화됨"으로 변경된 후에만 유효합니다.
:::
