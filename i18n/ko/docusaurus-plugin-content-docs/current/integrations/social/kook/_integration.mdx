## KOOK 개발자 센터에서 애플리케이션 생성하기 \{#create-an-application-in-the-kook-developer-center}

팁: 이미 완료한 섹션이 있다면 일부 단계를 건너뛸 수 있습니다.

:::note
OAuth2 애플리케이션을 생성하기 전에 KOOK OAuth2 권한을 신청해야 합니다. 자세한 내용은 OAuth2 페이지의 정보를 확인하거나 KOOK 공식 개발자 서버에서 KOOK 담당자에게 문의하세요.
:::

1. [KOOK 개발자 센터](https://developer.kookapp.cn/app/index)에 접속하여 KOOK 계정으로 로그인하세요. 계정이 없다면 새로 등록할 수 있습니다.
2. **애플리케이션 생성** 버튼을 클릭하여 애플리케이션을 생성하고, 이름(예: LogtoAuth)을 입력한 후 **확인**을 클릭하세요.
3. 생성한 애플리케이션 아이콘을 클릭한 후, **OAuth2** 페이지로 이동하세요.
4. **Callback URIs** 필드에 유효한 리디렉트(예: `${your_logto_origin}/callback/${connector_id}`)를 추가하세요. 리디렉트 URI는 logto 관리자 콘솔의 커넥터 상세 페이지에서 확인할 수 있습니다.
5. **OAuth2 링크 빌더**에서 **get_user_info** 스코프를 반드시 체크하세요.

## KOOK 커넥터 구성하기 \{#configure-your-kook-connector}

이전 섹션에서 OAuth2 페이지에서 받은 Client ID와 Client Secret을 clientId 및 clientSecret 필드에 입력하세요.

### 구성 타입 \{#config-types}

| Name         | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
