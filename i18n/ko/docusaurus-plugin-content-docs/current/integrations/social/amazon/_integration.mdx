### Amazon 앱 설정 \{#setup-a-amazon-app}

[Amazon Developer Portal](https://developer.amazon.com)로 이동하여 계정으로 로그인하세요. 계정이 없는 경우, 등록할 수 있습니다.

그런 다음, 앱을 생성하세요.

**1단계:** 홈페이지의 "Login with Amazon" 블록으로 이동하세요.

또는 [이 페이지](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html)를 방문할 수 있습니다.

가이드를 따라 새 보안 프로필을 생성하세요.

보안 프로필을 생성한 후, 표에서 Client ID와 Client Secret을 받게 됩니다.

**2단계:** 콜백 URL을 설정하세요.

표의 오른쪽에 있는 "Manage" 버튼을 클릭하고 "Web Settings" 탭으로 이동하세요.

"Allowed redirect URLs" 필드에 다음 URL을 추가하세요:

- `${your_logto_origin}/callback/${connector_id}`.

`connector_id`는 Logto Admin Console 커넥터 세부 정보 페이지의 상단 바에서 찾을 수 있습니다.

### 커넥터 구성 \{#configure-your-connector}

Logto 커넥터 구성에서, 이전 단계에서 얻은 값으로 다음 필드를 채우세요:

- **clientId:** 앱의 Client ID.
- **clientSecret:** 앱의 Client Secret.

`scope`는 OIDC 스코프의 공백으로 구분된 목록입니다. 제공되지 않으면 기본 스코프는 `openid profile`입니다.

### 구성 유형 \{#config-types}

| 이름         | 유형   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
