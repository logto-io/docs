# Amazon 소셜 로그인을 설정하세요 (Set up social login with Amazon)

Amazon 소셜 로그인을 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Amazon 커넥터를 사용하면 최종 사용자가 Amazon OAuth 2.0 인증 (Authentication) 프로토콜을 통해 자신의 Amazon 계정으로 애플리케이션에 로그인할 수 있습니다.

<Integration />

## Amazon 커넥터 테스트하기 \{#test-amazon-connector}

이제 완료되었습니다. Amazon 커넥터를 사용할 수 있습니다. [회원가입 및 로그인에서 소셜 커넥터 활성화](/connectors/social-connectors/#enable-social-sign-in)를 잊지 마세요.
