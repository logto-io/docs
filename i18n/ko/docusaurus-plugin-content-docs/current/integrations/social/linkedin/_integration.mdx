### LinkedIn 앱 설정하기 \{#setup-a-linkedin-app}

[LinkedIn Developers](https://www.linkedin.com/developers)로 이동하여 LinkedIn 계정으로 로그인하세요. 계정이 없다면, 하나 등록할 수 있습니다.

그런 다음, 앱을 생성하세요.

**1단계:** 앱 세부 정보 입력.

양식을 작성하고 앱을 생성하세요.

**2단계:** 콜백 URL 설정.

앱 세부 정보 페이지로 이동하여 "Auth" 탭의 "OAuth 2.0" 섹션에서 "Authorized redirect URLs for your app" 필드를 찾으세요.

우리의 경우, 이는 `${your_logto_endpoint}/callback/${connector_id}`가 됩니다. 예를 들어, `https://foo.logto.app/callback/${connector_id}`입니다. `connector_id`는 Logto Admin Console 커넥터 세부 정보 페이지의 상단 바에서 찾을 수 있습니다.

**3단계:** 제품 추가.

"Products" 탭으로 이동하여 "OpenID Connect를 사용한 LinkedIn 로그인" 제품을 추가하세요.

### 커넥터 구성하기 \{#configure-your-connector}

Logto 커넥터 구성에서, 앱의 "Auth" 탭, "Application credentials" 섹션에서 얻은 값으로 다음 필드를 채우세요:

- **clientId:** 앱의 Client ID.
- **clientSecret:** 앱의 Primary Client Secret.

`scope`는 OIDC 스코프의 공백으로 구분된 목록입니다. 제공되지 않으면 기본 스코프는 `openid profile email`입니다.

### 구성 유형 \{#config-types}

| 이름          | 유형   |
| ------------- | ------ |
| clientId      | string |
| clientSecret  | string |
| scope         | string |
