# WeChat 소셜 로그인을 설정하세요 (네이티브)

네이티브 앱(iOS 및 Android)에서 WeChat 소셜 로그인을 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

커넥터의 개념을 잘 모르거나 이 커넥터를 로그인 경험에 추가하는 방법을 모른다면, [Logto 튜토리얼](/connectors/social-connectors)을 참고하세요.

네이티브 앱에서는 웹을 WeChat의 로그인 방식으로 사용할 수 없습니다: WeChat 앱으로의 이동이 필요하며, 그들의 SDK 사용도 요구됩니다.

조금 복잡하게 느껴질 수 있지만 걱정하지 마세요. 이 튜토리얼에서 쉽게 안내해 드리겠습니다.

<Integration />

## WeChat 네이티브 커넥터 테스트하기 \{#test-wechat-native-connector}

이제 끝입니다. <MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">로그인 경험에서 소셜 커넥터 활성화</MainSiteUrl>를 잊지 마세요.

WeChat 네이티브 커넥터가 활성화되면, 앱을 빌드하고 실행하여 정상적으로 동작하는지 확인할 수 있습니다.

:::caution

WeChat은 WeChat 앱이 설치되지 않은 기기에 대한 별도의 계획이 없습니다. 이 경우 Logto는 로그인 중 해당 커넥터를 숨깁니다(이는 [공식 개발 가이드](https://developers.weixin.qq.com/doc/oplatform/en/Mobile_App/WeChat_Login/Development_Guide.html)에서 권장하는 방식입니다).

:::
