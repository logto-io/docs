# Apple 소셜 로그인을 설정하세요

Apple 소셜 로그인을 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

커넥터의 개념을 잘 모르거나, 이 커넥터를 Sign-in & account > Sign-up and sign-in에 추가하는 방법을 모른다면 [Logto 튜토리얼](/connectors/social-connectors)을 참고하세요.

<Integration />

### 스코프 구성 시 주의사항 \{#pitfalls-of-configuring-scope}

Apple로 이미 로그인한 후에 앱이 사용자의 이메일 주소를 요청하도록 구성한 경우, 해당 기존 사용자가 Apple ID로 다시 로그인하더라도 이메일 주소를 가져올 수 없습니다. 이를 해결하려면, 사용자가 [Apple ID 계정 관리 페이지](https://appleid.apple.com/account/manage)에 방문하여 "Apple로 로그인" 섹션에서 애플리케이션을 제거하도록 안내해야 합니다. 이는 앱 상세 페이지에서 "Apple로 로그인 사용 중지"를 선택하여 할 수 있습니다.

예를 들어, 앱이 사용자 이메일과 이름(`email name` 스코프)을 모두 요청하는 경우, 신규 사용자가 처음 로그인할 때 보게 되는 동의 화면은 다음과 비슷해야 합니다:

<img src="/img/assets/sign-in-with-apple-consent-screen.png" alt="Apple로 로그인 동의 화면" />

개발자 토론은 [여기](https://forums.developer.apple.com/forums/thread/132223)에서 확인하세요.

## Apple 커넥터 테스트하기 \{#test-apple-connector}

이제 끝입니다. Apple 커넥터는 웹 및 네이티브 앱 모두에서 사용할 수 있습니다. [Sign-up and sign-in에서 소셜 커넥터 활성화](/connectors/social-connectors/#enable-social-sign-in)를 잊지 마세요.
