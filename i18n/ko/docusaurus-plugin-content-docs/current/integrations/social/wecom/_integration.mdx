## 시작하기 \{#get-started}

관리자 계정으로 [WeCom WebUI](https://work.weixin.qq.com/)에 로그인하거나 WeCom 앱에서 **기업 관리(管理企业)**를 클릭하세요.

"앱 관리" 탭에서 페이지를 아래로 스크롤한 후 "앱 만들기"를 클릭하세요.

앱에 맞는 적절한 정보를 입력한 후 생성하세요.

이제 Agent ID (APPID가 아님)와 Secret을 얻었습니다.

### 웹사이트 정보 \{#website-info}

이 페이지에서 필요한 항목을 설정하세요. 예시는 다음과 같습니다:

- 허용된 사용자: _이 앱을 볼 수 있는 사람_
- 앱 홈페이지: _앱의 홈페이지. 예: `logto.io/demo-app`_

**중요**
이 페이지에는 "개발자 API(开发者接口)"와 관련된 세 가지 항목이 있습니다.

1. 웹 인증 및 JS-SDK;
2. WeCom을 통한 인증으로 로그인;
3. 기업 신뢰 IP;

각 항목은 WeCom의 안내에 따라 입력하세요.

### Corp ID \{#corp-id}

WeChat 개발에 익숙하다면 Corp ID의 사용이 APP ID와 동일하다는 것을 알 수 있습니다.

"내 기업(我的企业)" 탭 페이지 하단에서 Corp ID를 찾을 수 있습니다. 보통 **ww\*\*\*\*** 형식입니다.

## 커넥터 구성하기 \{#configure-the-connector}

이제 Agent ID, Secret, Corp ID를 모두 확보했습니다.

커넥터 양식을 완성해봅시다.

`Scope` 필드는 선택 사항이므로 비워둘 수 있습니다. 또는 `snsapi_base`나 `snsapi_privateinfo`를 입력할 수도 있습니다.

**저장하고 완료하세요**
