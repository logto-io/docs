# Alipay 소셜 로그인을 설정하세요 (Web)

웹 앱에서 Alipay 소셜 로그인을 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Alipay Web 커넥터는 데스크톱 웹 애플리케이션을 위해 설계되었습니다. Alipay의 OAuth 2.0 인증 (Authentication) 워크플로우를 활용하여, 사용자가 번거로운 회원가입 과정 없이 공개 Alipay 사용자 프로필을 사용해 다른 앱에 로그인할 수 있도록 해줍니다.

<Integration />

## Alipay 웹 커넥터 테스트하기 \{#test-alipay-web-connector}

이제 끝입니다. [로그인 경험에서 소셜 커넥터 활성화](/connectors/social-connectors/#enable-social-sign-in)를 잊지 마세요.

Alipay 웹 커넥터가 활성화되면, 웹 앱을 빌드하고 실행하여 정상적으로 동작하는지 확인할 수 있습니다.
