## Patreon 계정으로 로그인하기 \{#sign-in-with-patreon-account}

[Patreon 웹사이트](https://www.patreon.com/)에 접속하여 Patreon 계정으로 로그인하세요. 계정이 없다면 새로 등록할 수 있습니다.

## OAuth 앱 생성 및 구성하기 \{#create-and-configure-oauth-app}

[Patreon OAuth 앱 생성 가이드](https://www.patreon.com/portal/registration/register-clients)를 따라 새 애플리케이션을 등록하세요.

새 OAuth 애플리케이션의 **App Name**에 이름을 입력하고, **App URL**에 앱의 URL을 입력하세요. **App Description** 필드는 비워둘 수 있으며, **Redirect URIs**는 `${your_logto_origin}/callback/${connector_id}`로 설정하세요. `connector_id`는 Logto Admin Console 커넥터 상세 페이지 상단 바에서 확인할 수 있습니다.

:::note
로그인 시 "The redirect_uri MUST match the registered callback URL for this application."라는 오류 메시지가 나타난다면, Patreon OAuth 앱의 Redirect URI와 Logto 앱의 리디렉트 URL(프로토콜 포함)이 일치하는지 확인하여 문제를 해결하세요.
:::

## OAuth 앱 관리하기 \{#managing-oauth-apps}

Patreon의 [Clients & API Keys 페이지](https://www.patreon.com/portal/registration/register-clients)로 이동하면 기존 OAuth 앱을 추가, 수정 또는 삭제할 수 있습니다. 또한 OAuth 앱 상세 페이지에서 `Client ID`를 확인하고 `Client secrets`를 생성할 수 있습니다.

## 커넥터 구성하기 \{#configure-your-connector}

이전 섹션에서 확인한 *Client ID*와 *Client Secret*을 `clientId` 및 `clientSecret` 필드에 입력하세요.

`scope`는 [스코프](https://docs.patreon.com/#scopes)의 공백으로 구분된 목록입니다. 별도로 지정하지 않으면 기본값은 `identity identity[email]`입니다.

### 구성 타입 \{#config-types}

| Name         | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
