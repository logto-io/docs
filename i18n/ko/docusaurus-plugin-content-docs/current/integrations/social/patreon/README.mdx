# Patreon 소셜 로그인을 설정하세요

Patreon 소셜 로그인을 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Patreon 커넥터를 사용하면 최종 사용자가 Patreon OAuth 2.0 인증 (Authentication) 프로토콜을 통해 자신의 Patreon 계정으로 애플리케이션에 로그인할 수 있습니다.
