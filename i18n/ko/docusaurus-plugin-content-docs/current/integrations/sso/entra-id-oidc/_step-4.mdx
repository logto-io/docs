[Microsoft Graph API](https://docs.microsoft.com/en-us/graph/api/overview)에 접근하고 사용자 인가 (Authorization)로 작업을 수행하려면, Logto가 특정 API 스코프 (Scope)를 받아 토큰을 저장해야 합니다.

1. Microsoft Entra 관리 센터의 API 권한 구성과 Logto Microsoft Entra ID 커넥터에 필요한 스코프 (Scope)를 추가하세요.
2. Logto Microsoft Entra ID 커넥터에서 **지속적인 API 접근을 위한 토큰 저장**을 활성화하세요. Logto는 Microsoft 액세스 토큰 (Access token)과 리프레시 토큰 (Refresh token)을 Secret Vault에 안전하게 저장합니다.
3. 리프레시 토큰 (Refresh token)이 반환되도록 하려면, Microsoft Entra ID 애플리케이션 권한에 `offline_access` 스코프 (Scope)를 추가하고 Logto Microsoft Entra ID 커넥터 스코프에도 포함하세요. 이 스코프 (Scope)는 애플리케이션이 장기간 리소스에 접근할 수 있도록 해줍니다.
