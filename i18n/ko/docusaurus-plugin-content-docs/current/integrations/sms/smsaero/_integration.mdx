## 계정 등록 \{#register-account}

[SMSAero](https://smsaero.ru/)에서 새 계정을 만드세요. (이미 계정이 있다면 다음 단계로 넘어가세요.)

## 계정 자격 증명 얻기 \{#get-account-credentials}

커넥터를 작동시키기 위해 API 자격 증명이 필요합니다.  
[API 및 SMPP](https://smsaero.ru/cabinet/settings/apikey/)에서 시작하세요.

"API-key"를 복사하거나 새로 생성하세요.

## 커넥터 JSON 작성 \{#compose-the-connector-json}

_email_, _apiKey_, _senderName_ 필드를 본인의 이메일, API 키, 발신자 이름으로 채우세요.

발신자 이름은 SMSAero에서 제공하는 기본 발신자 이름을 사용하려면 "SMSAero"로 입력할 수 있습니다.

다양한 상황에 맞게 여러 SMS 커넥터 템플릿을 추가할 수 있습니다. 다음은 단일 템플릿을 추가하는 예시입니다:

- `content` 필드에는 임의의 문자열 내용을 입력하세요. 무작위 인증 코드를 위한 `{{code}}` 플레이스홀더를 반드시 남겨두세요.
- `usageType` 필드는 다양한 사용 사례에 따라 `Register`, `SignIn`, `ForgotPassword`, `Generic` 중 하나로 입력하세요.  
  전체 사용자 플로우를 활성화하려면 `Register`, `SignIn`, `ForgotPassword`, `Generic` 용 템플릿이 모두 필요합니다.
