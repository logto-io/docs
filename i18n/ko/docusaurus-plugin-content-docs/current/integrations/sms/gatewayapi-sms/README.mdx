# GatewayAPI SMS로 SMS 인증을 설정하세요

GatewayAPI SMS에 대한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

GatewayAPI는 유럽의 클라우드 서비스 제공업체로, SMS (단문 메시지 서비스)를 포함한 다양한 클라우드 서비스를 제공합니다. GatewayAPI SMS Connector는 Logto 팀에서 제공하는 플러그인으로, 이를 통해 Logto 최종 사용자는 SMS 인증 코드를 통해 Logto 계정에 등록하고 로그인할 수 있습니다.

## GatewayAPI에서 설정하기 \{#set-up-in-gatewayapi}

### GatewayAPI 계정 생성하기 \{#create-an-gatewayapi-account}

[GatewayAPI 웹사이트](https://www.gatewayapi.com/)로 이동하여 GatewayAPI 계정이 없으면 등록하세요.

### 계정 활성화하기 \{#enable-account}

SMS 서비스를 사용하기 전에 계정을 활성화해야 할 수도 있습니다. GatewayAPI 고객 서비스에 문의하여 계정을 활성화하세요.

### API 토큰 받기 \{#get-api-token}

GatewayAPI 콘솔에서 API Keys 페이지로 이동하여 API 토큰을 찾거나 새 API 토큰을 생성하세요.

## Logto에서 설정하기 \{#set-up-in-logto}

1. **Endpoint**: GatewayAPI 계정이 EU 지역에 있는 경우, https://gatewayapi.com/rest/mtsms 엔드포인트를 사용해야 합니다. GatewayAPI 계정이 US 지역에 있는 경우, https://gatewayapi.com/rest/mtsms 엔드포인트를 사용해야 합니다.
2. **API Token**: 이전 단계에서 생성한 API 토큰.
3. **Sender**: SMS를 보내는 데 사용할 발신자.
4. **Templates**: SMS를 보내는 데 사용할 템플릿. 기본 템플릿을 사용하거나 필요에 따라 수정할 수 있습니다.
