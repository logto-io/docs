### GatewayAPI 계정 생성하기 \{#create-an-gatewayapi-account}

[GatewayAPI 웹사이트](https://www.gatewayapi.com/)에 방문하여 GatewayAPI 계정이 없다면 회원가입을 진행하세요.

### 계정 활성화하기 \{#enable-account}

SMS 서비스를 사용하기 전에 계정을 활성화해야 할 수 있습니다. GatewayAPI 고객 서비스에 문의하여 계정을 활성화할 수 있습니다.

### API 토큰 받기 \{#get-api-token}

GatewayAPI 콘솔에서 API Keys 페이지로 이동하여 API 토큰을 찾거나 새 API 토큰을 생성하세요.

## Logto에서 설정하기 \{#set-up-in-logto}

1. **엔드포인트**: GatewayAPI 계정이 EU 지역에 있다면 https://gatewayapi.com/rest/mtsms 엔드포인트를 사용해야 합니다. GatewayAPI 계정이 US 지역에 있다면 https://gatewayapi.com/rest/mtsms 엔드포인트를 사용해야 합니다.
2. **API 토큰**: 이전 단계에서 생성한 API 토큰을 입력하세요.
3. **발신자**: SMS를 보낼 때 사용할 발신자를 입력하세요.
4. **템플릿**: SMS를 보낼 때 사용할 템플릿을 입력하세요. 기본 템플릿을 사용하거나 필요에 따라 수정할 수 있습니다.
