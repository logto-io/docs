# Aliyun Short Message Service로 SMS 인증을 설정하세요

Aliyun 단문 메시지 서비스를 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

알리윈( Aliyun )은 아시아의 주요 클라우드 서비스 제공업체로, SMS(단문 메시지 서비스)를 포함한 다양한 클라우드 서비스를 제공합니다. Aliyun SMS 커넥터는 Logto 팀에서 제공하는 플러그인으로, 이를 통해 Logto 최종 사용자는 SMS 인증 코드를 이용해 Logto 계정에 가입하고 로그인할 수 있습니다.
