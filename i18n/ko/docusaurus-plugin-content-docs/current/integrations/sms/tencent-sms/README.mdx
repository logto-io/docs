# Tencent 단문 메시지 서비스로 SMS 인증을 설정하세요

Tencent 단문 메시지 서비스용 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Tencent Cloud는 아시아의 주요 클라우드 서비스 제공업체로, SMS(단문 메시지 서비스)를 포함한 다양한 서비스를 제공합니다.

Tencent 단문 메시지 서비스 커넥터는 SMS 인증 코드를 사용하여 최종 사용자가 회원가입 또는 로그인을 할 수 있도록 돕는 공식 Logto 통합입니다.
