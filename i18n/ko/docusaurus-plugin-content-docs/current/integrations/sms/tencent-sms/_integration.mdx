### Tencent Cloud 계정 생성하기 \{#create-an-tencent-cloud-account}

[Tencent Cloud 웹사이트](https://cloud.tencent.com/)에 접속하여 계정이 없다면 회원가입을 진행하세요.

### Tencent Cloud SMS 활성화 및 구성하기 \{#enable-and-configure-tencent-cloud-sms}

1. [Tencent Cloud 웹사이트](https://cloud.tencent.com/)에 계정으로 로그인한 후 [SMS 서비스 콘솔 페이지](https://cloud.tencent.com/product/sms)로 이동하세요.
2. SMS 서비스 페이지에서 “무료 체험” (免费试用) 버튼을 클릭하고 안내에 따라 설정을 진행하세요.
3. “SMS 서비스 활성화 약관” (短信开通服务条款)에 동의하고 “서비스 활성화” (开通服务)를 클릭하여 진행하세요.
4. “[SMS 콘솔 개요](https://console.cloud.tencent.com/smsv2)” (短信服务控制台概览)에서 필요에 따라 사이드바에서 “국내 메시지”(国内消息) 또는 “국제/홍콩, 마카오, 대만 메시지”(国际/港澳台消息)를 선택하세요.
5. SMS 메시지용 서명과 템플릿을 추가하세요. 참고:
   - “서명” (签名)을 추가할 때 "적용 시나리오" (模版类型) 필드에서 “인증 코드” (验证码)를 선택하세요.
   - 템플릿을 추가할 때도 템플릿 유형으로 “인증 코드”를 선택하세요.
   - 템플릿 내용에 `{1}` 플레이스홀더를 포함하세요. 이 부분이 무작위로 생성된 코드로 대체됩니다.
   - 여러 개의 플레이스홀더가 있는 템플릿은 지원되지 않습니다. 하나의 플레이스홀더만 포함된 템플릿을 사용하거나 생성하세요.
6. 서명과 템플릿을 승인 요청으로 제출하세요. 제출 후:
   - 적용될 때까지 기다리세요.
   - “[SMS 콘솔 개요](https://console.cloud.tencent.com/smsv2)” (短信服务控制台概览)에서 테스트 SMS를 보낼 수 있습니다. 승인된 서명과 템플릿은 바로 테스트에 사용할 수 있으며, 아직 심사 중이라면 제공된 테스트용 템플릿을 사용할 수 있습니다.
   - 테스트를 위해 계정에 충분한 잔액이 있는지 확인하고, 테스트용 휴대폰 번호를 미리 등록하여 SMS를 정상적으로 수신할 수 있도록 하세요.

자세한 안내는 SMS 콘솔 개요의 “시작 가이드” (新手配置指引) 탭을 클릭하세요.

## 커넥터 JSON 작성하기 \{#compose-the-connector-json}

1. “[SMS 콘솔 개요](https://console.cloud.tencent.com/smsv2)”로 이동하여 우측 상단 아바타에 마우스를 올리고 “접근 관리 > 접근 키 > API 키”로 이동하세요. 보안 인증을 완료한 후 `AccessKey ID`와 `AccessKey Secret`을 생성할 수 있습니다. 안전하게 보관하세요.
2. “국내 메시지” (国内消息) 또는 “국제/홍콩, 마카오, 대만 메시지” (国际/港澳台消息) 탭에서 “서명 이름” (签名名称)과 “템플릿 코드” (模版 CODCE)를 확인하세요.
   - 테스트용 템플릿은 "테스트 전용 서명 템플릿" (测试专用签名模版) 아래의 “빠른 시작” (快速开始) 탭에서 확인하세요.

- Tencent Cloud SMS 커넥터를 다음 필드로 구성하세요:
  - `accessKeyId` 및 `accessKeySecret`: 1단계에서 얻은 자격 증명을 사용하세요.
  - `signName`: 2단계에서 얻은 서명 이름입니다. 모든 템플릿에 적용됩니다.
  - 필요에 따라 다양한 시나리오에 대해 여러 템플릿을 추가할 수 있습니다. 단일 템플릿 예시:
    - `templateCode`: 2단계에서 얻은 템플릿 코드입니다.
    - `usageType`: 각 시나리오에 대해 `Register`, `SignIn`, `ForgotPassword`, `Generic` 중 하나를 지정하세요 (Logto 전용 속성).
    - 모든 네 가지 시나리오에 대해 템플릿이 구성되어야 전체 플로우가 완성됩니다.
  - `sdkAppId`: “[애플리케이션 관리](https://console.cloud.tencent.com/smsv2/app-manage)” (应用管理)에서 확인할 수 있습니다.
  - `region`: “[발송 문서](https://cloud.tencent.com/document/api/382/52071#.E5.9C.B0.E5.9F.9F.E5.88.97.E8.A1.A8)” (发送文档)에서 지원되는 지역을 지정하세요.

### Tencent Cloud SMS 커넥터 테스트하기 \{#test-tencent-cloud-sms-connector}

휴대폰 번호를 입력하고 "전송"을 클릭하여 "저장 및 완료" 전에 설정이 정상 동작하는지 확인할 수 있습니다.

이제 끝입니다. <MainSiteUrl href="/connectors/sms-connectors#enable-phone-number-sign-up-or-sign-in">로그인 경험에서 커넥터 활성화</MainSiteUrl>를 잊지 마세요.
