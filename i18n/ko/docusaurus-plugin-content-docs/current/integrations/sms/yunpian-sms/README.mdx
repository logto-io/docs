# YunPian SMS 커넥터

YunPian SMS 서비스를 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

YunPian은 SMS를 포함한 다양한 서비스를 제공하는 커뮤니케이션 서비스 제공업체입니다. YunPian SMS 커넥터는 Logto 팀에서 제공하는 플러그인으로, YunPian의 SMS 서비스와 연동하여 Logto 최종 사용자가 SMS 인증 코드로 회원가입 및 로그인을 할 수 있도록 지원합니다.

## YunPian 콘솔에서 SMS 서비스 설정 \{#set-up-sms-service-in-yunpian-console}

### YunPian 계정 생성 \{#create-a-yunpian-account}

[YunPian 웹사이트](https://www.yunpian.com/)를 방문하여 계정을 등록하고 실명 인증을 완료하세요.

### API KEY 받기 \{#get-api-key}

1. YunPian 콘솔에 로그인하세요
2. "계정 설정" -> "서브 계정 관리"로 이동하세요
3. API KEY를 찾아 복사하세요

### SMS 템플릿 구성 \{#configure-sms-templates}

1. YunPian 콘솔에서 "국내 SMS" -> "서명 등록"으로 이동하세요
2. 서명을 생성 및 제출하고, 통신사 승인을 기다리세요
3. "국내 SMS" -> "템플릿 등록"으로 이동하여 "인증 코드"를 선택하세요
4. 인증 코드 템플릿을 생성하세요. 반드시 `#code#` 변수를 포함해야 합니다 (승인 속도를 높이려면 "공용 템플릿"을 사용할 수도 있습니다)
5. 템플릿 승인을 기다리세요
6. 국제 SMS를 보내야 하는 경우, 위의 단계를 반복하되 "국제 SMS" -> "템플릿 등록"을 선택하세요

## Logto에서 구성하기 \{#configure-in-logto}

1. Logto 콘솔에서 "커넥터"로 이동하세요
2. "YunPian SMS 서비스"를 찾아 클릭하세요
3. 구성 폼을 작성하세요:
   - API KEY: YunPian에서 받은 API KEY
   - SMS 템플릿: 사용 목적에 따라 템플릿을 구성하세요. YunPian에서 승인된 템플릿과 정확히 일치해야 합니다

## 참고 사항 \{#notes}

1. SMS 템플릿 내용은 YunPian에서 승인된 템플릿과 정확히 일치해야 합니다
2. YunPian 템플릿에서 인증 코드 변수 플레이스홀더는 `#code#`이고, 커넥터 구성에서는 `{{code}}`입니다
3. YunPian은 API KEY에 따라 기본 서명을 자동으로 추가하므로, 템플릿 내용에 서명을 포함할 필요가 없습니다
4. 정식 사용 전에 구성을 테스트하는 것을 권장합니다
