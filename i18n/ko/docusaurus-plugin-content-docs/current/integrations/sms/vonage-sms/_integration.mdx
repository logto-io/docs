## Vonage에서 설정하기 \{#set-up-in-vonage}

:::tip
이미 완료했다면 이 단계를 건너뛸 수 있습니다.
:::

이 커넥터를 사용하려면 Vonage에서 [계정을 등록](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account)해야 합니다. 이를 통해 API 키와 시크릿을 받아 이 커넥터를 통해 API에 접근할 수 있습니다.

계정을 만들면 Vonage API 대시보드 상단에서 API 키와 API 시크릿을 확인할 수 있습니다.

또한 SMS 메시지를 보내기 위해 [가상 번호를 임대](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number)해야 할 수도 있습니다.

자세한 내용은 [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview)를 참고하세요.

## Logto에서 설정하기 \{#set-up-in-logto}

1. **API Key**: Vonage API 키.
2. **API Secret**: Vonage API 시크릿.
3. **Brand Name**: SMS를 보낼 때 사용할 브랜드 이름입니다. 이는 `from` 필드라고도 하며, 자세한 내용은 [발신자 아이덴티티](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id)를 참고하세요.
4. **Templates**: SMS를 보낼 때 사용할 템플릿입니다. 기본 템플릿을 사용하거나 필요에 따라 수정할 수 있습니다.
