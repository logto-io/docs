# Vonage SMS로 SMS 인증을 설정하세요

Vonage SMS를 위한 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Vonage는 SMS (단문 메시지 서비스)를 포함한 강력한 클라우드 기반 커뮤니케이션 서비스를 제공하는 글로벌 커뮤니케이션 제공업체입니다. Vonage SMS 커넥터는 Logto 팀이 제공하는 플러그인으로, Logto 최종 사용자가 SMS 인증 코드를 통해 Logto 계정에 등록하고 로그인할 수 있도록 합니다.

## Vonage에서 설정하기 \{#set-up-in-vonage}

:::tip
이미 완료한 경우 이 단계를 건너뛸 수 있습니다.
:::

이 커넥터를 사용하려면 Vonage에서 [계정을 등록](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account)해야 합니다. 이를 통해 이 커넥터를 통해 API에 접근할 수 있는 API 키와 비밀을 얻을 수 있습니다.

계정을 생성한 후, Vonage API 대시보드 상단에서 API 키와 API 비밀을 찾을 수 있습니다.

SMS 메시지를 보내려면 [가상 번호를 임대](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number)해야 할 수도 있습니다.

자세한 내용은 [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview)를 참조하세요.

## Logto에서 설정하기 \{#set-up-in-logto}

1. **API Key**: Vonage API 키.
2. **API Secret**: Vonage API 비밀.
3. **Brand Name**: SMS를 보내는 데 사용할 브랜드 이름입니다. 이는 `from` 필드라고도 하며, 자세한 내용은 [발신자 아이덴티티](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id)를 참조하세요.
4. **Templates**: SMS를 보내는 데 사용할 템플릿입니다. 기본 템플릿을 사용하거나 필요에 따라 수정할 수 있습니다.
