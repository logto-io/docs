# Vonage SMS로 SMS 인증을 설정하세요

Vonage SMS용 공식 Logto 커넥터입니다.

<GuideTip />

## 시작하기 \{#get-started}

Vonage는 글로벌 커뮤니케이션 제공업체로, SMS(단문 메시지 서비스)를 포함한 강력한 클라우드 기반 커뮤니케이션 서비스를 제공합니다. Vonage SMS 커넥터는 Logto 팀에서 제공하는 플러그인으로, Logto 최종 사용자가 SMS 인증 코드를 통해 Logto 계정에 등록하고 로그인할 수 있도록 지원합니다.
