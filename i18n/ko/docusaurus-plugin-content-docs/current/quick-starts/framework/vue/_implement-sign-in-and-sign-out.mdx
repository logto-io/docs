### 로그인 및 로그아웃 구현 \{#implement-sign-in-and-sign-out}

우리는 인증 흐름을 쉽게 관리할 수 있도록 두 가지 컴포저블 `useHandleSignInCallback()` 및 `useLogto()`를 제공합니다.

<SignInNote calling=".signIn()" />

```tsx title="views/HomeView.vue"
import { useLogto } from '@logto/vue';

const { signIn, signOut, isAuthenticated } = useLogto();

const onClickSignIn = () => signIn('http://localhost:3000/callback');
const onClickSignOut = () => signOut('http://localhost:3000');
```

```html
<template>
  <button v-if="!isAuthenticated" @click="onClickSignIn">로그인</button>
  <button v-else @click="onClickSignOut">로그아웃</button>
</template>
```
