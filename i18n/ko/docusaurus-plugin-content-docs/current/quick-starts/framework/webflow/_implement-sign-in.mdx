#### 로그인 버튼 구현하기 \{#implement-a-sign-in-button}

Webflow 디자이너로 돌아가서, "로그인" 버튼을 홈페이지에 드래그 앤 드롭하고, 나중에 `getElementById()`를 사용하여 참조할 수 있도록 ID를 "sign-in"으로 지정하세요.

```html
<script type="module">
  const signInButton = document.getElementById('sign-in');
  const onClickSignIn = () => logtoClient.signIn('https://your-awesome-site.webflow.io/callback');
  signInButton.addEventListener('click', onClickSignIn);
</script>
```
