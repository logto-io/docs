# Passport.js 애플리케이션에 인증 (Authentication)을 추가하세요

이 가이드에서는 [Passport.js](https://www.passportjs.org/)와 OIDC 전략을 사용하여 Logto를 애플리케이션에 통합하는 방법을 안내합니다.

<GuideTip />

## 사전 준비 사항 \{#prerequisites}

- [Logto Cloud](https://cloud.logto.io) 계정 또는 [셀프 호스팅 Logto](/introduction/set-up-logto-oss).
- 생성된 Logto 전통 애플리케이션.
- 세션이 구성된 express 프로젝트. [Express.js 웹사이트](https://expressjs.com/)를 참고하세요.
