# Auth.js (Next Auth) 애플리케이션에 인증 (Authentication)을 추가하세요

이 가이드에서는 [Auth.js](https://authjs.dev/) (이전 명칭: Next Auth)와 함께 Logto를 Next.js 애플리케이션에 통합하는 방법을 안내합니다.

<GuideTip />

## 사전 준비 사항 \{#prerequisites}

- [Logto Cloud](https://cloud.logto.io) 계정 또는 [셀프 호스팅 Logto](/introduction/set-up-logto-oss).
- 생성된 Logto 전통 애플리케이션.
- Auth.js가 적용된 Next.js 프로젝트, [Auth.js 문서](https://authjs.dev/getting-started/installation)를 참고하세요.
