# Auth.js (Next Auth) 애플리케이션에 인증 (Authentication)을 추가하세요

이 가이드는 이전에 Next Auth로 알려진 [Auth.js](https://authjs.dev/)를 사용하여 Logto를 Next.js 애플리케이션에 통합하는 방법을 보여줍니다.

<GuideTip />

## 사전 준비 사항 \{#prerequisites}

- [Logto Cloud](https://cloud.logto.io) 계정 또는 [셀프 호스팅 Logto](/introduction/set-up-logto-oss).
- Logto 전통적인 애플리케이션 생성.
- Auth.js가 포함된 Next.js 프로젝트, [Auth.js 문서](https://authjs.dev/getting-started/installation)를 확인하세요.
