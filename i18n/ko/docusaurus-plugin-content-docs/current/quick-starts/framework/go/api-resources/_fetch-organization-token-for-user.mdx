#### 조직 API 리소스 \{#organization-api-resources}

조직 내 API 리소스에 대한 액세스 토큰 (Access token)을 가져오려면, `GetAccessTokenWithOptions` 메서드를 사용하여 API 리소스와 조직 ID를 모두 파라미터로 전달할 수 있습니다:

```go
accessToken, error := client.GetAccessTokenWithOptions(
  client.GetAccessTokenOptions{
    Resource: 'https://shopping.your-app.com/api',
    OrganizationId: organizationId,
  },
);
```
