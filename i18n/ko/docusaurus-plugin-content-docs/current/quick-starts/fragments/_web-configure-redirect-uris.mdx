{/* 이 프래그먼트는 리디렉션 URI와 로그아웃 후 리디렉션 URI 구성을 결합하여 특정 프레임워크에 유용할 수 있습니다. */}

export const defaultBaseUrl = 'http://localhost:3000/';
export const defaultRedirectUri = `${defaultBaseUrl}callback`;

#### 리디렉션 URI 구성 \{#configure-redirect-uris}

Logto Console의 애플리케이션 세부 정보 페이지로 이동합니다. 리디렉션 URI <code>{props.redirectUri ?? 'http://localhost:3000/callback'}</code>를 추가하세요.

<img alt="Logto Console의 리디렉션 URI" src="/img/assets/web-redirect-uri.png" width="600px" />

로그인과 마찬가지로, 사용자는 공유 세션에서 로그아웃하기 위해 Logto로 리디렉션되어야 합니다. 완료되면 사용자를 다시 웹사이트로 리디렉션하면 좋습니다. 예를 들어, 로그아웃 후 리디렉션 URI 섹션에 <code>{defaultBaseUrl}</code>를 추가하세요.

그런 다음 "저장"을 클릭하여 변경 사항을 저장하세요.
