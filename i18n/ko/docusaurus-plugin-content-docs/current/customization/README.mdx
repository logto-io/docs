# 커스터마이징

브랜딩, 사용자 인터페이스, 언어, 도메인 설정 등 모든 측면을 커스터마이징하여 Logto를 여러분만의 것으로 만드세요.

## 기본 제공 UI \{#prebuilt-ui}

기본 제공 UI 컴포넌트로 시작하여 필수 브랜드 요소를 커스터마이징하세요. 다음 중 하나를 만들 수 있습니다:

- 플랫폼 전반에 걸친 통합된 인증 (Authentication) 경험.
- 특정 애플리케이션 또는 조직을 위한 전용 로그인 페이지.

<DocCardList
  items={[
    {
      type: 'link',
      label: '브랜드 맞춤화',
      href: '/customization/match-your-brand',
      description:
        '즉시 사용할 수 있는 로그인 경험과 계정 센터를 위해 기본 브랜드 요소를 커스터마이징하는 방법을 알아보세요.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## 콘텐츠 & 언어 \{#content--languages}

## 커스텀 도메인 \{#custom-domain}

인증 (Authentication) URL에 여러분의 도메인을 사용하여 브랜드 인지도를 강화하세요. 기본 Logto 도메인을 여러분의 브랜드 도메인으로 대체할 수 있습니다.

  items={[
    {

## 라이브 프리뷰 \{#live-preview}

라이브 프리뷰 기능을 사용하여 다양한 인증 (Authentication) 플로우([회원가입](/end-user-flows/sign-up-and-sign-in/sign-up), [로그인](/end-user-flows/sign-up-and-sign-in/sign-in), [소셜 로그인](/end-user-flows/sign-up-and-sign-in/social-sign-in), [비밀번호 찾기](/end-user-flows/sign-up-and-sign-in/reset-password), [MFA](/end-user-flows/mfa), [SSO](/end-user-flows/enterprise-sso) 등)을 실험해보세요. 내장된 데모 앱을 통해 로그인 경험을 즉시 테스트할 수 있습니다.
