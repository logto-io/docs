새로운 FastAPI 프로젝트를 초기화하려면, 디렉터리를 생성하고 기본 구조를 설정하세요:

```bash
mkdir your-api-name
cd your-api-name
```

requirements 파일을 생성하세요:

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

의존성을 설치하세요:

```bash
pip install -r requirements.txt
```

기본 FastAPI 애플리케이션을 생성하세요:

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

개발 서버를 시작하세요:

```bash
uvicorn main:app --reload
```

:::note
경로 연산, 의존성 주입 및 기타 기능 설정 방법에 대한 자세한 내용은 FastAPI 문서를 참조하세요.
:::
