새로운 Flask 프로젝트를 초기화하려면, 디렉터리를 생성하고 기본 구조를 설정하세요:

```bash
mkdir your-api-name
cd your-api-name
```

requirements 파일을 생성하세요:

```txt title="requirements.txt"
Flask
```

의존성을 설치하세요:

```bash
pip install -r requirements.txt
```

기본 Flask 애플리케이션을 생성하세요:

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

개발 서버를 시작하세요:

```bash
python app.py
```

:::note
라우트, 블루프린트 및 기타 기능 설정 방법에 대한 자세한 내용은 Flask 문서를 참조하세요.
:::
