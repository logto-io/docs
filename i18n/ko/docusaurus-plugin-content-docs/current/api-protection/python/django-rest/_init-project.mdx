새로운 Django REST Framework 프로젝트를 초기화하려면 다음과 같이 하세요:

```bash
django-admin startproject your_api_name
cd your_api_name
```

필요한 패키지를 설치하세요:

```bash
pip install Django djangorestframework
```

기본 Django 앱을 생성하세요:

```bash
python manage.py startapp api
```

설정에 DRF를 추가하세요:

```py title="your_api_name/settings.py"
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'rest_framework',
    'api',
]
```

기본 API 뷰를 생성하세요:

```py title="api/views.py"
from rest_framework.decorators import api_view
from rest_framework.response import Response

@api_view(['GET'])
def hello_view(request):
    return Response({"message": "Hello from Django REST Framework"})
```

URL 구성을 추가하세요:

```py title="api/urls.py"
from django.urls import path
from . import views

urlpatterns = [
    path('', views.hello_view, name='hello'),
]
```

```py title="your_api_name/urls.py"
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('api/', include('api.urls')),
]
```

개발 서버를 시작하세요:

```bash
python manage.py runserver
```

:::note
시리얼라이저, 뷰셋 및 기타 기능 설정 방법에 대한 자세한 내용은 Django REST Framework 문서를 참고하세요.
:::
