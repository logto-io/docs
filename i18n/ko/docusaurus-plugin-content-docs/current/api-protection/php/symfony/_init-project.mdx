API 개발을 위한 새로운 Symfony 프로젝트를 초기화하려면 Symfony CLI 또는 Composer를 사용하세요:

Symfony CLI 사용 (권장):

```bash
symfony new your-api-name --webapp
cd your-api-name
```

또는 Composer 사용:

```bash
composer create-project symfony/skeleton your-api-name
cd your-api-name
composer require webapp
```

API 개발을 위한 추가 패키지 설치:

```bash
composer require symfony/security-bundle
composer require symfony/serializer
composer require doctrine/annotations
```

개발 서버 시작:

```bash
symfony serve
```

또는 PHP 내장 서버 사용:

```bash
php -S localhost:8000 -t public/
```

이렇게 하면 기본 Symfony 프로젝트가 생성됩니다. API 개발을 위해 프레임워크를 다음과 같이 구성하세요:

```yaml title="config/packages/framework.yaml"
framework:
  secret: '%env(APP_SECRET)%'
  serializer:
    enabled: true
  property_access:
    enabled: true
```

:::note
컨트롤러, 서비스 및 기타 기능 설정에 대한 자세한 내용은 Symfony 문서를 참조하세요.
:::
