Fiber와 함께 새로운 Go 프로젝트를 초기화하려면 다음 단계를 따르세요:

```bash
go mod init your-api-name
go get github.com/gofiber/fiber/v2
```

그런 다음, 기본 Fiber 서버를 다음과 같이 설정하세요:

```go title="main.go"
package main

import (
    "log"

    "github.com/gofiber/fiber/v2"
)

func main() {
    app := fiber.New()

    log.Fatal(app.Listen(":3000"))
}
```

:::note
라우트, 미들웨어 및 기타 기능 설정에 대한 자세한 내용은 Fiber 문서를 참고하세요.
:::
