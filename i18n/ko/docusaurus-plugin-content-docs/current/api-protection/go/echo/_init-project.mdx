Echo로 새로운 Go 프로젝트를 초기화하려면 다음 단계를 따라 진행하세요:

```bash
go mod init your-api-name
go get github.com/labstack/echo/v4
```

그런 다음, 기본 Echo 서버를 설정하세요:

```go title="main.go"
package main

import (
    "github.com/labstack/echo/v4"
)

func main() {
    e := echo.New()

    e.Logger.Fatal(e.Start(":3000"))
}
```

:::note
라우트, 미들웨어 및 기타 기능 설정에 대한 자세한 내용은 Echo 문서를 참고하세요.
:::
