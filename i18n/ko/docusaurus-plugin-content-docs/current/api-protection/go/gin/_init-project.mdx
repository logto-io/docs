Gin으로 새로운 Go 프로젝트를 초기화하려면 다음 단계를 따르세요:

```bash
go mod init your-api-name
go get github.com/gin-gonic/gin
```

그런 다음, 기본 Gin 서버를 설정하세요:

```go title="main.go"
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()

    r.Run(":3000") // 0.0.0.0:3000에서 리슨 및 서비스합니다
}
```

:::note
라우트, 미들웨어 및 기타 기능 설정에 대한 자세한 내용은 Gin 문서를 참조하세요.
:::
