NestJS로 새로운 Node.js 프로젝트를 초기화하려면, Nest CLI를 사용하여 빠르게 설정할 수 있습니다. 단계는 다음과 같습니다:

```bash
npm i -g @nestjs/cli
nest new my-api
cd my-api
```

또는, 수동으로 프로젝트를 생성할 수도 있습니다:

```bash
npm init -y
npm install @nestjs/core @nestjs/common @nestjs/platform-express reflect-metadata rxjs
```

TypeScript를 사용하는 경우, TypeScript 환경을 적절히 설정하는 것을 잊지 마세요.

그런 다음, 기본적인 NestJS 서버 설정을 만듭니다:

```ts title="main.ts"
import { NestFactory } from '@nestjs/core';
import { AppModule } from './app.module';

async function bootstrap() {
  const app = await NestFactory.create(AppModule);
  await app.listen(3000);
  console.log('Server running on http://localhost:3000');
}
bootstrap();
```

```ts title="app.module.ts"
import { Module } from '@nestjs/common';
import { AppController } from './app.controller';
import { AppService } from './app.service';

@Module({
  imports: [],
  controllers: [AppController],
  providers: [AppService],
})
export class AppModule {}
```

:::note
컨트롤러, 서비스 및 기타 기능 설정 방법에 대한 자세한 내용은 NestJS 문서를 참고하세요. 또한, NestJS CLI 명령어를 활용하여 컴포넌트와 모듈을 쉽게 생성할 수 있습니다.
:::
