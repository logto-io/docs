새로운 Node.js 프로젝트를 Express로 초기화하려면 다음 단계를 따르세요:

```bash
npm init -y
npm install express
```

TypeScript를 사용하는 경우, TypeScript 환경도 적절히 설정해야 합니다.

그런 다음, 기본 Express 서버를 다음과 같이 설정하세요:

```ts title="app.ts"
import express from 'express';

const app = express();

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

:::note
라우트, 미들웨어 및 기타 기능 설정에 대한 자세한 내용은 Express 문서를 참고하세요.
:::
