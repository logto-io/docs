새로운 Rails API 프로젝트를 초기화하려면 Rails 생성기를 사용할 수 있습니다:

```bash
rails new your-api-name --api
cd your-api-name
```

개발 서버를 시작하세요:

```bash
rails server
```

기본 API 컨트롤러를 생성하세요:

```ruby title="app/controllers/api/base_controller.rb"
class Api::BaseController < ApplicationController
  def index
    render json: { message: 'Hello from Rails API' }
  end
end
```

라우트를 추가하세요:

```ruby title="config/routes.rb"
Rails.application.routes.draw do
  namespace :api do
    root 'base#index'
  end
end
```

:::note
컨트롤러, 모델 및 기타 기능 설정 방법에 대한 자세한 내용은 Rails 공식 문서를 참고하세요.
:::
