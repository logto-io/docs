새로운 Grape API 프로젝트를 초기화하려면, 디렉터리를 생성하고 기본 구조를 설정하세요:

```bash
mkdir your-api-name
cd your-api-name
```

Gemfile을 생성하세요:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

의존성을 설치하세요:

```bash
bundle install
```

기본 Grape API를 생성하세요:

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Grape API에서 인사드립니다' }
  end
end
```

config.ru 파일을 생성하세요:

```ruby title="config.ru"
require_relative 'api'

run API
```

개발 서버를 시작하세요:

```bash
bundle exec rackup
```

:::note
리소스 설정, 파라미터 검증 및 기타 기능에 대한 자세한 내용은 Grape 문서를 참고하세요.
:::
