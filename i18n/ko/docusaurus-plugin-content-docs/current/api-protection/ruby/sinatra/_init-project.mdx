새로운 Sinatra 프로젝트를 초기화하려면, 디렉터리를 생성하고 기본 구조를 설정하세요:

```bash
mkdir your-api-name
cd your-api-name
```

Gemfile을 생성하세요:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'sinatra'
```

의존성을 설치하세요:

```bash
bundle install
```

기본 Sinatra 애플리케이션을 생성하세요:

```ruby title="app.rb"
require 'sinatra'
require 'json'

get '/' do
  content_type :json
  { message: 'Hello from Sinatra API' }.to_json
end
```

개발 서버를 시작하세요:

```bash
ruby app.rb
```

:::note
라우트, 미들웨어 및 기타 기능 설정에 대한 자세한 내용은 Sinatra 문서를 참조하세요.
:::
