# Logto로 API를 보호하세요

[역할 기반 접근 제어 (RBAC)](/authorization/role-based-access-control) 및 [JSON Web Token (JWT)](https://auth.wiki/jwt) 검증을 구현하여 API 엔드포인트를 무단 접근으로부터 보호하는 방법을 알아보세요.

우리 가이드에서는 미들웨어 설정, 권한 모델, 다양한 프로그래밍 언어 및 프레임워크에서의 실용적인 예제를 다룹니다.

## 시작하기 \{#get-started}

### API 보안이 처음이신가요? \{#new-to-api-security}

API 인증 (Authentication) 및 인가 (Authorization)가 처음이라면, 다음 기본 개념부터 시작하세요:

- **[인증 (Authentication) vs. 인가 (Authorization)](/concepts/authn-vs-authz)** - 주요 차이점 이해하기
- **[역할 기반 접근 제어 (RBAC)](/authorization/role-based-access-control)** - 권한과 역할이 어떻게 작동하는지 배우기
- **[JSON Web Token (JWT)](https://auth.wiki/jwt)** - JSON Web Token이 API를 어떻게 보호하는지 알아보기

### 언어나 프레임워크를 선택하세요 \{#choose-your-language-or-framework}

API 보호를 구현할 준비가 되셨나요? 사이드바에서 기술 스택을 선택하여 단계별 통합 가이드, 코드 예제, 모범 사례를 확인하세요.
