Logto는 SMS 내용을 맞춤화할 수 있도록 네 가지 유형의 템플릿을 제공합니다. 이 템플릿들은 사용 목적에 따라 Register, SignIn, ForgotPassword, Generic으로 분류됩니다. 다양한 사용 사례에 따라 서로 다른 템플릿을 사용하는 것이 강력히 권장되며, 그렇지 않을 경우 속도 제한에 걸려 서비스가 일시적으로 중단될 수 있습니다.

## SMS 템플릿 유형 및 예시 \{#sms-template-types-and-examples}

다음은 참고용 예시입니다:

| usageType                | 시나리오                                                                                                                                                                                                                                                                                                                                                                                      | 템플릿 예시                                                      |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| SignIn                   | 사용자가 [휴대폰 번호로 로그인](/end-user-flows/sign-up-and-sign-in/sign-in)하고 비밀번호 대신 SMS 인증 코드를 입력하여 인증합니다.                                                                                                                                                                                                                                                           | Logto 로그인 인증 코드: `{{code}}`. 10분 후 만료됩니다.          |
| Register                 | 사용자가 [휴대폰 번호로 계정을 생성](/end-user-flows/sign-up-and-sign-in/sign-up)하고, Logto에서 해당 번호로 전송된 인증 코드를 입력하여 인증합니다.                                                                                                                                                                                                                                          | Logto 회원가입 인증 코드: `{{code}}`. 10분 후 만료됩니다.        |
| ForgotPassword           | 사용자가 로그인 중 비밀번호를 잊은 경우, 먼저 휴대폰 번호로 본인 인증을 선택하여 [비밀번호를 재설정](/end-user-flows/sign-up-and-sign-in/reset-password)할 수 있습니다.                                                                                                                                                                                                                       | Logto 비밀번호 재설정 인증 코드: `{{code}}`. 10분 후 만료됩니다. |
| Generic                  | 이 템플릿은 다양한 상황에서 일반 백업 옵션으로 사용할 수 있습니다. 예를 들어 커넥터 구성 테스트, [로그인 후 휴대폰 번호 인증 또는 연결](/end-user-flows/account-settings/by-management-api#email-and-phone-number-verification) 등입니다.                                                                                                                                                     | Logto 인증 코드: `{{code}}`. 10분 후 만료됩니다.                 |
| OrganizationInvitation   | 이 템플릿을 사용하여 사용자가 조직에 가입할 수 있도록 [초대 링크를 전송](/end-user-flows/organization-experience/invite-organization-members#configure-your-email-connector)합니다.                                                                                                                                                                                                           | Logto 조직 초대 인증 코드: `{{code}}`. 10분 후 만료됩니다.       |
| UserPermissionValidation | 앱 사용 중 은행 이체, 사용 중인 리소스 삭제, 멤버십 해지 등 [추가 사용자 인증이 필요한](/end-user-flows/account-settings/by-account-api#verify-by-sending-a-verification-code-to-the-users-email-or-phone) 고위험 또는 비교적 높은 위험 수준의 작업이 있을 수 있습니다. `UserPermissionValidation` 템플릿을 사용하여 이러한 상황에서 사용자가 받는 SMS 인증 코드의 내용을 정의할 수 있습니다. | Logto 인증 코드: `{{code}}`. 10분 후 만료됩니다.                 |
| BindNewIdentifier        | 사용자가 프로필을 수정할 때 [현재 계정에 휴대폰 번호를 연결](/end-user-flows/account-settings/by-account-api#manage-phone)할 수 있습니다. 이 경우, `BindNewIdentifier` 템플릿을 사용하여 인증 SMS의 내용을 맞춤화할 수 있습니다.                                                                                                                                                              | Logto 계정 연결 인증 코드: `{{code}}`. 10분 후 만료됩니다.       |
| MfaVerification          | [SMS MFA](/end-user-flows/mfa/sms-mfa)가 활성화된 경우, 이 템플릿은 다단계 인증 (MFA) 과정에서 사용자에게 인증 코드를 전송하는 데 사용됩니다.                                                                                                                                                                                                                                                 | Logto 2단계 인증 코드: `{{code}}`. 10분 후 만료됩니다.           |
| BindMfa                  | [SMS MFA](/end-user-flows/mfa/sms-mfa)가 활성화된 경우, 이 템플릿은 MFA용 SMS 인증 코드를 설정하는 데 사용됩니다. 사용자가 계정의 MFA 요소로 휴대폰 번호를 연결하거나 구성할 때 이 인증 코드를 받게 됩니다.                                                                                                                                                                                   | Logto 2단계 인증 추가 코드: `{{code}}`. 10분 후 만료됩니다.      |

이러한 파라미터를 이해하는 것이 중요합니다:

- 인증 코드는 10분 동안 유효합니다. 현재 만료 시간은 커스터마이즈를 지원하지 않습니다.
- Logto는 SMS 템플릿의 `{{code}}` 플레이스홀더를 인증 코드로 대체합니다. 따라서 템플릿에 반드시 플레이스홀더가 포함되어 있어야 합니다.

:::note
일부 국가 및 지역에서는 규정 준수 요구 사항으로 인해 승인되지 않은 내용을 SMS로 전송하는 것이 허용되지 않을 수 있습니다. SMS 템플릿은 사용 전에 SMS 제공업체에 등록 및 승인을 받아야 합니다. 이러한 경우, 내용은 템플릿 ID로 인덱싱되어 해당 템플릿과 매칭될 수 있습니다.
:::
