# 감사 로그의 이벤트 유형

<CloudLink to="/audit-logs">Logto Console > 감사 로그</CloudLink>에서 이벤트 유형을 필터링할 수
있습니다.

:::note

Logto는 이제 [Experience API](https://openapi.logto.io/group/endpoint-experience)를 통해 최종 사용자 상호작용과 관련된 로그 조회를 지원합니다.

[Management API](/integrate-logto/interact-with-management-api) 및 [Account API](https://openapi.logto.io/group/endpoint-account-center)의 감사 로그는 곧 제공될 예정입니다.

요구 사항을 공유하고 싶으시면 언제든지 [문의해 주세요](https://logto.io/contact).

:::

## 토큰 교환

| Key                               | Name                               |
| --------------------------------- | ---------------------------------- |
| ExchangeTokenBy.AuthorizationCode | 코드로 토큰 교환                   |
| ExchangeTokenBy.ClientCredentials | 클라이언트 자격 증명으로 토큰 교환 |
| ExchangeTokenBy.RefreshToken      | 리프레시 토큰으로 토큰 교환        |
| ExchangeTokenBy.TokenExchange     | 토큰 교환                          |

## 커스텀 토큰 클레임(Claim)

| Key                            | Name                                                       |
| ------------------------------ | ---------------------------------------------------------- |
| JwtCustomizer.AccessToken      | 사용자 커스텀 액세스 토큰(Access token) 클레임(Claim) 조회 |
| JwtCustomizer.ClientCredential | M2M 커스텀 액세스 토큰(Access token) 클레임(Claim) 조회    |
