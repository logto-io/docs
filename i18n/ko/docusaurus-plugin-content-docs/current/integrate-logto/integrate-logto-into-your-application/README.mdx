# Logto를 애플리케이션에 통합하세요

Logto를 사용하여 사용자 대상 앱이든 기계 간 서비스든 애플리케이션에 인증 (Authentication)을 추가하려면 다음 단계를 따르세요:

1. <CloudLink to="/applications">콘솔 > 애플리케이션</CloudLink>로 이동하세요.

2. "애플리케이션 생성"을 클릭하여 새 애플리케이션을 추가하세요.

3. [애플리케이션 프레임워크](/quick-starts)를 선택하여 시작하세요. 프레임워크를 찾을 수 없는 경우, 애플리케이션 생성 페이지 오른쪽 하단의 "프레임워크 없이 앱 생성" 버튼을 클릭하여 [애플리케이션 유형](/integrate-logto/application-data-structure/#application-types)을 선택해 앱을 생성하거나, [SDK 규칙](/developers/sdk-conventions)을 참고하여 기능 요청을 제출하거나 SDK를 기여할 수 있습니다.

4. 프레임워크를 선택하면 해당 프레임워크 SDK의 빠른 시작 가이드가 표시됩니다. 안내에 따라 애플리케이션을 구성하고 통합하세요. 통합 과정에서 필요한 개념을 이해하는 데 도움이 필요하다면, [Logto 인증 (Authentication) 플로우 이해하기](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/)를 참고하여 더 깊이 이해할 수 있습니다.

:::note
콘솔의 가이드는 Logto SDK를 사용한 빠른 시작용입니다. 고급 SDK 사용법을 포함한 전체 통합 가이드는 [빠른 시작](/quick-starts) 섹션을 참고하세요.
:::
