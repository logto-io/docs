# 사용자 관리

사용자 관리는 Logto에서 중요한 중점 영역입니다. Logto Console 또는 Management API를 사용하여 사용자 데이터를 조회하고 관리할 수 있으며, 두 방법 모두 효과적입니다. 이러한 도구를 통해 다음과 같은 작업을 수행할 수 있습니다:

- 사용자 이름, ID, 전체 이름, 전화번호 또는 이메일로 사용자 검색
- 소셜 계정 연결 해제, 프로필 정보 업데이트, 사용자 삭제, 비밀번호 재설정 등 사용자에 대한 여러 작업 수행
- 사용자 로그 확인
- 접근 제어를 위한 역할 할당
- 개인 액세스 토큰 생성 및 관리
- 사용자 세션 관리
