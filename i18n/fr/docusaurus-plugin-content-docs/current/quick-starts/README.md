# Démarrages rapides

Bienvenue dans les démarrages rapides Logto. Cette documentation contient des guides utiles pour intégrer Logto à votre application.

La plupart des guides utilisent les SDK officiels Logto. Si vous utilisez des outils de développement alimentés par l’IA, essayez notre [Logto MCP Server](/logto-cloud/logto-mcp-server) pour une intégration assistée par l’IA.

N’hésitez pas à nous contacter si vous avez des questions.

:::info
Cette page se concentre sur l’intégration du client Logto, y compris les modèles backend-for-frontend. Pour la protection côté serveur de l’API, consultez [Protection de l’API](/api-protection).
:::
