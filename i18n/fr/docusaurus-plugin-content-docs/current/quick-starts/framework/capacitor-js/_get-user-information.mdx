### Afficher les informations de l'utilisateur \{#display-user-information}

Pour afficher les informations de l'utilisateur, vous pouvez utiliser la méthode `getIdTokenClaims()`. Par exemple, dans une application Capacitor :

```tsx
const userClaims = await logtoClient.getIdTokenClaims();
console.log(userClaims);
```

Pour demander des portées supplémentaires, vous pouvez passer les portées à l'objet `LogtoConfig` lors de l'initialisation du client. Par exemple :

```tsx
const logtoConfig = {
  scopes: ['email', 'phone', 'custom_data', 'organizations'],
};
```
