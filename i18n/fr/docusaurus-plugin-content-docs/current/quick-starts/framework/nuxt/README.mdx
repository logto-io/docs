## Utilisation dans le middleware ou les routes API \{#use-in-middleware-or-api-routes}

Les composables `useLogtoClient()` et `useLogtoUser()` ne sont pas disponibles dans le middleware ou les routes API. Vous pouvez utiliser la fonction `logtoEventHandler()` pour obtenir le client Logto et d'autres contextes :

```ts title="/server/api/auth/get-access-token.ts"
import { logtoEventHandler } from '#logto';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
