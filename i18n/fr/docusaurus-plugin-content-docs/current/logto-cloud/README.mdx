# Service cloud Logto

Logto Cloud offre une gamme de services de gestion et d'exploitation pour vous aider à gérer les identités et les ressources de manière fluide et facile. Hébergé par Logto, il inclut des fonctionnalités telles que la prise en charge multi-régions, la gestion des locataires, la facturation et les prix, la gestion des membres et le contrôle d'accès basé sur les rôles de la console.

Si vous avez des questions sur les services cloud et avez besoin d'un support supplémentaire, veuillez nous contacter.
