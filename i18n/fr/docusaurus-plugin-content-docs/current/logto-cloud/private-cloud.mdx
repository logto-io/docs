# Logto Cloud Privé

Déployez Logto dans un environnement Azure dédié pour une isolation renforcée, une conformité accrue et un contrôle total de l’infrastructure.

## Vue d’ensemble \{#overview}

Logto Cloud Privé est une option de déploiement dédiée pour les clients qui ont besoin de plus d’isolation et de contrôle de l’infrastructure qu’un environnement cloud public mutualisé.

Il est conçu pour les organisations ayant des exigences plus strictes en matière de sécurité, de conformité, d’approvisionnement ou de déploiement, tout en conservant la même expérience produit principale que Logto Cloud.

Voici comment les options cloud diffèrent :

- **Logto Cloud** : cloud public géré, chemin le plus rapide vers la production
- **Logto Cloud Privé** : déploiement dédié pour les clients ayant des exigences plus strictes en matière de résidence des données ou de conformité

## Quand choisir le Cloud Privé \{#when-to-choose-private-cloud}

Logto Cloud Privé est généralement adapté lorsque votre organisation requiert un ou plusieurs des éléments suivants :

- Une isolation d’environnement renforcée
- Un engagement de disponibilité garanti par un SLA, avec support et collaboration de l’équipe Logto
- Des exigences de conformité spécifiques qui ne peuvent pas être satisfaites par un cloud public mutualisé
- Un déploiement dans une région ou une infrastructure spécifique non prise en charge par Logto Cloud
- Une connexion à des ressources internes inaccessibles depuis un environnement cloud public

Si vous n’avez pas ces exigences, [Logto Cloud](https://cloud.logto.io/?sign_up=1) est généralement le meilleur choix car il est plus simple à adopter et plus facile à exploiter.

### Portée du déploiement \{#deployment-scope}

Logto Cloud Privé est actuellement disponible uniquement sur **Azure**.

Les régions prises en charge dépendent de la disponibilité des services requis par Logto dans la région Azure concernée. Si vous avez une région cible en tête, contactez-nous et nous pourrons confirmer la faisabilité selon vos besoins.

### Sécurité et conformité \{#security-and-compliance}

Le Cloud Privé offre un environnement dédié avec des frontières d’isolation plus claires qu’un déploiement cloud public mutualisé.

Nous abordons les questions de sécurité et de conformité de manière pragmatique et orientée besoins. Si votre équipe a besoin d’une revue de sécurité, d’une discussion sur la conformité ou d’éclaircissements spécifiques au déploiement, contactez-nous et nous pourrons examiner les détails ensemble.

## Contactez-nous \{#contact-us}

Si vous envisagez le Cloud Privé, [contactez-nous](https://logto.io/contact?src=private-cloud) avec :

- votre région Azure cible
- vos exigences en matière de sécurité ou de conformité
- le contexte de migration, le cas échéant

Nous vous aiderons à évaluer la pertinence et à discuter des prochaines étapes.
