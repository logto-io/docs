Logto fournit quatre modèles différents pour personnaliser le contenu des SMS, qui sont classés en fonction de leur type d'utilisation : Register, SignIn, ForgotPassword et Generic. Il est fortement recommandé d'utiliser différents modèles pour divers cas d'utilisation, sinon cela pourrait atteindre la limite de débit, entraînant une interruption temporaire de votre service.

## Types de modèles SMS et exemples

Voici quelques exemples à titre de référence :

| Usage          | Scénario                                                                                                                                                                     | Exemple de modèle                                                                                                        |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| Register       | Les utilisateurs créent un compte en utilisant leur e-mail et le vérifient en entrant un code de vérification envoyé par Logto à leur e-mail.                                | Votre code de vérification d'inscription Logto est `{{code}}`. Le code restera actif pendant 10 minutes.                 |
| SignIn         | Les utilisateurs se connectent en utilisant leur e-mail et vérifient en entrant un code de vérification au lieu d'entrer un mot de passe.                                    | Votre code de vérification de connexion Logto est `{{code}}`. Le code restera actif pendant 10 minutes.                  |
| ForgotPassword | Si les utilisateurs oublient leur mot de passe lors de la connexion, ils peuvent choisir de vérifier leur identité en utilisant l'e-mail qu'ils ont déjà vérifié avec Logto. | Votre code de vérification de changement de mot de passe Logto est `{{code}}`. Le code restera actif pendant 10 minutes. |
| Generic        | Ce modèle peut être utilisé comme option de secours générale pour divers scénarios, y compris les tests de configurations de connecteurs, etc.                               | Votre code de vérification Logto est `{{code}}`. Le code restera actif pendant 10 minutes.                               |

Il est important de comprendre ces paramètres :

- Le code de vérification est valable pendant 10 minutes. Nous ne supportons actuellement pas la personnalisation du temps d'expiration.
- Logto remplacera l'espace réservé `{{code}}` dans le modèle SMS par un code de vérification. Par conséquent, veuillez vous assurer que le modèle a un espace réservé réservé.

:::note
Certains pays et régions peuvent ne pas autoriser l'envoi de contenu non approuvé via SMS en raison des exigences de conformité. Les modèles SMS doivent être enregistrés et approuvés par le fournisseur de SMS avant de pouvoir être utilisés. Dans de tels cas, le contenu pourrait être indexé par ID de modèle vers le modèle correspondant.
:::
