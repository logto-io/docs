# Structure des données d’organisation

Au niveau de l'Organisation, il existe plusieurs attributs dont les valeurs peuvent être créées, mises à jour, lues et supprimées (CRUD).

## ID de l’Organisation

L'ID de l’Organisation est un identifiant unique pour chaque Organisation. Il est utile pour mettre en œuvre des expériences de connexion au niveau de l’Organisation et récupérer des jetons d’organisation.

## Nom

Le nom prend en charge la connexion au niveau de l’Organisation et peut être intégré dans les interfaces de produits au niveau de l’Organisation selon les besoins.

## Description

Le champ de description vous permet d'ajouter du texte pour aider à identifier et étiqueter l’Organisation.

## Logos de l’Organisation

Pour afficher dynamiquement le logo de l’Organisation de votre client dans l’Expérience de connexion, vous pouvez télécharger les logos de l’Organisation sur la page des paramètres de l’Organisation.

Voir [logos spécifiques à l’organisation](/customization/match-your-brand/#organization-specific-logos) pour plus de détails.

## Données personnalisées

Les données personnalisées sont un objet JSON utilisé pour stocker des informations supplémentaires sur l’Organisation. Cela donne aux développeurs la flexibilité de stocker des détails supplémentaires sur l’Organisation à des fins de développement.
