# Aperçu en direct

Vous pouvez utiliser la fonctionnalité **Aperçu en direct** pour tester facilement vos configurations d' expérience de connexion.

En naviguant vers <CloudLink to="/sign-in-experience">Console > Expérience de connexion</CloudLink> et en cliquant sur le bouton "Aperçu en direct" dans le coin supérieur droit, un nouvel onglet de navigateur s'ouvrira avec une démonstration entièrement fonctionnelle des capacités de Logto. Cette démonstration met en avant notre approche de conception d' expérience pour les flux de connexion et d'inscription. Elle est hautement personnalisable et peut être intégrée de manière transparente dans votre application.

:::note

**Remarque** : Dans cet aperçu, vous devez d'abord créer un compte pour tester l' expérience. Vous pouvez gérer ce compte plus tard dans <CloudLink to="/user-management">Console > Gestion des utilisateurs</CloudLink>.

:::

Si vous essayez de vous connecter avec le compte utilisateur que vous avez créé et que le processus réussit sans problème, vous serez redirigé vers une page de succès qui affiche vos identifiants et ID.
