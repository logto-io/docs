# Personnalisation

Adaptez Logto à votre image en personnalisant chaque aspect — de l'identité visuelle et l'interface utilisateur aux langues et aux paramètres de domaine.

## Interface préconstruite \{#prebuilt-ui}

Commencez avec nos composants d'interface préconstruits et personnalisez les éléments essentiels de votre marque. Créez soit :

- Une expérience d'authentification unifiée sur l'ensemble de votre plateforme.
- Des pages de connexion dédiées pour des applications ou organisations spécifiques.

  items={[

## Contenu & langues \{#content--languages}

Logto prend en charge 15 langues par défaut. Étendez la prise en charge linguistique ou modifiez les traductions existantes selon vos besoins.

  items={[
    {

## Domaine personnalisé \{#custom-domain}

Renforcez la présence de votre marque en utilisant votre propre domaine pour les URLs d’authentification. Remplacez le domaine Logto par défaut par votre domaine personnalisé.

  items={[
    {
      type: 'link',
      label: 'Domaine personnalisé',
      href: '/logto-cloud/custom-domain',

      customProps: {
        icon: <UserFlowIcon />,
      },
    },
  ]}
/>
