# Personnalisation

Faites de Logto votre propre outil en personnalisant chaque aspect - de l'image de marque et de l'interface utilisateur aux langues et aux paramètres de domaine.

## Interface utilisateur préconstruite

Commencez avec nos composants d'interface utilisateur préconstruits et personnalisez les éléments de marque essentiels. Créez soit :

- Une expérience d'authentification unifiée sur votre plateforme.
- Des pages de connexion dédiées pour des applications ou organisations spécifiques.

<DocCardList
  items={[
    {

## Contenu & Langues

Logto prend en charge 15 langues par défaut. Étendez le support linguistique ou modifiez les traductions existantes pour répondre à vos besoins.

## Domaine personnalisé

Renforcez la présence de votre marque en utilisant votre domaine pour les URL d'authentification. Remplacez le domaine par défaut de Logto par votre domaine de marque.

  items={[
    {
      type: 'link',
      label: 'Domaine personnalisé',
      href: '/logto-cloud/custom-domain',

      customProps: {
        icon: <UserFlowIcon />,

    },
  ]}
/>
