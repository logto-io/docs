# Installer Logto

## Interactif \{#interactive}

Exécutez la commande ci-dessous pour procéder à une installation interactive dans votre terminal :

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## Options et installation silencieuse \{#options-and-silent-installation}

Alternativement, vous pouvez ajouter les options ci-dessous pour ignorer certaines questions :

```bash
# Toutes ces options sont facultatives
--db-url      L'URL Postgres vers la base de données Logto
-p, --path    Chemin vers votre instance Logto, doit être un chemin inexistant
--ss          Ignorer l'ensemencement de la base de données Logto
--oc          Ajouter des connecteurs officiels après l'installation
```

Exécutez `logto init --help` pour obtenir des informations complètes d'aide.

Voici un exemple pour effectuer une installation silencieuse (par exemple, dans CI) :

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

Cela fera les choses suivantes dans l'ordre :

1. Télécharger et extraire Logto dans `./logto`
2. Essayer d'initialiser et d'ensemencer la base de données avec l'URL `postgresql://your-postgres-dsn:5432/logto`
3. Ajouter les connecteurs officiels Logto à `./logto/packages/core/connectors`
