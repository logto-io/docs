# Configurer les permissions

## Configurer les permissions API via Logto Console \{#configure-api-permissions-via-logto-console}

Les permissions sont définies au niveau de **la ressource API**. Accédez à la <CloudLink to="/api-resources">Console > Ressources API</CloudLink>, cliquez sur la ressource API que vous souhaitez gérer, puis allez à l'onglet "Permissions".

Il est important de fournir un nom et des détails pour la permission, car cela facilitera l'identification et la gestion des permissions.

## Configurer les permissions API via Logto Management API \{#configure-api-permissions-via-logto-management-api}

Gérez en utilisant le Logto Management API. Effectuez un appel au point de terminaison relatif. Consultez cette [référence](https://openapi.logto.io/operation/operation-listresourcescopes).

| méthode | chemin                                                                                                               | description                                                                                                           |
| ------- | -------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| GET     | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-listresourcescopes)              | Obtenez les portées définies pour une ressource API.                                                                  |
| POST    | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-createresourcescope)             | Créez une nouvelle portée pour une ressource API.                                                                     |
| DELETE  | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-deleteresourcescope) | Supprimez une portée de ressource API de la ressource donnée.                                                         |
| PATCH   | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-updateresourcescope) | Mettez à jour une portée de ressource API pour la ressource donnée. Cette méthode effectue une mise à jour partielle. |
