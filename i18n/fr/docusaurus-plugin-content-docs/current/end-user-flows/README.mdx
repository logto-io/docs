# Parcours utilisateur final

Les parcours utilisateur final couvrent tous les processus de vérification lors des interactions utilisateur, classés comme suit :

- **Parcours d’authentification (Authentication flows)** : Impliquent la connexion, l’inscription et la réinitialisation du mot de passe de l’utilisateur. Logto propose des expériences prêtes à l’emploi configurées dans <CloudLink to="/sign-in-experience">Console > Expérience de connexion</CloudLink>, et vous pouvez [apporter votre propre interface](/customization/bring-your-ui) via les [Experience APIs](https://openapi.logto.io/group/endpoint-experience).
- **Parcours de compte (Account flows)** : Impliquent les paramètres du compte, le profil utilisateur et la vérification de sécurité. Utilisez les [Account APIs](https://openapi.logto.io/group/endpoint-account-center) ou les [Management APIs](https://openapi.logto.io/operation/operation-getuser) pour les mettre en œuvre.
- **Parcours d’organisation (Organization flows)** : Pour les produits multi-locataires, comme les services B2B ou SaaS. Impliquent la création d’organisation, l’invitation de membres et la gestion. Utilisez les [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) pour intégrer cela dans vos applications.
