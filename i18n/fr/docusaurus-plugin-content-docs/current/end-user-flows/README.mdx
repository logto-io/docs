# Flux des utilisateurs finaux

Les flux des utilisateurs finaux couvrent tous les processus de vérification pour les interactions des utilisateurs, classés comme suit :

- **Expérience de connexion** : Implique la connexion de l'utilisateur, l'inscription et la réinitialisation du mot de passe. Logto offre des expériences prêtes à l'emploi configurées dans Console, et vous pouvez apporter votre interface utilisateur via les Experience APIs.
- **Expérience de compte** : Implique les paramètres du compte, le profil utilisateur et la vérification de sécurité. Utilisez les Account APIs ou les Management APIs pour implémenter.
- **Expérience d’organisation** : Pour les produits multi-locataires, comme les services B2B ou SaaS. Implique la création d’Organisation, l’invitation de membres et la gestion. Utilisez les Management APIs pour intégrer cela dans vos applications.
