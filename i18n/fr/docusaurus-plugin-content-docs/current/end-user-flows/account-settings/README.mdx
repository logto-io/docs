# Paramètres du compte

Logto fournit deux collections d'APIs de paramètres de compte pour permettre aux utilisateurs de gérer leur compte et leurs profils stockés dans Logto.

## APIs de gestion des utilisateurs

Les Management APIs de Logto offrent la fonctionnalité la plus complète pour gérer les utilisateurs et leurs données de profil. Vous pouvez implémenter un système de gestion des utilisateurs auto-hébergé ou une page de paramètres de compte pour les utilisateurs finaux en utilisant ces APIs.

## APIs de compte

En plus des Management APIs, Logto fournit également un ensemble d'APIs de compte pour les utilisateurs finaux afin de leur permettre de gérer directement les paramètres de leur compte et les données de leur profil. Contrairement aux Management APIs, les APIs de compte sont conçues pour être utilisées dans le contexte de l'utilisateur final avec un contrôle détaillé des permissions utilisateur. Vous pouvez utiliser les APIs de compte directement dans votre application côté client pour créer une page de paramètres de compte conviviale.
