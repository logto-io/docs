# Paramètres du compte

Logto fournit deux collections d'APIs de paramètres de compte pour permettre aux utilisateurs de gérer leur compte et leurs profils stockés dans Logto.

## APIs de gestion des utilisateurs

Les APIs de gestion de Logto offrent la fonctionnalité la plus complète pour gérer les utilisateurs et leurs données de profil. Vous pouvez implémenter un système de gestion des utilisateurs auto-hébergé ou une page de paramètres de compte utilisateur final en utilisant ces APIs.

En savoir plus sur comment utiliser les [APIs de gestion des utilisateurs](/end-user-flows/account-settings/by-management-api) pour créer votre propre page de paramètres de compte.

## APIs de compte

En plus des APIs de gestion, Logto fournit également un ensemble d'APIs de compte utilisateur final pour permettre aux utilisateurs finaux de gérer directement leurs paramètres de compte et leurs données de profil. Contrairement aux APIs de gestion, les APIs de compte sont conçues pour être utilisées dans le contexte de l'utilisateur final avec un contrôle détaillé des permissions utilisateur. Vous pouvez utiliser les APIs de compte directement dans votre application côté client pour créer une page de paramètres de compte conviviale.

En savoir plus sur comment utiliser les [APIs de compte](/end-user-flows/account-settings/by-account-api) pour créer votre propre page de paramètres de compte.
