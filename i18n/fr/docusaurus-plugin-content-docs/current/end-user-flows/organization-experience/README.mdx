# Expérience d’organisation

L’[expérience d’organisation](/organizations) fait référence aux interfaces utilisateur et aux flux utilisés par vos clients professionnels et leurs employés, en particulier dans les [applications multi-locataires](https://auth.wiki/multi-tenancy). Logto propose plusieurs guides pour vous aider à intégrer rapidement cette expérience dans votre application à l’aide de Logto Management API.
