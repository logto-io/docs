# Expérience d’organisation

L’[expérience d’organisation](/organizations) est l’ensemble des interfaces et des flux que vos clients professionnels et leurs employés utilisent — en particulier dans les [applications multi-locataires](https://auth.wiki/multi-tenancy). Ce guide montre comment l’intégrer à votre application en utilisant le Management API de Logto.

Cette section vous aide à concevoir l’**expérience d’organisation** pour vos utilisateurs finaux — par exemple :

1. Les administrateurs peuvent créer leurs propres organisations.
2. Les administrateurs peuvent gérer les membres de l’organisation.
3. Les administrateurs peuvent inviter des membres à rejoindre leurs organisations.
4. et plus encore.

## Comprendre le flux d’authentification \{#understand-the-authentication-flow}

```mermaid
sequenceDiagram
  actor Utilisateur
  participant Frontend
  participant Backend
  participant Logto

  Utilisateur->>Frontend: Demande de création d’organisation
  Frontend->>Backend: POST /organizations
  Backend->>Logto: POST /api/organizations
  Logto-->>Backend: Organisation créée
  Backend->>Logto: POST /api/organizations/{id}/users
  Logto-->>Backend: Utilisateur ajouté à l’organisation
  Backend-->>Frontend: Réponse de succès
  Frontend-->>Utilisateur: Afficher le message de succès
```

Pour intégrer le Management API de Logto, commencez par comprendre le flux d’authentification de base. Il comporte deux exigences clés :

### Protégez votre API backend \{#protect-your-backend-api}

- Les appels du frontend à votre API backend nécessitent une authentification.
- Protégez les points de terminaison de l’API en validant le jeton d’accès Logto de l’utilisateur.
- Assurez-vous que seuls les utilisateurs authentifiés peuvent accéder à vos services.

### Accédez au Management API de Logto \{#access-the-logto-management-api}

- Votre service backend appelle en toute sécurité le Management API de Logto.
- Suivez le guide [Interagir avec le Management API](/integrate-logto/interact-with-management-api) pour la configuration.
- Utilisez l’authentification machine à machine pour obtenir les identifiants d’accès.

Les prochains chapitres expliquent comment configurer le Management API de Logto et présentent les cas d’usage courants pour construire votre expérience d’organisation.
