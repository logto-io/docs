# Expérience d'organisation

L'expérience d'[Organisation](/organizations) fait référence aux interfaces utilisateur et aux flux utilisés par vos clients professionnels et leurs employés, en particulier dans les [applications multi-locataires](https://auth.wiki/multi-tenancy). Logto fournit plusieurs guides pour vous aider à intégrer rapidement cette expérience dans votre application en utilisant le Logto Management API.
