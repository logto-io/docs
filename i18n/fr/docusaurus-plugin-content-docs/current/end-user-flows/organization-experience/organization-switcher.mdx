# Sélecteur d’organisation

Dans l’Expérience (Experience) d’organisation, les utilisateurs peuvent passer d’une organisation à une autre. À chaque changement, l’application récupère un nouveau jeton d’organisation (Organization token) pour charger les informations pertinentes.

<img src="/img/assets/organization-switcher.png" alt="Sélecteur d’organisation" />

## Comment l’implémenter \{#how-to-implement-it}

1. Créez un menu déroulant ou un composant d’interface similaire pour lister toutes les organisations disponibles pour l’utilisateur.
2. Lorsque l’utilisateur sélectionne une autre organisation, récupérez son ID et appelez la méthode SDK `getOrganizationToken(organizationId)` pour obtenir le jeton d’organisation (Organization token) correspondant. (Le SDK gère la mise en cache des jetons pour vous.)
3. Stockez l’ID de l’organisation sélectionnée (dans l’URL ou le stockage client), et utilisez-le toujours pour récupérer le jeton d’organisation (Organization token) associé aux ressources spécifiques à l’organisation.
