# Introduction à Logto

Bienvenue dans la documentation de Logto ! Logto est une solution de gestion des identités et des accès (IAM) conçue pour les applications modernes et les produits SaaS. Elle fournit un système d'authentification et d'Autorisation sécurisé, évolutif et personnalisable pour vos applications.

## Explorer par fonctionnalités \{#explore-by-features}

Logto offre une suite robuste de fonctionnalités, y compris l'Authentification, l'Autorisation, et la gestion des utilisateurs et des Organisations, adaptées aux besoins modernes en matière d'identité. Notre solide support CIAM (Customer Identity and Access Management) lui confère un avantage clair dans le domaine.
