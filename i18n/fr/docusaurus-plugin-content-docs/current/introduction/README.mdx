# Introduction

Bienvenue dans la documentation Logto ! Logto est une solution de gestion des identités et des accès (IAM) conçue pour les applications modernes et les produits SaaS. Elle fournit un système d’authentification (Authentication) et d’autorisation (Authorization) sécurisé, évolutif et personnalisable pour vos applications.
