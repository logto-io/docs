# Introduction à Logto

Bienvenue dans la documentation de Logto ! Logto est une solution de gestion des identités et des accès (IAM) conçue pour les applications modernes et les produits SaaS. Elle offre un système d'authentification et d'autorisation sécurisé, évolutif et personnalisable pour vos applications.
