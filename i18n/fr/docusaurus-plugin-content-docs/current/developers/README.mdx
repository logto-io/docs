# Développeur

Logto est un service de [gestion des identités et des accès (IAM)](https://auth.wiki/iam) basé sur les protocoles [OAuth 2](https://auth.wiki/oauth-2.0) et [OIDC](https://auth.wiki/openid-connect). Les services IAM comme Logto servent souvent de fondation à d'autres services web ; divers états d'autorisation au sein de ces services web sont directement affectés par Logto.

Afin de fournir de la commodité à nos utilisateurs, Logto offre une série de fonctionnalités couramment utilisées par les développeurs.
