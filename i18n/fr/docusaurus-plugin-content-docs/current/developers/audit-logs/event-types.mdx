# Types d’événements des journaux d’audit

Vous pouvez filtrer les types d’événements dans <CloudLink to="/audit-logs">Logto Console > Journaux d’audit</CloudLink>.

:::note

Logto prend désormais en charge la récupération des journaux liés aux interactions des utilisateurs finaux via les [Experience APIs](https://openapi.logto.io/group/endpoint-experience).

Les journaux d’audit pour les [Management APIs](/integrate-logto/interact-with-management-api) et les [Account APIs](https://openapi.logto.io/group/endpoint-account-center) arrivent bientôt.

N’hésitez pas à [nous contacter](https://logto.io/contact) si vous souhaitez partager vos besoins.

:::
