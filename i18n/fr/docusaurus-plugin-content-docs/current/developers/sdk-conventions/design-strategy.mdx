# Stratégie de conception

- Chaque langage de programmation doit avoir un dépôt git isolé nommé `${language}`.
- Chaque dépôt de langage de programmation doit être un mono repo.
- Les SDKs et leurs projets d'exemple associés doivent être placés sous ce dépôt.

Exemples :
