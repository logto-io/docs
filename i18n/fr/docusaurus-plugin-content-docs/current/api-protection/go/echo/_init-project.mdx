Pour initialiser un nouveau projet Go avec Echo, vous pouvez suivre ces étapes :

```bash
go mod init your-api-name
go get github.com/labstack/echo/v4
```

Ensuite, créez une configuration de serveur Echo basique :

```go title="main.go"
package main

import (
    "github.com/labstack/echo/v4"
)

func main() {
    e := echo.New()

    e.Logger.Fatal(e.Start(":3000"))
}
```

:::note
Consultez la documentation Echo pour plus de détails sur la configuration des routes, des middlewares et d'autres fonctionnalités.
:::
