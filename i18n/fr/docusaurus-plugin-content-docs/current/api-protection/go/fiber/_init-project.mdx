Pour initialiser un nouveau projet Go avec Fiber, vous pouvez suivre ces étapes :

```bash
go mod init your-api-name
go get github.com/gofiber/fiber/v2
```

Ensuite, créez une configuration de serveur Fiber basique :

```go title="main.go"
package main

import (
    "log"

    "github.com/gofiber/fiber/v2"
)

func main() {
    app := fiber.New()

    log.Fatal(app.Listen(":3000"))
}
```

:::note
Consultez la documentation Fiber pour plus de détails sur la configuration des routes, des middlewares et d'autres fonctionnalités.
:::
