Pour initialiser un nouveau projet Go avec Chi, vous pouvez suivre ces étapes :

```bash
go mod init your-api-name
go get github.com/go-chi/chi/v5
```

Ensuite, créez une configuration de serveur Chi basique :

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
Consultez la documentation de Chi pour plus de détails sur la configuration des routes, des middlewares et d'autres fonctionnalités.
:::
