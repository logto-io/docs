Pour initialiser un nouveau projet Grape API, créez un répertoire et mettez en place la structure de base :

```bash
mkdir your-api-name
cd your-api-name
```

Créez un fichier Gemfile :

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

Installez les dépendances :

```bash
bundle install
```

Créez une API Grape basique :

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

Créez un fichier config.ru :

```ruby title="config.ru"
require_relative 'api'

run API
```

Démarrez le serveur de développement :

```bash
bundle exec rackup
```

:::note
Consultez la documentation de Grape pour plus de détails sur la mise en place des ressources, la validation des paramètres et d'autres fonctionnalités.
:::
