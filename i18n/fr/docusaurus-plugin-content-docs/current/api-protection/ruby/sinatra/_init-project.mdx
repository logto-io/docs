Pour initialiser un nouveau projet Sinatra, créez un répertoire et mettez en place la structure de base :

```bash
mkdir your-api-name
cd your-api-name
```

Créez un fichier Gemfile :

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'sinatra'
```

Installez les dépendances :

```bash
bundle install
```

Créez une application Sinatra basique :

```ruby title="app.rb"
require 'sinatra'
require 'json'

get '/' do
  content_type :json
  { message: 'Hello from Sinatra API' }.to_json
end
```

Démarrez le serveur de développement :

```bash
ruby app.rb
```

:::note
Consultez la documentation de Sinatra pour plus de détails sur la configuration des routes, des middlewares et d'autres fonctionnalités.
:::
