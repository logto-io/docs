Pour initialiser un nouveau projet Node.js avec NestJS, vous pouvez utiliser le CLI Nest pour une configuration rapide. Voici les étapes :

```bash
npm i -g @nestjs/cli
nest new my-api
cd my-api
```

Alternativement, vous pouvez créer un projet manuellement :

```bash
npm init -y
npm install @nestjs/core @nestjs/common @nestjs/platform-express reflect-metadata rxjs
```

Si vous utilisez TypeScript, n'oubliez pas de configurer votre environnement TypeScript en conséquence.

Ensuite, créez une configuration de serveur NestJS basique :

```ts title="main.ts"
import { NestFactory } from '@nestjs/core';
import { AppModule } from './app.module';

async function bootstrap() {
  const app = await NestFactory.create(AppModule);
  await app.listen(3000);
  console.log('Server running on http://localhost:3000');
}
bootstrap();
```

```ts title="app.module.ts"
import { Module } from '@nestjs/common';
import { AppController } from './app.controller';
import { AppService } from './app.service';

@Module({
  imports: [],
  controllers: [AppController],
  providers: [AppService],
})
export class AppModule {}
```

:::note
Consultez la documentation NestJS pour plus de détails sur la configuration des contrôleurs, services et autres fonctionnalités. Vous pouvez également explorer les commandes du CLI NestJS pour générer facilement des composants et des modules.
:::
