Pour initialiser un nouveau projet Node.js avec Koa, vous pouvez suivre ces étapes :

```bash
npm init -y
npm install koa @koa/router
```

Si vous utilisez TypeScript, n'oubliez pas de configurer votre environnement TypeScript en conséquence.

Ensuite, créez une configuration de serveur Koa basique :

```ts title="app.ts"
import Koa from 'koa';

const app = new Koa();

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});
```

:::note
Consultez la documentation de Koa pour plus de détails sur la configuration des routes, des middlewares et d'autres fonctionnalités.
:::
