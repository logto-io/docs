Pour initialiser un nouveau projet Node.js avec Fastify, vous pouvez suivre ces étapes :

```bash
npm init -y
npm install fastify
```

Si vous utilisez TypeScript, n'oubliez pas de configurer votre environnement TypeScript en conséquence.

Ensuite, créez une configuration de serveur Fastify basique :

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Server running on http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
Consultez la documentation Fastify pour plus de détails sur la configuration des routes, des plugins et d'autres fonctionnalités.
:::
