Pour initialiser un nouveau projet Node.js avec Hapi, vous pouvez suivre ces étapes :

```bash
npm init -y
npm install @hapi/hapi
```

Si vous utilisez TypeScript, n'oubliez pas de configurer votre environnement TypeScript en conséquence.

Ensuite, créez une configuration de serveur Hapi basique :

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Server running on http://localhost:3000');
```

:::note
Consultez la documentation Hapi pour plus de détails sur la configuration des routes, des plugins et d'autres fonctionnalités.
:::
