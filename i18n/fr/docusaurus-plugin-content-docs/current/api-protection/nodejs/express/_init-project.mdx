Pour initialiser un nouveau projet Node.js avec Express, vous pouvez suivre ces étapes :

```bash
npm init -y
npm install express
```

Si vous utilisez TypeScript, n'oubliez pas de configurer votre environnement TypeScript en conséquence.

Ensuite, créez une configuration de serveur Express basique :

```ts title="app.ts"
import express from 'express';

const app = express();

app.listen(3000, () => {
  console.log('Serveur en cours d’exécution sur http://localhost:3000');
});
```

:::note
Consultez la documentation Express pour plus de détails sur la configuration des routes, des middlewares et d'autres fonctionnalités.
:::
