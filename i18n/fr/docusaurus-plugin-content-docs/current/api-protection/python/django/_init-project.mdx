Pour initialiser un nouveau projet Django, vous pouvez utiliser les commandes intégrées de Django :

```bash
django-admin startproject your_api_name
cd your_api_name
```

Installez Django si ce n'est pas déjà fait :

```bash
pip install Django
```

Créez une application Django de base :

```bash
python manage.py startapp api
```

Créez une vue API basique :

```py title="api/views.py"
from django.http import JsonResponse

def hello_view(request):
    return JsonResponse({"message": "Hello from Django"})
```

Ajoutez la configuration des URL :

```py title="api/urls.py"
from django.urls import path
from . import views

urlpatterns = [
    path('', views.hello_view, name='hello'),
]
```

```py title="your_api_name/urls.py"
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('api/', include('api.urls')),
]
```

Démarrez le serveur de développement :

```bash
python manage.py runserver
```

:::note
Consultez la documentation Django pour plus de détails sur la configuration des modèles, des vues et d'autres fonctionnalités.
:::
