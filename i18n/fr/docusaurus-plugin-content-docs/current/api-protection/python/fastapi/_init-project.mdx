Pour initialiser un nouveau projet FastAPI, créez un répertoire et mettez en place la structure de base :

```bash
mkdir your-api-name
cd your-api-name
```

Créez un fichier de dépendances :

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

Installez les dépendances :

```bash
pip install -r requirements.txt
```

Créez une application FastAPI basique :

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

Démarrez le serveur de développement :

```bash
uvicorn main:app --reload
```

:::note
Consultez la documentation FastAPI pour plus de détails sur la configuration des opérations de chemin, l'injection de dépendances et d'autres fonctionnalités.
:::
