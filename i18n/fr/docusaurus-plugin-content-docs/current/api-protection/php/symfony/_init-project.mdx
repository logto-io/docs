Pour initialiser un nouveau projet Symfony pour le développement d’API, utilisez le CLI Symfony ou Composer :

Avec le CLI Symfony (recommandé) :

```bash
symfony new your-api-name --webapp
cd your-api-name
```

Ou avec Composer :

```bash
composer create-project symfony/skeleton your-api-name
cd your-api-name
composer require webapp
```

Installez des paquets supplémentaires pour le développement d’API :

```bash
composer require symfony/security-bundle
composer require symfony/serializer
composer require doctrine/annotations
```

Démarrez le serveur de développement :

```bash
symfony serve
```

Ou en utilisant le serveur intégré de PHP :

```bash
php -S localhost:8000 -t public/
```

Cela crée un projet Symfony de base. Configurez le framework pour le développement d’API :

```yaml title="config/packages/framework.yaml"
framework:
  secret: '%env(APP_SECRET)%'
  serializer:
    enabled: true
  property_access:
    enabled: true
```

:::note
Consultez la documentation Symfony pour plus de détails sur la configuration des contrôleurs, services et autres fonctionnalités.
:::
