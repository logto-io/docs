# Protection d’API

Découvrez comment mettre en œuvre le contrôle d’accès basé sur les rôles (RBAC) et la validation de JSON Web Token (JWT) pour protéger vos points de terminaison d’API contre les accès non autorisés.

Nos guides couvrent la configuration du middleware, les modèles de permissions et des exemples pratiques dans plusieurs langages de programmation et frameworks.

## Commencer \{#get-started}

### Nouveau dans la sécurité des API ? \{#new-to-api-security}

Si vous débutez avec l’authentification et l’autorisation d’API, commencez par ces notions fondamentales :

- **[Authentification vs. autorisation](/concepts/authn-vs-authz)** — Comprenez les différences clés
- **[Contrôle d’accès basé sur les rôles (RBAC)](/authorization/role-based-access-control)** — Apprenez comment fonctionnent les permissions et les rôles
- **[JSON Web Token (JWT)](https://auth.wiki/jwt)** — Découvrez comment les JSON Web Tokens sécurisent vos API
