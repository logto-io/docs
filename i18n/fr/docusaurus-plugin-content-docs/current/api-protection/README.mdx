# Protégez vos API avec Logto

Découvrez comment mettre en œuvre le [contrôle d’accès basé sur les rôles (RBAC)](/authorization/role-based-access-control) et la validation de [JSON Web Token (JWT)](https://auth.wiki/jwt) pour protéger vos points de terminaison API contre les accès non autorisés.

Nos guides couvrent la configuration du middleware, les modèles de permissions et des exemples pratiques dans plusieurs langages de programmation et frameworks.
