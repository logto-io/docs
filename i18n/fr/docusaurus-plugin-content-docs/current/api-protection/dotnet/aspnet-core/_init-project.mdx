Pour initialiser un nouveau projet .NET Web API, vous pouvez utiliser le CLI .NET :

```bash
dotnet new webapi -n YourApiName
cd YourApiName
```

Ajoutez le package NuGet requis pour l’authentification JWT :

```bash
dotnet add package Microsoft.AspNetCore.Authentication.JwtBearer
```

Créez un contrôleur API de base :

```csharp title="Controllers/ApiController.cs"
using Microsoft.AspNetCore.Mvc;

namespace YourApiName.Controllers
{
    [ApiController]
    [Route("api/[controller]")]
    public class ApiController : ControllerBase
    {
        [HttpGet]
        public IActionResult Get()
        {
            return Ok(new { message = "Hello from .NET API" });
        }
    }
}
```

Démarrez le serveur de développement :

```bash
dotnet run
```

:::note
Consultez la documentation ASP.NET Core pour plus de détails sur la configuration des contrôleurs, du middleware et d’autres fonctionnalités.
:::
