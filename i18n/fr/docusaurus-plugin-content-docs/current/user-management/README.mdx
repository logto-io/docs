# Gestion des utilisateurs

La gestion des utilisateurs est un domaine clé pour Logto. Vous pouvez accéder à vos données utilisateur et les gérer en utilisant soit la Console Logto, soit le Management API, deux outils efficaces. Avec ces outils, vous pouvez effectuer des tâches telles que

- Rechercher des utilisateurs par nom d'utilisateur, ID, nom complet, numéro de téléphone ou e-mail.
- Effectuer plusieurs actions sur l'utilisateur, telles que dissocier des comptes sociaux, mettre à jour les informations du profil, supprimer des utilisateurs ou réinitialiser les mots de passe.
- Consulter les journaux des utilisateurs.
- Attribuer des rôles pour le contrôle d'accès.
- Créer et gérer des jetons d’accès personnels.
- Gérer les sessions utilisateur.
