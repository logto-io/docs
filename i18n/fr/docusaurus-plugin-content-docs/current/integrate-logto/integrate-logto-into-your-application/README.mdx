# Intégrer Logto dans votre application

Suivez ces étapes pour ajouter l'authentification à vos applications avec Logto, qu'il s'agisse d'une application destinée aux utilisateurs ou d'un service machine à machine :

1. Accédez à <CloudLink to="/applications">Console > Applications</CloudLink>

2. Cliquez sur "Créer une application" pour ajouter une nouvelle application

3. Choisissez le framework de votre application pour commencer. Si vous ne trouvez pas votre framework, cliquez sur le bouton "Créer une application sans framework" en bas à droite de la page de création d'application pour créer une application en sélectionnant un [Type d'application](/integrate-logto/application-data-structure/#application-types) ou soumettez une demande de fonctionnalité ou contribuez à un SDK en suivant nos [Conventions SDK](/developers/sdk-conventions).

4. Après avoir sélectionné votre framework, vous verrez un guide de démarrage rapide pour le SDK du framework. Suivez les étapes pour configurer et intégrer votre application. Si vous avez besoin d'aide pour comprendre les concepts impliqués dans le processus d'intégration, vous pouvez vous référer à [Comprendre le flux d'authentification Logto](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) pour une compréhension plus approfondie de l'intégration.

:::note
Le guide dans la console est uniquement pour un démarrage rapide avec Logto en utilisant notre SDK. Pour des guides d'intégration complets, y compris l'utilisation avancée du SDK, consultez la section [Démarrages rapides](/quick-starts).
:::

5. Une fois terminé, vous êtes prêt à explorer davantage Logto :
