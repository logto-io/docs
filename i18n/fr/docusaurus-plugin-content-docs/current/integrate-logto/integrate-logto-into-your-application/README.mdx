# Intégrer Logto dans votre application

Suivez ces étapes pour ajouter l’authentification à vos applications avec Logto, qu’il s’agisse d’une application destinée aux utilisateurs ou d’un service machine à machine :

1. Accédez à <CloudLink to="/applications">Console > Applications</CloudLink>

2. Cliquez sur « Créer une application » pour ajouter une nouvelle application

3. Choisissez votre [framework d’application](/quick-starts) pour commencer. Si vous ne trouvez pas votre framework, cliquez sur le bouton « Créer une application sans framework » en bas à droite de la page de création d’application pour créer une application en sélectionnant un [type d’application](/integrate-logto/application-data-structure/#application-types) ou soumettez une demande de fonctionnalité ou contribuez à un SDK en suivant nos [conventions SDK](/developers/sdk-conventions).

4. Après avoir sélectionné votre framework, vous verrez un guide de démarrage rapide pour le SDK du framework. Suivez les étapes pour configurer et intégrer votre application. Si vous avez besoin d’aide pour comprendre les concepts impliqués dans le processus d’intégration, vous pouvez consulter [Comprendre le flux d’authentification Logto](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) pour approfondir votre compréhension de l’intégration.

:::note
Le guide dans la console sert uniquement à un démarrage rapide avec Logto en utilisant notre SDK. Pour des guides d’intégration complets, y compris l’utilisation avancée du SDK, consultez la section [Guides de démarrage rapide](/quick-starts).

Le guide de démarrage rapide montre principalement comment implémenter la connexion. Si vous souhaitez accéder directement à l’inscription, au mot de passe oublié ou à des méthodes d’authentification spécifiques comme l’inscription par e-mail ou la connexion sociale, consultez la documentation [Paramètres d’authentification](/end-user-flows/authentication-parameters).
:::

5. Une fois terminé, vous êtes prêt à explorer davantage Logto :

   <Url href="/organizations">Organisations (Organizations)</Url>
