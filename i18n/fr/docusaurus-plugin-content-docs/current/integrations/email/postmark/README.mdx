# Configurer la vérification par e-mail avec le service de messagerie Postmark

Connecteur Logto pour le service de messagerie Postmark.

<GuideTip />

## Commencer \{#get-started}

Postmark est une plateforme de messagerie pour les e-mails transactionnels et marketing. Nous pouvons utiliser sa fonction d'envoi d'e-mails pour envoyer un _code de vérification_.

## Enregistrer un compte Postmark \{#register-postmark-account}

Créez un nouveau compte sur le [site Web de Postmark](https://postmark.com/). Vous pouvez passer cette étape si vous avez déjà un compte.

## Configurer votre connecteur \{#configure-your-connector}

Remplissez le champ `serverToken` avec le jeton de serveur que vous trouvez dans les paramètres de votre serveur dans Postmark.

Remplissez le champ `fromEmail` avec l'adresse _From Address_ de l'expéditeur.

Pour activer les flux utilisateur complets, des modèles avec usageType `Register`, `SignIn`, `ForgotPassword` et `Generic` sont requis.

Voici un exemple de modèle JSON de connecteur Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Tester le connecteur de messagerie Postmark \{#test-postmark-email-connector}

Vous pouvez saisir une adresse e-mail et cliquer sur "Envoyer" pour vérifier si les paramètres fonctionnent avant de cliquer sur "Enregistrer et Terminer".

C'est tout. N'oubliez pas de <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">Activer le connecteur dans l'expérience de connexion</MainSiteUrl>.

## Types de configuration \{#config-types}

| Nom         | Type   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propriétés du modèle | Type        | Valeurs énumérées                                       |
| -------------------- | ----------- | ------------------------------------------------------- |
| usageType            | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias        | string      | N/A                                                     |
