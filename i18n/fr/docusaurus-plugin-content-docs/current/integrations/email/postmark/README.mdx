# Configurer la vérification par e-mail avec le service de messagerie Postmark

Connecteur Logto pour le service de messagerie Postmark.

<GuideTip />

## Commencer

Postmark est une plateforme de messagerie pour les e-mails transactionnels et marketing. Nous pouvons utiliser sa fonction d'envoi d'e-mails pour envoyer un _code de vérification_.

## Enregistrer un compte Postmark

Créez un nouveau compte sur le [site Web de Postmark](https://postmark.com/). Vous pouvez passer cette étape si vous avez déjà un compte.

## Configurer votre connecteur

Remplissez le champ `serverToken` avec le Jeton de Serveur que vous trouvez dans les paramètres de votre serveur dans Postmark.

Remplissez le champ `fromEmail` avec l' _Adresse de l'expéditeur_.

Afin d'activer les flux utilisateur complets, des modèles avec usageType `Register`, `SignIn`, `ForgotPassword` et `Generic` sont requis.

Voici un exemple de modèle de connecteur Postmark en JSON.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Tester le connecteur de messagerie Postmark

Vous pouvez taper une adresse e-mail et cliquer sur "Envoyer" pour voir si les paramètres fonctionnent avant "Enregistrer et Terminer".

C'est tout. N'oubliez pas de [Activer le connecteur dans l'expérience de connexion](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)

## Types de configuration

| Nom         | Type   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propriétés du modèle | Type        | Valeurs énumérées                                       |
| -------------------- | ----------- | ------------------------------------------------------- |
| usageType            | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias        | string      | N/A                                                     |
