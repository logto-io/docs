# Configurer la vérification par e-mail avec AWS Direct Mail

Le connecteur officiel Logto pour le service de messagerie directe AWS.

<GuideTip />

## Commencer \{#get-started}

Amazon SES est un fournisseur de services de messagerie cloud qui peut s'intégrer à toute application pour l'envoi d'e-mails en masse.

L'équipe Logto appelle les API Amazon Simple Email Service, grâce auxquelles les utilisateurs finaux de Logto peuvent s'inscrire et se connecter à leur compte Logto via un code de vérification par e-mail.
