## Configurer un service de messagerie dans la console de service AWS \{#configure-a-mail-service-in-the-aws-service-console}

### Enregistrer un compte AWS \{#register-aws-account}

Rendez-vous sur [AWS](https://aws.amazon.com/) et créez un compte.

### Créer une identité \{#create-a-identity}

- Accédez à la console `Amazon Simple Email Service`
- Créez une identité, choisissez l'une des options suivantes :
  - Créer un domaine
  - Créer une adresse e-mail

### Configuration du connecteur \{#configuration-of-the-connector}

1. Cliquez sur votre nom d'utilisateur dans le coin supérieur droit de la console Amazon pour accéder à `Security Credentials`. Si vous n'en avez pas, créez une `AccessKey` et conservez-la soigneusement.
2. Complétez les paramètres du connecteur `Amazon Simple Email Service` :
   - Utilisez l'`AccessKey ID` et l'`AccessKey Secret` obtenus à l'étape 1 pour remplir respectivement `accessKeyId` et `accessKeySecret`.
   - `region` : Renseignez le champ `region` avec la région de l'identité que vous utilisez pour envoyer des e-mails.
   - `emailAddress` : L'adresse e-mail que vous utilisez pour envoyer des e-mails, au format `Logto\<noreply@logto.io>` ou `\<noreply@logto.io>`

Les paramètres suivants sont optionnels ; la description des paramètres est disponible dans la [documentation de l’API AWS SES](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html).

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### Tester le connecteur Amazon SES \{#test-the-amazon-ses-connector}

Vous pouvez saisir une adresse e-mail et cliquer sur "Envoyer" pour vérifier si les paramètres fonctionnent avant de cliquer sur "Enregistrer et terminer".

C'est tout. N'oubliez pas de <MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">Activer le connecteur dans l'inscription et la connexion</MainSiteUrl>.

### Types de configuration \{#configure-types}

| Nom                                       | Type               |
| ----------------------------------------- | ------------------ |
| accessKeyId                               | string             |
| accessKeySecret                           | string             |
| region                                    | string             |
| emailAddress                              | string (OPTIONNEL) |
| emailAddressIdentityArn                   | string (OPTIONNEL) |
| feedbackForwardingEmailAddress            | string (OPTIONNEL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPTIONNEL) |
| configurationSetName                      | string (OPTIONNEL) |
| templates                                 | Template[]         |

| Propriétés du template | Type        | Valeurs énumérées                                    |
| ---------------------- | ----------- | ---------------------------------------------------- |
| subject                | string      | N/A                                                  |
| content                | string      | N/A                                                  |
| usageType              | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
