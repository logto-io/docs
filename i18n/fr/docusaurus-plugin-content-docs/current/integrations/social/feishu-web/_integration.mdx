## Commencer avec la connexion sociale Feishu \{#getting-started-with-feishu-social-sign-in}

Le connecteur web Feishu est conçu pour les applications web de bureau et utilise le processus d'authentification OAuth 2.0.

## Enregistrer un compte développeur Feishu \{#register-a-feishu-developer-account}

Si vous n'avez pas de compte développeur Feishu, veuillez vous inscrire sur la [Plateforme Ouverte Feishu](https://open.feishu.cn/app).

## Créer une application \{#create-an-application}

1. Sur la [Console Développeur](https://open.feishu.cn/app), cliquez sur "Créer une application personnalisée".
2. Remplissez le nom de l'application, la description, sélectionnez une icône et cliquez sur le bouton "Créer".
3. Dans la barre latérale gauche, cliquez sur "Paramètres de sécurité", remplissez l'"URL de redirection" comme `${logto_endpoint}/callback/${connector_id}`. La valeur correspondante peut être trouvée dans le champ `Callback URI` sur la page des détails du connecteur Feishu dans la Console Logto.
4. Dans "Identifiants & Informations de base", vous pouvez obtenir l'"App ID" et le "App Secret".

:::note
Pour une utilisation interne non-entreprise, vous devez également cliquer sur le bouton "Créer une version" dans la page "Gestion des versions et publication".
L'"App ID" et le "App Secret" ne prendront effet qu'après que le statut de l'application passe à "Activé".
:::
