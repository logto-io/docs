## Premiers pas avec la connexion sociale Feishu \{#getting-started-with-feishu-social-sign-in}

Le connecteur web Feishu est conçu pour les applications web de bureau et utilise le processus d’authentification OAuth 2.0.

## Enregistrer un compte développeur Feishu \{#register-a-feishu-developer-account}

Si vous ne possédez pas de compte développeur Feishu, veuillez vous inscrire sur la [plateforme Feishu Open](https://open.feishu.cn/app).

## Créer une application \{#create-an-application}

1. Sur la [console développeur](https://open.feishu.cn/app), cliquez sur "Créer une application personnalisée".
2. Renseignez le nom de l’application, la description, sélectionnez une icône, puis cliquez sur le bouton "Créer".
3. Dans la barre latérale gauche, cliquez sur "Paramètres de sécurité", puis renseignez l’"URL de redirection" avec `${logto_endpoint}/callback/${connector_id}`. La valeur correspondante se trouve dans le champ `Callback URI` sur la page de détails du connecteur Feishu dans la Console Logto.
4. Dans "Identifiants & Infos de base", vous pouvez obtenir l’"ID de l’application" et le "Secret de l’application".

:::note
Pour une utilisation interne non-entreprise, vous devez également cliquer sur le bouton "Créer une version" dans la page "Gestion des versions et publication".
L’"ID de l’application" et le "Secret de l’application" ne prendront effet qu’après que le statut de l’application soit passé à "Activé".
:::
