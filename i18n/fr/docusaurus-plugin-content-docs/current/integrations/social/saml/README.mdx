# Configurer la connexion sociale avec SAML

Le connecteur officiel Logto pour le protocole SAML.

<GuideTip />

## Commencer \{#get-started}

Le connecteur SAML permet la connexion de Logto à un fournisseur d'identité sociale arbitraire qui prend en charge le protocole SAML.

:::note

Le connecteur SAML est un type spécial de connecteur dans Logto, vous pouvez ajouter quelques connecteurs basés sur SAML.

:::
