# Configurer la connexion sociale avec Kakao

Le connecteur Kakao fournit un moyen succinct pour votre application d'utiliser le système d'Authentification OAuth 2.0 de Kakao.

<GuideTip />

## Configurer un projet dans la console des développeurs Kakao

- Visitez la [console des développeurs Kakao](https://developers.kakao.com/console/app) et connectez-vous avec votre compte Kakao.
- Cliquez sur **Ajouter une application** pour créer un nouveau projet ou choisissez un projet existant.

## Configurer la connexion Kakao

### Activer la connexion Kakao

- Cliquez sur **Paramètres du produit -> Connexion Kakao** dans le menu.
- Activez `Activation de la connexion Kakao`
- Ajoutez l'URL ci-dessous dans `Redirect URI`
  - `http(s)://YOUR_URL/callback/${connector_id}` (Le `connector_id` peut être trouvé dans la barre supérieure de la page des détails du connecteur de la console d'administration Logto.)
  - (Veuillez remplacer `YOUR_URL` par votre URL `Logto`, et choisissez `http` ou `https` selon votre situation.)

### Paramètre de confidentialité

- Cliquez sur **Paramètres du produit -> Connexion Kakao -> Élément de consentement** dans le menu.
- Changez l'état de `Surnom`, `Image de profil` et `Email` en **Consentement requis** (Vous pourriez ne pas pouvoir changer `Email` en **Consentement requis** en raison de la configuration de votre projet.)

### Paramètre de sécurité (Optionnel)

- Cliquez sur **Paramètres du produit -> Connexion Kakao -> Sécurité** dans le menu.
- Cliquez sur le `Code secret client` pour générer un code secret.
- Changez l'`État d'activation` en Activer. (Si vous l'activez, le `code secret` est nécessaire.)

## Configurer Logto

### Types de configuration

| Nom          | Type    |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId

`clientId` est la `clé API REST` de votre projet.
(Vous pouvez la trouver dans le `résumé` de votre projet depuis la console des développeurs Kakao.)

#### clientSecret

`clientSecret` est le `Code secret` de votre projet.
(Veuillez vérifier [Paramètre de sécurité (Optionnel)](#security-setting-optional))
