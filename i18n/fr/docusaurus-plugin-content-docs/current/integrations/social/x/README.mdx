# Configurer la connexion sociale avec X (Set up social login with X)

Le connecteur officiel Logto pour la connexion sociale X (anciennement Twitter).

<GuideTip />

## Commencer \{#get-started}

Le connecteur X (Twitter) permet aux utilisateurs finaux de se connecter à votre application en utilisant leurs propres comptes X (Twitter) via le protocole d'authentification X OAuth 2.0.

<Integration />

## Tester le connecteur X (Twitter) \{#test-x-connector}

C'est tout. Le connecteur X (Twitter) devrait maintenant être disponible. N'oubliez pas d'[activer le connecteur social dans l'inscription et la connexion](/connectors/social-connectors/#enable-social-sign-in).
