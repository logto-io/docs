### Configurer une application Amazon \{#setup-a-amazon-app}

Allez sur le [Portail des développeurs Amazon](https://developer.amazon.com) et connectez-vous avec votre compte. Si vous n'avez pas de compte, vous pouvez en créer un.

Ensuite, créez une application.

**Étape 1 :** Allez dans le bloc "Login with Amazon" sur la page d'accueil.

Ou vous pouvez visiter [cette page](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html).

Suivez le guide pour créer un nouveau profil de sécurité.

Après avoir créé le profil de sécurité, vous obtiendrez un Client ID et un Client Secret à partir du tableau.

**Étape 2 :** Configurer les URLs de rappel.

Cliquez sur le bouton "Manage" sur le côté droit du tableau et allez dans l'onglet "Web Settings".

Dans le champ "Allowed redirect URLs", ajoutez l'URL suivante :

- `${your_logto_origin}/callback/${connector_id}`.

Le `connector_id` peut être trouvé sur la barre supérieure de la page des détails du connecteur dans la Logto Admin Console.

### Configurer votre connecteur \{#configure-your-connector}

Dans votre configuration de connecteur Logto, remplissez les champs suivants avec les valeurs obtenues à l'étape précédente :

- **clientId :** Le Client ID de votre application.
- **clientSecret :** Le Client Secret de votre application.

`scope` est une liste d'espaces délimités de portées OIDC. Si non fourni, la portée par défaut est `openid profile`.

### Types de configuration \{#config-types}

| Nom          | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
