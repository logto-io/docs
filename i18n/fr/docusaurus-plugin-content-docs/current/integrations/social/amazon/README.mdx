# Configurer la connexion sociale avec Amazon (Set up social login with Amazon)

Le connecteur officiel Logto pour la connexion sociale Amazon.

<GuideTip />

## Commencer \{#get-started}

Le connecteur Amazon permet aux utilisateurs finaux de se connecter à votre application en utilisant leurs propres comptes Amazon via le protocole d’authentification OAuth 2.0 d’Amazon.

<Integration />

## Tester le connecteur Amazon \{#test-amazon-connector}

C’est tout. Le connecteur Amazon devrait maintenant être disponible. N’oubliez pas d’[activer le connecteur social dans l’inscription et la connexion](/connectors/social-connectors/#enable-social-sign-in).
