# Configurer la connexion sociale avec LinkedIn

Le connecteur officiel Logto pour la connexion sociale LinkedIn.

<GuideTip />

## Commencer \{#get-started}

Le connecteur LinkedIn permet aux utilisateurs finaux de se connecter à votre application en utilisant leurs propres comptes LinkedIn via le protocole d'authentification LinkedIn OAuth 2.0.
