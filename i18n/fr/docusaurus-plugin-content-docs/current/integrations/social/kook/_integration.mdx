## Créer une application dans le Centre de développement KOOK \{#create-an-application-in-the-kook-developer-center}

Conseil : Vous pouvez ignorer certaines sections si vous les avez déjà terminées.

:::note
Vous devez demander l'autorisation OAuth2 de KOOK avant de créer une application OAuth2. Veuillez consulter les informations sur la page OAuth2 ou demander plus d'informations au personnel de KOOK sur le serveur officiel des développeurs KOOK.
:::

1. Rendez-vous sur le [Centre de développement KOOK](https://developer.kookapp.cn/app/index) et connectez-vous avec votre compte KOOK. Vous pouvez créer un nouveau compte si vous n'en avez pas.
2. Cliquez sur le bouton **Créer une application** pour créer une application, choisissez un nom pour celle-ci (par exemple, LogtoAuth) et cliquez sur **Confirmer**.
3. Cliquez sur l'icône de l'application que vous avez créée, puis allez sur la page **OAuth2**.
4. Ajoutez les redirections valides (par exemple, `${your_logto_origin}/callback/${connector_id}`) dans les champs **Callback URIs**. Vous pouvez trouver l'URI de redirection sur la page des détails du connecteur dans la console d'administration Logto.
5. N'oubliez pas de cocher la portée **get_user_info** dans le **générateur de lien OAuth2**.

## Configurer votre connecteur KOOK \{#configure-your-kook-connector}

Renseignez les champs clientId et clientSecret avec le Client ID et le Client Secret que vous avez obtenus sur la page OAuth2 mentionnée dans la section précédente.

### Types de configuration \{#config-types}

| Nom          | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
