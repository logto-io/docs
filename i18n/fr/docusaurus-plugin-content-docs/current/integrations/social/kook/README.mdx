# Configurer la connexion sociale avec KOOK

Le connecteur officiel Logto pour les applications web de connexion sociale KOOK.

<GuideTip />

## Commencer \{#get-started}

Le connecteur KOOK permet aux utilisateurs finaux de se connecter à votre application en utilisant leur propre compte KOOK via le protocole d’authentification OAuth 2.0 de KOOK.
