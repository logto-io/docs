# Configurer la connexion sociale avec WeChat (Native) (Set up social login with WeChat (Native))

Le connecteur officiel Logto pour la connexion sociale WeChat dans les applications natives (iOS et Android).

<GuideTip />

## Commencer \{#get-started}

Si vous ne connaissez pas le concept de connecteur ou si vous ne savez pas comment ajouter ce connecteur à votre expérience de connexion, veuillez consulter le [tutoriel Logto](/connectors/social-connectors).

Dans les applications natives, vous ne pouvez pas utiliser le web comme méthode de connexion WeChat : une navigation vers l’application WeChat est requise, et il est également nécessaire d’utiliser leur SDK.

Nous savons que cela peut sembler intimidant, mais ne vous inquiétez pas. Nous allons gérer cela facilement dans ce tutoriel.

<Integration />

## Tester le connecteur WeChat natif \{#test-wechat-native-connector}

C'est tout. N'oubliez pas d'[activer le connecteur social dans l'expérience de connexion](/connectors/social-connectors/#enable-social-sign-in).

Une fois le connecteur WeChat natif activé, vous pouvez construire et lancer votre application pour vérifier si cela fonctionne.

:::caution

WeChat n’a pas prévu de solution pour les appareils sans l’application WeChat installée. Logto masquera ce connecteur lors de la connexion dans ce cas (ce qui est la méthode recommandée par le [guide de développement officiel](https://developers.weixin.qq.com/doc/oplatform/en/Mobile_App/WeChat_Login/Development_Guide.html)).

:::
