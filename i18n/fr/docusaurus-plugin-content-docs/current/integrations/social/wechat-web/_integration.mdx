## Créer une application web dans la plateforme ouverte WeChat \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

Vous pouvez ignorer certaines sections si vous avez déjà terminé.

:::

### Créer un compte \{#create-an-account}

Ouvrez https://open.weixin.qq.com/, cliquez sur le bouton "Sign Up" dans le coin supérieur droit, puis terminez le processus d'inscription.

### Créer une application web \{#create-a-web-app}

Connectez-vous avec le compte que vous venez de créer. Dans l'onglet "Web Application" (网页应用), cliquez sur le grand bouton vert "Créer une application web" (创建网页应用).

<img src="/img/assets/app-tabs.png" alt="Onglets d'application" />

Remplissons les informations requises dans le formulaire de demande.

<img src="/img/assets/create-web-app.png" alt="Créer une application web" />

#### Informations de base \{#basic-info}

La plupart d'entre elles sont assez simples. Après avoir rempli le formulaire, cliquez sur "Étape suivante" pour continuer.

#### Informations sur le site web \{#website-info}

Remplissez "Domaine de rappel d'autorisation" (授权回调域) avec votre domaine Logto. Par exemple, `logto.io`.

#### En attente du résultat de l'examen \{#waiting-for-the-review-result}

Après avoir complété les informations sur le site web, cliquez sur "Soumettre pour examen" pour continuer. En général, l'examen est rapide et se termine en 1 à 2 jours.

Nous soupçonnons que le réviseur est attribué aléatoirement à chaque soumission, car la norme est fluctuante. Vous pouvez être rejeté la première fois, mais ne vous découragez pas ! Indiquez votre situation actuelle et demandez au réviseur comment la modifier.

### Configurer votre connecteur WeChat \{#configure-your-wechat-connector}

Remplissez le champ `clientId` et `clientSecret` avec _Client ID_ et _Client Secret_ que vous avez obtenus à partir des pages de détails de l'application OAuth.

Remplissez le champ `scope` avec soit 'snsapi_userinfo' soit 'snsapi_base'. Vous pouvez laisser ce champ vide car il est OPTIONNEL. La valeur par défaut est définie sur 'snsapi_userinfo'. Vous pouvez consulter la [différence](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) entre les différentes valeurs.
