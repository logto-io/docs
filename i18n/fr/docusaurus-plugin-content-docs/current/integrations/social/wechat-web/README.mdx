# Configurer la connexion sociale avec WeChat (Web)

Le connecteur officiel Logto pour la connexion sociale WeChat dans les applications web.

<GuideTip />

## Commencer \{#get-started}

Si vous ne connaissez pas le concept de connecteur ou si vous ne savez pas comment ajouter ce connecteur à votre expérience de connexion, veuillez consulter le [tutoriel Logto](/connectors/social-connectors).

:::caution

Ce connecteur est uniquement destiné aux applications web. Si vous cherchez la méthode pour vous connecter avec WeChat dans des applications natives, veuillez consulter le [connecteur WeChat natif](/integrations/wechat-native).

:::

## Créer une application web sur la plateforme ouverte WeChat \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

Vous pouvez passer certaines sections si vous les avez déjà terminées.

:::

### Créer un compte \{#create-an-account}

Ouvrez https://open.weixin.qq.com/, cliquez sur le bouton "S'inscrire" dans le coin supérieur droit, puis terminez le processus d'inscription.

### Créer une application web \{#create-a-web-app}

Connectez-vous avec le compte que vous venez de créer. Dans l'onglet "Application Web" (网页应用), cliquez sur le grand bouton vert "Créer une application web" (创建网页应用).

<img src="/img/assets/app-tabs.png" alt="Onglets d'application" />

Remplissons les informations requises dans le formulaire d'application.

<img src="/img/assets/create-web-app.png" alt="Créer une application web" />

#### Informations de base \{#basic-info}

La plupart des champs sont assez explicites. Après avoir rempli le formulaire, cliquez sur "Étape suivante" pour continuer.

#### Informations du site web \{#website-info}

Renseignez le champ "Domaine de rappel d'autorisation" (授权回调域) avec votre domaine Logto. Par exemple, `logto.io`.

#### En attente du résultat de la révision \{#waiting-for-the-review-result}

Après avoir complété les informations du site web, cliquez sur "Soumettre pour révision" pour continuer. En général, la révision est rapide et se termine en 1 à 2 jours.

Nous pensons que le réviseur est attribué aléatoirement à chaque soumission, car le standard varie. Vous pouvez être refusé la première fois, mais ne vous découragez pas ! Expliquez votre situation et demandez au réviseur comment la modifier.

### Configurer votre connecteur WeChat \{#configure-your-wechat-connector}

Renseignez les champs `clientId` et `clientSecret` avec le _Client ID_ et le _Client Secret_ obtenus depuis les pages de détails de l'application OAuth.

Renseignez le champ `scope` avec soit 'snsapi_userinfo' soit 'snsapi_base'. Vous pouvez laisser ce champ vide car il est OPTIONNEL. La valeur par défaut est 'snsapi_userinfo'. Vous pouvez consulter la [différence](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) entre les différentes valeurs.

### Tester le connecteur WeChat web \{#test-wechat-web-connector}

C'est tout. N'oubliez pas [d'activer le connecteur social dans l'inscription et la connexion](/connectors/social-connectors/#enable-social-sign-in).

Une fois le connecteur WeChat web activé, vous pouvez vous reconnecter à votre application pour vérifier si cela fonctionne.
