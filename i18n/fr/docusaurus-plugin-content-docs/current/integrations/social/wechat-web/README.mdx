# Configurer la connexion sociale avec WeChat (Web)

Le connecteur officiel Logto pour la connexion sociale WeChat dans les applications web.

<GuideTip />

## Commencer \{#get-started}

Si vous ne connaissez pas le concept du connecteur ou ne savez pas comment ajouter ce connecteur à votre expérience de connexion, veuillez consulter le [tutoriel Logto](/connectors/social-connectors).

:::caution

Ce connecteur est uniquement pour les applications web. Si vous recherchez la méthode pour vous connecter avec WeChat dans des applications natives, veuillez consulter le [connecteur natif WeChat](/integrations/wechat-native).

:::

## Créer une application web dans la plateforme ouverte WeChat \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

Vous pouvez passer certaines sections si vous avez déjà terminé.

:::

### Créer un compte \{#create-an-account}

Ouvrez https://open.weixin.qq.com/, cliquez sur le bouton "Sign Up" dans le coin supérieur droit, puis terminez le processus d'inscription.

### Créer une application web \{#create-a-web-app}

Connectez-vous avec le compte que vous venez de créer. Dans l'onglet "Web Application" (网页应用), cliquez sur le grand bouton vert "Create a web app" (创建网页应用).

<img src="/img/assets/app-tabs.png" alt="Onglets d'application" />

Remplissons les informations requises dans le formulaire de demande.

<img src="/img/assets/create-web-app.png" alt="Créer une application web" />

#### Informations de base \{#basic-info}

La plupart d'entre elles sont assez simples. Après avoir terminé le formulaire, cliquez sur "Next step" pour continuer.

#### Informations sur le site web \{#website-info}

Remplissez "Authorization callback domain" (授权回调域) avec votre domaine Logto. Par exemple, `logto.io`.

#### En attente du résultat de la révision \{#waiting-for-the-review-result}

Après avoir complété les informations sur le site web, cliquez sur "Submit Review" pour continuer. En général, la révision est rapide et se termine en 1 à 2 jours.

Nous soupçonnons que le réviseur est attribué aléatoirement à chaque soumission, car la norme est fluctuante. Vous pouvez être rejeté la première fois, mais ne vous découragez pas ! Indiquez votre situation actuelle et demandez au réviseur comment la modifier.

### Configurer votre connecteur WeChat \{#configure-your-wechat-connector}

Remplissez le champ `clientId` et `clientSecret` avec _Client ID_ et _Client Secret_ que vous avez obtenus des pages de détail de l'application OAuth.

Remplissez le champ `scope` avec soit 'snsapi_userinfo' soit 'snsapi_base'. Vous pouvez laisser ce champ vide car il est OPTIONNEL. La valeur par défaut est définie sur 'snsapi_userinfo'. Vous pouvez consulter la [différence](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) entre les différentes valeurs.

### Tester le connecteur web WeChat \{#test-wechat-web-connector}

C'est tout. N'oubliez pas d'[activer le connecteur social dans l'expérience de connexion](/connectors/social-connectors/#enable-social-sign-in).

Une fois le connecteur web WeChat activé, vous pouvez vous reconnecter à votre application pour voir si cela fonctionne.
