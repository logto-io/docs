## Se connecter avec un compte Patreon \{#sign-in-with-patreon-account}

Allez sur le [site Patreon](https://www.patreon.com/) et connectez-vous avec votre compte Patreon. Vous pouvez créer un nouveau compte si vous n'en avez pas.

## Créer et configurer une application OAuth \{#create-and-configure-oauth-app}

Suivez le guide [création d'une application OAuth Patreon](https://www.patreon.com/portal/registration/register-clients) et enregistrez une nouvelle application.

Nommez votre nouvelle application OAuth dans **App Name** et renseignez l'**App URL** de l'application. Vous pouvez laisser le champ **App Description** vide et personnaliser les **Redirect URIs** comme `${your_logto_origin}/callback/${connector_id}`. Le `connector_id` se trouve dans la barre supérieure de la page de détails du connecteur dans la Console d'administration Logto.

:::note
Si vous rencontrez le message d'erreur "The redirect_uri MUST match the registered callback URL for this application." lors de la connexion, essayez d'aligner l'URI de redirection de votre application OAuth Patreon et l'URL de redirection de votre application Logto (y compris le protocole) pour résoudre le problème.
:::

## Gérer les applications OAuth \{#managing-oauth-apps}

Rendez-vous sur la [page Clients & API Keys](https://www.patreon.com/portal/registration/register-clients) sur Patreon, où vous pouvez ajouter, modifier ou supprimer des applications OAuth existantes. Vous pouvez également trouver le `Client ID` et générer des `Client secrets` dans les pages de détails de l'application OAuth.

## Configurer votre connecteur \{#configure-your-connector}

Renseignez les champs `clientId` et `clientSecret` avec le _Client ID_ et le _Client Secret_ obtenus dans les pages de détails de l'application OAuth mentionnées dans la section précédente.

`scope` est une liste d’[autorisations (scopes)](https://docs.patreon.com/#scopes) séparées par des espaces. Si non renseigné, la portée par défaut est `identity identity[email]`.

### Types de configuration \{#config-types}

| Nom          | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
