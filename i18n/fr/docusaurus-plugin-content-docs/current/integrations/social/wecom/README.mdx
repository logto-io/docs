# Configurer la connexion sociale avec WeCom (OAuth 2.0)

Le connecteur personnalisé pour la connexion sociale WeCom (peut-être appelé WXwork).

<GuideTip />

## Commencer \{#get-started}

Connectez-vous à [WeCom WebUI](https://work.weixin.qq.com/) avec un compte administrateur ou cliquez sur **Gérer l'entreprise (管理企业)** depuis l'application WeCom.

Dans l'onglet "Gestion des applications", faites défiler la page vers le bas et cliquez sur "Créer une application".

Remplissez les informations appropriées selon votre application. Ensuite, créez.

Nous avons maintenant l'ID d'agent (PAS APPID) et le Secret.

### Informations sur le site web \{#website-info}

Définissez les éléments dont vous avez besoin sur cette page. Cela ressemblerait à :

- Utilisateurs autorisés : _qui peut voir cette application_
- Page d'accueil de l'application : _La page d'accueil de votre application. Par exemple, `logto.io/demo-app`_

**Important**
Il y a trois éléments sur cette page concernant l' "API développeur (开发者接口)".

1. Autorisation Web et JS-SDK ;
2. Connexion via l'autorisation par WeCom ;
3. IP de confiance de l'entreprise ;

Remplissez-les selon le guide de WeCom.

### ID de l'entreprise \{#corp-id}

Si vous êtes familier avec le développement WeChat, vous remarquerez que l'utilisation de l'ID de l'entreprise est la même que l'APP ID.

Vous pouvez trouver l'ID de l'entreprise en bas de la page de l'onglet "Mon entreprise (我的企业)". Cela ressemble à **ww\*\*\*\*** .

## Configurer le connecteur \{#configure-the-connector}

Nous avons donc l'ID d'agent, le Secret et l'ID de l'entreprise.

Complétons le formulaire pour le connecteur.

Vous pouvez laisser le champ `Scope` vide car il est facultatif. Alternativement, vous pouvez remplir `snsapi_base` ou `snsapi_privateinfo`.

**Enregistrez et c'est terminé**
