# Configurer la vérification par SMS avec GatewayAPI SMS

Le connecteur officiel Logto pour GatewayAPI SMS.

<GuideTip />

## Commencer \{#get-started}

GatewayAPI est un fournisseur de services cloud en Europe, offrant de nombreux services cloud, y compris le SMS (service de messages courts). Le connecteur GatewayAPI SMS est un plugin fourni par l'équipe Logto pour appeler le service SMS de GatewayAPI, grâce auquel les utilisateurs finaux Logto peuvent s'inscrire et se connecter à leur compte Logto via un code de vérification SMS.
