# Configurer la vérification par SMS avec GatewayAPI SMS

Le connecteur officiel Logto pour GatewayAPI SMS.

<GuideTip />

## Commencer \{#get-started}

GatewayAPI est un fournisseur de services cloud en Europe, offrant de nombreux services cloud, y compris le SMS (service de messages courts). Le connecteur SMS GatewayAPI est un plugin fourni par l'équipe Logto pour appeler le service SMS GatewayAPI, grâce auquel les utilisateurs finaux de Logto peuvent s'inscrire et se connecter à leur compte Logto via un code de vérification par SMS.

## Configuration dans GatewayAPI \{#set-up-in-gatewayapi}

### Créer un compte GatewayAPI \{#create-an-gatewayapi-account}

Allez sur [le site Web de GatewayAPI](https://www.gatewayapi.com/) et enregistrez votre compte GatewayAPI si vous n'en avez pas un.

### Activer le compte \{#enable-account}

Vous devrez peut-être activer votre compte avant d'utiliser le service SMS. Vous pouvez contacter le service client de GatewayAPI pour activer votre compte.

### Obtenir le jeton API \{#get-api-token}

Accédez à la page des clés API depuis la console GatewayAPI, et trouvez le jeton API ou créez un nouveau jeton API.

## Configuration dans Logto \{#set-up-in-logto}

1. **Endpoint** : Si votre compte GatewayAPI est dans la région UE, vous devez utiliser l'endpoint https://gatewayapi.com/rest/mtsms. Si votre compte GatewayAPI est dans la région US, vous devez utiliser l'endpoint https://gatewayapi.com/rest/mtsms.
2. **Jeton API** : Le jeton API que vous avez créé à l'étape précédente.
3. **Expéditeur** : L'expéditeur que vous souhaitez utiliser pour envoyer le SMS.
4. **Modèles** : Les modèles que vous souhaitez utiliser pour envoyer le SMS, vous pouvez utiliser les modèles par défaut ou les modifier selon vos besoins.
