# Configurer la vérification par SMS avec Vonage SMS (Set up SMS verification with Vonage SMS)

Le connecteur officiel Logto pour Vonage SMS.

<GuideTip />

## Commencer \{#get-started}

Vonage est un fournisseur mondial de communications, offrant des services de communication cloud robustes, y compris la messagerie SMS (service de messages courts). Le connecteur Vonage SMS est un plugin fourni par l’équipe Logto pour permettre aux utilisateurs finaux Logto de s’inscrire et de se connecter à leur compte Logto via des codes de vérification SMS.
