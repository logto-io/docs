# Configurer la vérification par SMS avec Vonage SMS

Le connecteur officiel Logto pour Vonage SMS.

<GuideTip />

## Commencer

Vonage est un fournisseur de communications mondial, offrant des services de communication robustes basés sur le cloud, y compris le SMS (service de messages courts). Le connecteur Vonage SMS est un plugin fourni par l'équipe Logto pour permettre aux utilisateurs finaux de Logto de s'inscrire et de se connecter à leur compte Logto via des codes de vérification par SMS.

## Configuration dans Vonage

:::tip
Vous pouvez passer cette étape si vous l'avez déjà complétée.
:::

Pour travailler avec ce connecteur, vous devrez [créer un compte](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) chez Vonage. Cela vous donnera une clé API et un secret que vous pourrez utiliser pour accéder aux API via ce connecteur.

Une fois que vous avez un compte, vous pouvez trouver votre clé API et votre secret API en haut du tableau de bord API Vonage.

Et vous pourriez avoir besoin de [louer un numéro virtuel](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) pour envoyer des messages SMS.

Consultez l' [API SMS Vonage](https://developer.vonage.com/en/messaging/sms/overview) pour plus d'informations.

## Configuration dans Logto

1. **Clé API** : Votre clé API Vonage.
2. **Secret API** : Votre secret API Vonage.
3. **Nom de la marque** : Le nom de la marque que vous souhaitez utiliser pour envoyer le SMS, cela s'appelle aussi le champ `from`, voir l' [Identité de l'expéditeur](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) pour plus d'informations.
4. **Modèles** : Les modèles que vous souhaitez utiliser pour envoyer le SMS, vous pouvez utiliser les modèles par défaut ou les modifier selon vos besoins.
