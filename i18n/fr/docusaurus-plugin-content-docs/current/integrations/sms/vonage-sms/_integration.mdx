## Configuration dans Vonage \{#set-up-in-vonage}

:::tip
Vous pouvez ignorer cette étape si vous l'avez déjà complétée.
:::

Pour utiliser ce connecteur, vous devrez [créer un compte](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) chez Vonage. Cela vous fournira une clé API et un secret que vous pourrez utiliser pour accéder aux API via ce connecteur.

Une fois que vous avez un compte, vous pouvez trouver votre clé API et votre secret API en haut du tableau de bord API Vonage.

Et il se peut que vous deviez [louer un numéro virtuel](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) pour envoyer des messages SMS.

Consultez la [Vonage SMS API](https://developer.vonage.com/en/messaging/sms/overview) pour plus d'informations.

## Configuration dans Logto \{#set-up-in-logto}

1. **Clé API** : Votre clé API Vonage.
2. **Secret API** : Votre secret API Vonage.
3. **Nom de la marque** : Le nom de la marque que vous souhaitez utiliser pour envoyer le SMS, cela correspond également au champ `from`, voir [Identité de l'expéditeur](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) pour plus d'informations.
4. **Modèles** : Les modèles que vous souhaitez utiliser pour envoyer le SMS, vous pouvez utiliser les modèles par défaut ou les modifier selon vos besoins.
