## Configurer dans Vonage \{#set-up-in-vonage}

:::tip
Vous pouvez passer cette étape si vous l'avez déjà complétée.
:::

Pour travailler avec ce connecteur, vous devrez [créer un compte](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) chez Vonage. Cela vous fournira une clé API et un secret que vous pourrez utiliser pour accéder aux API via ce connecteur.

Une fois que vous avez un compte, vous pouvez trouver votre clé API et votre secret API en haut du tableau de bord API Vonage.

Et vous pourriez avoir besoin de [louer un numéro virtuel](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) pour envoyer des messages SMS.

Consultez l'[API SMS Vonage](https://developer.vonage.com/en/messaging/sms/overview) pour plus d'informations.

## Configurer dans Logto \{#set-up-in-logto}

1. **API Key** : Votre clé API Vonage.
2. **API Secret** : Votre secret API Vonage.
3. **Brand Name** : Le nom de marque que vous souhaitez utiliser pour envoyer le SMS, cela est également appelé le champ `from`, voir l'[Identité de l'expéditeur](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) pour plus d'informations.
4. **Templates** : Les modèles que vous souhaitez utiliser pour envoyer le SMS, vous pouvez utiliser les modèles par défaut ou les modifier selon vos besoins.
