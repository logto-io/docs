## Créer un compte \{#register-account}

Créez un nouveau compte sur [SMSAero](https://smsaero.ru/). (Passez à l'étape suivante si vous en possédez déjà un.)

## Obtenir les identifiants du compte \{#get-account-credentials}

Nous aurons besoin des identifiants API pour faire fonctionner le connecteur. Commençons par
la page [API et SMPP](https://smsaero.ru/cabinet/settings/apikey/).

Copiez la "clé API" ou générez-en une nouvelle.

## Composer le JSON du connecteur \{#compose-the-connector-json}

Renseignez les champs _email_, _apiKey_ et _senderName_ avec votre e-mail, clé API et nom d'expéditeur.

Vous pouvez renseigner le nom d'expéditeur avec "SMSAero" pour utiliser le nom d'expéditeur par défaut fourni par SMSAero.

Vous pouvez ajouter plusieurs modèles de connecteur SMS pour différents cas d'usage. Voici un exemple d'ajout d'un seul modèle :

- Remplissez le champ `content` avec un contenu arbitraire de type chaîne de caractères. N'oubliez pas de laisser l'espace réservé `{{code}}` pour le code de vérification aléatoire.
- Remplissez le champ `usageType` avec l'une des valeurs suivantes : `Register`, `SignIn`, `ForgotPassword`, `Generic` selon les cas d'utilisation.
  Pour activer tous les parcours utilisateur, les modèles avec usageType `Register`, `SignIn`, `ForgotPassword` et `Generic` sont requis.

### Tester le connecteur SMSAero \{#test-smsaero-connector}

Vous pouvez saisir un numéro de téléphone et cliquer sur "Envoyer" pour vérifier si les paramètres fonctionnent avant de cliquer sur "Enregistrer et terminer".

C'est tout. N'oubliez pas d’<MainSiteUrl href="/connectors/sms-connectors/#enable-phone-number-sign-up-or-sign-in">activer le connecteur dans l’expérience de connexion</MainSiteUrl>.

### Types de configuration \{#config-types}

| Nom        | Type        |
| ---------- | ----------- |
| email      | string      |
| apiKey     | string      |
| senderName | string      |
| templates  | Templates[] |

| Propriétés du modèle | Type        | Valeurs énumérées                                    |
| -------------------- | ----------- | ---------------------------------------------------- |
| content              | string      | N/A                                                  |
| usageType            | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
