Para inicializar um novo projeto FastAPI, crie um diretório e configure a estrutura básica:

```bash
mkdir seu-nome-da-api
cd seu-nome-da-api
```

Crie um arquivo de requisitos:

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Crie uma aplicação FastAPI básica:

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

Inicie o servidor de desenvolvimento:

```bash
uvicorn main:app --reload
```

:::note
Consulte a documentação do FastAPI para mais detalhes sobre como configurar operações de rota, injeção de dependências e outros recursos.
:::
