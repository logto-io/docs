Para iniciar um novo projeto Flask, crie um diretório e configure a estrutura básica:

```bash
mkdir seu-nome-da-api
cd seu-nome-da-api
```

Crie um arquivo de dependências:

```txt title="requirements.txt"
Flask
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Crie uma aplicação Flask básica:

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

Inicie o servidor de desenvolvimento:

```bash
python app.py
```

:::note
Consulte a documentação do Flask para mais detalhes sobre como configurar rotas, blueprints e outros recursos.
:::
