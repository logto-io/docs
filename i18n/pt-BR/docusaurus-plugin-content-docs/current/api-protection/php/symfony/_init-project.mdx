Para inicializar um novo projeto Symfony para desenvolvimento de API, utilize o Symfony CLI ou o Composer:

Usando o Symfony CLI (recomendado):

```bash
symfony new your-api-name --webapp
cd your-api-name
```

Ou usando o Composer:

```bash
composer create-project symfony/skeleton your-api-name
cd your-api-name
composer require webapp
```

Instale pacotes adicionais para desenvolvimento de API:

```bash
composer require symfony/security-bundle
composer require symfony/serializer
composer require doctrine/annotations
```

Inicie o servidor de desenvolvimento:

```bash
symfony serve
```

Ou usando o servidor embutido do PHP:

```bash
php -S localhost:8000 -t public/
```

Isso cria um projeto Symfony básico. Configure o framework para desenvolvimento de API:

```yaml title="config/packages/framework.yaml"
framework:
  secret: '%env(APP_SECRET)%'
  serializer:
    enabled: true
  property_access:
    enabled: true
```

:::note
Consulte a documentação do Symfony para mais detalhes sobre como configurar controllers, serviços e outros recursos.
:::
