Para inicializar um novo projeto Go com Fiber, você pode seguir estes passos:

```bash
go mod init seu-nome-da-api
go get github.com/gofiber/fiber/v2
```

Em seguida, crie uma configuração básica de servidor Fiber:

```go title="main.go"
package main

import (
    "log"

    "github.com/gofiber/fiber/v2"
)

func main() {
    app := fiber.New()

    log.Fatal(app.Listen(":3000"))
}
```

:::note
Consulte a documentação do Fiber para mais detalhes sobre como configurar rotas, middlewares e outros recursos.
:::
