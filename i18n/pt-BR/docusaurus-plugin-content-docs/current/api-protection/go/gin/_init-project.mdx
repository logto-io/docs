Para inicializar um novo projeto Go com Gin, você pode seguir estes passos:

```bash
go mod init seu-nome-da-api
go get github.com/gin-gonic/gin
```

Em seguida, crie uma configuração básica de servidor Gin:

```go title="main.go"
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()

    r.Run(":3000") // escuta e serve em 0.0.0.0:3000
}
```

:::note
Consulte a documentação do Gin para mais detalhes sobre como configurar rotas, middlewares e outros recursos.
:::
