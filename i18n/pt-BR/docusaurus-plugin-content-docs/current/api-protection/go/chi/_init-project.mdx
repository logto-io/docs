Para inicializar um novo projeto Go com Chi, você pode seguir estes passos:

```bash
go mod init seu-nome-da-api
go get github.com/go-chi/chi/v5
```

Em seguida, crie uma configuração básica de servidor Chi:

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
Consulte a documentação do Chi para mais detalhes sobre como configurar rotas, middlewares e outros recursos.
:::
