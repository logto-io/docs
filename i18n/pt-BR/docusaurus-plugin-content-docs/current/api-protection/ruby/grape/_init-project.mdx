Para inicializar um novo projeto Grape API, crie um diretório e configure a estrutura básica:

```bash
mkdir seu-nome-api
cd seu-nome-api
```

Crie um Gemfile:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

Instale as dependências:

```bash
bundle install
```

Crie uma Grape API básica:

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

Crie um arquivo config.ru:

```ruby title="config.ru"
require_relative 'api'

run API
```

Inicie o servidor de desenvolvimento:

```bash
bundle exec rackup
```

:::note
Consulte a documentação do Grape para mais detalhes sobre como configurar recursos, validação de parâmetros e outros recursos.
:::
