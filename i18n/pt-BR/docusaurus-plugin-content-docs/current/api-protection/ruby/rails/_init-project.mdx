Para inicializar um novo projeto Rails API, você pode usar o gerador do Rails:

```bash
rails new seu-nome-api --api
cd seu-nome-api
```

Inicie o servidor de desenvolvimento:

```bash
rails server
```

Crie um controlador básico de API:

```ruby title="app/controllers/api/base_controller.rb"
class Api::BaseController < ApplicationController
  def index
    render json: { message: 'Hello from Rails API' }
  end
end
```

Adicione as rotas:

```ruby title="config/routes.rb"
Rails.application.routes.draw do
  namespace :api do
    root 'base#index'
  end
end
```

:::note
Consulte a documentação do Rails para mais detalhes sobre como configurar controladores, modelos e outros recursos.
:::
