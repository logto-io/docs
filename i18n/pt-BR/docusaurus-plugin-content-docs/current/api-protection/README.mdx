# Proteja suas APIs com Logto

Aprenda como implementar [controle de acesso baseado em papel (RBAC)](/authorization/role-based-access-control) e validação de [JSON Web Token (JWT)](https://auth.wiki/jwt) para proteger seus endpoints de API contra acessos não autorizados.

Nossos guias abrangem configuração de middleware, modelos de permissão e exemplos práticos em várias linguagens de programação e frameworks.
