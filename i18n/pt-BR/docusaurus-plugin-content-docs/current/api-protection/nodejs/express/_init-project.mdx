Para inicializar um novo projeto Node.js com Express, você pode seguir estes passos:

```bash
npm init -y
npm install express
```

Se você estiver usando TypeScript, lembre-se de configurar seu ambiente TypeScript adequadamente.

Em seguida, crie uma configuração básica de servidor Express:

```ts title="app.ts"
import express from 'express';

const app = express();

app.listen(3000, () => {
  console.log('Servidor rodando em http://localhost:3000');
});
```

:::note
Consulte a documentação do Express para mais detalhes sobre como configurar rotas, middlewares e outros recursos.
:::
