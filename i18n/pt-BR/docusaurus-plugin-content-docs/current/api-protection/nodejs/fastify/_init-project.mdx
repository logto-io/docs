Para inicializar um novo projeto Node.js com Fastify, você pode seguir estes passos:

```bash
npm init -y
npm install fastify
```

Se você estiver usando TypeScript, lembre-se de configurar seu ambiente TypeScript adequadamente.

Em seguida, crie uma configuração básica de servidor Fastify:

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Servidor rodando em http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
Consulte a documentação do Fastify para mais detalhes sobre como configurar rotas, plugins e outros recursos.
:::
