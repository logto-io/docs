Para inicializar um novo projeto Node.js com Hapi, você pode seguir estes passos:

```bash
npm init -y
npm install @hapi/hapi
```

Se você estiver usando TypeScript, lembre-se de configurar seu ambiente TypeScript adequadamente.

Em seguida, crie uma configuração básica de servidor Hapi:

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Servidor rodando em http://localhost:3000');
```

:::note
Consulte a documentação do Hapi para mais detalhes sobre como configurar rotas, plugins e outros recursos.
:::
