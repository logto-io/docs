1. Vá para o [Microsoft Entra admin center](https://entra.microsoft.com/) e faça login como administrador.

2. Navegue até Identity > Applications > App registrations.

<center>
  <img src={createApplication} alt="Criar Aplicativo" />
</center>

3. Selecione `New registration`.

4. Insira o nome do aplicativo e selecione o tipo de conta apropriado para seu aplicativo.

5. Selecione `Web` como a plataforma do aplicativo.

6. Copie e cole o `redirect URI` da página de configurações de SSO do Logto. O `redirect URI` é o URL para onde o usuário é redirecionado após se autenticar com o Microsoft Entra ID.

<center>
  <img src={configApplication} alt="Configurar Aplicativo" />
</center>

6. Clique em `Register` para criar o aplicativo.
