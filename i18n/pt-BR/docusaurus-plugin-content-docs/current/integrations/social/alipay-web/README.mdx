# Configurar login social com Alipay (Web)

O conector oficial do Logto para login social com Alipay em aplicativos web.

<GuideTip />

## Primeiros passos \{#get-started}

O conector Alipay Web foi projetado para aplicativos Web de desktop. Ele aproveita o fluxo de autenticação OAuth 2.0 do Alipay e permite que usuários do Alipay façam login em outros aplicativos usando perfis públicos do Alipay, sem passar por um processo de registro trabalhoso.

<Integration />

## Testar o conector Alipay web \{#test-alipay-web-connector}

É isso. Não se esqueça de <MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">habilitar o conector social na experiência de login</MainSiteUrl>.

Depois que o conector Alipay web estiver habilitado, você pode construir e executar seu aplicativo web para ver se está funcionando.
