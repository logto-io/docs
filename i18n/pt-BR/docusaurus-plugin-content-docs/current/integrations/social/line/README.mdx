# Configurar login social com Line

O conector oficial do Logto para login social com Line.

<GuideTip />

## Primeiros passos \{#get-started}

O conector Line permite que os usuários finais façam login no seu aplicativo usando suas próprias contas Line via o protocolo de autenticação OAuth 2.0 do Line.

<Integration />

## Testar o conector Line \{#test-line-connector}

É isso. O conector Line já deve estar disponível. Não se esqueça de [Ativar o conector social no Cadastro e login](/connectors/social-connectors/#enable-social-sign-in).
