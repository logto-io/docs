# Configurar login social com Amazon

O conector oficial do Logto para login social com Amazon.

<GuideTip />

## Introdução \{#get-started}

O conector Amazon permite que os usuários finais façam login no seu aplicativo usando suas próprias contas Amazon via o protocolo de autenticação OAuth 2.0 da Amazon.

<Integration />

## Testar conector Amazon \{#test-amazon-connector}

É isso. O conector Amazon deve estar disponível agora. Não se esqueça de [Habilitar conector social na experiência de login](/connectors/social-connectors/#enable-social-sign-in).
