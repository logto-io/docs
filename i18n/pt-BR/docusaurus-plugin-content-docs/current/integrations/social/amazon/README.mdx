# Configurar login social com Amazon (Set up social login with Amazon)

O conector oficial do Logto para login social com Amazon.

<GuideTip />

## Primeiros passos \{#get-started}

O conector Amazon permite que os usuários finais façam login em seu aplicativo usando suas próprias contas Amazon por meio do protocolo de autenticação OAuth 2.0 da Amazon.

<Integration />

## Testar o conector Amazon \{#test-amazon-connector}

É isso. O conector Amazon já deve estar disponível. Não se esqueça de [habilitar o conector social no cadastro e login](/connectors/social-connectors/#enable-social-sign-in).
