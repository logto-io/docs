# Configurar login social com Hugging Face

O conector oficial do Logto para login social do Hugging Face.

<GuideTip />

## Introdução \{#get-started}

O conector Hugging Face permite que os usuários finais façam login em seu aplicativo usando suas próprias contas Hugging Face via fluxo OAuth / OpenID Connect do Hugging Face.

<Integration />

## Testar conector Hugging Face \{#test-hugging-face-connector}

É isso. O conector Hugging Face deve estar disponível agora. Não se esqueça de <MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">Habilitar conector social na experiência de login</MainSiteUrl>.
