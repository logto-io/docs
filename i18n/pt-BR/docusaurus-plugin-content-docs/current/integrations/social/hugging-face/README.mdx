# Configurar login social com Hugging Face (Set up social login with Hugging Face)

O conector oficial do Logto para login social com Hugging Face.

<GuideTip />

## Primeiros passos \{#get-started}

O conector Hugging Face permite que os usuários finais façam login em seu aplicativo usando suas próprias contas Hugging Face via o fluxo OAuth / OpenID Connect do Hugging Face.

<Integration />

## Testar o conector Hugging Face \{#test-hugging-face-connector}

É isso. O conector Hugging Face já deve estar disponível. Não se esqueça de [habilitar o conector social no cadastro e login](/connectors/social-connectors/#enable-social-sign-in).
