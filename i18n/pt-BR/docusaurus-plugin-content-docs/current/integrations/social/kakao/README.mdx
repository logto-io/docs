# Configurar login social com Kakao

O conector Kakao fornece uma maneira sucinta para seu aplicativo usar o sistema de autenticação OAuth 2.0 do Kakao.

<GuideTip />

## Configurar um projeto no Console de Desenvolvedores do Kakao \{#set-up-a-project-in-the-kakao-devlopers-console}

- Visite o [Console de Desenvolvedores do Kakao](https://developers.kakao.com/console/app) e faça login com sua conta Kakao.
- Clique em **Adicionar um aplicativo** para criar um novo projeto ou escolha um projeto existente.

## Configurar o Login Kakao \{#configure-kakao-login}

### Ativar o Login Kakao \{#activate-kakao-login}

- Clique em **Configurações do Produto -> Login Kakao** no menu.
- Ative `Ativação do Login Kakao`
- Adicione a URL abaixo em `Redirect URI`
  - `http(s)://YOUR_URL/callback/${connector_id}` (O `connector_id` pode ser encontrado na barra superior da página de detalhes do conector no Logto Admin Console.)
  - (Por favor, substitua `YOUR_URL` pela sua URL do `Logto` e escolha `http` ou `https` conforme sua situação.)

### Configuração de Privacidade \{#privacy-setting}

- Clique em **Configurações do Produto -> Login Kakao -> Item de Consentimento** no menu.
- Altere o estado de `Apelido`, `Imagem de perfil` e `Email` para **Consentimento obrigatório** (Você pode não conseguir alterar `Email` para **Consentimento obrigatório** devido à configuração do seu projeto.)

### Configuração de Segurança (Opcional) \{#security-setting-optional}

- Clique em **Configurações do Produto -> Login Kakao -> Segurança** no menu.
- Clique em `Código secreto do cliente` para gerar o código secreto.
- Altere o `Estado de ativação` para Habilitar. (Se você habilitar, o `código secreto` será necessário.)

## Configurar Logto \{#configure-logto}

### Tipos de configuração \{#config-types}

| Nome         | Tipo    |
| ------------ | ------- |
| clientId     | string  |
| clientSecret | string? |

#### clientId \{#clientid}

`clientId` é a `chave da API REST` do seu projeto.
(Você pode encontrá-la no `resumo` do seu projeto no console de desenvolvedores do Kakao.)

#### clientSecret \{#clientseceret}

`clientSecret` é o `Código Secreto` do seu projeto.
(Por favor, verifique [Configuração de Segurança (Opcional)](#security-setting-optional))
