O conector oficial do Logto para o protocolo SAML.

## Introdução \{#get-started}

O conector SAML permite a conexão do Logto a um provedor de identidade social arbitrário que suporta o protocolo SAML.

:::note

O conector SAML é um tipo especial de conector no Logto, você pode adicionar alguns conectores baseados em SAML.
