# Configurar login social com X

O conector oficial do Logto para login social com X (anteriormente Twitter).

<GuideTip />

## Introdução \{#get-started}

O conector X (Twitter) permite que os usuários finais façam login no seu aplicativo usando suas próprias contas X (Twitter) via protocolo de autenticação X OAuth 2.0.

<Integration />

## Testar conector X (Twitter) \{#test-x-connector}

É isso. O conector X (Twitter) deve estar disponível agora. Não se esqueça de <MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">Habilitar conector social na experiência de login</MainSiteUrl>.
