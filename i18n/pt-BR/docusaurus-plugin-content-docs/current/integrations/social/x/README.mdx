# Configurar login social com X (Set up social login with X)

O conector oficial do Logto para login social com X (anteriormente Twitter).

<GuideTip />

## Primeiros passos \{#get-started}

O conector X (Twitter) permite que os usuários finais façam login em seu aplicativo usando suas próprias contas X (Twitter) via o protocolo de autenticação X OAuth 2.0.

<Integration />

## Testar o conector X (Twitter) \{#test-x-connector}

É isso. O conector X (Twitter) já deve estar disponível. Não se esqueça de [Ativar o conector social no Cadastro e login](/connectors/social-connectors/#enable-social-sign-in).
