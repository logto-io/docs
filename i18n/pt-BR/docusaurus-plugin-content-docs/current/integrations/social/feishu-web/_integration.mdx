## Introdução ao login social com Feishu \{#getting-started-with-feishu-social-sign-in}

O conector web do Feishu é projetado para aplicativos web de desktop e utiliza o processo de autenticação OAuth 2.0.

## Registrar uma conta de desenvolvedor Feishu \{#register-a-feishu-developer-account}

Se você ainda não possui uma conta de desenvolvedor Feishu, registre-se na [Plataforma Open Feishu](https://open.feishu.cn/app).

## Criar um aplicativo \{#create-an-application}

1. No [Console do Desenvolvedor](https://open.feishu.cn/app), clique em "Criar aplicativo personalizado".
2. Preencha o nome do aplicativo, descrição, selecione um ícone e clique no botão "Criar".
3. Na barra lateral esquerda, clique em "Configurações de segurança", preencha o "URL de redirecionamento" como `${logto_endpoint}/callback/${connector_id}`. O valor correspondente pode ser encontrado no campo `Callback URI` na página de detalhes do conector Feishu no Logto Console.
4. Em "Credenciais e informações básicas", você pode obter o "App ID" e o "App Secret".

:::note
Para uso interno não empresarial, você também precisa clicar no botão "Criar uma versão" na página "Gerenciamento de versão e lançamento".
O "App ID" e o "App Secret" só terão efeito após o status do aplicativo mudar para "Ativado".
:::
