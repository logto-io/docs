# Configurar login social com WeChat (Nativo) (Set up social login with WeChat (Native))

O conector oficial do Logto para login social do WeChat em aplicativos nativos (iOS e Android).

<GuideTip />

## Primeiros passos \{#get-started}

Se você não conhece o conceito de conector ou não sabe como adicionar este conector à sua experiência de login, consulte o [tutorial do Logto](/connectors/social-connectors).

Em aplicativos nativos, você não pode usar a web como método de login do WeChat: é necessário navegar até o aplicativo WeChat, além de ser obrigatório o uso do SDK deles.

Sabemos que isso pode parecer assustador, mas não se preocupe. Vamos lidar com isso facilmente neste tutorial.

<Integration />

## Testar o conector nativo do WeChat \{#test-wechat-native-connector}

É isso. Não se esqueça de [habilitar o conector social na experiência de login](/connectors/social-connectors/#enable-social-sign-in).

Depois que o conector nativo do WeChat estiver habilitado, você pode compilar e executar seu aplicativo para verificar se está funcionando.

:::caution

O WeChat não possui uma solução para dispositivos sem o aplicativo WeChat instalado. O Logto ocultará este conector durante o login nesses casos (que é a forma recomendada pelo [guia oficial de desenvolvimento](https://developers.weixin.qq.com/doc/oplatform/en/Mobile_App/WeChat_Login/Development_Guide.html)).

:::
