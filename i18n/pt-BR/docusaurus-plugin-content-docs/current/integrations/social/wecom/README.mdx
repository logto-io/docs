# Configurar login social com WeCom (OAuth 2.0)

O conector personalizado para login social do WeCom (talvez chamado WXwork).

<GuideTip />

## Introdução

Faça login no [WeCom WebUI](https://work.weixin.qq.com/) com uma conta de administrador ou clique em **Gerenciar a empresa(管理企业)** no aplicativo WeCom.

Na aba "Gerenciamento de Aplicativos", role a página para baixo e clique em "Criar um aplicativo".

Preencha as informações apropriadas de acordo com seu aplicativo. Em seguida, crie.

Agora temos o Agent ID (NÃO APPID) e o Secret.

### Informações do site

Defina as coisas que você precisa nesta página. Seria algo como:

- Usuários permitidos: _quem pode ver este aplicativo_
- Página inicial do aplicativo: _A página inicial do seu aplicativo. Por exemplo, `logto.io/demo-app`_

**Importante**
Há três itens nesta página relacionados ao "Developer API(开发者接口)".

1. Autorização Web e JS-SDK;
2. Login via autorização pelo WeCom;
3. IP confiável da empresa;

Preencha-os de acordo com o guia do WeCom.

### Corp ID

Se você está familiarizado com o desenvolvimento do WeChat, pode notar que o uso do Corp ID é o mesmo que o APP ID.

Você pode encontrar o Corp ID na parte inferior da página da aba "Minha Empresa(我的企业)". Parece algo como **ww\*\*\*\*** .

## Configurar o conector

Então, temos o Agent ID, Secret e Corp ID.

Vamos completar o formulário para o conector.

Você pode deixar o campo `Scope` em branco, pois é opcional. Alternativamente, você pode preencher com `snsapi_base` ou `snsapi_privateinfo`.

**Salvar e pronto**
