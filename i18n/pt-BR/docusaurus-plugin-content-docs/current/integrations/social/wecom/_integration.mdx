## Primeiros passos \{#get-started}

Faça login no [WeCom WebUI](https://work.weixin.qq.com/) com uma conta de administrador ou clique em **Gerenciar a empresa(管理企业)** no aplicativo WeCom.

Na guia "Gerenciamento de aplicativos", role a página para baixo e clique em "Criar um aplicativo".

Preencha as informações apropriadas de acordo com seu aplicativo. Em seguida, crie.

Agora temos o Agent ID (NÃO APPID) e o Secret.

### Informações do site \{#website-info}

Defina as informações necessárias nesta página. Deve ser algo como:

- Usuários permitidos: _quem pode ver este aplicativo_
- Página inicial do aplicativo: _A página inicial do seu aplicativo. Ex: `logto.io/demo-app`_

**Importante**
Há três itens nesta página relacionados à "API do desenvolvedor(开发者接口)".

1. Autorização Web e JS-SDK;
2. Login via autorização pelo WeCom;
3. IP confiável da empresa;

Preencha-os de acordo com o guia do WeCom.

### Corp ID \{#corp-id}

Se você está familiarizado com o desenvolvimento WeChat, pode notar que o uso do Corp ID é o mesmo que o APP ID.

Você pode encontrar o Corp ID na parte inferior da guia "Minha empresa(我的企业)". Ele se parece com **ww\*\*\*\*** .

## Configurar o conector \{#configure-the-connector}

Agora temos o Agent ID, Secret e Corp ID.

Vamos preencher o formulário para o conector.

Você pode deixar o campo `Scope` em branco, pois é opcional. Alternativamente, você pode preencher com `snsapi_base` ou `snsapi_privateinfo`.

**Salve e pronto**
