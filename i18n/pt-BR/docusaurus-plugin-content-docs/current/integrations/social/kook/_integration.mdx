## Crie um aplicativo no Centro de Desenvolvedores KOOK \{#create-an-application-in-the-kook-developer-center}

Dicas: Você pode pular algumas seções se já tiver concluído.

:::note
Você precisa solicitar permissão OAuth2 do KOOK antes de criar um Aplicativo OAuth2. Por favor, confira as informações na página OAuth2 ou pergunte à equipe do KOOK no servidor oficial de desenvolvedores KOOK para mais informações.
:::

1. Acesse o [Centro de Desenvolvedores KOOK](https://developer.kookapp.cn/app/index) e faça login com sua conta KOOK. Você pode registrar uma nova conta se ainda não tiver uma.
2. Clique no botão **Criar Aplicativo** para criar um aplicativo, escolha um nome para ele (por exemplo, LogtoAuth) e clique em **Confirmar**.
3. Clique no ícone do aplicativo que você criou e vá para a página **OAuth2**.
4. Adicione os redirecionamentos válidos (por exemplo, `${your_logto_origin}/callback/${connector_id}`) nos campos **Callback URIs**. Você pode encontrar o URI de redirecionamento na página de detalhes do conector no console de administração do Logto.
5. Não se esqueça de marcar o escopo **get_user_info** no **Construtor de link OAuth2**.

## Configure seu conector KOOK \{#configure-your-kook-connector}

Preencha os campos clientId e clientSecret com o Client ID e Client Secret que você obteve na página OAuth2 mencionada na seção anterior.

### Tipos de configuração \{#config-types}

| Nome         | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
