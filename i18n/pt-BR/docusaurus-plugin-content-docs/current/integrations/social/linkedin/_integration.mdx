### Configurar um aplicativo LinkedIn \{#setup-a-linkedin-app}

Vá para o [LinkedIn Developers](https://www.linkedin.com/developers) e faça login com sua conta do LinkedIn. Se você não tiver uma conta, pode se registrar para obter uma.

Em seguida, crie um aplicativo.

**Passo 1:** Preencha os Detalhes do Aplicativo.

Complete o formulário e crie o aplicativo.

**Passo 2:** Configure os URLs de retorno de chamada.

Vá para a página de detalhes do aplicativo e encontre a aba "Auth", seção "OAuth 2.0" e encontre o campo "Authorized redirect URLs for your app".

No nosso caso, será `${your_logto_endpoint}/callback/${connector_id}`. por exemplo, `https://foo.logto.app/callback/${connector_id}`. O `connector_id` pode ser encontrado na barra superior da página de detalhes do conector no Logto Admin Console.

**Passo 3:** Adicione o produto.

Vá para a aba "Products" e adicione o produto "Sign In with LinkedIn using OpenID Connect".

### Configure seu conector \{#configure-your-connector}

Na configuração do seu conector Logto, preencha os seguintes campos com os valores obtidos na aba "Auth" do seu aplicativo, seção "Application credentials":

- **clientId:** O Client ID do seu aplicativo.
- **clientSecret:** O Primary Client Secret do seu aplicativo.

`scope` é uma lista de escopos OIDC delimitada por espaços. Se não for fornecido, o escopo padrão é `openid profile email`.

### Tipos de configuração \{#config-types}

| Nome         | Tipo   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
