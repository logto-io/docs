# Configurar login social com LinkedIn

O conector oficial do Logto para login social com LinkedIn.

<GuideTip />

## Introdução \{#get-started}

O conector LinkedIn permite que os usuários finais façam login no seu aplicativo usando suas próprias contas LinkedIn através do protocolo de autenticação LinkedIn OAuth 2.0.
