# Configurar login social com LinkedIn (Set up social login with LinkedIn)

O conector oficial do Logto para login social com LinkedIn.

<GuideTip />

## Primeiros passos \{#get-started}

O conector do LinkedIn permite que os usuários finais façam login em seu aplicativo usando suas próprias contas do LinkedIn por meio do protocolo de autenticação OAuth 2.0 do LinkedIn.

<Integration />

## Testar o conector do LinkedIn \{#test-linkedin-connector}

É isso. O conector do LinkedIn já deve estar disponível. Não se esqueça de [habilitar o conector social no Cadastro e login](/connectors/social-connectors/#enable-social-sign-in).
