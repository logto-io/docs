# Configurar login social com LinkedIn

O conector oficial do Logto para login social do LinkedIn.

<GuideTip />

## Introdução \{#get-started}

O conector LinkedIn permite que os usuários finais façam login no seu aplicativo usando suas próprias contas LinkedIn via o protocolo de autenticação LinkedIn OAuth 2.0.

<Integration />
