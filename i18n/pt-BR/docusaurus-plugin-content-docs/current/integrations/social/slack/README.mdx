# Configurar login social com Slack

O conector oficial do Logto para login social com Slack.

<GuideTip />

## Primeiros passos \{#get-started}

O conector do Slack permite que os usuários finais façam login no seu aplicativo usando suas próprias contas do Slack através do protocolo de autenticação Slack OAuth 2.0.

<Integration />

## Testar o conector do Slack \{#test-slack-connector}

É isso. O conector do Slack já deve estar disponível. Não se esqueça de [habilitar o conector social no cadastro e login](/connectors/social-connectors/#enable-social-sign-in).
