# Configurar login social com Slack

O conector oficial do Logto para login social com Slack.

<GuideTip />

## Introdução \{#get-started}

O conector Slack permite que os usuários finais façam login no seu aplicativo usando suas próprias contas Slack via protocolo de autenticação Slack OAuth 2.0.

<Integration />

## Testar conector Slack \{#test-slack-connector}

É isso. O conector Slack deve estar disponível agora. Não se esqueça de [Habilitar conector social na experiência de login](/connectors/social-connectors/#enable-social-sign-in).
