# Configurar login social com Alipay (Nativo)

O conector oficial do Logto para login social do Alipay em aplicativos nativos de dispositivos móveis.

<GuideTip />

## Primeiros passos \{#get-started}

O conector Alipay Native funciona em conjunto com o Logto SDK em plataformas móveis. Ele aproveita o fluxo de autenticação OAuth 2.0 do Alipay e permite que usuários do Alipay façam login em outros aplicativos usando perfis públicos do Alipay, sem passar por um processo de registro trabalhoso.

<Integration />

### Testar o conector Alipay nativo \{#test-alipay-native-connector}

É isso aí. Não se esqueça de <MainSiteUrl href="/connectors/social-connectors#enable-social-sign-in">habilitar o conector social na experiência de login</MainSiteUrl>.

Depois que o conector Alipay nativo estiver habilitado, você pode compilar e executar seu aplicativo para ver se está funcionando.
