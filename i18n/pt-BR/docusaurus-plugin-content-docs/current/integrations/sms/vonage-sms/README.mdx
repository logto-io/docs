# Configurar verificação SMS com Vonage SMS

O conector oficial do Logto para Vonage SMS.

<GuideTip />

## Primeiros passos \{#get-started}

Vonage é um provedor global de comunicações, oferecendo serviços robustos de comunicação em nuvem, incluindo SMS (serviço de mensagens curtas). O Conector Vonage SMS é um plugin fornecido pela equipe Logto para permitir que usuários finais do Logto se registrem e façam login em sua conta Logto via códigos de verificação SMS.
