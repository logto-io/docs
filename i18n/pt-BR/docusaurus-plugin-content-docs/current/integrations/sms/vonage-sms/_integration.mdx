## Configurar no Vonage \{#set-up-in-vonage}

:::tip
Você pode pular esta etapa se já a tiver concluído.
:::

Para trabalhar com este conector, você precisará [criar uma conta](https://developer.vonage.com/en/account/guides/dashboard-management#create-and-configure-a-vonage-account) no Vonage. Isso fornecerá uma chave de API e um segredo que você pode usar para acessar as APIs por meio deste conector.

Depois de criar uma conta, você pode encontrar sua chave de API e segredo de API no topo do Painel de API do Vonage.

E talvez seja necessário [alugar um número virtual](https://developer.vonage.com/en/numbers/guides/number-management#rent-a-virtual-number) para enviar mensagens SMS.

Consulte a [API de SMS do Vonage](https://developer.vonage.com/en/messaging/sms/overview) para mais informações.

## Configurar no Logto \{set-up-in-logto}

1. **API Key**: Sua chave de API do Vonage.
2. **API Secret**: Seu segredo de API do Vonage.
3. **Brand Name**: O nome da marca que você deseja usar para enviar o SMS, também chamado de campo `from`. Veja [Identidade do Remetente](https://developer.vonage.com/en/messaging/sms/guides/custom-sender-id) para mais informações.
4. **Templates**: Os modelos que você deseja usar para enviar o SMS. Você pode usar os modelos padrão ou modificá-los conforme necessário.
