# Configurar verificação SMS com Aliyun Short Message Service

O conector oficial do Logto para o serviço de mensagem curta da Aliyun.

<GuideTip />

## Primeiros passos \{#get-started}

Aliyun é um dos principais provedores de serviços em nuvem na Ásia, oferecendo diversos serviços em nuvem, incluindo SMS (serviço de mensagem curta). O Conector Aliyun SMS é um plugin fornecido pela equipe Logto para chamar o serviço de SMS da Aliyun, com o qual os usuários finais do Logto podem se registrar e fazer login em sua conta Logto via código de verificação por SMS.
