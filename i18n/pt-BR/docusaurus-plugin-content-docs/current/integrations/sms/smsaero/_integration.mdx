## Registrar conta \{#register-account}

Crie uma nova conta na [SMSAero](https://smsaero.ru/). (Pule para o próximo passo se você já tiver uma.)

## Obter credenciais da conta \{#get-account-credentials}

Vamos precisar das credenciais da API para fazer o conector funcionar. Vamos começar em
[API e SMPP](https://smsaero.ru/cabinet/settings/apikey/).

Copie a "API-key" ou gere uma nova.

## Compor o JSON do conector \{#compose-the-connector-json}

Preencha os campos _email_, _apiKey_ e _senderName_ com seu email, chave de API e nome do remetente.

Você pode preencher o nome do remetente com "SMSAero" para usar o nome de remetente padrão fornecido pela SMSAero.

Você pode adicionar vários modelos de conector SMS para diferentes casos. Aqui está um exemplo de como adicionar um único modelo:

- Preencha o campo `content` com conteúdos arbitrários do tipo string. Não se esqueça de deixar o placeholder `{{code}}` para o código de verificação aleatório.
- Preencha o campo `usageType` com `Register`, `SignIn`, `ForgotPassword` ou `Generic` para diferentes casos de uso.
  Para habilitar todos os fluxos de usuário, são necessários modelos com usageType `Register`, `SignIn`, `ForgotPassword` e `Generic`.

### Testar o conector SMSAero \{#test-smsaero-connector}

Você pode inserir um número de telefone e clicar em "Enviar" para ver se as configurações funcionam antes de "Salvar e Concluir".

É isso. Não se esqueça de <MainSiteUrl href="/connectors/sms-connectors/#enable-phone-number-sign-up-or-sign-in">Habilitar o conector na experiência de login</MainSiteUrl>.

### Tipos de configuração \{#config-types}

| Nome       | Tipo        |
| ---------- | ----------- |
| email      | string      |
| apiKey     | string      |
| senderName | string      |
| templates  | Templates[] |

| Propriedades do Template | Tipo        | Valores enum                                         |
| ------------------------ | ----------- | ---------------------------------------------------- |
| content                  | string      | N/A                                                  |
| usageType                | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
