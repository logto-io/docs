# Conector YunPian SMS

O conector oficial do Logto para o serviço YunPian SMS.

<GuideTip />

## Primeiros passos \{#get-started}

YunPian é um provedor de serviços de comunicação que oferece diversos serviços, incluindo SMS. O Conector YunPian SMS é um plugin fornecido pela equipe Logto para integrar com o serviço de SMS da YunPian, permitindo que usuários finais do Logto se registrem e façam login via códigos de verificação por SMS.

## Configurar o serviço de SMS no Console YunPian \{#set-up-sms-service-in-yunpian-console}

### Criar uma conta YunPian \{#create-a-yunpian-account}

Acesse o [site da YunPian](https://www.yunpian.com/) para registrar uma conta e concluir a verificação de identidade.

### Obter API KEY \{#get-api-key}

1. Faça login no Console YunPian
2. Vá para "Configurações da Conta" -> "Gerenciamento de Subcontas"
3. Encontre e copie a API KEY

### Configurar modelos de SMS \{#configure-sms-templates}

1. No Console YunPian, vá para "SMS Doméstico" -> "Registro de Assinatura"
2. Crie e envie uma assinatura, aguarde a aprovação da operadora
3. Vá para "SMS Doméstico" -> "Registro de Modelo" e selecione "Código de Verificação"
4. Crie um modelo de código de verificação, certifique-se de incluir a variável `#code#` (você também pode usar "Modelos Comuns" para agilizar o processo de aprovação)
5. Aguarde a aprovação do modelo
6. Se precisar enviar SMS internacional, repita os passos acima, mas selecione "SMS Internacional" -> "Registro de Modelo"

## Configurar no Logto \{#configure-in-logto}

1. No Console Logto, vá para "Connectors"
2. Encontre e clique em "YunPian SMS Service"
3. Preencha o formulário de configuração:
   - API KEY: A API KEY obtida na YunPian
   - Modelos de SMS: Configure os modelos conforme o uso, garantindo que correspondam exatamente aos modelos aprovados na YunPian

## Observações \{#notes}

1. O conteúdo do modelo de SMS deve corresponder exatamente ao modelo aprovado na YunPian
2. O espaço reservado da variável do código de verificação nos modelos YunPian é `#code#`, enquanto na configuração do conector é `{{code}}`
3. A YunPian adiciona automaticamente a assinatura padrão com base na API KEY, não é necessário incluir a assinatura no conteúdo do modelo
4. Recomenda-se testar a configuração antes do uso formal
