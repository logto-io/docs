# Configurar verificação SMS com GatewayAPI SMS

O conector oficial do Logto para GatewayAPI SMS.

<GuideTip />

## Primeiros passos \{#get-started}

GatewayAPI é um provedor de serviços em nuvem na Europa, oferecendo diversos serviços em nuvem, incluindo SMS (serviço de mensagem curta). O Conector GatewayAPI SMS é um plugin fornecido pela equipe Logto para chamar o serviço GatewayAPI SMS, com o qual os usuários finais do Logto podem se registrar e fazer login em sua conta Logto via código de verificação SMS.
