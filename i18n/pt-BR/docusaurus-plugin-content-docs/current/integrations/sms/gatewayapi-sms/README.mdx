# Configurar verificação SMS com GatewayAPI SMS

O conector oficial do Logto para GatewayAPI SMS.

<GuideTip />

## Introdução \{#get-started}

GatewayAPI é um provedor de serviços em nuvem na Europa, oferecendo muitos serviços em nuvem, incluindo SMS (serviço de mensagens curtas). O GatewayAPI SMS Connector é um plugin fornecido pela equipe Logto para chamar o serviço SMS do GatewayAPI, com a ajuda do qual os usuários finais do Logto podem se registrar e fazer login em sua conta Logto via código de verificação SMS.

## Configuração no GatewayAPI \{#set-up-in-gatewayapi}

### Criar uma conta GatewayAPI \{#create-an-gatewayapi-account}

Vá para [o site do GatewayAPI](https://www.gatewayapi.com/) e registre sua conta GatewayAPI se você ainda não tiver uma.

### Habilitar conta \{#enable-account}

Você pode precisar habilitar sua conta antes de usar o serviço SMS. Você pode entrar em contato com o serviço de atendimento ao cliente do GatewayAPI para habilitar sua conta.

### Obter token de API \{#get-api-token}

Vá para a página de API Keys no console do GatewayAPI e encontre o token de API ou crie um novo token de API.

## Configuração no Logto \{#set-up-in-logto}

1. **Endpoint**: Se sua conta GatewayAPI estiver na região da UE, você deve usar o endpoint https://gatewayapi.com/rest/mtsms. Se sua conta GatewayAPI estiver na região dos EUA, você deve usar o endpoint https://gatewayapi.com/rest/mtsms.
2. **API Token**: O token de API que você criou na etapa anterior.
3. **Remetente**: O remetente que você deseja usar para enviar o SMS.
4. **Templates**: Os templates que você deseja usar para enviar o SMS, você pode usar os templates padrão ou modificá-los conforme necessário.
