### Criar uma conta GatewayAPI \{#create-an-gatewayapi-account}

Vá para [o site da GatewayAPI](https://www.gatewayapi.com/) e registre sua conta GatewayAPI se você ainda não tiver uma.

### Ativar conta \{#enable-account}

Você pode precisar ativar sua conta antes de usar o serviço de SMS. Você pode entrar em contato com o serviço de atendimento ao cliente da GatewayAPI para ativar sua conta.

### Obter token de API \{#get-api-token}

Vá para a página de API Keys no console da GatewayAPI e encontre o token de API ou crie um novo token de API.

## Configurar no Logto \{#set-up-in-logto}

1. **Endpoint**: Se sua conta GatewayAPI estiver na região da UE, você deve usar o endpoint https://gatewayapi.com/rest/mtsms. Se sua conta GatewayAPI estiver na região dos EUA, você deve usar o endpoint https://gatewayapi.com/rest/mtsms.
2. **Token de API (API Token)**: O token de API que você criou na etapa anterior.
3. **Remetente**: O remetente que você deseja usar para enviar o SMS.
4. **Modelos**: Os modelos que você deseja usar para enviar o SMS, você pode usar os modelos padrão ou modificá-los conforme necessário.
