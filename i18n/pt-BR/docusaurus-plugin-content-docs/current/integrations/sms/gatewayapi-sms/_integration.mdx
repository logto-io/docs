### Criar uma conta GatewayAPI \{#create-an-gatewayapi-account}

Acesse [o site da GatewayAPI](https://www.gatewayapi.com/) e registre sua conta GatewayAPI se ainda não tiver uma.

### Ativar conta \{#enable-account}

Você pode precisar ativar sua conta antes de usar o serviço de SMS. Você pode entrar em contato com o atendimento ao cliente da GatewayAPI para ativar sua conta.

### Obter token de API \{#get-api-token}

Acesse a página de API Keys no console da GatewayAPI e encontre o token de API ou crie um novo token de API.

## Configurar no Logto \{#set-up-in-logto}

1. **Endpoint**: Se sua conta GatewayAPI estiver na região da UE, você deve usar o endpoint https://gatewayapi.com/rest/mtsms. Se sua conta GatewayAPI estiver na região dos EUA, você deve usar o endpoint https://gatewayapi.com/rest/mtsms.
2. **API Token**: O token de API que você criou na etapa anterior.
3. **Remetente (Sender)**: O remetente que você deseja usar para enviar o SMS.
4. **Modelos (Templates)**: Os modelos que você deseja usar para enviar o SMS; você pode usar os modelos padrão ou modificá-los conforme necessário.
