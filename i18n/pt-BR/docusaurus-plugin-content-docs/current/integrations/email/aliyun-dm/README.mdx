# Configurar verificação de email com Aliyun Direct Mail (Set up email verification with Aliyun Direct Mail)

O conector oficial do Logto para o serviço de direct mail da Aliyun.

<GuideTip />

## Primeiros passos \{#get-started}

Aliyun é um dos principais provedores de serviços em nuvem na Ásia, oferecendo diversos serviços em nuvem, incluindo DM (direct mail). O Conector DM da Aliyun é um plugin fornecido pela equipe Logto para chamar as APIs do serviço DM da Aliyun, com o qual os usuários finais do Logto podem se registrar e fazer login em sua conta Logto via código de verificação por email (ou seja, código de verificação).
