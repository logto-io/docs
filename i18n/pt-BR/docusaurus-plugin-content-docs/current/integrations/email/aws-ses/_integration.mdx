## Configurar um serviço de e-mail no console de serviços da AWS \{#configure-a-mail-service-in-the-aws-service-console}

### Registrar conta AWS \{#register-aws-account}

Acesse [AWS](https://aws.amazon.com/) e registre uma conta.

### Criar uma identidade \{#create-a-identity}

- Acesse o Console do `Amazon Simple Email Service`
- Crie uma identidade, escolha uma das seguintes opções:
  - Criar um domínio
  - Criar um endereço de e-mail

### Configuração do conector \{#configuration-of-the-connector}

1. Clique no seu nome de usuário no canto superior direito do console da Amazon para acessar `Credenciais de Segurança`. Se você não tiver uma, crie uma `AccessKey` e salve-a cuidadosamente.
2. Complete as configurações do conector `Amazon Simple Email Service`:
   - Use o `AccessKey ID` e o `AccessKey Secret` obtidos no passo 1 para preencher `accessKeyId` e `accessKeySecret`, respectivamente.
   - `region`: Preencha o campo `region` com a região da identidade que você usa para enviar e-mails.
   - `emailAddress`: O endereço de e-mail que você usa para enviar e-mails, no formato `Logto\<noreply@logto.io>` ou `\<noreply@logto.io>`

os seguintes parâmetros são opcionais; a descrição dos parâmetros pode ser encontrada na [documentação da API AWS SES](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html).

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### Testar o conector Amazon SES \{#test-the-amazon-ses-connector}

Você pode digitar um endereço de e-mail e clicar em "Enviar" para verificar se as configurações funcionam antes de clicar em "Salvar e Concluir".

É isso. Não se esqueça de <MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">Habilitar o conector no cadastro e login</MainSiteUrl>.

### Tipos de configuração \{#configure-types}

| Nome                                      | Tipo              |
| ----------------------------------------- | ----------------- |
| accessKeyId                               | string            |
| accessKeySecret                           | string            |
| region                                    | string            |
| emailAddress                              | string (OPCIONAL) |
| emailAddressIdentityArn                   | string (OPCIONAL) |
| feedbackForwardingEmailAddress            | string (OPCIONAL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPCIONAL) |
| configurationSetName                      | string (OPCIONAL) |
| templates                                 | Template[]        |

| Propriedades do Template | Tipo        | Valores enum                                         |
| ------------------------ | ----------- | ---------------------------------------------------- |
| subject                  | string      | N/A                                                  |
| content                  | string      | N/A                                                  |
| usageType                | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
