## Configurar conector SMTP \{#set-up-smtp-connector}

SMTP é um protocolo de transmissão que não é exclusivo de alguns provedores de serviço de e-mail específicos, mas pode funcionar com todos os provedores.

Agora estamos oferecendo guias sobre como usar o conector SMTP para enviar e-mails seguindo os provedores abaixo para seu melhor entendimento:

- _Gmail_ é o provedor de serviço de e-mail mais popular do mundo.
- _Aliyun direct mail_ e _SendGrid mail_. Alguns de vocês podem estar familiarizados com esses dois provedores de serviço de e-mail porque a equipe Logto forneceu conectores correspondentes; você provavelmente já tem uma ideia geral sobre eles.

Esperamos que você consiga configurar outros provedores de e-mail com os exemplos a seguir :rocket:

### Configuração para uso com Gmail \{#set-up-for-gmail-use}

Você pode obter uma nova conta do Gmail em [Gmail](https://mail.google.com/), ou pode usar uma conta existente se já tiver uma.

Um [post oficial do Gmail](https://support.google.com/a/answer/176600) mostra como determinar os valores das propriedades necessárias para operar o Gmail via um conector SMTP.

### Integração com SendGrid SMTP API \{#integrate-with-sendgrid-smtp-api}

Inicialmente, assumimos que você já possui uma conta SendGrid. Caso não tenha, crie uma nova conta no [site do SendGrid](https://app.sendgrid.com/).

Você pode encontrar um guia passo a passo em ["Integrating with the SMTP API"](https://docs.sendgrid.com/for-developers/sending-email/integrating-with-the-smtp-api).

Desenvolvedores podem acessar detalhes do _remetente_ em ["Sender Management"](https://mc.sendgrid.com/senders).

### Configuração com conta Aliyun direct mail \{#configure-with-aliyun-direct-mail-account}

Faça login no [site da Aliyun](https://cn.aliyun.com/). Registre uma nova conta se ainda não tiver uma.

Siga o guia [Send emails using SMTP](https://www.alibabacloud.com/help/en/directmail/latest/send-emails-using-smtp) e conclua as 'tarefas' para obter as configurações e informações necessárias.

Você pode acessar a [página de endereço do serviço SMTP](https://www.alibabacloud.com/help/en/directmail/latest/smtp-service-address) para escolher um host de endereço de serviço SMTP e número de porta adequados.

Para verificar "Endereços de remetente", você pode encontrar a entrada no painel de navegação à esquerda no [console DirectMail](https://dm.console.aliyun.com/). Você deve ver `Sender address` e `SMTP Password` aqui.

:::note
Apenas um modelo de exemplo é fornecido nos casos anteriores para manter as coisas simples. Você deve adicionar mais modelos para outros casos de uso.
Você deve alterar os valores entre "\<" e ">" de acordo com as configurações da sua conta Gmail, SendGrid ou Aliyun e optar por manter os outros campos sem "\<" e ">".
Adicione `{{code}}` como um placeholder no conteúdo dos modelos para exibir o código de verificação aleatório ao enviar e-mails.
:::

### Testar conector SMTP \{#test-smtp-connector}

Você pode digitar um endereço de e-mail e clicar em "Enviar" para ver se as configurações funcionam antes de "Salvar e Concluir".

É isso. Não se esqueça de <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">habilitar o conector na experiência de login</MainSiteUrl>.

### Tipos de configuração \{#config-types}

| Nome      | Tipo       |
| --------- | ---------- |
| host      | string     |
| port      | string     |
| fromEmail | string     |
| templates | Template[] |

| Propriedades do Template | Tipo        | Valores enum                                         |
| ------------------------ | ----------- | ---------------------------------------------------- |
| subject                  | string      | N/A                                                  |
| content                  | string      | N/A                                                  |
| usageType                | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| contentType              | enum string | 'text/plain' / 'text/html'                           |

**Opções de autenticação Username e password**

| Nome | Tipo                   | Valores enum |
| ---- | ---------------------- | ------------ |
| user | string                 | N/A          |
| pass | string                 | N/A          |
| type | enum string (OPCIONAL) | 'login'      |

Você também pode configurar [Opções de autenticação OAuth2](https://nodemailer.com/smtp/oauth2/) e outras configurações avançadas. Veja [aqui](https://nodemailer.com/smtp/) para mais detalhes.

Fornecemos um exemplo de configuração com todos os parâmetros configuráveis na caixa de texto para ajudá-lo a configurar sua própria configuração. (Você é responsável pela configuração, alguns valores são apenas para demonstração e podem não se adequar ao seu caso de uso.)
