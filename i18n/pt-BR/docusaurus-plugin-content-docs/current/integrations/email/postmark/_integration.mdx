## Registrar conta Postmark \{#register-postmark-account}

Crie uma nova conta no [site do Postmark](https://postmark.com/). Você pode pular esta etapa se já possui uma conta.

## Configurar seu conector \{#configure-your-connector}

Preencha o campo `serverToken` com o Server Token que você encontra nas configurações do seu servidor no Postmark.

Preencha o campo `fromEmail` com o _Endereço do Remetente_ (From Address).

Para habilitar todos os fluxos de usuário, são necessários templates com usageType `Register`, `SignIn`, `ForgotPassword` e `Generic`.

Aqui está um exemplo de JSON de template de conector Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Testar conector de email Postmark \{#test-postmark-email-connector}

Você pode digitar um endereço de email e clicar em "Enviar" para verificar se as configurações funcionam antes de "Salvar e Concluir".

É isso. Não se esqueça de <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">Habilitar o conector na experiência de login</MainSiteUrl>.

## Tipos de configuração \{#config-types}

| Nome        | Tipo   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propriedades do Template | Tipo        | Valores enum                                         |
| ------------------------ | ----------- | ---------------------------------------------------- |
| usageType                | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| templateAlias            | string      | N/A                                                  |
