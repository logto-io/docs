# Configurar verificação de email com serviço de email Postmark

Conector Logto para o serviço de email Postmark.

<GuideTip />

## Introdução \{#get-started}

Postmark é uma plataforma de email para emails transacionais e de marketing. Podemos usar sua função de envio de email para enviar um _código de verificação_.

## Registrar conta Postmark \{#register-postmark-account}

Crie uma nova conta no [site do Postmark](https://postmark.com/). Você pode pular esta etapa se já tiver uma conta.

## Configurar seu conector \{#configure-your-connector}

Preencha o campo `serverToken` com o Token do Servidor que você encontra nas configurações do seu servidor no Postmark.

Preencha o campo `fromEmail` com o _Endereço de Remetente_.

Para habilitar fluxos completos de usuários, são necessários templates com usageType `Register`, `SignIn`, `ForgotPassword` e `Generic`.

Aqui está um exemplo de JSON de template de conector Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Testar conector de email Postmark \{#test-postmark-email-connector}

Você pode digitar um endereço de email e clicar em "Enviar" para ver se as configurações funcionam antes de "Salvar e Concluir".

É isso. Não se esqueça de [Habilitar conector na experiência de login](/connectors/email-connectors/#enable-email-sign-up-or-sign-in)

## Tipos de configuração \{#config-types}

| Nome        | Tipo   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propriedades do Template | Tipo        | Valores do enum                                        |
| ------------------------ | ----------- | ------------------------------------------------------ |
| usageType                | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias            | string      | N/A                                                    |
