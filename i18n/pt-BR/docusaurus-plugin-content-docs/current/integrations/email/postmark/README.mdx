# Configurar verificação de email com Postmark email service

Conector Logto para o serviço de email Postmark.

<GuideTip />

## Introdução \{#get-started}

Postmark é uma plataforma de e-mails para transacionais e marketing. Podemos usar sua função de envio de e-mails para enviar um _código de verificação_.

## Registrar conta no Postmark \{#register-postmark-account}

Crie uma nova conta no [site do Postmark](https://postmark.com/). Você pode pular esta etapa se já possui uma conta.

## Configure seu conector \{#configure-your-connector}

Preencha o campo `serverToken` com o Server Token que você encontra nas configurações do seu servidor no Postmark.

Preencha o campo `fromEmail` com o _From Address_ do remetente.

Para habilitar todos os fluxos de usuário, são necessários templates com usageType `Register`, `SignIn`, `ForgotPassword` e `Generic`.

Aqui está um exemplo de JSON de template do conector Postmark.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Testar o conector de email Postmark \{#test-postmark-email-connector}

Você pode digitar um endereço de e-mail e clicar em "Enviar" para ver se as configurações funcionam antes de "Salvar e Concluir".

É isso. Não se esqueça de <MainSiteUrl href="/connectors/email-connectors/#enable-email-address-for-end-user-flows">Habilitar o conector no cadastro e login</MainSiteUrl>.

## Tipos de configuração \{#config-types}

| Nome        | Tipo   |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Propriedades do Template | Tipo        | Valores enum                                         |
| ------------------------ | ----------- | ---------------------------------------------------- |
| usageType                | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| templateAlias            | string      | N/A                                                  |
