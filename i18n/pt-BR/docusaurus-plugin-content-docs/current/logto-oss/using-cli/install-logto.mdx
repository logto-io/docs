# Instalar Logto

## Interativo

Execute o comando abaixo para proceder com uma instalação interativa no seu terminal:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## Opções e instalação silenciosa

Alternativamente, você pode adicionar as opções abaixo para pular algumas perguntas:

```bash
# Todas essas opções são opcionais
--db-url      A URL do Postgres para o banco de dados Logto
-p, --path    Caminho para sua instância Logto, deve ser um caminho inexistente
--ss          Pular a semeadura do banco de dados Logto
--oc          Adicionar conectores oficiais após a instalação
```

Execute `logto init --help` para obter informações completas de ajuda.

Aqui está um exemplo para realizar uma instalação silenciosa (por exemplo, em CI):

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

Ele fará as seguintes coisas em ordem:

1. Baixar e extrair Logto em `./logto`
2. Tentar inicializar e semear o banco de dados com a URL `postgresql://your-postgres-dsn:5432/logto`
3. Adicionar conectores oficiais do Logto em `./logto/packages/core/connectors`
