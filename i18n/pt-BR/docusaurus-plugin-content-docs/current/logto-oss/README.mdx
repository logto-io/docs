# Logto OSS

O Logto oferece versões open-source (OSS) e [cloud](/logto-cloud). Esta seção foca na solução OSS: <Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

O Logto OSS é uma solução de identidade gratuita que oferece aos desenvolvedores total capacidade de personalização e controle completo sobre seus dados. O Logto OSS oferece flexibilidade para adaptar o código conforme necessidades específicas, sendo ideal para quem prioriza soberania de dados e deseja uma solução de provedor de identidade personalizada que possa crescer junto com seu aplicativo.

## Recursos suportados pelo Logto OSS \{#feature-supported-by-logto-oss}

## Recursos suportados pelo Logto OSS \{#feature-supported-by-logto-oss}

O Logto OSS suporta a maioria das capacidades principais do serviço Logto e é regularmente atualizado e mantido.

Alguns recursos avançados são atualmente exclusivos da versão Logto Cloud, incluindo:

| Limitações de recursos no OSS                                                              | Descrição                                                                                                                                                    |
| ------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [Console Logto: múltiplos tenants](/logto-cloud/tenant-settings)                           | Crie e gerencie múltiplos tenants Logto em um console.                                                                                                       |
| [Console Logto: convite de colaboradores](/logto-cloud/tenant-member-management)           | Convide e gerencie múltiplos membros de um tenant Logto.                                                                                                     |
| Console Logto: Ativar MFA                                                                  | Aumente a segurança exigindo autenticação multifatorial ao acessar o Console Logto.                                                                          |
| [Logto Protected App](/integrate-logto/protected-app)                                      | Integração rápida e sem SDK de autenticação, alimentada pelo Cloudflare.                                                                                     |
| [Serviço de email integrado do Logto](/connectors/email-connectors/built-in-email-service) | Utilize o serviço de email integrado e gratuito para envio de emails.                                                                                        |
| [Traga sua UI](/customization/bring-your-ui)                                               | Personalize a UI ou os fluxos da experiência de login usando este recurso (no OSS, é possível fazer fork do código no GitHub para personalizar diretamente). |
| [SSO iniciado pelo IdP](/end-user-flows/enterprise-sso/idp-initiated-sso)                  | Ative autenticação única corporativa iniciada pelo provedor de identidade.                                                                                   |
| [Apps SAML](/integrate-logto/saml-app)                                                     | Integre aplicativos via protocolo SAML. A versão OSS é limitada a 3 apps SAML.                                                                               |

Dicas: Multi-tenancy, convites de membros e MFA não estão disponíveis para sua equipe acessar um console Logto open-source. No entanto, você pode implementar esses recursos em seu próprio produto usando Logto OSS, tornando-os disponíveis para seus usuários finais.
