# Logto OSS

Logto oferece versões open-source (OSS) e [cloud](/logto-cloud). Esta seção se concentrará na solução OSS: <Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS é uma solução de identidade gratuita que oferece aos desenvolvedores total capacidade de personalização e controle completo sobre seus dados. Logto OSS oferece a flexibilidade de adaptar a base de código para atender a necessidades específicas, sendo uma escolha perfeita para aqueles que priorizam a soberania dos dados e desejam uma solução de provedor de identidade personalizada que possa crescer com seu aplicativo.

## Recursos suportados pelo Logto OSS

Logto OSS suporta a maioria das capacidades principais do serviço Logto e é regularmente atualizado e mantido.

Alguns recursos avançados são atualmente exclusivos da versão Logto Cloud, incluindo [Logto Protected App](/integrate-logto/protected-app/), [serviço de email gratuito](/connectors/email-connectors/built-in-email-service), [traga sua UI](/customization/bring-your-ui), [SSO iniciado pelo IdP](/end-user-flows/enterprise-sso/idp-initiated-sso) e IdP SAML.
