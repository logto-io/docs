# Desenvolver seu conector

Atualmente, o Logto forneceu muitos conectores de login social amplamente utilizados, como WeChat, Alipay, Google, Facebook, etc., e conectores de SMS / Email, como AliCloud, SendGrid e Twilio.

Aqui está um diagrama de arquitetura ilustrando como os conectores funcionam com o serviço central do Logto.

![Estrutura do Conector](./assets/logto-connector-framework.png)

Eles devem ser capazes de satisfazer a maioria das necessidades dos usuários nesta fase, mas continuaremos progredindo.

Enquanto trabalhamos em mais conectores, você também pode desenvolver seus próprios conectores. Você é incentivado a compartilhar seus conectores com toda a comunidade.

Vamos ver como contribuir para o projeto Logto começando com a construção de um conector.
