# Implementar verificação de segurança via Management API

## Verificação de senha \{#password-verification}

Quando os usuários tentam alterar sua senha, o sistema deve solicitar que eles insiram novamente sua senha atual para verificar sua identidade. Essa etapa garante que apenas o titular autorizado da conta possa alterar a senha, prevenindo o acesso não autorizado à conta.

| método | caminho                                                                                                  | descrição                                        |
| ------ | -------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| POST   | [/api/users/\{userId\}/password/verify](https://openapi.logto.io/operation/operation-verifyuserpassword) | Verificar a senha atual do usuário pelo user ID. |
