# Verificação de segurança

Quando usuários autenticados tentam realizar ações sensíveis ou de alto risco — como alterar sua senha, fazer um pagamento ou acessar informações sensíveis como contracheques ou detalhes de contas bancárias — medidas de segurança adicionais são essenciais. Esse fluxo é especialmente crítico em aplicativos sensíveis, como serviços bancários, de saúde e governamentais.

Esse processo, conhecido como _verificação de segurança_, exige que os usuários re-verifiquem sua identidade para confirmar que são os titulares autorizados da conta. Ao implementar a verificação de segurança, o sistema reforça a proteção contra acesso não autorizado, adicionando uma camada crítica de segurança para ações de alto risco e ajudando a proteger dados sensíveis.

Saiba como adicionar o processo de verificação de segurança ao seu aplicativo usando as [Management APIs](/end-user-flows/security-verification/by-management-api) do Logto.
