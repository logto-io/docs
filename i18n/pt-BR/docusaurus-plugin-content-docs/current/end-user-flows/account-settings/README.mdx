# Configurações de conta

Logto fornece duas coleções de APIs de configurações de conta para permitir que os usuários gerenciem suas contas e perfis armazenados no Logto.

## APIs de Gerenciamento de Usuários

As Management APIs do Logto oferecem a funcionalidade mais abrangente para gerenciar usuários e seus dados de perfil. Você pode implementar um sistema de gerenciamento de usuários auto-hospedado ou uma página de configurações de conta de usuário final usando essas APIs.

Saiba mais sobre como usar as [APIs de Gerenciamento de Usuários](/end-user-flows/account-settings/by-management-api) para construir sua própria página de configurações de conta.

## Account APIs

Além das Management APIs, o Logto também fornece um conjunto de Account APIs para permitir que os usuários finais gerenciem diretamente suas configurações de conta e dados de perfil. Ao contrário das Management APIs, as Account APIs são projetadas para serem usadas no contexto do usuário final com controle detalhado de permissões de usuário. Você pode usar as Account APIs diretamente em seu aplicativo do lado do cliente para construir uma página de configurações de conta amigável ao usuário.

Saiba mais sobre como usar as [Account APIs](/end-user-flows/account-settings/by-account-api) para construir sua própria página de configurações de conta.
