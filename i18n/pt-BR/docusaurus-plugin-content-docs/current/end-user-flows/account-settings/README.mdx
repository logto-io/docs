# Configurações de conta

Logto fornece duas coleções de APIs de configurações de conta para permitir que os usuários gerenciem suas contas e perfis armazenados no Logto.

## APIs de Gerenciamento de Usuários

As Management APIs do Logto oferecem a funcionalidade mais abrangente para gerenciar usuários e seus dados de perfil. Você pode implementar um sistema de gerenciamento de usuários auto-hospedado ou uma página de configurações de conta de usuário final usando essas APIs.

## APIs de Conta

Além das Management APIs, o Logto também fornece um conjunto de APIs de conta de usuário final para permitir que os usuários finais gerenciem diretamente suas configurações de conta e dados de perfil. Ao contrário das Management APIs, as APIs de conta são projetadas para serem usadas no contexto do usuário final com controle detalhado de permissões de usuário. Você pode usar as APIs de conta diretamente em seu aplicativo do lado do cliente para construir uma página de configurações de conta amigável ao usuário.
