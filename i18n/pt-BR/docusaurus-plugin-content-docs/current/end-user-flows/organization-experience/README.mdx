# Experiência da organização

A experiência da [Organização](/organizations) refere-se às interfaces de usuário e fluxos usados por seus clientes empresariais e seus funcionários, particularmente em [aplicativos multi-tenant](https://auth.wiki/multi-tenancy). O Logto fornece vários guias para ajudá-lo a integrar rapidamente essa experiência em seu aplicativo usando o Logto Management API.
