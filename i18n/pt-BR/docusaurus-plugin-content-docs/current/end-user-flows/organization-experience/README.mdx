# Experiência de organização

A experiência de [Organização (Organization)](/organizations) refere-se às interfaces e fluxos usados por seus clientes empresariais e seus funcionários, especialmente em [aplicativos multi-inquilino](https://auth.wiki/multi-tenancy). O Logto fornece vários guias para ajudá-lo a integrar rapidamente essa experiência ao seu aplicativo usando a Logto Management API.
