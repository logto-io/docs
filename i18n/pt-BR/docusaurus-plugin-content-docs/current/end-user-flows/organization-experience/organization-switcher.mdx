# Seletor de organização

Na experiência de organização, os usuários podem alternar entre organizações. Cada vez que alternam, o aplicativo busca um novo token de organização para carregar as informações relevantes.

<img src="/img/assets/organization-switcher.png" alt="Seletor de organização" />

## Como implementar \{#how-to-implement-it}

1. Crie um dropdown ou componente de interface semelhante para listar todas as organizações disponíveis para o usuário.
2. Quando o usuário selecionar uma organização diferente, obtenha seu ID e chame o método do SDK `getOrganizationToken(organizationId)` para buscar o token de organização correspondente. (O SDK gerencia o cache de tokens para você.)
3. Armazene o ID da organização selecionada (na URL ou no armazenamento do cliente) e sempre use-o para buscar o token de organização pareado para recursos específicos da organização.
