# Fluxos do usuário final

Os fluxos do usuário final cobrem todos os processos de verificação para interações do usuário, categorizados da seguinte forma:

- **Experiência de login**: Envolve login do usuário, registro e redefinição de senha. Logto oferece experiências prontas configuradas no Console, e você pode trazer sua interface de usuário via Experience APIs.
- **Experiência de conta**: Envolve configurações de conta, perfil do usuário e verificação de segurança. Use Account APIs ou Management APIs para implementar.
- **Experiência de organização**: Para produtos de multi-tenancy, como serviços B2B ou SaaS. Envolve criação de organização, convite de membros e gerenciamento. Use Management APIs para integrar isso em seus aplicativos.
