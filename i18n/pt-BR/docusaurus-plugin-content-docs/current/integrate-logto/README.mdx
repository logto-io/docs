# Integrar autenticação Logto

O Logto oferece soluções abrangentes de autenticação para aplicações web, mobile e desktop, suporta autenticação [máquina para máquina (M2M)](/quick-starts/m2m) entre serviços e pode atuar como um Provedor de Identidade (IdP) para [aplicativos de terceiros](/integrate-logto/third-party-applications) através de protocolos padrão como [OpenID Connect (OIDC)](https://auth.wiki/openid-connect) e [OAuth 2.0](https://auth.wiki/oauth-2.0).

Comece sua integração selecionando a solução que melhor atende às suas necessidades:

## Adicionar autenticação aos seus aplicativos \{#add-authentication-for-your-applications}

Seja você desenvolvendo aplicativos voltados para o usuário (como web, mobile ou desktop) ou aplicativos máquina para máquina (M2M) para comunicação serviço a serviço, é possível implementar rapidamente recursos completos de [autenticação](/end-user-flows) e [gestão de usuários](/user-management) integrando o Logto.

## Adicionar aplicativos OIDC / OAuth \{#add-oidc-oauth-apps}

Quando você deseja permitir que aplicativos de terceiros autentiquem com seu sistema de identidade Logto, é possível configurar o Logto como um [Provedor de Identidade (IdP)](https://auth.wiki/identity-provider).

O Logto pode ser usado como IdP através de protocolos padrão como OAuth, OIDC, e serviços externos podem implementar "Entrar com Logto" assim como "Entrar com Google", permitindo que seus usuários autentiquem usando seu sistema de usuários Logto.

Ao criar aplicativos de terceiros no Logto, você estabelece conexões seguras entre seu sistema de identidade e serviços externos.

## Estenda os recursos do Logto com Management API \{#extend-logto-features-with-management-api}

A [Management API](https://openapi.logto.io/) do Logto permite controlar e automatizar programaticamente as funcionalidades do Logto em seus aplicativos.

Quando você precisa estender as capacidades do Logto para aprimorar seu produto (como [gerenciar organizações multi-inquilino](/organizations), ou automatizar fluxos de trabalho com o Logto), é possível aproveitar a Management API do Logto para construir recursos avançados que atendam às necessidades do seu produto.
