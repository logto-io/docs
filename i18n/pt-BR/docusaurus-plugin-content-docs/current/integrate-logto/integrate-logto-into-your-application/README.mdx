# Integrar Logto ao seu aplicativo

Siga estas etapas para adicionar autenticação aos seus aplicativos com Logto, seja um aplicativo voltado para o usuário ou serviço máquina para máquina:

1. Navegue até <CloudLink to="/applications">Console > Applications</CloudLink>

2. Clique em "Create application" para adicionar um novo aplicativo

3. Escolha o framework do seu aplicativo para começar. Se você não encontrar seu framework, clique no botão "Create app without framework" no canto inferior direito da página de criação do aplicativo para criar um aplicativo selecionando um [Application type](/integrate-logto/application-data-structure/#application-types) ou envie uma solicitação de recurso ou contribua com um SDK seguindo nossas [SDK conventions](/developers/sdk-conventions).

4. Após selecionar seu framework, você verá um guia de início rápido para o SDK do framework. Siga as etapas para configurar e integrar seu aplicativo. Se precisar de ajuda para entender os conceitos envolvidos no processo de integração, você pode consultar [Understanding Logto authentication flow](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) para uma compreensão mais profunda da integração.

:::note
O guia no console é apenas para um início rápido com Logto usando nosso SDK. Para guias de integração completos, incluindo uso avançado do SDK, confira a seção [Quick starts](/quick-starts).
:::

5. Uma vez concluído, você está pronto para explorar mais sobre Logto:
