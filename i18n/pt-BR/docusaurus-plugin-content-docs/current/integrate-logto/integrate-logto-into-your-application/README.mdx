# Integre o Logto ao seu aplicativo

Siga estes passos para adicionar autenticação aos seus aplicativos com o Logto, seja um app voltado para o usuário ou um serviço máquina para máquina:

1. Navegue até <CloudLink to="/applications">Console > Aplicativos</CloudLink>

2. Clique em "Criar aplicativo" para adicionar um novo aplicativo

3. Escolha seu [framework de aplicativo](/quick-starts) para começar. Se você não encontrar seu framework, clique no botão "Criar app sem framework" no canto inferior direito da página de criação do aplicativo para criar um app selecionando um [Tipo de aplicativo](/integrate-logto/application-data-structure/#application-types) ou envie uma solicitação de recurso ou contribua com um SDK seguindo nossas [convenções de SDK](/developers/sdk-conventions).

4. Após selecionar seu framework, você verá um guia de início rápido para o SDK do framework. Siga os passos para configurar e integrar seu aplicativo. Se precisar de ajuda para entender os conceitos envolvidos no processo de integração, consulte [Entendendo o fluxo de autenticação do Logto](/integrate-logto/integrate-logto-into-your-application/understand-authentication-flow/) para um entendimento mais profundo da integração.

:::note
O guia no console é apenas para início rápido com o Logto usando nosso SDK. Para guias completos de integração, incluindo uso avançado do SDK, confira a seção de [Inícios rápidos](/quick-starts).
:::

5. Depois de concluir, você está pronto para explorar mais sobre o Logto:
