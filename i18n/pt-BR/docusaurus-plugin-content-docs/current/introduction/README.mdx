# Introdução ao Logto

Bem-vindo à documentação do Logto! O Logto é uma solução de gerenciamento de identidade e acesso (IAM) projetada para aplicativos modernos e produtos SaaS. Ele fornece um sistema seguro, escalável e personalizável de autenticação (Authentication) e autorização (Authorization) para suas aplicações.
