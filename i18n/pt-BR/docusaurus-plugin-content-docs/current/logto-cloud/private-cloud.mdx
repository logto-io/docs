# Logto Private Cloud

Implemente o Logto em um ambiente Azure dedicado para maior isolamento, alinhamento com conformidade e controle de infraestrutura.

## Visão geral \{#overview}

Logto Private Cloud é uma opção de implantação dedicada para clientes que precisam de mais isolamento e controle de infraestrutura do que um ambiente de nuvem pública compartilhada.

É projetado para organizações com requisitos mais rigorosos de segurança, conformidade, aquisição ou implantação, mantendo a mesma experiência principal do produto que o Logto Cloud.

Veja como as opções de implantação diferem:

- **Logto Cloud**: nuvem pública gerenciada, caminho mais rápido para produção
- **Logto Private Cloud**: implantação dedicada para clientes com requisitos mais rigorosos de infraestrutura ou conformidade
- **Logto OSS**: auto-hospedado e auto-gerenciado

## Quando escolher Private Cloud \{#when-to-choose-private-cloud}

Logto Private Cloud geralmente é a escolha certa quando sua organização exige um ou mais dos seguintes:

- Maior isolamento de ambiente
- Compromisso de tempo de atividade respaldado por SLA, com suporte e colaboração da equipe Logto
- Requisitos específicos de conformidade que não podem ser atendidos por uma nuvem pública compartilhada
- Implantação em uma região ou infraestrutura específica que não é suportada pelo Logto Cloud
- Conexão com recursos internos que não podem ser acessados a partir de um ambiente de nuvem pública

Se você não possui esses requisitos, [Logto Cloud](https://cloud.logto.io/?sign_up=1) geralmente é a melhor escolha, pois é mais simples de adotar e mais fácil de operar.

### Escopo da implantação \{#deployment-scope}

Logto Private Cloud está atualmente disponível apenas no **Azure**.

As regiões suportadas dependem se a região do Azure fornece os serviços necessários para o Logto. Se você tiver uma região alvo em mente, entre em contato conosco e podemos confirmar a viabilidade com base em seus requisitos.

### Segurança e conformidade \{#security-and-compliance}

Private Cloud oferece um ambiente dedicado com limites de isolamento mais claros do que uma implantação em nuvem pública compartilhada.

Mantemos as discussões sobre segurança e conformidade práticas e orientadas por requisitos. Se sua equipe precisar de uma revisão de segurança, discussão de conformidade ou esclarecimento específico de implantação, entre em contato conosco e podemos analisar os detalhes diretamente.
