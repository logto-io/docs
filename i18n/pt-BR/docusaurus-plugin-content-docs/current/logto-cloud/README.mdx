# Serviço em nuvem Logto

[Logto Cloud](https://cloud.logto.io) oferece uma gama de serviços de gerenciamento e operacionais para ajudá-lo a gerenciar identidades e recursos de forma suave e fácil. Hospedado pelo Logto, inclui recursos como [suporte a múltiplas regiões](/logto-cloud/tenant-settings#tenant-region), gerenciamento de locatários, [faturamento e preços](/logto-cloud/billing-and-pricing), [gerenciamento de membros](/logto-cloud/tenant-member-management) e controle de acesso baseado em papel no console.

Se você tiver alguma dúvida sobre os serviços em nuvem e precisar de suporte adicional, entre em contato conosco.
