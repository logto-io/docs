# Entenda como as organizações funcionam

Para obter uma visão holística de entendimento, é importante compreender as entidades envolvidas nos recursos de organização, pois estes são os fundamentos para integrar organizações ao seu aplicativo.

## Organização (Organization) \{#organization}

Organização consiste em um grupo de [usuários (identidades)](/user-management/user-data). Ela pode representar equipes, clientes empresariais e empresas parceiras que podem acessar seu aplicativo.

A introdução de uma organização como entidade é importante, pois não apenas agrupa usuários, mas também fornece um contexto para isolamento de locatários em aplicativos multi-inquilino.

## Membro da organização (Organization member) \{#organization-member}

No Logto, um usuário que possui a associação a uma organização é chamado de membro da organização (ou seja, membro) dentro do contexto dessa organização.

## Aplicativo M2M da organização (Organization M2M application) \{#organization-m2m-application}

Aplicativo M2M da organização refere-se aos [aplicativos máquina para máquina](/quick-starts/m2m) que estão associados a uma organização. Assim como os [membros da organização](#organization-member), os aplicativos da organização estão restritos ao contexto de uma organização.

Depois de associar um aplicativo a uma organização, o aplicativo pode receber [papéis da organização](/authorization/role-based-access-control#configure-organization-roles). Isso é útil quando você deseja permitir acesso sem interação aos recursos da organização.

:::note

Outros tipos de aplicativos não suportam associação à organização e papéis de organização, pois são projetados para interação do usuário.

:::

Consulte esta [seção](/organizations/organization-management) para gerenciar organizações, membros e aplicativos.

## Modelo de organização (Organization template) \{#organization-template}

O modelo de organização é projetado para necessidades de controle de acesso em aplicativos multi-inquilino. Ele inclui papéis da organização e permissões da organização. Confira este [modelo de organização](/authorization/organization-template) para saber mais sobre o design de controle de acesso em organização.
