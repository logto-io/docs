# Estrutura de dados da organização

No nível da organização, existem vários atributos cujos valores podem ser criados, atualizados, lidos e excluídos (CRUD).

## ID da Organização

O _ID da organização_ é um identificador único para cada organização. É útil para implementar experiências de login no nível da organização e recuperar tokens de organização.

## Nome

O _nome_ suporta login no nível da organização e pode ser integrado em interfaces de produtos no nível da organização conforme necessário.

## Descrição

O campo _descrição_ permite adicionar texto para ajudar a identificar e rotular a organização.

## Logos da organização

Para mostrar dinamicamente o logotipo da organização do seu cliente na experiência de login, você pode fazer upload dos logos da organização na página de configurações da organização.

Veja [logos específicos da organização](/customization/match-your-brand/#organization-specific-logos) para mais detalhes.

## Dados personalizados

_Dados personalizados_ é um objeto JSON usado para armazenar informações extras sobre a organização. Ele oferece aos desenvolvedores flexibilidade para armazenar detalhes adicionais da organização para fins de desenvolvimento.
