# Gerenciamento de usuários

O gerenciamento de usuários é uma área de foco significativa para o Logto. Você pode acessar seus dados de usuário e gerenciá-los usando o Logto Console ou o Management API, ambos eficazes. Com essas ferramentas, você pode realizar tarefas como

- Pesquisar usuários por nome de usuário, ID, nome completo, número de telefone ou email.
- Realizar várias ações no usuário, como desvincular contas sociais, atualizar informações de perfil, excluir usuários ou redefinir senhas.
- Verificar logs de usuários.
- Atribuir papéis para controle de acesso.
- Criar e gerenciar tokens de acesso pessoal.
