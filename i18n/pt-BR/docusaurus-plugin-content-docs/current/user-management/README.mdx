# Gerenciamento de usuários

O gerenciamento de usuários é uma área de grande foco para o Logto. Você pode acessar e gerenciar os dados dos seus usuários utilizando tanto o Logto Console quanto a Management API, ambos eficazes. Com essas ferramentas, você pode realizar tarefas como

- Pesquisar usuários por nome de usuário, ID, nome completo, número de telefone ou e-mail.
- Realizar várias ações no usuário, como desvincular contas sociais, atualizar informações do perfil, excluir usuários ou redefinir senhas.
- Verificar logs do usuário.
- Atribuir papéis para controle de acesso.
- Criar e gerenciar tokens de acesso pessoal.
- Gerenciar sessões de usuários.
