# Tipos de eventos dos logs de auditoria

Você pode filtrar os tipos de eventos em <CloudLink to="/audit-logs">Logto Console > Logs de Auditoria</CloudLink>.

:::note

O Logto agora suporta a recuperação de logs relacionados às interações dos usuários finais via [Experience APIs](https://openapi.logto.io/group/endpoint-experience).

Os logs de auditoria para [Management APIs](/integrate-logto/interact-with-management-api) e [Account APIs](https://openapi.logto.io/group/endpoint-account-center) estarão disponíveis em breve.

Sinta-se à vontade para [entrar em contato conosco](https://logto.io/contact) caso queira compartilhar seus requisitos.

:::
