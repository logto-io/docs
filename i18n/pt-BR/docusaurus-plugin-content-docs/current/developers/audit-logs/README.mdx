# Logs de auditoria

O log de auditoria do Logto permite monitorar facilmente a atividade e os eventos dos usuários. Ele fornece uma base sólida para diversos cenários de negócios de gerenciamento de usuários e verificação de integridade.

## Visualizar todos os logs \{#view-all-logs}

Navegue até <CloudLink to="/audit-logs">Console > Logs de auditoria</CloudLink>. O Logto captura e organiza eventos de autenticação em uma tabela. Ele registra o nome do evento, usuário, aplicativo e carimbo de data / hora. Você pode refinar os resultados filtrando pelo nome do evento e nome do aplicativo. Ao clicar em um evento específico, serão exibidos detalhes adicionais.

:::warning
Os logs de auditoria contêm apenas logs que ocorrem durante o processo de autenticação do usuário; logs de operações da Management API não são registrados.
:::

## Capturar atividade do usuário no nível do tenant \{#capture-user-activity-at-the-tenant-level}

Os logs do Logto oferecem detalhes abrangentes, garantindo facilidade de ação e segurança do cliente. Eles capturam e registram as seguintes informações:

- Tipo de evento (a lista completa de eventos de log de auditoria pode ser encontrada [aqui](/developers/audit-logs/event-types))
- Aplicativo envolvido
- Endereço IP
- Usuário envolvido
- ID do log
- Carimbo de data / hora
- User-agent

Ao manter esses registros de eventos, as organizações podem detectar possíveis riscos de segurança de forma eficaz e agir rapidamente para evitar acessos não autorizados ao sistema.

## Realizar uma análise detalhada no nível do usuário \{#perform-a-detailed-analysis-at-the-user-level}

Os administradores podem realizar uma análise detalhada dos logs associados a usuários específicos, facilitando investigações abrangentes sobre eventos específicos. O processo de navegação é simples e intuitivo.

Para acessar logs específicos do usuário, siga estes passos:

1. Navegue até <CloudLink to="/users">Console > Gerenciamento de usuários</CloudLink>.
2. Selecione o usuário desejado e vá para a página de detalhes.
3. Clique em "Logs do usuário". A tabela resultante exibirá exclusivamente eventos de log realizados e acionados por aquele usuário em particular.
