# Estratégia de design

- Cada linguagem de programação deve ter um repositório git isolado chamado `${language}`.
- Cada repositório de linguagem de programação deve ser um mono repo.
- Tanto os SDKs quanto seus projetos de exemplo associados devem ser colocados neste repositório.

Exemplos:

<details>
  <summary>JS</summary>

- js (core)
- react
- react-sample

</details>

<details>
  <summary>Kotlin</summary>

- kotlin (core)
- android
- android-sample

</details>
