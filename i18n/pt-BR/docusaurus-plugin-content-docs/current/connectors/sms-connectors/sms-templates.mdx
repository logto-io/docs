Logto fornece quatro modelos diferentes para personalizar o conteúdo de SMS, que são categorizados com base no tipo de uso: Registro, Login, EsqueceuSenha e Genérico. É altamente recomendável que você use diferentes modelos para vários casos de uso, ou pode atingir o limite de taxa, levando a uma interrupção temporária do seu serviço.

## Tipos de modelos de SMS e exemplos \{#sms-template-types-and-examples}

Aqui estão alguns exemplos apenas para referência:

| Uso           | Cenário                                                                                                                                          | Exemplo de Modelo                                                                                                 |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| Registro      | Usuários criam uma conta usando seu email e a verificam inserindo um código de verificação enviado pelo Logto para seu email.                    | Seu código de verificação de inscrição do Logto é `{{code}}`. O código permanecerá ativo por 10 minutos.          |
| Login         | Usuários fazem login usando seu email e verificam inserindo o código de verificação em vez de inserir uma senha.                                 | Seu código de verificação de login do Logto é `{{code}}`. O código permanecerá ativo por 10 minutos.              |
| EsqueceuSenha | Se os usuários esquecerem sua senha durante o login, eles podem optar por verificar sua identidade usando o email que já verificaram com Logto.  | Seu código de verificação de alteração de senha do Logto é `{{code}}`. O código permanecerá ativo por 10 minutos. |
| Genérico      | Este modelo pode ser usado como uma opção de backup geral para vários cenários, incluindo testar configurações de conectores e assim por diante. | Seu código de verificação do Logto é `{{code}}`. O código permanecerá ativo por 10 minutos.                       |

É importante entender esses parâmetros:

- O código de verificação é válido por 10 minutos. Atualmente, não suportamos personalização no tempo de expiração.
- Logto substituirá o espaço reservado `{{code}}` no modelo de SMS por um código de verificação. Portanto, certifique-se de que o modelo tenha um espaço reservado reservado.

:::note
Alguns países e regiões podem não permitir o envio de conteúdo não aprovado via SMS devido a requisitos de conformidade. Os modelos de SMS precisam ser registrados e aprovados pelo provedor de SMS antes de serem usados. Nesses casos, o conteúdo pode ser indexado pelo ID do modelo para o modelo correspondente.
:::
