# Personalização

Deixe o Logto com a sua cara personalizando todos os aspectos — desde a identidade visual e interface do usuário até idiomas e configurações de domínio.

## UI pré-construída \{#prebuilt-ui}

Comece com nossos componentes de UI pré-construídos e personalize elementos essenciais da marca. Crie:

- Uma experiência de autenticação unificada em toda a sua plataforma.
- Páginas de login dedicadas para aplicativos ou organizações específicas.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Combine com sua marca',
      href: '/customization/match-your-brand',
      description:
        'Saiba como personalizar os elementos básicos da marca para uma experiência de login e central de contas pronta para uso.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## Conteúdo & Idiomas \{#content--languages}

O Logto oferece suporte a 15 idiomas por padrão. Expanda o suporte a idiomas ou modifique traduções existentes conforme sua necessidade.

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Idiomas & Conteúdo',
      href: '/customization/localized-languages',
      description:
        'Saiba como adicionar novos idiomas e personalizar o conteúdo para seus usuários.',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## Domínio personalizado \{#custom-domain}

Fortaleça a presença da sua marca usando seu próprio domínio para URLs de autenticação. Substitua o domínio padrão do Logto pelo seu domínio personalizado.

  items={[
    {
      type: 'link',
      label: 'Domínio personalizado',

      customProps: {

  ]}
/>
