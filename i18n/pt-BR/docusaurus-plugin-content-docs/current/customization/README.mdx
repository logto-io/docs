# Personalização

Faça do Logto o seu próprio, personalizando todos os aspectos - desde a marca e interface do usuário até as configurações de idiomas e domínio.

## UI Pré-construída

Comece com nossos componentes de UI pré-construídos e personalize elementos essenciais da marca. Crie:

- Uma experiência de autenticação unificada em sua plataforma.
- Páginas de login dedicadas para aplicativos ou organizações específicas.

  items={[
    {
      type: 'link',
      label: 'Combine com sua marca',

        'Aprenda a personalizar os elementos básicos da marca para uma experiência de login pronta para uso.',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## Conteúdo & Idiomas

Logto oferece suporte para 15 idiomas por padrão. Expanda o suporte a idiomas ou modifique traduções existentes para atender às suas necessidades.

## Domínio personalizado

Fortaleça a presença da sua marca usando seu domínio para URLs de autenticação. Substitua o domínio padrão do Logto pelo seu domínio de marca.
