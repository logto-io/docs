```dart title="lib/main.dart"
// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",
  appId: "<your-app-id>",
  // highlight-start
  scopes: ["email", "phone"],
  // highlight-end
);
```

Também fornecemos um enum `LogtoUserScope` embutido no pacote SDK para ajudá-lo a usar os escopos (Scopes) predefinidos.

```dart
// LogtoConfig
final logtoConfig = const LogtoConfig(
  endpoint: "<your-logto-endpoint>",
  appId: "<your-app-id>",
  // highlight-start
  scopes: [LogtoUserScope.email.value, LogtoUserScope.phone.value],
  // highlight-end
);
```
