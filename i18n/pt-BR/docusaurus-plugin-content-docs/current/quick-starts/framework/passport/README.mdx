# Adicionar autenticação ao seu aplicativo Passport.js

Este guia mostrará como integrar o Logto ao seu aplicativo com [Passport.js](https://www.passportjs.org/) e estratégia OIDC.

<GuideTip />

## Pré-requisitos

- Uma conta [Logto Cloud](https://cloud.logto.io) ou um [Logto auto-hospedado](/introduction/set-up-logto-oss).
- Um aplicativo tradicional Logto criado.
- Um projeto express com sessão configurada. Confira o [site do Express.js](https://expressjs.com/).
