### Exibir informações do usuário

Para exibir as informações do usuário, você pode usar o método `getIdTokenClaims()`. Por exemplo, em um aplicativo Capacitor:

```tsx
const userClaims = await logtoClient.getIdTokenClaims();
console.log(userClaims);
```

Para solicitar escopos adicionais, você pode passá-los para o objeto `LogtoConfig` ao inicializar o cliente. Por exemplo:

```tsx
const logtoConfig = {
  scopes: ['email', 'phone', 'custom_data', 'organizations'],
};
```

Então você pode acessar as reivindicações adicionais no valor de retorno de `client.getIdTokenClaims()`:
