Se organização é um conceito novo para você, por favor, leia [🏢 Organizações (Multi-tenancy)](/organizations) para começar.

Você precisa adicionar o escopo <code>{props.organizationScope}</code> ao configurar o cliente Logto:

{props.configOrganizationCode}

Uma vez que o usuário esteja autenticado, você pode buscar o token de organização para o usuário:

{props.getOrganizationAccessTokenCode}
