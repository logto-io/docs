O aplicativo M2M faz uma solicitação `POST` para o endpoint de token para obter um token de acesso, adicionando os seguintes parâmetros usando o formato `application/x-www-form-urlencoded` no corpo da entidade da solicitação HTTP:

- **grant_type**: Deve ser definido como `client_credentials`
- **resource**: O recurso que você deseja acessar
- **scope**: O escopo da solicitação de acesso

Você também precisa incluir as credenciais do seu aplicativo M2M no cabeçalho da solicitação para que o endpoint de token possa autenticar seu aplicativo M2M.

Isso é feito incluindo as credenciais do aplicativo na forma de [autenticação Basic](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Authorization#basic_authentication) no cabeçalho `Authorization` da solicitação: codifique em base64 `{App ID}:{App Secret}` e defina como valor após o prefixo `Basic`.
