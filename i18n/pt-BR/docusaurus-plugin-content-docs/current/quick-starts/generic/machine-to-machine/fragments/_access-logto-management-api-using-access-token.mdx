<TabItem value="with-@logto-api-sdk" label="Node.js com `@logto/api` SDK">

Com o SDK `@logto/api`, você pode interagir diretamente com qualquer Management API usando o trecho de código abaixo. O token de acesso (Access token) é armazenado em cache internamente e atualizado automaticamente se necessário.

```js
import { createManagementApi } from '@logto/api/management';

const { apiClient } = createManagementApi('your-tenant-id', {
  clientId: 'your-client-id',
  clientSecret: 'your-client-secret',
});

const response = await apiClient.GET('/api/applications');
console.log(response.data);
```

<TabItem value="nodejs" label="Node.js">

<AccessTokenUsage />

```js
const logtoEndpoint = 'https://your.logto.endpoint'; // Substitua pelo seu endpoint Logto
const accessToken = 'eyJhb...2g'; // Token de acesso (Access Token)

const fetchLogtoApplications = async () => {
  return await fetch(`${logtoEndpoint}/api/applications`, {
    method: 'GET',
    headers: {
      Authorization: `Bearer ${accessToken}`,
    },
  });
};
```

</TabItem>

<TabItem value="curl" label="cURL">

<AccessTokenUsage />

```bash
curl --location \
  --request GET 'https://your.logto.endpoint/api/applications' \
  --header 'Authorization: Bearer eyJhbG...2g'
```

Lembre-se de substituir os valores reais pelos seus próprios. O valor após `Bearer` deve ser o token de acesso (Access token) (JWT) que você recebeu.
