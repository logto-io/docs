# Configurar permissões

## Configurar permissões de API via Logto Console \{#configure-api-permissions-via-logto-console}

As permissões são definidas no nível **do recurso de API (API resource)**. Navegue até o <CloudLink to="/api-resources">Console > Recursos de API</CloudLink>, clique no recurso de API que você deseja gerenciar e, em seguida, vá para a aba "Permissões".

É importante fornecer um nome e detalhes para a permissão, pois isso facilitará a identificação e o gerenciamento das permissões.

## Configurar permissões de API via Logto Management API \{#configure-api-permissions-via-logto-management-api}

Gerencie usando o Logto Management API. Faça uma chamada para o endpoint relativo. Confira esta [referência](https://openapi.logto.io/operation/operation-listresourcescopes).

| método | caminho                                                                                                              | descrição                                                                                                    |
| ------ | -------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| GET    | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-listresourcescopes)              | Obter escopos definidos para um recurso de API.                                                              |
| POST   | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-createresourcescope)             | Criar um novo escopo para um recurso de API.                                                                 |
| DELETE | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-deleteresourcescope) | Excluir um escopo de recurso de API do recurso fornecido.                                                    |
| PATCH  | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-updateresourcescope) | Atualizar um escopo de recurso de API para o recurso fornecido. Este método realiza uma atualização parcial. |
