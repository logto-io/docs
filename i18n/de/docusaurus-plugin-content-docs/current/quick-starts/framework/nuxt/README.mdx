## Verwendung in Middleware oder API-Routen \{#use-in-middleware-or-api-routes}

Die Composables `useLogtoClient()` und `useLogtoUser()` sind in Middleware oder API-Routen nicht verfügbar. Du kannst die Funktion `logtoEventHandler()` verwenden, um den Logto-Client und weiteren Kontext zu erhalten:

```ts title="/server/api/auth/get-access-token.ts"
import { logtoEventHandler } from '#logto';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
