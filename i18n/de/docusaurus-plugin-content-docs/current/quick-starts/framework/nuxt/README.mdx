## Verwendung in Middleware oder API-Routen \{#use-in-middleware-or-api-routes}

Die Komposablen `useLogtoClient()` und `useLogtoUser()` sind in Middleware oder API-Routen nicht verf端gbar. Du kannst die Funktion `logtoEventHandler()` verwenden, um den Logto-Client und andere Kontexte zu erhalten:

```ts title="api/auth/get-access-token.ts"
import { logtoEventHandler } from '@logto/nuxt';

export default defineEventHandler(async (event) => {
  const config = useRuntimeConfig(event);
  await logtoEventHandler(event, config);
  const accessToken = await event.context.logtoClient.getAccessToken();
  return { accessToken };
});
```
