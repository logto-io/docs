# Authentifizierung zu deiner Auth.js (Next Auth)-Anwendung hinzufügen

Dieser Leitfaden zeigt dir, wie du Logto in deine Next.js-Anwendung mit [Auth.js](https://authjs.dev/) integrierst, früher bekannt als Next Auth.

<GuideTip />

## Voraussetzungen \{#prerequisites}

- Ein [Logto Cloud](https://cloud.logto.io)-Konto oder ein [selbst gehostetes Logto](/introduction/set-up-logto-oss).
- Eine erstellte traditionelle Logto-Anwendung.
- Ein Next.js-Projekt mit Auth.js, siehe die [Auth.js-Dokumentation](https://authjs.dev/getting-started/installation).
