### Routen registrieren

Das SDK bietet eine Hilfsfunktion `handleAuthRoutes`, um 3 Routen zu registrieren:

1. `/logto/sign-in`: Anmeldung mit Logto.
2. `/logto/sign-in-callback`: Anmelde-Callback verarbeiten.
3. `/logto/sign-out`: Abmeldung mit Logto.

F端ge den folgenden Code zu deiner App hinzu:

```ts title="app.ts"
import { handleAuthRoutes } from '@logto/express';

app.use(handleAuthRoutes(config));
```

### Anmeldung und Abmeldung implementieren

Mit den registrierten Routen implementieren wir nun die Anmelde- und Abmeldebuttons auf der Startseite. Wir m端ssen den Benutzer bei Bedarf zur Anmelde- oder Abmelderoute umleiten. Um dabei zu helfen, verwende `withLogto`, um den Authentifizierungsstatus in `req.user` zu injizieren.

```ts title="app.ts"
import { withLogto } from '@logto/express';

app.get('/', withLogto(config), (req, res) => {
  res.setHeader('content-type', 'text/html');

  if (req.user.isAuthenticated) {
    res.end(`<div>Hallo ${req.user.claims?.sub}, <a href="/logto/sign-out">Abmelden</a></div>`);
  } else {
    res.end('<div><a href="/logto/sign-in">Anmelden</a></div>');
  }
});
```
