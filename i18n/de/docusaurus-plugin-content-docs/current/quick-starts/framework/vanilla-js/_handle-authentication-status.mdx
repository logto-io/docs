Im Logto SDK können wir im Allgemeinen `logtoClient.isAuthenticated` verwenden, um den Authentifizierungsstatus zu überprüfen. Wenn der Benutzer angemeldet ist, wird der Wert `true` sein, andernfalls wird der Wert `false` sein.

In deiner Vanilla-JS-App kannst du den `isAuthenticated`-Status verwenden, um die Anmelde- und Abmeldebuttons programmatisch anzuzeigen und auszublenden. Schauen wir uns an, wie das geht.

```js
const redirectUrl = 'http://localhost:3000/callback';
const baseUrl = 'http://localhost:3000';

// Bedingte Darstellung der Anmelde- und Abmeldebuttons
const render = async (logtoClient) => {
  const isAuthenticated = await logtoClient.isAuthenticated();
  const container = document.querySelector('#container');

  const onClickSignIn = () => logtoClient.signIn(redirectUrl);
  const onClickSignOut = () => logtoClient.signOut(baseUrl);

  const button = document.createElement('button');
  button.innerHTML = isAuthenticated ? 'Abmelden' : 'Anmelden';
  button.addEventListener('click', isAuthenticated ? onClickSignOut : onClickSignIn);

  container.append(button);
};
```
