## SMTP-Connector einrichten \{#set-up-smtp-connector}

SMTP ist ein Übertragungsprotokoll, das nicht exklusiv für bestimmte E-Mail-Dienstanbieter ist, sondern mit allen Anbietern funktioniert.

Wir bieten nun Anleitungen, wie du den SMTP-Connector zum Versenden von E-Mails mit den folgenden Anbietern nutzen kannst, um dir ein besseres Verständnis zu ermöglichen:

- _Gmail_ ist der weltweit beliebteste E-Mail-Dienstanbieter.
- _Aliyun Direct Mail_ und _SendGrid Mail_. Einige von euch sind vielleicht mit diesen beiden E-Mail-Dienstanbietern vertraut, da das Logto-Team entsprechende Connectors bereitgestellt hat; du hast wahrscheinlich bereits eine allgemeine Vorstellung davon.

Wir hoffen, dass du mit den folgenden Beispielen die Einrichtung aller anderen E-Mail-Anbieter nachvollziehen kannst :rocket:

### Einrichtung für die Nutzung von Gmail \{#set-up-for-gmail-use}

Du kannst ein neues Gmail-Konto bei [Gmail](https://mail.google.com/) erstellen oder ein bestehendes Konto verwenden, falls du bereits eines hast.

Ein [offizieller Gmail-Beitrag](https://support.google.com/a/answer/176600) zeigt, wie du die erforderlichen Eigenschaften bestimmst, um Gmail über einen SMTP-Connector zu betreiben.

### Integration mit SendGrid SMTP API \{#integrate-with-sendgrid-smtp-api}

Wir gehen zunächst davon aus, dass du bereits ein SendGrid-Konto hast. Falls nicht, erstelle ein neues Konto auf der [SendGrid-Website](https://app.sendgrid.com/).

Eine Schritt-für-Schritt-Anleitung findest du unter ["Integrating with the SMTP API"](https://docs.sendgrid.com/for-developers/sending-email/integrating-with-the-smtp-api).

Entwickler können _Absender_-Details unter ["Sender Management"](https://mc.sendgrid.com/senders) einsehen.

### Konfiguration mit Aliyun Direct Mail-Konto \{#configure-with-aliyun-direct-mail-account}

Melde dich auf der [Aliyun-Website](https://cn.aliyun.com/) an. Registriere ein neues Konto, falls du noch keines hast.

Folge der Anleitung [Send emails using SMTP guide](https://www.alibabacloud.com/help/en/directmail/latest/send-emails-using-smtp) und erledige die dort genannten „Aufgaben“, um die erforderlichen Einstellungen und Informationen zu erhalten.

Du kannst auf die [SMTP-Service-Adressseite](https://www.alibabacloud.com/help/en/directmail/latest/smtp-service-address) gehen, um einen passenden SMTP-Service-Adress-Host und Port auszuwählen.

Um „Absenderadressen“ zu prüfen, findest du den Zugang im linken Navigationsbereich der [DirectMail-Konsole](https://dm.console.aliyun.com/). Hier solltest du `Absenderadresse` und `SMTP-Passwort` sehen.

:::note
In den vorherigen Fällen wurde nur eine Beispielvorlage bereitgestellt, um die Dinge einfach zu halten. Du solltest weitere Vorlagen für andere Anwendungsfälle hinzufügen.
Du solltest Werte, die mit "\<" und ">" umschlossen sind, entsprechend deiner Gmail-, SendGrid- oder Aliyun-Kontoeinstellungen ändern und andere Felder ohne "\<" und ">" beibehalten.
Füge `{{code}}` als Platzhalter im Inhalt der Vorlagen ein, um den zufälligen Bestätigungscode in gesendeten E-Mails anzuzeigen.
:::

### SMTP-Connector testen \{#test-smtp-connector}

Du kannst eine E-Mail-Adresse eingeben und auf „Senden“ klicken, um zu prüfen, ob die Einstellungen funktionieren, bevor du auf „Speichern und Fertig“ klickst.

Das war's. Vergiss nicht, den <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">Connector in der Anmeldeerfahrung zu aktivieren</MainSiteUrl>.

### Konfigurationstypen \{#config-types}

| Name      | Typ        |
| --------- | ---------- |
| host      | string     |
| port      | string     |
| fromEmail | string     |
| templates | Template[] |

| Template-Eigenschaften | Typ         | Enum-Werte                                           |
| ---------------------- | ----------- | ---------------------------------------------------- |
| subject                | string      | N/A                                                  |
| content                | string      | N/A                                                  |
| usageType              | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| contentType            | enum string | 'text/plain' / 'text/html'                           |

**Benutzername und Passwort Auth-Optionen**

| Name | Typ                    | Enum-Werte |
| ---- | ---------------------- | ---------- |
| user | string                 | N/A        |
| pass | string                 | N/A        |
| type | enum string (OPTIONAL) | 'login'    |

Du kannst auch [OAuth2 Auth-Optionen](https://nodemailer.com/smtp/oauth2/) und andere erweiterte Konfigurationen einstellen. Siehe [hier](https://nodemailer.com/smtp/) für weitere Details.

Wir haben ein Beispiel für eine Konfiguration mit allen konfigurierbaren Parametern im Textfeld bereitgestellt, um dir beim Einrichten deiner eigenen Konfiguration zu helfen. (Du bist für die Konfiguration verantwortlich, einige Werte dienen nur zu Demonstrationszwecken und passen möglicherweise nicht zu deinem Anwendungsfall.)
