## SMTP-Connector einrichten \{#set-up-smtp-connector}

SMTP ist ein Übertragungsprotokoll, das nicht exklusiv für bestimmte E-Mail-Dienstanbieter ist, sondern mit allen Anbietern funktionieren kann.

Wir bieten nun Anleitungen an, wie man den SMTP-Connector verwenden kann, um E-Mails mit den folgenden Anbietern zu versenden, um dein Verständnis zu verbessern:

- _Gmail_ ist der weltweit beliebteste E-Mail-Dienstanbieter.
- _Aliyun direct mail_ und _SendGrid mail_. Einige von euch sind möglicherweise mit diesen beiden E-Mail-Dienstanbietern vertraut, da das Logto-Team entsprechende Connectors bereitgestellt hat; du wirst wahrscheinlich eine allgemeine Vorstellung von ihnen haben.

Wir hoffen, dass du mit den folgenden Beispielen die Einrichtung aller anderen E-Mail-Anbieter herausfinden kannst :rocket:

### Einrichtung für die Nutzung von Gmail \{#set-up-for-gmail-use}

Du kannst ein neues Gmail-Konto bei [Gmail](https://mail.google.com/) erstellen oder ein bestehendes Konto verwenden, wenn du eines hast.

Ein [offizieller Gmail-Beitrag](https://support.google.com/a/answer/176600) zeigt, wie man die erforderlichen Eigenschaftenwerte bestimmt, um Gmail über einen SMTP-Connector zu betreiben.

### Integration mit SendGrid SMTP API \{#integrate-with-sendgrid-smtp-api}

Zunächst gehen wir davon aus, dass du bereits ein SendGrid-Konto hast. Falls nicht, erstelle ein neues Konto auf der [SendGrid-Website](https://app.sendgrid.com/).

Du findest eine Schritt-für-Schritt-Anleitung zur ["Integration mit der SMTP API"](https://docs.sendgrid.com/for-developers/sending-email/integrating-with-the-smtp-api).

Entwickler können auf _Absender_-Details im ["Sender Management"](https://mc.sendgrid.com/senders) zugreifen.

### Konfiguration mit Aliyun direct mail Konto \{#configure-with-aliyun-direct-mail-account}

Melde dich auf der [Aliyun-Website](https://cn.aliyun.com/) an. Registriere ein neues Konto, wenn du keines hast.

Folge der [Anleitung zum Versenden von E-Mails mit SMTP](https://www.alibabacloud.com/help/en/directmail/latest/send-emails-using-smtp) und erledige diese 'Aufgaben', um die erforderlichen Einstellungen und Informationen zu erhalten.

Du kannst zur [SMTP-Dienstadressenseite](https://www.alibabacloud.com/help/en/directmail/latest/smtp-service-address) gehen, um einen geeigneten SMTP-Dienstadress-Host und eine Portnummer auszuwählen.

Um "Absenderadressen" zu überprüfen, findest du den Zugang im linken Navigationsbereich auf der [DirectMail-Konsole](https://dm.console.aliyun.com/). Du solltest hier `Absenderadresse` und `SMTP-Passwort` sehen.

:::note
In den vorherigen Fällen wird nur eine Beispielvorlage bereitgestellt, um die Dinge einfach zu halten. Du solltest mehr Vorlagen für andere Anwendungsfälle hinzufügen.
Du solltest Werte, die mit "\<" und ">" umschlossen sind, gemäß deinen Gmail-, SendGrid- oder Aliyun-Kontoeinstellungen ändern und andere Felder ohne "\<" und ">" beibehalten.
Füge `{{code}}` als Platzhalter im Inhalt der Vorlagen hinzu, um zufällige Bestätigungscodes in gesendeten E-Mails anzuzeigen.
:::

### SMTP-Connector testen \{#test-smtp-connector}

Du kannst eine E-Mail-Adresse eingeben und auf "Senden" klicken, um zu sehen, ob die Einstellungen funktionieren, bevor du "Speichern und Fertig" auswählst.

Das war's. Vergiss nicht, den [Connector in der Anmeldeerfahrung zu aktivieren](/connectors/email-connectors/#enable-email-sign-up-or-sign-in).

### Konfigurationstypen \{#config-types}

| Name      | Typ        |
| --------- | ---------- |
| host      | string     |
| port      | string     |
| fromEmail | string     |
| templates | Template[] |

| Template-Eigenschaften | Typ         | Enum-Werte                                              |
| ---------------------- | ----------- | ------------------------------------------------------- |
| subject                | string      | N/A                                                     |
| content                | string      | N/A                                                     |
| usageType              | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| contentType            | enum string | 'text/plain' \| 'text/html'                             |

**Benutzername und Passwort Auth-Optionen**

| Name | Typ                    | Enum-Werte |
| ---- | ---------------------- | ---------- |
| user | string                 | N/A        |
| pass | string                 | N/A        |
| type | enum string (OPTIONAL) | 'login'    |

Du kannst auch [OAuth2 Auth-Optionen](https://nodemailer.com/smtp/oauth2/) und andere erweiterte Konfigurationen einrichten. Siehe [hier](https://nodemailer.com/smtp/) für weitere Details.

Wir haben ein Beispielkonfiguration mit allen konfigurierbaren Parametern im Textfeld bereitgestellt, um dir bei der Einrichtung deiner eigenen Konfiguration zu helfen. (Du bist für die Konfiguration verantwortlich, einige Werte dienen Demonstrationszwecken und passen möglicherweise nicht zu deinem Anwendungsfall.)
