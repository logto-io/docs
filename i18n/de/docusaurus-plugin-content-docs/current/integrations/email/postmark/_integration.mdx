## Postmark-Konto registrieren \{#register-postmark-account}

Erstelle ein neues Konto auf der [Postmark-Website](https://postmark.com/). Du kannst diesen Schritt überspringen, wenn du bereits ein Konto hast.

## Connector konfigurieren \{#configure-your-connector}

Fülle das Feld `serverToken` mit dem Server Token aus, den du unter den Einstellungen für deinen Server in Postmark findest.

Fülle das Feld `fromEmail` mit der _Absenderadresse_ aus.

Um vollständige Benutzerflüsse zu ermöglichen, sind Vorlagen mit usageType `Register`, `SignIn`, `ForgotPassword` und `Generic` erforderlich.

Hier ist ein Beispiel für das Postmark-Connector-Template im JSON-Format.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Postmark-E-Mail-Connector testen \{#test-postmark-email-connector}

Du kannst eine E-Mail-Adresse eingeben und auf „Senden“ klicken, um zu prüfen, ob die Einstellungen funktionieren, bevor du auf „Speichern und Fertig“ klickst.

Das war's. Vergiss nicht, den <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">Connector in der Anmeldeerfahrung zu aktivieren</MainSiteUrl>.

## Konfigurationstypen \{#config-types}

| Name        | Typ    |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Template-Eigenschaften | Typ         | Enum-Werte                                           |
| ---------------------- | ----------- | ---------------------------------------------------- |
| usageType              | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| templateAlias          | string      | N/A                                                  |
