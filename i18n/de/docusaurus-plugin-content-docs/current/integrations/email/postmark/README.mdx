# E-Mail-Verifizierung mit Postmark-E-Mail-Dienst einrichten

Logto Connector für den Postmark-E-Mail-Dienst.

<GuideTip />

## Erste Schritte \{#get-started}

Postmark ist eine Plattform für transaktionale und Marketing-E-Mails. Wir können seine E-Mail-Versandfunktion nutzen, um einen _Verifizierungscode_ zu senden.

## Postmark-Konto registrieren \{#register-postmark-account}

Erstelle ein neues Konto auf der [Postmark-Website](https://postmark.com/). Du kannst diesen Schritt überspringen, wenn du bereits ein Konto hast.

## Deinen Connector konfigurieren \{#configure-your-connector}

Fülle das `serverToken`-Feld mit dem Server-Token aus, das du unter den Einstellungen für deinen Server in Postmark findest.

Fülle das `fromEmail`-Feld mit der _From Address_ des Absenders aus.

Um vollständige Benutzerflüsse zu ermöglichen, sind Vorlagen mit dem usageType `Register`, `SignIn`, `ForgotPassword` und `Generic` erforderlich.

Hier ist ein Beispiel für eine Postmark-Connector-Vorlage im JSON-Format.

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## Postmark-E-Mail-Connector testen \{#test-postmark-email-connector}

Du kannst eine E-Mail-Adresse eingeben und auf "Senden" klicken, um zu sehen, ob die Einstellungen funktionieren, bevor du auf "Speichern und Fertig" klickst.

Das war's. Vergiss nicht, <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">Connector in der Anmeldeerfahrung zu aktivieren</MainSiteUrl>.

## Konfigurationstypen \{#config-types}

| Name        | Typ    |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| Vorlageneigenschaften | Typ         | Enum-Werte                                              |
| --------------------- | ----------- | ------------------------------------------------------- |
| usageType             | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
| templateAlias         | string      | N/A                                                     |
