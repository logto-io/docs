# Soziale Anmeldung mit KOOK einrichten (Set up social login with KOOK)

Der offizielle Logto Connector für KOOK Social Sign-in Webanwendungen.

<GuideTip />

## Erste Schritte \{#get-started}

Der KOOK Connector ermöglicht es Endbenutzern, sich über ihr eigenes KOOK-Konto mithilfe des KOOK OAuth 2.0 Authentifizierungsprotokolls bei deiner Anwendung anzumelden.
