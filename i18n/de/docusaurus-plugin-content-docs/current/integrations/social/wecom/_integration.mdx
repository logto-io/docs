## Erste Schritte \{#get-started}

Melde dich mit einem Administratorkonto bei [WeCom WebUI](https://work.weixin.qq.com/) an oder klicke im WeCom-App auf **Unternehmen verwalten (管理企业)**.

Wechsle zum Tab "App-Management", scrolle nach unten und klicke auf "App erstellen".

Fülle die entsprechenden Informationen zu deiner App aus. Dann erstelle die App.

Jetzt haben wir die Agent ID (NICHT APPID) und das Secret.

### Website-Informationen \{#website-info}

Lege auf dieser Seite die benötigten Einstellungen fest. Das könnte sein:

- Erlaubte Benutzer: _wer diese App sehen kann_
- App-Startseite: _Die Startseite deiner App. Z. B. `logto.io/demo-app`_

**Wichtig**
Auf dieser Seite gibt es drei Punkte bezüglich der "Developer API (开发者接口)":

1. Web-Autorisierung und JS-SDK;
2. Anmeldung über Autorisierung durch WeCom;
3. Enterprise Trusted IP;

Fülle diese gemäß der Anleitung von WeCom aus.

### Corp ID \{#corp-id}

Wenn du mit der WeChat-Entwicklung vertraut bist, wirst du feststellen, dass die Verwendung der Corp ID der APP ID entspricht.

Du findest die Corp ID unten auf der Seite im Tab "Mein Unternehmen (我的企业)". Sie sieht etwa so aus: **ww\*\*\*\*** .

## Connector konfigurieren \{#configure-the-connector}

Jetzt haben wir die Agent ID, das Secret und die Corp ID.

Fülle das Formular für den Connector aus.

Das Feld `Scope` kannst du leer lassen, da es optional ist. Alternativ kannst du `snsapi_base` oder `snsapi_privateinfo` eintragen.

**Speichern und fertig**
