# Soziale Anmeldung mit Patreon einrichten

Der offizielle Logto Connector für die Patreon Social Sign-in.

<GuideTip />

## Erste Schritte \{#get-started}

Der Patreon Connector ermöglicht es Endbenutzern, sich über ihre eigenen Patreon-Konten mithilfe des Patreon OAuth 2.0 Authentifizierungsprotokolls bei deiner Anwendung anzumelden.
