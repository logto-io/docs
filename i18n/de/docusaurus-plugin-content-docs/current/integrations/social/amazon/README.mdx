# Soziale Anmeldung mit Amazon einrichten (Set up social login with Amazon)

Der offizielle Logto Connector für die Amazon Social Sign-In.

<GuideTip />

## Erste Schritte \{#get-started}

Der Amazon Connector ermöglicht es Endbenutzern, sich über ihre eigenen Amazon-Konten mithilfe des Amazon OAuth 2.0 Authentifizierungsprotokolls bei deiner Anwendung anzumelden.

<Integration />

## Amazon Connector testen \{#test-amazon-connector}

Das war's. Der Amazon Connector sollte jetzt verfügbar sein. Vergiss nicht, den [Social Connector für die Anmeldung und Registrierung zu aktivieren](/connectors/social-connectors/#enable-social-sign-in).
