# Soziale Anmeldung mit Amazon einrichten

Der offizielle Logto-Connector für die Amazon-Sozialanmeldung.

<GuideTip />

## Erste Schritte \{#get-started}

Der Amazon-Connector ermöglicht es Endbenutzern, sich mit ihren eigenen Amazon-Konten über das Amazon OAuth 2.0 Authentifizierungsprotokoll bei deiner Anwendung anzumelden.

<Integration />

## Amazon-Connector testen \{#test-amazon-connector}

Das war's. Der Amazon-Connector sollte jetzt verfügbar sein. Vergiss nicht, [Sozialen Connector in der Anmeldeerfahrung aktivieren](/connectors/social-connectors/#enable-social-sign-in).
