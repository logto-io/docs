### Amazon-App einrichten

Gehe zum [Amazon Developer Portal](https://developer.amazon.com) und melde dich mit deinem Konto an. Wenn du noch kein Konto hast, kannst du dich registrieren.

Erstelle dann eine App.

**Schritt 1:** Gehe zum Block "Login with Amazon" auf der Startseite.

Oder du kannst [diese Seite](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html) besuchen.

Folge der Anleitung, um ein neues Sicherheitsprofil zu erstellen.

Nach der Erstellung des Sicherheitsprofils erhältst du eine Client-ID und ein Client Secret aus der Tabelle.

**Schritt 2:** Callback-URLs einrichten.

Klicke auf die Schaltfläche "Verwalten" auf der rechten Seite der Tabelle und gehe zum Tab "Webeinstellungen".

Füge im Feld "Erlaubte Umleitungs-URLs" die folgende URL hinzu:

- `${your_logto_origin}/callback/${connector_id}`.

Die `connector_id` findest du in der oberen Leiste der Logto Admin Console auf der Seite mit den Connector-Details.

### Deinen Connector konfigurieren

In deiner Logto-Connector-Konfiguration fülle die folgenden Felder mit den Werten aus, die du im letzten Schritt erhalten hast:

- **clientId:** Die Client-ID deiner App.
- **clientSecret:** Das Client Secret deiner App.

`scope` ist eine durch Leerzeichen getrennte Liste von OIDC-Berechtigungen. Wenn nicht angegeben, ist die Standardberechtigung `openid profile`.

### Konfigurationstypen

| Name         | Typ    |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
