# Soziale Anmeldung mit Line einrichten

Der offizielle Logto-Connector für die Line-Sozialanmeldung.

<GuideTip />

## Erste Schritte \{#get-started}

Der Line-Connector ermöglicht es Endbenutzern, sich mit ihren eigenen Line-Konten über das Line OAuth 2.0 Authentifizierungsprotokoll bei deiner Anwendung anzumelden.

<Integration />

## Line-Connector testen \{#test-line-connector}

Das war's. Der Line-Connector sollte jetzt verfügbar sein. Vergiss nicht, [Sozialen Connector in der Anmeldeerfahrung aktivieren](/connectors/social-connectors/#enable-social-sign-in).
