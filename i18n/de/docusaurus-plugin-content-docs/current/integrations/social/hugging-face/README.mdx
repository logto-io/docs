# Soziale Anmeldung mit Hugging Face einrichten

Der offizielle Logto Connector für Hugging Face Social Sign-in.

<GuideTip />

## Loslegen \{#get-started}

Der Hugging Face Connector ermöglicht es Endbenutzern, sich über ihre eigenen Hugging Face Konten per Hugging Face OAuth / OpenID Connect Flow bei deiner Anwendung anzumelden.
