# Soziale Anmeldung mit LinkedIn einrichten

Der offizielle Logto Connector für LinkedIn Social Sign-In.

<GuideTip />

## Erste Schritte \{#get-started}

Der LinkedIn Connector ermöglicht Endbenutzern, sich mit ihren eigenen LinkedIn-Konten über das LinkedIn OAuth 2.0 Authentifizierungsprotokoll bei deiner Anwendung anzumelden.

<Integration />

## X (Twitter) Connector testen \{#test-x-connector}

Das war's. Der X (Twitter) Connector sollte jetzt verfügbar sein. Vergiss nicht, [Sozialen Connector in der Anmeldeerfahrung aktivieren](/connectors/social-connectors/#enable-social-sign-in).
