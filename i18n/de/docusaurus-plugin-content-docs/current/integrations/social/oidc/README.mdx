# Soziale Anmeldung mit OpenID Connect (OIDC) einrichten

Der offizielle Logto Connector für das OpenID Connect (OIDC) Protokoll.

<GuideTip />

## Erste Schritte \{#get-started}

Der OIDC Connector ermöglicht die Verbindung von Logto zu einem beliebigen sozialen Identitätsanbieter, der das OIDC-Protokoll unterstützt. Verwende den OIDC Connector, um deiner Anwendung folgende Möglichkeiten zu bieten:

- Soziale Anmeldebuttons hinzufügen
- Benutzerkonten mit sozialen Identitäten verknüpfen
- Benutzerprofilinformationen vom sozialen Anbieter synchronisieren
- Zugriff auf Drittanbieter-APIs durch sichere Token-Speicherung im Logto Secret Vault für Automatisierungsaufgaben (z. B. Bearbeiten von Google Docs, Verwalten von Kalenderereignissen in deiner App)

Um diese Authentifizierungsfunktionen einzurichten, erstelle zuerst einen OIDC Connector in Logto:

1. Gehe zu <CloudLink to="/connectors/social">Logto-Konsole > Connector > Social Connector</CloudLink>.
2. Klicke auf **Social Connector hinzufügen**, wähle **OIDC**, klicke auf **Weiter** und folge dem Schritt-für-Schritt-Tutorial, um die Integration abzuschließen.

:::note

Der OIDC Connector ist eine besondere Art von Connector in Logto. Du kannst mehrere auf dem OIDC-Protokoll basierende Connectors hinzufügen.

:::
