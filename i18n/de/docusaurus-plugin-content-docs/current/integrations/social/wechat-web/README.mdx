# Soziale Anmeldung mit WeChat (Web) einrichten (Set up social login with WeChat (Web))

Der offizielle Logto Connector für WeChat Social Sign-in in Webanwendungen.

<GuideTip />

## Erste Schritte \{#get-started}

Wenn du das Konzept des Connectors nicht kennst oder nicht weißt, wie du diesen Connector zu deiner Anmeldeerfahrung hinzufügst, siehe bitte das [Logto-Tutorial](/connectors/social-connectors).

:::caution

Dieser Connector ist nur für Webanwendungen geeignet. Wenn du nach einer Methode suchst, dich mit WeChat in nativen Apps anzumelden, siehe bitte den [WeChat Native Connector](/integrations/wechat-native).

:::
