# SMS-Verifizierung mit GatewayAPI SMS einrichten

Der offizielle Logto Connector für GatewayAPI SMS.

<GuideTip />

## Erste Schritte \{#get-started}

GatewayAPI ist ein Cloud-Service-Anbieter in Europa, der viele Cloud-Dienste anbietet, darunter SMS (Short Message Service). Der GatewayAPI SMS Connector ist ein Plugin, das vom Logto-Team bereitgestellt wird, um den GatewayAPI SMS-Dienst anzusprechen. Damit können Logto-Endbenutzer sich per SMS-Verifizierungscode bei ihrem Logto-Konto registrieren und anmelden.
