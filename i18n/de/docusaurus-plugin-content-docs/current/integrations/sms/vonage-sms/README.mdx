# SMS-Verifizierung mit Vonage SMS einrichten (Set up SMS verification with Vonage SMS)

Der offizielle Logto Connector für Vonage SMS.

<GuideTip />

## Erste Schritte \{#get-started}

Vonage ist ein globaler Kommunikationsanbieter und bietet leistungsstarke cloudbasierte Kommunikationsdienste, einschließlich SMS (Short Message Service). Der Vonage SMS Connector ist ein von Logto bereitgestelltes Plugin, das es Logto-Endbenutzern ermöglicht, sich über SMS-Verifizierungscodes bei ihrem Logto-Konto zu registrieren und anzumelden.
