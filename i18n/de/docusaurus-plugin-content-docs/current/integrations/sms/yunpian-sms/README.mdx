# YunPian SMS Connector

Der offizielle Logto Connector für den YunPian SMS-Dienst.

<GuideTip />

## Erste Schritte \{#get-started}

YunPian ist ein Kommunikationsdienstleister, der verschiedene Dienste einschließlich SMS anbietet. Der YunPian SMS Connector ist ein Plugin, das vom Logto-Team bereitgestellt wird, um die Integration mit dem SMS-Dienst von YunPian zu ermöglichen. Dadurch können Logto-Endnutzer sich per SMS-Verifizierungscode registrieren und anmelden.

## SMS-Dienst in der YunPian-Konsole einrichten \{#set-up-sms-service-in-yunpian-console}

### YunPian-Konto erstellen \{#create-a-yunpian-account}

Besuche die [YunPian-Website](https://www.yunpian.com/), um ein Konto zu registrieren und die Echtnamenverifizierung abzuschließen.

### API KEY erhalten \{#get-api-key}

1. Melde dich in der YunPian-Konsole an
2. Gehe zu „Kontoeinstellungen“ -> „Sub-Account-Verwaltung“
3. Finde und kopiere den API KEY

### SMS-Vorlagen konfigurieren \{#configure-sms-templates}

1. Gehe in der YunPian-Konsole zu „Inländische SMS“ -> „Signatur-Einreichung“
2. Erstelle und reiche eine Signatur ein, warte auf die Genehmigung durch den Netzbetreiber
3. Gehe zu „Inländische SMS“ -> „Vorlagen-Einreichung“ und wähle „Verifizierungscode“
4. Erstelle eine Verifizierungscode-Vorlage und stelle sicher, dass sie die Variable `#code#` enthält (du kannst auch „Allgemeine Vorlagen“ nutzen, um den Genehmigungsprozess zu beschleunigen)
5. Warte auf die Genehmigung der Vorlage
6. Wenn du internationale SMS versenden möchtest, wiederhole die obigen Schritte, wähle jedoch „Internationale SMS“ -> „Vorlagen-Einreichung“

## In Logto konfigurieren \{#configure-in-logto}

1. Gehe in der Logto-Konsole zu „Connectors“
2. Finde und klicke auf „YunPian SMS Service“
3. Fülle das Konfigurationsformular aus:
   - API KEY: Der von YunPian erhaltene API KEY
   - SMS-Vorlagen: Konfiguriere die Vorlagen entsprechend der Nutzung und stelle sicher, dass sie exakt mit den genehmigten Vorlagen in YunPian übereinstimmen

## Hinweise \{#notes}

1. Der Inhalt der SMS-Vorlage muss exakt mit der genehmigten Vorlage in YunPian übereinstimmen
2. Der Platzhalter für den Verifizierungscode in YunPian-Vorlagen ist `#code#`, während er in der Connector-Konfiguration `{{code}}` ist
3. YunPian fügt die Standardsignatur automatisch basierend auf dem API KEY hinzu, es ist nicht notwendig, die Signatur im Vorlageninhalt zu hinterlegen
4. Es wird empfohlen, die Konfiguration vor dem produktiven Einsatz zu testen
