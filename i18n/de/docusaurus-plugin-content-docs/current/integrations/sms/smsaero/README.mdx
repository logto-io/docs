# SMS-Verifizierung mit SMS Aero einrichten

Der offizielle Logto-Connector für den SMS Aero-Kurznachrichtendienst.

<GuideTip />

## Konto registrieren

Erstelle ein neues Konto auf [SMSAero](https://smsaero.ru/). (Springe zum nächsten Schritt, wenn du bereits eines hast.)

## Kontozugangsdaten erhalten

Wir benötigen die API-Zugangsdaten, um den Connector zum Laufen zu bringen. Beginnen wir mit
der [API und SMPP](https://smsaero.ru/cabinet/settings/apikey/).

Kopiere den "API-Schlüssel" oder generiere einen neuen.

## Den Connector-JSON zusammenstellen

Fülle die Felder _email_, _apiKey_ und _senderName_ mit deiner E-Mail, deinem API-Schlüssel und dem Absendernamen aus.

Du kannst den Absendernamen mit "SMSAero" ausfüllen, um den von SMSAero bereitgestellten Standard-Absendernamen zu verwenden.

Du kannst mehrere SMS-Connector-Vorlagen für verschiedene Fälle hinzufügen. Hier ist ein Beispiel für das Hinzufügen einer einzelnen Vorlage:

- Fülle das `content`-Feld mit beliebigen string-typisierten Inhalten aus. Vergiss nicht, den `{{code}}` Platzhalter für
  den zufälligen Verifizierungscode zu belassen.
- Fülle das `usageType`-Feld mit entweder `Register`, `SignIn`, `ForgotPassword`, `Generic` für verschiedene Anwendungsfälle aus.
  Um vollständige Benutzerflüsse zu ermöglichen, sind Vorlagen mit dem usageType `Register`, `SignIn`, `ForgotPassword` und `Generic`
  erforderlich.

### SMSAero-Connector testen

Du kannst eine Telefonnummer eingeben und auf "Senden" klicken, um zu sehen, ob die Einstellungen funktionieren, bevor du auf "Speichern und Fertig" klickst.

Das war's. Vergiss nicht,
den [Connector in der Anmeldeerfahrung zu aktivieren](/connectors/sms-connectors/#enable-phone-number-sign-up-or-sign-in).

### Konfigurationstypen

| Name       | Typ         |
| ---------- | ----------- |
| email      | string      |
| apiKey     | string      |
| senderName | string      |
| templates  | Templates[] |

| Template-Eigenschaften | Typ         | Enum-Werte                                              |
| ---------------------- | ----------- | ------------------------------------------------------- |
| content                | string      | N/A                                                     |
| usageType              | enum string | 'Register' \| 'SignIn' \| 'ForgotPassword' \| 'Generic' |
