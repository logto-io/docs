# Organisationserfahrung

Organisationserfahrung bezieht sich auf die Benutzeroberflächen und Abläufe, die von deinen Geschäftskunden und deren Mitarbeitern genutzt werden, insbesondere in Multi-Tenant-Anwendungen. Logto bietet mehrere Anleitungen, die dir helfen, diese Erfahrung schnell in deine App zu integrieren, indem du die Logto Management API verwendest.
