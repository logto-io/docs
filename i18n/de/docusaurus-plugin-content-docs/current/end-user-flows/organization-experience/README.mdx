# Organisationserfahrung

Die [Organisation](/organizations)-Erfahrung bezieht sich auf die Benutzeroberflächen und Abläufe, die von deinen Geschäftskunden und deren Mitarbeitenden genutzt werden, insbesondere in [Multi-Tenant-Anwendungen](https://auth.wiki/multi-tenancy). Logto bietet mehrere Anleitungen, um dir zu helfen, diese Erfahrung mithilfe der Logto Management API schnell in deine App zu integrieren.
