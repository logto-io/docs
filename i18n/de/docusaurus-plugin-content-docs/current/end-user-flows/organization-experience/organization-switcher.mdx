# Organisationswechsler

In der Organisationserfahrung können Benutzer zwischen Organisationen wechseln. Jedes Mal, wenn sie wechseln, ruft die App ein neues Organisationstoken ab, um die relevanten Informationen zu laden.

## So implementierst du es \{#how-to-implement-it}

1. Erstelle ein Dropdown-Menü oder ein ähnliches UI-Element, um alle dem Benutzer verfügbaren Organisationen aufzulisten.
2. Wenn der Benutzer eine andere Organisation auswählt, hole deren ID und rufe die SDK-Methode `getOrganizationToken(organizationId)` auf, um das entsprechende Organisationstoken abzurufen. (Das SDK übernimmt das Token-Caching für dich.)
3. Speichere die ausgewählte Organisations-ID (in der URL oder im Client-Speicher) und verwende sie immer, um das zugehörige Organisationstoken für organisationsspezifische Ressourcen abzurufen.
