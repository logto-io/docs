# Endbenutzer-Flows

Endbenutzer-Flows decken alle Verifizierungsprozesse für Benutzerinteraktionen ab, kategorisiert wie folgt:

- **Anmeldeerfahrung**: Beinhaltet Benutzeranmeldung, Registrierung und Passwortzurücksetzung. Logto bietet sofort einsatzbereite Erfahrungen, die in der Konsole konfiguriert werden, und du kannst deine Benutzeroberfläche über Experience APIs einbringen.
- **Kontenerfahrung**: Beinhaltet Kontoeinstellungen, Benutzerprofil und Sicherheitsüberprüfung. Verwende Account APIs oder Management APIs zur Implementierung.
- **Organisationserfahrung**: Für Multi-Tenancy-Produkte, wie B2B- oder SaaS-Dienste. Beinhaltet Organisationserstellung, Mitglieder-Einladung und Verwaltung. Verwende Management APIs, um dies in deine Apps zu integrieren.
