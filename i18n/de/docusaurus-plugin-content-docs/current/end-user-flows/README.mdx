# Endbenutzerflüsse

Endbenutzerflüsse decken alle Verifizierungsprozesse für Benutzerinteraktionen ab, kategorisiert wie folgt:

- **Authentifizierungsflüsse**: Beinhaltet Benutzeranmeldung, Registrierung und Passwortzurücksetzung. Logto bietet sofort einsatzbereite Erfahrungen, die in <CloudLink to="/sign-in-experience">Konsole > Anmeldeerfahrung</CloudLink> konfiguriert sind, und du kannst [dein UI einbringen](/customization/bring-your-ui) über [Experience APIs](https://openapi.logto.io/group/endpoint-experience).
- **Kontoflüsse**: Beinhaltet Kontoeinstellungen, Benutzerprofil und Sicherheitsüberprüfung. Verwende [Account APIs](https://openapi.logto.io/group/endpoint-account-center) oder [Management APIs](https://openapi.logto.io/operation/operation-getuser) zur Implementierung.
- **Organisationsflüsse**: Für Multi-Tenancy-Produkte, wie B2B- oder SaaS-Dienste. Beinhaltet Organisationserstellung, Mitglieder-Einladung und Verwaltung. Verwende [Management APIs](https://openapi.logto.io/operation/operation-listorganizations), um dies in deine Apps zu integrieren.
