# Soziale Connectors

Vereinfachen Sie das Benutzer-Onboarding und erhöhen Sie die Konversionsraten, indem Sie die soziale Anmeldung mit Logto aktivieren. Benutzer können sich schnell und sicher mit ihren bestehenden Social-Media-Konten anmelden, ohne ein Passwort erstellen oder einen komplexen Registrierungsprozess durchlaufen zu müssen. Logto bietet eine Vielzahl von vorgefertigten sozialen Connectors und unterstützt benutzerdefinierte Integrationen für maximale Flexibilität.

### Passen Sie Ihre sozialen Connectors an

Für benutzerdefinierte Anforderungen nutzen Sie die OAuth 2.0- und OIDC (OpenID Connect)-Standards, um Ihren bevorzugten Anbieter zu integrieren.

Wenn unsere Standard-Connectors Ihre spezifischen Anforderungen nicht erfüllen, zögern Sie nicht, uns zu kontaktieren. Für OSS-Nutzer können Sie [Ihren Connector entwickeln (OSS)](/logto-oss/develop-your-connector), wenn die Anforderung dringend ist. Wir begrüßen immer Beiträge; Ihr Einsatz könnte anderen Community-Mitgliedern mit denselben Bedürfnissen sehr helfen.

## Konfigurationsschritte

1. Navigieren Sie zu <CloudLink to="/connectors/social">Konsole > Connectors > Soziale Connectors</CloudLink>.
2. Klicken Sie auf "Sozialen Connector hinzufügen" und wählen Sie den gewünschten Typ aus.
3. Folgen Sie der README-Anleitung und füllen Sie die erforderlichen Felder aus und passen Sie die Einstellungen an.
4. Klicken Sie auf "Speichern und Fertig", um abzuschließen.
5. Testen Sie den Connector, indem Sie eine soziale Anmeldung initiieren.

Bitte beachten Sie die folgenden Einstellungen:

- **Name des Identitätsanbieters**: Jeder soziale Connector hat einen eindeutigen Namen des Identitätsanbieters (IdP), um Benutzeridentitäten zu unterscheiden. Während gängige Connectors einen festen IdP-Namen verwenden, erfordern benutzerdefinierte Connectors einen eindeutigen Wert. Erfahren Sie mehr über IdP-Namen für weitere Details.
- **Benutzerprofile synchronisieren**: Wählen Sie, wann Benutzerprofilinformationen (z. B. Avatar, Benutzername) synchronisiert werden sollen. Standardmäßig wird "nur bei Registrierung synchronisieren" verwendet. "Bei jeder Anmeldung synchronisieren" ist eine Alternative, kann jedoch benutzerdefinierte Benutzerdaten überschreiben.

## Soziale Anmeldung aktivieren

Sobald Sie einen sozialen Connector erfolgreich erstellt haben, können Sie ihn als Schaltfläche für die soziale Anmeldung in der <CloudLink to="/sign-in-experience/sign-up-and-sign-in">Anmeldeerfahrung</CloudLink> aktivieren.

1. Navigieren Sie zu <CloudLink to="/sign-in-experience/sign-up-and-sign-in">Konsole > Anmeldeerfahrung > Anmeldung und Registrierung</CloudLink>.
2. (Optional) Wählen Sie "Nicht zutreffend" für das Anmeldekennzeichen, wenn Sie nur die soziale Anmeldung benötigen.
3. Fügen Sie konfigurierte soziale Connectors zum Abschnitt "Soziale Anmeldung" hinzu.
4. Ordnen Sie die Connectors nach Bedarf neu an.
5. Klicken Sie auf "Änderungen speichern" und testen Sie die "Live-Vorschau".
