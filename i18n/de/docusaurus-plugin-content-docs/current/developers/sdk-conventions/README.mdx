# Plattform-SDK-Konventionen

Logto bietet einen sehr leistungsstarken und flexiblen Web-Authentifizierungsdienst.

In der praktischen Nutzung der Logto-Dienste ist es aus Bequemlichkeitsgründen oft notwendig, dass Entwickler das Logto SDK in ihre eigenen Client-Anwendungen integrieren, um den Anmeldestatus der Benutzer, Berechtigungen und mehr zu verwalten.

Du kannst die SDKs für alle von Logto unterstützten Programmiersprachen / Frameworks [hier](/quick-starts) finden.

Falls du Pech hast und das gewünschte SDK nicht findest, gibt es hier eine Konvention, der du folgen kannst, um das SDK für deine gewünschte Programmiersprache zu implementieren, was die Nutzung der Logto-Dienste erleichtert.
