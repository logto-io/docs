# Ereignistypen der Audit-Logs

Du kannst Ereignistypen in <CloudLink to="/audit-logs">Logto Console > Audit Logs</CloudLink> filtern.

:::note

Logto unterstützt jetzt das Abrufen von Protokollen im Zusammenhang mit Endbenutzerinteraktionen über die [Experience APIs](https://openapi.logto.io/group/endpoint-experience).

Audit-Logs für [Management APIs](/integrate-logto/interact-with-management-api) und [Account APIs](https://openapi.logto.io/group/endpoint-account-center) folgen in Kürze.

Teile uns gerne [deine Anforderungen mit](https://logto.io/contact), wenn du Wünsche hast.

:::
