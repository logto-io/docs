# Entwickler

Logto ist ein Identity and Access Management (IAM) Dienst, der auf den OAuth 2 und OIDC Protokollen basiert. IAM-Dienste wie Logto dienen oft als Grundlage für andere Webdienste; verschiedene Autorisierungszustände innerhalb dieser Webdienste werden direkt von Logto beeinflusst.

Um unseren Nutzern Komfort zu bieten, bietet Logto eine Reihe häufig verwendeter Entwicklerfunktionen.
