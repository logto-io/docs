# Anpassung

Mache Logto zu deinem eigenen, indem du jeden Aspekt anpasst - von Branding und Benutzeroberfläche bis hin zu Sprach- und Domaineinstellungen.

## Vorgefertigte UI

Beginne mit unseren vorgefertigten UI-Komponenten und passe wesentliche Markenelemente an. Erstelle entweder:

- Eine einheitliche Authentifizierungserfahrung über deine Plattform hinweg.
- Spezielle Anmeldeseiten für bestimmte Anwendungen oder Organisationen.

  items={[

## Inhalt & Sprachen

Logto bietet standardmäßig Unterstützung für 15 Sprachen. Erweitere die Sprachunterstützung oder passe bestehende Übersetzungen an, um deinen Anforderungen gerecht zu werden.

  items={[
