# Logto OSS

Logto bietet sowohl Open-Source- (OSS) als auch [Cloud](/logto-cloud) Versionen an. Dieser Abschnitt konzentriert sich auf die OSS-Lösung: <Url type="inline" href="https://github.com/logto-io/logto">github.com/logto-io/logto</Url>

Logto OSS ist eine kostenlose Identitätslösung, die Entwicklern vollständige Anpassungsmöglichkeiten und vollständige Kontrolle über ihre Daten bietet. Logto OSS bietet die Flexibilität, den Code an spezifische Bedürfnisse anzupassen. Es ist ideal für diejenigen, die Datensouveränität priorisieren und eine maßgeschneiderte Identitätsanbieter-Lösung wünschen, die mit ihrer Anwendung wachsen kann.

## Von Logto OSS unterstützte Funktionen

Logto OSS unterstützt die meisten Kernfunktionen des Logto-Dienstes und wird regelmäßig aktualisiert und gewartet.

Einige erweiterte Funktionen sind derzeit exklusiv für die Logto Cloud-Version, einschließlich [Logto Protected App](/integrate-logto/protected-app/), [kostenloser E-Mail-Dienst](/connectors/email-connectors/built-in-email-service), [Bring your UI](/customization/bring-your-ui), [IdP-initiiertes SSO](/end-user-flows/enterprise-sso/idp-initiated-sso) und SAML IdP.
