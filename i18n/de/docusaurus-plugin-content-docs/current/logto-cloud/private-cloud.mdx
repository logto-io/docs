# Logto Private Cloud

Setze Logto in einer dedizierten Azure-Umgebung ein, um stärkere Isolierung, Compliance-Konformität und Infrastrukturkontrolle zu erreichen.

## Überblick \{#overview}

Logto Private Cloud ist eine dedizierte Bereitstellungsoption für Kunden, die mehr Infrastruktur-Isolierung und Kontrolle benötigen als eine geteilte Public-Cloud-Umgebung.

Sie ist für Organisationen konzipiert, die strengere Anforderungen an Sicherheit, Compliance, Beschaffung oder Bereitstellung haben, während sie dennoch das gleiche Kernerlebnis wie Logto Cloud bietet.

So unterscheiden sich die Bereitstellungsoptionen:

- **Logto Cloud**: Verwaltete Public Cloud, schnellster Weg zur Produktion
- **Logto Private Cloud**: Dedizierte Bereitstellung für Kunden mit strengeren Infrastruktur- oder Compliance-Anforderungen
- **Logto OSS**: Selbst gehostet und selbst verwaltet

## Wann Private Cloud wählen \{#when-to-choose-private-cloud}

Logto Private Cloud ist in der Regel die richtige Wahl, wenn deine Organisation eines oder mehrere der folgenden Kriterien benötigt:

- Stärkere Isolierung der Umgebung
- SLA-gestützte Betriebszeit-Zusicherung mit Support und Zusammenarbeit durch das Logto-Team
- Spezifische Compliance-Anforderungen, die durch eine geteilte Public Cloud nicht erfüllt werden können
- Bereitstellung in einer bestimmten Region oder Infrastruktur, die von Logto Cloud nicht unterstützt wird
- Verbindung zu internen Ressourcen, die aus einer Public-Cloud-Umgebung nicht erreichbar sind

Wenn diese Anforderungen nicht bestehen, ist [Logto Cloud](https://cloud.logto.io/?sign_up=1) in der Regel die bessere Wahl, da sie einfacher zu übernehmen und leichter zu betreiben ist.

### Bereitstellungsumfang \{#deployment-scope}

Logto Private Cloud ist derzeit nur auf **Azure** verfügbar.

Unterstützte Regionen hängen davon ab, ob die Azure-Region die von Logto benötigten Dienste bereitstellt. Wenn du eine Zielregion im Sinn hast, kontaktiere uns und wir können die Machbarkeit anhand deiner Anforderungen bestätigen.

### Sicherheit und Compliance \{#security-and-compliance}

Private Cloud bietet eine dedizierte Umgebung mit klareren Isolationsgrenzen als eine geteilte Public-Cloud-Bereitstellung.

Wir halten Sicherheits- und Compliance-Diskussionen praxisnah und anforderungsgesteuert. Wenn dein Team eine Sicherheitsüberprüfung, eine Compliance-Besprechung oder eine bereitungsspezifische Klärung benötigt, kontaktiere uns und wir gehen die Details direkt mit dir durch.

## Kontaktiere uns \{#contact-us}

Wenn du Private Cloud in Betracht ziehst, [kontaktiere uns](https://logto.io/contact?src=private-cloud) mit:

- deiner gewünschten Azure-Region
- Sicherheits- oder Compliance-Anforderungen
- Migrationskontext, falls vorhanden

Wir helfen dir, die Eignung zu bewerten und die nächsten Schritte zu besprechen.
