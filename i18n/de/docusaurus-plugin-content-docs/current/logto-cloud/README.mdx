# Logto Cloud-Dienst

[Logto Cloud](https://cloud.logto.io) bietet eine Reihe von Verwaltungs- und Betriebsdiensten, die dir helfen, Identitäten und Ressourcen reibungslos und einfach zu verwalten. Gehostet von Logto, umfasst es Funktionen wie [Multi-Region-Unterstützung](/logto-cloud/tenant-settings#tenant-region), Mandantenverwaltung, [Abrechnung und Preise](/logto-cloud/billing-and-pricing), [Mitgliederverwaltung](/logto-cloud/tenant-member-management) und rollenbasierte Zugangskontrolle in der Konsole.

Wenn du Fragen zu Cloud-Diensten hast und zusätzliche Unterstützung benötigst, kontaktiere uns bitte.
