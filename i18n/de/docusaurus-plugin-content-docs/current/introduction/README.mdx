# Einführung in Logto

Willkommen zur Logto-Dokumentation! Logto ist eine Identitäts- und Zugriffsmanagement (IAM) Lösung, die für moderne Apps und SaaS-Produkte entwickelt wurde. Es bietet ein sicheres, skalierbares und anpassbares Authentifizierungs- und Autorisierungssystem für deine Anwendungen.
