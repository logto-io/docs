# Einführung in Logto

Willkommen bei der Logto-Dokumentation! Logto ist eine Identity and Access Management (IAM) Lösung, die für moderne Apps und SaaS-Produkte entwickelt wurde. Es bietet ein sicheres, skalierbares und anpassbares Authentifizierungs- und Autorisierungssystem für deine Anwendungen.

## Nach Funktionen erkunden \{#explore-by-features}

Logto bietet eine robuste Suite von Funktionen, einschließlich Authentifizierung, Autorisierung sowie Benutzer- und Organisationsverwaltung, die auf moderne Identitätsbedürfnisse zugeschnitten sind. Unsere starke Unterstützung im Bereich Customer Identity and Access Management (CIAM) verschafft uns einen klaren Vorteil auf diesem Gebiet.
