# Berechtigungen konfigurieren

## API-Berechtigungen über die Logto-Konsole konfigurieren \{#configure-api-permissions-via-logto-console}

Berechtigungen werden auf der Ebene der **API-Ressource** festgelegt. Navigiere zu <CloudLink to="/api-resources">Konsole > API-Ressourcen</CloudLink>, klicke auf die API-Ressource, die du verwalten möchtest, und gehe dann zum Tab "Berechtigungen".

Es ist wichtig, einen Namen und Details für die Berechtigung bereitzustellen, da dies die Identifizierung und Verwaltung der Berechtigungen erleichtert.

## API-Berechtigungen über die Logto Management API konfigurieren \{#configure-api-permissions-via-logto-management-api}

Verwalte die Berechtigungen über die Logto Management API. Führe einen Aufruf zum relativen Endpunkt durch. Sieh dir diese [Referenz](https://openapi.logto.io/operation/operation-listresourcescopes) an.

| Methode | Pfad                                                                                                                 | Beschreibung                                                                                                                |
| ------- | -------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| GET     | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-listresourcescopes)              | Hole die für eine API-Ressource definierten Berechtigungen.                                                                 |
| POST    | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-createresourcescope)             | Erstelle eine neue Berechtigung für eine API-Ressource.                                                                     |
| DELETE  | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-deleteresourcescope) | Lösche eine API-Ressourcenberechtigung von der angegebenen Ressource.                                                       |
| PATCH   | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-updateresourcescope) | Aktualisiere eine API-Ressourcenberechtigung für die angegebene Ressource. Diese Methode führt ein partielles Update durch. |
