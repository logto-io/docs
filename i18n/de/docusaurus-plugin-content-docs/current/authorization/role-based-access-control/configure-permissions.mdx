# Berechtigungen konfigurieren

## Konfiguration über die Logto-Konsole

Berechtigungen werden auf der Ebene der **API-Ressource** festgelegt. Navigiere zu <CloudLink to="/api-resources">Konsole > API-Ressourcen</CloudLink>, klicke auf die API-Ressource, die du verwalten möchtest, und gehe dann zum Tab "Berechtigungen".

Es ist wichtig, einen Namen und Details für die Berechtigung anzugeben, da dies die Identifizierung und Verwaltung von Berechtigungen erleichtert.

## Konfiguration über die Logto Management API

Verwalte die Berechtigungen mit der Logto Management API. Führe einen Aufruf zum relativen Endpunkt durch. Sieh dir diese [Referenz](https://openapi.logto.io/operation/operation-listresourcescopes) an.

| Methode | Pfad                                                                                                                 | Beschreibung                                                                                                                         |
| ------- | -------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| GET     | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-listresourcescopes)              | Berechtigungen (Scopes) abrufen, die für eine API-Ressource definiert sind.                                                          |
| POST    | [/api/resources/\{resourceId\}/scopes](https://openapi.logto.io/operation/operation-createresourcescope)             | Eine neue Berechtigung (Scope) für eine API-Ressource erstellen.                                                                     |
| DELETE  | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-deleteresourcescope) | Eine API-Ressourcenberechtigung (Scope) von der angegebenen Ressource löschen.                                                       |
| PATCH   | [/api/resources/\{resourceId\}/scopes/\{scopeId\}](https://openapi.logto.io/operation/operation-updateresourcescope) | Eine API-Ressourcenberechtigung (Scope) für die angegebene Ressource aktualisieren. Diese Methode führt ein partielles Update durch. |
