# Organisationen

Die Organisationsfunktion ist besonders effektiv in Multi-Tenant-SaaS und [Business-to-Business (B2B)-Apps](/introduction/plan-your-architecture/b2b). Neben einzelnen Benutzern können deine Kunden auch aus Teams, Organisationen oder ganzen Unternehmen bestehen.

Mit diesem grundlegenden Element kannst du die unverzichtbaren Funktionen für [Multi-Tenancy-Apps](https://auth.wiki/multi-tenancy) aufbauen, wie zum Beispiel:

- Ein Produkt, das von mehreren Organisationen genutzt werden kann.
- Bereitstellung von Organisationsmitgliedern auf [Einladung](/end-user-flows/organization-experience/invite-organization-members) oder [Just-in-Time](/organizations/just-in-time-provisioning)-Basis.
- [Zugangskontrollen](/authorization/organization-template), die durch Rollen definiert werden, die Mitgliedern innerhalb einer Organisation zugewiesen sind.
- Verknüpfe Organisationen mit [Enterprise SSO (Single Sign-On)](/end-user-flows/enterprise-sso)-Erfahrung.

Der Begriff „Organisation“ wird auch in anderen Formen verwendet, wie „Workspace“, „Team“, „Unternehmen“ usw. In Logto verwenden wir „Organisation“ als generischen Begriff, um das Konzept der Multi-Tenancy darzustellen.

## Planungshinweise \{#planning-considerations}

Beim Einsatz von Organisationen kann die Integration unterschiedlich aufwendig sein. Wir empfehlen, diese Aspekte zu berücksichtigen, um deine Entwicklung individuell anzupassen.

- **Organisationserfahrung**: Wie soll die Organisationserfahrung aussehen? Kann ein Organisationsadministrator seine Organisation selbst erstellen? Können Organisationsadministratoren Mitglieder einladen? Möchtest du ein individuelles Dashboard erstellen, das Administratoren die Selbstverwaltung der identitätsbezogenen Informationen ihrer Organisationsmitglieder ermöglicht?
- **Rollen und Zugangskontrolle**: Muss die Anwendung Benutzern bestimmte Rollen innerhalb ihrer Organisationen zuweisen?
- **Organisationen über Logto Console und Management API verwalten:** Welche Konfigurationen möchtest du direkt in der Logto Console und Management API einrichten?
