# Benutzerverwaltung

Die Benutzerverwaltung ist ein bedeutender Schwerpunktbereich für Logto. Du kannst auf deine Benutzerdaten zugreifen und sie entweder über die Logto Console oder die Management API verwalten – beide sind effektiv. Mit diesen Tools kannst du Aufgaben wie die folgenden ausführen:

- Suchen von Benutzern nach Benutzername, ID, vollständigem Namen, Telefonnummer oder E-Mail.
- Verschiedene Aktionen am Benutzer durchführen, wie das Trennen von sozialen Konten, das Aktualisieren von Profilinformationen, das Löschen von Benutzern oder das Zurücksetzen von Passwörtern.
- Überprüfen von Benutzerprotokollen.
- Zuweisen von Rollen (Roles) für die Zugangskontrolle.
- Persönliche Zugangstokens erstellen und verwalten.
- Verwalten von Benutzersitzungen.
