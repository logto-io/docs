# Benutzerverwaltung

Die Benutzerverwaltung ist ein bedeutender Schwerpunkt für Logto. Du kannst auf deine Benutzerdaten zugreifen und sie entweder über die Logto-Konsole oder die Management API verwalten, die beide effektiv sind. Mit diesen Tools kannst du Aufgaben wie die folgenden ausführen:

- Suchen nach Benutzern anhand von Benutzernamen, ID, vollständigem Namen, Telefonnummer oder E-Mail.
- Mehrere Aktionen am Benutzer durchführen, wie das Trennen von sozialen Konten, Aktualisieren von Profilinformationen, Löschen von Benutzern oder Zurücksetzen von Passwörtern.
- Überprüfen von Benutzerprotokollen.
- Zuweisen von Rollen für die Zugangskontrolle.
- Erstellen und Verwalten von persönlichen Zugangstokens.
