Um ein neues Sinatra-Projekt zu initialisieren, erstelle ein Verzeichnis und richte die grundlegende Struktur ein:

```bash
mkdir your-api-name
cd your-api-name
```

Erstelle eine Gemfile:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'sinatra'
```

Installiere die Abhängigkeiten:

```bash
bundle install
```

Erstelle eine grundlegende Sinatra-Anwendung:

```ruby title="app.rb"
require 'sinatra'
require 'json'

get '/' do
  content_type :json
  { message: 'Hello from Sinatra API' }.to_json
end
```

Starte den Entwicklungsserver:

```bash
ruby app.rb
```

:::note
Siehe die Sinatra-Dokumentation für weitere Details zur Einrichtung von Routen, Middleware und anderen Funktionen.
:::
