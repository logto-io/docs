Um ein neues Grape API-Projekt zu initialisieren, erstelle ein Verzeichnis und richte die Grundstruktur ein:

```bash
mkdir your-api-name
cd your-api-name
```

Erstelle eine Gemfile:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

Installiere die Abhängigkeiten:

```bash
bundle install
```

Erstelle eine grundlegende Grape API:

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

Erstelle eine config.ru-Datei:

```ruby title="config.ru"
require_relative 'api'

run API
```

Starte den Entwicklungsserver:

```bash
bundle exec rackup
```

:::note
Siehe die Grape-Dokumentation für weitere Details zur Einrichtung von Ressourcen, Parameter-Validierung und anderen Funktionen.
:::
