Um ein neues Go-Projekt mit Gin zu initialisieren, kannst du folgende Schritte ausführen:

```bash
go mod init your-api-name
go get github.com/gin-gonic/gin
```

Erstelle dann ein grundlegendes Gin-Server-Setup:

```go title="main.go"
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()

    r.Run(":3000") // hören und bedienen auf 0.0.0.0:3000
}
```

:::note
Siehe die Gin-Dokumentation für weitere Details zur Einrichtung von Routen, Middleware und anderen Funktionen.
:::
