Um ein neues Go-Projekt mit Fiber zu initialisieren, kannst du folgende Schritte ausführen:

```bash
go mod init your-api-name
go get github.com/gofiber/fiber/v2
```

Erstelle anschließend ein grundlegendes Fiber-Server-Setup:

```go title="main.go"
package main

import (
    "log"

    "github.com/gofiber/fiber/v2"
)

func main() {
    app := fiber.New()

    log.Fatal(app.Listen(":3000"))
}
```

:::note
Siehe die Fiber-Dokumentation für weitere Details zur Einrichtung von Routen, Middleware und anderen Funktionen.
:::
