Um ein neues FastAPI-Projekt zu initialisieren, erstelle ein Verzeichnis und richte die Grundstruktur ein:

```bash
mkdir your-api-name
cd your-api-name
```

Erstelle eine Requirements-Datei:

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

Installiere die Abhängigkeiten:

```bash
pip install -r requirements.txt
```

Erstelle eine grundlegende FastAPI-Anwendung:

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

Starte den Entwicklungsserver:

```bash
uvicorn main:app --reload
```

:::note
Siehe die FastAPI-Dokumentation für weitere Details zur Einrichtung von Pfadoperationen, Dependency Injection und anderen Funktionen.
:::
