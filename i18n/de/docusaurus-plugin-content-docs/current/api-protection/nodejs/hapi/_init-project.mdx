Um ein neues Node.js-Projekt mit Hapi zu initialisieren, kannst du folgende Schritte ausführen:

```bash
npm init -y
npm install @hapi/hapi
```

Wenn du TypeScript verwendest, denke daran, deine TypeScript-Umgebung entsprechend einzurichten.

Erstelle dann ein grundlegendes Hapi-Server-Setup:

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Server läuft auf http://localhost:3000');
```

:::note
Siehe die Hapi-Dokumentation für weitere Details zur Einrichtung von Routen, Plugins und anderen Funktionen.
:::
