Um ein neues Node.js-Projekt mit Fastify zu initialisieren, kannst du folgende Schritte ausführen:

```bash
npm init -y
npm install fastify
```

Wenn du TypeScript verwendest, denke daran, deine TypeScript-Umgebung entsprechend einzurichten.

Erstelle dann ein grundlegendes Fastify-Server-Setup:

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Server running on http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
Siehe die Fastify-Dokumentation für weitere Details zur Einrichtung von Routen, Plugins und anderen Funktionen.
:::
