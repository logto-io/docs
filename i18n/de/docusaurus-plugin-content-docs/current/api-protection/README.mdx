# API-Schutz

Lerne, wie du [rollenbasierte Zugangskontrolle (RBAC)](/authorization/role-based-access-control) und [JSON Web Token (JWT)](https://auth.wiki/jwt) Validierung implementierst, um deine API-Endpunkte vor unbefugtem Zugriff zu schützen.

Unsere Anleitungen behandeln die Einrichtung von Middleware, Berechtigungsmodelle und praktische Beispiele in mehreren Programmiersprachen und Frameworks.

## Loslegen \{#get-started}

### Neu bei API-Sicherheit? \{#new-to-api-security}

Wenn du neu bei Authentifizierung (Authentication) und Autorisierung (Authorization) für APIs bist, beginne mit diesen Grundlagen:

- **[Authentifizierung vs. Autorisierung](/concepts/authn-vs-authz)** – Verstehe die wichtigsten Unterschiede
- **[Rollenbasierte Zugangskontrolle (RBAC)](/authorization/role-based-access-control)** – Erfahre, wie Berechtigungen und Rollen funktionieren
- **[JSON Web Token (JWT)](https://auth.wiki/jwt)** – Entdecke, wie JSON Web Tokens deine APIs absichern

### Wähle deine Sprache oder dein Framework \{#choose-your-language-or-framework}

Bereit, den API-Schutz zu implementieren? Wähle deinen Technologie-Stack in der Seitenleiste aus, um Schritt-für-Schritt-Integrationsanleitungen mit Codebeispielen und Best Practices zu erhalten.
