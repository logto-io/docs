Um ein neues Symfony-Projekt für die API-Entwicklung zu initialisieren, verwende die Symfony CLI oder Composer:

Mit der Symfony CLI (empfohlen):

```bash
symfony new your-api-name --webapp
cd your-api-name
```

Oder mit Composer:

```bash
composer create-project symfony/skeleton your-api-name
cd your-api-name
composer require webapp
```

Installiere zusätzliche Pakete für die API-Entwicklung:

```bash
composer require symfony/security-bundle
composer require symfony/serializer
composer require doctrine/annotations
```

Starte den Entwicklungsserver:

```bash
symfony serve
```

Oder mit dem integrierten Server von PHP:

```bash
php -S localhost:8000 -t public/
```

Dies erstellt ein grundlegendes Symfony-Projekt. Konfiguriere das Framework für die API-Entwicklung:

```yaml title="config/packages/framework.yaml"
framework:
  secret: '%env(APP_SECRET)%'
  serializer:
    enabled: true
  property_access:
    enabled: true
```

:::note
Siehe die Symfony-Dokumentation für weitere Details zur Einrichtung von Controllern, Services und anderen Funktionen.
:::
