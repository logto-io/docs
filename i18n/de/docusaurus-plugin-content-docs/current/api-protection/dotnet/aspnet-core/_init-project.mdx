Um ein neues .NET Web API-Projekt zu initialisieren, kannst du die .NET CLI verwenden:

```bash
dotnet new webapi -n YourApiName
cd YourApiName
```

Füge das erforderliche NuGet-Paket für JWT-Authentifizierung hinzu:

```bash
dotnet add package Microsoft.AspNetCore.Authentication.JwtBearer
```

Erstelle einen einfachen API-Controller:

```csharp title="Controllers/ApiController.cs"
using Microsoft.AspNetCore.Mvc;

namespace YourApiName.Controllers
{
    [ApiController]
    [Route("api/[controller]")]
    public class ApiController : ControllerBase
    {
        [HttpGet]
        public IActionResult Get()
        {
            return Ok(new { message = "Hello from .NET API" });
        }
    }
}
```

Starte den Entwicklungsserver:

```bash
dotnet run
```

:::note
Weitere Informationen zum Einrichten von Controllern, Middleware und anderen Funktionen findest du in der ASP.NET Core-Dokumentation.
:::
