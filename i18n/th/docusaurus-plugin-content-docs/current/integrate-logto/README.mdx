# ผสานรวมการยืนยันตัวตน (Authentication) ของ Logto

Logto มอบโซลูชันการยืนยันตัวตนที่ครบถ้วนสำหรับแอปพลิเคชันเว็บ โมบายล์ และเดสก์ท็อป รองรับ [การยืนยันตัวตนระหว่างเครื่อง (Machine-to-Machine; M2M)](/quick-starts/m2m) ระหว่างบริการ และสามารถให้บริการเป็นผู้ให้บริการข้อมูลระบุตัวตน (IdP) สำหรับ [แอปพลิเคชันของบุคคลที่สาม](/integrate-logto/third-party-applications) ผ่านโปรโตคอลมาตรฐาน เช่น [OpenID Connect (OIDC)](https://auth.wiki/openid-connect) และ [OAuth 2.0](https://auth.wiki/oauth-2.0)

เริ่มต้นการผสานรวมของคุณโดยเลือกโซลูชันที่ตรงกับความต้องการมากที่สุด:

## เพิ่มการยืนยันตัวตนให้กับแอปพลิเคชันของคุณ \{#add-authentication-for-your-applications}

ไม่ว่าคุณจะสร้างแอปพลิเคชันสำหรับผู้ใช้ปลายทาง (เช่น เว็บ โมบายล์ หรือเดสก์ท็อป) หรือแอปพลิเคชันแบบเครื่องต่อเครื่อง (M2M) สำหรับการสื่อสารระหว่างบริการ คุณสามารถนำเสนอฟีเจอร์ [การยืนยันตัวตน (Authentication)](/end-user-flows) และ [การจัดการผู้ใช้ (User management)](/user-management) ได้อย่างรวดเร็วด้วยการผสานรวม Logto

ค้นหาคู่มือการผสานรวมสำหรับเฟรมเวิร์กหรือภาษาโปรแกรมที่คุณต้องการ:

## เพิ่มแอป SAML \{#add-saml-apps}

Logto (ในฐานะผู้ให้บริการข้อมูลระบุตัวตน) รองรับการผสานรวมกับแอปพลิเคชันที่ใช้โปรโตคอล SAML (ในฐานะ Service Provider) ผ่านการผสานรวมแอป SAML คุณสามารถมอบประสบการณ์ Single Sign-On (SSO) ที่ปลอดภัยและได้มาตรฐานให้กับผู้ใช้ระดับองค์กร

## เพิ่มแอป OIDC / OAuth \{#add-oidc-oauth-apps}

เมื่อคุณต้องการให้แอปพลิเคชันของบุคคลที่สามสามารถยืนยันตัวตนกับระบบ Logto ของคุณได้ คุณสามารถตั้งค่า Logto เป็น [ผู้ให้บริการข้อมูลระบุตัวตน (IdP)](https://auth.wiki/identity-provider)

Logto สามารถใช้เป็น IdP ผ่านโปรโตคอลมาตรฐาน เช่น OAuth, OIDC และบริการภายนอกสามารถนำ “เข้าสู่ระบบด้วย Logto” ไปใช้ได้เช่นเดียวกับ “เข้าสู่ระบบด้วย Google” ทำให้ผู้ใช้ของพวกเขาสามารถยืนยันตัวตนผ่านระบบผู้ใช้ของคุณใน Logto

โดยการสร้างแอปพลิเคชันของบุคคลที่สามใน Logto คุณจะสร้างการเชื่อมต่อที่ปลอดภัยระหว่างระบบข้อมูลระบุตัวตนของคุณกับบริการภายนอก

## ขยายความสามารถของ Logto ด้วย Management API \{#extend-logto-features-with-management-api}

Logto [Management API](https://openapi.logto.io/) ช่วยให้คุณควบคุมและทำงานอัตโนมัติของฟังก์ชัน Logto ในแอปพลิเคชันของคุณได้แบบโปรแกรม

เมื่อคุณต้องการขยายความสามารถของ Logto เพื่อยกระดับผลิตภัณฑ์ของคุณ (เช่น [การจัดการองค์กรแบบหลายผู้เช่า](/organizations) หรือการทำงานอัตโนมัติร่วมกับ Logto) คุณสามารถใช้ Logto Management API เพื่อสร้างฟีเจอร์ขั้นสูงให้เหมาะกับความต้องการของผลิตภัณฑ์
