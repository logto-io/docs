# ปกป้อง Go API ของคุณด้วยการควบคุมการเข้าถึงตามบทบาท (RBAC) และการตรวจสอบ JWT

เรียนรู้วิธีรักษาความปลอดภัยให้กับ Go API ของคุณด้วย [การควบคุมการเข้าถึงตามบทบาท (RBAC)](/authorization/role-based-access-control) และ [JSON Web Token (JWT)](https://auth.wiki/jwt)

แต่ละหน้าจะครอบคลุมรูปแบบการใช้งาน การตั้งค่า middleware และตัวอย่างโดยละเอียดสำหรับโมเดลสิทธิ์ต่าง ๆ เพื่อช่วยคุณปกป้อง endpoint ของ API ด้วยการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่เหมาะสม
