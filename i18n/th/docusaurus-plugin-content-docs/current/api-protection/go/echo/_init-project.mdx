ในการเริ่มต้นโปรเจกต์ Go ใหม่ด้วย Echo คุณสามารถทำตามขั้นตอนดังนี้:

```bash
go mod init your-api-name
go get github.com/labstack/echo/v4
```

จากนั้น สร้างการตั้งค่าเซิร์ฟเวอร์ Echo พื้นฐาน:

```go title="main.go"
package main

import (
    "github.com/labstack/echo/v4"
)

func main() {
    e := echo.New()

    e.Logger.Fatal(e.Start(":3000"))
}
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), มิดเดิลแวร์ (middleware) และฟีเจอร์อื่น ๆ ได้ที่เอกสารของ Echo
:::
