ในการเริ่มต้นโปรเจกต์ Go ใหม่ด้วย Chi คุณสามารถทำตามขั้นตอนดังนี้:

```bash
go mod init your-api-name
go get github.com/go-chi/chi/v5
```

จากนั้น สร้างการตั้งค่าเซิร์ฟเวอร์ Chi พื้นฐาน:

```go title="main.go"
package main

import (
    "net/http"

    "github.com/go-chi/chi/v5"
)

func main() {
    r := chi.NewRouter()

    http.ListenAndServe(":3000", r)
}
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), มิดเดิลแวร์ (middleware) และฟีเจอร์อื่น ๆ ได้ที่เอกสารของ Chi
:::
