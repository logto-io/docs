# ปกป้อง API Rust ของคุณด้วย RBAC และการตรวจสอบ JWT

เรียนรู้วิธีรักษาความปลอดภัยให้กับ API Rust ของคุณโดยใช้ [การควบคุมการเข้าถึงตามบทบาท (RBAC)](/authorization/role-based-access-control) และ [JSON Web Token (JWT)](https://auth.wiki/jwt)

แต่ละหน้าจะครอบคลุมรูปแบบการใช้งาน การตั้งค่า middleware และตัวอย่างโดยละเอียดสำหรับโมเดลสิทธิ์ที่แตกต่างกัน เพื่อช่วยให้คุณปกป้อง endpoint ของ API ด้วยการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่เหมาะสม
