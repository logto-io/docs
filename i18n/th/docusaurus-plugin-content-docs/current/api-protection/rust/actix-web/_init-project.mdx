ในการเริ่มต้นโปรเจกต์ Actix Web ใหม่ ให้สร้างไดเรกทอรีและตั้งค่าโครงสร้างพื้นฐานดังนี้:

```bash
cargo new your-api-name
cd your-api-name
```

เพิ่ม dependencies ของ Actix Web ในไฟล์ `Cargo.toml` ของคุณ:

```toml title="Cargo.toml"
[dependencies]
actix-web = "4.0"
tokio = { version = "1.0", features = ["full"] }
serde = { version = "1.0", features = ["derive"] }
serde_json = "1.0"
```

สร้างแอปพลิเคชัน Actix Web พื้นฐาน:

```rust title="src/main.rs"
use actix_web::{web, App, HttpServer, Result};
use serde_json::{json, Value};

#[actix_web::main]
async fn main() -> std::io::Result<()> {
    HttpServer::new(|| {
        App::new()
            .route("/", web::get().to(hello_handler))
    })
    .bind("127.0.0.1:8080")?
    .run()
    .await
}

async fn hello_handler() -> Result<web::Json<Value>> {
    Ok(web::Json(json!({ "message": "Hello from Actix Web" })))
}
```

เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา:

```bash
cargo run
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), มิดเดิลแวร์ (middleware) และฟีเจอร์อื่น ๆ ได้จากเอกสารของ Actix Web
:::
