# ปกป้อง .NET API ของคุณด้วย RBAC และการตรวจสอบ JWT

เรียนรู้วิธีรักษาความปลอดภัยให้กับ .NET API ของคุณโดยใช้ [การควบคุมการเข้าถึงตามบทบาท (RBAC)](/authorization/role-based-access-control) และ [JSON Web Token (JWT)](https://auth.wiki/jwt)

{/* คำเตือน: เราไม่สามารถใช้ตัวแปรในหัวข้อหรือข้อความแนะนำได้ เนื่องจากจะไม่ถูกแสดงผลอย่างถูกต้อง */}

แต่ละหน้าจะครอบคลุมรูปแบบการใช้งาน การตั้งค่า middleware และตัวอย่างโดยละเอียดสำหรับโมเดลสิทธิ์ที่แตกต่างกัน เพื่อช่วยคุณปกป้อง endpoint ของ API ด้วยการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่เหมาะสม
