ในการเริ่มต้นโปรเจกต์ Rails API ใหม่ คุณสามารถใช้ Rails generator ได้ดังนี้:

```bash
rails new your-api-name --api
cd your-api-name
```

เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา:

```bash
rails server
```

สร้าง API controller พื้นฐาน:

```ruby title="app/controllers/api/base_controller.rb"
class Api::BaseController < ApplicationController
  def index
    render json: { message: 'Hello from Rails API' }
  end
end
```

เพิ่ม routes:

```ruby title="config/routes.rb"
Rails.application.routes.draw do
  namespace :api do
    root 'base#index'
  end
end
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่า controller, model และฟีเจอร์อื่น ๆ ได้จากเอกสารของ Rails
:::
