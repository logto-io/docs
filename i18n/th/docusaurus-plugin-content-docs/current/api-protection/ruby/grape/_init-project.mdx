ในการเริ่มต้นโปรเจกต์ Grape API ใหม่ ให้สร้างไดเรกทอรีและตั้งค่าโครงสร้างพื้นฐานดังนี้:

```bash
mkdir your-api-name
cd your-api-name
```

สร้างไฟล์ Gemfile:

```ruby title="Gemfile"
source 'https://rubygems.org'

gem 'grape'
```

ติดตั้ง dependencies:

```bash
bundle install
```

สร้าง Grape API พื้นฐาน:

```ruby title="api.rb"
require 'grape'

class API < Grape::API
  format :json

  get :hello do
    { message: 'Hello from Grape API' }
  end
end
```

สร้างไฟล์ config.ru:

```ruby title="config.ru"
require_relative 'api'

run API
```

เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา:

```bash
bundle exec rackup
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าทรัพยากร การตรวจสอบความถูกต้องของพารามิเตอร์ และฟีเจอร์อื่น ๆ ได้จากเอกสารของ Grape
:::
