# ปกป้อง Ruby API ของคุณด้วย RBAC และการตรวจสอบ JWT

เรียนรู้วิธีรักษาความปลอดภัย Ruby API ของคุณด้วย [การควบคุมการเข้าถึงตามบทบาท (RBAC)](/authorization/role-based-access-control) และ [JSON Web Token (JWT)](https://auth.wiki/jwt)

{/* คำเตือน: ไม่สามารถใช้ตัวแปรในหัวข้อหรือข้อความแนะนำได้ เนื่องจากจะไม่ถูกแสดงผลอย่างถูกต้อง */}

แต่ละหน้าครอบคลุมรูปแบบการใช้งาน การตั้งค่า middleware และตัวอย่างโดยละเอียดสำหรับโมเดลสิทธิ์ที่แตกต่างกัน เพื่อช่วยให้คุณปกป้อง API endpoint ของคุณด้วยการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่เหมาะสม
