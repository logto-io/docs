ในการเริ่มต้นโปรเจกต์ Flask ใหม่ ให้สร้างไดเรกทอรีและตั้งค่าโครงสร้างพื้นฐานดังนี้:

```bash
mkdir your-api-name
cd your-api-name
```

สร้างไฟล์ requirements:

```txt title="requirements.txt"
Flask
```

ติดตั้ง dependencies:

```bash
pip install -r requirements.txt
```

สร้างแอปพลิเคชัน Flask พื้นฐาน:

```py title="app.py"
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def hello():
    return jsonify({"message": "Hello from Flask"})

if __name__ == '__main__':
    app.run(debug=True)
```

เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา:

```bash
python app.py
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่า routes, blueprints และฟีเจอร์อื่น ๆ ได้ที่เอกสาร Flask
:::
