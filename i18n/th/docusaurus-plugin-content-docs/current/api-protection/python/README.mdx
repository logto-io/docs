# ปกป้อง API Python ของคุณด้วย RBAC และการตรวจสอบ JWT

เรียนรู้วิธีรักษาความปลอดภัยให้กับ API Python ของคุณโดยใช้ [การควบคุมการเข้าถึงตามบทบาท (RBAC)](/authorization/role-based-access-control) และ [JSON Web Token (JWT)](https://auth.wiki/jwt)

แต่ละหน้าจะครอบคลุมรูปแบบการใช้งาน การตั้งค่ามิดเดิลแวร์ และตัวอย่างโดยละเอียดสำหรับโมเดลสิทธิ์ที่แตกต่างกัน เพื่อช่วยให้คุณปกป้อง API endpoint ของคุณด้วยการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่เหมาะสม
