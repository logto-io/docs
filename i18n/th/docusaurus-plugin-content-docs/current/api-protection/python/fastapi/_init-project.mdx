ในการเริ่มต้นโปรเจกต์ FastAPI ใหม่ ให้สร้างไดเรกทอรีและตั้งค่าโครงสร้างพื้นฐานดังนี้:

```bash
mkdir your-api-name
cd your-api-name
```

สร้างไฟล์ requirements:

```txt title="requirements.txt"
fastapi
uvicorn[standard]
```

ติดตั้ง dependencies:

```bash
pip install -r requirements.txt
```

สร้างแอปพลิเคชัน FastAPI พื้นฐาน:

```py title="main.py"
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI"}
```

เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา:

```bash
uvicorn main:app --reload
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่า path operations, dependency injection และฟีเจอร์อื่น ๆ ได้ที่เอกสาร FastAPI
:::
