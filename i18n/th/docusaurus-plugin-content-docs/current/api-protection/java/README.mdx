# ปกป้อง Java API ของคุณด้วยการควบคุมการเข้าถึงตามบทบาท (RBAC) และการตรวจสอบ JWT

เรียนรู้วิธีรักษาความปลอดภัยให้กับ Java API ของคุณโดยใช้ [การควบคุมการเข้าถึงตามบทบาท (RBAC)](/authorization/role-based-access-control) และ [JSON Web Token (JWT)](https://auth.wiki/jwt)

แต่ละหน้าจะครอบคลุมรูปแบบการใช้งาน การตั้งค่ามิดเดิลแวร์ และตัวอย่างโดยละเอียดสำหรับโมเดลสิทธิ์ต่าง ๆ เพื่อช่วยคุณปกป้อง API endpoint ของคุณด้วยการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่เหมาะสม
