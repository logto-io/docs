ในการเริ่มต้นโปรเจกต์ Node.js ใหม่ด้วย Fastify คุณสามารถทำตามขั้นตอนดังนี้:

```bash
npm init -y
npm install fastify
```

หากคุณใช้ TypeScript อย่าลืมตั้งค่าสภาพแวดล้อม TypeScript ให้เหมาะสมด้วย

จากนั้น สร้างการตั้งค่าเซิร์ฟเวอร์ Fastify พื้นฐาน:

```ts title="app.ts"
import fastify from 'fastify';

const server = fastify({ logger: true });

try {
  await server.listen({ port: 3000 });
  console.log('Server running on http://localhost:3000');
} catch (err) {
  server.log.error(err);
  process.exit(1);
}
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), ปลั๊กอิน (plugins) และฟีเจอร์อื่น ๆ ได้จากเอกสาร Fastify
:::
