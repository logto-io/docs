ในการเริ่มต้นโปรเจกต์ Node.js ใหม่ด้วย Express คุณสามารถทำตามขั้นตอนดังนี้:

```bash
npm init -y
npm install express
```

หากคุณใช้ TypeScript อย่าลืมตั้งค่าสภาพแวดล้อม TypeScript ให้เหมาะสมด้วย

จากนั้น สร้างเซิร์ฟเวอร์ Express พื้นฐานดังนี้:

```ts title="app.ts"
import express from 'express';

const app = express();

app.listen(3000, () => {
  console.log('เซิร์ฟเวอร์ทำงานที่ http://localhost:3000');
});
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), มิดเดิลแวร์ (middleware) และฟีเจอร์อื่น ๆ ได้ที่เอกสาร Express
:::
