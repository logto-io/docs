ในการเริ่มต้นโปรเจกต์ Node.js ใหม่ด้วย Hapi คุณสามารถทำตามขั้นตอนดังนี้:

```bash
npm init -y
npm install @hapi/hapi
```

หากคุณใช้ TypeScript อย่าลืมตั้งค่าสภาพแวดล้อม TypeScript ให้เหมาะสมด้วย

จากนั้น สร้างการตั้งค่าเซิร์ฟเวอร์ Hapi พื้นฐาน:

```ts title="app.ts"
import Hapi from '@hapi/hapi';

const server = Hapi.server({
  port: 3000,
  host: 'localhost',
});

await server.start();
console.log('Server running on http://localhost:3000');
// เซิร์ฟเวอร์กำลังทำงานที่ http://localhost:3000
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), ปลั๊กอิน (plugins) และฟีเจอร์อื่น ๆ ได้จากเอกสาร Hapi
:::
