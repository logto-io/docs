ในการเริ่มต้นโปรเจกต์ Node.js ใหม่ด้วย Koa คุณสามารถทำตามขั้นตอนดังนี้:

```bash
npm init -y
npm install koa @koa/router
```

หากคุณใช้ TypeScript อย่าลืมตั้งค่าสภาพแวดล้อม TypeScript ของคุณให้เหมาะสม

จากนั้น สร้างเซิร์ฟเวอร์ Koa พื้นฐานดังนี้:

```ts title="app.ts"
import Koa from 'koa';

const app = new Koa();

app.listen(3000, () => {
  console.log('เซิร์ฟเวอร์ทำงานที่ http://localhost:3000');
});
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าเส้นทาง (routes), มิดเดิลแวร์ (middleware) และฟีเจอร์อื่น ๆ ได้จากเอกสาร Koa
:::
