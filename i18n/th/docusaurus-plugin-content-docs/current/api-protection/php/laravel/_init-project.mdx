ในการเริ่มต้นโปรเจกต์ Laravel ใหม่ คุณสามารถใช้ Laravel installer หรือ Composer ได้ดังนี้:

ใช้ Laravel installer (แนะนำ):

```bash
composer global require laravel/installer
laravel new your-api-name
cd your-api-name
```

หรือใช้ Composer โดยตรง:

```bash
composer create-project laravel/laravel your-api-name
cd your-api-name
```

เริ่มเซิร์ฟเวอร์สำหรับพัฒนา:

```bash
php artisan serve
```

คำสั่งนี้จะสร้างโครงสร้างโปรเจกต์ Laravel พื้นฐาน สำหรับการพัฒนา API คุณอาจต้องการลบ middleware และ route ที่เกี่ยวข้องกับเว็บบางส่วนออก:

```php title="bootstrap/app.php"
<?php

use Illuminate\Foundation\Application;
use Illuminate\Foundation\Configuration\Exceptions;
use Illuminate\Foundation\Configuration\Middleware;

return Application::configure(basePath: dirname(__DIR__))
    ->withRouting(
        web: __DIR__.'/../routes/web.php',
        api: __DIR__.'/../routes/api.php',
        commands: __DIR__.'/../routes/console.php',
        health: '/up',
    )
    ->withMiddleware(function (Middleware $middleware) {
        // กำหนดค่า middleware สำหรับ API ตามที่ต้องการ
    })
    ->withExceptions(function (Exceptions $exceptions) {
        //
    })->create();
```

:::note
ดูรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่า controller, middleware และฟีเจอร์อื่น ๆ ได้จากเอกสารของ Laravel
:::
