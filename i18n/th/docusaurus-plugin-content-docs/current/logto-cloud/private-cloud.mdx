# Logto Private Cloud

ปรับใช้ Logto ในสภาพแวดล้อม Azure เฉพาะสำหรับการแยกข้อมูลที่แข็งแกร่งขึ้น สอดคล้องกับข้อกำหนดด้านการปฏิบัติตามกฎระเบียบ และควบคุมโครงสร้างพื้นฐานได้มากขึ้น

## ภาพรวม \{#overview}

Logto Private Cloud เป็นตัวเลือกการปรับใช้แบบเฉพาะสำหรับลูกค้าที่ต้องการการแยกโครงสร้างพื้นฐานและการควบคุมมากกว่าสภาพแวดล้อมคลาวด์สาธารณะร่วม

ออกแบบมาสำหรับองค์กรที่มีข้อกำหนดด้านความปลอดภัย การปฏิบัติตามกฎระเบียบ การจัดซื้อ หรือการปรับใช้ที่เข้มงวดกว่า โดยยังคงประสบการณ์ผลิตภัณฑ์หลักเดียวกับ Logto Cloud

นี่คือความแตกต่างของตัวเลือกคลาวด์แต่ละแบบ:

- **Logto Cloud**: คลาวด์สาธารณะที่มีการจัดการ พร้อมใช้งานได้เร็วที่สุด
- **Logto Private Cloud**: การปรับใช้แบบเฉพาะสำหรับลูกค้าที่มีข้อกำหนดด้านที่ตั้งข้อมูลหรือการปฏิบัติตามกฎระเบียบที่เข้มงวดกว่า

## เมื่อใดควรเลือก Private Cloud \{#when-to-choose-private-cloud}

Logto Private Cloud มักเหมาะสมเมื่อองค์กรของคุณต้องการสิ่งใดสิ่งหนึ่งต่อไปนี้:

- การแยกสภาพแวดล้อมที่แข็งแกร่งกว่า
- ข้อผูกพันด้านความพร้อมใช้งาน (SLA) พร้อมการสนับสนุนและความร่วมมือจากทีม Logto
- ข้อกำหนดด้านการปฏิบัติตามกฎระเบียบเฉพาะที่คลาวด์สาธารณะร่วมไม่สามารถตอบสนองได้
- การปรับใช้ในภูมิภาคหรือโครงสร้างพื้นฐานเฉพาะที่ Logto Cloud ไม่รองรับ
- การเชื่อมต่อกับทรัพยากรภายในที่ไม่สามารถเข้าถึงได้จากคลาวด์สาธารณะ

หากคุณไม่มีข้อกำหนดเหล่านี้ [Logto Cloud](https://cloud.logto.io/?sign_up=1) มักเป็นตัวเลือกที่ดีกว่า เพราะใช้งานง่ายกว่าและดูแลรักษาง่ายกว่า

### ขอบเขตการปรับใช้ \{#deployment-scope}

ขณะนี้ Logto Private Cloud มีให้บริการบน **Azure** เท่านั้น

ภูมิภาคที่รองรับขึ้นอยู่กับว่า Azure ในภูมิภาคนั้นให้บริการที่ Logto ต้องการหรือไม่ หากคุณมีภูมิภาคเป้าหมายในใจ กรุณาติดต่อเราเพื่อยืนยันความเป็นไปได้ตามข้อกำหนดของคุณ

### ความปลอดภัยและการปฏิบัติตามกฎระเบียบ \{#security-and-compliance}

Private Cloud มอบสภาพแวดล้อมเฉพาะที่มีขอบเขตการแยกข้อมูลชัดเจนกว่าการปรับใช้บนคลาวด์สาธารณะร่วม

เราให้ความสำคัญกับการพูดคุยเรื่องความปลอดภัยและการปฏิบัติตามกฎระเบียบอย่างตรงประเด็นและขับเคลื่อนด้วยข้อกำหนด หากทีมของคุณต้องการการตรวจสอบความปลอดภัย การพูดคุยเรื่องการปฏิบัติตามกฎระเบียบ หรือคำชี้แจงเฉพาะการปรับใช้ กรุณาติดต่อเราเพื่อพูดคุยรายละเอียดโดยตรง
