# Logto Private Cloud

ปรับใช้ Logto ในสภาพแวดล้อม Azure เฉพาะสำหรับการแยกข้อมูลที่แข็งแกร่งขึ้น สอดคล้องกับข้อกำหนด และควบคุมโครงสร้างพื้นฐานได้มากขึ้น

## ภาพรวม \{#overview}

Logto Private Cloud คือทางเลือกการปรับใช้แบบเฉพาะสำหรับลูกค้าที่ต้องการการแยกโครงสร้างพื้นฐานและการควบคุมมากกว่าสภาพแวดล้อมคลาวด์สาธารณะร่วม

ออกแบบมาสำหรับองค์กรที่มีข้อกำหนดด้านความปลอดภัย การปฏิบัติตามข้อกำหนด การจัดซื้อ หรือการปรับใช้ที่เข้มงวดกว่า โดยยังคงประสบการณ์ผลิตภัณฑ์หลักเหมือนกับ Logto Cloud

นี่คือความแตกต่างของตัวเลือกการปรับใช้แต่ละแบบ:

- **Logto Cloud**: คลาวด์สาธารณะที่มีการจัดการ พร้อมใช้งานได้เร็วที่สุด
- **Logto Private Cloud**: การปรับใช้แบบเฉพาะสำหรับลูกค้าที่มีข้อกำหนดด้านโครงสร้างพื้นฐานหรือการปฏิบัติตามข้อกำหนดที่เข้มงวดกว่า
- **Logto OSS**: โฮสต์และจัดการด้วยตนเอง

## เมื่อใดควรเลือก Private Cloud \{#when-to-choose-private-cloud}

Logto Private Cloud มักจะเหมาะสมเมื่อองค์กรของคุณต้องการสิ่งใดสิ่งหนึ่งต่อไปนี้:

- การแยกสภาพแวดล้อมที่แข็งแกร่งกว่า
- ข้อผูกพันด้านความพร้อมใช้งาน (SLA) พร้อมการสนับสนุนและความร่วมมือจากทีม Logto
- ข้อกำหนดด้านการปฏิบัติตามข้อกำหนดเฉพาะที่ไม่สามารถทำได้ในคลาวด์สาธารณะร่วม
- การปรับใช้ในภูมิภาคหรือโครงสร้างพื้นฐานเฉพาะที่ Logto Cloud ไม่รองรับ
- การเชื่อมต่อกับทรัพยากรภายในที่ไม่สามารถเข้าถึงได้จากคลาวด์สาธารณะ

หากคุณไม่มีข้อกำหนดเหล่านี้ [Logto Cloud](https://cloud.logto.io/?sign_up=1) มักจะเป็นตัวเลือกที่ดีกว่า เพราะใช้งานง่ายกว่าและดูแลรักษาง่ายกว่า

### ขอบเขตการปรับใช้ \{#deployment-scope}

ขณะนี้ Logto Private Cloud มีให้บริการบน **Azure** เท่านั้น

ภูมิภาคที่รองรับขึ้นอยู่กับว่า Azure region นั้นให้บริการที่ Logto ต้องการหรือไม่ หากคุณมีภูมิภาคเป้าหมายในใจ กรุณาติดต่อเราเพื่อยืนยันความเป็นไปได้ตามข้อกำหนดของคุณ

### ความปลอดภัยและการปฏิบัติตามข้อกำหนด \{#security-and-compliance}

Private Cloud มอบสภาพแวดล้อมเฉพาะที่มีขอบเขตการแยกที่ชัดเจนกว่าการปรับใช้ในคลาวด์สาธารณะร่วม

เราให้ความสำคัญกับการพูดคุยเรื่องความปลอดภัยและการปฏิบัติตามข้อกำหนดอย่างเป็นรูปธรรมและขับเคลื่อนด้วยความต้องการ หากทีมของคุณต้องการการตรวจสอบความปลอดภัย การพูดคุยเรื่องการปฏิบัติตามข้อกำหนด หรือคำชี้แจงเฉพาะการปรับใช้ กรุณาติดต่อเราเพื่อพูดคุยรายละเอียดโดยตรง
