# Logto MCP Server

Logto MCP Server คือเซิร์ฟเวอร์ [MCP (Model Context Protocol)](https://modelcontextprotocol.io/) ระยะไกล เพิ่ม URL นี้ลงในแอปพลิเคชัน AI ของคุณ ลงชื่อเข้าใช้ด้วยบัญชี Logto Cloud ของคุณ แล้วเริ่มพูดคุยกับ Logto ได้เลย

:::tip
Logto MCP Server ถูกสร้างขึ้นด้วย Logto และ [MCP Auth](https://mcp-auth.dev) หากคุณต้องการสร้างเซิร์ฟเวอร์ MCP ระยะไกลของคุณเอง ดูที่ [สร้างเซิร์ฟเวอร์ MCP ระยะไกลของคุณเอง](#build-your-own-remote-mcp-server)
:::

## สิ่งที่คุณสามารถทำได้ \{#what-you-can-do}

- **เพิ่มการยืนยันตัวตนให้กับโปรเจกต์ของคุณ**: AI จะตรวจจับ framework ของคุณ สร้างแอป Logto และสร้างโค้ดสำหรับเชื่อมต่อที่ใช้งานได้จริง รองรับ framework มากกว่า 38 ตัว เช่น React, Next.js, Vue, Nuxt, SvelteKit, Express, Go, Python, iOS, Android, Flutter และอื่น ๆ
- **ถามคำถามเกี่ยวกับ Logto**: AI จะค้นหาข้อมูลจากเอกสาร Logto และให้คำตอบพร้อมแหล่งอ้างอิง

ความสามารถอื่น ๆ กำลังจะมาเร็ว ๆ นี้ เช่น การตั้งค่าประสบการณ์การลงชื่อเข้าใช้ การจัดการ RBAC และสิทธิ์ (permissions) พร้อมคำแนะนำแนวปฏิบัติที่ดีที่สุด การตั้งค่าหลายผู้เช่าสำหรับสร้างแอป B2B ด้วยองค์กร และอื่น ๆ

## ตั้งค่าในแอปพลิเคชัน AI ของคุณ \{#set-up-in-your-ai-application}

### Claude Desktop \{#claude-desktop}

1. เปิด **Settings** จากเมนูหลัก ไปที่ **Connectors** แล้วคลิก **Add custom connector**
2. กรอกชื่อ `Logto MCP Server` และ URL ของเซิร์ฟเวอร์ MCP ระยะไกล: `https://mcp.logto.io`
3. ขยาย **Advanced settings** และกรอก OAuth Client ID: `mcp-client` จากนั้นคลิก **Add**
4. ในรายการ **Connectors** หา **Logto MCP Server** แล้วคลิก **Connect**
5. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าลงชื่อเข้าใช้ Logto Cloud ลงชื่อเข้าใช้และอนุญาตเพื่อเชื่อมต่อ

### Cursor \{#cursor}

1. เปิด command palette (`Command + Shift + P`) แล้วเลือก **View: Open MCP Settings**
2. ในส่วน **Installed MCP Servers** คลิกปุ่ม **Add Custom MCP**
3. ในไฟล์ `mcp.json` ที่เปิดขึ้นมา ให้เพิ่มการตั้งค่าต่อไปนี้:

   ```json
   {
     "mcpServers": {
       "Logto": {
         "command": "npx",
         "args": [
           "mcp-remote",
           "https://mcp.logto.io",
           "--allow-http",
           "--static-oauth-client-info",
           "{ \"client_id\": \"mcp-client\", \"scope\": \"mcp:all\" }",
           "--resource",
           "https://mcp.logto.io/"
         ]
       }
     }
   }
   ```

หน้าลงชื่อเข้าใช้ Logto Cloud จะปรากฏขึ้นโดยอัตโนมัติ ลงชื่อเข้าใช้ด้วยบัญชี Logto Cloud ของคุณเพื่อเชื่อมต่อ

### OpenCode \{#opencode}

1. เพิ่มการตั้งค่าต่อไปนี้ลงในไฟล์ `opencode.json` ของคุณ:

   ```json
   {
     "$schema": "https://opencode.ai/config.json",
     "mcp": {
       "Logto": {
         "enabled": true,
         "type": "remote",
         "url": "https://mcp.logto.io",
         "oauth": {
           "clientId": "mcp-client"
         }
       }
     }
   }
   ```

2. รันคำสั่งต่อไปนี้เพื่อยืนยันตัวตน:
   ```bash
   opencode mcp auth Logto
   ```
   หมายเหตุว่า `Logto` ตรงกับชื่อ MCP server ในไฟล์การตั้งค่าของคุณ

หน้าต่างเบราว์เซอร์จะเปิดขึ้น ลงชื่อเข้าใช้ด้วยบัญชี Logto Cloud ของคุณเพื่ออนุญาตและเชื่อมต่อ

### VS Code \{#vs-code}

1. เปิด command palette (`Command + Shift + P`) แล้วเลือก **MCP: Add Server...**
2. เลือก **HTTP** เป็นประเภทเซิร์ฟเวอร์
3. กรอก URL: `https://mcp.logto.io` คุณสามารถเลือกติดตั้งใน workspace เพื่อการแยกและจัดการที่ดียิ่งขึ้น
4. เมื่อเริ่มกระบวนการ OAuth ให้กรอก App ID: `mcp-client` เว้น App Secret ว่างไว้แล้วกด Enter เพื่อข้าม
5. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าลงชื่อเข้าใช้ Logto Cloud ลงชื่อเข้าใช้ด้วยบัญชี Logto Cloud ของคุณเพื่อเชื่อมต่อ

### Windsurf \{#windsurf}

1. เปิด command palette (`Command + Shift + P`) แล้วเลือก **Windsurf: MCP Marketplace**
2. ในส่วน **Installed MCPs** คลิกไอคอนตั้งค่าทางขวาเพื่อสร้างและเปิดไฟล์ `~/.codeium/windsurf/mcp_config.json`
3. เพิ่ม [การตั้งค่า](https://docs.windsurf.com/windsurf/cascade/mcp#mcp_config-json) ต่อไปนี้:

   ```json
   {
     "mcpServers": {
       "Logto": {
         "command": "npx",
         "args": [
           "mcp-remote",
           "https://mcp.logto.io",
           "--allow-http",
           "--static-oauth-client-info",
           "{ \"client_id\": \"mcp-client\", \"scope\": \"mcp:all\" }",
           "--resource",
           "https://mcp.logto.io/"
         ]
       }
     }
   }
   ```

หน้าลงชื่อเข้าใช้ Logto Cloud จะปรากฏขึ้นโดยอัตโนมัติ ลงชื่อเข้าใช้ด้วยบัญชี Logto Cloud ของคุณเพื่อเชื่อมต่อ

### ลูกค้า MCP อื่น ๆ \{#other-mcp-clients}

Logto MCP Server สามารถใช้งานร่วมกับลูกค้า MCP ใด ๆ ที่รองรับการอนุญาต OAuth ใช้การตั้งค่าต่อไปนี้:

- **MCP server URL:** `https://mcp.logto.io`
- **OAuth client ID:** `mcp-client`
- **Resource:** `https://mcp.logto.io/` (หากลูกค้าของคุณต้องการ)
- **Scope:** `mcp:all` (หากลูกค้าของคุณต้องการ)

หากคุณเป็นนักพัฒนาแอป AI และต้องการรองรับการเชื่อมต่อกับ Logto MCP Server แต่ Logto MCP Server ยังไม่รองรับ redirect URI ของแอปของคุณ กรุณา [ติดต่อเรา](https://logto.io/contact) พร้อมรายละเอียดแอปของคุณ

## สร้างเซิร์ฟเวอร์ MCP ระยะไกลของคุณเอง \{#build-your-own-remote-mcp-server}

Logto MCP Server ถูกสร้างขึ้นด้วย Logto และ [MCP Auth](https://mcp-auth.dev) หากคุณต้องการสร้างเซิร์ฟเวอร์ MCP ระยะไกลของคุณเองพร้อมการยืนยันตัวตน ดูแหล่งข้อมูลต่อไปนี้:

- [Remote MCP server in action: a new entry point for SaaS products in the AI era](https://blog.logto.io/remote-mcp-server-in-action)
- [เข้าร่วม Discord ของเรา](https://discord.gg/fUQmaFyAam): แบ่งปันประสบการณ์และแนวปฏิบัติที่ดีที่สุดในการสร้างเซิร์ฟเวอร์ MCP ระยะไกล
- [MCP Auth](https://mcp-auth.dev): ทุกสิ่งที่คุณต้องการเพื่อเพิ่มการยืนยันตัวตนระดับ production ให้กับเซิร์ฟเวอร์ MCP ของคุณ ไม่ต้องเสียเวลาหลายสัปดาห์อ่านสเปกหรือเชื่อมโยงระบบเอง
