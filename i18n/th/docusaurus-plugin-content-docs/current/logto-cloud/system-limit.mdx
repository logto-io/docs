# ขีดจำกัดของระบบ

ที่ Logto เรากำหนดขีดจำกัดที่เอื้อเฟื้อในทุกแผน พร้อมตัวเลือกแบบจ่ายตามการใช้งานจริง (pay-as-you-go) ที่ยืดหยุ่น เพื่อให้ผู้ใช้จ่ายเฉพาะสิ่งที่ใช้จริงเท่านั้น

คุณอาจสังเกตเห็นว่ารายการบางอย่างในหน้าราคา ถูกระบุว่า _ไม่จำกัด_ หรือ _จ่ายตามการใช้งานจริงอย่างต่อเนื่องโดยไม่มีเพดาน_ ซึ่งหมายความว่าสามารถใช้งานได้โดยทั่วไปโดยไม่มีข้อจำกัด แต่ Logto ขอสงวนสิทธิ์ในการปรับขีดจำกัดเหล่านี้ตามความเหมาะสมเพื่อรักษาการใช้งานที่เป็นธรรมสำหรับผู้ใช้ทุกคน กล่าวอีกนัยหนึ่ง ขีดจำกัดของเอนทิตีเป็นเพดานที่เข้มงวดเพื่อปกป้องสุขภาพโดยรวมของแพลตฟอร์ม ไม่ใช่ส่วนหนึ่งของราคา แม้อาจแตกต่างกันในแต่ละกลุ่มแผน

หากกรณีการใช้งานของคุณสมเหตุสมผลแต่ถึงขีดจำกัดของระบบ โปรด [ติดต่อเรา](https://logto.io/contact) และแบ่งปันความคิดเห็นของคุณ สิ่งนี้ช่วยให้เราเข้าใจรูปแบบการใช้งานจริงและปรับขีดจำกัดของระบบให้เหมาะสมยิ่งขึ้นเพื่อสนับสนุนลูกค้าประจำของเรา

## การป้องกันอัตราขีดจำกัดระดับผู้เช่า (Tenant) \{#tenant-level-rate-limit-protection}

### ผู้เช่า Dev \{#dev-tenants}

สำหรับ [ผู้เช่า Development](/logto-cloud/tenant-settings#development) ผู้ใช้สามารถใช้ประโยชน์จากฟีเจอร์และข้อเสนอฟรีของ Logto ได้ เพื่อป้องกันการใช้งานที่ผิดวัตถุประสงค์และกำหนดความคาดหวังที่ชัดเจน เราจึงกำหนดขีดจำกัดของระบบบางประการ ขีดจำกัดเหล่านี้ช่วยให้เราบริหารจัดการแพลตฟอร์มอย่างยั่งยืน ในขณะที่ยังคงให้เข้าถึงฟรีสำหรับการทดสอบและพัฒนา
