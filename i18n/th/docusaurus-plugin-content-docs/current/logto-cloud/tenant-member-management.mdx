# การจัดการสมาชิกผู้เช่า (Tenant member management)

## เชิญผู้ร่วมงาน \{#invite-collaborators}

ไปที่ <CloudLink to="/tenant-settings/members">Console > การตั้งค่าผู้เช่า > สมาชิก</CloudLink> คุณจะเห็นสมาชิกปัจจุบันของคุณและสามารถเชิญเพิ่มได้ กรอกอีเมลและกำหนดบทบาท (สามารถทำแบบกลุ่มได้) จากนั้นกดส่ง

Logto ให้คุณเชิญสมาชิกทีมเข้าสู่ผู้เช่าของคุณใน Logto Cloud ได้ 2 บทบาท:

- **Admin:** เข้าถึงและจัดการทรัพยากรของผู้เช่าได้ทั้งหมด
- **Collaborator:** จัดการทรัพยากรที่เกี่ยวข้องกับเอกลักษณ์ได้ แต่มีข้อจำกัดดังนี้:
  - ไม่มีสิทธิ์เข้าถึงข้อมูลการเรียกเก็บเงิน
  - ไม่สามารถเชิญหรือจัดการสมาชิกคนอื่นได้
  - ไม่สามารถเปลี่ยนชื่อผู้เช่าได้
  - ไม่สามารถลบผู้เช่าได้

### ข้อจำกัดจำนวนสมาชิก \{#members-limit}

| ประเภทผู้เช่า          | ขีดจำกัดสมาชิก                                  |
| ---------------------- | ----------------------------------------------- |
| Dev tenant             | สูงสุด 3 สมาชิก (ติดต่อเราหากต้องการเพิ่ม)      |
| Free production tenant | 1 สมาชิก (admin เท่านั้น)                       |
| Pro production tenant  | เริ่มต้น 3 สมาชิก จากนั้น $8 ต่อสมาชิกเพิ่มเติม |

### จัดการคำเชิญ \{#manage-invitation}

แท็บคำเชิญจะแสดงคำเชิญที่ผ่านมาและสถานะของแต่ละรายการ

| สถานะ    | คำอธิบาย                               | การดำเนินการที่ทำได้ |
| -------- | -------------------------------------- | -------------------- |
| Pending  | ส่งคำเชิญแล้ว แต่ยังไม่ได้รับการตอบรับ | ส่งซ้ำ               |
| Accepted | ผู้ใช้กลายเป็นสมาชิกผู้เช่าแล้ว        | ลบ                   |
| Expired  | คำเชิญหมดอายุเกิน 7 วัน                | ลบ                   |
| Revoked  | ผู้ดูแลระบบยกเลิกคำเชิญ                | ลบ                   |

คุณไม่สามารถเชิญผู้ที่มีคำเชิญค้างอยู่หรือเป็นสมาชิกอยู่แล้วได้

### จัดการสมาชิกผู้เช่า (Admin เท่านั้น) \{#manage-tenant-members-admin-only}

ในฐานะผู้ดูแลระบบ (admin) คุณสามารถจัดการทีมของคุณได้หลายวิธี:

- **เชิญหรือถอดสมาชิก:** เพิ่มสมาชิกทีมใหม่หรือถอดสิทธิ์การเข้าถึงตามต้องการ
- **เปลี่ยนบทบาทสมาชิก:** กำหนดบทบาท 'admin' หรือ 'collaborator' ตามระดับการเข้าถึงที่ต้องการ
- **โอนความเป็นเจ้าของผู้เช่า:** หากจำเป็น คุณสามารถโอนความเป็นเจ้าของผู้เช่าให้สมาชิกคนอื่นได้ โดยให้บทบาท admin กับสมาชิกใหม่ก่อน แล้วเลือกว่าจะอยู่ต่อในฐานะสมาชิกทั่วไปหรือออกจากผู้เช่าเลยก็ได้
- **เพิ่มผู้ดูแลระบบหลายคน:** แบ่งปันความรับผิดชอบด้านการจัดการกับสมาชิกทีมคนอื่นเพื่อการบริหารร่วมกัน
