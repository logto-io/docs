# ประเภทเหตุการณ์ของบันทึกการตรวจสอบ (Audit Logs)

คุณสามารถกรองประเภทเหตุการณ์ใน <CloudLink to="/audit-logs">Logto Console > บันทึกการตรวจสอบ (Audit Logs)</CloudLink> ได้

:::note

ขณะนี้ Logto รองรับการดึงบันทึกที่เกี่ยวข้องกับการโต้ตอบของผู้ใช้ปลายทางผ่าน [Experience APIs](https://openapi.logto.io/group/endpoint-experience)

บันทึกการตรวจสอบสำหรับ [Management APIs](/integrate-logto/interact-with-management-api) และ [Account APIs](https://openapi.logto.io/group/endpoint-account-center) จะมาเร็วๆ นี้

หากคุณต้องการแบ่งปันความต้องการของคุณ [ติดต่อเรา](https://logto.io/contact) ได้เลย

:::

## การแลกเปลี่ยนโทเค็น (Exchange token) \{#exchange-token}

| Key                               | Name                                             |
| --------------------------------- | ------------------------------------------------ |
| ExchangeTokenBy.AuthorizationCode | แลกเปลี่ยนโทเค็นด้วยรหัส (Code)                  |
| ExchangeTokenBy.ClientCredentials | แลกเปลี่ยนโทเค็นด้วย Client Credentials          |
| ExchangeTokenBy.RefreshToken      | แลกเปลี่ยนโทเค็นด้วยโทเค็นรีเฟรช (Refresh Token) |
| ExchangeTokenBy.TokenExchange     | การแลกเปลี่ยนโทเค็น                              |

## การอ้างสิทธิ์โทเค็นแบบกำหนดเอง (Custom token claims) \{#custom-token-claims}

| Key                            | Name                                                                 |
| ------------------------------ | -------------------------------------------------------------------- |
| JwtCustomizer.AccessToken      | รับการอ้างสิทธิ์โทเค็นการเข้าถึง (Access Token) ของผู้ใช้แบบกำหนดเอง |
| JwtCustomizer.ClientCredential | รับการอ้างสิทธิ์โทเค็นการเข้าถึง (Access Token) ของ M2M แบบกำหนดเอง  |
