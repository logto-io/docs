# CSS แบบกำหนดเอง

หาก [การปรับแต่ง UI อย่างละเอียด](/customization/match-your-brand) ของเราไม่ตรงกับแบรนด์ของคุณทั้งหมด คุณสามารถใช้ CSS แบบกำหนดเองเพื่อปรับแต่งประสบการณ์การลงชื่อเข้าใช้ของคุณได้ เพียงเพิ่มโค้ด CSS ของคุณลงในตัวแก้ไขของเราและดูการเปลี่ยนแปลงได้ทันที

## CSS แบบกำหนดเองใน Logto Console \{#custom-css-in-logto-console}

1. ไปที่ <CloudLink to="/sign-in-experience/branding">Console > ประสบการณ์การลงชื่อเข้าใช้ > การสร้างแบรนด์ > CSS แบบกำหนดเอง</CloudLink>
2. แก้ไขโค้ด CSS ในตัวแก้ไขด้านซ้าย ผลลัพธ์จะปรากฏแบบเรียลไทม์ในหน้าต่างแสดงตัวอย่างทางขวาทันที หมายเหตุ: ตัวแก้ไขโค้ดรองรับเฉพาะโค้ด CSS ในขณะนี้ ไม่รองรับ HTML หรือ Javascript
3. บันทึกการเปลี่ยนแปลง
4. อย่าลืมคลิกปุ่ม “[Live Preview](/customization/live-preview)” เพื่อตรวจสอบการเปลี่ยนแปลงในทุกหน้า

:::note
คุณยังสามารถตั้งค่า CSS แบบกำหนดเองในระดับแอปพลิเคชันและระดับองค์กรได้ที่หน้ารายละเอียดแอปพลิเคชันและหน้าการตั้งค่าองค์กรตามลำดับ ลำดับความสำคัญคือ:

CSS แบบกำหนดเองระดับองค์กร -> CSS แบบกำหนดเองระดับแอปพลิเคชัน -> CSS แบบกำหนดเองของประสบการณ์การลงชื่อเข้าใช้แบบ Omni
:::

หากต้องการดูการออกแบบอินเทอร์เฟซแบบแยกส่วนสำหรับ Logto คุณสามารถไปที่ [Logto/packages/experience/src](https://github.com/logto-io/logto/tree/master/packages/experience/src) บน GitHub หรือเปิดเครื่องมือ "Inspect Elements" ของเบราว์เซอร์ในหน้า Logto Live preview

## ตัวอย่าง \{#examples}

ตัวอย่างเช่น หากคุณต้องการให้หน้าลงชื่อเข้าใช้ของคุณมีบรรยากาศแบบ _Night City_ สำหรับมุมมองเว็บ ลองใช้ CSS นี้:

:::note

เนื่องจาก Logto ใช้ CSS Modules คุณอาจเห็นค่าแฮชใน property `class` ของ element ใน DOM (เช่น `<div>` ที่มี `vUugRG_container`) หากต้องการ override เหล่านี้ คุณสามารถใช้ตัวเลือก CSS `$=` เพื่อเลือก element ที่ลงท้ายด้วยค่าที่ระบุ ในกรณีนี้ควรใช้ `div[class$=container]`

:::

## CSS แบบกำหนดเองผ่าน Management API \{#custom-css-by-management-api}

คุณยังสามารถใช้ [Management API](https://openapi.logto.io/operation/operation-uploadcustomuiassets) `PATCH /api/sign-in-exp` พร้อม body `{ "customCss": "arbitrary string" }` เพื่อกำหนด CSS แบบกำหนดเองสำหรับประสบการณ์การลงชื่อเข้าใช้ คุณจะเห็นค่าของ `customCss` ถูกแนบหลัง `<title>` ของหน้า หาก style มีลำดับความสำคัญสูงกว่า ก็จะ override ได้
