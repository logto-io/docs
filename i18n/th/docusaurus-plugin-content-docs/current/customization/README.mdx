# การปรับแต่ง (Customization)

ทำให้ Logto เป็นของคุณเองด้วยการปรับแต่งทุกแง่มุม ตั้งแต่แบรนด์และส่วนติดต่อผู้ใช้ ไปจนถึงภาษาและการตั้งค่าโดเมน

## UI ที่สร้างไว้ล่วงหน้า (Prebuilt UI) \{#prebuilt-ui}

เริ่มต้นด้วยคอมโพเนนต์ UI ที่สร้างไว้ล่วงหน้าของเรา และปรับแต่งองค์ประกอบแบรนด์ที่สำคัญ สร้างได้ทั้ง:

- ประสบการณ์การยืนยันตัวตน (Authentication) ที่เป็นหนึ่งเดียวทั่วทั้งแพลตฟอร์มของคุณ
- หน้าลงชื่อเข้าใช้เฉพาะสำหรับแอปพลิเคชันหรือองค์กรแต่ละแห่ง

<DocCardList
  items={[
    {
      type: 'link',
      label: 'ปรับให้ตรงกับแบรนด์ของคุณ',
      href: '/customization/match-your-brand',
      description:
        'เรียนรู้วิธีปรับแต่งองค์ประกอบแบรนด์พื้นฐานสำหรับประสบการณ์ลงชื่อเข้าใช้ที่พร้อมใช้งาน',
      customProps: {
        icon: <CustomizationIcon />,
      },
    },
  ]}
/>

## UI แบบกำหนดเอง (Custom UI) \{#custom-ui}

ควบคุมประสบการณ์การยืนยันตัวตน (Authentication) ได้อย่างเต็มที่ด้วยสองตัวเลือก:

<DocCardList
  items={[
    {
      type: 'link',
      label: 'Custom CSS',
      href: '/customization/custom-css',
      description: 'ปรับแต่งองค์ประกอบภาพด้วยการใช้ CSS ของคุณเองผ่านตัวแก้ไขโค้ดของเรา',
      customProps: {
        icon: <BrushIcon />,
      },
    },
    {
      type: 'link',
      label: 'นำ UI ของคุณมาใช้',
      href: '/customization/bring-your-ui',
      description:
        'นำเสนออินเทอร์เฟซแบบกำหนดเองเต็มรูปแบบโดยอัปโหลด HTML, CSS, JavaScript และไฟล์ asset ของคุณเป็น ZIP ทั้งหมดโฮสต์อย่างปลอดภัยบน Logto Cloud',
      customProps: {
        icon: <FlaskIcon />,
      },
    },
  ]}
/>

## เนื้อหาและภาษา (Content & Languages) \{#content--languages}

Logto รองรับ 15 ภาษาโดยค่าเริ่มต้น ขยายการรองรับภาษา หรือปรับแต่งคำแปลที่มีอยู่ให้ตรงกับความต้องการของคุณ

<DocCardList
  items={[
    {
      type: 'link',
      label: 'ภาษาและเนื้อหา',
      href: '/customization/localized-languages',
      description: 'เรียนรู้วิธีเพิ่มภาษาใหม่และปรับแต่งเนื้อหาสำหรับผู้ใช้ของคุณ',
      customProps: {
        icon: <LanguagesIcon />,
      },
    },
  ]}
/>

## โดเมนแบบกำหนดเอง (Custom domain) \{#custom-domain}

เสริมสร้างภาพลักษณ์แบรนด์ของคุณด้วยการใช้โดเมนของคุณเองสำหรับ URL การยืนยันตัวตน (Authentication) แทนที่โดเมน Logto เริ่มต้นด้วยโดเมนแบรนด์ของคุณ

<DocCardList
  items={[
    {
      type: 'link',
      label: 'โดเมนแบบกำหนดเอง',
      href: '/logto-cloud/custom-domain',
      description:
        'เพิ่มความน่าเชื่อถือและความไว้วางใจของผู้ใช้ด้วยการใช้โดเมนของคุณเองเป็น URL สำหรับลงชื่อเข้าใช้หรือสมัครสมาชิก',
      customProps: {

      },

  ]}
/>
