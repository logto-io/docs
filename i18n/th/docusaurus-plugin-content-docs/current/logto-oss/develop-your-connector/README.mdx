# พัฒนาตัวเชื่อมต่อของคุณ

ปัจจุบัน Logto ได้จัดเตรียมตัวเชื่อมต่อสำหรับการลงชื่อเข้าใช้โซเชียลที่ได้รับความนิยมมากมาย เช่น WeChat, Alipay, Google, Facebook ฯลฯ รวมถึงตัวเชื่อมต่อ SMS / อีเมล เช่น AliCloud, SendGrid และ Twilio

นี่คือแผนภาพสถาปัตยกรรมที่แสดงให้เห็นว่าตัวเชื่อมต่อทำงานร่วมกับบริการหลักของ Logto อย่างไร

<img src="/img/assets/logto-connector-framework.png" alt="โครงสร้างตัวเชื่อมต่อ" />

ตัวเลือกเหล่านี้น่าจะตอบโจทย์ความต้องการของผู้ใช้ส่วนใหญ่ในขณะนี้ แต่เรายังคงพัฒนาอย่างต่อเนื่อง

ในขณะที่เรากำลังพัฒนาตัวเชื่อมต่อเพิ่มเติม คุณก็สามารถพัฒนาตัวเชื่อมต่อของคุณเองได้เช่นกัน และเราขอเชิญชวนให้คุณแบ่งปันตัวเชื่อมต่อของคุณกับชุมชนทั้งหมด

มาดูกันว่าคุณจะมีส่วนร่วมกับโปรเจกต์ Logto ได้อย่างไร โดยเริ่มจากการสร้างตัวเชื่อมต่อ
