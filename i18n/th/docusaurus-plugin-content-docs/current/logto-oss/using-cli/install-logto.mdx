# ติดตั้ง Logto

## แบบโต้ตอบ (Interactive) \{#interactive}

รันคำสั่งด้านล่างเพื่อดำเนินการติดตั้งแบบโต้ตอบในเทอร์มินัลของคุณ:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @logto@latest
```

  </TabItem>

</Tabs>

## ตัวเลือกและการติดตั้งแบบเงียบ (Options and silent installation) \{#options-and-silent-installation}

หรือคุณสามารถเพิ่มตัวเลือกใด ๆ ด้านล่างนี้ (ทั้งหมดเป็นทางเลือก) เพื่อข้ามคำถามบางข้อ:

| Option                                         | คำอธิบาย                                                                                                                                                                                                                                             |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--db-url`                                     | URL ของ Postgres สำหรับฐานข้อมูล Logto                                                                                                                                                                                                               |
| `-p, --path`                                   | พาธไปยังอินสแตนซ์ Logto ของคุณ ต้องเป็นพาธที่ยังไม่มีอยู่                                                                                                                                                                                            |
| `--ss`                                         | ข้ามการ seed ฐานข้อมูล Logto                                                                                                                                                                                                                         |
| `--disable-admin-pwned-password-check, --dapc` | seed tenant ผู้ดูแลระบบโดยปิดการตรวจสอบรหัสผ่านรั่วไหลผ่าน Have I Been Pwned (HIBP) สำหรับการติดตั้งแบบ air-gapped หรือออฟไลน์ ดู [Seed สำหรับการติดตั้งแบบ air-gapped หรือออฟไลน์](/logto-oss/using-cli#seed-for-air-gapped-or-offline-deployments) |

รัน `logto init --help` เพื่อดูข้อมูลช่วยเหลือแบบเต็ม

ตัวอย่างการติดตั้งแบบเงียบ (เช่น ใน CI):

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @logto/cli init -p ./logto --db-url postgresql://your-postgres-dsn:5432/logto --oc
```

  </TabItem>

</Tabs>

จะดำเนินการดังต่อไปนี้ตามลำดับ:

1. ดาวน์โหลดและแตกไฟล์ Logto ไปที่ `./logto`
2. พยายาม init และ seed ฐานข้อมูลด้วย URL `postgresql://your-postgres-dsn:5432/logto`
3. เพิ่มตัวเชื่อมต่อ (Connectors) อย่างเป็นทางการของ Logto ไปที่ `./logto/packages/core/connectors`
