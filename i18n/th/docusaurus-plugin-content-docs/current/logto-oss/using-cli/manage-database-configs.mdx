# จัดการคอนฟิกฐานข้อมูล

Logto ช่วยดูแลคอนฟิกทางเทคนิคบางอย่าง เช่น คีย์ OIDC และ Cookie ก่อนที่เราจะสามารถจัดการผ่าน Admin Console หรือ Management API ได้ คุณสามารถใช้ CLI เพื่อบรรลุเป้าหมายนี้

## ดึงคอนฟิกด้วยคีย์ \{#get-config-by-key}

การใช้งาน:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get <key> [keys...]
```

  </TabItem>

</Tabs>

สมมติว่าคุณต้องการดึง OIDC private keys:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config get oidc.privateKeys
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config get oidc.privateKeys
```

  </TabItem>

</Tabs>

ผลลัพธ์จะมีลักษณะดังนี้:

```bash
oidc.privateKeys=["\n-----BEGIN PRIVATE KEY-----\nMIIJRAIBA..."]
```

รัน `logto db config get --help` เพื่อดูคีย์ทั้งหมดที่มี หรือดู [Configuration](/concepts/core-service/configuration/#database-configs) สำหรับคำอธิบายโดยละเอียด

## ตั้งค่าคอนฟิกด้วยคีย์ \{#set-config-by-key}

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set <key> [keys...]
```

  </TabItem>

</Tabs>

สมมติว่าคุณต้องการตั้งค่า OIDC Cookie keys:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
logto db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @logto/cli db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>

</Tabs>

โดยปกติ ค่าที่จะตั้งควรเป็นสตริง JSON ที่ถูกต้อง และต้องเป็นไปตาม type definition ที่กำหนดไว้ รัน `logto db config set --help` เพื่อดูคีย์ทั้งหมดที่มี หรือดู [Configuration](/concepts/core-service/configuration/) สำหรับคำอธิบายโดยละเอียด
