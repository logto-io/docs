# การมีส่วนร่วม

ขอขอบคุณที่สนใจร่วมพัฒนา Logto! เราซาบซึ้งในทุกการมีส่วนร่วมจากชุมชน เพื่อทำให้โปรเจกต์ของเรามีความแข็งแกร่ง ใช้งานง่าย และหลากหลายมากยิ่งขึ้น ไม่ว่าคุณจะช่วยแก้ไขบั๊ก ปรับปรุงเอกสาร สร้างตัวเชื่อมต่อของคุณเอง เพิ่มฟีเจอร์ใหม่ หรือให้ข้อเสนอแนะ ความพยายามของคุณล้วนมีคุณค่าอย่างยิ่ง

## จะมีส่วนร่วมได้อย่างไร? \{#how-to-contribute}

เราพยายามทำให้การมีส่วนร่วมเป็นเรื่องง่ายที่สุด ก่อนเริ่มต้น กรุณาศึกษา [คู่มือการมีส่วนร่วมบน GitHub](https://github.com/logto-io/logto/blob/master/.github/CONTRIBUTING.md) ของเรา คู่มือนี้จะอธิบายขั้นตอน มาตรฐานการเขียนโค้ด และวิธีการมีส่วนร่วมกับ Logto อย่างประสบความสำเร็จ

## ต้องการความช่วยเหลือ? \{#need-help}

หากคุณมีคำถามหรือต้องการคำแนะนำ สามารถติดต่อเราได้ผ่าน [GitHub discussions](https://github.com/logto-io/logto/discussions), [เปิด issue ใหม่](https://github.com/logto-io/logto/issues) หรือ [ติดต่อเราโดยตรง](https://logto.io/contact) แล้วเราจะช่วยให้คุณมีทุกอย่างที่จำเป็นสำหรับความสำเร็จ
