# แนะนำเบื้องต้น

ยินดีต้อนรับสู่เอกสาร Logto! Logto คือโซลูชันการจัดการเอกลักษณ์และการเข้าถึง (IAM) ที่ออกแบบมาสำหรับแอปและผลิตภัณฑ์ SaaS ยุคใหม่ มอบระบบการยืนยันตัวตน (Authentication) และการอนุญาต (Authorization) ที่ปลอดภัย ขยายได้ และปรับแต่งได้สำหรับแอปพลิเคชันของคุณ
