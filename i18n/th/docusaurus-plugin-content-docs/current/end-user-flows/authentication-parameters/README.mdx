# พารามิเตอร์การยืนยันตัวตน (Authentication parameters)

ในกระบวนการลงชื่อเข้าใช้แบบ OIDC มาตรฐาน แอปพลิเคชันไคลเอนต์จะเริ่มต้นคำขอการยืนยันตัวตน (authentication request) ที่เปลี่ยนเส้นทางผู้ใช้ไปยังหน้าเว็บประสบการณ์การลงชื่อเข้าใช้แบบสากลที่โฮสต์โดย Logto ขึ้นอยู่กับการตั้งค่าประสบการณ์การลงชื่อเข้าใช้ของคุณ ผู้ใช้สามารถลงชื่อเข้าใช้หรือลงทะเบียนโดยใช้ตัวระบุ วิธีการยืนยัน และตัวเชื่อมต่อโซเชียลหรือ SSO สำหรับองค์กรจากภายนอกได้หลากหลาย

Logto ยังรองรับพารามิเตอร์ OIDC มาตรฐานเพิ่มเติม เช่น:

- `login_hint`: ให้คำแนะนำสำหรับตัวระบุผู้ใช้ เช่น เติมอีเมล / ชื่อผู้ใช้ล่วงหน้า
- [`ui_locales`](/end-user-flows/authentication-parameters/ui-locales): ควบคุมภาษาขณะใช้งานสำหรับการโต้ตอบปัจจุบัน รวมถึง UI การลงชื่อเข้าใช้และอีเมลที่เกี่ยวข้อง

นอกเหนือจาก [พารามิเตอร์การยืนยันตัวตน OIDC มาตรฐาน](https://openid.net/specs/openid-connect-core-1_0.html#AuthRequest) ผลิตภัณฑ์ของเรายังมีพารามิเตอร์การยืนยันตัวตนแบบกำหนดเองหลายรายการที่ช่วยให้คุณปรับแต่งประสบการณ์การลงชื่อเข้าใช้ที่ต้องการสำหรับผู้ใช้ปลายทาง

สิ่งนี้มีประโยชน์อย่างยิ่งเมื่อคุณต้องการบังคับใช้กระบวนการลงชื่อเข้าใช้เฉพาะสำหรับกลุ่มผู้ใช้ที่แตกต่างกัน รวมถึงแต่ไม่จำกัดเพียงสถานการณ์ต่อไปนี้:

### ลงทะเบียนโดยตรงสำหรับผู้ใช้ใหม่ \{#direct-sign-up-for-new-users}

    สำหรับแคมเปญการลงทะเบียนแบบเจาะจงกลุ่มเป้าหมาย คุณอาจต้องการนำผู้ใช้ใหม่ไปยังหน้าลงทะเบียนโดยตรง ข้ามฟอร์มลงชื่อเข้าใช้เริ่มต้น เพื่อให้ประสบการณ์เริ่มต้นใช้งานราบรื่น

    **ใช้**: พารามิเตอร์ [First screen](/end-user-flows/authentication-parameters/first-screen) → `first_screen=register`

### ลงทะเบียนโดยกรอกอีเมลล่วงหน้าจากลิงก์สมัครสมาชิก \{#email-prefilled-sign-up-from-a-subscription-link}

    หากคุณเก็บรวบรวมอีเมลผ่านฟอร์มสมัครสมาชิกที่โฮสต์เอง พารามิเตอร์เหล่านี้จะช่วยเปลี่ยนเส้นทางผู้สมัครใหม่ไปยังหน้าลงทะเบียนโดยมีช่องอีเมลกรอกไว้ล่วงหน้า ทำให้พวกเขาสามารถดำเนินการลงทะเบียนได้ง่ายขึ้น

    **ใช้**: พารามิเตอร์ [First screen](/end-user-flows/authentication-parameters/first-screen) → `first_screen=identifier:sign_up&identifier=email&login_hint=foo@gmail.com`

### ลิงก์รีเซ็ตรหัสผ่านโดยตรง \{#direct-password-reset-link}

    สำหรับผู้ใช้ใหม่ที่ต้องตั้งรหัสผ่านใหม่ ส่งลิงก์โดยตรงและเปลี่ยนเส้นทางไปยังหน้าตั้งรหัสผ่านใหม่โดยมีช่องอีเมลกรอกไว้ล่วงหน้า

    **ใช้**: พารามิเตอร์ [First screen](/end-user-flows/authentication-parameters/first-screen) → `first_screen=reset_password`

### เข้าสู่ระบบโซเชียลทันที \{#instant-social-login}

    หากคุณมีปุ่มลงชื่อเข้าใช้ด้วยโซเชียลบนเว็บไซต์โดยตรง คุณสามารถใช้พารามิเตอร์เหล่านี้เพื่อข้ามฟอร์มลงชื่อเข้าใช้เริ่มต้นและให้ผู้ใช้ยืนยันตัวตนกับผู้ให้บริการโซเชียลที่เลือกได้ทันที

    **ใช้**: พารามิเตอร์ [Direct sign-in](/end-user-flows/authentication-parameters/direct-sign-in#social-sign-in) → `direct_sign_in=social:google`

### ลงชื่อเข้าใช้ SSO โดยตรงสำหรับลูกค้าองค์กร \{#direct-sso-sign-in-for-enterprise-customers}

    สำหรับผลิตภัณฑ์ที่ให้บริการลูกค้าองค์กรหลายราย พารามิเตอร์เหล่านี้ช่วยให้คุณแสดงปุ่ม SSO เฉพาะที่ลงชื่อเข้าใช้ด้วยผู้ให้บริการ SSO ของแต่ละองค์กร ข้ามฟอร์มลงชื่อเข้าใช้สากลและมอบประสบการณ์ที่ราบรื่นสำหรับแต่ละองค์กร

    **ใช้**: พารามิเตอร์ [Direct sign-in](/end-user-flows/authentication-parameters/direct-sign-in#enterprise-sso) → `direct_sign_in=sso:1234567890`

### แสดงวิธีลงชื่อเข้าใช้ที่แตกต่างกันสำหรับแต่ละเว็บไซต์ \{#custom-sign-in-methods-displayed-for-different-sites}

    หากผลิตภัณฑ์ของคุณให้บริการทั้งทีมภายในและผู้บริโภค หรือองค์กรที่แตกต่างกัน คุณสามารถปรับแต่งประสบการณ์การลงชื่อเข้าใช้ให้เหมาะสมได้ เช่น - สำหรับเครื่องมือภายใน แสดงเฉพาะการลงชื่อเข้าใช้ด้วยอีเมล / ชื่อผู้ใช้และรหัสผ่าน - สำหรับหน้าเว็บไซต์ที่ให้บริการผู้บริโภค แสดงตัวเลือกการลงชื่อเข้าใช้ทั้งหมด รวมถึง Google login

    **ใช้**: การตั้งค่า [first screen](/end-user-flows/authentication-parameters/first-screen) หรือ [direct sign-in](/end-user-flows/authentication-parameters/direct-sign-in) ที่แตกต่างกันตามประเภทเว็บไซต์

อ่านต่อเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับพารามิเตอร์การยืนยันตัวตนเพื่อปรับแต่ง [first screen](/end-user-flows/authentication-parameters/first-screen) หรือเปิดใช้งานประสบการณ์ [direct sign-in](/end-user-flows/authentication-parameters/direct-sign-in) สำหรับผู้ใช้ของคุณ
