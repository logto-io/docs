# กระบวนการสำหรับผู้ใช้ปลายทาง

กระบวนการสำหรับผู้ใช้ปลายทางครอบคลุมกระบวนการตรวจสอบทั้งหมดที่เกี่ยวข้องกับการโต้ตอบของผู้ใช้ แบ่งออกเป็นหมวดหมู่ดังนี้:

- **กระบวนการยืนยันตัวตน (Authentication flows)**: เกี่ยวข้องกับการลงชื่อเข้าใช้ การลงทะเบียน และการรีเซ็ตรหัสผ่าน Logto มีประสบการณ์สำเร็จรูปที่ตั้งค่าได้ใน <CloudLink to="/sign-in-experience">Console > ประสบการณ์การลงชื่อเข้าใช้</CloudLink> และคุณสามารถ [นำ UI ของคุณมาใช้](/customization/bring-your-ui) ผ่าน [Experience APIs](https://openapi.logto.io/group/endpoint-experience)
- **กระบวนการบัญชี (Account flows)**: เกี่ยวข้องกับการตั้งค่าบัญชี โปรไฟล์ผู้ใช้ และการตรวจสอบความปลอดภัย ใช้ [Account APIs](https://openapi.logto.io/group/endpoint-account-center) หรือ [Management APIs](https://openapi.logto.io/operation/operation-getuser) เพื่อพัฒนา
- **กระบวนการองค์กร (Organization flows)**: สำหรับผลิตภัณฑ์แบบหลายผู้เช่า เช่น B2B หรือบริการ SaaS เกี่ยวข้องกับการสร้างองค์กร การเชิญสมาชิก และการจัดการ ใช้ [Management APIs](https://openapi.logto.io/operation/operation-listorganizations) เพื่อผสานรวมเข้ากับแอปของคุณ
