# ประสบการณ์องค์กร (Organization experience)

ประสบการณ์องค์กร (Organization experience) หมายถึงอินเทอร์เฟซผู้ใช้และโฟลว์ที่ใช้โดยลูกค้าธุรกิจของคุณและพนักงานของพวกเขา โดยเฉพาะใน [แอปพลิเคชันแบบหลายผู้เช่า (multi-tenant applications)](https://auth.wiki/multi-tenancy) Logto มีคู่มือหลายรายการเพื่อช่วยให้คุณผสานประสบการณ์นี้เข้ากับแอปของคุณอย่างรวดเร็วโดยใช้ Logto Management API
