#### ทรัพยากร API ขององค์กร (Organization API resources) \{#organization-api-resources}

หากต้องการดึงโทเค็นการเข้าถึง (Access token) สำหรับทรัพยากร API ในองค์กร คุณสามารถใช้เมธอด `GetAccessTokenWithOptions` โดยระบุทั้ง API resource และ organization ID เป็นพารามิเตอร์:

```go
accessToken, error := client.GetAccessTokenWithOptions(
  client.GetAccessTokenOptions{
    Resource: 'https://shopping.your-app.com/api',
    OrganizationId: organizationId,
  },
);
```
