# เพิ่มการยืนยันตัวตนให้กับแอป Passport.js ของคุณ (Add authentication to your Passport.js application)

คู่มือนี้จะแสดงวิธีผสาน Logto เข้ากับแอปพลิเคชันของคุณโดยใช้ [Passport.js](https://www.passportjs.org/) และกลยุทธ์ OIDC

<GuideTip />

## ข้อกำหนดเบื้องต้น \{#prerequisites}

- บัญชี [Logto Cloud](https://cloud.logto.io) หรือ [Logto แบบโฮสต์เอง](/introduction/set-up-logto-oss)
- สร้างแอปพลิเคชันแบบ traditional ใน Logto แล้ว
- โปรเจกต์ express ที่ตั้งค่า session เรียบร้อยแล้ว ดูรายละเอียดได้ที่ [เว็บไซต์ Express.js](https://expressjs.com/)
