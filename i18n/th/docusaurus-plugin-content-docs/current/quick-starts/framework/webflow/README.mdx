# เพิ่มการยืนยันตัวตนให้กับแอป Webflow ของคุณ (Add authentication to your Webflow application)

คู่มือนี้จะแสดงวิธีการผสาน Logto เข้ากับเว็บไซต์ Webflow ของคุณ

<GuideTip />

## ข้อกำหนดเบื้องต้น \{#prerequisits}

1. การผสาน Logto กับ Webflow จำเป็นต้องใช้ฟีเจอร์ "Custom code" ของ Webflow ซึ่งต้องมีอย่างน้อยแผน "Basic"
2. เว็บไซต์ Webflow โดยสามารถใช้เว็บไซต์ที่มีอยู่แล้วหรือสร้างใหม่ก็ได้
