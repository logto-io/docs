# เริ่มต้นใช้งานอย่างรวดเร็ว

ยินดีต้อนรับสู่คู่มือเริ่มต้นใช้งาน Logto อย่างรวดเร็ว เอกสารนี้ประกอบด้วยแนวทางที่เป็นประโยชน์สำหรับการผสาน Logto เข้ากับแอปพลิเคชันของคุณ

คู่มือส่วนใหญ่ใช้ SDK อย่างเป็นทางการของ Logto หากคุณใช้เครื่องมือพัฒนาแบบขับเคลื่อนด้วย AI ลองใช้ [Logto MCP Server](/logto-cloud/logto-mcp-server) ของเราเพื่อการผสานรวมที่มีผู้ช่วย AI

หากคุณมีคำถามใด ๆ โปรดติดต่อเราได้เสมอ

:::info
หน้านี้เน้นการผสาน Logto ฝั่งไคลเอนต์ รวมถึงรูปแบบ backend-for-frontend สำหรับการปกป้อง API ฝั่งเซิร์ฟเวอร์ ดูที่ [การปกป้อง API](/api-protection)
:::
