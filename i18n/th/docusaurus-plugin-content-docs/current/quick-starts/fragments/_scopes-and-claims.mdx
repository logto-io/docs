Logto ใช้แนวทาง [ขอบเขต (Scope) และ การอ้างสิทธิ์ (Claim) ของ OIDC](https://openid.net/specs/openid-connect-core-1_0.html#Claims) เพื่อกำหนดขอบเขตและการอ้างสิทธิ์สำหรับดึงข้อมูลผู้ใช้จากโทเค็น ID (ID token) และ OIDC <a href="https://openid.net/specs/openid-connect-core-1_0.html#UserInfo">จุดปลาย userinfo</a> ทั้ง "ขอบเขต (Scope)" และ "การอ้างสิทธิ์ (Claim)" เป็นคำศัพท์จากข้อกำหนดของ OAuth 2.0 และ OpenID Connect (OIDC)

สำหรับการอ้างสิทธิ์ OIDC มาตรฐาน การรวมอยู่ในโทเค็น ID จะถูกกำหนดอย่างเคร่งครัดโดยขอบเขตที่ร้องขอ การอ้างสิทธิ์เพิ่มเติม (เช่น `custom_data` และ `organizations`) สามารถตั้งค่าเพิ่มเติมให้ปรากฏในโทเค็น ID ได้ผ่าน <a href="/developers/custom-id-token">การตั้งค่า Custom ID token</a>
