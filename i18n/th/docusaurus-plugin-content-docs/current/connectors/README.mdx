# ตัวเชื่อมต่อ (Connector)

ตัวเชื่อมต่อ (Connectors) คือสะพานเชื่อมระหว่าง Logto กับบริการภายนอก พวกเขาช่วยให้สามารถใช้วิธี [การยืนยันตัวตนแบบไม่ใช้รหัสผ่าน (passwordless-verification)](https://auth.wiki/passwordless) เช่น [การยืนยันอีเมล](/end-user-flows/sign-up-and-sign-in/sign-in), [การยืนยัน SMS](/end-user-flows/sign-up-and-sign-in/sign-in), [การเข้าสู่ระบบโซเชียล](/end-user-flows/sign-up-and-sign-in/social-sign-in), และ [Enterprise SSO](/end-user-flows/enterprise-sso) ได้

การลงทะเบียนบัญชีแบบดั้งเดิมด้วยชื่อผู้ใช้และรหัสผ่านมักเป็นกระบวนการที่น่าเบื่อและสร้างความหงุดหงิด ผู้ใช้มักลืมรหัสผ่านที่ซับซ้อนและลังเลที่จะสร้างบัญชีใหม่เพราะความไม่สะดวก หากผู้ใช้สามารถระบุอีเมลหรือเบอร์โทรศัพท์และยืนยันผ่านรหัสยืนยัน หรือเข้าสู่ระบบด้วยบัญชีอย่าง Google / Facebook หรือบัญชีองค์กรอย่าง Microsoft จะช่วยให้กระบวนการเข้าสู่ระบบง่ายขึ้นมากและลดการสูญเสียผู้ใช้ ตัวเชื่อมต่อจึงถูกสร้างขึ้นมาเพื่อแก้ปัญหานี้
