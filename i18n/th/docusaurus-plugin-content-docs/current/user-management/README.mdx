# การจัดการผู้ใช้

การจัดการผู้ใช้เป็นประเด็นสำคัญที่ Logto ให้ความสำคัญ คุณสามารถเข้าถึงข้อมูลผู้ใช้และจัดการได้ผ่านทั้ง Logto Console หรือ Management API ซึ่งทั้งสองวิธีมีประสิทธิภาพ ด้วยเครื่องมือเหล่านี้ คุณสามารถดำเนินการต่าง ๆ ได้ เช่น

- ค้นหาผู้ใช้ด้วยชื่อผู้ใช้, รหัสผู้ใช้, ชื่อเต็ม, เบอร์โทรศัพท์ หรืออีเมล
- ดำเนินการต่าง ๆ กับผู้ใช้ เช่น ยกเลิกการเชื่อมต่อบัญชีโซเชียล, อัปเดตข้อมูลโปรไฟล์, ลบผู้ใช้ หรือรีเซ็ตรหัสผ่าน
- ตรวจสอบบันทึกกิจกรรมของผู้ใช้
- กำหนดบทบาท (Role) เพื่อควบคุมการเข้าถึง
- สร้างและจัดการโทเค็นการเข้าถึงส่วนบุคคล (Personal access token)
