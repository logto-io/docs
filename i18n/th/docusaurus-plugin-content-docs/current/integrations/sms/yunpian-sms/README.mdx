# ตัวเชื่อมต่อ YunPian SMS

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับบริการ YunPian SMS

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

YunPian เป็นผู้ให้บริการด้านการสื่อสารที่มีบริการหลากหลาย รวมถึง SMS ตัวเชื่อมต่อ YunPian SMS เป็นปลั๊กอินที่ทีม Logto จัดเตรียมไว้เพื่อเชื่อมต่อกับบริการ SMS ของ YunPian ทำให้ผู้ใช้ปลายทางของ Logto สามารถลงทะเบียนและลงชื่อเข้าใช้ผ่านรหัสยืนยันทาง SMS ได้

## ตั้งค่าบริการ SMS ใน YunPian Console \{#set-up-sms-service-in-yunpian-console}

### สร้างบัญชี YunPian \{#create-a-yunpian-account}

ไปที่ [เว็บไซต์ YunPian](https://www.yunpian.com/) เพื่อสมัครบัญชีและดำเนินการยืนยันตัวตนตามชื่อจริงให้เสร็จสมบูรณ์

### รับ API KEY \{#get-api-key}

1. เข้าสู่ระบบ YunPian Console
2. ไปที่ "การตั้งค่าบัญชี" -> "การจัดการบัญชีย่อย"
3. ค้นหาและคัดลอก API KEY

### กำหนดค่าเทมเพลต SMS \{#configure-sms-templates}

1. ใน YunPian Console ไปที่ "SMS ภายในประเทศ" -> "การยื่นขอลายเซ็น"
2. สร้างและส่งลายเซ็น รอการอนุมัติจากผู้ให้บริการเครือข่าย
3. ไปที่ "SMS ภายในประเทศ" -> "การยื่นขอเทมเพลต" และเลือก "รหัสยืนยัน"
4. สร้างเทมเพลตรหัสยืนยัน โดยต้องมีตัวแปร `#code#` (คุณสามารถใช้ "เทมเพลตทั่วไป" เพื่อเร่งกระบวนการอนุมัติ)
5. รอการอนุมัติเพิ่มเติมสำหรับเทมเพลต
6. หากต้องการส่ง SMS ระหว่างประเทศ ให้ทำขั้นตอนข้างต้นซ้ำ แต่เลือก "SMS ระหว่างประเทศ" -> "การยื่นขอเทมเพลต"

## กำหนดค่าใน Logto \{#configure-in-logto}

1. ใน Logto Console ไปที่ "ตัวเชื่อมต่อ"
2. ค้นหาและคลิก "บริการ YunPian SMS"
3. กรอกแบบฟอร์มการตั้งค่า:
   - API KEY: API KEY ที่ได้รับจาก YunPian
   - เทมเพลต SMS: กำหนดค่าเทมเพลตตามการใช้งาน โดยต้องตรงกับเทมเพลตที่ได้รับอนุมัติใน YunPian

## หมายเหตุ \{#notes}

1. เนื้อหาเทมเพลต SMS ต้องตรงกับเทมเพลตที่ได้รับอนุมัติใน YunPian ทุกประการ
2. ตัวแปรรหัสยืนยันในเทมเพลตของ YunPian คือ `#code#` ส่วนในการตั้งค่าตัวเชื่อมต่อคือ `{{code}}`
3. YunPian จะเพิ่มลายเซ็นเริ่มต้นโดยอัตโนมัติตาม API KEY ไม่จำเป็นต้องใส่ลายเซ็นในเนื้อหาเทมเพลต
4. แนะนำให้ทดสอบการตั้งค่าก่อนใช้งานจริง
