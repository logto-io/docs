# ตั้งค่าการยืนยัน SMS ด้วย GatewayAPI SMS

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับ GatewayAPI SMS

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

GatewayAPI เป็นผู้ให้บริการคลาวด์ในยุโรปที่ให้บริการคลาวด์หลากหลาย รวมถึง SMS (short message service) ตัวเชื่อมต่อ GatewayAPI SMS เป็นปลั๊กอินที่ทีม Logto จัดเตรียมไว้เพื่อเรียกใช้บริการ GatewayAPI SMS ซึ่งช่วยให้ผู้ใช้ปลายทาง Logto สามารถลงทะเบียนและลงชื่อเข้าใช้บัญชี Logto ของตนผ่านรหัสยืนยันทาง SMS ได้

## ตั้งค่าใน GatewayAPI \{#set-up-in-gatewayapi}

### สร้างบัญชี GatewayAPI \{#create-an-gatewayapi-account}

ไปที่ [เว็บไซต์ GatewayAPI](https://www.gatewayapi.com/) และลงทะเบียนบัญชี GatewayAPI ของคุณหากยังไม่มี

### เปิดใช้งานบัญชี \{#enable-account}

คุณอาจต้องเปิดใช้งานบัญชีก่อนใช้บริการ SMS สามารถติดต่อฝ่ายบริการลูกค้าของ GatewayAPI เพื่อเปิดใช้งานบัญชีของคุณ

### รับ API token \{#get-api-token}

ไปที่หน้าคีย์ API จากคอนโซล GatewayAPI และค้นหา API token หรือสร้าง API token ใหม่

## ตั้งค่าใน Logto \{#set-up-in-logto}

1. **Endpoint**: หากบัญชี GatewayAPI ของคุณอยู่ในภูมิภาค EU ให้ใช้ endpoint https://gatewayapi.com/rest/mtsms หากบัญชี GatewayAPI ของคุณอยู่ในภูมิภาค US ให้ใช้ endpoint https://gatewayapi.com/rest/mtsms
2. **API Token**: API token ที่คุณสร้างในขั้นตอนก่อนหน้า
3. **Sender**: ผู้ส่งที่คุณต้องการใช้ในการส่ง SMS
4. **Templates**: เทมเพลตที่คุณต้องการใช้ในการส่ง SMS คุณสามารถใช้เทมเพลตเริ่มต้นหรือปรับแต่งตามต้องการ
