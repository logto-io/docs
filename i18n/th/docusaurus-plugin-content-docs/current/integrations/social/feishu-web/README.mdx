# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย Feishu (Set up social login with Feishu)

Feishu Web: แพลตฟอร์มการทำงานร่วมกันและการจัดการองค์กรขั้นสูง มอบประสบการณ์การทำงานร่วมกันในสำนักงานอย่างไร้รอยต่อ และช่วยให้เป้าหมายของทีมสอดคล้องกันเพื่อปลดล็อกศักยภาพขององค์กรและบุคคลอย่างเต็มที่

<GuideTip />

## เริ่มต้นใช้งานการเข้าสู่ระบบโซเชียลด้วย Feishu \{#getting-started-with-feishu-social-sign-in}

ตัวเชื่อมต่อ Feishu web ถูกออกแบบมาสำหรับแอปพลิเคชันเว็บบนเดสก์ท็อป และใช้กระบวนการยืนยันตัวตน OAuth 2.0

## ลงทะเบียนบัญชีนักพัฒนา Feishu \{#register-a-feishu-developer-account}

หากคุณยังไม่มีบัญชีนักพัฒนา Feishu โปรดลงทะเบียนที่ [Feishu Open Platform](https://open.feishu.cn/app)

## สร้างแอปพลิเคชัน \{#create-an-application}

1. ที่ [Developer Console](https://open.feishu.cn/app) คลิก "Create Custom App"
2. กรอกชื่อแอปพลิเคชัน คำอธิบาย เลือกไอคอน และคลิกปุ่ม "Create"
3. ที่แถบด้านซ้าย คลิก "Security Settings" กรอก "Redirect URL" เป็น `${logto_endpoint}/callback/${connector_id}` โดยค่าที่เกี่ยวข้องสามารถดูได้ในฟิลด์ `Callback URI` บนหน้ารายละเอียดตัวเชื่อมต่อ Feishu ใน Logto Console
4. ใน "Credentials & Basic Info" คุณจะได้รับ "App ID" และ "App Secret"

:::note
สำหรับการใช้งานภายในที่ไม่ใช่องค์กร คุณต้องคลิกปุ่ม "Create a version" ในหน้า "Version Management and Release" ด้วย
"App ID" และ "App Secret" จะมีผลหลังจากสถานะแอปเปลี่ยนเป็น "Enabled"
:::
