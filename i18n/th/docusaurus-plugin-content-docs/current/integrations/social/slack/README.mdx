# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย Slack

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับการลงชื่อเข้าใช้โซเชียลผ่าน Slack

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ตัวเชื่อมต่อ Slack ช่วยให้ผู้ใช้ปลายทางสามารถลงชื่อเข้าใช้แอปพลิเคชันของคุณด้วยบัญชี Slack ของตนเองผ่านโปรโตคอลการยืนยันตัวตน OAuth 2.0 ของ Slack

<Integration />

## ทดสอบตัวเชื่อมต่อ Slack \{#test-slack-connector}

เรียบร้อยแล้ว ขณะนี้ตัวเชื่อมต่อ Slack ควรพร้อมใช้งาน อย่าลืม [เปิดใช้งานตัวเชื่อมต่อโซเชียลในประสบการณ์การลงชื่อเข้าใช้](/connectors/social-connectors/#enable-social-sign-in)
