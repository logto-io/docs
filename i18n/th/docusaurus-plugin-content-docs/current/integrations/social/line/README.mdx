# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย Line

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับการเข้าสู่ระบบโซเชียลด้วย Line

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ตัวเชื่อมต่อ Line ช่วยให้ผู้ใช้ปลายทางสามารถลงชื่อเข้าใช้แอปพลิเคชันของคุณด้วยบัญชี Line ของตนเองผ่านโปรโตคอลการยืนยันตัวตน OAuth 2.0 ของ Line

<Integration />

## ทดสอบตัวเชื่อมต่อ Line \{#test-line-connector}

เรียบร้อยแล้ว ขณะนี้ตัวเชื่อมต่อ Line ควรพร้อมใช้งาน อย่าลืม [เปิดใช้งานตัวเชื่อมต่อโซเชียลในประสบการณ์การลงชื่อเข้าใช้](/connectors/social-connectors/#enable-social-sign-in)
