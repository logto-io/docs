# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย WeCom (OAuth 2.0)

ตัวเชื่อมต่อแบบกำหนดเองสำหรับการเข้าสู่ระบบโซเชียล WeCom (หรืออาจเรียกว่า WXwork)

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ลงชื่อเข้าใช้ [WeCom WebUI](https://work.weixin.qq.com/) ด้วยบัญชีผู้ดูแลระบบ หรือคลิก **จัดการองค์กร (管理企业)** จากแอป WeCom

ในแท็บ "การจัดการแอป (App Management)" เลื่อนหน้าลงแล้วคลิก "สร้างแอป (Create an app)"

กรอกข้อมูลที่เหมาะสมตามแอปของคุณ จากนั้นสร้างแอป

ตอนนี้คุณจะได้ Agent ID (ไม่ใช่ APPID) และ Secret

### ข้อมูลเว็บไซต์ \{#website-info}

ตั้งค่ารายการที่คุณต้องการในหน้านี้ ตัวอย่างเช่น:

- ผู้ใช้ที่อนุญาต: _ใครสามารถเห็นแอปนี้_
- หน้าแรกของแอป: _หน้าแรกของแอปคุณ เช่น `logto.io/demo-app`_

**สำคัญ**
ในหน้านี้จะมี 3 รายการเกี่ยวกับ "Developer API (开发者接口)"

1. การอนุญาตเว็บและ JS-SDK;
2. เข้าสู่ระบบผ่านการอนุญาตโดย WeCom;
3. IP ที่เชื่อถือได้ขององค์กร;

กรอกข้อมูลตามคู่มือของ WeCom

### Corp ID \{#corp-id}

หากคุณคุ้นเคยกับการพัฒนา WeChat คุณอาจสังเกตได้ว่าการใช้ Corp ID เหมือนกับ APP ID

คุณสามารถดู Corp ID ได้ที่ด้านล่างของหน้าแท็บ "องค์กรของฉัน (我的企业)" โดยจะมีลักษณะเป็น **ww\*\*\*\*** .

## กำหนดค่าตัวเชื่อมต่อ \{#configure-the-connector}

ตอนนี้เรามี Agent ID, Secret และ Corp ID แล้ว

มาดำเนินการกรอกแบบฟอร์มสำหรับตัวเชื่อมต่อกัน

คุณสามารถเว้นช่อง `Scope` ไว้ได้เพราะเป็นตัวเลือก หรือจะกรอก `snsapi_base` หรือ `snsapi_privateinfo` ก็ได้

**บันทึกและเสร็จสิ้น**
