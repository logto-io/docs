# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย WeChat (Web) (Set up social login with WeChat (Web))

ตัวเชื่อมต่อ (Connector) อย่างเป็นทางการของ Logto สำหรับการเข้าสู่ระบบโซเชียล WeChat ในเว็บแอป

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

หากคุณยังไม่เข้าใจแนวคิดของตัวเชื่อมต่อ หรือไม่ทราบวิธีเพิ่มตัวเชื่อมต่อนี้ในประสบการณ์การลงชื่อเข้าใช้ของคุณ โปรดดู [คู่มือ Logto](/connectors/social-connectors)

:::caution

ตัวเชื่อมต่อนี้ใช้สำหรับเว็บแอปเท่านั้น หากคุณกำลังมองหาวิธีเข้าสู่ระบบด้วย WeChat ในแอปเนทีฟ โปรดดู [WeChat native connector](/integrations/wechat-native)

:::

## สร้างเว็บแอปใน WeChat Open Platform \{#create-a-web-app-in-the-wechat-open-platform}

:::tip

คุณสามารถข้ามบางส่วนได้หากดำเนินการเสร็จแล้ว

:::

### สร้างบัญชี \{#create-an-account}

เปิด https://open.weixin.qq.com/ คลิกปุ่ม "Sign Up" ที่มุมขวาบน แล้วดำเนินการสมัครให้เสร็จสมบูรณ์

### สร้างเว็บแอป \{#create-a-web-app}

ลงชื่อเข้าใช้ด้วยบัญชีที่คุณเพิ่งสร้าง ในแท็บ "Web Application" (网页应用) คลิกปุ่มสีเขียวขนาดใหญ่ "Create a web app" (创建网页应用)

<img src="/img/assets/app-tabs.png" alt="App tabs" />

กรอกข้อมูลที่จำเป็นในแบบฟอร์มสมัครแอปพลิเคชัน

<img src="/img/assets/create-web-app.png" alt="Create web app" />

#### ข้อมูลพื้นฐาน \{#basic-info}

ส่วนใหญ่ค่อนข้างตรงไปตรงมา หลังจากกรอกแบบฟอร์มเสร็จแล้ว คลิก "Next step" เพื่อดำเนินการต่อ

#### ข้อมูลเว็บไซต์ \{#website-info}

กรอก "Authorization callback domain" (授权回调域) ด้วยโดเมน Logto ของคุณ เช่น `logto.io`

#### รอผลการตรวจสอบ \{#waiting-for-the-review-result}

หลังจากกรอกข้อมูลเว็บไซต์แล้ว คลิก "Submit Review" เพื่อดำเนินการต่อ โดยปกติการตรวจสอบจะรวดเร็วและเสร็จสิ้นภายใน 1-2 วัน

เราสันนิษฐานว่าผู้ตรวจสอบจะถูกสุ่มในแต่ละครั้งที่ส่ง เนื่องจากมาตรฐานมีความผันผวน คุณอาจถูกปฏิเสธในครั้งแรก แต่ขอให้พยายามต่อไป! แจ้งสถานะของคุณและสอบถามผู้ตรวจสอบว่าควรแก้ไขอย่างไร

### กำหนดค่าตัวเชื่อมต่อ WeChat ของคุณ \{#configure-your-wechat-connector}

กรอกช่อง `clientId` และ `clientSecret` ด้วย _Client ID_ และ _Client Secret_ ที่คุณได้รับจากหน้ารายละเอียด OAuth app

กรอกช่อง `scope` ด้วย 'snsapi_userinfo' หรือ 'snsapi_base' คุณสามารถเว้นว่างช่องนี้ได้เพราะเป็นตัวเลือก (OPTIONAL) ค่าเริ่มต้นจะเป็น 'snsapi_userinfo' คุณสามารถตรวจสอบ [ความแตกต่าง](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/Wechat_webpage_authorization.html) ของแต่ละค่าได้

### ทดสอบตัวเชื่อมต่อ WeChat web \{#test-wechat-web-connector}

เรียบร้อยแล้ว อย่าลืม [เปิดใช้งานตัวเชื่อมต่อโซเชียลในประสบการณ์การลงชื่อเข้าใช้](/connectors/social-connectors/#enable-social-sign-in)

เมื่อเปิดใช้งานตัวเชื่อมต่อ WeChat web แล้ว คุณสามารถลงชื่อเข้าใช้แอปของคุณอีกครั้งเพื่อตรวจสอบการทำงาน
