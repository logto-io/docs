# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย OpenID Connect (OIDC)

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับโปรโตคอล OpenID Connect (OIDC)

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ตัวเชื่อมต่อ OIDC ช่วยให้ Logto เชื่อมต่อกับผู้ให้บริการข้อมูลระบุตัวตนโซเชียลใด ๆ ที่รองรับโปรโตคอล OIDC ใช้ตัวเชื่อมต่อนี้เพื่อให้แอปของคุณสามารถ:

- เพิ่มปุ่มเข้าสู่ระบบโซเชียล
- เชื่อมโยงบัญชีผู้ใช้กับตัวตนโซเชียล
- ซิงค์ข้อมูลโปรไฟล์ผู้ใช้จากผู้ให้บริการโซเชียล
- เข้าถึง API ของบุคคลที่สามผ่านการจัดเก็บโทเค็นอย่างปลอดภัยใน Logto Secret Vault สำหรับงานอัตโนมัติ (เช่น แก้ไข Google Docs, จัดการกิจกรรมปฏิทินในแอปของคุณ)

เพื่อเริ่มตั้งค่าฟีเจอร์การยืนยันตัวตนเหล่านี้ ให้สร้างตัวเชื่อมต่อ OIDC ใน Logto ก่อน:

1. ไปที่ <CloudLink to="/connectors/social">Logto console > Connector > Social connector</CloudLink>
2. คลิก **Add social connector** เลือก **OIDC** คลิก **Next** และทำตามขั้นตอนในบทช่วยสอนเพื่อดำเนินการเชื่อมต่อให้เสร็จสมบูรณ์

:::note

OIDC connector เป็นตัวเชื่อมต่อประเภทพิเศษใน Logto คุณสามารถเพิ่มตัวเชื่อมต่อที่ใช้โปรโตคอล OIDC ได้หลายตัว

:::
