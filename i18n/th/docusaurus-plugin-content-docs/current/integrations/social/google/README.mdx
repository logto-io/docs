# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย Google (Set up social login with Google)

ผสานระบบการยืนยันตัวตน OAuth 2.0 ของ Google เพื่อเปิดใช้งาน Sign-in with Google, การเชื่อมโยงบัญชี และการเข้าถึง Google APIs อย่างปลอดภัย

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ตัวเชื่อมต่อ Google (Google connector) ช่วยให้ผสาน OAuth 2.0 เพื่อให้แอปของคุณสามารถ:

- เพิ่มการยืนยันตัวตน “Sign-in with Google”
- เชื่อมโยงบัญชีผู้ใช้กับอัตลักษณ์ Google
- ซิงค์ข้อมูลโปรไฟล์ผู้ใช้จาก Google
- เข้าถึง Google APIs ผ่านการจัดเก็บโทเค็นอย่างปลอดภัยใน Logto Secret Vault สำหรับงานอัตโนมัติ (เช่น แก้ไข Google Docs, จัดการกิจกรรม Calendar ในแอปของคุณ)

ในการตั้งค่าฟีเจอร์การยืนยันตัวตนเหล่านี้ ให้สร้างตัวเชื่อมต่อ Google ใน Logto ก่อน:

1. ไปที่ <CloudLink to="/connectors/social">Logto console > Connector > Social connector</CloudLink>
2. คลิก **Add social connector** เลือก **Google** คลิก **Next** และทำตามขั้นตอนในบทเรียนเพื่อผสานระบบให้สมบูรณ์

<Integration />

## ใช้งานตัวเชื่อมต่อ Google (Utilize the Google connector) \{#utilize-the-google-connector}

เมื่อคุณสร้างตัวเชื่อมต่อ Google และเชื่อมต่อกับ Google แล้ว คุณสามารถนำไปใช้ใน flow สำหรับผู้ใช้ปลายทาง เลือกตัวเลือกที่ตรงกับความต้องการของคุณ:

### เปิดใช้งาน “Sign-in with Google” \{#enable-sign-in-with-google}

1. ใน Logto Console ไปที่ <CloudLink to="/sign-in-experience/sign-up-and-sign-in">Sign-in & account > Sign-up and sign-in</CloudLink>
2. เพิ่มตัวเชื่อมต่อ Google ในส่วน **Social sign-in** เพื่อให้ผู้ใช้ยืนยันตัวตนด้วย Google
3. เลือกเปิดใช้งาน **Google One Tap** บนหน้าลงชื่อเข้าใช้และสมัครสมาชิก เพื่อประสบการณ์การยืนยันตัวตนที่รวดเร็วขึ้น

เรียนรู้เพิ่มเติมเกี่ยวกับ [ประสบการณ์การเข้าสู่ระบบโซเชียล](/end-user-flows/sign-up-and-sign-in/social-sign-in)

### เชื่อมโยงหรือยกเลิกการเชื่อมโยงบัญชี Google \{#link-or-unlink-a-google-account}

ใช้ Account API เพื่อสร้าง Account Center แบบกำหนดเองในแอปของคุณ ให้ผู้ใช้ที่ลงชื่อเข้าใช้สามารถเชื่อมโยงหรือยกเลิกการเชื่อมโยงบัญชี Google ได้ [ดูวิธีในบทเรียน Account API](/end-user-flows/account-settings/by-account-api#link-a-new-social-connection)

:::tip
คุณสามารถเปิดใช้งานตัวเชื่อมต่อ Google เฉพาะสำหรับการเชื่อมโยงบัญชีและการเข้าถึง API โดยไม่ต้องเปิดใช้งานสำหรับ social sign-in ก็ได้
:::

### เข้าถึง Google APIs และดำเนินการต่าง ๆ \{#access-google-apis-and-perform-actions}

แอปของคุณสามารถดึงโทเค็นการเข้าถึง Google ที่จัดเก็บไว้ใน Secret Vault เพื่อเรียกใช้ Google APIs และทำงาน backend อัตโนมัติ (เช่น จัดการไฟล์ Google Drive, สร้างกิจกรรม Calendar หรือส่งอีเมลผ่าน Gmail) ดูคู่มือการดึงโทเค็นที่จัดเก็บไว้สำหรับการเข้าถึง API

## จัดการอัตลักษณ์ Google ของผู้ใช้ \{#manage-user-s-google-identity}

หลังจากผู้ใช้เชื่อมโยงบัญชี Google แล้ว ผู้ดูแลระบบสามารถจัดการการเชื่อมต่อนี้ใน Logto Console ได้ดังนี้:

1. ไปที่ <CloudLink to="/users">Logto console > User management</CloudLink> และเปิดโปรไฟล์ของผู้ใช้
2. ในส่วน **Social connections** ค้นหารายการ Google แล้วคลิก **Manage**
3. ในหน้านี้ ผู้ดูแลระบบสามารถจัดการการเชื่อมต่อ Google ของผู้ใช้ ดูข้อมูลโปรไฟล์ทั้งหมดที่ได้รับและซิงค์จากบัญชี Google และตรวจสอบสถานะ access token ได้
