# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย SAML

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับโปรโตคอล SAML

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ตัวเชื่อมต่อ SAML ช่วยให้ Logto เชื่อมต่อกับผู้ให้บริการข้อมูลระบุตัวตน (IdP) โซเชียลใด ๆ ที่รองรับโปรโตคอล SAML

:::note

ตัวเชื่อมต่อ SAML เป็นตัวเชื่อมต่อชนิดพิเศษใน Logto คุณสามารถเพิ่มตัวเชื่อมต่อที่ใช้ SAML ได้หลายตัว

:::
