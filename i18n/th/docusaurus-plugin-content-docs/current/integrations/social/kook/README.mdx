# ตั้งค่าการเข้าสู่ระบบโซเชียลด้วย KOOK

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับแอปเว็บลงชื่อเข้าใช้โซเชียล KOOK

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

ตัวเชื่อมต่อ KOOK ช่วยให้ผู้ใช้ปลายทางสามารถลงชื่อเข้าใช้แอปพลิเคชันของคุณโดยใช้บัญชี KOOK ของตนเองผ่านโปรโตคอลการยืนยันตัวตน OAuth 2.0 ของ KOOK
