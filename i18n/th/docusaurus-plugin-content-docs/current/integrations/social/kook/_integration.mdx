## สร้างแอปพลิเคชันใน KOOK Developer Center \{#create-an-application-in-the-kook-developer-center}

เคล็ดลับ: คุณสามารถข้ามบางส่วนได้หากดำเนินการเสร็จแล้ว

:::note
คุณต้องสมัครขอสิทธิ์ KOOK OAuth2 ก่อนสร้าง OAuth2 Application โปรดตรวจสอบข้อมูลในหน้าของ OAuth2 หรือสอบถามเจ้าหน้าที่ KOOK ในเซิร์ฟเวอร์นักพัฒนาอย่างเป็นทางการของ KOOK เพื่อข้อมูลเพิ่มเติม
:::

1. ไปที่ [KOOK Developer Center](https://developer.kookapp.cn/app/index) และลงชื่อเข้าใช้ด้วยบัญชี KOOK ของคุณ หากยังไม่มีบัญชีสามารถลงทะเบียนใหม่ได้
2. คลิกปุ่ม **Create Application** เพื่อสร้างแอปพลิเคชัน ตั้งชื่อให้แอป (เช่น LogtoAuth) แล้วคลิก **Confirm**
3. คลิกไอคอนแอปพลิเคชันที่คุณสร้าง แล้วไปที่หน้า **OAuth2**
4. เพิ่ม redirect ที่ถูกต้อง (เช่น `${your_logto_origin}/callback/${connector_id}`) ในช่อง **Callback URIs** คุณสามารถดู redirect uri ได้ในหน้ารายละเอียดตัวเชื่อมต่อใน logto admin console
5. อย่าลืมเลือก scope **get_user_info** ใน **OAuth2 link builder**

## กำหนดค่าตัวเชื่อมต่อ KOOK ของคุณ \{#configure-your-kook-connector}

กรอกข้อมูลในช่อง clientId และ clientSecret ด้วย Client ID และ Client Secret ที่คุณได้รับจากหน้าของ OAuth2 ที่กล่าวถึงในส่วนก่อนหน้า

### ประเภทของการกำหนดค่า \{#config-types}

| Name         | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
