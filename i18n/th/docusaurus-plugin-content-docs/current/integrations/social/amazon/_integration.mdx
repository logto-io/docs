### ตั้งค่าแอป Amazon \{#setup-a-amazon-app}

ไปที่ [Amazon Developer Portal](https://developer.amazon.com) และลงชื่อเข้าใช้ด้วยบัญชีของคุณ หากยังไม่มีบัญชี คุณสามารถลงทะเบียนได้

จากนั้น สร้างแอป

**ขั้นตอนที่ 1:** ไปที่บล็อก "Login with Amazon" ในหน้าแรก

หรือคุณสามารถเยี่ยมชม [หน้านี้](https://developer.amazon.com/loginwithamazon/console/site/lwa/overview.html)

ทำตามคำแนะนำเพื่อสร้าง security profile ใหม่

หลังจากสร้าง security profile แล้ว คุณจะได้รับ Client ID และ Client Secret จากตาราง

**ขั้นตอนที่ 2:** ตั้งค่า callback URLs

คลิกปุ่ม "Manage" ทางด้านขวาของตาราง และไปที่แท็บ "Web Settings"

ในช่อง "Allowed redirect URLs" ให้เพิ่ม URL ต่อไปนี้:

- `${your_logto_origin}/callback/${connector_id}`

`connector_id` สามารถดูได้ที่แถบด้านบนของหน้ารายละเอียดตัวเชื่อมต่อใน Logto Admin Console

### กำหนดค่าตัวเชื่อมต่อของคุณ \{#configure-your-connector}

ในการกำหนดค่าตัวเชื่อมต่อ Logto ของคุณ ให้กรอกข้อมูลในช่องต่อไปนี้ด้วยค่าที่ได้รับจากขั้นตอนก่อนหน้า:

- **clientId:** Client ID ของแอปของคุณ
- **clientSecret:** Client Secret ของแอปของคุณ

`scope` คือรายการ OIDC scopes ที่คั่นด้วยช่องว่าง หากไม่ได้ระบุ จะใช้ scope เริ่มต้นคือ `openid profile`

### ประเภทของ config \{#config-types}

| Name         | Type   |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
