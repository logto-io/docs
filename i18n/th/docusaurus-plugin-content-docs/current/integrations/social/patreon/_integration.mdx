## ลงชื่อเข้าใช้ด้วยบัญชี Patreon \{#sign-in-with-patreon-account}

ไปที่ [เว็บไซต์ Patreon](https://www.patreon.com/) และลงชื่อเข้าใช้ด้วยบัญชี Patreon ของคุณ หากยังไม่มีบัญชี คุณสามารถลงทะเบียนใหม่ได้

## สร้างและกำหนดค่าแอป OAuth \{#create-and-configure-oauth-app}

ทำตามคู่มือ [การสร้างแอป Patreon OAuth](https://www.patreon.com/portal/registration/register-clients) และลงทะเบียนแอปพลิเคชันใหม่

ตั้งชื่อแอป OAuth ใหม่ของคุณในช่อง **App Name** และกรอก **App URL** ของแอป คุณสามารถเว้นว่างช่อง **App Description** ได้ และปรับแต่ง **Redirect URIs** เป็น `${your_logto_origin}/callback/${connector_id}` โดยสามารถดู `connector_id` ได้ที่แถบด้านบนของหน้ารายละเอียดตัวเชื่อมต่อใน Logto Admin Console

:::note
หากคุณพบข้อความแสดงข้อผิดพลาดว่า "The redirect_uri MUST match the registered callback URL for this application." ขณะเข้าสู่ระบบ ให้ลองปรับค่า Redirect URI ของแอป Patreon OAuth และ URL สำหรับ redirect ของแอป Logto ของคุณ (รวมถึง protocol) ให้ตรงกันเพื่อแก้ไขปัญหา
:::

## จัดการแอป OAuth \{#managing-oauth-apps}

ไปที่ [หน้า Clients & API Keys](https://www.patreon.com/portal/registration/register-clients) บน Patreon ซึ่งคุณสามารถเพิ่ม แก้ไข หรือลบแอป OAuth ที่มีอยู่ได้ คุณยังสามารถดู `Client ID` และสร้าง `Client secrets` ได้ในหน้ารายละเอียดของแอป OAuth

## กำหนดค่าตัวเชื่อมต่อของคุณ \{#configure-your-connector}

กรอกข้อมูลในช่อง `clientId` และ `clientSecret` ด้วย _Client ID_ และ _Client Secret_ ที่คุณได้รับจากหน้ารายละเอียดแอป OAuth ตามที่กล่าวไว้ในส่วนก่อนหน้า

`scope` คือรายการ [ขอบเขต (scopes)](https://docs.patreon.com/#scopes) ที่คั่นด้วยช่องว่าง หากไม่ได้ระบุ scope จะตั้งค่าเริ่มต้นเป็น `identity identity[email]`

### ประเภทของการกำหนดค่า \{#config-types}

| ชื่อ         | ประเภท |
| ------------ | ------ |
| clientId     | string |
| clientSecret | string |
| scope        | string |
