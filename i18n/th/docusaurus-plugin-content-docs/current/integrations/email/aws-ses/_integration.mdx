## ตั้งค่าบริการอีเมลใน AWS Service Console \{#configure-a-mail-service-in-the-aws-service-console}

### ลงทะเบียนบัญชี AWS \{#register-aws-account}

ไปที่ [AWS](https://aws.amazon.com/) และลงทะเบียนบัญชี

### สร้างอัตลักษณ์ (Identity) \{#create-a-identity}

- ไปที่คอนโซล `Amazon Simple Email Service`
- สร้างอัตลักษณ์ โดยเลือกหนึ่งในตัวเลือกต่อไปนี้
  - สร้างโดเมน
  - สร้างที่อยู่อีเมล

### การตั้งค่าตัวเชื่อมต่อ (Connector) \{#configuration-of-the-connector}

1. คลิกชื่อผู้ใช้ของคุณที่มุมขวาบนของคอนโซล Amazon เพื่อเข้าสู่ `Security Credentials` หากยังไม่มี ให้สร้าง `AccessKey` และเก็บรักษาไว้ให้ดี
2. กรอกการตั้งค่าของตัวเชื่อมต่อ `Amazon Simple Email Service` ให้สมบูรณ์:
   - ใช้ `AccessKey ID` และ `AccessKey Secret` ที่ได้จากขั้นตอนที่ 1 กรอกใน `accessKeyId` และ `accessKeySecret` ตามลำดับ
   - `region`: กรอกช่อง `region` ด้วยภูมิภาคของอัตลักษณ์ที่คุณใช้ส่งอีเมล
   - `emailAddress`: ที่อยู่อีเมลที่คุณใช้ส่งอีเมล รูปแบบเช่น `Logto\<noreply@logto.io>` หรือ `\<noreply@logto.io>`

พารามิเตอร์ต่อไปนี้เป็นทางเลือก สามารถดูคำอธิบายพารามิเตอร์ได้ใน [AWS SES API documentation](https://docs.aws.amazon.com/ses/latest/APIReference-V2/API_SendEmail.html)

- `feedbackForwardingEmailAddress`
- `feedbackForwardingEmailAddressIdentityArn`
- `configurationSetName`

### ทดสอบตัวเชื่อมต่อ Amazon SES \{#test-the-amazon-ses-connector}

คุณสามารถกรอกที่อยู่อีเมลและคลิก "Send" เพื่อตรวจสอบว่าการตั้งค่าทำงานหรือไม่ก่อนจะ "Save and Done"

เรียบร้อยแล้ว อย่าลืม <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">เปิดใช้งานตัวเชื่อมต่อในประสบการณ์การลงชื่อเข้าใช้</MainSiteUrl>

### ประเภทการตั้งค่า \{#configure-types}

| ชื่อ                                      | ประเภท            |
| ----------------------------------------- | ----------------- |
| accessKeyId                               | string            |
| accessKeySecret                           | string            |
| region                                    | string            |
| emailAddress                              | string (OPTIONAL) |
| emailAddressIdentityArn                   | string (OPTIONAL) |
| feedbackForwardingEmailAddress            | string (OPTIONAL) |
| feedbackForwardingEmailAddressIdentityArn | string (OPTIONAL) |
| configurationSetName                      | string (OPTIONAL) |
| templates                                 | Template[]        |

| คุณสมบัติของ Template | ประเภท      | ค่าที่เป็นไปได้                                      |
| --------------------- | ----------- | ---------------------------------------------------- |
| subject               | string      | N/A                                                  |
| content               | string      | N/A                                                  |
| usageType             | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
