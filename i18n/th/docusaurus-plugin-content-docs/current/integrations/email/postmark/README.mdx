# ตั้งค่าการยืนยันอีเมลด้วย Postmark (Set up email verification with Postmark email service)

ตัวเชื่อมต่อ Logto สำหรับบริการอีเมล Postmark

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

Postmark คือแพลตฟอร์มอีเมลสำหรับอีเมลธุรกรรมและการตลาด เราสามารถใช้ฟังก์ชันส่งอีเมลของมันเพื่อส่ง _รหัสยืนยัน_

## ลงทะเบียนบัญชี Postmark \{#register-postmark-account}

สร้างบัญชีใหม่ที่ [เว็บไซต์ Postmark](https://postmark.com/) คุณสามารถข้ามขั้นตอนนี้ได้หากมีบัญชีอยู่แล้ว

## กำหนดค่าตัวเชื่อมต่อของคุณ \{#configure-your-connector}

กรอกช่อง `serverToken` ด้วย Server Token ที่คุณพบในหน้าการตั้งค่าของ server ใน Postmark

กรอกช่อง `fromEmail` ด้วย _From Address_ ของผู้ส่ง

เพื่อให้สามารถใช้งาน flow ของผู้ใช้ได้ครบถ้วน จำเป็นต้องมีเทมเพลตที่มี usageType เป็น `Register`, `SignIn`, `ForgotPassword` และ `Generic`

ตัวอย่าง JSON ของเทมเพลตตัวเชื่อมต่อ Postmark มีดังนี้

```jsonc
[
  {
    "usageType": "Register",
    "templateAlias": "logto-register",
  },
  {
    "usageType": "SignIn",
    "templateAlias": "logto-sign-in",
  },
  {
    "usageType": "ForgotPassword",
    "templateAlias": "logto-forgot-password",
  },
  {
    "usageType": "Generic",
    "templateAlias": "logto-generic",
  },
]
```

## ทดสอบตัวเชื่อมต่ออีเมล Postmark \{#test-postmark-email-connector}

คุณสามารถกรอกที่อยู่อีเมลแล้วคลิก "Send" เพื่อดูว่าการตั้งค่าสามารถใช้งานได้หรือไม่ ก่อนจะ "Save and Done"

เรียบร้อยแล้ว อย่าลืม <MainSiteUrl href="/connectors/email-connectors/#enable-email-sign-up-or-sign-in">เปิดใช้งานตัวเชื่อมต่อในประสบการณ์การลงชื่อเข้าใช้</MainSiteUrl>

## ประเภทของการกำหนดค่า \{#config-types}

| ชื่อ        | ประเภท |
| ----------- | ------ |
| serverToken | string |
| fromEmail   | string |

| คุณสมบัติของเทมเพลต | ประเภท      | ค่าที่เป็นไปได้                                      |
| ------------------- | ----------- | ---------------------------------------------------- |
| usageType           | enum string | 'Register' / 'SignIn' / 'ForgotPassword' / 'Generic' |
| templateAlias       | string      | N/A                                                  |
