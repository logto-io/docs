# ตั้งค่าการยืนยันอีเมลด้วย SendGrid Email

ตัวเชื่อมต่อ Logto อย่างเป็นทางการสำหรับบริการอีเมล SendGrid

<GuideTip />

## เริ่มต้นใช้งาน \{#get-started}

SendGrid (หรือ Twilio SendGrid) เป็นแพลตฟอร์มการสื่อสารกับลูกค้าสำหรับอีเมลธุรกรรมและอีเมลการตลาด เราสามารถใช้ฟังก์ชันส่งอีเมลของมันเพื่อส่ง _รหัสยืนยัน_
