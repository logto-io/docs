1. ไปที่ [Microsoft Entra admin center](https://entra.microsoft.com/) และลงชื่อเข้าใช้ในฐานะผู้ดูแลระบบ

2. ไปที่ Identity > Applications > App registrations

<center>
  <img src="/img/assets/entra_id_oidc_create_application.webp" alt="สร้างแอปพลิเคชัน" />
</center>

3. เลือก `New registration`

4. กรอกชื่อแอปพลิเคชันและเลือกประเภทบัญชีที่เหมาะสมสำหรับแอปของคุณ

5. เลือก `Web` เป็นแพลตฟอร์มของแอปพลิเคชัน

6. คัดลอกและวาง `redirect URI` จากหน้าการตั้งค่า SSO ของ Logto โดย `redirect URI` คือ URL ที่ผู้ใช้จะถูกเปลี่ยนเส้นทางไปหลังจากยืนยันตัวตนกับ Microsoft Entra ID แล้ว

<center>
  <img src="/img/assets/entra_id_oidc_config_application.webp" alt="กำหนดค่าแอปพลิเคชัน" />
</center>

6. คลิก `Register` เพื่อสร้างแอปพลิเคชัน
