# กรณีการใช้งาน

Logto เป็นโครงสร้างพื้นฐานการยืนยันตัวตน (auth infrastructure) และโซลูชันการจัดการเอกลักษณ์และการเข้าถึง (IAM) ที่ทรงพลัง ซึ่งสามารถนำไปใช้ในหลากหลายสถานการณ์ เราได้สร้างชุดกรณีการใช้งานเพื่อช่วยให้คุณเข้าใจวิธีใช้ประโยชน์จาก Logto ในบริบทที่แตกต่างกัน
