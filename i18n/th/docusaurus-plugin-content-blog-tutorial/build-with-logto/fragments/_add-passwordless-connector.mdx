ตัวเชื่อมต่อ {props.connectorType} เป็นวิธีที่ใช้ในการส่งรหัสผ่านครั้งเดียว (OTP) สำหรับการยืนยันตัวตน (Authentication) โดยช่วยให้สามารถยืนยัน {props.passwordlessSignUpIdentifier} เพื่อรองรับการยืนยันตัวตนแบบไม่ใช้รหัสผ่าน รวมถึงการลงทะเบียนด้วย {props.connectorType} การลงชื่อเข้าใช้ การยืนยันตัวตนสองปัจจัย (2FA) และการกู้คืนบัญชี

คุณสามารถเชื่อมต่อ {props.connector} เป็นผู้ให้บริการ {props.connectorType} ของคุณได้อย่างง่ายดาย ด้วยตัวเชื่อมต่อ {props.connectorType} ของ Logto คุณสามารถตั้งค่าสิ่งนี้ได้ในไม่กี่นาที

ในการเพิ่มตัวเชื่อมต่อ {props.connectorType} ให้ทำตามขั้นตอนดังนี้:

1. ไปที่ [Console > Connector > ตัวเชื่อมต่ออีเมลและ SMS](https://cloud.logto.io/to/connectors/passwordless)
2. หากต้องการเพิ่มตัวเชื่อมต่อ {props.connectorType} ใหม่ ให้คลิกปุ่ม "Set up" และเลือก "{props.connector}"
3. ตรวจสอบเอกสาร README สำหรับผู้ให้บริการที่คุณเลือก
4. กรอกข้อมูลการตั้งค่าในส่วน "Parameter Configuration"
5. ปรับแต่งเทมเพลต {props.connectorType} ด้วย JSON editor
6. ทดสอบการตั้งค่าของคุณโดยการส่งรหัสยืนยันไปยัง {props.passwordlessSignUpIdentifier} ของคุณ
