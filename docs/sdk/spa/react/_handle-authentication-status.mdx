In Logto SDK, generally we can use `logtoClient.isAuthenticated` to check the authentication status, if the user is signed in, the value will be `true`, otherwise, the value will be `false`.

In Logto React SDK, the `isAuthenticated` status can be checked by using the `useLogto` hook. In the example code below, we can use `isAuthenticated` to programmatically show and hide the sign-in and sign-out buttons.

```tsx
const Home = () => {
  const { isAuthenticated, signIn, signOut } = useLogto();

  return (
    <div>
      {isAuthenticated ? (
        <button onClick={signOut}>Sign Out</button>
      ) : (
        <button onClick={() => signIn('http://localhost:3000/callback')}>Sign In</button>
      )}
    </div>
  );
};
```
