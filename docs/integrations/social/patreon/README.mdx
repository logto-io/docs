# Set up social login with Patreon

The official Logto connector for Patreon social sign-in.

<GuideTip />

## Get started \{#get-started}

The Patreon connector enables end-users to sign in to your application using their own Patreon accounts via the Patreon OAuth 2.0 authentication protocol.
