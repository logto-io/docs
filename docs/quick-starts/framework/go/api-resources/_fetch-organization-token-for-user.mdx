#### Organization API resources \{#organization-api-resources}

To fetch an access token for an API resource in an organization, you can use the `GetAccessTokenWithOptions` method with both the API resource and organization ID as parameters:

```go
accessToken, error := client.GetAccessTokenWithOptions(
  client.GetAccessTokenOptions{
    Resource: 'https://shopping.your-app.com/api',
    OrganizationId: organizationId,
  },
);
```
